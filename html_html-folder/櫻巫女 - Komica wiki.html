介紹[編輯]

  • 2018年8月1日在Youtube開始活動的女性虛擬Youtuber，現為hololive旗下所屬藝人。
    • 原為COVER旗下「さくらみこプロジェクト」的獨立Vtuber，剛開始活動時主要以投稿短片以及夾雜小量直播為主。某次到Hololive辦事時(約一期生還在面試的時期)被YAGOO親自遞上名片招募同年12月25日正式加入hololive並開始以直播為中心活動。
      • YAGOO:跟我簽訂契約成為hololive的VT吧( ◕‿‿◕ )
  • 設定是在電腦世界裡面侍奉神明的巫女，被神派遣到日本交流，現在一邊完成努力差事一邊以成為虛擬偶像巫女為目標。
    • 人物頭髮和主色調為接近櫻花的粉紅色，主造型帶有巫女服的特色。但露肩、側乳和緞帶內褲這點色氣元素就是另一回事了
      • 然而新版3D把緞帶內褲換成了毛內褲，巫女本人表示自己是不會放棄緞帶內褲的。
    • 然而現在看回已經變成hololive混沌勢之一了，雖然在台上的樣子還是會令YAGOO一時迴光返照
    • 插畫中常常會出現一隻紅色和一到多隻白色的兩種生物，白色的為觀眾們35P的化身最廣為人知，紅色的則是一隻叫做金時的式神由於鮮少被提到導致很多觀眾不曉得那是甚麼。
    • 之後35自行補完了設定，是被神明派去跑腿到秋葉原買餐包時遇到了空媽才決定想要當偶像，結果除了把原本幫神明跑腿買的麵包自行吃掉以外還不回去了。
      • 星姊:神明大人快把他開除掉。

個人資料[編輯]

  • 個性活潑和喜歡有趣的事物，但同時也喜歡事情變得混沌。YAGOO長期留在ICU的原因之一
  • 聲音偏向一點年幼的奶音，有時候講話時連日本人自己都聽不懂他在說甚麼甚至被戲稱是菁櫻語與公主的嬰語並列Holo中最難理解的語言。
    • 曾有一次扮演嬰兒的聲音被觀眾開玩笑說跟平常的聲音沒兩樣。
    • 不過小時候曾因為聲音突出被欺負過一度不想開口說話，直到參加校園內的廣播被稱讚聲音後而有了自信並成為日後成為實況主的契機。
    • 也曾因為聲音特殊高中時曾被誤會是很做作的人，但同學們實際上跟他聊過後才發現他根本是個大叔。
    • 實際上在Holo裡面也是個非常容易親近的人，比起同樣都是0期生雖然沒有打算散發前輩氣場卻讓後輩過於尊敬反而覺得有隔閡在的空媽，巫女活潑容易相處的態度反而讓很多後輩容易親近甚至不當他是前輩來尊敬。
    • 事實上也能發出一把大姐姐聲線。
  • 喜歡鯛魚燒、芝士白身魚和味噌鍋，最近也提到喜歡吃麥當勞的雙層吉士堡，但討厭青椒、紅椒及綠豆之類的食物。
    • 亦因此持有一台特別的鯛魚燒外形手機。倒不如說是能打電話的鯛魚燒更貼切
  • 喜歡玩沙盒類遊戲如Minecraft、ARK和GTA等。由其ARK更是達到廢人程度更被官方關注其推特。

    • 公布3D模組2.0版本的同時宣布販售以巫女為女主角的視覺小說遊戲，喜歡GALGAME的極致就是自己化身為遊戲裡的可攻略腳色。
      • 不過內容很短，慢慢跑半小時內也能跑完。
    • 喜歡玩GALGAME的契機是看了我的妹妹哪有那麼可愛的關係，喜歡女主角桐乃的關係進而對桐乃玩的GALGAME產生興趣，另外也是蘿莉控+妹控。
