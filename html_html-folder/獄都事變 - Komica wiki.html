解說[編輯]

  • 由リンネ堂於2014年12月30日發布的一款免費不恐怖解謎冒險遊戲，遊戲由RPG Maker VX軟體所製。
    • 注意：本作品包含了部分流血、獵奇等相關描寫。
  • 為作者的處女作，相較於一般恐怖遊戲有更多動作要素，遊戲整體難度不高。
  • 在近年蘿莉正太主角當道中少數以青年為主角的恐怖遊戲。
  • 另外一個特色為辨別度低的角色頭像。
    • 每位角色臉型幾乎相同且髮型都被帽子蓋住[1]。
    • 能明確分辨角色的部分只剩下眼睛顏色。
    • 鼻子完全省略。
  • 目前最新版本為1.05。
  • 已知有韓文、中文、英文、俄文、西班牙文的翻譯版本。
  • 共有兩個END，分歧點在劇情末端。
    • 分歧在於有無把某物交給同伴修理並物歸原主。
  • 於2015年5月27日漫畫化，漫画：あおのなち　原作：リンネ堂　原作協力：SUNPLANT。
    • 角色全長出鼻子，帥氣度大增
  • 於2016年7月27日移植到RPG製作大師MV。
    • 可用手機遊玩，在電腦上遊玩則可用滑鼠操作。

故事簡介[編輯]


「將亡者逮捕過來。」




遵照這個指示的斬島來到了棲息著怪談的廢棄學校－－




角色[編輯]

斬島（きりしま kirishima）[編輯]

  • 玩家操作角色，本故事的主角。
  • 獄卒，眼睛為藍紫色，黑色短髮和稍長的鬢角。
  • 身高178cm/體重68kg[2]。
  • 因為同事木舌讓追捕的亡者逃脫，被派往亡者逃進的廢校收拾爛攤子。
  • 個性認真且過於率直，因為太坦率而完全不會說謊。
  • 冷靜沉著且力量強大，讓人覺得非常可靠。
    • 使恐怖遊戲一點也不恐怖的原因之一。
  • 也有身為獄卒毫不留情的一面，又踹又砍是家常便飯。
    • 會處罰惡作劇的妖怪。
    • 會把工作中打瞌睡的同事踹到醒。
  • 興趣是鍛鍊身體。
  • 體力值459點[3]，氣值59點[4]。
  • 右撇子，武器為日本刀，刀名「カナキリ」(漢化為「金切」[5])。
  • 漫畫裡提及其生前因為擁有藍眼睛被村人視為怪物，其母親為了保護他不被村人迫害而死。

肋角（ろっかく rokkaku）[編輯]

  • 獄卒們的上司。
  • 眼睛為紅色，皮膚黝黑。
  • 身高198cm，身材結實。
  • 指派斬島接替木舌的工作，之後就一直待在辦公室看書看風景
  • 作為獄卒們的老師，同時也是有如父親一樣的存在。
    • 主角一輩的獄卒據說都是被他撿回來的。
  • 未持有武器，但即使所有獄卒一起上也打不過他，非常強大。
  • 看起來很嚇人，但其實很會照顧人，不過生氣起來還是很恐怖。
  • 喜歡抽菸，燃燒出的煙霧為紫色。
  • 其書櫃裡放了能幫主角回血的OK繃，能搜刮到99個上限。

佐疫（さえき saeki）[編輯]

  • 獄卒，主角的親友。
  • 眼睛為天空色，頭髮是灰色系的焦茶色。
  • 身高177cm，體型比斬島纖細。
  • 遊戲中因為未當班而留守在獄卒機關裡。
    • 但之後為了幫助斬島解謎也前往廢校。
  • 和和風的主角對應，設計原型是西洋風的天使。
    • 設計原型是「吹號角宣告是芥末日世界末日的天使」，所以還真的帶著號角(位於腰間)。
  • 個性溫柔又聰明，教科書一般的優等生。
  • 興趣是音樂，平常會練鋼琴。
  • エンジェル佐疫 [6]
    • 是唯一積極表現出想幫助主角的角色。
    • 但是，主角會坦率的回答不需要幫忙。
  • 異常高的女子力。
  • 病嬌的潛力股。
  • 肩上披著昭和時代學生服的披肩，是個四次元口袋。
  • 武器為槍，遊戲中未登場。
    • 作者設定其披風下藏了各種類型的槍枝，但並非無限。

田噛（たがみ tagami）[編輯]

  • 獄卒，被派來支援斬島，平腹的搭檔。
  • 只會打混
  • 眼睛為橘色，黑髮，靠近右眼的瀏海比較長。
  • 身高176cm，體型和斬島差不多。
  • 很怕麻煩，總是有氣無力的樣子，其實頭腦很好。
    • 但聰明都用在逃避麻煩事上。
    • 口頭禪是「真沒勁...」。
  • 興趣是彈吉他。
  • 武器為鶴嘴鎬[7]，遊戲中被平腹擅自拿走還搞丟了。
    • 因此氣得把對方打飛出去。
    • 還回來後還是把平腹揍了一頓。
    • 漫畫版中把清開課桌椅的工作推給平腹時主動把鶴嘴鎬給他使用。
  • 和平腹是「阿吽[8]」般的關係。
  • 遊戲後期交給斬島道具後完全放棄工作直接回本部了。
    • 代替後來到廢校的佐疫站在肋角辦公室外的走廊。
  • 雖然態度上完全沒有想工作的幹勁，但給了主角不少好懂的提示。

平腹（ひらはら hirahara）[編輯]

  • 獄卒，被派來支援斬島，田嚙的搭檔。
  • 和上面一樣只會打混
  • 眼睛為暗黃色，髮色是接近橘色的亮茶色，髮型是看起來就很擅長運動的那種短髮。
  • 身高179cm，體型比斬島強壯一點。
  • 感情起伏非常激烈又輕率行事的人
  • 像小孩子般憑著好奇心行動，講話不會考慮先後。
  • 一旦找不到有趣的事就會失去幹勁，開始摸魚打瞌睡。
    • 即使值勤中也會豪邁的趴在地板上睡覺。
    • 遊戲中如果粗暴的叫醒他，會發生大慘事。
    • 漫畫第一回就引發大慘事。
  • 興趣是漫畫和遊戲(的樣子)。
  • 武器為鏟子，但空手也有很強的戰鬥力。
    • 怪力屬性，身體比頭腦更早行動的類型。
    • 對應用右手拿著鶴嘴鎬的田嚙而把鏟子拿在左手，但作者稱並不是左撇子。
  • 和田嚙是「阿吽」般的關係。

谷裂（たにざき tanizaki）[編輯]

  • 獄卒，被派來支援斬島。眼睛為紫色，光頭。
  • 身高182cm，所有獄卒中肌肉最發達的一位。
  • 有在做事但貢獻度比前面兩人還低
  • 遊戲前半段都站在學校外守門，負責清除接近廢校的魑魅魍魎。
    • 但是沒有出過廢校的斬島和玩家看不到。
    • 嘴巴很壞，每次看到主角都會說教。
  • 很頑固，是對自己和別人都很嚴厲的類型。只要是敵對者都會不容辯解的徹底解決掉。
    • 可說是所有登場獄卒中最接近「鬼」形象的角色。
  • 孤獨一匹狼。
  • 興趣是鍛鍊身體，視斬島為勁敵(但斬島不是很在意)。
  • 武器為紫色的狼牙棒。
  • 主角Game Over時負責收屍的人。

木舌（きのした kinoshita）[編輯]

  • 獄卒，因追捕亡者而失聯。
  • 砲灰(噓)
  • 身高188cm，是獄卒中體型最高大的，僅次於肋角。
  • 眼睛為綠色，七三分的黑髮。
  • 設計原型是金太郎那樣溫柔的大力士。
  • 是個不拘小節的人，臉上總是掛著笑容。
  • 再次登場時大概是整個遊戲最恐怖的地方。
  • 追捕亡者時因為一時大意被對方奪去雙眼，之後瞬間移動到廢校[9]靠聽覺發現前來的斬島，請他找來兩顆綠色眼球塞回去後跟個沒事人一樣。
  • 武器是大斧，遊戲中未登場。
  • 是個酒鬼，在官方公認的twitter小劇場裡總是離不開酒的話題。
    • 對酒喜愛的程度到「會在吃刨冰時淋上啤酒」。
    • 因為喝太多身上脂肪量比較高。
  • 在獄卒中是大哥哥般的存在。

???/真希 (マキ maki) [編輯]

  • 亡者，獄卒們追捕的對象。
  • 路人
  • 髮色和服飾都很明亮，膚色也比獄卒們看來更健康。
  • 興趣是花之類的很有女孩子氣的東西。
  • 其生前的回憶散落在廢校各處。
    • 成長過程受到母親[10]、同學、同事的無端指責和謾罵，不理解大家為何這麼討厭自己。[11]
    • 生前唯一理解她的人只有同學優子，其在真希生日時曾送她一串項鍊作為禮物。
    • 然而優子後來因車禍而死。在職場上遭同事造謠攻擊的真希得知此事後疑因打擊過大而跳樓，享年27歲。
    • 成為亡者後將過去欺負自己的人一一殺害[12]
  • 被木舌追捕到無路可逃後反奪走對方雙眼
  • 逃進廢校後似乎在尋找弄丟的項鍊而四處徘徊。
  • 沒有收集齊金屬器的碎片拼湊回項鍊的話，會因為聽不進斬島的勸阻執迷不悟，最後被斬島強行捉回，接受嚴厲的懲罰，下場不明[13]。
  • 反之如果收集齊金屬器的碎片並修復成項鍊交給她的話，會想起生前與優子的約定，化解怨念老實地跟斬島離開。之後根據肋角的說法是「雖然免不了懲罰但看在有悔意的份上，能從輕發落；之後就會返回輪迴，應該能夠過上更靠譜的人生吧」。
  • 俺は悪くねぇ！

斬島?（きりしま? kirishima）(引きずり込まれる鏡) [編輯]

  • 鏡妖，不是居住在鏡中而是有生命的鏡子。
    • 變化為經過鏡前的人，並透過這種方式奪取對方的存在。
    • 被奪取者會被拉入鏡中的廢校。
    • 實力在怪談中算強，但還是不比本尊的斬島。
  • 雖然和平腹吵過一架，但是對斬島卻一言不發。
    • 因為是鏡子，不會說話也是沒辦法的事。
    • 作者表示鏡子的本質是模仿，如果不斷和他說話也許會變得能說也不一定。
    • 意思是平腹無聊到和鏡子說話了嗎。
  • 因為會出現本體不會出現的表情，意外的人氣。
    • 一定是多了鼻子的關係

錐華 (キリカ kirika) [14][編輯]

  • 僅出現在漫畫中。
  • 炊事員的蛇女。

綾子 (アヤコ ayako) [15][編輯]

  • 僅出現在漫畫中。
  • 浣衣的雙口女。(頭背面似乎有一張嘴)。

災藤[編輯]

  • 僅出現在漫畫中，官方部落格有作者繪製的頭圖。
  • 身高192公分，灰藍色的眼睛，灰白色的頭髮。
  • 肋角的補佐官以及特務室的副長。
  • 優雅、神秘，平時很溫柔但生起氣來很可怕。
  • 沒有特別愛用的武器但很強。
  • 教佐疫鋼琴的老師。
  • 興趣是購物。

廢校裡的怪談[編輯]

骨骼模型(しゃべる骸骨)[編輯]

  • 會說話的骨骼模型，位於理科教室。
  • 廢校最後的良心，提供主角各種情報。
    • 但是，主角大多時候會坦率的回答不需要幫忙。
    • 在鏡中世界崩解後，被脫困的人體模型拆成碎片。
    • 因為像素的關係很多玩家沒看出碎片的正體。

人體模型(動く人体模型)[編輯]

  • 會動的人體模型，關在理科教室隔壁的準備室。
    • 因為廢校是座小學，所以是小孩子的人體模型。
  • 非常兇暴而且無法溝通。
  • 因為會拆解骨骼模型而被關起來
    • 之後被骨骼模型取代成為理科教室之主。
  • 本名是「みきお」，不良少年屬性。

黑球(ひとりでに弾むボール)[編輯]

  • 獨自彈跳的黑球。
    • 原本被作者設定成人頭，但因為後來會被戳爆，才改成球。
  • 筋肉腦組合平腹&斬島最初的犧牲者
  • 其實只是撿了人家掉的鶴嘴鋤。

無人的腳步聲(誰もいないのに聞こえる足音)[編輯]

  • 明明沒有任何人在卻聽得到的腳步聲。
  • 膽小又愛出風頭的孩子。
    • 跟在斬島身後，真面目是一雙紅色的腳。
    • 每次斬島在走廊上聽到腳步聲時要選擇「回頭」，循環三次才能看到他。

廁所的花子(トイレの花子さん)[編輯]

  • 學校怪談的定番。
  • 水邊賢明的少女。
    • 會無視陰沈的孩子。真希：「...」。
    • 只會和天真直率的孩子玩耍。斬島：「...」。

保健室的收藏癖(保健室の収集癖)[編輯]

  • 眼睛忽大忽小的黑色球體。
  • 叫一聲就能對話的能力。
  • 即使不出聲也能對話的能力。
  • 廢校共通，玩弄主角之後迅速逃離現場的能力。

眼球妖怪(目玉おばけ)[編輯]

  • 全身都是眼球的黑團，興趣是收集眼球。
  • 其實只是撿了人家掉的眼球。
  • 不會攻擊，又一個不還手被砍到死的可憐妖怪。

廢校之眼(廃校の目)[編輯]

  • 舊校舍本身變成的妖怪。
    • 難怪會廢校。
  • 可以自由改變校舍的內部。
    • 像是弄亂教室結構、在黑板上寫字之類的。
    • 各話負責寫標題的人。
    • 黑板左下有人體模型君寫著みきお（Mikio）和はな[16]（Hana）的情人傘。
  • 對亡者和獄卒的追逐戰很有興趣，一直在窺視著。

用語[編輯]

亡者[編輯]

  • 死去的人類的化身，外表還維持著生前的模樣。

獄卒[編輯]

  • 給予死後脫離輪迴、危害人類的亡者懲罰的地獄的鬼。
  • 有從長角、長尖牙到和人類長的一模一樣的各種種族。遊戲中只有長得和人類一樣的種族登場，男生的服飾為昭和時期的男學生服，此外也有女性的獄卒。
  • 沒有生死的概念，即使是致死的重傷，只要經過一定的時間就能再生。

獄都[編輯]

  • 那個世界的首都。混雜著昭和時期的懷舊和近未來的文化。獄卒平常居住的地方。
