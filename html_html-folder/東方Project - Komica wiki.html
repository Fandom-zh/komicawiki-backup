簡介[編輯]

  • 由上海愛莉絲幻樂團（上海アリス幻樂団）出品的一系列同人遊戲，從初代至今有超過20年歷史。
    • 類型為「巫女飛行射擊遊戲」
  • 系列原作者為ZUN（太田順也，又稱神主）
    • 主題曲：童祭 ～ Innocent Treasures（幻創神主・ZUN入退場テーマ）
  • 以獨特的世界觀、角色和音樂而著名，目前同人界中最有影響力的原創作品。
    • 舊五作（WIN系列前）的ACG二創成分較明顯[2]。例如靈夢的原型就有人認為是源自TAITO發行的巫女STG《奇奇怪界》的主角小夜。
      • 夢時空靈夢結局還把某戀愛遊戲的女主角搬進來
      • 某些曲子的既視感略重，夢時空某張魔理沙立繪還是描圖自蘭斯4的ブラックナース（慎入）
    • WIN系列開始各角色典故大都出自日本神話或妖怪，很少放大量的ACG捏他。
    • 製作者ZUN身兼程式、音樂、美術、劇本，幾乎一人包辦整個遊戲製作。但萃夢想等不是射擊遊戲的系列作則和同人社團黃昏邊境合作開發。
  • 也算是STG界最知名的作品系列之一，但由於其主要人氣並非集中在遊戲面上，而導致一部分STG玩家對東方的粉絲圈有些意見甚至相當不以為然。結果現在TAITO和CAVE都要推出東方官方授權的遊戲了......
    • 但也有人認為要不是東方這IP的知名度夠高，憑其他STGIP的知名度根本拉不了多少人入STG這個(ry
    • 因為豐富的二創而帶來大量不關心STG甚至連體驗版都不碰的エアプ，對粉絲間或粉絲與非粉絲的交流造成負面影響。
    • 有些人還會變成在各種與東方無關的場合中插入東方相關話題的東方廚。例如拿東方四處和其他射擊遊戲比較難度，但實際上在整個STG圈中，東方系列的難度差不多位於入門級。
      • 不過在Undertale廚出現後就出現了雙方廚互比難度的狀況，然而Tobyfox本人就是東方眾還和ZUN聊過天，所以還是別戰了吧。
  • 曾經和TYPE-MOON的月姬與龍騎士07的暮蟬悲鳴時並列同人界三大作品，並且和V家、偶像大師合稱NICONICO御三家。
    • 不過和TYPE-MOON與龍騎士07作品都有被改編成動畫不同，神主本人對東方改編動畫的態度十分保留，其中一個原因是神主認為製作動畫已超出自己能力範圍必須委託他人處理，但這讓神主有自己的作品不再由自己掌控的感覺[3]。
      • 另外有粉絲認為動畫是最容易收看但卻也是最花成本的製作媒介，且一旦製作就很可能導致角色定型化並失去了創作的空間[4]。
        • 舉一個有點極端的例子：作為主角的靈夢在不少同人都有極度貪財且容易為此耍小手段的性格，但在原作中關於這點並不是那麼明顯，然而動畫化時若為了增加趣味而去強化這方面的演出，就可能讓本來維持在二創，支持與不支持的都可以自由創作的靈夢性格變成半官設，導致其他人創作的時候很容易碰上動畫黨警察被指指點點說「這角色不是這個樣子」「這角色應該要這個樣子」，徒增困擾。
      • 更麻煩的情況下甚至可能招來角色黑，想想看灰姑娘女孩的未央因為動畫改編而招來多少黑大概就能略知一二了
    • 因此現階段東方依舊沒有官方的改編動畫，就算是幻想萬華鏡與跟奧運一樣每四年出現一個新作的夢想夏鄉也都強調自己是二次或三次創作。
  • 大多角色和ZUN一樣嗜酒
  • 雖然不少二次創作有濃厚的百合味，但原作中完全沒有百合成份。
  • 2010年後，因為艦隊Collection、ラブライブ！等新作品崛起，加上偶像大師也藉著灰姑娘女孩和百萬人演唱會等作品成功開發新客群，雖然本家仍然維持每一到兩年推出新作，但本身作為STG入門有一定難度，二創影片也隨著NICO退燒而逐漸變少，加上隨時間經過相關話題的新鮮度也降低，因此新入坑者人數減少而稍微變得比較影薄，但從Comiket的攤數以及bilibili的點閱來看依然保有一定人氣。
    • 因此支持者在中國被戲稱越共只要神主一聲令下，前東方民都會馬上趕回來
    • 跟艦隊收藏一樣成天被喊過氣，但一有相關動態就會趕來一堆人
  • 從天空璋開始，系列作品陸續在Steam上架
    • 不過到目前為止Steam版沒有任何新增要素，基本上就只是多了一個網路通販的管道，不只沒翻譯沒支援成就系統，連Bug都不修[5]

歷代作品一覽[編輯]

PC-98時代遊戲部份[編輯]

PC-98為平台發表的作品，發行時的社團名稱為「ZUN Soft」。
由於當時PC-98已處於衰退期，這幾款遊戲並不出名，但受到日後Windows上新作的影響而重拾注目。
目前遊戲商品已經絕版，硬體亦不易取得，但有心的玩家可透過模擬器一睹其風貌。
畫面表現和Win時代前期比起來異常瞎眼，尤其背景有炫技過頭的疑慮，還會妨礙到物件辨識度

Windows時代遊戲部份[編輯]

作者停滯了四年[8] 後，繼續以Windows為平台發行作品，並將社團名改為現在的上海アリス幻樂団。
遊戲整體水準較舊作有大幅度的進步，尤其背景音樂與人物設定有優異的表現，在同人遊戲之中是極為出色的作品，也帶起許多同人社團的二次創作風潮。

常用語[編輯]

符卡（Spell Card, 簡稱SC）[編輯]

  • 為敵人使用的一種特殊彈幕攻擊，發動中會在畫面上出現彈幕的名稱及特殊的背景。沒有前述演出的彈幕則通稱非符
  • 敵方發動SC時，如果在擊破前我方沒有使用炸彈（或靈擊等消彈或無敵技）也沒有被擊中，就可以獲得額外的分數
    • 唯一的例外是天空璋的季節解放，就算在符卡戰使用也不會判定為收取失敗（體驗版的春季解放例外）
  • 少部份SC是被敵方單方面攻擊只能閃躲直到時間結束，統稱耐久彈幕、耐久符、時符
  • 一部分作品有滿足特定條件才會出現的符卡，如Last Spell和Last Word等

命點[編輯]

  • 以電腦的角度來說，那一點叫判定點。
  • 從妖妖夢開始按shift鍵低速移動時可以看見使用的角色身上出現一個圓點，子彈只有碰到那點才算被擊中並扣殘機。
  • 不管命點有沒有顯示，命中判定永遠只有那一點。

擦彈/Graze[編輯]

  • 指與子彈擦身而過，可賺取大量分數或刻符（永夜抄中的一種獎勵點，又稱時點）。
  • 少數遊戲（如風神錄與鬼形獸）為例外，角色擦彈只會發出特殊音效而不會加分或取得任何道具。

炸彈、靈擊[編輯]

  • 跟一般射擊遊戲一樣是自機使用的廣範圍攻擊，發動炸彈時自機無敵且可消去敵人的彈幕轉為道具，在某些作品中可以將畫面中的道具吸過來
  • 通常對最後一關BOSS的最後一個SC與EXTRA關卡BOSS的所有SC無法造成傷害（還是有除彈效果），且對炸彈發動中我方射出的子彈也無敵
  • 部分作品看玩家所選裝備來決定炸彈，有些作品甚至連自機處在高速還是低速也會影響炸彈性能
  • 在永夜抄之前各自機的炸彈就是該角色的符卡，但風神錄、地靈殿則取消此設定，直到星蓮船才重新採用
    • 風神錄中炸彈改稱為靈擊，無論選哪個自機或裝備都是往前方射出一個大型圓環，在BOSS戰時使用靈擊會額外產生全畫面的攻擊判定
      • 靈夢的靈擊範圍大時間短，魔理沙的範圍小但滯留時間較長
    • 地靈殿稱為遠隔靈擊，性能看所選夥伴（使用的攻擊方式）決定
  • 在中彈後一瞬間放炸彈的話可以不掉殘機繼續進行遊戲
  • 在妖妖夢等部分遊戲中，靈擊指滿足特定條件時才能使用的特殊消彈技，跟炸彈是不同的東西

殘機的碎片/炸彈的碎片[編輯]

  • 在地靈殿中引入的新系統，為紫色星形，蒐集五個可獲得一個殘機。輝針城後則改為粉紅色心形。
  • 於星蓮船則追加炸彈的碎片，為綠色星形，蒐集三個可獲得一個炸彈。
  • 神靈廟之後的作品大都有沿用上述系統，但每作集滿一個殘機或炸彈所需的碎片數不一定相同。
    • 殘機所需碎片數：5（地、紺ex）、3（星、輝、紺、鬼、虹）、8~25（神，隨取得次數上升）
    • 炸彈所需碎片數：3（星、鬼、虹）、8（神、輝）、5（紺、天）
  • 也有直接加殘機和炸彈的道具，但出現次數較少。

火力道具[編輯]

  • 外表為一個紅色的P，又稱P道具或P點。每吃到一定量則自機的火力上升，永夜抄以前是本體射出的子彈變多，風神錄以後改為增加射出子彈的子機。
  • 有時會出現比較大的P，火力上升的幅度也較多。在特殊條件下更會出現直接讓火力上升到最大值的金色F道具。
  • P道具的上升值與火力上下限依遊戲發展可分為三個時期：
    • 紅、妖、永：下限為0，上限為128。小P+1，大P+8。
    • 風、地：下限為0.00，上限為5.00（風）/4.00（地）。小P+0.05，大P+1.00。但地靈殿中魔理沙選愛莉絲當夥伴的場合則小P上升值和火力上限會增加。
    • 星蓮船以後：下限為1.00，上限為4.00。小P+0.01，大P+1.00。不過部分遊戲有例外。

藍色得分道具[編輯]

  • 外表為一個藍色的点，又稱藍點或點道具。吃到時會增加分數。
  • 妖妖夢與永夜抄每吃到一定數量的藍點會增加一個殘機，紅、風、天則是每當分數到達一定值加一殘機，因此對於全破遊戲也是重要的要素。
  • 藍點的分數不是固定值，吃到道具時越靠近畫面上端則分數越高，一定高度以上則分數不再上升且吃到時的分數會以黃色顯示（未達此高度則是白色）。
  • 藍點的分數最大值可以藉由擦彈或信仰點等方式增加，增加方式和計算公式依遊戲而異。
  • 在虹龍洞變為金色，可以在過關後用得分道具換取一張卡片強化角色。

回收線[編輯]

  • 當自機位置在一定高度以上則P點和藍點等道具會自動飛向自機，可觸發這種回收方法的最低高度就稱為回收線。但飛碟等部分道具不對應自動回收。
  • 歷代作品觸發自動回收條件：
    • 紅魔鄉：火力值達到128並到達回收線。
    • 妖妖夢：同上，但EX與PH關卡不需達成火力條件也能觸發自動回收。
    • 永夜抄：火力值達到128或自機為低速狀態下到達回收線，但玩家使用魔理沙時只需達成最後一個條件就能觸發自動回收。
    • 後續作品：無論什麼時候都能上線觸發自動回收，且魔理沙的回收線比其他人還要低一些
  • 從風神錄開始，系列作會在第一關開頭顯示回收線的位置。但是從來沒畫準過，有玩家做過測試發現實際回收位置比畫面顯示的更低
  • 另外自機在低速狀態下會自動吸收身邊一定範圍內的道具，與當前高度無關。

信仰道具/綠色得分道具[編輯]

  • 外表為一個綠色的点因此也叫做綠點，當自機吃到信仰道具時會回復下述的連擊槽並提升自己的信仰點，信仰點越高則藍點的分數最大值越高。
  • 連擊槽會隨時間下降，但可藉由擊破敵人或吃到各種道具回復，如果連擊槽降到零則信仰點會逐漸減少。連擊槽與信仰點均顯示於畫面左下角。
  • 當玩家擊破敵人，或以靈擊和炸彈等方式消除敵彈後會出現綠點，後者產生的綠點不需要跑到回收線也能自動回收。
  • 本道具從風神錄開始登場，地靈殿也有沿用此系統但沒有特別取名字。星蓮船後只有消彈時出現的綠點有被後續作品採用。
  • 你就是因為信仰不足敗北的早苗啊！

星星道具[編輯]

  • 外表為一個灰色的星星，吃到時會增加分數。當玩家以靈擊或炸彈等方式消除敵彈後出現，不需要跑到回收線也能自動回收。
  • 只出現在紅、妖、永三作，風神錄後被功能類似的綠點取代。

Lunatic[編輯]

  • 在東方中指最難的難易度，由簡單至困難為Easy・Normal・Hard・Lunatic。
  • 不接關全破Normal以上難度（部分作品Easy也可）會出現特殊的Extra關卡，難度大約介於Normal和Hard之間。

處理落[編輯]

  • 右下角的fps（frames per second：每秒顯示幀數）低於60，遊戲會變慢。原因為系統不堪負荷。可以用外掛突破60。

カリスマ[編輯]

  • charisma，中文翻譯為魅力，不過多數同人作品會解釋為領袖魅力。

蓬萊之藥[編輯]

  • 在月球時八意永琳利用蓬萊山輝夜的能力製造的禁藥。
  • 吃了蓬萊之藥會不老不死，統稱蓬萊人，但仍然會饑餓和感到痛楚。
  • 蓬萊山輝夜、藤原妹紅和八意永琳（參照永夜抄幽冥組的Good Ending）都是蓬萊人。

彈幕戰[編輯]

  • 幻想鄉有些妖怪非常強，萬一認真戰鬥起來可能有嚴重後果，甚至可能毀壞幻想鄉。因此博麗的巫女建議把戰鬥方式系統化（命名決鬥法案）。
    • 理念：令妖怪容易引起異變、令人類容易解決異變、否定完全實力主義。
    • 打敗博麗的巫女的唯一可能性
  • 彈幕只是攻擊手段的通稱，實際上不使用彈幕攻擊也可以。
  • 使用特定彈幕攻擊時必須先用符卡告訴對手即將進行攻擊，注意符卡本身只是一張普通的紙，彈幕攻擊靠的是當事人自身的能力。
  • 明明只是幻想鄉的內規但就是有辦法讓幻想鄉以外的人也跟著遵守

博麗大結界[編輯]

  • 分隔幻想鄉和外界的結界，實際位置不明, 只知道博麗神社在結界的邊緣上。
  • 據說八雲家就是在境界上, 也有一說是在外界。
  • 在外界只能看見無人的博麗神社。
  • 靠近邊緣的地區容易發現外界的物品, 所以收集家喜歡到神社尋找。
  • 由八雲紫(實際行動主要由八雲藍負責)和博麗的巫女共同維持。

隙間[編輯]

  • 空間之間的隙縫，是八雲紫和八雲藍的住所。
  • 偶爾會有人掉進或被拖進這些隙縫而失蹤，這種現象就是人們所謂的神隠。
  • 隙間界從東方萃夢想中看到的形象是一條裂縫，內有許許多多的眼睛和手。
  • 八雲紫就是憑藉著操縱以及穿越隙間的能力，隨意地穿梭於世界各地。
  • 強得犯規、非常方便的能力。
