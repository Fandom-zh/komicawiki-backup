 學生會執行篇 第一部 漫畫/動畫 第1－21箱/S1第1－11箱 2009年5月11日－9月28日/2012年4月4日－6月13日
就讀箱庭學園的少女黑神目高，是一名成績優秀，運動萬能，外表秀麗，家財萬貫，十全十美的一年級生，
於第98代學生會長選舉中以壓倒性的98%支持度勝出。
她根據選舉公約設置了「目標箱」，並招攬了青梅竹馬人吉善吉和其他特別的朋友作為成員，一起解決學生的問題。

然而，隨著意見箱所引發的事顯得越來越高調，最後引發了與風紀委員會的對立，最後在黑神目高與雲仙冥利的正面衝突下落幕。
 
 十三班的十三人篇 第一部 漫畫/動畫 第22－55箱/S2第1－11箱 2009年10月5日－2010年6月21日/2012年10月10日－12月19日
怪物黑神目高與雲仙的對立結束後，箱庭學園的理事長不知火褲注意到了目高的異常性，並邀約參與製造完美之人的計畫燒瓶計畫，
在與燒瓶計畫的中心都城王土的接觸後，目高和善吉決定接受目高兄長，也是前任燒瓶計畫統籌黑神真黑的特訓，
並宣布在一天之內破壞燒瓶計畫。

在地下設施內一一擊敗了所謂在異常者的班級十三班中做為核心的十三人，
途中與目高除籍的姐姐，更是現任燒瓶計畫統籌的名瀨夭歌/黑神鯨重逢，
甚至期間目高喪失了記憶，與善吉等人對立。突破一切難關，黑神目高正面和都城王土展開決鬥，
最後王土承認敗給了目高的異常性完成，眾人銷毀了地下研究設施的資料，讓燒瓶計畫停擺作為結束。

在以為一切都結束時，現在的轉學生、過去的學長、負之完全、負十三班的球磨川禊，來襲。
 
 球磨川事件篇 第一部 漫畫 第56－92箱 2010年6月28日－2011年4月4日
兇人球磨川和負十三班得到了理事會的支持，利用規則漏洞正式提出罷免學生會，目高便申請發動選舉戰(戰選)。
得到原學生會長(英雄)日之影空洞和善吉母親人吉瞳的支持，與負十三班正式對立。

庶務戰中善吉戰勝了球磨川，但也導致善吉近乎死亡，
最後在夢中得到惡平等安心院馴染的幫助且得到了新的異常性。
期間其餘負十三班偷襲了特訓中的阿久根、日之影等人，順利得讓眾人無法再戰。

書計戰時名瀨得到了新的過負荷冰凍火柱擊敗能引發舊傷的志布志飛沫。
但因為規則的關係，決定認輸已讓接下來的比賽繼續進行，徹底擊敗過負荷。

會計戰時人吉透過欲視力與母親的合作和球磨川與江迎的組合激烈衝突，
最後在江迎的自我犧牲下，不但保全了眾人的生命，也擊碎了球磨川的計畫和內心，並證明了與過負荷相互了解的可能性。

副會長戰由兇惡的蝶蝶崎蛾蛾丸與復活的日之影空洞對戰，
在眾人的支持下，喪失了異常的日之影擊敗了能轉移傷害的蛾蛾丸。

最後是得到了最初的過負荷腳本製作的球磨川與黑神目高的會長戰對決，
就算成為和球磨川一般的存在，正面接下腳本製作的目高內心並未服輸，
隨著目高在過去敵人們的加油聲中，給予球磨川最後一擊，選戰結果勝利，98代學生會再次的獲勝。
並邀約球磨川成為副會長，將一切暫時畫下句點。第一部完結。

用語介紹編輯

  • 箱庭學園
    • 介紹：
      • 故事的舞台，創立年份近100年的高中。目前實際人數超過1500人[1]。
      • 佔地及設施相當大，由一般校舍[2]及操場[3]、社團教室[4]、游泳池(含室內外)、13層鐘樓及地下13層研究設施、植物園「樹影篩落」、箱庭學園舊宿舍(軍艦塔、負十三班教室)、建設中的校舍[5]等場地組成。
      • 由安心院馴染於200年時創立的「黑箱塾」所改制而成。
      • 如同名稱一般，將各地的異常、菁英聚集起來，推動大量金錢和技術投入的「玻璃瓶計劃/燒瓶計畫」，可以說箱庭學園本身就是為了燒瓶計畫而存在。
      • 由龐大的學生機構群領導，其中做為君主般存在地位的學生會長及委員長聯合負責守護學園的日常。
      • 學校內推動君主制大丈夫?
    • 班級編制：
      • 第1到4班為普通科，5、7、9班為體育科，6、8班為藝術科。以上的學生多被稱為「一般/普通」。
      • 第10班為特別普通科，11班為特別體育科，12班是特別藝術科。以上的都是以特殊優待生的資格入取。以上的學生多被稱為「特別/菁英」。
      • 而13班則是「異常」班，即使人數超標也無妨，其中的核心就是十三班的十三人。
      • 在十三班的十三人篇中期以緊急事項的結果而成立了類似的負13班，也就是「過負荷/負之異常」班級，雖然草率成型，人數卻多達千人。
    • 社團：
      • 有菁英生為重點人員栽培的社團，包含運動類、藝術類等。
      • 也有非菁英生組成的社團，如劍道社。
      • 在漆黑的新娘禮服篇前運動類社團被目高全滅(合掌)
    • 箱庭學園學生會執行部
      • 學園內代表學生的最高組織，負責管理大型典禮活動、管理委員會和各大學部的工作。
      • 按照慣例，由會長、副會長、書記、會計、庶務五人組成。
      • 相關校規：第四十五條：關於學生會執行部的罷免。
      • 相關校規：第四十五條-第三項：詳細條款，若學生會執行部有明顯缺失的場合下，只要有全校過半數的署名，那麼全成員即日起被罷免。
      • 相關校規：第四十五條-第七項：詳細條款，會長被解職時，其他成員也應立即解職。
      • 相關校規：第四十五條-第十三項：詳細條款，在不造成其他的麻煩情況下，由解職請求者從即日起直到下期選舉擔任臨時的學生會長。
      • 相關校規：學生會守則 第二條：關於成員選取，學生會執行部由會長、副會長、書記、會計、庶務五人組成。會長當選後應當迅速選出勝任其他職位之學生。
      • 相關校規：學生會守則 第十七條：學生會掌長因為職務的需求，可以任意舉辦學生總會，招集全校所有學生。包含無需上課的十三班學生。
      • 其他-塾則(黑箱塾時期的塾規)：第一百五十九項：關於塾頭(學生會長)解職項目，如塾頭被請求解職的場合下，可要求進行選出下一期的塾頭。
      • 執行部的制服是黑色，這是做為白色制服的一般學生的影子而存在著。
    • 目標箱（目高箱）：
      • 自第98代箱庭學園學生會執行部後設置的屋形意見箱，讓成員可優先處理部分學生的問題和請求。
      • 每當目標箱中的一件事被解決之後，學生會辦公室就會多放一朵花，但是由於解決問題的數量和速度過於異常，導致學生會辦公室空間不足，因而將多出來的花移置校園。
      • 於第100代學生會長選舉當天(平安夜)已經有了4822封求助信。
    • 風紀委員會：
      • 將維護學園治安作為首要目標的游擊部隊，脫離理事會、教師辦公室等等權力的制約的的獨立特務機構。又稱"學園警察"。由於條件賦予准許攜帶武器。
      • 風紀委員長底下有設立副委員長及各大分部隊。
    • 選舉管理委員會：
      • 無限地把公平，公正的選舉的進行，正確的裁判作為目的的學園組織。
      • 在基本的是只選舉期間職務，因而把自己稱為背陰處的機構，還不忘要別人忘記他們。
      • 執行期間包含會長選舉及特別舉辦的學生會戰舉(選舉戰)。
      • 特色是一律都帶著眼罩或面紗。

    • 保健委員會：
    • 飼育委員會：
    • 食育委員會：唯一的雙委員長制，各負責"獵理"及"超理"。

  • 燒瓶計畫/玻璃瓶計畫：
    • 介紹：
      • 由安心院自一百多年前，將試管計畫轉型而成的計畫。其目的都是完成人類
      • 最初的目的在於製造完美的人，靈感來自英雄-獅子目言彥及不知火半纏的製作技能的技能。
      • 以不知火之村為代表，數十個財團甚至國家軍部都在贊助者的名單中。百年來不曾間斷。
      • 雖說最終目的是完成人類，但能夠吸引投資的原因在於能達成天才的量產化。
      • 目的(天才的量產化)的理論建立後，將會把箱庭學園全部作為實驗箱一般使用，約有98%學生會被破壞。
    • 架構：
      • 安心院交由理事長不知火褲[6]主導，投入了10萬人進行研究。
      • 經過不斷的實驗，此屆的十三班的十三人稱之為歷代最為異常。
      • 場地在箱庭學園鐘塔及地下研究設施。另外箱庭醫院也作為計畫的前線基地。
    • 鐘塔及地下研究設施：
      • 箱庭學園作為燒瓶計劃的研究設施，從地面以上的13層鐘樓加上地面以下的13層，形成如同圓形實驗瓶的設施。
      • 拒絕之門：位於鐘樓1樓的位置，是地下設施唯一的出入口，外面用電子鎖鎖上，而暗號是6位數字，被通過一人後每次號碼就會變更，能通過的機率是一百萬分之一超低機率，阻擋了能參加燒瓶計劃的異常程度的異常者以外之人。
      • 地下十三樓直達電梯：一樓的設施，簡單來說就是能到達各樓層的電梯，然外頭也用電子鎖鎖上，難度卻在拒絕的門之上，因而能夠使用的人只有7個人（裏之6人和雲仙冥利）。[7]
      • 地下一樓設施 巨大迷路：[8]
      • 地下二樓設施 日本庭園：[9]日式的房屋、石燈籠，日本庭園的踏腳石，池子，小花園等被舖土的地板的，近似真實的自然空間。
      • 地下三樓設施 實驗動物園：[10]有很多很多動物，全部都由名瀨親手完全肉體改造，因此不怕黑神的「動物趨避」。
      • 地下四樓設施 名瀬夭歌的實驗室：[11]
      • 地下五樓設施 停車場：[12]這裡的車全是百町破魔矢的收藏，後來因兇化合宿的關係遭到負13班的襲擊而毀壞
      • 地下六樓設施 圖書館：[13]

      • 地上十三層鐘塔：作為越野識途比賽的場地，其中機械室在第五樓。
    • 箱庭綜合醫院：
      • 燒瓶計畫的前線基地，被懷疑是異常的小孩會被送往這邊檢查和分析，進行異常者的管理與控制的醫療機關。
      • 由醫科名門鶴喰家主持，長子鶴喰梟為院長，長女鶴喰鳩則擔任護士，鳩在十六年前生下目高後就心肌梗塞死去。
      • 十三年前，人吉瞳也曾在此擔任醫生，但因為球磨川的"兩邊都是皮鞭"，被迫把球磨川列為無異常，之後自動離開這家醫院。
      • 同樣是十三年前，在鶴喰梟的安排下，目高和善吉在這裡相遇，是一切的初始之地。
      • 人吉瞳辭職後，作為最後的患者，六歲的蝶蝶崎蛾蛾丸和五歲的志布志飛沫毀了這家醫院，化作廢址。
      • 三年前的漆黑宴，作為勝利者的鶴喰梟被月下冰人會暗殺，但梟博士利用STYLE(風格)的振動，用言語刺激皮膚驅動血管而活著。
      • 活著的鶴喰梟在箱庭綜合醫院廢址的第四病樓-四樓院長室依然進行著研究，並暗中進行策畫。
  • 黑神家族：
    • 暗中控制世界經濟和政治的巨大家族企業都是這樣設定的。
      • 黑神企業在當代總帥-黑神舵樹之子。黑神真黑(十三歲)的參謀之下，從日本走向了國際，成為國際級大企業。
      • 在國外有400所，國內有100所的家族城市，皆為黑神舵樹擁有，有錢到能輕易的把白金礦山送給別人。
    • 其中除了本家，還有七大分家，分別是贅波、葉野、潛木、桃園、壽、杠、鶴喰。
      • 黑神家的後繼者傳統上要和這七家全部締結婚約關係，如同為了進入本家的勢力婚姻。
      • 之後為了進入本家，所舉辦的選拔宴會，就是所謂的漆黑宴。由分家創立的媒人機構月下冰人會主辦。
    • 不知火家族作為影舞者組織白縫，擔任黑神家的分身。
      • 黑神舵樹 <-> 不知火褲
      • 黑神目高 <-> 不知火半袖
    • 漆黑宴：
      • 由七大分家為了進入本家所舉辦的醜惡宴會，交由月冰會主持。
      • 二十年前的漆黑宴，由黑神舵樹與七位分家的七名婚約者全部結婚。
      • 三年前的漆黑宴，由鶴喰梟勝出，但沒有和黑神目高結婚。認為違反規則的月冰會於是殺了鶴喰梟與其他六名婚約者。
    • 本屆漆黑宴：
      • 本屆狀況比較特別：
      • 1.舉辦原因是三年前的漆黑宴並沒有選出入贅黑神家之人，勝利者(鶴喰梟)並沒有與目高結婚。才舉辦這次漆黑宴。
      • 2.建立在黑神目高本人不追求自己戀愛幸福的前提。當目高追求幸福後，便會影響到漆黑宴和月冰會的根本。
      • 3.該屆漆黑宴表面上由月冰會主持，但鶴喰梟早已控制整個局面，七名婚約者都是梟博士派來的。於是今屆漆黑宴的目的變為渴求梟博士所愛之人--鶴喰鳩之血脈(替代品)，也就是目高。
      • 4.目高為了追求幸福，同時作為鶴喰家的代表及獎品，目的是擊敗其餘六位分家婚約者。
      • 成員：
      • 獎品：黑神目高。
      • 月冰會主持者：兔洞武器子。
      • 不知火家族影舞者：怪儡、蜃氣郎、幻實、錯悟、仮輝、偽造。
      • 婚約者之零-杠：杠換替 (轉喻、謊言使者)。
      • 婚約者之一-潛木：潛木鼬鼠 (誤轉換使者)。
      • 婚約者之二-壽：壽常套 (童謠使者)。
      • 婚約者之三-桃園：桃園喪 (名片使者)。
      • 婚約者之四-葉野：葉野遂 (漢字使者)。
      • 婚約者之五-鶴喰：鶴喰鷗 (挑釁使者)、黑神目高、不知火半袖、不知火半纏、安心院馴染、球磨川禊、箱庭學園第100代學生會成員[22]。
      • 婚約者之六-贅波：贅波生煮 (逆接使者)。
      • 流程：
      • 漆黑宴第一回合：[23]安心院馴染 VS 七名婚約者影舞者(不知火家族的代表)：勝。
      • 漆黑宴第二回合-預賽1：[24]其餘六名婚約者[25] VS 贅波生煮：勝，贅波淘汰。
      • 漆黑宴第二回合-預賽2：[26]人吉善吉 VS 贅波生煮：勝。
      • 漆黑宴第二回合-預賽3：[27]其餘四名婚約者 VS 潛木鼬鼠：勝，潛木淘汰。
      • 漆黑宴第二回合-預賽4：[28]名瀨夭歌 VS 葉野遂：勝：葉野淘汰。
      • 漆黑宴第二回合-預賽5：[29]球磨川禊 VS 潛木鼬鼠：勝。
      • 漆黑宴第二回合-決賽：[30]黑神目高 VS 其餘三名婚約者：勝，杠、壽、桃園淘汰。
  • 不知火(白縫)家族：
    • 建立：來自只是存在在這裡的惡平等不知火半纏，為了替補身為安心院馴染影舞者工作的備份。
      • 經過萬年的演變，成為不知火之村。
    • 工作是黑神家的影舞者(白縫)及保存稀有品種(獅子目言彥)。
      • 擔任黑神家影舞者的工作，是為了傳承言彥而活用自身特技的薪水豐厚的工作
      • 透過一代接著一代的影舞者，在20歲當天傳承獅子目言彥。
