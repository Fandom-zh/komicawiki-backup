簡介[編輯]

  • 英國皇家海軍是英國軍隊中歷史最悠久，實力最強的軍種。在十六世紀到二次大戰其間一直穩坐於海軍界中的一哥位置，是當時的英國將其影響力遠播至全球的重要工具
  • 儘管在第二次世界大戰中其世界第一的寶座讓予了工業力全開的美國海軍，但是不論在大西洋，地中海，以至遠東地區都有著皇家海軍船艦奮戰的身影。
  • 幻萌的親女兒，和美國一樣匯聚了畫風漂亮的畫師，因此英國艦娘基本上都很有人氣。
    • 英吹少女
    • 最多、最好的戰艦和最強大的戰巡
      • 一個標準的攻堅艦隊通常都是以3-4艘英國戰艦/戰巡以及2-3艘美日航母所組成的
    • 唯二的獨有艦種淺水重炮艦
    • 相對地小船不突出
      • 但還是有能一頭撞殘戰艦的妖怪小學生存在的
    • 普遍有著高火力，但部份艦隻的裝甲薄弱


海軍部V級驅逐艦[編輯]

  • 又稱舊V級，皇家海軍在一戰時期於戰時應急計畫下投建的第三個艦型，在噸位及武裝上遠比舊R級及舊S級強大
    • 為應對來自空中的新威脅，加裝了一門高射炮或砰砰炮作防空用途
  • 本級一共建造了28艘，共中18艘在二戰爆發時仍然在服役
  • 由於是一戰船，在眾多驅逐艦中屬性是墊底的，只比紫石英強上一點

吸血鬼 (HMAS Vampire)[編輯]

  • 圖鑑編號：271，海軍部V4號艦
  • 1917年5月21日下水，於1933年10月11日轉讓予澳大利亞海軍，於1942年4月8日在亭可馬里港附近被日軍轟炸機擊沈
  • 雖然是英國出身，但國藉上是屬於澳大利亞，不能享受英國限定的buff加成
  • 小惡魔性格的護士娘，聲稱從血液中就能看出一個人的內心
    • 在最潦倒的時候曾獲反擊收留，因此很是愛慕她
  • 在畫師設定中是貨真價實的吸血鬼，雖然在血族中比較年輕但還是最少有上百歲了ロリBBAgfychwehuhsid(被吸乾)
    • 因為工業革命的緣故被迫流離失所，在最潦倒的時候被反擊救起，在她的引薦下加入艦隊成為艦娘
      • 因此對反擊有十分深厚的感情[1]，對於保薦她的聲望和威爾士親王都有好感
      • 派趣版另有立繪，但由於風評被害的關係顧及部分玩家感受就不貼了，還請自行餵古狗

A級驅逐艦[編輯]

  • 皇家海軍以於1927年建造的伏擊號(HMS Ambuscade)及亞馬遜號（HMS Amazon）為基礎，於同年計畫投建的第一批新型驅逐艦
    • 英國在此時到戰前建造的驅逐艦主要思路都是每年計畫建造一級，每級9艘；每一級驅逐艦都能自成一隊驅逐艦支隊(Destroyer Flotilla)
  • 此型驅逐艦共建造了兩個批次(A及B級)，A級建造了11艘，其中兩艘於加拿大建造並於加拿大海軍服役
    • 在二戰中A級驅逐艦只有5艘能存活到戰後
  • 在畫師設定中，由於A級驅逐艦屬於更早期的型號，所以造型上採用了英國線列步兵(龍蝦兵)的樣式，武裝上也是更簡陋的步槍型主炮

熱心 (HMS Ardent)[編輯]

  • 圖鑑編號：172，A級6號艦
  • 1929年6月26日下水，1940年6月8日於斯卡帕灣被德軍戰巡沙恩霍斯特號擊沉

阿卡司塔 (HMS Acasta)[編輯]

  • 圖鑑編號：172，A級3號艦
  • 1929年8月8日下水，1940年6月8日於斯卡帕灣被德軍戰巡格奈森諾號擊沉
    • 但在被擊沈前發射的魚雷也重創了沙恩霍斯特，迫使她回港維修

G級驅逐艦[編輯]

  • 皇家海軍於1933年計畫建造的艦隊驅逐艦，在F級驅逐艦的基礎上縮小了船身的尺寸
  • 和其改造型號H級驅逐艦一起，共建造了24艘，其中G級有9艘，H級15艘
    • 她們在戰況激烈的大西洋戰場上表現出色，但損傷也十分慘重，只有7艘能活到戰後被拆解

螢火蟲 (HMS Glowworm)[編輯]

  • 圖鑑編號：82→1082，G級4號艦
  • 1935年7月22日下水，1940年4月8日在 威瑟堡演習行動 中戰沒
    • 是在所有G級驅逐艦中最先服役的
  • 屬於英國海軍在強化單艦雷擊方面的實驗艦，裝備了新型的五聯裝魚雷發射管
    • 不過改前的立繪卻是四聯裝魚雷，改造後總算還原回去了
  • 初期四艘可選艦娘之一
  • 從各方面[2] 看都是最有主角格的艦娘
  • 手持魔杖的魔法少女，改造後身邊還跟著三台機械使魔浮游炮
  • アリス・カータレット[3]
  • 沒爹愛，沒有新衣服，衣服都是穿標槍穿舊的
    • 然而也是換裝系統實裝後第一批獲得新換裝的艦娘之一
  • 熱衷碰撞
    • 從1934年完工到1940年戰沉為止，螢火蟲總共發生過三次碰撞事故
    • 最後一頭撞在德軍重巡希佩爾號上而戰沉
    • 還原戰史，因此改造後的技能是衝撞攻擊「無畏撞擊」，隨機發動對敵方造成和裝甲值掛勾的固定傷害
      • 為了最大化衝撞的傷害，通常會讓她帶滿裝甲板，然後和信賴一起出擊信賴騎士
      • 但還是撞不沉希佩爾[4]
      • 在逸仙改實裝以後，在她的裝甲buff加成下，無畏撞擊終於能撞沈希佩爾改了，太好了呢螢醬！
      • 在畫師的設定中，中破立繪中從膝部伸出的刺刀就是用來給她發動技能用的所以正確的名字其實是「無畏膝擊」
  • 對玩家的稱呼是「指揮官」

J級驅逐艦[編輯]

  • 由於部族級的高昂造價，所以在1936年起計劃建造的J級驅逐艦又走回小型船身，強調雷擊能力的設計
    • 采用劃時代的單煙囪設計，節省甲板面積以放置更多的武裝
    • 船身的設計也成為後來的戰時應急驅逐艦的範本
  • 此型驅逐艦共建造了三個批次(J，K及N級)共24艘，其中J級有8艘
    • 在二戰中J級共損失了6艘，只有領艦杰維斯(HMS Jervis)和標槍(HMS Javelin)存活到戰後

黑背豺 (HMS Jackal)[編輯]

  • 圖鑑編號：85→1085，J級2號艦
  • 1938年10月25日下水，1942年5月12日在一次阻擊行動期間被德軍轟炸機擊沉
  • 很符合命名意象的黑皮獸耳娘，手上帶著金屬製的狼爪
    • 娘化金剛狼
  • 中破時頭髮會變黑變長，藏起來的狼耳也會完全展開，而且髮梢還著火了
    • 瞳色也變成危險的橙色
  • 似乎有「嗷嗚」這樣的口癖，一見面就嗷嗚著把指揮官推倒了
    • 3.0實裝後為了配合聲優演繹而改成一見面就對指揮官嗅個不停
  • 改造後的技能是「飛撲接球攔截護航」，在炮擊戰中有一定機率能為僚艦擋下並免疫一次傷害[5]
    • 似乎是捏他在1940年和標槍等四艘驅逐艦一同抵禦德軍驅逐艦的突襲的事跡
      • 但是那場戰鬥的結果是德軍全身而退，反而僚艦標槍卻吃了魚雷被重創
  • 畫師設定：
    • 北非殖民地來英國學習的異族公主，雖然似乎有著強大的戰鬥技巧，但比較擅長隱藏自己。習慣隱藏自我，話很少的感覺；大破之後會狂暴的有些難以自控；對其他人都會說客套話，對好感高的人才顯得不太一樣。

天后 (HMS Juno)[編輯]

  • 圖鑑編號：84→1084，J級4號艦
  • 1938年12月8日下水，1941年5月21日在克里特島戰役期間被意軍的轟炸機擊沉
  • 因為美國有同音意的USS Juneau，故意譯為「天后」而USS Juneau音譯為「朱諾」
  • 自稱「妾身」，但用語沒有初春的古風明明是英國船
  • 喜歡給其他人算命明明是(ry
  • 擅長翻花繩[6]明明(ry
  • 中破會黑化
    • 畫師的說法：比較腹黑和精於算計，說話之前都會思考一番，對好感度高的人，略帶病嬌的愛
  • 英國驅逐艦的女子力擔當
    • 曾經的結婚台詞中宣稱要「天天給提督做飯」究竟是太喜歡提督只給他做還是難吃到只有提督才吃得下就不得而知了
  • 改造後變得很騷
  • 技能「對潛專精」能提高對潛艇的命中率30%和對魚雷的迴避30%
    • 多半會成為英國驅逐艦的改造標配技能

標槍 (HMS Javelin)[編輯]

  • 圖鑑編號：83→1083，J級8號艦
  • 1938年12月11日下水，1949年6月11日退役，隨後被拆解
  • 因為名字的關係，手持著由魚雷和高頻測向儀組成的長槍
  • 改造後的技能「戰鬥續行絕處逢生」可以使她免疫一次致命的傷害，並增加耐久上限
    • 捏他自標槍號在二戰中兩度[7] 受重創但仍然能活到戰後的經歷
    • 但一般來說如果大破的話是不會冒險進擊的，而且一場戰鬥中只能免疫一次傷害，在同一場戰鬥中再吃到傷害的話還是會沈船
      • 結果這特性只能在演習中能有效運用，而且主要都是被某些惡質的提督用來惡心其他人
    • 不過技能可以令標槍的耐久最高能提升至40血，能觸發4倍血限定的中破保護機制，大大提高生存能力
  • 是其畫師無糖白蓮最喜愛的角色，但他對於標槍的角色設定卻出乎意料的鬱
    • 貴族家庭出身，但由於在幼年的一次事故中在沒可能生還的情況下倖存而遭到家人畏忌
    • 加入艦隊之後也得不到指揮官重視，然後在一次例行任務中，她的小隊遇上敵方的主力艦隊，只有她拖著致命傷活著回來，還得到"死神"的污名
    • 在傷勢痊癒後，為求立功的她主動要求使用實驗中的新型探測裝備(改造)
      • 但是新型探測裝備的運作原理是和深海側建立聯繁，在上次重創時受到深海因子感染的她因而暴走並攻擊隊友
    • 於是當她回港時馬上就被憲兵逮捕了，雖然她並沒有攻擊隊友的記憶，但也只能在好心人的協助下亡命天涯
      • 在無糖對最近實裝的吸血鬼的背景設定中甚至暗示標槍最後還是死掉了

紫石英 (HMS Amethyst F116)[編輯]

  • 圖鑑編號：81→1081，改黑天鵝級16號艦
  • 由於是比驅逐艦更小一圈的護衛艦(Sloop)的緣故，各方面的屬性都是驅逐艦裡面最下位的程度
    • 以20節的低速成為唯一的低速驅逐艦
    • 不過作為對潛護航用的Sloop，反潛值比所有驅逐艦都要高
  • 左眼戴眼罩的中二病，會給自已的攻擊取中二臭必殺技名
    • 小鳥遊六花
    • 在畫師設定中這是用來掩飾自己來自家道中落家族的窮酸
      • 洋傘和洋服都是自己造的
    • 在改造後因為下述事件而改正了
  • 曾經在國共內戰期間亂入到雙軍交戰的前線中，結果被共軍以山炮和榴彈炮組成的炮兵團打的落荒而逃
    • 由於護衛艦的裝甲薄弱，她甚至一度被打至重傷擱淺
    • 隨後英方派遣倫敦號重巡，伴侶號驅逐艦以及黑天鵝號護衛艦前來營救，也被共軍的炮火打的滿頭包，不得不撒退
      • 據當時的報載，甚至有國軍的炮兵加入對兩艦的炮擊
    • 最後在夜間借助一艘碰巧經過的客輪的掩護才得以脫出長江水域，然而卻害得那艘客輪躺炮而被擊沉
    • 此事後來被稱為 「紫石英」號事件(Amethyst Incident)，被認為象徵著西方列強自鴉片戰爭以後對中國的「炮艦政策」的結束
  • 改造後因為上述事件的關件，性格變得謹慎起來，中二病都改掉了
    • 在畫師設定中，因為被女王懲罸的關係而變成真正的貧窮少女
      • 改造後的服裝是用改前的服裝改造的，打了不少補丁
    • 屁股上留著在上述事件中被兔子打的烙印
    • 不過眼罩仍然留著，是為了遮蓋左眼的傷疤

海軍上將級戰列巡洋艦[編輯]

  • 圖鑑編號：1->1001， 海軍上將級1號艦
  • 英國皇家海軍歷史上第4艘以「胡德」（Hood）命名的軍艦。由於其同級艦安森、豪、羅德尼均在下水前取消建造，其成為英國實際建造的最後一艘戰列巡洋艦。
    • 1916年9月1日在約翰·布朗造船公司動工，1918年8月22日下水，1920年5月15日服役。
    • 在其服役生涯中多次作為展示英國國威的禮儀艦巡遊世界各國
      • 最為著名的1923年11月27日至1924年9月28日的「帝國巡遊」。[10]
    • 1941年5月24日威爾斯親王號戰列艦一起攔截德國俾斯麥號戰列艦，在丹麥海峽追蹤到俾斯麥號。最後意外被俾斯麥號的380毫米主炮炮彈貫穿甲板裝甲，引發彈藥庫爆炸，艦體斷裂迅速沉沒。全艦1418名官兵陣亡，僅有3人獲救。[11]
  • 改造後技能為「皇家海軍的榮耀」，為全體增加10%暴擊率及被暴擊率的旗艦限定技能
    • 英國船有雙倍加成，因此被稱為英吹四艇[12] 之首
  • 16/7/22的更新後新增了第二技能「胡德旅行團帝國巡遊」，是增加全體航速的的旗艦限定技能
    • 大幅提升戰鬥中T優的機率，也能使迂回更容易成功
  • 溫柔賢淑的大姐姐，優雅而善解人意，喜歡看書和下午茶（紅茶黨）

    • 有無數的仰慕者，包括俾斯麥
  • 眼鏡娘，視力不太好，1935年1月曾在演習時與聲望相撞。
    • 著名的丹麥海峽之戰中，也曾將歐根親王誤認為俾斯麥。雖然主要原因是糟糕的能見度和歐根親王與俾斯麥艦型過於類似。

  • 教科書式的傲嬌
  • 深海細作
  • 常和博格爭論誰是姐姐
  • 改造後的技能和博格一樣是「反潛護航」
    • 贈送的攻擊機「復仇者即是進口版TBF」是反潛值最高(10)的攻擊機

皇家方舟 (HMS Ark Royal)[編輯]

  • 圖鑑編號：119，皇家方舟級1號艦
  • 英國第一艘非改裝而是單獨建造的航空母艦，也是艦R第一艘實裝的英國正航
  • 附送的劍魚810中隊正是打壞俾斯麥陀機的那一隊
    • 實際上擊傷俾斯麥的劍魚機中隊是818中隊。
  • 目前唯一一艘持弓的非日系航母
    • 拿的是英國長弓
  • 奧斯卡的座艦三號[25]

單造航空母艦[編輯]

獨角獸 (HMS Unicorn)[編輯]

  • 圖鑑編號：127，無級別單造艦
  • 因為鈴蘭的要求，獨角獸的設計充滿魔幻風格
    • 純白禮服，點綴著花朵，飛機跑道化作音符絲帶，飛機則變為了翠色小鳥，全身上下唯一的機械構物就是手上的豎琴－你真的是「艦娘」嗎？可prpr最重要

羅伯茨級淺水重炮艦[編輯]

  • 在前期地圖當保母，後期因為航速過慢，生存力太弱而無用武之地
  • 沒存在感，改造衛星被忘記放在場刊

羅伯茨 (HMS Roberts)[編輯]

  • 圖鑑編號：62→1062，羅伯茨級1號艦
  • 頂個蟹
  • 率先得到改裝，技能「火力支援」能大幅提高其火力及命中率，而且加幅會隨著平均航速的降低而上升
    • 使用戰列核心改造
    • 炮塔上的機槍位置太微妙結果令一眾紳士想歪到一邊

阿貝克隆比 (HMS Abercrombie)[編輯]

  • 圖鑑編號：63，羅伯茨級2號艦
  • 頂個鳥
  • 隱藏巨乳
  • 改裝技能和羅伯茨一樣
