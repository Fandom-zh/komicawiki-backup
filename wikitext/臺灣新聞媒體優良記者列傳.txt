== {{color|white||&lt;s&gt;這是不存在的說明&lt;/s&gt;}} ==
*{{color|white||&lt;s&gt;也許你是小民，沒什麼理由好被採訪。不過你被捲入任何事件遇到記者的機會不等於0&lt;/s&gt;}}
*{{color|white||&lt;s&gt;也許像同人展那種記者的主要攻擊目標，根本沒有方法可以阻擋記者，只能任其凌虐。&lt;/s&gt;}}
*{{color|white||&lt;s&gt;就算如此，這篇仍希望能成為各種意義上的預防針，此為本篇存在目的之一。&lt;/s&gt;}}
*{{color|white||&lt;s&gt;你不會希望難得出現在電視上卻是在附和與陪襯記者耍低能的&lt;/s&gt;}}
*{{color|white||&lt;s&gt;就算遇到躲不掉的狀況，起碼像知道「幹！這傢伙是當年採訪腳尾米導演的！」時，你可以預知回家後看到自己在電視上被打成一問三不知的白痴&lt;/s&gt;}}
*{{color|white||&lt;s&gt;「珍惜生命，請遠離記者。」早已是生活常識，同時請瞭解記者就算智障也有百百種的智障玩法。&lt;/s&gt;}}
*{{color|white||&lt;s&gt;我們當然會祈禱各位不要遇到牠們，而且可以主動避開這堆垃圾，但更希望各位不幸遇到時能夠好好應對然後全身而退。&lt;/s&gt;}}
*{{color|white||&lt;s&gt;達成的要件很簡單："知彼"知己。&lt;/s&gt;}}

== 格式 ==
*暫定採用姓氏注音音頭排列
*請注上事蹟日期與當時所在單位
*就算真的是編寫當日發生的事，也請注上日期──這條目是以讓後人&lt;s&gt;永記教訓&lt;/s&gt;永久景仰為目標的。

== 優良記者常見特徵&lt;s&gt;常幹的蠢事&lt;/s&gt; ==
*提出各種出於善意，會令人感到溫暖的問題關懷人們，例如詢問被從火災救出剛醒來的小孩「你知道爸爸媽媽都被燒死了嗎？難不難過？」
*制定新式單位計算法，協助科學家們使臺灣走在國際最尖端
*前無古人後無來者的超強洞察力，只要到過案發現場就能知道兇手真面目
**並能使用超強大的領導魅力領導眾人張揚正義，2012年1月一位張姓&lt;s&gt;嫌疑人&lt;/s&gt;殺人凶手即是最好例證。
*掃除社會害蟲為己志，其體現在對宅男的大力追殺，並且對有害社會的次文化如ACGN等不遺餘力的攻擊。
*不畏困難，再多的黑衣人也擋不住攝影機拍攝真相；聰明伶俐，受訪對象偽裝成滿頭斑駁身體孱弱的老者也無法欺騙我們停下採訪的腳步

== 待尋找中一定得記述下來流傳後世的關鍵人物 ==
*解說：記者總是將製造優良&lt;s&gt;垃圾&lt;/s&gt;新聞的功勞&lt;s&gt;責任&lt;/s&gt;光環轉給公司，而公司也很謙讓的總是把光環收下再私底下轉交給各個學者讓他們將頭功登記在觀眾上面，現在，該把功勞物歸原主了。
*各家新聞台責任編輯、總編、老闆，現在這一切優質新聞他們必須居頭功

== 列傳序 ==
*自不知何時起，新聞已同時兼有綜藝節目、色情資訊傳播、造神祭壇、黑道恐嚇代言、清涼寫真集、科學名詞制定、造詞、次文化與弱勢的批鬥場、私家偵探辦案等多到無法詳盡列出的功能與身份。（詳細請參閱[[台灣媒體亂象]]）
*現今優質媒體生態非一人一時之力可成，為無數記者及無數主管血汗奮鬥至今並且持續不懈才得以建立。
*著序筆者對本行先烈與現正奮鬥中的勇士們深感敬佩，望國內廣大人民享用優質媒體之時勿忘根本。
*遇到現正奮鬥中的勇士，請不要擋到牠的路。若有需要，也請好好配合牠&lt;s&gt;迫害&lt;/s&gt;追殺各種牠們認為有害社會的一切事物，例如犯罪嫌疑人家屬、宅族群、弱勢團體、動漫作品等等一切牠們認定的垃圾。
*若對方一時昏頭以為您有資格被牠破例特別報導，請您立即婉拒並提醒牠，您這小民並沒有資格在這璀璨的一頁上留下痕跡。
*不要擋路並不只是不要讓牠浪費時間採訪您，凡在路上遇見，請繞道而行，或者上前稱讚一下牠的事蹟；在餐廳同室，請即刻迴避；同車而居，務必請牠等下一班，不要讓自己不小心玷汙了尊貴的記者。
*現列傳紀念故人以及現行努力中的優良記者，除可使人緬懷先烈事蹟時方便使用、亦為前述減少現今記者所遭受的阻力。
*&lt;s&gt;編寫這序就已經夠讓我想吐的了，&lt;/s&gt;事實上，如2012年1月的台人在日遇害案件至少就有30位記者{{ref|報紙四大：中自聯蘋；電視：台中華民年T三東中緯非。單點名大的就至少15名記者，日本那邊一個、台灣這邊&lt;s&gt;騷擾&lt;/s&gt;關心家屬又要一個，所以至少30位，有錯還麻煩修正。}}因此事蹟而取得入傳資格，由於編者&lt;s&gt;不想再強姦自己眼睛跟頭腦至少29次&lt;/s&gt;心力不足，故希望此條目能由大眾一同&lt;s&gt;僅提供受害經驗或者印象深刻的&lt;/s&gt;努力完成使之流傳後世。
