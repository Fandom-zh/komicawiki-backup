概要編輯

  • 為Niconico動畫上一個著名的次文化圈總稱。和淫夢民有著相當深厚的關聯。
    • 是淫夢史上最大的風評被害事件，導致大部分東方聲劇及參與配音的聲優都被拖下水。
    • 因為這件事的緣故，產生了「早期的東方聲劇基本都是風評被害，後期的東方聲劇基本都是看準クッキー☆的人氣而生的賣名劇[1]」的說法。
    • 雖然看似有很多東方角色，但並不被日本東方圈視為其一部分，大部分日本クッキー民也和東方沒有太多交集，不過海外クッキー眾因為接觸管道的因素而常有東餅雙修的狀況出現。
      • 兩作一起出現的場合稱為「一轉攻勢」。

作品一覽編輯

  • 由於作品數量繁多，下文只介紹較具代表性的聲劇介紹。

一切的元兇：クッキー☆編輯

  • 全稱：「【東方合同動画企画】魔理沙とアリスのクッキーKiss」
    • 動畫內容為東方百合向的集體創作故事[2]，企劃主辦人為蓮奈理緒；劇情大意是魔理沙製作情人節巧克力餅乾送給愛麗絲的過程，全長30餘分。
  • 乍看之下相當大手筆，但是由於成品水準不高，在東方圈中評價不佳。
    • 繪師眾多導致畫風一直變化[3]。
    • 素人程度的配音水準，棒讀只是基本款，部分聲優因宅錄而有明顯的錄音瑕疵，其餘至錄音室的聲優也因時間關係要一發錄完。
    • 煩悶枯燥的劇情（吐槽滿點的對話，讓人笑不起來的搞笑橋段「⑨番茶如何呢（激寒）」，意義不明的戰鬥等等）。
    • 無法掌握節奏的失敗後製。
  • 通常一般人在苦行觀賞本篇時只會看到MRS、RIM、SIK、ALC等人在博麗神社場景就脫落了，進到紅魔館立教大學後的場景則稱為「未公開場景（未公開シーン）」。
  • 而因為棒讀的聲優演出與淫夢二章有那麼一點雷同，因此在投稿兩天後就火速吸引了淫夢廚的注意，並被掛上了淫夢的tag
    • 原本只是好事的淫夢民跟東方眾在動畫中的爭吵，但因為一個音MAD好死不死同時使用了クッキー☆及淫夢素材製作，此舉等同將兩者牽線，在此MAD一炮而紅後本家也開始被大量淫夢彈幕的入侵。
      • 到後來蓮奈理緒索性設置過濾轉換系統，把所有彈幕裡的淫夢用語替換成「クッキー☆」，卻只讓事情更加嚴重
    • 此時東方眾們仍與淫夢民頑劣抵抗，但是之後出了ㄧ些大大小小的事情（參與者以侵害權利者為由要求刪除クッキー☆MAD與淫夢民鬧翻，好事鄉民翻出了蓮奈理緒的不良記錄，後期更有企劃參與者內鬨等）之後更加加劇了淫夢廚的入侵，最後被東方眾拋棄歸入淫夢動畫的分支。
  • 現在一般認知中「クッキー☆」系列動畫屬於淫夢的一部分，與東方無關，所以クッキー☆動畫通常不會標上東方的標籤，若是使用東方曲則會給予「東方蠻法天」[4]的次標籤。
    • 本動畫於2013年11月12日0時0分（台灣時間）達成百萬再生，是淫夢關連中第二個百萬再生動畫，也很不幸地是東方配音系列中第一個達到百萬再生的動畫。
    • 目前播放數已突破500萬再生

クッキー☆☆編輯

  • 指TIS姉貴製作的一期的「聖誕企劃2011」、二期的「正月企劃」（中文翻譯版）[5]以及三期的「聖誕企劃2013」。
    • 最初的聖誕企劃僅是因クッキー☆的風評被害而被稱為「クッキー☆☆」，然而其作為MAD素材的潛力不足，並沒有受到重視。
    • 企劃者TIS姉貴在了解クッキー☆事件後所作的，竟是瞄準淫夢民製作正月企劃；再加上mo兄貴以傑邦尼般的速度在24小時內製作出MMD版，兩者相乘引發廣大迴響，甚至一舉攻入ノンケ市場。
  • 由於TIS的聲劇作品僅有聲音，並沒有繪畫的部分（僅有封面），因此都需要透過MMD的重新演繹後才能正式素材化同時進入クッキー☆劇場，也因為MMD模組的取材與角色性格的連結，使得觀眾更容易透過劇場動畫選用的MMD模組輕易判別出角色。換言之，MMD化是クッキー☆☆順利成為クッキー☆的分支的一大功臣。

クッキー☆☆☆編輯

  • 指由同人團體「滿褔神社」製作的一系列東方二次創作動畫《幻想万華鏡》。以重現原作（射擊遊戲）故事為旨，雖有字幕對白但不設配音。
  • 本該和クッキー☆扯不上關係，但偏偏有參演過クッキー☆☆的聲優（AZS姉貴、VVAN姉貴等）上傳過經自己配音的影片，致本系列無辜受牽連。
    • 所幸火種很快被撲滅，漸漸為クッキー☆眾淡忘。

カス☆編輯

  • 由美翔カズヤ所主催，集合15名繪師及聲優而成的新的東方聲劇企劃，名稱為東方夏休み　ヴォイスドラマ企画
  • 劇情跟說明文很明顯是致敬賣名クッキー☆本家，基於クッキー☆架構下的多人合作二次創作聲劇（跟其他由TIS主導的クッキー☆☆仍然是基於東方為架構的聲劇企畫不同），但是過度低劣至於褻瀆原作等級的繪畫連當成素材都不可能了，因此投稿後被淫夢民們毫不留情的批判。
  • 雖然掛著クッキー☆的名號卻不為クッキー☆系MAD作者所採用 ，反而是カズヤ本人比較常出現在MAD變成惡搞對象，悲しいなぁ…
    • 不過還是有人賞臉把裡面的ALC其中一個人物發掘成為另一系列劇場的主角了，詳見JGN

満月（まき）作品編輯

  • 代表作有萬聖節企劃、地靈殿企劃、命蓮寺大冒險等。
    • 最新作有開心戀聖誕（俗稱いしころ☆）
  • 會對應募聲優說明風評被害的風險。
    • 嘴上說會保障參加者隱私，卻大肆投放クッキー☆☆MAD（尤其以RI姉貴為題材居多）挖人瘡疤。人間之屑嗎這傢伙
  • 但由於各種因素[6]，影片再生數遠遠不及前人，被譏諷為「売名失敗兄貴」
    • 2017年元旦投稿的新作總算做足本錢，把RRM姉貴也請來了，在一個月內累積破萬再生，雖然不是很多，但對MK而言已經是個重大突破了。

メッモー☆編輯

  • 影片全名「アヤツリ異変...ナゾの影」，是使用DSi軟體「うごくメモ帳」製作點陣圖動畫的東方聲劇作品。主催者りーな是現役小學生。
    • 連載中，現已上載11集。
  • 原創狐妖「ミラミカル・リラン」四處找架，並用「使人發狂的能力」逼得一眾對手失常。主角群為解決這異變而奔走...如此介紹活像常見的東方二創。
    • 但蹩腳的作畫和失真的配音，加上是東方題材，還有作者故意對クッキー☆致敬（只是轉載者在耍廚），行差踏錯的話很容易會クッキー淪為☆☆☆☆。

金のかかったクッキー☆編輯

  • 花大錢作的クッキー☆，特指《艦隊Collection (動畫版)》（第一季）。
  • 大公司資本加持下使本動畫擁有（看似）華麗的作畫以及豪華的聲優陣，但劇情大崩壞使得它被拿來與クッキー☆比較。
    • 別拿這套爛片和クッキー☆比，簡直汙衊了クッキー☆

クッソー☆編輯

  • 阿刀田阿子所企劃的一系列聲劇，一系列共四部作品，名稱由來為初作的浣腸排便要素。
  • 現存ｳﾞｫｲｽﾄﾞﾗﾏｷｶｸ作者，也是繼TIS之後挖掘クッキー☆新一代聲優的主導者，雖然規模沒有TIS時期來得大，但是話題跟捏它性都有過之而無不及。
    • 前兩部作品評價普通，但作者有意無意的導入能夠跟淫夢廚產生共鳴的捏他，似乎是嘗到甜頭在2015後投稿的兩部作品內容開始大量毫不遮掩的對淫夢跟クッキー☆本家致敬。但主要還是靠著聲優的賣力演出跟BB作者的形象塑造下，成為承接TIS在近期中作品引用(BB素材、劇場等)最為活躍的クッキー☆派生作品。
  • 有名作品：
    • the reimu's happy holidays，超越ALISON兄貴的瘋狂內容364364
    • 霊夢と魔理沙のチョコレート★ハート，故事內容平淡無奇跟クッキー☆差不多對靈夢超級不友善……，但是處處埋下跟能與クッキー☆連結的小捏它，此外本部作品的聲優包含RRM姉貴（靈夢擔當）跟AKR姉貴（幽香超早口おばさん擔當）以及NYN跟ICG都在本作品的演出十分出色而一舉成名。
    • 聖誕企劃2015，作者表示此為最終作。特色是在同一作有五名由各自不同聲優擔演的魔理沙，以及極其露骨的メタ発言（主要關於參演東方聲劇這行為的）。另外作者更獲りーな姉貴許可而讓ミラミカルリラン客串出演。

   自己矛盾☆編輯

  • 熟肉見此
  • 全稱：魔理沙とアリスの自己矛盾☆，為在クッキー☆10周年活動突然登場的聲劇企劃，不以東方為題而是以クッキー☆為主題。
  • 以極高的製作水準（動畫、劇情、聲優演技）和發人深省的內容在クッキー☆圈內引進了巨大討論。
  • 原作者表示：在這部聲劇劇本完成開始製作（2018年中）到成品出爐時已經過了一年半，クッキー☆圈的狀態也已發生巨大變化。因此請抱著觀賞以クッキー☆為主題的時代劇的心情看下去。

相關動畫編輯

傳統クッキー☆音MAD編輯

  • 素材選用以クッキー☆本家前5分鐘為主，MAD的後半段轉變成為淫夢本家的MAD（向 「霊夢 so クッキー☆」致敬）。由於強烈的反差以及中毒性極高，仿效的動畫越來越多，因此成為其系列的一個最大特色。
    • 也有一種逆向操作手法，一開始是淫夢MAD，進入副歌時才改變成為クッキー☆MAD，這時稱為「一轉幸成」。
    • 還有一種更惡劣的方式是縮圖跟標題都クッキー☆相關，結果內容含有クッキー☆要素的畫面只有前幾秒就立即ㄧ轉攻勢。
  • 無論如何，將淫夢跟クッキー☆兩者屬性差異極大的題材混在一起就是クッキー☆系MAD的醍醐味。

クッキー☆淫夢共演音MAD編輯

  • 省略一轉攻勢，直接使用クッキー☆及淫夢素材同時演出。

  • 沒有任何淫夢要素的純クッキー☆MAD，偶爾會添入確定演出片段故作玄虛。
 

クッキー☆動畫版編輯

  • 將クッキー☆本家的重點場景動畫化，也有BB版本供MAD作者使用，不愧是基佬特有的技術力。
    • 日後MAD的素材選用也開始偏向動畫版クッキー☆，但靜止畫的原版還是主要的使用來源。

ほのぼの神社アレンジ編輯

  • 以歌曲「ほのぼの神社」為基礎作任何形式改編的動畫系列。
    • 「ほのぼの神社」是クッキー☆本家第一幕使用的BGM，為東方[7]曲「二色蓮花蝶 〜 Ancients」的改編曲。
      • 製作者為再調整の人，也是參與クッキー☆企劃的配樂之一，在クッキー☆淫夢化後便發表了該曲（動畫還不避嫌的放上Van Houten Cocoa的圖片，クッキー☆製作者為淫夢民說不可避），至此開始大流行。各種ほのぼの神社的重編版本如雨後春筍般不斷冒出，特徵是縮圖一定要放上ぷはーRIM的笑顏。
  • 之後順理成章地變成BB先輩劇場及BBクッキー☆劇場的常用BGM用曲，由於改編種類眾多可以依據劇情需要選擇適合的版本。
    • 知名NICO組曲作者也來湊上一腳。

新約ほのぼの神社編輯

    • クッキー☆☆前段的BGM。
    • 非東方アレンジ曲、而是使用免費素材BGM。
      • 來自H/MIX GALLERY，曲名Arukas Load，作者秋山裕和。
      • 脱出ゲームBear's life 2 ～はじめてのお使い～使用本曲作為BGM。
  • 相關作品多不勝數，但素質也參差不齊，如同BB先輩一般，僅有少數高質素或有名的會被挖掘且被劇場引用，絕大多數都僅止於展示合成的作品而已，苦無登場機會。
  • ほのぼの神社アレンジwiki

BBクッキー☆劇場編輯

  • 如同淫夢本家的BB先輩劇場一樣，使用各式クッキー☆素材編輯成的短篇劇場動畫，以下詳述。

星空Project編輯

  • 以UDK的詩「星空」作為歌詞的歌曲創作計畫。
    • 該詩內頻繁的使用了「星」一字，讓クッキー眾吐槽就算是三流偶像歌曲也不會有如此筆墨拮据的重複用詞

クッキー☆キャラソン編輯

  • クッキー☆系列出演者所唱過的歌曲。
  • 在クッキー☆☆引發風潮之後，KNN姉貴的歌ってみた被發掘，她充滿個性的歌唱博得高人氣，也帶動了其他クッキー☆☆出演者知名度的上升，如AZS姉貴的歌唱力也給了視聽者深刻的印象。

MMDクッキー☆編輯

  • 以クッキー☆為主題用MMD的方式來演繹的動畫，在這之前都是以純BB素材來製作，題材上多少有點侷限。
    • 拿掉跟クッキー☆的關聯就跟一般的東方MMD沒兩樣
  • 受到クッキー☆☆MMD版的啟蒙，每一個クッキー☆角色都有各自固定的既有風評被害東方角色模組，例如KNN姉貴所固定使用的nya式比良坂霊夢，現已成為KNN這個角色的定型。
  • 除了採用既有模組的風評被害模式，亦有根據クッキー☆本家繪畫為基礎所設計的原創模組，其中以青ポット兄貴所設計的青ポット式為最大宗。

一切的元兇：霊夢 so クッキー☆編輯

  • 使用原曲「I'm so happy（Ryu☆所創作的音樂遊戲用曲）」、クッキー☆以及淫夢為素材作成的音MAD。
    • 同時將「I'm so happy」及「クッキー☆」拉向深淵的最大元凶。
  • 儘管這部動畫的編輯技巧並不算特別出色，但是其以獨特的素材選擇以及後半的一轉攻勢一舉成名同時系列化。
  • 此音MAD造成的影響如下：
    • クッキー☆MAD的流行。
    • クッキー☆本家自此受到淫夢民的入侵。
    • I'm so happy徹底的淫夢化並產生眾多分支，就算動畫本身根本不含淫夢成分也避不過怒濤的淫夢彈幕。
  • I'm so happy系列的三個種類：
    • ○○ so クッキー☆：仿照靈夢的本家MAD使用其他クッキー☆角色製作，一轉攻勢部份也使用其他的淫夢角色相對應。
      • 在新的クッキー☆素材被發掘後，便立即有作者製作相對應的○○ so クッキー☆MAD已經是例行公事。
    • ○○ so happy：使用任意題材仿照「霊夢 so クッキー☆」的手法製作，一轉攻勢的部分可任意選用（不限定淫夢要素）。
      • 已經變成I'm so happy音MAD的既定格式，期待一轉攻勢的內容已經變成此類音MAD的樂趣。
    • I'm so happy -EX mix-；以I'm so クッキー☆ -EX mix-（原曲為「I'm so happy」的完整版）為範本所製作的強化版。
      • 「八王子」捏他的起源。
    • 除了I'm so happy以外，Ryu☆姊妹曲的音MAD也被染指（例如Sakura Storm及Sakura Reflection，其特徵跟I'm so happy如出一轍）

MGRoid編輯

  • 將MGR姉貴的聲音調配出如虛擬歌唱軟體般的效果的MAD
    • 著名代表為由にす兄貴製作，以平澤進創作的歌曲製作的三部音MAD《論理空軍☆》、《Big Brother☆》、《LAB=01☆》
    • 也因此MGR姉貴被取了個平澤進之娘的外號，甚至有人叫她平澤巡（MGR=めぐる→巡る）
    • 平澤進本人表示：這Remix做的挺不錯的嘛

BBクッキー☆劇場編輯

  • 基於淫夢本家的「BB先輩劇場」複製產生的延伸系列，早期的クッキー☆MAD仍然以音MAD為大宗，直到近期BB素材充實之後逐漸流行起來，同時也發展出與本家動畫截然不同的角色性格及特質。
  • 由於「BB先輩劇場」素材量以及成熟性，BBクッキー☆劇場含有淫夢成分是在所難免的。即使沒有淫夢角色出演也難免會援引淫夢音效等小素材。
