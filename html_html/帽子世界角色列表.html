主角[編輯]

  • ドーラ / 朵菈
    • 玩具世界的管理人。
    • 初期能力以靈巧見長。
    • 敵人時使用武器為銃，魔法為土，在高價值觀擊敗時可以提升熟練度上限，99%時為30。
    • 價值觀是「世界全部都是製造出來的東西」。
    • 帽子名：カテジサスタリム / 卡緹吉薩斯塔莉姆（Cartesian dualism substance dualism）。
      • 帽子解放的形象是提線的傀儡木偶。
    • 自家位置：無人主題樂園。
    • 初期特權：改造強化裝備、安裝歐帕茲。
    • 全破特典：自動採收地圖上目前所在的房間內所有素材或水晶。
    • 收集帽子的目的：完善製作人造人的理論並解除俄羅斯魔人偶的封印。
      • 由於前期收集帽子的目的理由太過莫名荒唐，使一開始的茱諾很不願意參與加入，直到朵菈喊她四眼田雞不得已只好參與幫助朵菈實施收集帽子計畫。
    • 第一人稱是「ボク」的僕娘，還會被洋子拿來吐槽。
      • 被普莉姆蘿茲說第一人稱是ボク的都不是正常人。
        • 雖然這說法怎麼看都是指桑罵槐暗諷用「僕」自稱的薇歐蕾特，然而當朵菈吐槽這點時普莉姆蘿茲卻表示薇歐蕾特因為長相夠好看而抵銷了。
    • 原本舊約是穿著內褲到處跑，新約修正成給朵菈穿上短褲了，可是褲子不穿好拉鍊沒拉好的呈半開狀....
    • 作者製作的所有角色中朵菈為最早誕生的角色，而最初朵菈的髮型和菈薇是相同的。
    • 自己的路線後期會揭曉一開始玩家操作的朵菈其身分是作為俄羅斯魔人偶菲優媞奴的材料而分裂出的半身-闇朵菈。
      • 闇朵菈的造型基本上為沿用舊約時期的朵菈2P版，除此之外膚色較原版黑，同時也是唯三有兩套戰鬥外型的主角。
      • 被奪去帽子的朵菈本尊在闇朵菈的真身揭曉後才會加入隊伍。
    • 在洋子篇一開始就被知曉真相的洋子救出並打敗菲優媞奴，與菲優媞奴融合之後於30章獲得了絕望魔法，同時也有著七大麻煩的身份。
      • 作者表示既然半身的朵菈就能當Top10了，完整的朵菈可能有望衝擊珍妮絲的地位。
      • 畢竟三姐妹都是俄羅斯人嘛。
    • 在現實世界偕同姊妹去聖彼得堡皇家莫斯科大劇院觀賞人偶劇時遇難（遇難原因沒有講清楚），因為帽子世界計畫而得以延續生命。
      • 被運送至帽子世界時第二位管理人出現，甦醒則是在達莉亞成為第三位管理人時。
    • 在洋子篇結局與姊妹在學園都市一帶定居。
  • シキ / 四季
    • 時之世界的管理人，魔法大師聯盟的時魔法大師。
    • 初期能力以速度見長。
    • 敵人時使用武器為弓（魔法大師聯盟作為敵人時使用武器改為杖），魔法為時，在高價值觀擊敗時可以提升傷害值上限，99%時為99999。
    • 價值觀是「過去無法返回」。
    • 帽子名：イリバルデュシャス / 伊莉巴爾狄烏夏司（Irreversible reaction duodecimal system）。
      • 帽子解放的形象是東方紅龍。
    • 自家位置：時間列車．世界樹。
      • 同時自己也是身為時間列車．世界樹的車掌，在自家可以選擇車外的各種景色主題（隨著遊戲進度變化能選的景色主題也會越多）或是嫌列車運作聲響太吵的話，可以自行選擇關閉列車運作聲響。
    • 初期特權：時魔法解禁、戰鬥時可看見敵我雙方的行動順序條[1]。
      • 同時也是唯一擁有反時停被動的管理人，新約被移至個人路線終盤習得，作為敵人時不管價值觀為何都會使用。
      • 舊約若是第一輪選擇她會嚴重破壞平衡。
      • 洋子線處於貓耳狀態時會有能夠貫穿對方蘋果耐性[2]的被動。
    • 全破特典：自動補充消耗道具。
    • 收集帽子的目的：在自己的帽子被搶走前先下手為強，實際上是為了讓時間列車倒退以取回大家喪失的記憶。
    • 固有能力為預測未來的「時詠み」。
    • 因為是梅兒一手拉拔大的，所以儘管時和夢的價值觀相性差，但和梅兒關係很好，因此在洋子篇在跟時鐘少女一戰結束後自己一同失去了時之帽子和時之世界後，和燕子多半都是輔助梅兒為主。
    • 個性極度頹廢，生活上很多方面的事物都靠搭檔燕子幫自己處理，甚至在其他管理人進攻時之世界時還幾乎處於完全放棄的狀態，因此作者對一周目選四季當主角的人有種微妙的想法。
      • 最初原案設定名字叫（カンナ / 神乃）[3]，後來改名原因是作者覺得神乃這名字形象太強烈，故因此決定更改名字叫四季。
      • 舊約髮色是褐色，新約變更成為赤褐色。
    • 在最終決戰中會擁有貓耳，洋子篇則是打倒時鐘少女完後會擁有貓耳，因為沒有肉體若要到現實世界就只能附在貓咪的身上。
    • 在現實世界的本名為「觀澤詩紀 / 観澤詩紀」，為了拯救被瀨法司了推下月台的鴨池燕子而被火車輾過去幾乎瀕死，因為帽子世界計畫而得以延續生命。
      • 但因為心理創傷過大而沒辦法完整消除記憶，不然會影響到整個帽子世界。
      • 因此記得舊約中變成帽子以及四季篇中變成貓的事，在洋子篇中曾經因為洋子看起來忘記她曾冒著生命危險救她的事而和洋子大吵一架，最大原因是洋子攻打時之世界讓本身就頹廢的四季更加憂鬱。
    • 在洋子篇結局與燕子回歸校園生活，早期版本沿用在帽子世界時的服裝，ver2.20更新後兩人改成穿各自高中的制服。
  • ナタリー / 娜塔莉
    • 法之世界的管理人。
    • 初期能力以魔力見長。
    • 敵人時使用武器為杖，魔法為全（除了時屬性），在高價值觀擊敗時可以提升瑪那上限，99%時為128。
    • 價值觀是「紀律與協調」。
    • 帽子名：ワドラボヤンティ / 瓦多菈博央緹（Work study school learn clairvoyance teach）。
      • 帽子解放的形象是被繃帶禁錮的蒙眼天使，自我吞噬戰的法帽戰天使會解脫禁錮自身的繃帶。
    • 自家位置：能看到一望無際大海的港城。
    • 初期特權：可以看見敵方的能力和弱點個性所有資料，包含身上的配方和武器裝備掉落率（但對上某些BOSS無法使用）。
    • 全破特典：可以在突破後的迷宮進行瞬移傳送。
    • 收集帽子的目的：妨礙監護人選拔，失敗後為了打倒球狀人類而暫不歸還帽子。
    • 個性和外表[4]甚至口語遣詞都是大小姐風格的女生，雖然表面講維持自己世界觀的紀律與協調，但其實個性有些唯我獨尊獨裁的一面。
      • 舊約娜塔莉篇還被洋子吐槽妳這個性不得了，新約娜塔莉篇的芙蕾德莉卡更稱呼她是「自我中心世界」的管理人。
    • 固有能力為觀測真實的「千里眼」（劇情中發動會有特殊CG演出）。
    • 新約在菈薇篇初期會加入，並且不會離隊。
      • 不過在法之世界的事件結束後，將其自編成除外依舊能進行挑戰，順便放閃
    • 在自己的路線中打倒帕皮後會解禁娜塔莉專用的寶石「大師全魔法」，能使用全部的魔法&大魔法並享有最高級優惠，裝有該屬性寶石才能速發的魔法也能速發，可以繼承到菈薇篇繼續用。
    • 在現實世界被違規駕駛撞死，因為帽子世界計畫而得以延續生命。
    • 在洋子篇雖然是最早發病的管理人，不過遭遇自我吞噬的時間點比其他人還晚。
      • 原因是18章被瑪麗召喚的雙重懲罰的存在讓法之世界價值觀過高而減緩吞噬速度，所以被洋子和菈薇放置一陣子，因此9章洋子就特別要求菈薇先暫時別去治療娜塔莉。
    • 在洋子篇結局為了成為老師而與菈薇前往學園都市進修，在背景的機場還能看到幾個支線任務的NPC路過。
  • ジャニス / 珍妮絲
    • 戰之世界的管理人，曾經擔任Provider，在艾麗絲掀起的帽子戰爭擔任被搶奪帽子的梅莉莎守護的一方，但保護梅莉莎失敗後艾麗絲本人和梅莉莎的帽子一起消失，這事件過後珍妮絲就選擇辭掉了Provider的職位。
    • 初期能力以力量見長。
    • 敵人時使用武器為雙劍，魔法為火，在高價值觀擊敗時可以提升能力值上限，99%時為50。
    • 價值觀是「敗者什麼都得不到」。
    • 帽子名：クオストワット / 庫歐茲特瓦特（Quarrel forces fight war battle）。
      • 帽子解放的形象是餓狼，自我吞噬戰最終決戰的戰之帽子會變成全身紅身上插滿刀槍的超大隻餓狼增加氣勢。
    • 自家位置：荊棘之城。
    • 初期特權：二刀流解禁、可在地圖上施展二段跳及下踢，\ﾄﾞｩｴ/。
      • 新約的二刀流可學習全範圍的專屬必殺技「月閃裂削斬（ムーンスクレイパー）」與「明鏡止水」。
      • 下踢可以直接清除較弱的敵人，就算對手較強使其下踢也可以達成我方先制攻擊。
    • 在洋子篇中會解放全觸發寶石，除了識破和鎖定率外都包含在裡面，而格擋的機率還會提升至兩倍。
    • 全破特典：可以使用帽子解放（僅限管理人自己所屬的世界領域）。
    • 個性沉默寡言不善言談，多半交給搭檔梅莉莎或茱諾幫自己發言。
      • 可本質仍是個熱血戰鬥狂，跟洋子對話幾乎都會扯上與戰鬥有關的事物，搞得洋子很傻眼。
      • 跟巧克力對話可得知「珍妮絲本人除了常常鍛鍊與戰鬥外，其實閒暇之餘也會看食譜之類的書籍，也就是喜歡自己做點心」這種意外的反差萌的一面。
    • 收集帽子的目的：針對上下的五人（甚至最終也包含自己在內總共六人）履行Provider「死神」的職務，並討伐帽子變成的自我吞噬。
    • 設定上是主角六人中實力最強的管理人[5]，而劇情與最終BOSS戰也可說是六人之最（令人絕望的意味）。
      • 因為自我吞噬戰是總力戰的緣故，珍妮絲篇終盤所有同伴都會強制入隊（包括原本必須解支線任務才能加入的芙蕾德莉卡跟凱莉也是）。
      • 如果最終戰全員生還的話會於淚水行星中的大海醒來，所以把全部人練好練滿吧，珍妮絲篇和洋子篇全員上場有練有保障。
    • 在現實世界的本名為「珍妮絲‧法庫哈（ジャニス・ファーカー）」，麝香同母異父的姊姊。
      • 原本是救生員，在搶救溺水的遊客時遇難，因而成為第一位接受帽子世界計畫延續生命的個案。
      • 在玩家間有著這名遊客是梅兒的猜測，不過這個說法被作者給否定了，在梅兒溺水前，珍妮絲便已經在生命維持裝置中延命了。
    • 在洋子篇一度因為洋子於各地奪取帽子、妨礙麝香的Provider職務，以及倒退時之列車等理由將洋子列為與七大麻煩同等級的危險人物。
      • 在NPG機關佔領戰後被BigMama說服，決定遵循自身意念而戰並與洋子陣營會合對抗自我吞噬。
    • 在洋子篇結局會換上新衣裝，與麝香正式相認，在看了姊妹的合照後想起了過去姊妹兩人相處的記憶，因為沒了先前的工作，在找到想做的新事物以前只好厚著臉皮暫時寄人籬下在麝香身邊。
  • ラヴィ / 菈薇
    • 命之世界的管理人，個人路線最後會成為愛之管理人。
    • 初期能力以體力和魔力見長。
    • 敵人時使用武器為槍，魔法為水，在高價值觀擊敗時可以提升生命上限，99%時為1280。
    • 價值觀是「生死皆平等」。
    • 帽子名：アキュリメルフ / 雅裘麗玫爾芙（Alive cure recover animate heal life）。
      • 因為價值觀是包含了生與死這兩種相反的東西的緣故，敵方時帽子解放形象是鐮刀死神，我方時或發動基因根源時的帽子解放則是祈禱的聖母，自我吞噬戰的命帽戰則採用前者。
      • 作為BOSS戰時若成功發動基因根源的話，除了獲得全能力提升之外還會自動回復生命值。
      • 帽子之力不知為什麼常常被人拿去用。[6]
    • 自家位置：聖地．生命樹。
    • 初期特權：戰鬥結束後生命值自動完全回復，OS解除並防止負面效果。
    • 全破特典：可以自由調整已攻略世界的價值觀數值。
    • 收集帽子的目的：找出完善管理人特有疾病的治療方法以拯救娜塔莉，並且幫助失去身體的達莉亞復原（可實質上只能做到一個，而對菈薇而言是全都要做到）。
    • 個性如自己外表一樣溫柔討厭紛爭（不過舊約菈薇被設定比較嚴肅一些[7]）。
      • 然而在舊約菈薇篇被洋子問及為何挑起搶奪帽子鬥爭時會自我強迫性稱自己愛戰鬥愛殺戮。
      • 除了跟自己親近的娜塔莉外，對其他人總是以敬語與人對話，都會在叫他人名字後面加「さん」（中文譯作先生或小姐）。
    • 新約在娜塔莉篇初期會加入，並且不會離隊。
      • 不過在命之世界的事件結束後，將其自編成除外依舊能進行挑戰，順便放閃
    • 在朵菈和四季篇中，前兩小關會傻傻的把帽子交出去並邀請她們吃午餐喝茶聊天，當朵菈和四季拿到帽子後娜塔莉會突然殺出來的出現把命帽給搶回來，而在聖地．生命樹對上她時會因為娜塔莉去洗澡所以能選擇直接獲得帽子或戰鬥。
      • 前者是因為菈薇想幫朵菈，後者是因為菈薇聽四季說她以後會去搶四季的帽子，所以為了不造成四季困擾就把帽子給四季以防止這件事發生。
      • 雖然可以不進行戰鬥就直接獲得帽子，但如果在選項之前SKIP掉的話就只能乖乖打了。

  • 魔トリョーシカ / 俄羅斯魔人偶
    • 朵菈篇的最終BOSS。
      • 舊約為隱藏BOSS，造型是比較成熟的瑪莉亞人形。
      • 新約的造型則是改成朵菈外表+穿上黑暗魔女裝+頭上戴著兩個骷髏吊飾和上部有三隻眼+鯊魚尖牙的邪惡帽子，雖然大幅萌化，背景卻比現在還要掉SAN值。

      • 在別人的眼中看來是個想吃人腦的怪物，實力強悍連管理人朵菈都無法破壞，最後被普莉姆蘿茲給封印。

    • 價值觀是「希望會變成絕望」，因此擁有的特殊能力是「強化魔法反轉」。
      • 會為我方上鳳凰跟自動回血後再使用強化魔法反轉，鳳凰會變成即死，自動回血會變成毒，因為是把buff直接換掉，所以OS直接無效，不想耗費觸發寶石次數的話土魔法的「消失回歸」是唯一的解法。
      • 另外，三種跟魔法輸出有關的詠唱魔法（雙重詠唱，犧牲詠唱和聖禮詠唱）也會被反轉成魔力下降。
      • 同樣的，定期會使用魅惑，墮天使的繃帶可以派上不少用場。
    • 強度調節方法為闇之世界的價值觀。

      • 舊約為本篇（六位主角的篇章）的最終BOSS。
      • 新約追加第二套高貴女王禮服裝的立繪（僅限在非現實世界時的帽子世界時期有這樣打扮）。
    • 最終BOSS的造型是圓環環圈的大眼睛，且戰鬥時環境搭乘的升降梯也會隨著此戰進度而越來越急速。
    • 戰鬥時會使用主角們所擁有的各種帽子之力。
      • 戰鬥後期會流出血淚並施展帽子之力的OS「Always Oversoul」。
      • 舊約時依照對應帽子還會施展讓眾多玩家頭痛的蘋果和基因根源，新約被改成其他招式。
      • 洋子篇還會在「Always Oversoul」狀態附加「Program ALL」直接使用六主角的帽子之力並且一開始會就放基因根源，必須盡快打倒，拖越久越有全滅風險。
    • 新約正式公布本名為梅姆‧奧肯（メイム・オーケン），同時真實身分為梅兒的生母，為了拯救成為植物人的女兒而將自己化為程式創造出帽子世界。
      • 肉體年齡33歲，實際的年齡不明。
      • 洋子篇中會在NPG機關佔領戰中進行戰鬥，在被打敗後變回人類原型並一度希望由梅兒這個自己的親生女兒親手手刃了自己，但在洋子的勸阻下梅兒選擇放棄殺死自己的母親並願意和自己和好，之後和珍妮絲見面並讓珍妮絲釋懷放下重擔。
      • 洋子篇結局會與丈夫和女兒一家三人一起移居德國。
    • 強度調節方法位於麝香的房間內的電腦（需全破一次）。
  • マリスレーゼ / 瑪麗蘇蕾澤


    • 洋子篇的最終BOSS，為瑪麗蘇蕾澤大教會信仰的神，同時也是現實世界的舊管理人。
      • 被涅勒斯四世打開棺材召喚後因力量過於強大連涅勒斯四世都無法駕馭，所以在被解開棺材封印後直接附身在伊索拉身上並攻擊十三使徒以及涅勒斯四世，伊索拉被打敗之後就會現出真身。
    • 總共有三次戰鬥，分別是附身在伊索拉的普通狀態、女超級賽亞人災厄狀態以及真身，事實上真身戰是劇情演出，如同四季篇最終戰。
      • 此戰無法使用奇蹟世界，發動後的可選效果只會出現「讚美主！讚美瑪麗蘇蕾澤！」這類對戰鬥毫無助益的文字。
    • 伊索拉的普通狀態會使用瀕死（斬血99%），不用治癒就是等死，還會全體消狀態跟無限次數的妨礙詠唱，OS絕招「ホーリーヴェノキシア」為隨機的高傷40段傷害，單純的復活寶石沒用，若在釋放之前被打倒會直接變身而不使用。
    • 災厄模式開場會先制使用搶奪寶石，優先順序為時魔法（用過蘋果）>什麼都不搶（大師魔法或大師全魔法，用過蘋果）>愛魔法＞大師全魔法＞絕望魔法，如果無以上寶石則不會發生任何事。
      • 還會多一招OS逆轉（OS歸0並按照%數斬血，如果滿會直接死亡），被打倒後會離開伊索拉並現出真身。
      • 雖然不管什麼事只要洋子裝真紅のべヘリット歐帕茲&全員生存三寶治癒復活大師寶石都能解決就是了，賽蕾娜（過去梅莉莎）另當別論。
    • 真身在第一回合會使用時空間計算等技能讓自己一招有10幾萬傷害實際上只有個位數就是了，並每回合補99999血，第二回合其他六主角會出場並給予洋子BUFF和回復，第三回合會全體OS並帽子解放，傷害上限值變為999999一次秒掉瑪麗蘇蕾澤。
