目錄

解說編輯

  • 任天堂的模擬經營遊戲《動物森友會》系列[2]第7作，於2020年3月20日發售。
    • 系列首部登陸NS平台的作品，但「我的設計」仍可讀取3DS版本的QR code
    • 宣傳標語為「正因為什麼都沒有，所以什麼都做得到」
    • 與以往經營村莊不同，本作改為經營無人島，但仍然保留了極大的開放性與自由度。
  • 於本作玩家扮演的角色為參加「無人島移居計劃」的人類，玩家可以隨自己所想把無人島改成自己喜歡的樣子。
    • 因此本作擔當的角色是「島主」，而不是「村民」
    • 同樣地玩家於本作沒什麼必須要做的事，因此可以隨心所慾做自己想做的事因此可以看到各種大神的超強創作
  • 因受疫情影響本遊戲僅發售3天便達成日本國內共售出超過188萬套的成就。發售6周全球銷量已經達到1341萬份，超過前作《來吧！動物森友會》的8年間的總銷量。
    • 根據老任在同年8月6日公布的財布顯示，因為本遊戲的大賣使老任達成了自身歷史上最好的季度成績，營業利潤為上年同期的5.3倍。本遊戲更成為老任第二暢銷的遊戲。相關新聞可點此處瀏覽

系統編輯

無人島編輯

  • 玩家居住的與經營的地方。因參加狸克的「無人島移居計劃」而移居至此。
    • 玩家在前往島嶼之前可以從4個島嶼種類中選擇1個自己喜歡的，但除外觀外其實沒什麼差別。
  • 島嶼名字決定後便不能更改，如要更改只能刪檔重玩
    • 初次登陸無人島時一定會有另外兩位動物作鄰居。隨著遊戲流程玩家可以招攬更多的動森鄰居。上限10位。
  • 每個人的島都有獨自的原生水果與花，可以通過去外島/別人的島撿拾偷竊其他種類的水果與花
  • 島嶼中間是廣場，當達成條件時K.K.便會在每週六到廣場上開演唱會。島上的新聞可以在報告板查閱。
    • 服務處與旗幟都位於廣場上，另請注意廣場上不能放置物品（如把樹種在廣場旁邊，當樹果掉下時便會直接在廣場上消失）
  • 雖然是無人島但當主角去了居住後就不再是「無人」島了吧（遊戲角色也會吐槽這點）
  • 原本與世隔絕的小島突然被人類與一群動物來訪並破壞島上原本的環境與生態系發展成各種樣子，人類太可惡

家編輯

  • 玩家在遊戲內的家居。
  • 最初只是一個小帳篷，玩家以哩數還清移居費用後便可以向狸克申請債務建造房屋。
    • 房屋正式建成後便可以利用儲物功能和裝飾家居。
  • 最初只有一個非常小的大廳，可以透過還債後再向狸克申請金額越來越大的債務進行擴建
    • 擴建順序為：擴建大廳（2次）→增設房間（3次）→增設2樓→增設地下室
    • 達成一定進度後可以透過付錢改變家的外觀。

服務處編輯

  • 位於廣場上，狸克與西施惠（擴建後才出現）的工作地點。
    • 狸克會協助玩家辦理房屋擴建的貸款，也可以進行橋樑與斜坡等的建設，或是區域規劃移動部份建築物的位置等。
  • 起初只是個小型帳篷，隨著進度達成就會改建成大型建築物。改建後西施惠便會在此處出現。
    • 西施惠則能協助玩家設定島嶼旗幟、島嶼主題曲，確認島嶼評價。
  • 服務處內則有名為「狸端機」的多功能ATM。除本身的ATM功能外也可以用作訂購商品、利用哩數換取物品等。
    • 每天第一次使用狸端機時會視作登入並贈送額外的狸數，獎勵最高疊加7天共300哩數。
      • 於服務處改建後的第二天使用則無視前面日子直接當成第7天。
  • 服務處改建前有小型工作台讓玩家使用。

機場編輯

  • 玩家可以從島嶼的機場離開自己的家，往別的島遊玩或是採集資源，通稱出外島。
    • 若要前往外島則需事前通過狸端機以哩數交換機票，其後才能在機場處尋找莫里乘搭飛機。
  • 外島有分為普通的素材島嶼（會出現各種果樹、樹木等）以及各有特色的特別島嶼。例如有無限狼蛛/蠍子的「狼蛛島」、有著各種特殊花色的「花之島」等等
    • 若在前往外島前無人島上有事前預售作房屋的土地，則能在外島遇上未有居所的島民，其後可以通把對方邀請到自己的無人島上居住離島ガチャ（主線上第一次尋找3位新島民時則不需要預售土地）
    • 在外島上莫德里機長會提供以哩數換道具的服飾，玩家被打暈時也是由他救助玩家。
  • 另外也可以在機場處選擇前往其他玩家的島，或是邀請其他玩家到自己的島，可以分為朋友限定、密碼搜尋等。前往玩家的島時則不需要機票。
    • 在別玩家的島上可能會有部份居民想要搬走，此時也可以通過對話增加他搬到自己島上的成功率。
    • 另前往巴獵島也不需要機票。

NOOK商店編輯

  • 可以把各種東西都拿給商店內的豆狸和粒狸收購
    • 注意本島原生的水果只能賣出100元，但可以通過到別人島的商店販賣解決。
  • 原本只是辦事處內的小型地攤，隨著流程就能擴建商店。
    • 商店由路邊攤升級成大型商店後營業時間會變成早上8時-晚上10時。也只有升級後才會收購大頭菜。
    • 每天都會選出1種DIY物品向玩家以2倍價格收購。
    • 再達成條件就可以進行擴建（會停業一天施工，持有大頭菜時請注意），擴建後除了提供更多物品供選購之外，以2倍價格收購的物品會增至2種。
  • 商店每天也會有特別的家具商品出售
    • 商店關門後也可以使用門外的收購箱收購物品，但會收取總金額的20%手續費。
  • 冷知識：不同語言版本下商店的招牌也會有所不同。

博物館編輯

  • 可以把初次捕獲到的魚、蟲或鑑定到的化石送給傅達，數量足夠後便會建成博物館。
    • 2020年4月23日更新後可以捐贈藝術品真貨，同年7月3日更新後追加捐贈海洋生物。
  • 為了響應國際博物館日，於2020年5月18日起至5月31日舉辦了博物館蓋章拉力賽，能夠在三個生物展館裡蓋下3個獎章以換取壁飾。

＊ 2.0更新後與傅達交談可擴建出咖啡廳敲碗了一年終於來了。有時各個島民包括西施惠都會到咖啡廳上休息。

碼頭編輯

  • 位於島上一角的碼頭。本來純粹是特定魚類的出現點，2021年11月5日更新後可以在此乘坐航平的快艇遊特別的島嶼（每天限定一次）。
    • 可到達的特別島嶼別於飛機時的島嶼，可獲得的素材也有很大差別。
      • 例如有獲得新植物的「奇異植物島」、「農作物島」，可從石頭敲出流星碎片的「流星碎片島」等。可以獲得各種全新的素材。

季節編輯

  • 於遊戲開始時玩家可選自己的島位於南半球還是北半球，會影響目前島上出現什麼生物與灌木叢開花的種類。
  • 此外不同季節時也會有不同的天氣變化，島上也會出現各式的額外物種素材供收集，也可得對應季節素材的DIY。

採集要素編輯

種植編輯

  • 本作可供種植的有樹、花、灌木叢和蔬菜。
  • 樹
    • 分為果樹與普通樹兩種。果樹在成熟後會長出果實，把果實搖下後便可以拿來吃或是種植。
    • 種植樹木時請確保幼苗之間有一格的空隔。另幼苗附近不能同時有太多樹/樹苗不然不會成長
    • 本作的果實有櫻桃、梨子、橙、蘋果與桃子。
    • 普通的樹有闊樹葉與針葉樹，在搖動時會隨機掉出傢俱、鈴錢、樹枝或是蜂巢，有時也會搖出蜘蛛和蓑衣蟲。
      • 搖出蜂巢的話會有整群的黃蜂追著玩家攻擊，被刺中1次的話會受傷，再刺中就會昏倒。
        • 黃蜂可以利用拉炮擊退、或看準時機以網子捕獲，也可以跳到建築物或是瑪利歐合作的水管內避難前提是要跑得比牠們快
      • 秋天時闊葉樹有低機率搖出橡栗，針葉樹有低機率搖出松果。
      • 闊葉樹會隨著季節、活動不同而有不同外觀：例如賞櫻季時會變成櫻花樹，冬天時會披上積雪
        • 另有1種特殊的闊葉樹是通過把鈴錢種在有發光的特殊地洞（每天限1次）內生成的發財樹，會依據玩家種植時所埋的鈴錢而「結出」3袋鈴錢。當採收後就會變回一般的闊葉樹。
      • 椰子樹只能在沙地上種植，搖它只會掉椰子不會掉其他東西。
      • 竹也分類為樹木的一種。只是搖動時不會掉出東西，果實則是隨機長在附近地面的竹筍（一棵竹只會結竹筍一次）。
  • 花
    • 花雖然不留空格也沒問題，但留有空格可以增加配種的成功率。有不少花的顏色都需要通過配種才能取得。
    • 本作的花有銀蓮花、百合、三色堇、玫瑰、鬱金香、菊花、波斯菊、風信子。當中以玫瑰有著最多的特殊花色。
    • 此外還有一種稀有花「鈴蘭」，需要島嶼評價遠五星才會隨機在島上自然生長出來且無法交配和採花，顏色只有白色。
  • 灌木叢
    • 算是花和樹的合體，種植上雖然沒有限制也不能配種但長大後會阻擋去路。
    • 不同灌木叢只會在特定日子才開花。日子不對時就只是一堆草。
    • 2.0更新後會多出「雞蛋花樹」的特殊品種，只能通過乘船到達的特殊外島才能遇上。
  • 蔬菜
    • 舊版只有為萬聖節而設的南瓜，於2021年11月5日更新後提供更多食用植物給玩家種植。
      • 品種有小麥、甘蔗、番茄、胡蘿蔔、南瓜、和馬鈴薯。
    • 玩家需要在然然的植物店買下菜苗方可種植，最初種下時需要等3天收成，之後每隔2天可收成一次，每天澆水可以增加收成量。
      • 南瓜苗在10月裡Nook商店也可以買到，同時期然然的路邊攤的南瓜苗會以半價發售。
    • 小麥之類的菜苗則可以通過乘船方式到達特殊外島採集。另外採收了的作物也可以當成種子種植。
    • 南瓜有4種顏色，橙色南瓜出現率為最高（約7成機率），低機率出現黃色、白色和綠色南瓜。
      • 此外把南瓜直接種在地上也會變成南瓜苗，長出來的南瓜的顏色和種下去的南瓜一樣。
      • 橙色南瓜除了用作料理外也是製作萬聖節DIY的素材，其他3種南瓜則可用作改造萬聖節DIY物品。

捕蟲編輯

  • 島上到處都有蟲子出現，玩家可以製作捕蟲網捕捉。
  • 部份蟲子需要以特定方法誘使牠們出現。
  • 初次捉到的蟲子可以捐贈給傅達放到博物館的展館中，或拿去商店販賣。
    • 如果島上有龍克斯的話可以以較高價錢販賣。另有3隻同樣的蟲子的話也可以給他製成蟲子模型。
      • 下訂後的蟲子模型會在第二天以郵寄形式寄給玩家，注意每次只能下訂一款。
    • 不販賣的話也可以直接裝飾放在家內或是島上。

釣魚編輯

  • 本代的水域分為6種，為大海、河川、池塘、瀑布、碼頭、出海口。每個地方可以釣出的魚都會有所不同，隨著月份與南/北半球的不同出沒的魚類也會不同。
    • 例如鬼頭刀只能在碼頭釣出，並且北半球的島嶼只能在5-10月才能釣出。11月到隔年4月無法釣出鬼頭刀。
  • 魚影分為小魚影至大魚影數個分別，不同魚影釣出的魚也會不同。例如小魚影會釣出海馬或是小丑魚等小型魚類
    • 另外有著背上有鰭的鯊魚魚影、全身像蛇的鰻魚魚影等等殊魚影。
  • 初次釣出的魚可以捐贈給傅達放到博物館的水族館中，或拿去商店販賣。
    • 如果島上有俞司廷的話可以以較高價錢販賣。另有3隻同樣的魚類的話也可以給他製成魚類模型。
      • 下訂後的魚類模型會在第二天以郵寄形式寄給玩家，注意每次只能下訂一款。
    • 不販賣的話也可以直接裝飾放在家內或是島上可是那個水箱之小就像虐待動物

潛水編輯

  • 2020年7月3日更新追加的功能。在穿上潛水衣後即可跳到大海中游泳與潛水。
  • 可以在海中捕獲到一些海洋生物，例如海葵、海膽等。
    • 若於大海中成功捕獲帆立貝的話會出現新NPC「阿獺」（每日限一次，而且玩家持有物已滿、持有昨天捕獲的帆立貝、在離島時不會出現），把帆立貝交給他可取得特別的「人魚系列家具」的DIY方程式、「人魚系列」服飾或珍珠還附送一句哲理名言。
  • 值得一提的是俞司廷不會收購大海生物，也因此無法製作相關的動物模型。但可以捐給博物館。

藝術品編輯

  • 2020年4月23日更新後實裝的系統。只能在不定時到島上的狐利的小船內購買。
  • 有機會會買到贗品，而贗品博物館不會接受商店也不會收購，因此除了丟到垃圾桶外就只能留著。
    • 但部份贗品可以互動故反而比真品更有價值。[3]
  • 玩家有時候會收到島民買下並寄過來的藝術品，同樣會出現贗品。

大頭菜編輯

  • 大家熟知的本遊戲的股票[4]，只能在每週日上午在曹賣處購買
  • 大頭菜在Nook商店裡的賣價在每天的上午和下午都會有變動（升跌幅非常大，最小可以10元最大時可以逾600元）
    • 而且到了翌週的週日就會腐爛，不能以市價賣出[5][6]，是一種有風險也有機會一夜致富的期貨。
    • 因為每個玩家的大頭菜價格變化都不一樣的關係，有不少玩家會在自己的島出現高賣價時開放給其他人入島散貨，甚至徵收入島費/物品賺外快的也有。

  • 某公主：「大頭菜是用來砸人的東西。」

DIY方程式編輯

  • 本作製作道具的重要方法。雖隨著主線進度都會送一些但要全收集還是免不了跟別人交換。
    • 個人收集方法有：射氣球、與製造DIY中的島民交談獲得（不分本島還是別人的島）、本島或外島的飄流瓶。
  • 持有方程式後再持有相應的材料就可以在工作台製作出相應的道具。
  • 2.0更新後新增了名為「食譜」的新種DIY，基本上就是把成品換成食物的DIY。
    • 食譜的獲取方法除了與上面一樣外，還可以通過釣到相應的魚類、咖啡老闆贈送的方式。
    • 製作食物除了可以在DIY工作台外，也可以在相應的煮食系家具上製作。（如爐灶廚房）
      • 在DIY工作台製作時會看到玩家用鎚子處理食材，而且還會直接用雜草來調味，這樣的料理真的大丈夫？

      • 不少人都會把遮陽帽畫成蛋榚、鬆餅等食物

      • 但港澳台方面不受影響，因此中國大陸玩家可以利用VPN連接港澳台的eshop購買。同時現在大陸仍有部份商店沒有下架。部份網店則以別名「集合吧！猛男撿樹枝」的名義販售。
  • 事實上這些能自由發佈個人資訊和設計的遊戲通常都有規則限制不當內容，本遊戲內的夢遊系統也有提供舉報機制，若在夢遊裡遇到有違規疑慮的島，可使用舉報機制處理。

政治宣傳問題編輯

  • 除了上述內容外，也有政治人物曾經以本遊戲為政治宣傳媒介。
    • 如2020年9月，日本民主黨前幹事石原破茂就利用本遊戲為自己的競選團隊作宣傳，結果一天後立即被任天堂以「限制參與政治活動」為理由要求停止。
    • 2020年美國總統大選裡，民主黨參選人拜登也有發佈了一個島為自己拉票。到了同年11月，美國任天堂就發佈了一篇本遊戲的遊玩指南，其中一項表明「不要創作含低俗、歧視及人身攻擊成份的內容。也不要把政治帶入遊戲」，也就是說拜登此舉從此之後在美國會變成違規。問題是選舉早已結束了，而且拜登也贏了

與肺炎疫情的關係編輯

  • 本作推出時由於現實世界正值武漢肺炎肆虐全球，許多地域戶外活動被限制，因此在家遊玩遊戲成為不少人的唯一活動。
    • 同時推出的本作就因此成為最好的遊玩對象，因此達成了系列最高的銷售量。
  • 同時因為本遊戲的極高自由度，因此出現了利用遊戲代替各種儀式的情況
    • 例如有新人利用遊戲舉辦婚禮、也有學生舉辦畢業典禮
    • 甚至有部份政府部門、廠商都用此遊戲來作宣傳
    • 美國紐約大都會博物館與台灣故宮博物院甚至把館內部分或全部展品QR化開放予玩家免費下載，讓玩家在遊戲內也可以欣賞真實的展品。

接觸此遊戲的名人們編輯

  • 因全球疫情問題導致全球多家攝影所停止拍攝，各國均有當地以至國際名人被發現原來有玩本作的新聞，蔚為一時風潮。
  • 有遊玩本作的各方名人計有演員、聲優、歌手以至運動員等，以下為部分舉例（歡迎追加）：
    • 在《仮面ライダーゼロワン》中飾演不破諫（假面騎士Vulcan）的演員岡田龍太郎便在Youtube上多次以本作作遊戲實況，還曾邀請イズ的演員鶴嶋乃愛上島比賽釣魚結果被對方洗了一臉
    • 於電影《魔戒》系列中飾演主角佛羅多的Elijah Wood曾到訪網友的島上賣大頭菜並與其他玩家互動
    • 台灣歌唱組合「五月天」的主唱阿信在FB上直接PO文問有沒有玩家可以讓他賣大頭菜
    • 日本AV女優波多野結衣在遊戲裡重現了AV拍攝現場連情節也想了一輪

被玩家們爭竉的傑克編輯

  • 今集其中一個新增的動物村民「傑克（ジャック）」為一隻擁有異色瞳和戴上方框眼鏡，流露出斯文氣質的雄性貓（此外笑的時候會露出虎牙）。
    • 因為實在非常好看的關係很多玩家都想招他上島，但也因為他是今集的新角色的關係，無法用過往發售的Amiibo卡牌直接召喚，只能透過島上的露營帳篷或離島旅遊的隨機相遇碰碰運氣。[21]
    • 網終上甚至出現傑克販賣潮，售價當然非常昂貴。
      • 甚至有駭客組織為打壓傑克販賣潮而大量複製傑克，免費發放給想要他的玩家。P.S.：接受以非正規途徑得來的居民乃不當行為，可能會承受被ban的代價！
      • 僅次於他的新居民「茶茶丸🥺」也沒有被賣得這麼誇張
  • 當然也有不少玩家認為這樣會破壞本遊戲的氣氛，因此創作了各種梗圖來「抗議」（例如箱中的傑克），甚至有激進玩家要求任天堂刪除傑克這個角色就算刪了他大眾的目光也只會集中於另一個居民吧。
    • 玩家們以村民外表決定是否留他在島上的「嚴選居民」行為也成為了網路熱門話題之一（見下）

嚴選居民問題編輯

  • 本作可以邀請最高10位的島民來自己的島上居住成為鄰居，除了遊戲開始時隨機分配的兩位外，遊戲中總共有390多位的島民可以招攬。
    • 招攬方法為通過機場往外島去，跟在在那兒遇上的生物對話後就可以邀請到島上居住（事前需有已預售的土地）
      • 隨著遊戲進度也會解放利用AIBO卡招攬島民的方法，也可以到別人的島人招攬別人有意離開其原住處的島民
  • 但是因為本作島民眾多，不免會遇上某些外觀醜陋不怎麼樣的動物，招致部份玩家對該島民的討厭與反感（最佳例子便是大猩猩）
    • 還有鄉民弄出島民人氣等級列表，把所有角色以SR-F級分類
    • 而若不慎有不是心儀的島民搬來的話，部份玩家會用各種玩法把對方給趕走。
      • 諸如以物品、陷阱等圍堵別人的家、以網子瘋狂打島民等，直至對方搬走為止。
  • 由於此等「以貌取人」的行為讓不少玩家聯想到現實的欺凌問題，因而招致部份玩家反感，認為就算不喜歡該島民也不應該用這種方式趕走別人，因而掀起一場爭論
    • 甚至有繪師以此作題材繪畫漫畫諷刺此等行為而大炎上
  • 但其實島民搬走與否是隨機的，不存在逼害行為就會加速搬走的情況。相反好感度越高反而越容易搬走。
    • 因此越想對方搬走反而應送更多的禮物（最佳選擇是多出來的化石）才是最好做法
