最終陣容
  • 光之美少女系列的第七輯（第五代），簡稱「HCPC」或「ハトプリ」、「鳩プリ」，另有「抓心/奪心組（Heart catch 直譯）」等稱呼。在2010年2月7日接續超展開的《フレッシュプリキュア!》播出。
    • 如同風格大變的上輯，人設一出就被預言搞不好會成為黑歷史
    • 但當官網公佈水樹奈奈擔任第一主角聲優的時候，再次吸引大友們的注意。
    • 除了主角們的聲優，連配角到路人同學多數都由知名聲優演出，果然是東映的陰謀
  • 號稱回歸原點，以三、四代的戰隊元素來演繹雙人搭檔。
  • 接在《仮面ライダーディケイド》（誤）後面又是二人組，因而成為《仮面ライダーダブル》的熱門惡搞對象
     
    • 事實上播出後因為可惡搞的點太多，騎士捏他以比例來說反而不如上代常見，反倒是在《ダブル》的惡搞中出現了不少抓心的捏他。
    • 碰巧的是，在接檔的《仮面ライダーOOO》中，主角火野映司要收集オーメダル的劇情和本作要收集心之種非常相似，結果光之美少女和假面騎士的關係繼續糾纏不清……
  • 劇組再次改動。
    • 這次的監製是在前三代由製作升上來的長峰達也。
    • 系列構成是以子供向製作出名的山田隆司，在本作中使用化名「栗山緑」，目前兼任暗之美少女預定岳母的專任秘書
    • 人設是前三代的OP/ED以及電影原畫的馬越嘉彥。
    • 順帶一提，此三位也是東映子供向神作《小魔女DoReMi》系列的靈魂人物。
      • 如果單看人設一大票人會覺得這是小魔女第五輯，也有很多人覺得標題應該改作《不可思議星球的光之小女警Z》，因此光美初代人設的忠實支持者在水樹出馬前一度罷看。
      • 不過馬越過去成熟畫風作品的特色也融入其中，被開場誤導成新造人間ガリレオSins的觀眾也不在少數
      • 最後協議由兩名小麻煩魔女與兩名卡夏組成隊伍，違和度直飆歷代寶座
    • 另外「地獄兄弟中的弟」米村正二也是今年編劇之一，幸好沒把騎士棚的詛咒帶過來
    • 由於資深的小魔女系劇組撐腰，一掃前幾作黑歷史的陰影，從而成為公認的光美傑作。
      • 大友對作品的風評、周邊的相關收入都是歷代最高，真正地達成了名利雙收的巔峰。
      • 本作是2010年日本オタク大賞得主，為唯一取得此獎項的光美作品。
  • 如同新鮮組，開創破壞系列慣例
    • 世代傳承的設定使光之美少女大量出現，連帶突破年齡上限，水天使和莓天使大喜
      • 在光之美少女殿堂中證實有許許多多女性與妖精為心之大樹奉獻心力，幼女幻想破滅
      • 現役成員最年長的是高中二年級，十七歲一度留職停薪的月影百合。
      • 已退休的花咲薰子不僅是在就業時入軍因而是紀錄上最高齡，就連後來在劇中變身時也是以年過半百的年齡變身，這樣還能叫「美少女」嗎？！
      • 除了超齡藤色外，還有熟度顛倒的藍黃與拉高平均主角智商的粉紅……
    • 首次出現2+1+1的隊伍配置，中後期（30話以後）的追加戰士（月光天使）。
      • 但在第1话有过先行登场。
      • 因為同樣的原因，其實很不好編排，囙此中後期入隊的光之美少女在下一代組曲組之後乾脆消失匿迹了。
    • 因為劇情本身太惡搞所以預告尾圖惡搞消失，同樣的圖一用就是十集。
    • 主角都有病（無誤）。
    • 洗腦電波消失，變身大絕咒語要由精靈轉達，連名字口號都要自己想，造就史上最弱傳說……
      • 不過在《All Stars DX2》中主角兩人還是中了洗腦電波的「我剛剛在說啥」詛咒……
    • 精靈不是警報器而是偵察機，且戰鬥時不是躲到旁邊而是留在場上擋路
    • 內藏緊身褲設定縮到最短，外表已無法辨識，證實不是薛丁格的內褲
      • 不過其實日光初登場那一話有清楚畫出安全褲就是了……
         
    • 首創變身中途衣服會變成白色睡衣長裙，服裝更是前所未有的露背裝，但是沒有露肚臍所以第二部無望
      • 白衣變身的設定一開始被懷疑是遭到和諧，但由於變身遭到解除時也是變成白衣而非原本服裝，看起來反而意外的糟糕……
    • 雜兵名字不再使用日本語當語源而是英文，「Desertian」之名來自英文的「沙漠」+「外星人」。
    • 以路人為雜兵的召喚媒介（也就是老前輩《美少女戦士セーラームーン》模式），也因此出現史上第一位被變成雜兵的主角及史上第一批行動連幹部都無法預料的雜兵。
    • 第一次主角變身器沒有電子遊戲功能，看來可能賣不出去
      • 其實這代有電子遊戲功能的是用來放大便心靈種子的馬桶心靈之壺（ココロポット）。
    • 除個人變身外，還製作了兩人一起變身的動畫，新鮮組淚目
      • 這時的蕾也由受變攻
      • 史上最閃變身
      • ふたりはプリキュア・Heat-Trigger
    • 但到三人世界就破局了，回覆到剪片模式，且竟然是拿蕾海兩人與日光單人剪片，令日光格格不入。
      • 推測日光的精美變身不過是日後摸魚方便，也可能是繪理香對會長宣示蕾的主權
      • 即使如此，以光點與節奏轉換的剪片水準還是遠在新鮮組之上。
      • 理所當然後期的四人變身也是2+1+1的剪片，證明蕾海兩人情宜堅不可摧
    • 歷代第一個沒有和肉棒役固定班底男配角拍拖的主角。
      • 承上，也因此是歷代第一部在聖誕節沒有演男女感情戲的光之美少女，取而代之的是光之美少女（們）代替聖誕老人實現她們的願望。
  • 部份設定則繼續沿用前作已經被破壞掉的設定。
    • 變身會改變髮色，但戰隊組模式的高彩度作畫加上配色效果可說是歷代最刺眼。
    • 大絕同樣是讓人很舒服的淨化系，囧度更是有過之而無不及。
      • 史上第一支純手動裝備，大絕放出後還要自己死命轉半天，不過主角兩人看起來轉得很樂在其中
      • 規格不合的日光和為了保留氣質的前輩月光則是採用前代水果組的轉法。
    • 劇情中仍會不時出現前作捏他。
      • 例如新鮮組的雙人拳擊、雙人踢擊和初代的蹬牆攻擊。
      • 其他還有校園名模、狗仔偷拍狂、女性足球同好會等令人產生既視感的人設，暗婊前作
    • 設定和前輩們大量重複，SS受害尤其嚴重。
      • 主題是花，SS和戰隊組的前輩用過花屬性，好死不死兩個一起中獎……
      • 大地之花的頭銜也被花天使二代借走了……
      • 強化型態月屬性被月光盜用，從金色變為銀色（？）。
  • 戰鬥中經常使用攻擊技能作為輔助，合體技更是史上最多。
    • 普通攻擊經常出現偷學前輩的雙人拳擊和雙人踢擊，只差沒喊招式名。
    • 第4集就出現合體大絕「光之美少女 繁花極強音（Precure Floral Power Fortissimo）」，其實是「石破LOVE LOVE天驚拳」式的「Hell and Heaven」，可以直接把心之花由敵人體內取出，但最後還是要來一次破壞氣勢的轉轉轉淨化。
      • 同樣是史上最閃大絕
      • 前輩月光獨自一人就能使用，不過因為蕾海兩人用起來太閃也和日光一起用過，最終決戰時則是月光與蕾、日光與海分別共同發動此招。
    • 第9集出現名為「光之美少女 雙重射擊（Precure Double Shoot）」的情人氣功彈小型合體技，其實只是兩人一起放個人技。
    • 第10集出現名為「光之美少女大爆炸（Precure 大爆炸）」的偽自爆清場技，也可以作為個人技使用。
    • 第30集中出現由「蕾天使屁拳」進化而成的雙人技「雙人屁拳（Double 屁股 Punch）」。
    • 第31集出現三人合體技。和變身一樣，三人世界再次偷懶，直接把日光的絕招與前兩人招式強制合併為「光之美少女 閃耀極強音（Precure Shining Fortissimo）」。
      • 可能因為是衝組兩人加支援一人的組合，招式名還是叫作ff而非fff（Fortissimo）。
      • 不過抹除法具外問題，水準至少還是在新鮮組上……
    • 第38集以撼心幻影鏡（Heart Catch Mirage）的力量成功變身成超級幻影（Super Silhouette）形態，並能使用四人合體技「光之美少女 撼心交響樂（Precure Heart Catch Orchestra）」。
      • 其實是巨大化 + 鐵！拳！制！裁！
      • 吐嘈點之多不輸上一代
      • 正體其實是前作的光之玫瑰園主人芙蘿拉
    • 在最後一集，因為四人同心而合體成最強的「光之美少女 - 無限幻影（Heart Catch Precure - Infinity Silhouette）」。
      • 外表跟蕾天使相似，單是體型已經是突破天際。
      • 和魔法少女まどか★マギカ的粉圓女神並稱「魔法少女的守護神」
    • 雖然主角四人無一玩音樂，但合體技卻全都是音樂術語，絕對是名不符實
  • 我方成員名字以植物為主題（其中えりか為歐石楠的學名），敵幹部名字則以沙漠的生物為主題。
    • 自介代表著變身後的屬性（陸、海、空），故又稱天裝美少女，姓氏依慣例也影射屬性。
    • 另外角色以花朵作為代表形象，並會出現在背景和大絕中；花的種類和本人的心之花無關。
      • 蕾天使→櫻花、海天使→藍色菊花（可能是雛菊）、日光天使→向日葵、月光天使→玫瑰。
      • 過場圖的四角裝飾代表著四位主角的櫻花、雛菊（？）、向日葵、玫瑰，等於一開始就把還是隱藏角色的日光月光捏完了
  • 聲優的巧合性。
    • 去年玩上癮，名偵探殺人祭典沒有停止……
      • 名偵探柯南班底主角（高山南）、毛利蘭（山崎和佳奈）、佐藤美和子（湯屋敦子）出場。
      • 偵探學園Q的主角美南惠（桑島法子）、天草流（遠近孝一）、遠矢邦子（金田朋子）與教師片桐紫乃（久川綾）參戰，更造成無數路人死傷。
    • 同時延續戰隊組以來的地獄詛咒
    • 因為東映要省錢，之後播出的《デジモンクロスウォーズ》，所使用的聲優和本動畫有著大量重疊。
      • 基本上都跟從數碼寶貝和小魔女之間的協調方法：一套當主角，另一套就當配角。例如演主角工藤大樹的高山南、演陽本明的白石涼子以及演Shoutmon的阪本千夏就在此作中分別擔任暗之美少女、園藝部部長水島綾和奶奶花咲薰子。
      • 唯一例外是桑島法子，現時兩套她都是在主角群之列（明堂院樹和天野寧寧/キュートモン）。
      • 菊池こころ不論在數碼寶貝或光之美少女的世界都當桑島身邊的淫獸……
  • 台灣東森翻譯跟前作比起來有好有壞。
    • 標題名被翻成《光之美少女甜蜜天使》，和原名的關係度頗低，不過可能是因為直翻本來就怪怪的，前作的幸福精靈這下完全變黑歷史了
    • 主角的花咲蕾被強制改名為花小蜜，明堂院樹則被改名為明堂院愛樹……[2]
      • 其他人的名字倒是翻的中規中矩，特別是一堆人以為也會被亂改的繪理香是完完全全照本翻，稱號以及招式的翻譯也多半是符合網路慣用翻譯。
      • 不過幸好TVB有把一些不適合的翻譯修正過來了。

特色編輯

  • 本代是自SS後再次以二人組方式作為初期伍成員的一代光美（如上）。
    • 但跟以前不同的是可以單人變身，沒有二人一起才能變身的限制，但有閃瞎觀眾雙眼的二人變身兼用卡（如上）。
  • 本作是歷代首次以粉紅、藍作為初始成員的光美。
  • 本作是歷代首次全員為粉紅、藍、黃、紫四人的光美。
  • 本作出現歷代第一位轉學生粉紅，也是歷代第一位眼鏡娘粉紅。
  • 本作是歷代首次出現一個以上追加成員的一部。
  • 本作首次加入了「前輩光美」的概念，並且是歷代第一次出現非正規光美的一部。
  • 本作是歷代首次出現非國中生的光美：歷代首次出現高中生正規光美成員的一部。
  • 本作亦毫不忌諱的描寫「關係親密的人或同伴被敵人攻擊而死亡」的事件，而且還出現了至少三次，都是在紫色身上
  • 本作的藍色光美是到目前為止最反傳統的，也是目前為止唯一一位學業苦手的藍色，同時也是在All Stars系列中唯一一位自初登場起每一年都有台詞的非粉紅光美。
  • 本作是自SS組後再次為所有戰士皆為普通地球人的一代，同時本作也是在有追加成員的情況下歷代首個所有戰士皆為普通地球人的一代。
  • 本作開始光美們的隊呼固定為該作完整標題，並在初期成員到齊之後推出。
    • 目前唯一的例外是擁抱組，一直到追加成員入隊之後才推出隊呼。
  • 本作是歷代第一部異世界（妖精居住的地方）就設定在地球（普通人類居住的地方）的一部。
  • 本作的主要角色於後輩熱帶組TV版33話客串，而後者是屬於宣傳秋季電影版的亂入客串性質。
    • 台灣播出這集時也因為該作中文配音陣容的緣故，得以讓其中的三位聲優回歸演出各自的角色，而飾演的花咲天使的楊凱凱小姐也於該話客串。

心之大樹（こころの大樹）編輯

世界樹的一部份集結人們心靈根源的屍魂界世界，根本就是光之玫瑰園，也不排除是芙蘿拉歸位前的暫行機關，此外還是個移動要塞。其實是保護地球的自衛機制。

  • 心之大樹【CV：鶴弘美[4]】
    • 心之大樹的本體，亦是光美神殿的管理者。
    • 由除魔師轉生而成
 
  • 希芙莉（シフレ）【CV：川田妙子】
    • 心之大樹的精靈，蕾天使的精靈夥伴。
    • 襲胸淫獸1號，受過專業的玩偶偽裝訓練
    • 真身是泉之鄉的精靈女王，前途不可限量[5]
    • 爹蘇。
    • O口O
    • 給予蕾天使變身用的光之種子（プリキュアの種），並於蕾天使打敗敵人後大出得到一顆「星宿[6]心靈種子（心の種）」。

道具編輯

心之種子（こころの種）編輯

  • 光之美少女的收集物，形狀跟一硬幣相若，存放於心靈之壺中
  • 每打贏一隻沙漠異形，妖精便會產生出一顆
    • 一開始生產心之種子的動畫是妖精先抽蓄一陣搖晃屁股之後再產出不知名的金黃色的液體球，之後金黃色的液體球會化成心之種子
    • 家長和大友們：「這是在大便沒錯吧?」
    • 因為以上的畫面有家長感到非常下流跑去向BPO（日本播放倫理·節目改進機構）抗議，之後的畫面就被修改成心之種子直接伴隨著光飛出來[28]但大友對於「心之種子＝大便」的印象還是揮之不去
    • 在此之後的光之美少女系列作中，就會盡量避免所謂下流梗的描寫
  • 不過有時候會作畫失誤把心靈之壺裡面的心之種子畫成光之種子

光之種子（プリキュアの種）編輯

  • 光之美少女的變身道具，與心之種子外型相同，但中間會有象徵光之美少女的心型標誌，顏色為各自的代表色
    • 比較特別的地方在於此物平常由光美的搭檔妖精保管，要變身的時候才會拿出來
    • 即使破碎了也能讓持有者在普通人狀態下使用少量能力，不過會特別消耗體力
  • 波普莉另外持有一枚橙色的光之種子，並非變身用，而是打開光美宮殿大門的鑰匙
    • 在光美們通過考驗之後，該種子變化為一枚金黃色的特別種子
    • 金色種子繼承橙色種子的功能，並在光美們完成最後考驗後可以用於變身為「超級幻影」型態

心之香水（ココロパフューム）編輯

  • 初期二人的共用變身器，以前的月光天使與花朵天使也是使用此型號的變身器
    • 在變身後亦可與其他心之種子配合使用以得到某些buff

閃耀香水（シャイニーパフューム）編輯

  • 樹專用變身器，性能上大致跟心之香水一様，外型也只有一點點的細節差異

心靈之壺（ココロポット）編輯

  • 用以存放心之種子，尿壺一個
    • 當收集夠多心之種子之後，充滿力量的心之大樹會隨之產生奇蹟
      • 第一次發光為波普莉誕生，第二次發光為可隆現身引導百合
    • 在百合重新獲得變身力量後蓋子部分馬桶蓋成為其變身器，保管者由可貝轉為百合[29]

撼心幻影鏡（ハートキャッチミラージュ）編輯

  • 存放於光之美少女宮殿的強化道具，能夠強化光之美少女的能力，激發潛能。想要得到此物，就必須來到宮殿接受考驗──擊敗上一代的光之美少女，並於通過考驗後由心之大樹親自賜與。
    • 當年的花朵天使就是靠著此物才能與杜恩勉強打成平手也就是說其實不一定要湊齊4個人也能用嘛
    • 月光天使也曾經想得到此物，卻在當天接受考驗之前就敗給暗之美少女
    • 蕾、繪理香與樹在接受考驗時，因為作為前輩的薰子和百合都失去了變身能力，因此由可貝代勞
  • 除了強化光之美少女們的能力外，還擁有將光美們傳送到心之大樹身邊、照出他人的心之花、隨時觀察心之大樹等功能
    • 光美四人在通過第二次考驗後，可使用此物變身為「超級幻影（スーパーシルエット）」型態
      • 四人可以以此形態發動四人大絕「光之美少女撼心交響曲（プリキュア・ハートキャッチ・オーケストラ）」
      • 於最終戰中四人+三妖精更能合體成為「無限幻影（無限（むげん）シルエット）」（漫畫版則是花蕾天使獨自變身而成）
