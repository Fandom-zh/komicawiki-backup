2nya (ふたにゃー, 双喵)


目錄

解說編輯

簡介編輯

  • 2nya 是一個仿雙葉風格的中文語系圖片討論板，主要的管理人是 小企鵝。成立時間在 2005-06 左右，維持至 2006-06 左右。
  • 主要的管理團隊為 風魂 、 風誠皓殿 、 永恆 、 Piex 和 真理。
  • 由於同性質 2cat / Komica 的出現，和站方無暇管理的緣故，無人管理狀態之後隨著的是關站。
  • 與傳統的匿名貼圖站不同，2nya 加入了會員系統，加入會員必須考試以取得會員申請資格，會員可以使用更多服務，例如點選詳細希望連結、進入會員專板等。
  • 2006-06 左右完全關閉服務，雖未正式宣告但已代表著2nya的死亡。目前DNS對應為127.0.0.1，也就是瀏覽者自己的電腦。

名稱由來編輯

  • 2nya 双喵的名稱應該是起源自遊戲會社 CIRCUS 在 2005 年愚人節的企劃 最終試験くじにゃ！ 的名字和人物。當時官網有兩張限定期間桌布可以下載，因為圖內就是兩隻貓娘，加上標題的にゃ，因此可能為 2nya ふたにゃー双喵的名稱起源。
  • 名稱與 2cat 雙貓聯合站有所衝突，但基本上名稱起源不一樣，且用字也不同。一個是 双喵2nya 另一個是 雙貓2cat，在當時使用者常常搞混。

歷史編輯

  • 双葉ちゃん（ふたば☆ちゃんねる）是一個日本有名的圖片討論板網站，主要的程式 futaba.php 是使用レッツPHP!的圖片討論板程式 gazou.php 所改造，網站上有配布程式的原始碼。這個原始碼後來被修改成多種不同語言和功能的版本，並且被其他不同語言的使用者拿來架設類似双葉風格的網站，其中較知名的包括如英文的 4chan[1] 和中文的 2cha[2] 等。
  • 2nya 最早是由 風魂 和 小企鵝 等人因為 V-Day 後 2cha 的消失想再架設一個 2cha ，於是在小企鵝的自架主機上架設，名稱還是取為 2cha。對外開放後怕到時候與復站的 2cha 搞混，於是更名為 2nya。
  • 一開始使用的是 scribe 修改自 双葉ちゃん 的 futaba.php[3]，爾後由 小企鵝 寫了一套完整的版面及後端系統，採用了 MySQL 資料庫作媒介。
  • 因為管理團隊大多是來自於 遊戲頻道 的使用者，故基本上也會有一些相吸作用，吸引了不少來自遊戲頻道的網友前來。
  • 由於同性質 2cat / Komica 的出現，和站方無暇管理的緣故，無人管理狀態之後隨著的是關站。

發展編輯

  • 2006-05 2nya 內部測試，當時的名稱是 2cha 而網址為 2cha.ngct.net。
  • 2006-06 2nya 使用的貼圖版系統 NYABB 開發完成，改名為 2nya 並對外公開版面。
  • 2005-09 由於站方伺服器軟體的更新而造成部分版面連接大圖時出現 Forbidden 或者是小圖無法顯示，經過修改後回復正常。
  • 2005-10 站方決定終止開發其使用的系統 NYABB，因為眾多問題導致開發停止，原計畫的第二代双喵也胎死腹中。
  • 2005-11 2nya 開始無法進入
  • 2006-03 可能是因為 kirur 的請求，2nya 再度開放。但關閉時間太久造成人流喪失，加上 2cat / Komica / 2cha 的強盛，目前為沉寂狀態。
  • 2006-06 無法連線進入，網址對應為 127.0.0.1 ，關站確認。
