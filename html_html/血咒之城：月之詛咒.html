血咒之城：月之詛咒

Bloodstained: Curse of the Moon / 血咒之城：月之詛咒[1] / 血污（赤痕）：月之詛咒[2]
本條目內容不足需要擴充，歡迎有興趣的朋友加以補完。
本條目內含劇透，沒玩過遊戲又不想被斬月砍死的話請不要反白。


目錄

簡介編輯

  • 本作(月之詛咒無印)為《血咒之城：暗夜儀式》的外傳，為於2018年推出的2D動作遊戲，畫面走FC時期的8bit風格
    • 主角為斬月，米莉安、阿爾弗雷德與吉貝爾則作為同伴人物出現
    • 整體風格類似《惡魔城傳說》，除關卡有大量分支路線之外，還能藉由切換同伴以通過部分機關地形
    • 明明是外傳，推出時間卻比本傳早上一年
    • 據玩家普遍評價，本作的BOSS各方面演出明顯比本傳好
  • 由於銷量優良和風評良好，在2020年暑假檔期販售月之詛咒2代，一樣是維持8bit風格
    • 主角還是斬月，但是同伴變成了教會人員多明妮克、退役軍人羅伯特與魔導機甲小八（機甲裡面是隻柯基犬）
    • 可以選擇原本一代的陣容，但是須完成特定進度才能解鎖此功能
    • 其中身為本篇黑幕的多明妮克參戰在玩過暗夜儀式的玩家間引發了話題
      • 對此製作組作出的回應是該作品是和本篇暗夜儀式相同世界觀但走向不同的平行世界
    • Boss有做更動，部分來自暗夜儀式，也新增其他Boss成為玩家的挑戰；之後新增的Boss是否回流到暗夜儀式則有待商確
    • 在各關卡間追加過場事件，可以看到主角一行人在各關卡中間的有趣互動
  • 二代依照通關進度，可以分為四個主要章節與一個額外模式：
    • EPISODE 1 -The Demon`s Crown-
      • 最初可選的章節，可使用斬月、多明妮克、羅伯特與小八
    • EPISODE 2 -Unbreakable Bonds-
      • EPISODE 1過關後可選擇，可操作人物為除多明妮克以外的三人
      • 按照關卡中途有否獲得斬魔刀會影響結局是否將多明妮克救出
    • EPISODE EX -Oath of Reunion-
      • EPISODE 2未獲得斬魔刀過關後可選擇
      • 接續EPISODE 2的壞結局，因為羅伯特與小八負傷退場的緣故，斬月、米莉安、阿爾弗雷德與吉貝爾四人再度集結重啟多明妮克救出計畫；此章一開始斬月就會得到斬魔刀
    • FINAL EPISODE -月を斬る者-
      • EPISODE 2獲得斬魔刀下過關後可選擇
      • 接續EPISODE 2的好結局，被救出的多明妮克察覺到惡魔的根據地在月球上，七人為了登上月球而著手開發登月魔導機甲
      • 本處可以自由選擇關卡收集零件及讓同伴加入，收集完後可以搭乘機甲登上月球[3]突入最終關卡
      • 如果不找任何同伴、僅收集零件就突入最後關卡的話，斬月會拿到由鍊金術鍛造的大斬月刀，但是就只能用斬月破關
    • SINGLE MODE
      • 在FINAL EPISODE不找任何同伴破關後可選擇，為操控任一可用角色破關的特殊模式

系統編輯

  • 一代同伴出現後可以決定要收留、殺掉或無視，這些選項會左右最後的結局
    • 全收留結局：打倒格莫瑞後所有人都乏力，而黑暗力量要襲擊眾人的當下，斬月自己一人擋下攻擊而被黑暗力量詛咒，三人決定要救他而開啟Nightmare模式
      • Nightmare模式特殊結局：魔化的斬月被其他三位夥伴解救其靈魂後，於疑似未來的另一個世界重生（場面惡搞洛克人2的片頭）
    • 全殺光結局：打倒格莫瑞後斬月吸收了黑暗力量而魔化
    • 全無視結局：打倒格莫瑞後斬月乏力，黑暗力量要襲擊他，最終其他三人犧牲性命把黑暗力量給擊破
    • 真正好一點的結局反而是既不能全殺、也不能全收和全無視
  • 二代的同伴強制全部收留，製作組笑稱是斬月也成長了，都幾歲的大叔了還要到二代才不會亂殺人…
    • EPISODE 1結局：打倒巴力西卜後所有人都乏力，而黑暗力量要襲擊眾人的當下，多明妮克自己一人擋下攻擊而被黑暗力量詛咒，三人決定要救她而開啟EPISODE 2劇情
    • EPISODE 2壞結局：打倒梅菲斯特後斬月因迷惘而無法將其完全斬斷，斬月以外的同伴被黑暗力量給打重傷必須休養，這時前代同伴三人組就會加入並開啟EPISODE EX -Oath of Reunion-劇情
    • EPISODE 2好結局+EPISODE EX：打倒梅菲斯特後斬月成功救走被黑暗力量吸收的多明妮克，隨後從對話中得知詛咒之月那邊仍有魔物大軍，這時前代同伴三人組就會加入一行人
    • FINAL EPISODE結局：製作完可以上太空的魔導機甲後眾人飛到月球，並在此地終結掉墮天使沙利葉
    • FINAL EPISODE Single結局：製作完可以上太空的魔導機甲後眾人飛到月球，斬月連同月球本體一併把墮天使沙利葉給砍成兩半（如果是兩人遊玩的話月球會被砍成四塊）
  • 幾乎所有的Boss會有死亡前的最終攻擊[4]
    • 二代第一章的前三關會由該關登場同伴處理掉，但仍然有此攻擊存在
    • 一般模式下即使被打中也能以剩一格血存活，但在Boss Rush模式下被打死就死了、而且會判定失敗要重打

主要角色編輯

  • 斬月（Zangetsu）
    • 本作主角，來自東洋的退魔士
    • 武器為斬月刀，一代藉由血祭其他同伴可以獲得強化
    • 二代基本動作不變，但強化後攻擊花招變的更多
      • 在性格上也有所成長，然而卻從此脫離常識人範疇，像是沉迷吸狗，把狗取名的像忍犬，不在意有女性在場就把衣服脫掉之類的
    • 二代第一章第四關後可以拿到噬魂者之刀，攻擊招式大強化。但是此武器無法徹底斬殺第二章的最終Boss
    • 第二章必須要在前三關找齊關鍵刀劍道具，才能在第四關某處拿到能斬殺最終Boss的斬魔刀
      • 須注意在第二章一拿到斬魔刀能力招式就會回到初期狀態
    • 終章在不帶任何同伴的情況下收集登月魔導機甲的零件登月的話，可以獲得大斬月刀；此時最終關卡的標題，最終Boss的攻擊方式與結局演出都會改變
    • 滿足特定條件破關後可以選擇究極模式的斬月，除了習得所有招式外另外還追加一招儲氣斬，跑RTA必備要角
  • 米莉安（ミリアム / Miriam）
    • 本篇《暗夜儀式》的主角，被植入惡魔結晶的少女，一代於STAGE 1破關時出現
      • 斬月殺死她後會得到跳斬，攻擊判定增加甚至有機會一刀有兩下攻擊判定
    • 以聖鞭為武器，擁有滑行與跳高能力（與多明尼克都是三格高度，其他人只有兩格），還可以在蹲下時進行移動
      • 還會運用各種實體副武器協助戰鬥
    • 性格與原作一樣天然呆，根據二代最終章的陣地對話，甚至相信月球上有月兔存在
      • 還真的被她應驗，但是月兔在此為沙利葉的守護者而且明顯比人還要大
  • 阿爾弗雷德（アルフレッド / Alfred）
    • 鍊金術師，一代於STAGE 2破關時出現
      • 斬月殺死他後會取得二段跳能力
    • 主打多花樣的鍊金術從效果上看已經是魔法了，雖然火力強大但血量非常低
      • 二代此技能有作調整，主要是將一代被評為不好用的分身技調整；普攻可以回復魔力值
    • 二代最終章負責主持登月魔導機甲的開發，卻也因此成為唯一一個完全沒有日常對話的同伴
  • 吉貝爾（ジーベル / Gebel）
    • 被植入惡魔結晶的青年，一代於STAGE 3破關時出現
      • 斬月殺死他後會取得衝刺能力，唯一可以用跑的角色
    • 一般攻擊為向斜上方發射的蝙蝠（無法打到貼地或離地面僅一格高度的怪物），自己也可以變成蝙蝠短暫飛行
      • 二代在變成蝙蝠攻擊怪物時會回復HP
      • 二代最終章的陣地對話甚至會羨慕可以攻擊貼地怪物的小八，公式が病気（蓋章）
  • 多明妮克（ドミニク / Dominique）
    • 教會所屬的驅魔人，二代於STAGE 1破關時出現
      • 因為在本篇是幕後黑手的關係，在本作出場時也一度被認為會在背後搞鬼⋯結果這回除了白到發光之外還幫主角群揭穿黑幕底細
    • 手持長槍且與米莉安一樣的高跳躍能力和多樣式攻擊，而且擁有少數直接回血甚至復活我方的手段，但是血量比起米莉安還要少一些
    • 第一章結局為了完成父母討伐梅菲斯特的遺願而做好自願被附身並與其同歸於盡的覺悟，在後續章節被獲得斬魔刀的斬月救出
  • 羅伯特（ロバート / Robert）
    • 退役軍人，二代於STAGE 2破關時出現
    • 手持火槍所以是遠距攻擊，只有他可以攀爬在牆上與匍匐在地，當然血量為所有角色中最低（官方曰抗魔能力太低）
    • 因為家人死於惡魔的緣故非常憎恨惡魔與鍊金術，更因為教會對自己家人見死不救而對教會沒有好感
      • 原本對多明妮克與小八完全沒有好臉色，隨著一路上的旅程而逐漸改觀
      • 第二章甚至因為多明妮克的犧牲而暗自垂淚然後被小八安慰，取代一代斬月成為本作傲嬌擔當（蓋章）
  • 小八（ハチ / Hachi）
    • 操控魔導機甲的柯基犬，二代於STAGE 3破關時出現
    • 可以漂浮且可以開啟無敵狀態，本身具備破壞尖刺地形功能，同時血量極高，是個很強力的血坦
    • 根據最終章的陣地對話，原本差點被斬月取名為「斬月丸」，被羅伯特駁回後才取名為小八

Boss編輯

一代編輯

  • 暴食火車（グラットントレイン / Glutton Train）
    • STAGE 1「月光に誘われて」的Boss
    • 與火車頭融為一體的惡魔
  • 佛卡洛（フォカロル / Focalor）
    • STAGE 2「冷獄に潜むもの」的Boss
    • 第41柱魔神，擅長控制水的惡魔
    • 第一個沒有在暗夜儀式本篇出現的Boss
  • 華利弗（ヴァルファール / Valefar）
    • STAGE 3「絢爛たる虚無」的Boss
    • 第6柱魔神，從硬幣中汲取能量的惡魔
  • 華列克（ウァラク / Valac）
    • STAGE 4「天を穿つ冒涜の証」的Boss
    • 第62柱魔神，雙頭魔龍
    • 最終攻擊由第二型態取代，此時身軀部分會上升將低處的地面佔據使立足點減少
  • 安德雷斐斯（アンドレアルフス / Andrealphus）
    • STAGE 5「暗夜裂く銀影」的Boss
    • 第65柱魔神，擁有漂亮羽毛的孔雀惡魔
    • 場景沿用了《血之輪迴》的死神地形，巨大桅杆兩邊都是無底洞，本身也會使用吹飛攻擊將人吹進洞中
  • 無血（ブラッドレス / Bloodless）
    • STAGE 6「殺戮劇に幕引きを」的Boss
    • 操縱血液的吸血鬼
  • 巴欽（バティン / Bathin）
    • STAGE 7「禁忌嘲う悪魔の手」的Boss
    • 第18柱魔神，擁有光速移動能力的惡魔
    • 最終攻擊會極限加速至超越時間產生分身
  • 格莫瑞（グレモリー / Gremory）
    • STAGE 8「惨月 / 斬月」的Boss
    • 第56柱魔神，擁有預知能力的惡魔，斬月的死對頭
    • 二代STAGE 7「静寂と鳴音の狭間 / 心一つに、斬れ / 何度立ち塞がろうとも」亦有登場
    • 二代攻擊方式與一代第一型態變化不大，且因為不是最終BOSS這次只有一個型態，且會施展最終攻擊
  • 魔化斬月
    • Nightmare模式STAGE 8「逆月の鎮魂歌」的BOSS

二代編輯

  • 龍共生體（ドラゴシンビオート / Drago-symbiote）
    • STAGE 1「竜影、月に踊る / 絆を胸に / 最後の希望」的Boss
    • 與擁有降雷能力的惡魔形成共生體的巨龍
    • 少數最終攻擊可中斷的BOSS，斬月可使用蓄力斬時在衝過來時使用，會被切成兩半，不過前面的閃電還是要躲就是了
  • 薩流比絲（サリュビス / Lubius）
    • STAGE 2「欺き夢繰る妖女 / 幻夢斬り断つ紅蓮剣 / 蒼き薔薇の姫闘士」的Boss
    • 本體是蘑菇型態的魔物，因主角一行人在開打前吸入過多胞子（粉紅色粉末）使得斬月見到其變成遊女的外型，所以說斬月你喜歡這種的啊？
    • 不同人所見到的形象都不一樣，例如羅伯特所見到的是死去的家人
    • EX章在破完該關後由於胞子的作用還沒有退去，導致主角一行人被彼此看作成各種魔物
  • 角鬥士戰車（グラディアトルドーザー / Gladiator Dozer）
    • STAGE 3「飛瀑裂く凶戦車（チャリオット） / 轟鎧撃ち抜く銀弾主 / 錬金術と共にあれ」的Boss
    • 被角鬥士的亡魂支配的水生魔物
    • 少數最終攻擊可中斷的BOSS，斬月可使用蓄力斬時在撞過來時使用，會以切成兩半的狀態掉進河裡
  • 威沛（ウェパル / Vepar）
    • STAGE 4「女王の剣の葬送曲（フューネラルマーチ） / 海割る鋼の破壊王 / 悲しき運命（さだめ）の男」的Boss
    • 第42柱魔神，支配水的惡魔
    • 在本篇是第一個打到的Boss，這裡則較晚登場
    • 最終攻擊由第二型態取代，斬殺上面的頭部後還有下半部的頭要打
  • 熔岩曼陀（ラバマンドラ / Lavamandra）
    • STAGE 5「天地貫く緋骨の鎖 / 火よりも熱き血潮で / 止められる者は、ない」的Boss
    • 由火山亡魂聚集而成的巨蛇
  • 泰坦卡門（ティタンカーメン / Titankhamun）
    • STAGE 6「巨なる棺は誰が為に / 遺物に構う暇はない / 棺を踏み割りその先へ」的Boss
    • 宿有古代國王靈魂的巨大棺木，名稱故意致敬現實擁有詛咒傳說的圖坦卡門
    • 最終攻擊會讓滿滿的蟲子從棺木爆出來，密集恐懼症患者慎入
  • 巴力西卜（ベルゼブブ / Beelzebub）
    • 第一章STAGE 8「悪魔の王冠」的Boss
    • 即為蒼蠅王，有兩個型態，被打倒後會巨大化
    • 戰敗後被寄宿其身上的梅菲斯特背刺而死
  • 梅菲斯特（メフィスト / Mephisto）
    • 第二章與EX章STAGE 8「熱き絆は砕けない！ / 再会の誓い」的Boss
    • 須藉由附身其他生物才能實體化的惡魔，造型基本上就是本篇《暗夜儀式》存檔點的惡魔像
    • 第一章結局出現，殺死原本的宿主巴力西卜後轉而附身多明妮克
    • 會與附身對象形成共生關係，必須獲得斬魔刀才能在給予致命一擊的同時救出多明妮克
  • 墮天使沙利葉（堕天使サリエル / Sariel）
    • 最終章FINAL STAGE「月の涙 / 大・斬・月」的Boss
    • 以月球為根據地的墮天使，一切事端的元兇
