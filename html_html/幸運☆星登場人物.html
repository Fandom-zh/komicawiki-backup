    • 身體虛弱，時常病倒，但自從入學陵櫻學園以後情況有好轉，似乎是因為認識了岩崎南(岩崎みなみ)這個好友的緣故。
  • 岩崎南(岩崎みなみ)
    • CV：松來未祐 / 茅原實里

    • 表情沒有太大的變化，給人很酷的感覺，但其實內在相當溫柔。
    • 與高良美幸(高良みゆき)住的很近，由於一樣喜愛閱讀，所以年幼時就認識美幸。
    • 據美幸從電話中的說詞，小南家境富裕，庭院中設有大陽傘，且養著一隻大白狗。
    • 相當在意自己的胸圍，當提到與胸部發育有關的事情時，就會開始撫摸起自己的胸部。
    • 曾以冷酷的態度驅趕在小優身邊開惡質玩笑的男生們。
    • 由於其聲優的關係，似乎已經多出「長門」的綽號。

    • 在她畫漫畫的妄想世界裡小優與小南似乎是一對的，小優還常常摸小南的胸部。
    • 被岩崎南家的狗所討厭

  • 帕特莉西亞‧馬汀(パトリシア‧マーティン/Patricia Martin)

    • 後來轉學到小優的班上成為同學。
  • 若瀨和泉(若瀬いずみ)
    • 出身地：埼玉県;生日(星座)：1月3日(摩羯座);血型：B型;身高：164cm

    • 漫畫版第7卷成為小優班上的班長，性格嚴謹及成績優秀，相當受同學青睞。[5][6][7]

      • 和泉：這種地方要不是近距離停下來看到對方根本不會注意。結果下一秒...

    • 漫畫版第8卷，與田村`帕蒂結為同好及好友，約定好不要曝露她是宅女的秘密。[8]

家人們編輯

  • 成實唯(成実ゆい)
    • CV：齋藤千和 / 西原さおり
    • 此方的表姐，是一名交通安全科的警察。
    • 人妻, 原名小早川唯
    • 雖然平常看似人畜無害，但只要開車一被超過，就會立刻變身成暴走族，甚至可以在時速40公里下甩尾和使出水溝蓋跑法超車。
    • 小時候因為看見賽跑比賽時用的鳴槍器，以為是真槍，因此有了想當警察的念頭(雖然此方認為她在唬爛就是了)
    • 經常的喝醉酒跑來泉家，也常向此方抱怨老公不在身邊。
    • 做事隨便的讓此方印象深刻。
  • 泉總次郎(泉そうじろう)
    • CV：平松広和
    • ♂
    • 本質是一個標準的御宅族，也是泉此方成長成為那樣的女孩的最大主因。
    • 名句為「我真可說是人生的贏家啊！」。對於能娶得一位蘿莉般的青梅竹馬(泉彼方)，有一個到了十八歲仍然長得像蘿莉、又了解父親喜好的女兒(泉此方)，再追加同樣體格的姪女搬了過來住(小早川優)，不少島民都深深認同這句話。「人生的贏家」也成為了島民間的成句。
    • 動畫第8集首次登場，但不僅被女兒此方吐嘈[11]，參加陵櫻高中的運動會還被工作人員當成癡漢強行拖走。
  • 泉彼方(泉かなた)
    • CV：島本須美
    • 眾角色中唯一已去世的人（沒誤）
    • 此方的母親，和此方一樣都很矮小，但沒有此方的呆毛和愛哭痣。(在原作最早期時有呆毛)
    • 曾經對丈夫總次郎希望此方「身高不要像我，個性不要像你」，不過悲慘的是最後此方把這兩點都繼承起來了。

    • 宮河姐妹的姊姊，一名御宅族，靠著在同人店打工維生。
    • 由於喜愛購買同人誌，因此生活過的不算太好。
    • 最初在遊戲中登場，爾後於動畫12集中帶著妹妹日影一起出現。[13]
    • 之後於16話正式登場。
    • 外傳漫畫與動畫《宮河家の空腹》的擔當雙主角之一。
  • 宮河日影(宮河ひかげ)
    • CV：田村由香里(遊戲版) / 鹿野優以(本傳動畫) / 川崎琴(外傳動畫)
    • 宮河姐妹的妹妹，就讀櫻園小學四年級，是所有角色中年紀最小的。
    • 由於姊姊的嗜好，因此不是很喜歡御宅族和相關事物。
    • 和姊姊日向一樣最初在遊戲中登場，後來在動畫12集中被日向一起拉來參加COMIKE活動，也一起和姊姊日向在16話正式登場。
    • 外傳漫畫與動畫《宮河家の空腹》的擔當雙主角之一。


突然出現、一閃即過的特別角色編輯

  • 山本寬
    • 1~4集導演，於動畫第三集登場，擔任的是因為此方的暴露舉動而不知該把視線往哪擺的路人甲。[14]
  • 兄澤命斗(兄沢命斗)
    • 動畫第10集、12和13集中出場，由島本和彥老師設計的ANIMATE代表人物，出場時候的畫面十分熱血，甚至還會魯路修的GEASS。讓人快樂的是，他的聲優關智一是配過相良宗介的「男高中生代表」。
    • 中文版由於店長聲優劉傑有配過魯路修，使用GEASS的片段領班特別指定用魯路修的聲線配
  • 星河ラミカ
    • 跟著兄澤一起於第10集登場的女店員，不過也只出現這麼一次[15]。
  • 茅原實里(茅原みのり)
    • 動畫第12集中出場的長門有希聲優，在京阿尼的攤子擔任銷貨人員，不過只有一句台詞。
  • 杉田店員、COSPLAY喫茶店顧客
    • 動畫第12集中登場的角色，相貌是取自同姓的聲優杉田智和[16]，聲音也是由這位聲優配音，而COSPLAY喫茶店顧客則是在16集時出場。
  • 立木文彥&くじら
    • 皆為聲優，在動畫中分飾多名路人角色(立木配男生，くじら配女生)。
  • KERORO小隊
    • 於動畫13話登場，是小隊中主要的三個角色KERORO、GIRORO、TAMAMA偽裝成人類模樣和兄澤命斗聯合擔任ANIMATE店員，同時三人也擔任了13話的預告。
      • 中文版配音找來KERORO的原班人馬配
  • 一對無名情侶
    • 於第13話片尾中出現，看著白石耍白的可憐情侶......[17]
  • 平野綾(平野あや)
    • 於15話中「涼宮春日的激奏」活動中登場。
  • 美水鏡(美水かがみ)
    • 作品原作者，於每集換場畫面時以自己的自畫像（一個長有貓尾巴的頭）方式登場，另外在15集的D.C.惡搞畫面中也有客串。雖然以名字和自畫像當基準，乍看之下可能會以為他是女生，不過實際上美水老師是男生。
  • 長門店員
    • 16話中Cosplay喫茶店的店員之一。雖然說是Cosplay，但是除了造型已經為了配合動畫風格而稍作改變以外，怎麼看都是長門有希本人。
  • 涼宮春日(涼宮ハルヒ)
    • 在第20集內賣飲料廣告，因為突然增加的乳量讓一干島民噴出鼻血，實況串內瞬間哀鴻遍野......
  • 小野大輔(小野だいすけ)
    • 於20話「LUCKYCHANNEL」登場擔任小神晶臨時助手，身為《涼宮春日的憂鬱》中古泉一樹和《AIR》國崎往人的配音員，登場的時候也和小神大玩聲優捏它。
  • 負責大人(ゴットゥーザ様)
    • 說是ゴットゥーザ様，其實就是後藤邑子。於23話已荒廢的「LUCKYCHANNEL」中登場，以大姐頭的口氣對白石稔與小神晶說教，最後騎著愛車藍色的CB400SF拉風的退場。
    • 現實中「ゴットゥーザ様」是杉田智和替後藤邑子取的綽號。
