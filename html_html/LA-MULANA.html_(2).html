LA-MULANA

LA-MULANA（ラ・ムラーナ）
本條目內容不足需要擴充，歡迎有興趣的朋友加以補完。
本條目可能含有部分劇情內容，沒玩過遊戲又不想被遺跡內的陷阱整死或被Mulbruk甩巴掌的話請不要繼續看下去。

目錄

概要編輯

  • 由GR3 PROJECT（NIGORO的前身）開發的動作冒險遊戲。
    • 概念上屬於MSX《魔城傳說II》的精神延續。
  • 初版以MSX的風格呈現，畫面也是MSX模式。
  • 在遊戲性上也充滿了惡質的陷阱與高難度。
  • 後來在Wii、Steam與PSV推出重製版，畫面與迷宮內容都大幅更新之外還實裝了成就系統。
  • 2018年於Steam推出續作《LA-MULANA 2》，預計在2019年移植到PS4、Xbox One、Nintendo Switch等家用主機上。

登場人物編輯

  • ルエミーザ・小杉 / Lemeza Kosugi
    • 一代主角，日裔第三代的考古學家。
  • ルミッサ・小杉 / Lumisa Kosugi
    • 二代主角，Lemeza的女兒。
  • ショーン・小杉 / Shawn Kosugi
    • Lemeza的父親，歷史學權威。
    • 為了追尋人類的起源而前往ラ・ムラーナ遺跡卻就此失去聯繫。
  • ゼレプド長老 / Xelpud
    • 守護ラ・ムラーナ遺跡的長老，Shawn的舊識。
    • 會給予主角傳郵件的程式並在主角冒險途中不斷發郵件各種意義上都很嘈吵。
    • 二代將ラ・ムラーナ遺跡改造成觀光勝地，依然會給主角吵死人的郵件。
  • アルセダーナ / Alsedana
    • 沉睡在ラ・ムラーナ遺跡的賢人，主司死亡。
    • 二代中盤因為衰老而死亡。
  • ギルトリーヨ / Giltoriyo
    • 沉睡在ラ・ムラーナ遺跡的賢人，主司知識。
    • 二代為了尋找丟失的遺物而四處奔波。
  • フォボウス / Fobos
    • 沉睡在ラ・ムラーナ遺跡的賢人，主司生命。
    • 二代為了調查イグ・ラーナ遺跡而留在迷いの門。
  • サマランタ / Samaranta
    • 沉睡在ラ・ムラーナ遺跡的賢人，主司力量。
    • 二代在ラ・ムラーナ遺跡負責復原遺跡。
  • フレイヤ / Freja
    • 妖精種族的女王，為了封印遺跡而向賢者求助。
    • 二代在遺跡毀滅之際打算與其共存亡，最後被妖精們救出。
  • 回復妖精エイル
    • 支援主角的妖精，可以在一定時間幫主角回復生命
  • 武器妖精カーラ
    • 支援主角的妖精，會對周圍的敵人自動攻擊
  • お宝妖精エルルーン
    • 支援主角的妖精，一定時間內擊敗敵人獲得的道具量上升
  • 鍵妖精ヘリヤ
    • 支援主角的妖精，可以開啟特定區域的封鎖
  • ムーブルク / Mulbruk
    • 移植版登場，沉睡在ラ・ムラーナ遺跡的少女。
    • 二代參與イグ・ラーナ遺跡的調查，性格似乎更加殘念，還會因為主角身材比較好而心生嫉妒。
    • 在同作者的有病作品《薔薇與椿與LA-MULANA》擔任主角。

武器道具編輯

主武器編輯

  • 鞭
    • 初期武器，可以隨故事進度強化。
  • 匕首
    • 射程短，不過可以攻擊較矮小的單位。
  • 鏈劍
    • 以突刺攻擊為主。
    • 在部分關卡同時作為鑰匙使用。
  • 斧
    • 攻擊範圍廣，攻速慢。
  • 刀
    • 攻擊力高，連續攻擊性能較弱。

副武器編輯

  • 手裏劍
    • 直線飛行的遠程武器，威力弱但可以連續發射。
  • 車手裏劍
    • 貼地行進的貫通武器，可以觸發部分機關。
  • 大地之槍
    • 朝正下方發射的長槍。
  • 擲彈筒
    • 朝正上方發射的砲彈。
  • 炸彈
    • 造成廣範圍暴風，暴風對主角也有傷害判定。
  • 環刃
    • 可以回收的迴旋鏢。
  • 撒菱
    • 設置在後方的忍具，對主角亦有傷害判定。
  • 手槍
    • 威力強大的射擊武器，子彈只能在商店購買且較為昂貴。
  • 盾
    • 防禦子彈的盾牌。
  • Ankh Jewel
    • 喚醒各關守護者的寶玉。

道具編輯

  • 重物
    • 啟動機關的道具，可在打倒敵人時獲得或是商店購買。
  • 生命寶玉
    • 增加生命值上限。
  • 掃描器
    • 裝備時允許調查石碑，搭配特定軟體可以翻譯內容。
    • 也可以調查遺跡內出現的屍體。
  • 時之燈
    • 可以停止時間。
  • MULANA護符/寶石
    • 在詛咒寶箱前使用時可以破除詛咒。

種族編輯

  • 第一子
    • 母親最早創造的種族，以人頭蛇身為特徵。
    • 型態與母親最為類似，因為企圖追求母親的力量而滅亡。
  • 第二子
    • 體型巨大的巨人族。
    • 對於是否送母親上天產生分歧，最後在內戰中滅亡。
  • 第三子
    • 有翼人族，內部又可細分出天使與惡魔等種族。
  • 第四子
    • 魚人族，和其餘數子相比較缺乏智慧。
    • 一代只有一位魚商人出現，二代完全沒有出場。
  • 第五子
    • 擁有相當的智慧的矮人族。
    • 因為企圖封印母親而被毀滅。
  • 第六子
    • 由世界樹產生的泥人，從第六子以後的人類開始肉體會衰老。
  • 第七子
    • 為封印母親而誕生的人族，但仍因存有母親的基因而無法進行封印。
    • 初始村內的NPC與四賢者皆為第七子。
  • 第八子
    • 現生人類，因為沒有母親基因而能夠進行封印計畫。
