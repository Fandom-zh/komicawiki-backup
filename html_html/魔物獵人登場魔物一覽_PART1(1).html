魚類編輯

  • MH3新增物種，但沒有收錄在筆記的小型魔物圖鑑當中，到了MHW才只有一種被登記為小型魔物。絕大多數都能被魚叉一擊斃殺。
  • 這邊只列出可以被討伐的個體，暫不記載以釣魚方式來取得的個體。

ガライーバ 咬魚編輯

 

  • 登場作：MH3、MH3G、MHW、MHW:IB、MHR、MHR:SB
  • 棲息在淡水水域當中的小型魔物，攻擊性強。只要看到獵人踏進水裡就會開始攻擊，就算跑到陸地上也可以繼續追擊；不過無法離開水中太久。
  • 由於在MHW之前沒有正式名稱，所以被叫做鯰魚(ナマズ)。
  • MH3、MH3G時期的攻擊性不如MHW之後的作品來的強烈。

サメ 鯊魚編輯

  • 登場作：MH3、MH3G
  • 棲息在海水水域當中的小型魔物，攻擊性強。
  • 古代鮫の皮是製作鯊魚長槍的素材

カツオ 鰹魚編輯

  • 登場作：MH3、MH3G
  • 棲息在海水水域當中的小型魔物，生肉魔物。
  • 有機會能挖到狂走精華。

アロワナ 骨舌魚編輯

  • 登場作：MH3、MH3G
  • 棲息在淡水水域當中的小型魔物，生肉魔物。
  • 有機會能挖到增強劑。

マンボウ 曼波魚編輯

  • 登場作：MH3、MH3G
  • 棲息在海水水域當中的小型魔物，生肉魔物。
  • 有機會能挖到生命粉塵。

ナマズ 水母編輯

  • 登場作：MH3、MH3G
  • 棲息在海水水域當中的小型魔物，生肉魔物。
    • 等一下，魚類？

草食種編輯

ケルビ編輯

凱爾比／斑鹿／精靈鹿
  • 登場作：MH、MHG、MHP、MH2、MHP2、MHF、MHP2G、MH3、MHP3、MH3G、MH4、MH4G、MHX、MHXX、MHW、MHW:IB、MHR、MHR:SB

 

  • 草食種，有分雌性和雄性；肉質鮮美，尤其白肝(ホワイトレバー)因其口感非常受到歡迎，頭上的角可以做為藥用素材
  • 打撃系攻擊打倒必能剝取精靈鹿的角
  • 迅竜的獵物
  • 很多人一開始不知道，在牠原地撞角的時侯有一下動作是帶有吹飛效能的……
  • 只要出問題，鹿角ノ弾弓都能搞定(MH3G限定)

ガウシカ編輯

加烏鹿／雪鹿／大角鹿
  • 登場作：MH2、MHP2、MHF、MHP2G、MHP3、MH3G、MHX、MHXX、MHW:IB、MHR、MHR:SB

 

  • 可以當作是雪地版的精靈鹿，不過角沒有藥效。

モス編輯

摩斯／苔皮豬／蘑菇豬／嗜菇豬／菌豬
  • 登場作：MH、MHG、MHP、MH2、MHP2、MHF、MHP2G、MHX、MHXX、MHW、MHW:IB

 

  • 「モス」有苔蘚的意思
  • 為慶祝踏入豬年(日本地區)，MHW於2019年元旦推出一天限定任務「菌豬是菌但也是菇菌的菌」，有著豐厚的報酬金壓歲錢嗎
    • 任務中所有菌豬全部巨大化，攻擊力更是一擊必殺級，裝衣、盾牌全部沒有用……
    • 貝西「摩斯」/ベヒー「モス」，所以每頭也懂得エクリプモスメテオ/黃道菌豬也是很合理的事(大誤

ポポ編輯

波波／長毛獸／巨牙獸
  • 登場作：MHG[1]、MH2、MHP2、MHF、MHP2G、MH3、MHP3、MH3G、MH4、MH4G、MHX、MHXX、MHW:IB、MHR、MHR:SB

 

  • 其舌頭因美味一樣很受歡迎，可以人工圈養的魔物之一(人工生肉&駝獸意味)
  • 轟竜喜愛的獵物
  • 不是神奇寶貝
  • 亞種是巨獸

アプトノス編輯

阿普特諾斯／草食竜
  • 登場作：MH、MHG、MHP、MH2、MHP2、MHF、MHP2G、MH3、MHP3、MH3G、MH4、MH4G、MHX、MHXX、MHW、MHW:IB

 

  • 可以人工圈養的魔物之一量產生肉二號
  • 跟火竜夫妻一樣全勤(在Rise就突然消失了，可能絕種了)
  • 外觀與現實恐龍的副龍櫛龍相似，地位也一樣。
  • MHW中常常被大凶豺竜整個活吞下肚

アプケロス編輯

阿普克洛斯／甲殼竜／背甲龍
  • 登場作：MH、MHG、MHP、MH2、MHP2、MHF、MHP2G、MH4G、MHX、MHXX、MHW、MHW:IB

 

  • 草食竜中較帶有攻擊性，獵人在前面走過一定會追逐獵人，不過速度慢得要命
  • 新大陸的似乎比較溫和，並不會主動攻擊獵人

リノプロス 草食竜編輯

利諾普洛斯／硬殼草食竜／原角龍
  • 登場作：MH3、MHP3、MH3G、MH4、MH4G、MHX、MHXX、MHR、MHR:SB

 

  • 在MH3時被視為是野豬的接班人，有著比甲殼竜(アプケロス)更強烈的攻擊性，主要出現在砂原跟火山
    • 其衝撞速度雖不如野豬，但獵人正應付大型魔物無暇注意周遭時，這傢伙就會把獵人給撞飛導致攻擊或逃命的最佳時間就此消逝。如破壞大劍的蓄力斬，朝正在對中了落穴陷阱的魔物鬼人亂舞的獵人給撞飛。甚至把獵人給撞回魔物長且致命的原地攻擊招式裡面也是常常發生的事
  • 火箭生肉
  • 唯一一隻有專屬防具的草食種魔物，女性裝備外觀是一隻熊的樣子(劍士槍手皆如此)

ズワロポス 垂皮竜編輯

茲瓦洛波斯
  • 登場作：MHP3、MH3G、MH4、MH4G、MHX、MHXX、MHR、MHR:SB

 

  • 在激戰時，會冷不防的出現在獵人身後攻擊獵人
  • 有自己的結算道具「垂皮油」。

エピオス編輯

艾比歐斯／小水生獸／水蜥竜
  • 登場作：MH3、MH3G

 

  • 水中生肉，在草食種當中對獵人完全無害的生肉，即使是在水裡跟大型魔物交戰，牠也只是在旁邊像顆球般的在原地亂游

ムーファ 雲羊鹿編輯

  • 登場作：MHX、MHXX

 

  • 山羊或羊駝草泥馬形的草食種

リモセトス 蓋鳴竜編輯

  • 登場作：MHX、MHXX

 

  • 相當巨大長頸的草食種，一般姿態時已有獵人兩倍以上的高，形象上像迷惑龍
  • 造出的頭部裝備簡直是草泥馬頭

ケストドン 冠突竜編輯

  • 登場作：MHW、MHR、MHR:SB

 

  • 雄性個體頭部較為寬大，體型上也比雌性個體來的大，形象上像現實恐龍中的冥河龍屬
  • 會主動攻擊玩家，但會被艾路說服成為隨從
  • 大蟻塚地區的UBER，但居然可以輕易地用如獸龍種般的前足來攀爬藤蔓好攀越地形

ガストドン 突擊竜編輯

  • 登場作：MHW

 

  • 外表上有著一顆爆炸頭還有角加上那個配色，其實你是爆炸頭水牛吧
  • 會主動攻擊玩家，但會被艾路說服成為隨從
  • 龍結晶地區的UBER，跟冠突竜一樣可以輕易地用如獸龍種般的前足來攀爬藤蔓好攀越地形

メルクー編輯

麗羊獸
  • 登場作：MHR:SB
  • 破曉新增的草食種
  • 絢輝龍幼體
  • 很影薄

エルペ編輯

阿魯佩／稜角羚
  • 登場作：MHF
  • 高地的山羊，性格溫和到不可思議，會親近獵人甚至根據獵人的動作作出反應

ブルック編輯

布魯克／黑茶氂
  • 登場作：MHF
  • 高地的生肉，針纏竜喜愛的獵物
  • 跟硬殼草食竜一樣，激戰時會冷不防的從視線死角衝過來


獸人種編輯

アイルー編輯

艾路／愛露／礙路
  • 登場作：MH、MHG、MHP、MH2、MHP2、MHF、MHP2G、MH3、MHP3、MH3G、MH4、MHX、MHXX、MHW、MHW:IB、MHR、MHR:SB

 

  • 代表BGM「猫汁大作戦！」
  • 身形瘦小的貓，但也有像MHW的料理長那樣強壯的個體
  • 野生的艾路被討伐後會挖洞逃走，可以撿取他們逃跑時掉落的道具
  • 獵人們的好幫手[2]但在這之前更是魔物們的好同伴，獵人們最不想看見的存在(貓車的意味)
    • 另外死亡會引起任務報酬金減少的原因正是艾路們用車子推獵人回去營地的費用收費很黑箱，越多星的任務收費越貴
  • 結果跑去很奇怪的地方擔任客座武將了（完全無誤）

メラルー編輯

梅拉路／梅拉露
  • 登場作：MH、MHG、MHP、MH2、MHP2、MHF、MHP2G、MH3、MHP3、MH3G、MH4、MH4G、MHX、MHXX、MHR、MHR:SB

 

  • 會偷東西的黑貓，多數地區可以在牠們的巢穴中找回自己的物品，也有少數地區一沒了就是遺憾……
  • 比古龍種更具殺傷力[3]

チャチャブー 奇面族編輯

茶茶普
  • 登場作：MH2、MHP2、MHF、MHP2G、MH3、MHO、MH3G、MH4G(外傳任務)

 
 

  • 戴著面具，數量一多，除了閃光彈和逃走能救到獵人外就是貓車了
  • MH3、MH3G中代替艾路成為了獵人的狩獵夥伴。因為貓不喜歡水，但是MH3以及3G均要下水游泳採集&戰鬥，所以隨行夥伴只好換成奇面族。
  • MH3茶茶（チャチャ）先登場，MH3G另追加一隻卡楊巴（カヤンバ），雖然都是奇面族，但身體外型還是有差異
    • 沒辦法！你以為野生的生活好過啊！（瘦的意味，MH3後的有點發福了，之前的肚子上是肌肉）
  • 能換面具，且每個面具都有各自不同的技能，甚至能節省道具欄的空間，非常方便且種類豐富！
    • 橡果的面具(ドングリのお面)：茶茶專用的初期面具，沒有技能
    • 蟹爪的面具(カニ爪のお面)：卡楊巴專用的初期面具，沒有技能
    • 油燈的面具(ランプのお面)：阿拉丁的神燈，提供照明；生氣時油燈會到處噴火
    • 古代的面具(古代のお面)：長的像青蛙潛艇的面具，技能是在水中供給獵人氧氣，游泳變快，會用魚叉，擅長水戰
    • 瑪卡壺的面具(マカ壺のお面)：技能是道具轉換，類似MH3之前的醃製壺，給他一個道具能把它變成另一個道具
    • 烤肉架的面具(肉焼き器のお面)：一樣是烤肉架但比奇面王的高級(?)，技能就是烤肉，與連續燒肉架作用相同，烤肉時會只剩一個腦袋在地上
    • 毛茸茸的面具(ふさふさのお面)：毛茸茸的兔子，技能是千里眼，不需投擲記號彈就會標出大型魔物的位置；採取的頻率很高，適合採集任務或搬蛋任務
    • 殘念的面具(殘念のお面)：卡楊巴專用，技能是自動格擋大型魔物的攻擊
    • 最好的面具(最高のお面)：兩隻都能用，全金的面具，技能跟殘念的面具一樣
    • 砥石面具(砥石のお面)：超高級的，還是會轉的那種……技能就跟砥石一樣，能將武器的斬味一次回復
    • 否定的面具(ウチケシのお面)：對他下指令就會開始跳舞，舞蹈完後能消去身上的火水雷冰龍等狀態，但不能消除毒麻眠；生氣後會在地上到處灑麻痺彈
    • 臭臭面具(くっさくさのお面)：頭頂著糞球的防毒面罩，能對他下指令讓他對魔物投擲臭玉；講話會變成黑人饒舌風……
    • 落穴面具(落とし穴のお面)：對他下指令就會鑽入地下形成落穴
    • 大砲面具(大砲のお面)：對他下指令就會變成大砲，然後另外一隻就會變成砲彈跳入然後射出，當然，當砲彈的被射出去後血量是當場歸零
    • 艾路面具(アイルーフェイク)：仿艾路的麻布白貓臉面具，技能是投擲炸彈
    • 梅拉路面具(メラルーフェイク)：仿梅拉路的麻布黑貓臉面具，技能是偷取魔物的素材，隨時可能就會帶顆寶玉回來，一隻魔物只會偷一次
    • Keroro面具(ケロロフェイク)：無言……，有青蛙釣餌的效果[4]及在陷阱上挑撥魔物上當（這……）
    • 替身的面具(スタンドのお面)：JOJO的合作面具，造型是錢寶寶的樣子，會將撿到的物品變成硬幣，硬幣只能作賣錢用
    • 海賊Ｊ的面具(海賊Jのお面)：帶著海賊帽的哥爾·D·羅傑頭顱，技能是護主，當獵人陷入危機時，全身會閃耀金色光輝，攻擊力上升且不斷猛攻
    • 不倒翁的面具(ダルマのお面)：造型是艾路臉的達摩不倒翁，技能是回歸球，道具選單中出現回歸球
    • 丸鳥的面具(ガーグァのお面)：造型像丸鳥的面具，特性是在被打敗撤退時，會在地上留下道具
    • 火竜的面具(れうすのお面)：造型仿造火竜臉的面具，對他下指令就會使用火竜咆吼，有音爆彈效果，連續按會嗽聲，生氣時會用火球攻擊；裝上之後不會講話，只會發出嘎嘎的吼叫聲
  • MHO變成藍皮膚，面具則因武器有變化
    • 棒槌的面具：帶這面具的奇面族武器會是有尖刺的棒槌，面具類似木質鐘罩並且在後腦勺部分有個彎角
    • 吹箭的面具：帶這面具的奇面族武器是吹箭，不過沒什麼命中率，面具類似縮小的將軍鐮蟹的龍頭殼，兩側插了羽毛
    • 盾牌的面具：帶這面具的奇面族沒有武器，面具本身就是一個塔盾，最底下兩側有類似成年波波牙的裝飾，在蹲下時正面是必定彈刀，之後會做推擊攻擊推倒在近身範圍的獵人

キングチャチャブー 奇面王編輯

茶茶普王
  • 登場作：MH2、MHP2、MHF、MHP2G

 

  • 現獸人種唯一的(中)大型魔物
  • 頭上頂著燒肉架作為皇冠
  • 看來小隻，其實相當可怕(明明只是被斬到一下，角色只搖了一下，血量卻像被魔物火車一樣……)

ガジャブー 奇面族編輯

  • 登場作：MHW、MHW:IB

 

  • 居住在新大陸的奇面族群
  • 與舊大陸的チャチャブー一樣會主動襲擊獵人，還會向獵人投擲麻痺鏢、睡眠鏢等，當獵人專注在與大型魔物戰鬥時被送上貓車的間接兇手……
    • 但另一方面也會攻擊大型魔物，反而會無意幫助了玩家
  • 與チャチャブー不同的是，完成支線任務後會給艾路新的道具，之後地圖上也會出現跟獵人友好的奇面族(隨機配置或去奇面族巢穴都會遇到)，可以招募他們幫忙狩獵，還可以使用奇面族的捷徑，穿輕裝拿片手劍的也就罷了，那些穿盔甲拿大劍/長槍的獵人到底是怎麼鑽過去的呢？
    • 另外地圖上也會隨機配置一隻定點支援，戰鬥中靠近的話會跳出來對大型魔物進行炮擊，但無法指定目標
  • 在與「Witcher 3」的連動特別任務中，首次出現名為「族長」的個體，因被鹿首精的影響而失控
    • 與舊作的奇面王不同，儘管擁有大型魔物般的血量及攻擊力，此個體仍然被視為小型魔物而已

    • 【赤色眼爪】力量型態，攻撃力2.00倍
    • 【白色眼爪】全力型態，結合了部分靛色的結晶、綠色的火焰、青色的速度、赤色的筋肉[93]

    • 如果其中一隻變成綠色遠距火焰型態的話，很可能會被從遠端噴來的火焰給燒死
