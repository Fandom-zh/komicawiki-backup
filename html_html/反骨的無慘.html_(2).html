簡介[編輯]

為 ◆2sRGUbBO9j2n 所創的「安科帖(あんこ帖)」，是一種以AA角色為形象出演、並在情節轉折處由作者準備多重選項但以"骰子"決定實際劇情走向的同人創作形式

  • "骰子"是公正的第三方隨機值產生器，由於對安科劇情影響甚大，又常被敬稱為「骰子神」或「骰子監督」
    • "骰子"的標記法為[xDy]，x為骰子顆數、y為骰子最大值，例如:1D100就會出現1~100之間的隨機值
  • 如同標題一樣，本創作是以鬼滅之刃的無慘作為主角形象來挑戰反逆的魯路修劇情的故事
  • 雖然不如作者上一作迪亞哥‧布蘭多似乎在挑戰Grand Order的樣子般引起話題，但爆骰指數完全不落下風依舊充滿各種超展開與令人目瞪口呆的劇情發展

登場主要角色[編輯]

鬼舞辻無慘[編輯]

  • 稱號：日本解放陣線領導人、護國之鬼
  • 特殊能力持有：無→ GEASS「絕對反骨」→ CODE
  • 本作主角，反骨心的集塊
    • 沸點極低，"脾氣非常暴躁"不足以形容的傢伙、碎碎念之王
      • 憤怒判定幾乎每一回都要擲好幾次
    • 不想讓他當上司但是更不想讓他當部下的男人
    • 不過對於100%正論反駁的抵抗力是零，原本再生氣都會乖乖冷靜下來
    • 除了反骨心超強以外，其實沒啥政治慾望或野心
  • 護國之鬼
    • 脾氣差卻足以服眾的原因，是因為真材實料的才幹，不論作為戰術戰略指揮或是前線駕駛員都非常有才能
      • 戰術戰略敏感度超級高，作為駕駛員則是圓桌級
      • 偏愛斬首戰術(大概是因為在鬼滅那棚被虐慘了)不過鬼滅原作去找產屋敷也算是斬首戰術
    • 在有能的基礎上還維持著全作第一的過勞度，連作者要他休息，都會被硬凹回辦公務的執著度
      • 畢竟是組織內極少數的頭腦擔當，他不工作就沒人能夠彌補他的重要性
  • 戀愛廚
    • 對他人(ZERO)的、對自己的配對都超級認真

魯路修[編輯]

  • 稱號：ZERO、合眾國日本總理
  • 特殊能力持有：GEASS「絕對服從」
  • 原作主角
    • 在經歷瑪麗安娜暗殺事件之後，遇上C.C.給予GEASS
    • 化名ZERO，為了知道真相踏上了反逆不列顛尼亞之路
    • 有最高級別的頭腦，但駕駛技術則只能算中上
  • 超越親情等級的超級妹控
  • 溫柔的世界
    • 迴避原作九成九的鬱展開，明明不當主角卻過得比原作更爽了

黑死牟(樞木巖勝)[編輯]

  • 稱號：鬼之懷刀、上弦之一
  • 鬼之懷刀
    • 無慘最忠實的部下，無慘的命令會絕對服從的男人，連無慘懶得說出口的話都能即時翻譯給別人聽
    • 作為一名兵士是日本的王牌，駕駛技術圓桌級，白兵戰也能開無雙(但兩樣能力都略輸朱雀)
  • 兄上
    • 樞木朱雀的親哥哥，和小時候的朱雀感情還不錯，也常以類似義兄的立場照顧著小時候的魯路修和娜娜莉
  • 受詛咒般的樞木血脈
    • 其實跟鬼滅原作形象沒甚麼不同，不過在本作中發揚光大成為令讀者胃痛的主因之一

樞木朱雀[編輯]

  • 稱號：EU的惡夢、娜娜莉專任騎士
  • 魯路修最好的朋友
    • 在瑪麗安娜暗殺事件之後，魯路修、娜娜莉被送到日本時，結下了堅固的友誼
  • 名譽不列顛尼亞人
    • 在日本和不列顛尼亞戰爭時，刺殺了身為日本總理的父親，因為此經歷成為了名譽不列顛尼亞人
    • 為了朱雀自己的目標，朱雀進一步加入了不列顛尼亞軍
  • 外掛系男子
    • 原作戰力頂點之一，駕駛技術和白兵戰都踏入非人境界
  • 受詛咒般的樞木血脈
    • 其實跟原作形象沒甚麼不同，不過在本作中發揚光大成為令讀者胃痛的主因之一

查爾斯·Di·不列顛尼亞[編輯]

  • 稱號：第九十八代不列顛尼亞皇帝
  • 特殊能力持有：GEASS「修改記憶」
  • BOSS系男子
    • 嘴上常說著弱肉強食之理，對皇帝的身分也多有矜持，在世界各地引發侵略戰爭
    • 真正的目標是完成「諸神的黃昏」
    • 本作中變得比原作圓滑點但依然是敵役擔當
  • 笨爸爸
    • 本作中雖然被魯路修單方面敵視，但實際上卻是沒捨棄過親情的笨爸爸
    • 對自己的兒女滿溢著濃濃的父親愛

瑪麗安娜[編輯]

  • 稱號：閃光的瑪麗安娜
  • 特殊能力持有：GEASS「意識轉移」
  • 魯路修、娜娜莉的母親
    • 因為她被暗殺的事件而使得魯路修敵視查爾斯
    • 在柯內莉亞和尤菲米亞的印象中是位氣質皇妃
  • BOSS系女子
    • 如果說朱雀的駕駛技術和白兵戰是踏入非人境界，那瑪麗安娜就是神與惡魔的領域、毫無疑問的當世最強
    • 真正形象好動而與常人有所偏差，與其說是腦袋掉了幾顆螺絲，不如說打一開始她的整組螺絲就都不是圓形的
    • 贊同查爾斯完成「諸神的黃昏」的計畫，並一開始就知情幫忙

V.V.[編輯]

  • 稱號：嚮團之主
  • 特殊能力持有：CODE
  • 死弟控
    • 查爾斯的哥哥，因為早年就得到CODE而保持正太身體
    • 主張親子之情遠不如兄弟之情
  • 萬惡之源
    • 雖是和查爾斯同樣有為了不列顛尼亞、為了完成「諸神的黃昏」的目標，但有自己的其他考量、會暗地行動
    • 日本與不列顛尼亞戰爭時，在查爾斯不知道的情況下獨斷執行策反戰術
    • 瑪麗安娜暗殺事件的元凶

C.C.[編輯]

  • 稱號：披薩魔女
    • 熱愛披薩、喜愛披薩、連AA立繪也有吃披薩造型的女人
    • 無慘甚至為了她而開始了鑽研作披薩
  • 特殊能力持有：CODE
  • 不死魔女
    • 原本是「被愛」GEASS的持有者，但被CODE持有者硬塞CODE之後失去GEASS能力
    • 之後一路且活且過的到了本篇時間點，因為永生對內心的磨耗其實想死
    • 知性和氣質都不同凡響，不過對一般人的嘴巴很毒
    • 反過來說，如果是特別的人，這位不死魔女還是會嬌的
    • 給予魯路修GEASS的人，並不時會擔任魯路修戰略相談的對象

藤堂鏡志朗[編輯]

  • 稱號：奇蹟的藤堂、上弦之二
  • 奇蹟的藤堂
    • 在日本與不列顛尼亞的戰爭中創造了嚴島的奇蹟的男人，駕駛技術圓桌級
    • 不過在本作中戰力發揮的場面不多，有出場幾乎都是被無慘罵或是無慘的吐槽役
    • 但是，在推薦戀愛遊戲的眼光上，真的變成奇蹟級了
      • 【1D100:100】奇蹟的藤堂絕非浪得虛名！
    • 事後，藤堂表示就是希望無慘能找回因戰爭而淡薄的人性，才挑了"那片"戀愛遊戲
      • 奇蹟的藤堂絕非浪得虛名！

用語[編輯]

瑪麗安娜暗殺事件

  • 簡介
    • 某日，皇妃瑪麗安娜受不明人士暗殺，直接目睹事件經過的娜娜莉因精神衝擊而失明並不良於行
    • 魯路修前去找查爾斯詢問，卻反被查爾斯以弱肉強食論為理由把魯路修和娜娜莉近乎流放的送去日本當人質
    • 不列顛尼亞之後又開啟對日本的戰爭，不顧魯路修和娜娜莉生死，故魯路修決心找出事件真相並視查爾斯為敵
  • 真相
    • V.V.死弟控因為嫉妒瑪麗安娜和查爾斯感情太好，於是策劃暗殺行動
    • 然而瑪麗安娜使用GEASS，在肉體死前讓意識逃到了路過的亞妮身上
    • 查爾斯知道了事件經過，但為了「諸神的黃昏」計畫不能和V.V.撕破臉，決定送魯路修和娜娜莉去日本避險
    • 瑪麗安娜的意識則繼續躲在亞妮身上，提防V.V.逮到機會再對瑪麗安娜不利
    • 而不列顛尼亞之後開啟對日本的戰爭的理由原作和本作不同請自行觀看劇情區

KMF

  • 總之就是大型機器人，魯路修原作就有的要素
  • 本作中出現的大部分是原作機體，不過(因為骰子神的調皮)偶爾會混進奇怪的東西

櫻石

  • 重要的能源資源，總之當作萬能版石油就好
  • 日本的天然存量與出口量最大，占世界產量的70%

圓桌武士

  • 原指不列顛尼亞理論上最強的十人將兵稱號，也用來當作無雙級戰力單位
  • 同樣是圓桌級還難說，不到圓桌級幾乎就是被秒殺的份
