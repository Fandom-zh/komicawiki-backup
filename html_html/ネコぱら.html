貓娘樂園

NEKOPARA / ネコぱら / 貓娘樂園 / NEKO%RA / NEKOPAPA / NEKO%% / 啪貓 / 快給我變
本條目內容不足需要擴充，歡迎有興趣的朋友加以補完。
請以愛貓的角度來看待此條目，愛貓就不要%貓喔。

目錄

概要編輯

  • NEKOPARA是由同人社團NEKO WORKs開發出一款第一人稱射擊[1]戀愛冒險美少女貓遊戲。
  • 早在2009年遊戲還沒開發前，巧克力和香草為NEKO WORKs社團的看板娘，之後還出了相關的line貼圖，後來在2014年才推出NEKOPARA的遊戲。
  • 2017年OVA動畫化，2020年1月電視動畫化。
    • 2020年11月17日晚霹靂台灣台在Facebook粉絲團宣布11/18起於晚間23:30撥出可惜沒有閩南語配音
  • 2021年4月1日宣布製作男版《貓郎樂園-Catboys Paraside（ネコぱら- Catboys Paradise）》，原本只是2019年的小玩笑，但在其他玩家的熱烈迴響下，NEKO WORKs 決定讓此遊戲企劃成真，強調他們是真的有在製作此作品。

劇情編輯

  • 主人公水無月嘉祥離開過去的老家在橫濱市經營一家糕點店「La Soleil」，當他要打開行李時，發現行李中混了兩隻從老家來的「人型貓」巧克力與香草，原本嘉祥想要將她們送回去，但是貓咪們不願意，於是就變成嘉祥和兩隻貓娘一起經營糕點店。
  • 之後就是在日常篇穿插跟各位貓娘嘿咻的場景(要買18+補丁DLC)

登場人物編輯

水無月家編輯

  • 家中後院很大，可以擺下一整套極限體能王的外景

水無月嘉祥編輯

  • CV：立花慎之介（OVA、TV動畫）
  • 本作男主角，糕點店「La Soleil」的主人。
  • 雖然是十八禁美少女遊戲的男主角，不過他在遊戲裡有畫出臉%%時就沒畫臉了。
    • 雖然應該是漫畫版和OVA的影響。
  • 非常關心家裡的六隻貓，最初巧克力和香草被撿回家時也是因為他的關心才讓她們願意親近。
  • 精力過人，每次可以連續來好幾發， 第四部聖誕夜後可以被乳交+足交後全部六隻再射一輪而且射很快
  • 神木一根，六隻貓都發情了不知道多少次還能完全沒注意到她們是因為自己而發情，甚至常常連她們在發情都沒注意到，還要是她們自己要求才會跟她們做
    • 漫畫版比較好一點，有注意到香草的發情。
    • TV動畫因為不是18+，所以有被砍戲份

水無月時雨編輯

  • CV：佐倉江美（遊戲） / 永井真衣（OVA） / M・A・O（TV動畫）
  • 水無月嘉祥的妹妹也是貓咪們的主人，非常喜歡貓咪。
  • 在和貓咪們一起工作的時候會戴上貓耳。
  • 水無月家六隻貓都是由她一手養大和調教，對她們來說比起姐姐更像母親。
    • 然後不知道為什麼每次哥哥吃掉她帶大的貓娘之後情緒都會變得很high，還會喊出各種奇怪的問題發言和瘋狂大笑
  • 重度兄控玩家表示到底哪一集才推倒時雨[2]
  • \I wanna be NEKO〜〜〜〜〜〜〜〜〜〜！/[3]
  • 動畫中新增加會拿相機拍貓娘的痴女屬性

貓娘們編輯

巧克力(ショコラ)編輯

  • CV：ヒマリ（遊戲） / 朋永真季（OVA） / 八木侑紀（TV動畫）
  • 黑貓，和香草是雙胞胎。(兩人之中巧克力是姊姊)
  • 頭腦不是很好，個性天然呆。
    • 大家懷疑嘉祥有女朋友的時候只有她直接去問本人。
  • 非常喜歡嘉祥，就算沒發情也會常常黏著他。
    • 順便一提，她是故事裡最早發情的，也是第一個被嘉祥吃掉的，而且是被香草引導的
  • 初到水無月家時因為本來是被拋棄的孩子而完全不肯親近人（香草也是）。
    • 開始親近人的契機是後來兩人重感冒時嘉祥用心照顧，也是那時開始喜歡嘉祥
    • 所以天然呆的個性其實是那時候燒壞腦袋了？

香草/香子蘭(バニラ)編輯

  • CV：中村あむ（遊戲） / いのくちゆか（中文：豬口有佳）（OVA） / 佐伯伊織（TV動畫）
  • 白貓，和巧克力是雙胞胎，個性聰明。
  • 無口，但喜歡吐槽。
    • 正確來說，她是習慣觀察大局之後提出一針見血的意見。
  • 雖然也喜歡嘉祥，但總是強調自己喜歡的是巧克力
  • 胸部較巧克力小
  • 通常都是連接在巧克力之後做的姉妹丼
  • 和巧克力本來都是棄貓，是在某個下雨天被嘉祥和時雨發現然後撿回家的。
  • 克里姆的媽媽

楓（メイプル)編輯

  • CV：小倉結衣（遊戲、OVA） / 伊藤美來（TV動畫）
  • 女版塔爾斯[4]
  • 能轉動尾巴來飛行
  • 水無月家貓咪六姐妹的次女，捲耳貓。
  • 非常膽小，晚上去上廁所時總是要把走廊上的燈全部打開。
  • 充滿時髦意識，很注重流行事物，並曾教巧克力使用智慧型手機。
  • 有著音樂天分，唱歌非常好聽，但是本人一直不敢表現出來。
  • 傲嬌，不過通常都是桂讓她嬌。
    • 桂初到水無月家時很怕生，當時的楓對她說教了一番把她帶出來，所以桂很喜歡她。
    • 桂會變成M大概是她的傑作
    • 真正的可可蘿(聲優同一個人)

桂（シナモン)編輯

  • CV：有坂羽由（遊戲） / 岡嶋妙（OVA） / のぐちゆり（TV動畫）
  • 水無月家貓咪六姐妹的三女，蘇格蘭摺耳貓。
  • 平時打扮成魔法少女水手服的模樣(其實貓娘穿著都是時雨的品味)，貓娘姐妹裡胸部最大。
    • 動畫最終話時提及胸圍是92.5cm，而且還在成長中。
    • 因為其巨乳還常常被其他姐妹欺負，會拍打她的胸部出氣，然後桂本人會因為自己是M的關係而興奮
  • 抖M癡女貓，光是看生物課本講述植物繁殖就會發情（無誤）
    • 對於M這點有自覺，被欺負時會表示「就算人家是M，這樣做也太過份了啦！」。
  • 跟楓的感情很好，而且是有肉體關係的程度（無誤）。
  • 同樣喜歡嘉祥，姐妹中少數提過想要嘉祥吃掉她的。哈很久了
    • 嘉祥替她買了幫楓伴奏的樂器時因為太開心，還表示只要嘉祥開口就可以用她的巨乳當枕頭。
  • 繪師養的貓
    • 傳說桂的人設藍本來自於繪師自己養的蘇格蘭摺耳貓所以理所當然的畫的色氣十足。
    • 即使是在非限制級的動畫版裡也經常會發情(無誤)，頻率是幾乎每集都會

椰子（ココナツ)編輯

  • CV：手塚りょうこ（遊戲） / 立石めぐみ（OVA） / 水谷麻鈴（TV動畫）
  • 水無月家貓咪六姐妹的四女，緬因貓。
  • 身材高挑，熟女，還有著其實以貓咪來說很常見的異色瞳孔。
    • 不過熟女只是外表而已，她的心智年齡很幼小，大概跟巧克力和香草差不多。
    • 由於品種是大型貓，所以身體成長很快速，在OVA小貓篇中外表還是蘿莉，到了本篇時急速成長到現在的模樣讓她有些不知所措。
  • 常常會跟紅豆打架。
    • 事實上是因為兩人都沒辦法坦率面對對方的結果，在收養巧克力和香草之前椰子是最黏紅豆的。
  • 因為身體的快速成長，加上多了巧克力和香草兩個妹妹，所以原本個性就像個孩子的她產生了「要表現得像個姐姐」的自覺。
    • 因此是時雨調教的貓娘中行為最端莊的，本人以此而自豪。
    • 但她的本性其實是活潑的孩子，而且跟時雨一樣是兄控，是個會稱呼嘉祥「哥哥」的僕少女。
      • 不論是哥哥的稱呼還是「僕」的自稱都強迫自己改掉了，但是後來在嘉祥和紅豆的開導下不再壓抑。
    • 而且，手腳非常笨拙（不過運動神經很好），所以很害怕被當成「除了身體很大一無是處的貓」。

紅豆（アズキ)編輯

  • CV：成瀬未亜（遊戲、OVA） / 井澤詩織（TV動畫）
  • 水無月家貓咪六姐妹的長女，曼赤肯貓。
  • 雖然比其他貓咪年長，但外表卻像蘿莉。
    • 這是因為品種是小型貓。以人類來說就是合法蘿莉。
  • 說話毒舌，非常任性。
    • 是那種就算踢到鐵板也會一直嘴砲的類型。
    • 但其實本性很成熟負責，不只是會關心著妹妹們，和嘉祥也常有成熟的互動（←絕對不是紳士意，雖然他們也做了很多次）。
      • 尤其特別關心椰子，就算是兩人變得經常吵架之後也還是暗中擔心她[5]。對椰子的暱稱是「ナツ」。
  • 相對於手拙的椰子，她有著嘉祥教她做糕點時很輕鬆就上手的手腕。
  • 不喜歡人家叫她「短腿貓」。
  • 每次做完會尿失禁

可可蘿（カカオ）編輯

  • CV：森嶋優花（TV動畫）
  • 動畫原創角色。與巧克力邂逅的走失年幼貓娘，最喜歡吃小魚乾。暫時和巧克力及香草一起住在「La Soleil」蛋糕店。
  • 出乎意料的能幹，而且行動力很強，唯一的弱點是怕水。
  • 台詞很少，聲優很省事

其他編輯

牛奶（ミルク）編輯

  • 在嘉祥的店的附近的公園的某個章魚燒攤販的看板娘。
  • 原本也是流浪貓，被沒有家人的老闆娘撿回來當作女兒般疼愛。
  • 有次差點出車禍，千鈞一髮之際被椰子救下，成為了後來開導椰子的契機。
  • 非主要角色中唯一有動畫建模的角色。
    • 另外她和老闆娘也是少數有重複登場的非主要角色，可惜全勤記錄在VOL.3就破功了。
  • 於VOL.4再登場，並在半年間成長到少女外表。
