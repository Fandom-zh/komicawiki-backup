心跳魔女神判

心跳魔女神判/どきどき魔女神判！/どき魔女
本詞條含有遊戲劇透，沒玩過遊戲又不想變成究極大善人的人請斟酌查閱。
嫌疑人真身部分煩請隱藏，直接劈頭就寫某人是魔女會被變成究極大善人。

目錄

概說編輯

  • 由SNK公司代理的NDS掌機遊戲，遊戲外盒就寫白【可以使用NDS觸碰功能】令嫌疑人心跳加速，進而顯現出身為魔女的鐵證，魔女紋章之詭異遊戲。
    • 雖然感覺很鹹濕，不過還是處於CERO-D[1]之範疇，所以不會有露點鏡頭出現，就算有也是男的[2]。
  • 一共發售無印一代、二代Duo以及一代PLUS。
    • 時間軸是無印一代接二代，然後無印一代與一代PLUS兩者為平行世界。
  • 無印一代定位還不明，二代以後的作品有導入戀愛遊戲的要素，會依照人物的好感度決定終章結束後的結局。
  • 不過因為畫風令多數人來說很難適應[3]，後來該遊戲之NDS掌機賣肉番地位被閃亂神樂給取代（不過pixiv上還不少兩者相結合的圖…）。

主要登場人物介紹編輯

無印一代編輯

  • 西村アクジ【CV：尾崎未來】
    • 系列作的主角，星嶺鷹守學園三年級生，總是身穿有別於學校制服的黑色制服之惡人[4]。
      • 二代因為學力問題，轉學天櫻守學園時被迫降級成二年級，雖然二代事件結束後轉回原校三年級。
    • 陰錯陽差之下接下天使界給予的魔女搜索之任務，開始魔女搜尋後宮搜集之冒險。
  • 天使ルル【CV：園田ひろこ】
    • 陪伴著西村アクジ的守護小天使。
    • 超級好色，會對身材出眾的女性毛手毛腳。
      • 同時也是個超級破壞狂，動不動就拿出大槌把一切砸爛（不過無法在戰鬥中使用）。
    • 於二代時可以變身成人類的外觀，但是好色性格完全沒變。
    • 戰鬥時除了給與主角天使劍以外，也能消耗一些SP使自己對敵人做出突襲攻擊（物理攻擊，觸控筆按住螢幕一段時間）。
      • 二代以後主角的天使劍可隨意揮擊（可隨意畫線，無印一代就只能畫直線比較難用）。
  • 赤井まほ【CV：佐藤まさみ】
    • 星嶺鷹守學園二年級生，愛貓家。
    • 真身為魔女，一代PLUS更追加為大魔女紅獅子的唯一女兒之設定。
    • 可以變身成幼女以及上圍突出的女學生外觀（一代PLUS的後者原型乃其母親）；無印一代就只是利用魔法改變外型，一代PLUS就把這設定稱作能夠改變時間之強大魔力。
    • 戰鬥時能消耗SP發射直線型魔法彈（點擊螢幕），集氣（觸控筆按住螢幕一段時間）會變成雙顆魔法彈（無印）或是從天而降的貓罐頭（一代PLUS）。
    • 一代PLUS因為原居住地過於簡陋[5]，第一章結束後與主角住在一起，新模式顯示為其房間而非某角色的家。
  • 安倍マリア【CV：上田真紗子】
    • 星嶺鷹守學園三年級生，盡做些奇特的實驗。
    • 真身為人類，但在魔女神判之後自稱大魔女，也許與其姓氏有關（大陰陽師安倍晴明）。
    • 戰鬥時於無印一代僅有投擲藥水瓶攻擊（無印為點擊，PLUS改成按住螢幕一段時間），集氣版為黃色藥水（按住螢幕一段時間，會出現集氣完成的特效），能令對手麻痺一段時間。
      • 一代PLUS追加定時炸彈（點擊螢幕）。
  • 小田れんげ【CV：後藤惠梨子】
    • 星嶺鷹守學園二年級生，電玩宅女。真身為魔女。
    • 跑速非常快
    • 戰鬥時為盾牌角色，僅有吸收盾來吸收對手的魔法攻擊（點擊螢幕），吸收一定量的魔法彈後會反射出特殊的魔法彈攻擊對手。無法防禦物理以及音波類攻擊。
  • 御堂あやめ【CV：壱智村小真】
    • 星嶺鷹守學園三年級生，家中開神社所以是巫女，也會去女僕憩茶打工所以也具備女僕特性。
    • 真身為魔女，能召喚火焰侍神。
    • 無印僅有火焰彈攻擊（點擊螢幕，可以移動位置持續攻擊直到SP用完）。
      • 一代PLUS追加陰陽吸收盾（按住螢幕一段時間），可以吸收音波類攻擊；面對Final Boss背德魔獸的咆嘯攻擊的唯二解法。[6]
  • 望月ゆーま【CV：坂田有希】
    • 星嶺鷹守學園一年級生，男性，但是穿上女生制服卻意外的很美。
    • 真身為魔女（？），保護該地區的獸人；一代PLUS可以在故事模式中決定是否要對他進行魔女神判。
    • 攻擊為爪擊（畫線）以及集氣版的地面衝擊波（按住螢幕一段時間），爪擊攻擊速度比主角的天使劍要來的快。一代PLUS在簡單難度之下也能反彈對手的魔法彈。
  • 綿引メリー【CV：坂田有希】
    • 星嶺鷹守學園一年級生，終日穿者兔子布偶裝的奇特學生。真身為魔女。
    • 無印一代時身分只是普通學生，但是一代PLUS卻改成超有錢的千金大小姐。
    • 完全不會游泳。
    • 基本上會對其他女性具有敵意，但會對主角抱有好感。
    • 攻擊為放置受到傷害會爆炸的替身以及召喚隕石攻擊，隕石越大攻擊時間就越長但攻擊力也越大。
  • 聖夜イブ【CV：伊藤葉純】
    • 星嶺鷹守學園校醫，身分以及穿著還有身形超級容易讓人有エロ想法。
    • 真身為天使，該地區的駐地天使，與聖夜ノエル為姊妹關係。
    • 攻擊為曲線型魔法彈[7]以及天降聖之雷（無印與PLUS時攻擊方式完全相反）
  • 聖夜ノエル【CV：伊藤葉純】
    • 星嶺鷹守學園圖書館管理員，在知道名字以前都以「司書」作其名字。
    • 真身為天使，無印一代的黑幕。
      • 發動智之迷宮，打算發動儀式成為取代神之熾天使[8]；聖夜イブ是她姊，但是其人類外型與聖夜イブ完全讓人想不透她們是姊妹關係。
      • 一代PLUS在此事件後降級成與天使ルル一樣的位階作為懲罰；二代在完成第五章之後，成為天櫻守學園圖書館管理員兼該地區的駐地天使。
      • 二代在第六章之中，會有選項決定是否直接對她施展魔女神判。[9]
    • 攻擊為大型天使劍，攻擊力比主角的還要高出一截但是只能直線揮擊（畫線）；與姊姊同類型的魔法彈（點擊螢幕），還有傷害減半護罩[10]（按住螢幕一段時間）；此外當在場上維持一段時間還會自動召喚小神獸攻擊。
      • 只有在一代PLUS才會以同伴身分一同作戰。

二代Duo編輯

  • 惡魔クロ【CV：壱智村小真】
    • 陪伴者西村アクジ的守護小天使惡魔。
    • 本名「クローディア・リバルディア」，於400年前被天使界給俘虜作為下僕，現為天使ルル的弟子。
      • 雖說在天使界待了400年，但是惡魔性格仍然未改，主角人在正中午的校舍頂樓就會大叫快要死掉，然後遇到一些狀況就會說出想要收割靈魂等等。
    • 戰鬥時風格與天使ルル完全相同。
  • 羽織くれは【CV：櫻井浩美】
    • 天櫻守學園三年級生，同時也是該學校的學生會長，幾乎無所不管的大小姐個性。
    • 真身為魔女，也可以變成同髮色之飛鳥外型，所以才有粉紅色羽毛之證物。
    • 攻擊為直線型魔法彈（點擊螢幕）以及能吸收音波攻擊的護盾（按住一段時間）。
  • 比路ソフィ【CV：宮崎羽衣】
    • 天櫻守學園三年級生，身穿中世紀女僕裝，並且有獅子「ライ太」與鸚鵡「ペロケ」作伴。
    • 真身為魔女。
    • 攻擊為召喚ライ太突擊（點螢幕）和ペロケ的音波攻擊（按住一段時間）。對手的魔法吸收盾無法吸收這兩種攻擊。
  • 友里かもめ【CV：伊藤葉純】
    • 天櫻守學園二年級生，熱愛ADK遊戲《ニンジャコンバット》，還把主角叫成該遊戲男主角的名字的腐女（將アクジ為ジョー，セイジ則是ハヤブサ，並妄想BL情節）。
    • 真身為魔女，戰鬥時為忍者外觀。
    • 攻擊為消耗SP的召雷彈（點擊螢幕）和飛鏢（按住一段時間）。
      • 飛鏢還有準備需時較長的毒飛鏢（按住一段時間直到出現集氣特效），一樣能令被命中的對手一段時間內無法動彈。
  • 双葉なみ、双葉なぎ【CV：味里】
    • 天櫻守學園一年級生，雙子，當中一個是[[偽娘]]。
    • 真身為魔女，而且兩個人都是。[11]}}
    • 攻擊為從天而降的冰之魔法彈（點擊螢幕）、冰霜塔屏障（按住一段時間）以及減半傷害之冰雪護罩（按住一段時間直到出現集氣特效)。
      • 冰雪遮罩發動期間無法換人。
  • 渚烈火【CV：小林優】
    • 天櫻守學園二年級生，劍道社社長。
    • 其實只是人類，本身不具備魔力，火焰力量來源來自於其母親遺留給她的太刀，揮舞時才具備火焰之魔力。
    • 攻擊為火焰一閃（畫線）以及五連火焰彈（點擊螢幕）。火焰一閃可以與對手的魔法彈攻擊相抵消。
  • 理澄ねおん【CV：野川櫻】
    • 偶像歌手，年紀可能為全系列作之中最小。真身為魔女。
    • 戰鬥時為盾牌角色，僅有吸收盾來吸收對手的魔法攻擊（點擊螢幕），吸收一定量的魔法彈後會反射出特殊的魔法彈攻擊對手。無法防禦物理及音波類攻擊。
  • 櫻ひめき【CV：小林由依】
    • 記者。在主角大幅度使用數位相機的年代，仍使用傳統底片型照相機的復古派人物。
    • 是人類，原本是MMS反天使小隊隊長。
      • 過去曾參與人類對天使之間的戰爭，所以才被天使界故意用高度危險的魔女嫌疑人來告知主角[12]。
    • 攻擊為準備時間較長的能量破壞炮（點擊螢幕）和可以一次發射相當多顆的能量彈（按住一段時間，可移動位置）。能量破壞炮要是完全命中會給對手相當大的損傷。
  • 聖花ころん【CV：古山貴實子】
    • 天櫻守學園二年級生。個性開朗，能直接看到惡魔クロ。
    • 真身為天神，二代黑幕。
      • 二代故事開始14年前發動禁忌魔法，結果自己變成嬰孩出現在人間界被マスター收養；隨後在故事第七章疑似吃下不是人間界的藥，結果天神人格覺醒。
    • 攻擊風格與無印一代的聖夜ノエル相似，不過血量剩一半時會發動多重魔法彈攻擊，必須判斷該拿魔法劍反彈或者召喚護盾吸收之。

一代PLUS（新增）編輯

  • 夏南きあら【CV：鈴木美子】
    • 一代PLUS的封面人物，厭惡主角但對赤井まほ很有好感。
    • 真身為惡魔，本名卡蜜拉，欲把赤井まほ獻給惡魔王當作妻子，一代PLUS的黑幕。
    • 與赤井まほ一樣能變化外型年齡，遊樂場進場騙兒童票然後變成大人外型玩刺激型遊樂設施
    • 本身戰鬥風格與聖夜ノエル相似但能力不及無印一代的聖夜ノエル，但是主要劇情戰鬥會強制召喚真BOSS「背德魔獸」，血量分開計算下魔獸血量非常高又封印主角的大絕系統，使得此對手超級難纏。
      • Boss Rush模式下可以透過強烈攻擊來中止她召喚背德魔獸。

名詞解釋編輯

  • 魔女神判
    • 取得天使界的許可後，透過手上的光環將戰鬥中落敗的魔女嫌疑人進入精神世界，然後透過觸碰+視姦+吹氣+拿出證物等手段令目標心跳加速，進而顯現出身為魔女的鐵證→魔女紋章作終。
      • 當然要是發動神判時不小心讓對象動怒或者不快（背景會變成冰冷的深藍色），也會因而落敗=Game over。
    • 除了找尋魔女紋章之主要用途，也能透過此機能來執行另外之用途。
  • 究極大騸善人
    • 魔女搜索人在與魔女嫌疑人戰鬥中要是失敗就會變成此人，全身上下被騸得完全沒有任何邪念，而且會四處行善，甚至會把身上僅有的內衣褲脫下來給受凍的路人，自己僅以樹葉來遮羞。
    • 由於行善程度誇張到令路人覺得這人太噁心，也有可能就是警告主角不能落敗之警語。
      • 而咱們主角失敗的外型會變成--三藏法師(部分沒有誤)
  • 天使劍
    • 顧名思義就是由天使界給與魔女搜索人的基本武器，同時也存在者只有大天使才能用的進階版本。
    • 可以將魔法彈像打羽毛球般給回擊回去，也具備身為劍的其它功能。不過直接對目標攻擊時並不會產生任何傷口，而是給予相當程度的精神傷害。
  • 駐地天使
    • 除了守護小天使以外，為了一些目的，天使界會另外派大天使住在人間界，平時就以人類外型為主。
    • 因為任務性質，除非該駐地天使明講或者由事件得知其身分外，身分保密到連守護小天使都無法認出其真實身分之存在。
  • 逆パンダ
    • 一切成謎的逆色貓熊，會偽裝成各種不同的樣子出現在人間界，因外形可愛因此其形狀的公仔蠻受女性愛戴。
    • 依照找到的數量來解禁其他要素，全部找齊是100隻。
  • うさ丸
    • 行跡更為奇特的兔子，偽裝功力雖與逆パンダ差不多，然數量相對稀少要找到它不是很容易，一個章節才會找到一個。
    • 只在二代Duo才會出現，與逆パンダ一樣依照找到的數量來解禁其他要素，一共有八隻。
  • (暫無名稱)
    • 背面上半身與逆パンダ一樣，但其實是人魚的詭異生物，一個章節才會找到一個。

遊戲模式介紹編輯

  • 戰鬥模式
    • 故事主線全部完成後才能解禁，在隊伍完整的情況之下進入戰鬥模式，戰鬥對象為隨機順序出現。也可以自己(同伴)打自己(對手)。
    • 難度會越來越高，從一開始隨便就會KB(令對手無法攻擊)對手到後來對手也會防禦自己的攻擊。會一直戰鬥到主角HP用完倒下為止，每打倒一個對手會些許回復HP(要是有受傷的話，該場無傷會顯示PERFECT)。
    • 一代PLUS會出現由機器人NEO模仿二代角色的攻擊方式，名稱後面的英文字就是該角色名稱的簡寫。
    • 各代皆會依照打倒魔女的記錄來解禁其他要素。
  • 自由神判模式
    • 一樣故事主線全部完成後才能解禁，針對選擇之角色進行魔女神判。
    • 會記錄成功與失敗次數，所以還是別太紳士比較好…。
