解說[編輯]

  • 永井豪與DYNAMIC推出的超級機器人系列作品，包含動畫、漫畫等作品
    • 與《蓋特機器人》和《機動戰士鋼彈》並列為超級機器人大戰的御三家，且是御三家中最早的作品
    • 為第一部直接由人類駕駛機器的巨大機器人動畫作品[3][4]

作品列表[編輯]

TV動畫[編輯]

  • 無敵鐵金剛（マジンガーZ）
    • TV動畫第一部，1972年12月3日 - 1974年9月1日播映，全92話
      • OP：「マジンガーZ」、ED：「ぼくらのマジンガーZ（我們的無敵鐵金剛）」
      • 插曲：「Zのテーマ」、「空飛ぶマジンガーZ」
    • 和一般人理解的『超級機器人作品=半喜劇腦筋格鬥作品』不同，和敵人鬥智、開發新功能、新戰力的劇情也很多
    • 台灣上映時並不是全92話都有撥放
  • 金剛大魔神（グレートマジンガー）
    • TV動畫第二部，1974年9月8日 - 1975年9月28日播映，全56話
      • OP：「おれはグレートマジンガー」、ED：「勇者はマジンガー」
      • 插曲：「鉄也のテーマ」
    • 序盤安排無敵鐵金剛被邁錫尼帝國打至大破後由金剛大魔神接棒
    • 後期無敵鐵金剛修理及強化之後回歸戰線
    • 同樣在台灣沒有全話數撥放
  • 金剛戰神紅蓮線神（UFOロボ グレンダイザー）
    • TV動畫第三部，1975年10月5日 - 1977年2月27日播映，全74話
      • OP：「とべ! グレンダイザー」、ED：「宇宙の王者グレンダイザー」
    • 本作中甲兒從美國留學回來，無敵鐵金剛則進了博物館
    • 比起上兩部的收視率明顯下降，但在法國卻達到100%收視率的驚人紀錄
  • 太古巨魔神神魔神（ゴッドマジンガー）
    • TV動畫第四部，1984年4月15日 - 9月30日播映，全23話
      • OP：「Dreamy My Love」、ED：「時間の誘惑」
    • 以穆大陸文明的穿越劇為主題，與過往魔神系列作無關聯性的作品
  • 真無敵鐵金剛 衝擊！Z篇（真マジンガー 衝撃！Z編）
    • TV動畫第五部，2009年4月4日 - 9月26日播映，全26話
      • 前期OP：「感じてKnight」、前期ED：「Brand New World」
      • 後期OP：「守護神-The guardian」、後期ED：「強き者よ」
    • 永井豪全明星大亂鬥，可以説是第一部的重新製作

OVA[編輯]

  • 魔神凱撒（マジンカイザー）
    • 第一部OVA，2001年發行，全7話
      • OP：「FIRE WARS」、ED：「TORNADO」
      • 插曲：「マジンカイザーのテーマ」
  • 魔神凱撒 死鬥！暗黑大將軍（マジンカイザー 死闘！暗黒大将軍）
    • 第二部OVA，2003年發行，全1話
      • OP：「The Gate of The Hell」、ED：「戦士よ眠れ…」
      • 插曲：「魔神見参!!」
  • 魔神凱撒SKL（マジンカイザーSKL）
    • 第三部OVA，2011年發行，全3話
      • OP：「The ETERNAL SOLDIERS」、ED：「Juggernaut」
      • 插曲：「THE DANGER ZONE」、「LEGEND of KAISER」
    • 以太平洋島嶼奇械島的三方戰爭為主題，另一個與過往魔神系列作無關聯性的作品
    • 在機戰UX首次參戰甚至還取代了原本常駐參戰的《無敵鐵金剛》成為該作唯一魔神系作品，後來BX則跟《無敵鐵金剛》聯袂出演還有原創合體技

劇場動畫[編輯]

  • 劇場版無敵鐵金剛系列（劇場版マジンガーシリーズ）
    • 於1973至1976年間推出的六部劇場版，作品依序為：
      • 1973年7月18日上映的《マジンガーZ対デビルマン》
      • 1974年7月25日上映的《マジンガーZ対暗黒大将軍》
      • 1975年3月21日上映的《グレートマジンガー対ゲッターロボ》
      • 1975年7月26日上映的《グレートマジンガー対ゲッターロボG 空中大激突》
      • 1976年3月20日上映的《UFOロボ・グレンダイザー対グレートマジンガー》
      • 1976年7月22日上映的《グレンダイザー ゲッターロボG グレートマジンガー 決戦！大海獣》
    • 以魔神間跨作品互動為主軸，另外也有櫻多吾作與石川賢繪製的漫畫版
  • 劇場版 無敵鐵金剛 / INFINITY無限鐵金剛（劇場版 マジンガーZ / INFINITY）
    • 2018年1月13日上映，基於初代動畫版的世界觀推出的劇場版，時代設定為初代完結十年後
      • OP：「マジンガーZ」（INFINITY version）、ED：「The Last Letter」
    • 另有長田馨與小澤高廣創作的前傳漫畫《インターバルピース》描寫INFINITY事件前眾人的故事


漫畫[編輯]

  • 無敵鐵金剛（マジンガーZ）
    • 於1972年10月2日 - 1973年8月13日在少年JUMP連載
    • 實際上和TV動畫是平行發展的關係，並沒有誰才是原作之分
  • 櫻多版無敵鐵金剛（桜多版マジンガーシリーズ）
    • 由櫻多吾作擔任作畫的無敵鐵金剛漫畫版
    • 著重於動畫版各篇之間的聯繫，有時也會把被視為平行世界的劇場版劃入本篇
  • 超級機器人烈傳（スーパーロボット烈伝）
    • 石川賢作畫，集結無敵鐵金剛、金剛大魔神、金剛戰神、蓋特機器人與鋼鐵吉克等DYNAMIC系機器人的跨界作品
  • 魔神天使（マジンガーエンジェル）
    • 新名昭彥作畫，由歷代女主角領銜主演的作品
    • 多為捏他本篇劇情的輕鬆喜劇風格，也有部分永井豪作品的角色客串
  • 真無敵鐵金剛ZERO（真マジンガーZERO）
    • 《衝擊！Z篇》的衍生漫畫，田畑由秋腳本，余湖裕輝作畫
    • 以時間迴圈為主題，基本上每一次迴圈都是大量便當
      • ミネルバのなく頃に 魔神し編
      • 真マジンガー☆ZERO
      • Mazinger;Zero

其他[編輯]

  • 鐵金剛少女Z（ロボットガールズZ）
    • 由東映製作的萌え擬人化動畫作品，將收錄的機器人都萌化成美少女機娘
    • 本作是跨界作品，登場機器人不只有無敵鐵金剛系列，還收錄了像是蓋特機器人、太空魔龍ガイキング等東映曾經放映過的超級機器人動畫作品
    • 後來有遊戲化成『ロボットガールズZ ONLINE』、『ロボットガールズNEO』等作品

      • 操作系統也相當的有風格，使用汽車駕駛盤的方式操作
    • 多數人的強烈印象是在機戰中被打爆的修理費只要10元，且同蓋特機器人的巴武藏一樣是自爆指令的代表機之一
    • 補給役
    • 麻美要員
  • 龍女金剛/米妮瓦X（ミネルバX）
    • CV：菊地美香
    • 由兜十藏設計，原本打算作為無敵鐵金剛搭擋機的機體
    • 因為機械獸的攻擊導致搭擋迴路受損而失控，最後被沙也加摧毀
    • 《ZERO》起初也是巨大機體，在多次世界輪迴後變為人類體格的人造人
      • 擁有可超越光速的光子加速器，為了迴避無敵鐵金剛化為終焉魔神的未來而多次穿越時空
    • 《鐵金剛少女Z》中大家都被轉成機娘了，就他被轉成機器人少年
  • 地獄王戈登（地獄王ゴードン）
    • 地獄博士的乘機，於原作漫畫版、《魔神凱撒》與《衝擊！Z篇》中登場
    • 潛藏在地獄城內，作為地獄博士的最後王牌出現
    • 在《ZERO》有其強化型「戈登地獄（ゴードンヘル）」

金剛大魔神[編輯]

  • 金剛大魔神（グレートマジンガー）
    • CV：水瀨祈
    • 鐵也的乘機，兜劍造為了對抗邁錫尼帝國而秘密開發的機體
    • 裝甲由「超合金Z」四倍强度的「超合金New Z」構成
    • 代表招式為從手指發射電光的「金剛閃電（サンダーブレーク）」
  • 維納斯A / 新木蘭號（ビューナスA）
    • CV：藤田咲
    • 炎純的乘機，金剛大魔神的搭擋機體
  • 小天使/小鐵人（ロボットジュニア）
    • CV：本多真梨子
    • 志郎的乘機

金剛戰神[編輯]

  • 克雷飛天神紅蓮飛神（グレンダイザー）
    • CV：荒浪和沙
    • 狄克的乘機，被弗利特星奉為守護神的機器人
    • 機體可以套進稱爲スペイザー的支援機
    • 代表招式為從雙角發射電光的「宇宙雷鳴（スペースサンダー）」
  • 飛天翼號/雙航機（ダブルスペイザー）
    • CV：德井青空
    • 甲兒的乘機，在地球另外製作的克雷飛天神的支援機之一
  • 海天翼號/海航機（マリンスペイザー）
    • CV：斉藤佑圭
    • 牧葉光的乘機，在地球另外製作的克雷飛天神的支援機之一
  • 地龍號/鑽航機（ドリルスペイザー）
    • CV：白石涼子
    • 瑪莉亞的乘機，在地球另外製作的克雷飛天神的支援機之一
  • 　上述三架在遠征貝加星進行復國戰時有另外製作宇宙戰用的機型。
  • 雖然陣容豪華又具備多種變形合體，但在機戰的命運常因為魔神皇帝與真蓋特太好賣而逐漸被邊緣化，往往有魔神皇帝登場的機戰就不會出現金剛戰神系列
    • 不過最近在法國又煥發第二春，將於2023年改編成動作遊戲。

太古巨魔神[編輯]

  • 太古巨魔神（ゴッドマジンガー）
    • 穆王國的守護神
    • 在穆王國陷入危難之際甦醒與大和合而為一

魔神凱撒[編輯]

  • 魔神凱撒 / 魔神皇帝[12]（マジンカイザー）
    • 究極的無敵鐵金剛，擁有「超越神或是消滅惡魔」的強大力量
    • 最初登場的機戰F完結編設定為經蓋特線強化的無敵鐵金剛後繼機，機戰alpha改為魔神系列因為太過強大而被封印的原型機[13]，之後取消了這些設定就是單純的後繼機
    • OVA機設的腰變得有點性感
      • 在《魔神凱撒》中，不但把被捕虜兼魔改的魔神Z打成廢鐵，還失控把大魔神打趴，從此以後OVA的大魔神就像高級雜魚一般整天等魔神凱撒救
      • 其強度到達了在續作《死鬥！暗黑大將軍》中一出來就讓邁錫尼軍大亂並輕鬆幹掉一票人
        • 而其他光子力研究所的機體則是死的死殘的殘
    • 不同於過往的無敵鐵金剛，本身已具備藏在左右肩的雙劍「ショルダースライサー/カイザーブレード（肩）」 ，也能從胸口拔出大劍「カイザーブレード（胸）/ファイナルカイザーブレード」
      • 續作中因為太早出場會沒戲看所以故事中大部份時間都在宇宙格納庫等甲兒逃回日本
      • 因為以上原因被島民們視之為兜家的忠實女僕
    • 在機戰X的設定為在《衝擊！Z篇》宙斯神的指引下，跟下方的魔神皇帝G一起從可能性的光之彼岸召喚過來的最強的無敵鐵金剛的象徵，並重新用回F完結編～alpha系列的機設，所以胸口的大劍沒了
      • 根據「凱撒新星（カイザーノヴァ）」的技能動畫來推測，似乎ZERO一樣擁有7種魔神力

魔神凱撒SKL[編輯]

  • 魔神凱撒SKL（マジンカイザーSKL）
    • 海動與真上的乘機，為雙人駕駛的魔神
    • 主要以操作大劍或雙槍戰鬥，結合溫蓋兒的翅膀後可使用魔神原本的力量
  • 溫蓋兒/翼流（ウイングル）
    • 史嘉蕾的乘機，史嘉蕾死後成為由木的乘機
    • 可以將翅膀轉移到魔神凱撒SKL以解放其真正實力
    • 造型源自《惡魔人》的露奶雞腳苦命女死麗濡以及女惡魔人
  • 精神齒輪（サイコギア）
    • 八陵郭防衛隊的乘機，依照乘員有著各自的造型
  • 爆劉鬼（ばくりゅうき）
    • 基巴乘機
  • 惡靈（ガイストテレス）
    • 加蘭乘機
  • 鋼鐵凱撒（アイアンカイザー）
    • 控制奇械島重力爐的「另一個魔神」
    • 形象源自漫畫《手天童子》的同名機體
    • 原先封印在加蘭城內，被偷襲加蘭城的基巴搶來當作自己的座機

其他[編輯]

  • 無敵鐵金剛ZERO（マジンガーZERO）
    • 無敵鐵金剛在解放所有力量後化身而成的「終焉魔神」
    • 共計擁有7種的魔神力。其中擁有觀測平行世界的「高次預測」能力，以及干涉機率的「因果律兵器」的能力
    • 一度被寺田評為「太強而難以參戰」的機體，到機戰V終於參戰，也確實展現了神擋殺神的強大實力
    • 在島民的創作下變成了兜家的病嬌太妹
    • V可以爽爽用，X變成敵方還只能用一關
  • 大魔神凱撒（グレートマジンカイザー）
    • 《ZERO》登場的鐵也的後繼機，金剛大魔神的魔神力全開後化身而成
    • 超越「偉大的勇者」的「偉大的皇」，也是唯一能ZERO對抗的機體
    • 機戰沒有採用，而是設計原創的魔神皇帝G
  • 魔神皇帝G（マジンエンペラーG）
    • 機戰V原創鐵也的後繼機，為金剛大魔神藉由蓋特線強化而成的型態
    • 跟OVA版魔神凱撒一樣，本身已具備藏在大腿兩側的雙劍，也能召喚出大劍「エンペラーソード」
    • 在機戰X的設定為在《衝擊！Z篇》宙斯神的指引下，跟上方的魔神凱撒一起從可能性的光之彼岸召喚過來的最強的無敵鐵金剛的象徵
      • 與魔神凱撒有合體技「魔神雙皇擊」
  • INFINITY（インフィニティ）
    • 《INFINITY》劇場版登場的「第三魔神」
    • 被發現沈睡在富士山底下，全長約600m
    • 因其擁有改寫世界的力量而被復活的地獄博士盯上
    • 似乎是上古文明製造的魔神，也有一說是來自平行世界的魔神
      • 機戰T則設定成邁錫尼為了侵略瑟費洛等其他世界而開發的魔神，也被設定成魔法騎士一眾魔神的原型機
    • T因為是敵方專用而被砍招，只剩全飛彈發射（MAP兵器）及ブレストインフェルノ可以用

其他逸話[編輯]

  • 雖然《無敵鐵金剛》有其原文的主題曲，但於1978、1985年台灣上映時因當時政府政策不得出現日本文化而必需採用中文樂曲與中文人名，所以除了角色名稱被大幅度更動外（見上），主題曲也整個被更換
    • 金剛大魔神與金剛戰神在台灣首播時也作同樣處理，而前者更因為華視電視台亂搞讓人名變得更加混亂
    • 不過因為動畫在當時的台灣造成巨大風潮，台灣自行創作的中文主題曲就變成當年觀眾的既有印象，成為不少人的經典回憶
    • 因此創作《無敵鐵金剛》台灣中文主題曲的作家汪石泉於2018年去世時，在告別式現場眾人起身合唱中文主題曲用以紀念
