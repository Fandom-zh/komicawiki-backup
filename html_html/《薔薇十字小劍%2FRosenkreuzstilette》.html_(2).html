Rosenkreuzstilette

薔薇十字小劍（Rosenkreuzstilette/ローゼンクロイツスティレッテ）
注意！此遊戲為同人作品，與洛克人原創公司Capcom並無關聯。

　


目錄

簡介編輯

  • 由[erka:es]製作的同人遊戲
  • 有街機模式（無劇情對話）和故事模式可供選擇
  • Boss血量剩一半時會變紅血
    • 此狀態下攻擊模式會改變
  • Game over畫面才是重點（無誤）
    • 雖然本篇的劇情相當嚴肅，不過Game over畫面總是能夠涵蓋許多動漫的捏他
  • 兩部作品先後在2017/2/3與2017/10/17登上Steam平台。

Rosenkreuzstilette登場人物編輯

  • RKS001 スピリティア・ローゼンベルク
     
    • 本作主角
    • 魔力："魂の銃" ゼーレゲヴェーア Mega Buster
    • 為了阻止ゼッペリン伯爵的政變被迫和同伴戰鬥
    • 神尾觀鈴
    • \ Y /
    • グローラ模式下成為頭目，血量為其他人的兩倍
    • 在續作《Freudenstachel》中被黑十字軍綁架
  • リリィ
     
    • 住在黑森林的森之妖精，大約只有手掌大小
  • RKS002 フロイディア・ノイヴァール
     
    • 魔力："歓喜の棘" フロイデンシュタッヘル Needle Cannon
    • 八大頭目中兼任序關頭目的特殊案例
      • 因此在八大頭目戰中一開始就紅血
      • 終盤卻兼任布魯斯
    • 雖然是冰關，卻充滿快速人關卡的即死雷射
      • 相剋武器為時間暫停，果然是快速人
    • 續作《Freudenstachel》的主角，為了救出スピリティア而展開整合RKS以及討伐黑十字軍的旅程
  • RKS003 ツォルネ・ゼッペリン
     
    • 魔力："怒りの爆弾" ツォルネスボムベ Hyper Bomb強化版
    • ゼッペリン伯爵的養女，無藥可救的父控
    • 攻擊方式基本上致敬了轟炸超人
      • 續作《Freudenstachel》還有特殊平局的演出，在使用必殺技時畫面會被磚塊填滿，完全填滿後會雙殺
    • 笨蛋
  • RKS004 トラウアーレ・ヴレーデ
     
    • 魔力："なげきのよろい" クラーゲハルニッシュAir Shooter強化版
      • 因此這位大概又是打不倒的(誤)
      • 續作《Freudenstachel》攻擊模式有修改，變得比較接近陀螺人
    • 不管髮型還是武器都和某妖將非常相似
    • 與ツォルネ是閨蜜，在ツォルネ先被打敗後找她還會病嬌化
  • RKS005 ルステ・トイバー
     
    • 魔力："快楽の息吹" ルストアーテム Metal Blade
    • 挑戰之前請確認這不是糟糕化的某妹樣
    • 八大逆襲戰時可用二發ルストアーテムKO，你是金屬人嗎
    • 在續作《Freudenstachel》中戴上面具並自稱正義のヒーロー，不過模樣實在太好認了
  • RKS006 グローラ・ザイファルト
     
    • 魔力："うらみのつるぎ" グロールシュヴェート 洛克人Zero的Z Saber
    • 在劇情定位上算是苦勞人，兩代都是被怨念控制而暴走
    • 隱藏可用角色
  • RKS007 ズィヒテ・マイスター
     
    • 魔力："予定された未来" ゲプランテツークンフト The World，Time Stopper
    • 時停能力+飛刀武器+時鐘塔舞台，其實你是DIO的女兒吧
      • 在續作《Freudenstachel》中還出現了疑似致敬壓路機的壓石頭攻擊
  • RKS008 リーベア・パレーシュ
     
    • 魔力："愛の嵐" リーベスシュトゥルム Rain Flush
    • 重度兄控
    • 攻擊模式酷似Toadman
      • 攻擊模式在續作《Freudenstachel》中變成Magnetman，且代替フロイディア成為冰關的頭目
      • 其實和フロイディア交手的リーベア是ホムンクルス，本尊只在劇情中登場
    • 擅長製造萌音
  • RKS009 シュヴェーア＝ムータ・カサソラ・メルクル
     
    • 魔力："心の壁" ガイスターヴァント Skull Barrier
    • 稱號為RKSの孔明，所以關卡陷阱特別多
      • 在續作《Freudenstachel》中的陷阱更能夠使玩家跳回選關畫面...
    • 頭上的花枝才是本體
  • ミヒャエル・ゼッペリン伯爵
     
    • 帝國貴族
    • 為了避免擁有魔力的人遭迫害而創立RKS
    • 在續作《Freudenstachel》中以類似ガンマ的姿態復活，卻也造成和ガンマ一樣可被Top Spin一擊死
  • イーリス・ゼッペリン
    • ゼッペリン伯爵的女兒，於序幕關被抓
    • 本作的最終頭目，魔力使ローゼンクロイツ的轉世，甚至也像威利博士一般會搭乘戰機，被打爆後也會土下座然後趁スピリティア不注意時落跑
    • 在續作《Freudenstachel》中繼續擔任最終頭目，被打爆後還是會土下座不過仍被フロイディア冰封
  • ライムント・ザイファルト
    • グローラ的祖父暨劍術指導
