泛用人型角色編輯

戰士編輯

 
  • 系列中一代即登場的職業，擅長武器是劍、斧，部分作品也有拿槍。
  • 蓬亂的頭髮上綁了髮帶，赤裸著上身。
  • HP和ATK很高。
  • 在各作一開始就可以入隊，不過戰力很弱，到後期沒什麼存在感。

女戰士編輯

  • 系列中一代即登場的職業，擅長武器除共通的剣，也拿槍與弓。

魔法師編輯

 
  • 系列中一代即登場的職業，擅長武器是杖。
  • 穿著連衣裙，綁著髮帶的女性，天生就具有魔法天分。
    • 髮帶由魔法協會贈送，能夠更好的控制魔力、自動接收協會資訊，能夠方便學習的環境造就她性格變得天真爛漫。
  • 外觀顏色和屬性有關，宣傳圖常見的是紅色，即紅魔法師(火屬性)。
    • 其餘為藍色的藍魔法師(水屬性)、綠色的魔法師(風屬性)、金色的星魔法師(星屬性)、紫色的虹魔法師(星屬性)、黑色的銀河魔法師(星屬性)。
  • 和骷髏的能力都十分相近，差別在魔法師的HIT和SP較高。
  • ぽこん

骷髏編輯

 
  • 系列中一代即登場的職業，擅長武器是杖。
  • 戴著人臉兜帽及眼鏡的男性。
    • 只有在魔法協會的兢爭中脫穎而出的人才能獲得特殊的服飾，辛苦的過程讓個性變得較認真嚴肅。
    • 一代時沒有穿上衣，頭部完全被兜帽蓋住。
  • 和魔法師的能力都十分相近，差別在骷髏的SPD較高，SP雖少但技能傷害大。

格鬥家編輯

  • 系列中一代即登場的職業，擅長武器不用說是拳頭，但三代以後換原田雄一負責後外表有不少變化。

女格鬥家編輯

  • 系列中一代即登場的職業，擅長武器與上者一樣是拳頭，但三代以後換原田雄一負責後外表有不少變化。

女僧侶/女治療師編輯

 
  • 系列中一代即登場的職業，負責治療的職業，擅長用杖、弓。
  • 身著全身長袍，露出乳溝和側大腿，手腕處有著枷鎖，總是閉著眼睛。
    • 設定上原先是墮天使，來到魔界傳授治療的方法，被充滿惡意的惡魔們拒絕，直到他們發現治療魔法是必須的。
  • 重要能力是RES，其越高就越能提高回復能力。

僧侶/治療師編輯

  • 系列中一代即登場的職業，擅長武器也一樣是杖、弓。與女治療師登場次數相比少了兩次(二、四代)。
  • 外觀與女治療師相似，不過認為治療是種罪惡，在衣服內側藏有尖刺，用於懲罰自己。
    • 二代的形象變得可愛：雙馬尾、遮眼的長瀏海、戴著耳環，但很困擾常被誤認為女性。
  • INT較女治療師高，補助能力略弱但也稍微增強攻擊能力。

盜賊編輯

 
  • 系列中一代即登場的職業，能力值皆很差，還算擅長的武器是銃、弓。
  • 一代時的外觀是男性，之後就只有女性的盜賊登場。
    • 戴著貓耳兜帽和護目鏡；穿露臍上衣，似貓掌的鞋子。
  • 唯一可以在戰鬥中偷竊敵人身上道具的職業。
  • 不單只是道具這種實際的存在，厲害的盜賊據說還能偷走人的情感與能力。
  • 人氣很高，曾在《魔界戰記4》的泛用角色投票拿下第一名。

忍者編輯

 
  • 系列中一代即登場的職業，提升戰士或格鬥家的等級來解鎖，擅長武器為拳頭、劍。
  • 全身幾乎被包覆著，行動隱蔽的暗殺者，魔界的領導人們非常害怕他。
  • 魔能力會因為攻擊方向來提升能力，作為敵人時最好注意一下。
  • 每代皆登場，但人氣輸女忍一大截。

女忍者編輯

  • 從二代開始能自選的職業之一，擅長武器與上者一樣是拳頭、劍，但登場次數比上者少了2次(D2、六代)。

弓箭手編輯

 
  • 系列中一代即登場的職業，擅長的武器不用說就是弓
  • 金髮螺旋雙馬尾，個性溫柔，一族只有女性，住在遙遠的雪山中，天生對風向的感知非常好。
    • 胸部因為會妨礙拉弓而完全退化了。
  • 雖然設定提到討厭用弓以外的武器，但HIT很高，用銃也是不錯的選擇。
  • 僅三代有出現類似的男性職業遊俠，和弓箭手的相處已久但關係不好。

武士編輯

  • 僅二、三代登場的職業。

女武士編輯

 
  • 系列中一代即登場的職業，提升戰士和武術家的等級來解鎖，主要擅長的武器為劍，其他作品也能拿槍、斧、弓。
    • 二代僅在PSP版可用，還算是每代都有的職業。
  • 看起來像巫女，裹著纏胸布，胸部尺寸為貧乳。
  • 固有技能「妖刀紅死垂」，擊破越多敵人能讓威力變更強，曾有BUG可以無限提升。
  • 在4Return的宣傳廣告和紅魔法師唱雙簧，還被轉生成邪龍族。

魔法劍士編輯

 
  • 系列中一代即登場的職業，提高戰士和魔法使的等級自動解鎖，擅長武器為剣、杖。
  • 金色短髮，使用大劍的巨乳女性，和教授是好友。
  • 一代的各項能力都很低，基本上只是為了解鎖天使兵才會特別練。
    • 後期有改強，INT和RES很高。
  • 雖然是惡魔但為人正直且不違背誓言，從基地對話中可以看出來是個禮貌的人。

天使兵(女)編輯

 
  • 系列中一代即登場的職業，提升魔法劍士、女治療師、弓箭手的等級來解鎖，擅長劍、槍、弓、杖。
    • 三代後武器熟練度皆被削弱，建議拿弓與杖；四代後變成飛行單位。
  • 來自天界的馬尾辮天使，有點瞧不起惡魔，為了自願修行才來到魔界。看服裝也看不出來的隱藏巨乳。
  • 男性天使兵僅出現於D2、五代，考量到要角B的形象可能有BL傾向。

槍手編輯

 
  • 系列中一代即登場的職業，當任一角色的銃熟練度足夠後能於議會解鎖。擅長武器不用說是銃。
  • 頂著大牛仔帽，穿著披風，腰側可看到兩個槍套。
  • 擁有稱為邪眼的能力，能從遠處捕捉敵人的身影。
  • 雖然設定上是孤獨的雇傭兵，但攻擊容易觸發連攜，建議還是把他和其他角色放在一起。

女槍手編輯

  • 僅三代登場職業之一，擅長武器與上者一樣是銃，但再也沒登場於後續作品。
  • 頂著大紅牛仔帽，穿著紅比基尼與熱短褲，腰側可看到兩個槍套，眼鏡巨乳少女。

重騎士編輯

 
  • 二代開始登場的職業，當任一角色的槍熟練度足夠後能於議會解鎖，擅長武器為斧頭、槍，但五、六代缺席。
  • 用頸圈遮著嘴，身上穿著大量護具。
    • 理所當然的SPD和移動力非常低
    • 不過投擲距離有六格，為全職業最遠，配合丟出高移動力的角色能盡速佔據地圖有利位置。

女重騎士編輯

  • 從D2開始能自選的職業之一，擅長武器與上者一樣是斧頭、槍，背景設定說明因重騎士性別比例是女騎士佔多數並大於男騎士。
  • 身穿重鎧的深紫長髮爆乳女性。

應援師編輯

  • 僅三代登場職業之一，但再也沒登場於後續作品。
  • 超長淡紫色雙馬尾，穿著比基尼的巨乳女性。

教授編輯

 
  • 四代開始出現的職業，當女僧侶和陰陽師達到20級後可以解鎖，擅長武器是杖、銃。
  • 金中馬尾，有戴眼鏡的巨乳女性，熱衷探索未知的理論，和魔法劍士是好友。
  • 技能多用來提升能力，甚至連投擲利和移動力也可以，應用較廣的支援職業。
  • 魔能力可以讓隊伍獲得的經驗值增加20%，非常推薦練來放在隊伍中加速升級效率。

陰陽師編輯

  • 僅四代、D2登場職業之一，性能與賢者相同。

人造人編輯

  • 僅四代登場職業之一，性能與前三作登場的魔人相同，取代其地位。

海盜編輯

  • 僅五代登場職業之一，標準海盜風型男。

女賢者編輯

  • 僅五代登場職業之一，水藍色中髮也留長馬尾的巨乳女性。

暗黑騎士編輯

  • 僅五代登場職業之一。

魔界鬥士編輯

  • 僅五代登場職業之一，美式摔角手壯漢。

超能力者編輯

  • 六代登場職業，身穿軍服的少年卻性格很中二[1]。
  • 會按照能力差異來決定工作職位並就職 ，擅長武器是杖、銃。

機動戰姬編輯

  • 六代登場職業，由超魔導與超科學結合製造的巨乳女性人型機器人。
  • 能全部拿各武器，屬平均型。

泛用魔物型角色編輯

夜魔族編輯

 
  • 系列中一代即登場的職業，頭上長有犄角，有大片的蝠翼，尾巴的末端是愛心型，胸部尺寸估計是巨乳。
    • 前兩代原本服裝十分性感又暴露(由光之聖女傳說、馬爾王國的人設 野村良治負責)，但三代換原田雄一後服裝變得很保守，但依舊很性感
  • 被大部分的女性惡魔討厭著同族的艾特娜也是。但和貓娘族相處不錯。

貓娘族編輯

 
  • 系列中一代即登場的職業，頭上有貓耳，手與腳部是貓類的貓爪與掌，尾巴則是兩條。胸部尺寸比夜魔族還大，僅用少量的布蓋住。
    • 全族皆是武術達人，和夜魔族相處不錯。
  • HP和ATK高，魔變換的類型是拳，相當有武術的感覺。

屍族編輯

 
  • 系列中一代即登場的職業，外觀默認是綠色，身體腐爛有缺損。
  • 又稱殭屍，在魔界被復活的屍體都能算是屍族，嚴格來說並不能算是一個種族。
    • 因為早已死亡而有多處殘缺，特別是大腦，系列中屍族的思考邏輯都異於常人。
  • DEF較低，HP較高，善於抵抗異常狀態。

妖花族編輯

 
  • 於二代登場的職業，上半身是綠色皮膚的人形，腰部長著一朵花，腿似乎是根與莖。
  • 沒有雌雄的區別，說話語氣較陰柔。
    • 早期身體全裸，3代開始有葉子遮住乳頭。
  • ATK和RES高，生存能力弱，建議是作為支援角色在後方放補助技能。

龍王族編輯

  • 系列中一代即登場的職業，但登場次數很平均(2代原版沒登場，PSP移植版以炎帝族的名稱登場)

邪龍族編輯

 
  • 系列中一代即登場的職業，與上者登場次數相比較多
  • 默認外觀是象徵風屬性的綠色，站立著的西方龍。
    • 是巨型魔物之一，速度也不快，因此容易被攻擊，不過HP、ATK、RES皆高，還算容易生存的單位。


吸血鬼族編輯

  • 僅一代登場職業，外表是身穿貴族服飾的高大男性，因為太多特徵鮮明外表的職業顯得沒存在感，再也沒登場於後續作品。

魔翔族編輯

 
  • 二代開始出現的職業，透過推進主線關卡來解鎖，設計來源自UMA生物天蛾人。
  • 設定提到會散布鱗粉使其他種族染上疾病，技能也以各種病症命名。
  • 移動範圍高達6格，加上能夠飛行，機動性非常高。
    • 不過生命和防禦都很低，容易被秒殺。
    • 魔能力能在生命較低時再增加3格移動力[2]，速刷道具界的必備職業。

銃魔神族編輯

 
  • 二代開始出現的職業，有三隻眼睛，右手已化為槍砲，是來自其他次元的魔神。
  • 基礎能力都很好，但移動力僅有三格。
  • 每個階級的職稱都源自十二宮對應的天使，比如一階時的名字バキエル，為雙魚座天使。
  • 該種族的議員總是在議會中喝醉。
  • 可能來自毀滅戰士和德軍總部的世界。

兔鬥族編輯

  • 僅五代登場職業之一，外表不用說是擬人化的兔子，但為防禦敵方襲擊而會集結一同戰鬥的部隊，居民們滿懷敬意稱呼他們為：兔鬥。

九尾族編輯

  • 僅五代登場職業之一，外表是九條尾巴的狐娘。

詛眼族/咒眼族編輯

  • 六代登場職業，擁有豐滿身材(胸部尺寸比夜魔族、貓娘族更大)的牛系魔物。
    • 可是性格卻沒自信，多半是膽小鬼居多，不小心靠近的話還會被痛扁一頓。
