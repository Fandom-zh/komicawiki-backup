巴其海賊團

巴其海賊團→海賊派遣公司「巴其宅急便」→十字公會

目錄

解說編輯

  • 主角魯夫出海後遇上的第一個強敵，組織看起來就像是個馬戲團。
  • 原活躍於東方藍，為了向魯夫報仇而進入偉大的航道。
  • 頂上戰爭後實力大增，並當上七武海。
  • 巴其成為七武海後把海賊團轉營為「巴其宅急便」，主要業務為外判傭兵作為戰力給其他勢力。
  • 和之國篇後，巴其成為「四皇」之一，招收了曾為七武海的鷹眼和沙鱷並成立「十字公會」，實力甚至強橫到可以反通緝海軍。
    • 十字公會實質上仍由兩位前七武海掌管，只是因為巴其的手下製作文宣把他放在中間才讓外人誤會；二人還為此痛揍了巴其一頓（無誤）
  • 基本資料：
  • 船長/社長：巴其
    海賊船：巨頭號
    同盟關係：亞爾麗塔海賊團
    總懸賞金：最少1億4400萬貝里

船長/社長編輯

「千兩小丑」巴其（バギー，Buggy）編輯

  • CV：千葉繁／木村昴（少年時期）
  • 現今懸賞金：1500萬貝里→ 31億8900萬貝里
  • 「巴其宅急便」公司社長，兩年前在東海時為巴其海賊團船長，年輕時是羅傑海賊團的見習船員。
    • 當他們來到某座城鎮時因生病發高燒而無法跟隨羅傑前往拉乎德爾
  • 「四分五裂果實」能力者，能讓身體自由分裂並免疫斬擊系的攻擊[1]，分離的部位還能自由飛行，但操控範圍有限且雙腳必須著地
  • 有著一個小丑的大紅鼻子，但本人非常忌諱別人談及他的鼻子，甚至只是講到紅色都可能爆氣，也因為如此常常發狂及傷及無辜。
  • 以尋找寶藏及虐待弱者為樂，是個集自私、無情、貪婪於一身的海賊。曾因一時氣憤，而使用「巴其砲彈」濫炸城鎮。
    • 後來在魯夫把「巴其砲彈」以「橡膠氣球」彈回的時候，巴其竟以兩名手下當作擋箭牌來保護自己。
    • 貪財的程度比娜美更嚴重，因此對藏寶圖之類的話題沒有抵抗力
  • 巴其完全不理解羅傑海賊團內的成員為什麼對寶物沒有興趣，並認為大家根本不知道「海賊」的意思。他認為，海賊的本色就是「搶奪財富」，故每與敵船開戰都顯得十分亢奮和積極。
    • 不過當碰上敵眾我寡的戰局時又會縮回去
  • 某天，羅傑海賊團從敵船劫來一顆惡魔果實，在得悉果實可賣到1億貝里後，巴其便使計將它據為己有；計劃在事成後依靠賣果實得到的錢和藏寶圖上的寶物，成立最強的海賊團，搶奪更多的財寶。
    • 豈料，在他躲於暗角洋洋得意時，傑克不斷與他搭話，情急之下便把惡魔果實藏在口中，但竟不慎吞下了，還把藏寶圖丟進海中。
    • 他的發財大計瞬間化成泡影，成立海賊團出海的計劃也因而延後了10年；他將所有結果歸咎於傑克，因此對傑克恨之入骨。
  • 魯夫出海後第一個強敵，在戰鬥中利用果實能力重創了索隆，但後來兩人再戰時卻被娜美偷走分裂的身體而變成迷你巴其，被打飛至海平面彼端好討厭的感覺啊
    • 在迷你狀態下經歷過許多劫難，最後被「鐵棒」亞爾麗塔所救，由於兩人皆視魯夫為敵人，一拍即合便成為拍檔。
    • 之後在羅格鎮抓住魯夫並打算將其斬首示眾，但卻因為突如其來的大雷雨而失敗
    • 雖然追著魯夫進入偉大的航道，但卻執著於追尋「約翰船長的財寶」而到處探險，最後因為誤闖海軍基地被捕，最後被關進推進城第一層
  • 利用果實能力逃獄後遇上潛入的魯夫，受對方牽連一路掉到第四層，最後跟Mr.3聯手展開逃獄
    • 成功回到第二層後釋放眾多囚犯，被歌頌為「救世主巴其」，但隨後就被麥哲倫鎮壓並困住
    • 魯夫等人從底層殺出後再度會合，因為目標一致而組成共同戰線，並隨吉貝爾等人前去搶奪軍艦
  • 在前往馬林福特的途中過去的身分被海軍起底，在囚犯們大肆吹捧下腦袋發熱，決定前往頂上戰爭
    • 然而實際到達現場後反被嚇得半死，並嚷嚷著想要回家
  • 雖然實力不強，但靠著羅傑船員的光環以及頂上戰爭時的舉止贏得不少推進城囚犯的支持，頂上戰爭後勢力大增，因此被世界政府選為新的七武海
  • 成為七武海後，巴其利用其權勢在「新世界」卡萊巴利島創辦海賊派遣公司「巴其宅急便」，但意外獲知原本隸屬其麾下的S級傭兵兼頭牌：哈爾汀等5名巨人族宣誓加入草帽海賊團旗下而感到震驚。
  • 政府發布廢除王下七武海的制度的消息後，面對海軍中將史鐵雷斯率領海艦的圍捕，巴其認為海軍非常無情無義，並鼓勵部下全力反擊，自己卻想找機會離開。
  • 和之國篇後，巴其和魯夫同時成為「四皇」之一，和「紅髮」傑克、「黑鬍子」馬歇爾・D・汀奇平起平坐

幹部編輯

「馴獸師」摩奇（モージ，Mohji）編輯

  • CV：宗矢樹頼
  • 原巴其海賊團副船長兼馴獸師，總是騎着心愛的獅子利基。
  • 無惡魔果實能力，主要武器是鞭子，在戰鬥時以操控猛獸作為主要攻擊。
  • 轉營成為「巴其宅急便」幹部。

「特技」卡巴吉（カバジ，Cabaji）編輯

  • CV：遠藤守哉
  • 原巴其海賊團參謀長。
  • 無惡魔果實能力，主要武器是長劍，善用各式特技，如騎單輪車、吐火、耍陀螺。只要騎上單輪車，實力立刻倍增，甚至還能攀爬上牆或跳至高空。
  • 轉營成為「巴其宅急便」幹部。

「鐵棒」亞爾麗塔（アルビダ，Alvida）編輯

  • CV：松岡洋子
  • 原亞爾麗塔海賊團的船長。
  • 被魯夫打敗後吃下惡魔果實「滑嫩果實」，並遇上同被魯夫打敗的巴其，兩人自此成為同盟。
    • 「滑嫩果實」能力者的全身肌膚就會變得滑嫩，身型也由極度肥胖變成苗條美女。
    • 皮膚完全沒有摩擦力，不但能令刀子滑落，也可以議身體像溜冰板一樣高速移動。
  • 巴其被抓後接管其海賊團。
  • 轉營成為「巴其宅急便」幹部。

「黑金」加爾迪諾（ギャルディーノ，Galdino）編輯

  • CV：檜山修之
  • 原巴洛克華克高級特務，代號「Mr.3」。巴洛克華克被打敗後被關進推進城。
  • 推進城逃獄事件中遇上巴其，之後更成為「巴其宅急便」幹部。
  • 「融化果實」能力者，身上可任意產生蠟燭黏液，形狀千變變化，凝固後更硬如鋼鐵，可封鎖敵人行動、製作蠟像或牢房的鑰匙等。

傭兵編輯

「海賊傭兵」哈爾汀（ハイルディン，Hajrudin）編輯

  • 原為「巴其宅急便」旗下的優秀雇傭兵，多雷斯羅薩戰役結束後連同其他四名巨人提出辭呈，並加入草帽海賊團的旗下。
  • 詳見新巨兵海賊團

    • 協助魯夫逃離LEVEL 6，並趁機逃獄
