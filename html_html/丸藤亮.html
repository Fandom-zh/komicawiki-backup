丸藤亮

丸藤亮/丸（まる）藤（ふじ）亮（りょう）
這是一個編輯中的詞條，歡迎有興趣的SM愛好決鬥者加以補完。
以下含有部分劇情，不想被劇透或出錯融合卡自爆或被裏電子流交尾的話請不要繼續看下去

基本資料編輯

  • CV：前田剛

  • 稱號：凱薩→地獄凱撒
  • 杜馬三槍手的成員之一

能力編輯

  • 電子流堆牌，可以起手抽出3張電子龍、融合以及力量焊接
  • 抽牌抽第二張
 
  • 給予對手實體傷害
  • 飛鳥文化アタック

牌組編輯

 
  • 第一期使用「電子流」牌組
    • 以《電子龍（サイバー・ドラゴン）》及《電子終焉龍（サイバー・エンド・ドラゴン）》為主，並使用各種支援卡多次召喚終焉龍
  • 成為地獄凱薩並戰勝鮫島校長取得裡電子流牌組後變更為《電子龍》《電子暗黑（サイバー・ダーク）》的混合牌組
    • 沒有神抽的話就是超嚴重事故牌組，你要怎麼打這牌組啊[1]

經歷編輯

  • TURN-1初次登場，與明日香一起觀看十代與庫洛諾斯的決鬥
  • TURN-8與十代決鬥，以電子終焉龍獲得壓倒性勝利
  • TURN-19觀看十代與神樂坂決鬥，並稱讚神樂坂運用遊戲牌組的實力
  • TURN-20受到早乙女禮追求，以對方是小學生為由拒絕看來不是蘿莉控
  • TURN-30發現Darkness面具底下是吹雪時大吃一驚
  • TURN-32對陣Seven Stars成員的卡蜜拉，因翔被當作人質而敗陣
  • TURN-51~52與十代展開畢業決鬥，最後平手
  • TURN-57輸給艾德・菲尼克斯，從此一蹶不振
  • TURN-65時已無法立足於業界，只好轉向地下決鬥場
    • 在地下決鬥場與ラフェールマッドドッグ犬飼的決鬥中認清自身對勝利的渴求，因而以地獄凱薩之名重獲新生
  • TURN-83與電子流師範兼決鬥學院校長「鮫島」對決，因而獲得裏電子流牌組，並接受鮫島邀請參加GX大賽
  • TURN-89與化為Darkness的吹雪對決，第一次施展電子流式卡片防禦能量牆和介紹傳統技能電子交尾裏電子流
  • TURN-95傳承SM決鬥與翔決鬥
  • TURN-102與早乙女禮重逢，以對方決鬥的理由太過幼稚為由拒絕挑戰
    • 同時放棄GX大賽決賽資格
  • TURN-128為了打開次元裂縫而與分校學生約翰・安德森決鬥
  • TURN-146與無法使用融合的十代決鬥，因身體因素而中斷
  • TURN-147~148與尤貝爾附身的約翰・安德森決鬥，最後在召喚出電子終焉龍時心臟病發而倒下，並找回當初喪失的尊嚴
    • 不過在霸王篇結束後卻連同之前便當的同伴們吐便當了
  • TURN-163與精神流決鬥者「豬爪誠」對決，因身體因素（ry
  • TURN-164觀看繼承裡電子流牌組的翔與豬爪誠決戰然後把牌組丟給了弟弟
  • SD26後帶著當時沒大用處的超量怪回來了，然後到了904轉身成為GX最可怕的決鬥者

其它編輯

《Tag Force》系列關聯編輯

  • 因為AI不知變通，作為電腦玩家的凱薩總會作出一些驚人表現：
    • 利用《未來融合》召喚《奇美拉聚合終焉龍》後因效果自爆，正確用法應該是利用《超載融合》將送入墓地的怪獸移除融合
    • 在生命值所剩無幾時使用《力量焊接》，卻無法在該回合擊敗對手造成自滅
    • 不管該回合是否準備進行融合召喚，只要蓋卡有《電子融合支援》就一定會發動，導致自己白扣一半生命值雙打千萬別跟他一起

Dule Links相關編輯

  • 當融合召喚成功《奇美拉聚合終焉龍》時，角色台詞語音一定是「這是爲了活下去，我所做的最後的掙扎」
    • 雖然原作中這個台詞令人印象深刻，但是遊戯中經常會出現需要融合召喚複數隻《奇美拉聚合終焉龍》的情況，所以很容易變成一局内「掙扎」好幾次的笑點
    • 又或者，常常在先手還未損失任何LP的情況下召喚出《奇美拉聚合終焉龍》，於是又有了「滿血就開始掙扎」的笑點
    • 配合只要支付生命值時動不動都會叫一下的語音，「掙扎」成了凱撒亮和電子龍的代名詞
