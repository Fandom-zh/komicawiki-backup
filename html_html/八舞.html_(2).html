略歷編輯

八舞（やまい）編輯

  • 小說第五卷中登場的雙子精靈，識別名為Berserk
  • 二人的外表均擁有橘色的頭髮和水銀般的眼睛
  • 在世界各地上空現界的精靈，原本是同一個精靈。卻在某次的現界中分裂為耶俱矢與夕弦
  • 她們引發的空間震遠高於十香，但現界的地方大多位於空中而沒有製造人命傷亡，不過之後又會一邊互相戰鬥一邊高速移動，所經之處都會如同颶風過境般的破壞殆盡，還多次被一般民眾目擊，使Ratatoskr跟AST都把她們指定為優先目標，但由於她們行縱飄忽而無從下手。
  • 在遇見士道前，二人為了爭論誰是真正的八舞而比賽了99場。[12]
  • 二人在或美島附近進行第100合比賽的時候，巧遇正在修學旅行的士道等人，於是耶倶矢突發奇想的要以「誰能先得到士道的心」為考驗魅力的最後決戰項目。
  • 事實上，二人在心底均相當重視對方，希望對方能繼續存活。個別要求做為裁判的士道要選擇對方，而在士道與十香討論時不小心讓雙方聽到各自的真正心意，為此對對方憤怒的兩人決議直接要戰到一方倒下來分勝負。
  • 在士道的幫助下，二人互相吐露想一起活下去的真心話後和解。之後自願被士道封印靈力
  • 被封印靈力後，二人宣稱士道為二人的「共有財產」
  • 事後二人在令音的安排下，以孿生姊妹的身分就讀來禪高中的2年3班。也即是岡峰珠惠的堂妹，岡峰美紀惠的班別
  • 此後二人的關係猶如戀人般親密
    • 只要兩人在一起，在封印後的精靈中精神的安定性是最高的
  • 二人一同住在精靈公寓中的8樓的802室
  • 在第十卷與十香和美九遭到裝備DEM顯現裝置的折紙攻擊，險些被折紙殺害。折紙精靈化后在十香的掩護下帶著負傷的美九逃走
  • 十一卷士道改變世界線后，忘記了與舊世界折紙相關的記憶
  • 在士道被反轉折紙攻擊時和其他精靈出手相助。折紙被封印后，取回舊世界有關折紙的記憶
  • 原本是雙胞胎，但其中一名在母親腹中尚未出生時已夭折，殘餘細胞被整合入存活的胚胎，而倖存的風待八舞以攜帶已夭折的姊妹的部分細胞的狀態[13]下出生
  • 在接受澪的靈結晶後分裂成兩個人。

八舞耶倶矢（やまい かぐや）編輯

  • 外表是把長髮盤起來，體態較為纖細的少女但是動畫版身材被修正成和夕弦近似的比例
  • 中二病，喜歡取招式名，尤好德語
  • 在未被封印之前，會以戲劇般的措辭說話，所以會由夕弦充當翻譯向別人解釋說話的含意。
    • 被封印後的說話方式變得比較正常，但時不時的還是會舊病復發。
    • 事實上在夕弦不在的情況下，連琴里都壓制不住她的電波。
  • 擅長的樂器為爵士鼓
  • 不擅長心理遊戲，因為藏不住表情。
  • 在或美島分組時，被分配到十香、亞衣、麻衣、美衣的一組
  • 大概會擁有邪王真眼
    • 呼呼，你肯定是六花吧
  • 折紙被封印后，取回舊世界有關折紙的記憶
  • 十四卷與一眾精靈被吸入【幻書館】Ashufiriya中，扮演的故事角色為《糖果屋》中的漢賽爾。之後成功逃出。

八舞夕弦（やまい ゆづる）編輯

  • 外表是把長髮編成三股辮，體態較為豐滿，眼睛總像沒精神般的半開的少女
  • 個性與耶倶矢相反，性格較為冷靜溫馴
  • 擅長的樂器為電貝斯
  • 說話時會先說出自己的反應的兩字詞語
  • 在或美島分組時，被分配到折紙的一組，曾請教同組的折紙誘惑男人的方法，敬稱她為折紙大師
    • 所以比起在感情表達上比較孩子氣的耶倶矢，夕弦比較直白而開放
    • 對應愛看中二病相關書籍的耶俱矢，會在背地里看一些成人雜誌做參考
    • 一切都是胸部的錯
  • 十四卷與一眾精靈被吸入【幻書館】Ashufiriya中，扮演的故事角色為《糖果屋》中的葛麗塔。之後成功逃出。

  • 除智力外，其他屬性的數值平均且皆有高水準
    • 而敏捷的數值更是目前登場的精靈中最高的
    • 明明是一個精靈拆成兩個，能力值卻比其他精靈還高
  • 擁有極高敏捷與強大的遠程攻擊力，無論是一擊脫離，突襲或長距離火力輸出都非常適合。但是，因為其智力偏低，很容易陷入敵人的陷阱。
  • 颶風騎士．穿刺者（Raphael El Re'em）
     
    • 八舞耶俱矢的天使，外表是一柄長度超過她身高的巨大突擊槍。同時是颶風騎士Raphael的箭矢
    • 然而動畫裡卻可以從槍頭發射空氣彈
    • 小說的設計比動畫好看多了
  • 颶風騎士．束縛者（Raphael El Nahash）
     
    • 八舞夕弦的天使，外表是前端付有菱形之刃的鎖鍊靈擺。同時是颶風騎士Raphael的弓弦
    • 明明收納鎖鏈的手柄很短，卻能伸出長度接近無限的鎖鏈
  • 颶風騎士．天際疾馳者（Raphael El Kanaph）
     
    • 八舞雙子天使真正的姿態，外表是巨大的弓箭
    • 由於八舞的分裂使天使的力量也分散到耶倶矢與夕弦兩人身上，需兩人合力才能發揮真正的力量
    • 天使發動時，耶倶矢右邊和夕弦左邊肩膀上會個別裝上羽翼狀的無機物，同邊的手部會出現手甲
    • 擁有超強的破壞力，僅一擊便能把DEM社的戰艦Arbatel擊沉
  • 神威靈裝·八番（Elohim Tzabaoth）

     

     
    • 外表是紫藍色的皮帶拘束衣，是目前露出度最高的靈裝。
      • 耶俱矢的靈裝的顏色偏向紫色，而夕弦的靈裝則較偏向藍色
    • 項圈上扣有鎖頭，一邊的手腳個別裝有斷鏈銬（耶倶矢是右邊而夕弦是左邊）
    • 為什麼這麼暴露的靈裝，防御力居然是B級以上？
  • 速度非常快，在未被封印靈力時可以在數分鐘內移動上百公里的距離，這還是兩人一邊交手一邊移動而非全速前進的狀況
  • 動態視力很強，在被狂三的二之彈減緩時間的狀況下，依然能夠捕捉到以一之彈加速逃走的狂三
    • 此段連同二之彈在動畫版慘遭刪減

其他編輯

  • 士道十後宮（預定）中的“八”
    • 而且還是兩人，所以應該是士道的十一後宮
  • 代表生命之樹中第八個質點「榮耀」（Hod）
    • 橙色[14]的頭髮，水銀[15]般的瞳孔，爭奪“真正的八舞”這一榮耀的好勝心
    • 象徵天使拉斐爾（Raphael），神力天使，象徵物質型態的「鑄型」之意。[16]
  • 代表善惡之中樹第8虛數與惡德「貪欲」（Chemdah）
    • 象徵惡魔阿德拉米力克（Adramelek），象徵墮天使薩麥爾（Samael）。
  • 空間震的規模很大，但是不知為何都在空無一物的上空被觀測到，造成的損害不大
    • 但若兩人戰鬥產生的餘波則會造成廣範圍的災害，就像是高速移動的風暴一樣，而且有不少的目擊情報
    • 因為太麻煩而被指定為最優先處理的精靈，但又因為速度太快、移動範圍太廣而無法追蹤
  • 因為以各種方式進行決出真正八舞的比試，兩個人都相當多才多藝，不過相對於全能型的令音而言，二人只能屈居在對方之下
  • 目前唯一在天宮市之外的地點被攻略成功的精靈，就都合程度而言可說是全女主角之最
    • 事實上劇情中也承認能遇見並接觸八舞是非常巧合且幸運的
    • 但成功被攻略后就成了最路人的兩位女主角……
      • 不過據說原因是因為作者對於她們兩個的說話方式感到頭痛所以就乾脆把她們路人化了
  • 於動畫OVA中（對應小說短篇《十香Gamer Center》）在十香拍照片的地方附近玩遊戲設施[17]
    • 不過就跟動畫第三話中在夾娃娃機附近的四糸乃一樣，完全沒人發現她們的存在明明穿著如此暴露的衣服……
    • 另外，在動畫第二季第一集中出現的氣象報告中，有提到一對形成後又莫名其妙消失的雙子颱風，這對雙子颱風疑似她們兩個開打時形成的，之所以會形成後又莫名其妙消失，似乎也是因為她們消失的緣故
  • 雖然在被士道封印靈力後已經和解，但偶爾還是會以各種比賽互相較勁。
    • 在小說短篇《八舞Lunchtime》中接受士道的特訓時，就以特訓的內容進行比賽，還制定出“每輸一次就脫一件衣物”的懲罰
    • 在第十卷中還在上學時比賽誰比較早到學校
    • 在第十一卷中還以冰淇淋作賭注而進行電玩比賽
  • 在小說短篇《八舞Lunchtime》中，二人為了擊敗「來禪高中小賣部四天王」而接受士道的特訓
  • 代表動物是猴子。其實耶俱矢並不喜歡這樣的代表動物
