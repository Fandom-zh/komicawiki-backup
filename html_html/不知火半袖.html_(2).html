 リアルイーター / 正喰者 技能 過負荷 BGCOLOR(#39DDDF):
不知火半袖最初的過負荷，吞噬並轉化技能的技能。
條件是碰觸到對方後才能發動，能將對方的技能吞噬後任意改造和加設條件，如時間或次數限制。
這個技能被安心院稱之為世界上最可憎的技能，更是最接近半纏製造技能的技能的繼承表現。

曾吞噬並轉化的技能有：
力量的技能 不為人知的英雄[6]→速度的技能 光化靜翔[7][8]
掌握她人視界的技能 欲視力→掌握自身極限的體術 善吉模式

所有能力都是人們個性與才能的一體兩面，例如過於強大的日之影(異常:不為人知的英雄)或過於弱小的球磨川(過負荷:腳本製作)，
或著都城王土的(異常性:人心支配)，還有行橋未造的(異常性:受信感度)。
當正喰者改造技能後，同時也改造了個性與才能[9]，喪失了不為人知的英雄後，日之影便失去了異常強大的力量，
但也獲得了原本該有的氣勢及存在感。
 
 / / 不知火一族的回避特技 特技 / BGCOLOR(#39DDDF):
能毫髮無傷迴避攻擊的不死性特技，據半纏[10]所說是不知火之村中任誰都會使用的小特技，
推測是使用衣服造成的迴避。
但這個特技無法迴避獅子目言彥的攻擊。
如果說不知火之村的人都會用，那變態六人組[11]怎麼不去閃掉安心院的攻擊WWW
 
 / / 無限的餐具制 特技 / BGCOLOR(#39DDDF):
隨意拿出各式各樣的餐具攻擊，天知道是技能還是藏在衣服裡你是宗像家的嗎[12]
 
 / / 獅子目言彥 / / BGCOLOR(#39DDDF):
在第180箱中被言彥強行繼承了身體，獲得了不可逆的破壞能力、不同次元程度的防禦能力[13]
但之後也產生外表年齡停滯的後遺症。
  • 特性
    • 先手一擊斃殺：在漆黑宴時被善吉提起的，鬼牌組的共同特性，或著說是攻略方法。
      • 因為自身的不死性使然，不會防範和逃避第一次的必殺攻擊，而鬼牌組被先手一擊斃殺的理由各不同。

簡評編輯

  • 擅長從規則本身動手腳的幕後支配型人物，在背後操作了箱庭學園的裏側政治。
    • 也因此不直接去參加戰鬥，自身的戰鬥經驗因而遠遠不如身經百戰敗的球磨川
    • 善於推測對方的技謀然後攻破對方。
  • 因為幾乎完整繼承了半纏"創造能力的能力"，加上身為替身的優良素質，而被稱為不知火一族的菁英
    • 但也因此要繼承現代的言彥，最後將失去自我，這是身為精英卻更加(不幸=過負荷)的例子。
  • 過負荷-正喰者，被稱做是在安心院死後，世界上僅次於言彥，最為無敵強悍的能力。
    • 隨意改造(能力=個性)的能力，連球磨川和目高都不會有勝算。
      • 當然前提是要能施展成功，球磨川也是因此才能在最後壓制半袖。
