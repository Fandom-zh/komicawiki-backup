Chase (假面騎士)

Chase/チェイス/Roidmude 000/切傻
以下含有嚴重劇透，不想被信號斧處刑的話請不要看下去。

俺は生きとし生ける……

全ての自由のために戦う戦士！

仮面ライダーチェイサーだ！

目錄

解說編輯

  • 英文單字，意為追逐、競逐
  • 美國摩根大通旗下的銀行之一。
  • 《JOJO的奇妙冒險》第四部TV動畫的第二片頭曲。
  • 優木雪菜的單曲，但以上皆與本條目無關不過雪菜的歌還是有幾句台詞對應到本條目的主角
  • 《假面騎士Drive》的第三以及最初騎士，假面騎士Chaser的變身者。本條目的重點。

基本資料編輯

 
  • 演：上遠野太洸
  • 聲：遠藤大智（Drive Saga，擬態前）
  • 真身為最初的Roidmude-Proto Zero(000號)，後被其他Roidmude改造成處刑失敗品的「死神」，是系列作除Evolto外唯二兼具怪人、擬似騎士以及騎士身分的人物
    • 按Roidmude進化體的命名法則，Chase的名字由來可能是來自他的職責和曾經在Global Freeze時以Proto Drive的身份對抗Roidmude
  • 人間體的模樣複製自交通機動隊騎警「狩野洸一」，尤其是正經八百的死板性格更是同一樣，進之介初次遇上他便一眼看破他沒朋友
  • 墮落版的照井龍
  • 津上翔一（沢木哲也）二號

性格編輯

  • 不苟言笑，為此被某些人恐懼，但(正式加入後)也有特狀課的迷妹倒貼
    • 《Drive Saga》藉由Angel的羽毛迴路而獲得「感情」時的表現讓進之介和霧子（包括腰帶先生）噴果汁到剛身上，十分不習慣情感滿溢的它甚至覺得噁心
  • 想對人類的情感有更身的理解，在愛情的事上常鬧笑話但也多次幫忙作球
    • 43話先後單刀直入地問進之介「是否喜歡霧子」和對剛說「我喜歡霧子」讓兩人都馬上噴茶，之後還跑去突擊問霧子是否喜歡進之介
  • 恪遵自己接收的指令與假面騎士戰鬥、作為魔進Chaser時更幾乎不對人類出手，避開人群和Drive交戰
    • 就是被Medic以及Brain等修改程式，還是無法清除最初的指令:保護人類而戰鬥
  • 意外地遵守人類社會的規則
    • 在圖書館和進之介相遇時，主動要求到外面決鬥
    • 為了協助誘捕Torando操控載具考了駕照[1]中途還變身完停好車才去支援戰鬥，並深深以此為榮
  • 轉身成為假面騎士後天然呆化
    • （說出重大情報後被問，但為何不早說時候）「因為你們沒有問過」合理得無法吐槽

名言編輯

  • 「名乗る必要はない、これから倒す相手に」
    • 以Proto Drive身份初次應戰Roidmude時的台詞
  • 「俺は死神。仮面ライダーなど凌駕する!」
  • 「本は静かに読むのが人間のルールではないのか」
    • 到圖書館找進之介幹架時對他驚訝的反應做出的回答
    • 就像是反映他的真正身分與複製元的性格一樣，作為Roidmude卻會遵從人類的規矩
  • 「人間を救うのは…俺の本能なのかもしれない。」
    • 「人間を守ることが俺の本望なのかもしれない...霧子」
  • 「人間が俺にくれた……宝物だ」
    • 向Gold Drive自爆前，將Signal Chaser以及駕照給予剛時說出的遺言

經歷編輯

過去編輯

  • 被已經死亡、成為腰帶先生的克利姆喚醒，成為Proto Drive阻止Roidmude、為保護人類而戰
  • Global Freeze當夜憑一人之力擊倒多數Roidmude，救出了包含霧子在內的民眾若干，但無法破壞Viral Core
  • 一度將尚未進化的Heart逼入絕境，卻在完成進化的Medic到來救活同伴後被反轉，在腰帶先生面前遭到Heart擊破
    • 此事成為腰帶先生忌憚Heart，且對進之介極力保護的原因
  • 被Freeze,Heart以及Brain聯力改造成魔進Chaser，自此開始以死神之名活動

魔進Chaser時期編輯

  • 視假面騎士Drive為宿敵，不斷阻攔進之介變身而成的Drive，反指擁有能破壞Roidmude核心的能力，不給予它們「更生」機會的他才是真正的死神
    • 除了Drive以外，不聽幹部們命令的Roidmude也是它的狩獵對象
  • 以性惡論敵視人類，視他們為完全沒有下殺手價值的垃圾，因此即使知道進之介是Drive的變身者也會待他變身後才出手
    • 但自從與霧子相遇後開始變得有所動搖，腦海裡不但會浮現出她的樣貌，還有腰帶先生要它守護人類的記憶，連帶逐漸對自己的身世有所疑問
    • 第11話的戰鬥中因進之介的求饒勸說而手下留情，被同行的Brain打傷並露出真身
      • 之後先後向Heart和Brain求證編號一事，但Heart只說它「是偉大的戰士」，而Brain不置可否，直到第14話才告知其曾是「假面騎士」
    • Brain指它「受人類影響」把本來的使命給忘記，並要為此把它處理掉時被Heart勸止，Medic在得到Heart的首肯後以自己的方式把它再次洗腦「重置」，回歸成冷酷的殺戮機器
      • Medic指它會出現記憶錯亂的原因是Proto Drive的基礎程式無法刪除，而其洗腦方式則是把守護對象從人類竄改成Roidmude，副作用是無法再執行狩獵Roidmude的工作，於是這項工作由Medic親自接掌
  • Brain為了查出再次洗腦的祕密將自己的程式加入體內，意外地讓它在執行作戰途中再次突破洗腦並回復本性
    • Medic回收後被又一次洗腦，並加入會於短時間內提升破壞衝動的程式指令，讓它於第22話與Drive決戰前無差別地攻擊人類，使得進之介和腰帶先生不顧勸阻，也下定決心要全力擊破
    • 一輪戰鬥後敗於Drive Type Formula的必殺技之下，連核心也被破壞，軀體卻不知何故殘存下來，被路過的霧子帶回去以Mad Doctor治療，其Ride Chaser亦被送回Drive Pit作「戰利品」
  • 痊癒後的它再次陷入過去的迷惘中並對霧子不辭而別，期間曾經為協助她制訂對抗Roidmude融合進化體的手段而自殘、又被本想迎接的Heart看穿內心而令他自己選擇去路，最後選擇遵從自己最初被輸入的命令，為守護人類而戰

成爲假面騎士Chaser編輯

  • 為了尋回自己在Global Freeze前後的記憶委託霧子修復Shift Zero，之後進之介等人在它讀取修理好的Shift Zero內藏的記憶時，因從眼睛投射出影像而得知真影壯一的真身為Roidmude 001
  • 重新成為假面騎士後都是單獨行動，在Roidmude搗亂時會出現協助進之介一行人
    • 起初加入時對進之介、霧子和剛三人在面對牽涉家人的事情時情緒變得非理性一事感到不解，其後理解其價值時下定決心要拼上性命保護人類和家人
    • 對霧子萌生出愛意後先後求教於進之介和剛，但當看到霧子被問及情事時的慌張反應與進之介同出一轍後馬上便明白對方的心跡，留下了失戀的眼淚並轉而決定為守護霧子的愛而戰
  • 第28話為了讓進之介解除門禁而扮黑臉，現出真身配合特狀課演一場戲從此註定崩壞
    • 《Saga Chaser》中一度受到Roidmude Angel蠱惑要回復真身完成超進化，最後選擇保持本來的自我，擊敗她救了Heart等Roidmude以及被掌控的平民
    • 第36話在成功拯救ゆかり後、發送給進之介看的影片裡擺出微妙的讚好姿勢
    • 第39話為了協助誘捕Torando而特地跑去考駕照，練習到一半還要在車內變身去救援Mach兼搶功勞
  • 認定剛是自己的夥伴也常共鬥，然而對方始終否認甚至脫口而出人類和Roidmude不可能交好，於是此生留憾
  • 在阻止第2次Global Freeze的作戰中與剛一同為眾人斷後、加上剛負傷而應戰Gold Drive，但在對方壓倒性的戰鬥力下不斷敗退
    • 腰帶被打壞、變身成魔進Chaser也敗北下，把Singal Chaser和駕照託付給剛後自爆與Gold Drive同歸於盡，核心也同告毀滅

Zi-O編輯

後續編輯

  • 《超Movie大戰GENESIS》中因為進之介與天空寺尊回到10年前導致的蝴蝶效應，他與其他Roidmude一同復活並協助剛與マコト迎擊Roidmude和眼魔
    • 隨著進之介與尊打敗達文西眼魔修正歷史而無法再繼續存在，聽見剛肯定自己為摯友的話語後，滿足地再次消失
  • 在剛試圖讓它復活時，因為實驗失敗導致意識化為Shift Ride Crosser並短暫附身在狩野洸一上
    • Saga Brain裏其數據正緩慢地復甦僅1%但是Heart、Medic、Brain已經醒過來了，難道要等10周年才能完全醒來嗎？

魔進Chaser（魔進チェイサー）編輯

  • Chase使用Break Gunner（ブレイクガンナー）變身成的擬似騎士
    • \ Break Up! /
  • 通過插入不同的Roidmude所屬Viral Core可以切換不同的武裝形態

形態一覽編輯

  • Tail Wiper
    • 使用Cobra Viral Core所變化而成的長鞭武器
  • Fang Spidy
    • 使用Spider Viral Core所變化而成的大鉗武器
  • Wing Sniper
    • 使用Bat Viral Core所變化而成的弓型武器，可移動到背後變成蝙蝠翼用以飛行
  • Triple Tune
    • 將上面三種Viral Core連續使用的復合武裝形態，可以同時使用三種武裝進行混合式攻擊
  • 超魔進Chaser
    • 使用Rhino Super Viral Core所變化而成，全身變成金色為主的形態
    • 在最後擊破Angel時把角的部分射出去攻擊而損毀，後來被回收改造成Shift Viral Core

Proto Drive（プロトドライブ）編輯

  • 於本篇開始半年前的Global Freeze中出現，擊退Roidmude的先代假面騎士Drive
  • 頭盔上沒有Drive那樣的跑車的定風翼，黑色色調的身軀上沒有安裝輪胎，徒有能擊倒但沒法將Roidmude核心消滅的力量
    • 雖然與現在的Drive相比在性能和裝備上都較落後，但戰鬥力十分強橫，於Global Freeze時僅靠格鬥技和Shift Car的支援就幾乎將所有的Roidmude都擊破
  • 能透過插入Viral Core讀取Roidmude的記憶
    • 腰帶先生指他因此而被Heart打敗並戰死，實際上卻是經由某種途徑被意圖得到超進化契機的Roidmude幹部們而改造成現在的魔進Chaser，其時的記憶已被Brain洗去
    • Brain於第14話直接說明了Chase（Proto 0）就是當時的Proto Drive
  • Roidmude們稱呼Drive為「假面騎士」的起源[2]，也許是因為他和Drive長的相似、又騎著機車的關係
  • 必殺技：Proto Drop[3]

假面騎士Chaser（仮面ライダーチェイサー）編輯

  • 顏文字→（0W0≦）
  • 希望Chase能回歸正軌的霧子拜託りんな研發第二組Signal Bike Chaser，讓它能配以研發用的Mach Driver炎重新變身成假面騎士
  • 初變身時是先變成魔進Chaser，然後才CAST OFF變成這形態
  • 專用裝備為仿造紅綠燈設計的「信號斧(Signal Axe)」，需要時間蓄能發動Full Throttle攻擊「Across Breaker」
  • 過去變身魔進Chaser時使用的Break Gunner在變身後也能繼續用
  • 必殺技：
    • 通常：Chaser End
    • 武器連攜：Across Breaker
