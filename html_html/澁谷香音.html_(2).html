澁谷香音

澁谷香音（澁谷（しぶや） かのん）[1][2]/Shibuya Kanon/香音A夢/澁谷觀音[3]/飛天女神[4]

目錄

解說編輯

  • LoveLive!系列作品《LoveLive! Superstar!!》的主要角色
  • 生日：5月1日
  • 星座：金牛座
    • 系列主角中唯一的金牛座
  • 血型：A型
  • 身高：159cm
  • 代表色：萬壽菊橙（マリーゴールド）
  • 象徵圖案：耳機
  • CV：伊達小百合（伊達さゆり）
    • 經公開試音選拔的成員之一（另一位是青山渚）
    • 其他訊息詳見聲優的個人條目

簡介編輯

  • 結丘女子高等學校普通科，學園偶像團體「Liella!」的成員。
    • 第一季為1年級生，第二季起升上2年級。
  • 家裡經營咖啡廳，店長是香音的母親。
    • 店裡的吉祥物是紅角鴞圓滾滾（マンマル），店裡也有很多貓頭鷹裝飾。
    • 香音常常和可可在咖啡廳討論學園偶像的事，然後被旁邊的媽媽和妹妹吐槽。
    • 然而作為原型的店家不喜觀眾前來朝聖，於動畫開播前遭撤換
  • 家庭成員還有妹妹亞里亞和身為翻譯的父親，和其他父親一樣沒露臉也沒有台詞
  • 奶奶是西班牙人，所以有四分之一的西班牙血統，但在動畫中沒有提及
    • 目前並不會說西語（一單自介），倒是能說出一口流利的漢語
  • 小學就讀青山南國小；國中是花咲川[5]外西苑中學。
  • 喜歡的食物是番茄、烤蘋果、漢堡肉、咖啡歐蕾，還有奶奶做的海鮮飯
  • 擅長咖啡拉花
  • 喜歡的動物是貓頭鷹、兔子
  • 座右銘：「雲上總是天晴」（雲の上はいつも晴れ）
  • 瑜伽高手，找靈感時的瑜伽動作不少都是高難度，甚至在船上也能做，身體的平衡度和柔軟度極佳。
  • 時常聽音樂，喜歡唱歌，也有玩音遊；想透過歌聲帶給大家元氣
    • 還會彈吉他和作曲，剛好可以組五人樂團
  • 居家時有沖天炮髮型、眼鏡屬性
  • 常常戴著罩式耳機，款式據考察是SONY的WH-1000XM4
    • 或許是反映角色成長很少戴著了，最終回妹妹拿給她時還婉拒
  • 和千砂都是慣例的青梅竹馬，對千砂都暱稱「小千」（ちぃちゃん）
  • 意外地有懼高症，跳水時還要日團幫忙推一把
  • 本性消極怕事，但決定的事就會全力以赴。
  • 主角位，人物設定上打破了很多既有傳統，具有長髮（初設）、外國血統、吊眼梢與褲襪[6]等之前作品中會出現在其他成員上的配置
    • 髮色是橘色的，但姓氏沒有「高」字
    • 個性上亦與樂天開朗的三位高家人不同，偏向內向而不自信，但也有相比三人更加直率而帥氣的另一面
      • 動畫中可見到更多樣化，貼近真實女高中生的描寫：如在家中對家人冷淡的反應、對葉月的質問作出答辯的強勢樣貌
      • 受中之人的哭包屬性逆輸入，很容易哭，缺乏自信，但在關鍵時刻卻能挺身而出。
      • 承繼穗乃果和千歌的總受
      • 在生放送被中之人們吐槽說滿身都是弱點。
      • 因此被可可稱讚「很可愛」時妹妹都嚇傻了
      • 小時候個性相當熱血，妹妹的評價是『很吵』。簡單就是女版的松岡修造。
  • 一改以往主角是主動提出校園偶像的帶頭人，香音是反過來被可可勸誘加入學園偶像部，打破了之前作品中由第一主角帶頭的慣例。
  • 容易緊張，只要站在眾人面前就會緊張得唱不出來，最嚴重甚至會昏倒，因此考不進音樂科而進了一般科。
  • 第一位苦勞人領袖，怕麻煩又容易緊張，但總是被推上台前應付麻煩事。香音A夢
    • 明知吃力不討好且和自已毫無關係，但就是無法放著不管，總是為了別人讓自己累個半死。
    • 有點濫好人，對於夏美的不平等契約也沒動怒，甚至在被揭發後依然為著影片獲得大量點閱高興。

經歷編輯

正劇前編輯

  • 小學時在歌唱表演中怯場暈倒，之後就沒辦法在重要的場合唱出聲。
  • 國中時以結丘女子高校的音樂科為目標，但在甄試上無法唱歌而落選。
    • 最後考上普通科，放棄了在結丘盡情歌唱的夢想。

第一季編輯

  • 入學第一天因情不自禁的歌唱恰好被可可聽到，而被對方邀請成爲學園偶像。你好謝謝小籠包再見[7]
    • 起初自認爲不適合擔任而婉拒，但在被戀單方面禁止學園偶像成立後感到不滿，反而跟對方嗆了起來。
    • 但在被戀反問道自己想要做什麼時也無法回答，雖然沒有想成爲學園偶像還是提議幫可可招其他人選，兩人一番周折向菫發出邀請後的同時還要躲著戀的追查下還是沒有募集到成員。
  • 而可可在一番真情流露下還是希望和香音一同成爲學園偶像，自己卻一度因爲自卑而拒絕並表示歉意地離去
    • 在聽到可可説出「最喜歡香音的歌聲」後再次堅定了自己想要唱歌的想法，於是慣例地第1話個人校園Live不經意間在眾人面前唱出來了。
      • 當香音回到可可面前時，天空飄下代表著LoveLive!王座精神的白羽毛，並掉進香音的背包，但她並不知道。看來還不夠格承繼王位。
  • 此後性格也變得開朗，對於成爲學園偶像的想法和推動行爲也積極了起來，再次被戀否決提案後又在可可的餿主意下發起抗議活動鬧到了理事長那
    • 因爲有著倡導學生培養音樂興趣的宗旨而允許學園偶像的存在，但對方亦要求「要在偶像祭典上拿到第一名」才能進行相關活動
  • 香音與可可兩人開始接受千砂都的體能與舞蹈指導；回家後拿著可可寫的歌詞本研究，並徹夜作曲，一直以言語提醒自己「不放棄的心情」
    • 與可可以隊名「可香」於偶像祭參賽，為了在偶像祭能夠好好面對人群唱歌，千砂都甚至在章魚燒店令香音「代班」作為特訓手段
    • 背負著必須拿第一的壓力，練習過程中一度頹喪，此時可可告訴自己喜歡校園偶像的動機以鼓勵香音
  • 開演前在台上再次膽怯，身旁的可可則以耳語打氣，台下觀眾熱情地揮舞螢光棒回應，香音獲得了歌唱的勇氣，順利完成演出
    • 雖然最終未取得第一名，而是新人特別賞，但也因此特別許可建立學院偶像部和祖傳天台練習的使用權
  • 剛成立社團就遇上菫毛遂自薦地要入社，不過對方在得知自己被34:2:0的班級投票倍殺不能成為團隊C位後又光速退團
    • 尾隨對方時偶然看到小時候的錄像，於是被對方綁到房間裡試圖施法洗清記憶，之後也知道了堇的心結
    • 因為跟堇有著相同的經歷和煩惱，再三思考下還是沒有放棄拉菫入夥的想法，於是夜晚學著用星探的方式邀請她加入，最終打動了堇加入社團
  • 暑假期間被日團邀請香音等人以嘉賓身分參加神津島上的Live，獲悉有能夠唱歌的地點，香音爽快地答應了對方
    • 島上一日度假後的夜晚，香音並沒有睡著而是獨自前往海灘練瑜伽思索，因為不知道如何將千砂都的定位想像出來後寫進歌詞而煩惱
    • 發現人沒有在房內的可可則尾隨香音前往海灘，在關心香音之餘建議邀請千砂都入隊，而香音則回憶起與小千的往事有了點想法
      • 在跟Sunny Passion交談下終於能夠獨自創作屬於自己風格的歌曲，爲了表達喜悅也第一時間打電話給小千分享
      • 因爲察覺到電話中對方似乎有心事，在小千的舞蹈大賽當天趕往現場給她打氣，也因此得知了小千的一些想法（見角色條目）
      • 一番交談之下讓她能夠鼓起勇氣、自信地去參賽，最後也成功入夥成爲學院偶像
  • 被可可和千砂都提議競選學生會長，但因為膽怯的性格拒絕轉而去支持菫參選。但很快就以負20票的成績落選
    • 在戀正式擔任學生會長後單獨找她談論對學園偶像的看法，結果依舊是不被認可。
    • 然而隔天戀就發表了違反選前政見的言論，香音等人為了釐清戀真正的想法而而跟蹤她到家中。
      • 在經歷了與小不點（戀養的狗）的追逐後意外得知戀家已經瀕臨破產的局面，以及戀為了學校堅決要禁止學園偶像的心…
  • 試圖遊說音樂科與普通科兩方對學園祭的意見未果
    • 戀向香音等人哀求不要進行偶像活動，並說自己的母親試圖以此拯救學校但失敗，自己完全找不到有關的活動紀錄，故認為母親對此感到後悔
    • 不認同戀的陳述下向理事長借了校史室的鑰匙，想要找到偶像部的資料查明真相卻一無所獲
    • 朝會前戀主動向她表示母親常常說道「希望能在同一個地點延續偶像活動的心願」
      • 聯想到偶像部社辦以及先前給的兩把鑰匙後不顧全校集會時間奔向社辦，終於在地下室倉庫發現一個上鎖的箱子，獲得神宮音樂學校偶像部的活動日誌以及制服所以交接時都沒清點嗎
      • 秒速讀完日誌後趕往朝會的她獲得理事長准許上台向全校同學揭露日誌內容、宣讀當時社員們活動的點滴，以及戀的母親對學校的願望，戀得知母親真正的想法後聲淚俱下、並在集會後向香音再次致歉
      • 學園祭搭建學園偶像的舞台時邀請釋懷的戀加入社團，而戀釋出善意握手言和，兩科隔閡正式化解，團體也以五人之姿登台獻聲
  • 由於報名LoveLive大會需要團體名，除了負責作詞也開始構思團名，經歷嚴重的瓶頸後總算成事
  • 當可可對於菫在預賽被推選為C位有微言時提醒她去追蹤菫的訓練
  • 預賽入圍後接獲（與千砂都的）小學母校的通知，邀請香音一行人前去表演
    • 儘管和團員們得以一起在台上合聲，香音仍回想起了一些往事：
    • 雖然一直以來都能夠唱出歌聲，但實際上是因為都是與隊上的同伴一起唱，一個人唱歌時内心還是容易怯場；千砂都察覺到這一點，便向其他人提議小學表演當天讓香音solo
    • 當夜詢問亞里亞小時候的自己，對方回答「很吵、很熱血，但是不令人討厭」
    • 儘管團員沒有明說，香音仍在上台前通電表示自己知道千砂都想要自己成長的用意並致謝
    • 即將上台前，香音想起了過去在同一地點將要上台的自己，開始與自己心靈對話，告訴自已雖然害怕，但要記得自己最喜歡唱歌的那份心
      • 最終在沒有任何夥伴同台的情況下獨自唱出聲來，也意味著香音終於走出了内心的陰霾
  • 東京大賽以一名之差敗給日團後為著無法回報夥伴們的努力深感自責，不過在神路人與隊友的鼓勵下矢志要唱出屬於大家的歌奪勝。

第二季編輯

  • 為了雪恥暑假都和團員們在學校練習，恰好遇上迷路的希奈子。
    • 被希奈子叫前輩興奮到忘我，還要她多叫幾聲。
    • 幫希奈子帶路回家，順勢邀她到自己家作客，之後產生了和新生一起努力的想法。
    • 因為招不到其他人而對希奈子死纏爛打，透過直銷話術以及其他二期生的助攻下順利讓希奈子入團。
  • 見到希奈子無法跟上訓練又間接勸退新生，一度決議減省訓練量、將目標降為「入圍」。
    • 但自己也過意不去，於是在希奈子做出成為學園偶像的覺悟後撤回。
  • 由於Sunny Passion的高評價承受相當重的壓力，在代代木音樂祭又一次敗北後反而得以解放，重新回到挑戰者的立場。
  • 當一年生要獨自訓練時還擔心自己是不是逼得一年生太緊，在千砂都的安慰下才重新振作。
    • 剛好要送東西給父親而來到北海道，想順便帶慰問品給一年生時意外聽到希奈子想拉夏美進團，隨後扮成熊在其他二期生不知情下現身將夏美收編。
  • 為了分擔戀的工作，向理事長申請成為學生會副會長。
  • 預賽在神路人的建議下尋找能反映結丘特色的場地，開放校園日後經過希奈子等人的啟發選擇表參道作為舞台，在神路人的支援下順利晉級。
  • 東京大會勝出後透過社群網路了解葳恩的過去，同時葳恩嚮往入讀的維也納音樂學校也向她招手。
    • 原本想在結丘繼續作為學園偶像活動而無意留學，但在眾人的勸說下決定前往維也納實現「把歌聲帶往全世界」的夢想。
    • 然而在出發前遇到了穿著結丘制服的葳恩，告知因為還要拍第三季留學中止的消息…

其他編輯

 
二期的視覺圖中香音占了大半的畫面
其實是超巨大的香音站在後面
  • 由於戲份比重極大，且劇情上過度依賴、吹捧香音，得到「香音A夢」、「傳銷頭子」、「澁谷觀音」等稱號；動畫也被戲稱為香音傳（特別是二期）。
    • 最早出現此說是在1期9話四人逼她寫歌詞和構思團名的橋段，2期的表現更支持此說。
    • 可可和四季外的成員皆由香音拉入團，招募夏美時更是讓香音強行出現在北海道，通勤問題就別探討了。
      • 招募二期生的話術流於空洞也是部分觀眾不滿的一點。
    • 勝出東京大會後收到海外音樂學校的獎學金、瑪格麗特家准許葳恩跟在她身邊學習外，夏美更表示除她以外沒有可靠的學姊。
      • 謎之劇情：瑪格麗特家對親生女兒要求優勝才能入學，對香音可以破格包吃包住還送個陪讀生
    • 上述種種使星團和貼近群像劇的前三代演出落差甚大，二期最終回的標題「私を叶える物語」不同於過往的「私たち」、「みんな」也算是側面證明這點。
