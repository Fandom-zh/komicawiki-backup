勇者赫魯庫

勇者赫魯庫 / ヘルク / Helck
本條目內容不足需要擴充，歡迎有興趣的朋友加以補完。

目錄

解說編輯

  • 由七尾七希（七尾ナナキ）創作的奇幻冒險漫畫，在2013-2017年間於裏Sunday（裏サンデー）雜誌連載，全106話
  • 前期以類似搞笑漫畫的形式構成，實際上劇情主線相當嚴肅，搞笑的部分反而才是陪襯
  • 除本篇外還有以劇中角色嗶伊為主角的外傳《嗶伊～不思議的生物～》以及同世界觀的衍生作品《異劍戰記》
  • 於2022年宣布動畫化

劇情編輯

  • 在魔界的某個國家，一名人類的勇者親手打倒了魔王
  • 就在人們臉上笑容洋溢的時候，魔界即將舉行一場又一場賭上新任魔王寶座的熱血戰鬥！
  • 此時現身的居然是…！？神秘勇者赫魯庫的大冒險劇即將開幕！

登場角色編輯

赫魯庫一行人編輯

  • 赫魯庫（ヘルク）
    • 男主角，新魔王決定戰的決賽參加者之一，被指控殺害弟弟克雷斯的人類勇者
      • 除了有99級的最強戰鬥等級之外，料理與木工能力都相當優秀
    • 過去與克雷斯出身貧寒，在垂死之際被拉法葉德收養，從此改變了兩人的命運
    • 在克雷斯討伐魔王重傷歸來之後，為了安撫群眾的恐慌而前往魔王城並結識了阿茲杜勒，並認同其人類與魔族共存的理想
      • 不料在返國之際卻發現了賢者米卡洛斯與貴族企圖使用禁咒將全人類變成覺醒者的計畫，與其對抗之際被對方傳送到魔界
      • 在魔界徘徊時看見了魔界居民的良善一面，並決心為保護他們而戰
    • 與米卡洛斯等人一樣是「世界的意志」的接觸者，不過本人拒絕了該意志並決心抵抗之
  • 芭米里歐（ヴァミリオ）
    • 女主角，帝國四天王成員，別稱「紅之芭米里歐」
    • 擅長施展大範圍的火焰魔法
    • 初期對身為人類的赫魯庫相當不信任，為了就地監視而在選手攻略烏魯姆城時以工作人員「安」的身分接近他
    • 在知道赫魯庫的過去後向其公開自己的身分並全力協助他
  • 嗶伊（ピウイ）
    • 居住在孤島的神祕生物，本作吉祥物
      • 你好！！
    • 其歌聲擁有對抗禁咒的光明力量，在與米卡洛斯的決戰中成為使魔界陣營反敗為勝的關鍵

魔界國度編輯

  • 阿茲杜勒（アズドラ）
    • 帝國四天王成員，別稱「青之阿茲杜勒」
    • 擅長操控樹木，雖然自稱是四天王中最弱的，不過只要在有帝國樹木生長的土地上就能成為最強的存在
      • 因為最終決戰時的帥氣救場，在網上又有「阿祖A夢」的別名
    • 在過去曾與人類陣營的勇者烏利亞意氣相投而成為好友
    • 最後烏利亞卻因為覺醒者的禁咒而狂化而被迫將其討伐，也因此使阿茲杜勒決心拯救人類
  • 修拉（ヒュラ）
    • 新魔王決定戰的決賽參加者之一，因為其再生能力而有「不死身的修拉」之別名
    • 戰爭結束後，因為其餘參賽者棄權的緣故，成為了決定戰的優勝者擔任遞補托爾的魔王
  • 肯洛斯（ケンロス）
    • 新魔王決定戰的決賽參加者之一，擁有傲人的速度而有「神速的肯洛斯」之別名
    • 赫魯庫在魔界的第一個朋友
  • 多爾西（ドルーシ）
    • 新魔王決定戰的決賽參加者之一，擅長使用結界而有「鐵壁的多爾西」之別名
      • 結界能力為第二種能力，必須藉由吃下豆子才能發動
  • 亞斯塔（アスタ）
    • 芭米里歐屬下的情報員，伊斯塔的姐姐
    • 擁有與妹妹心電感應的第二種能力，觸發條件為喝咖啡
    • 為了調查人類王國而深入險境調查，一度遭對方俘虜並結識了夏露雅蜜
  • 伊斯塔（イスタ）
    • 芭米里歐屬下的情報員，亞斯塔的妹妹
    • 阿茲杜勒受傷後負責照顧他的生活起居
  • 哈爾比（ハルピィ）
    • 帝國搜索隊成員
  • 樹諾娃（シュノーヴァ）
    • 居住在孤島上的神祕魔女
    • 真實身分是初代紅之四天王的姐姐
  • 托爾（トール）
    • 帝國所屬魔王
    • 在故事序盤即遭到討伐，帝國為了選出接班人而舉辦新魔王決定戰
    • 戰爭結束後確定由修拉繼承其領土
  • 烏魯姆（ウルム）
    • 帝國所屬魔王
    • 在新魔王決定戰期間遭到討伐，因此阿茲杜勒便要求參賽選手奪回其居城當作決賽
    • 戰爭結束後為決定其接班人，帝國再度舉辦了新魔王決定戰
  • 西恩（シン）
    • 帝國所屬魔王
    • 俘虜了飛翼士兵哈拉歐爾
  • 荷恩（ホン）
    • 新魔王決定戰大會的副執行長
    • 基本上就是芭米里歐的出氣筒
  • 洛可可（ロココ）
    • 新魔王決定戰大會的司儀
    • 傳說其雙眼具有強大魅惑能力，平時總以瀏海遮住雙眼

人類國度編輯

  • 國王
    • 王國國王，擁有治癒勇者並使其復活的能力
    • 實為米卡洛斯藉由復活多名古代國王構成的傀儡，由「復活」、「治癒」、「幻象」、「支配」四元構成
    • 其中代表「支配」一方的國王已經被滅絕的古代人意識取代，為了阻止「世界的意志」而打算搶奪赫魯庫的身體
      • 在赫魯庫一行人接觸位於終端機的他之前利用支配能力操縱夏露雅蜜引導眾人
      • 在發現赫魯庫也是「世界的意志」接觸者後陷入瘋狂甚至被「世界的意志」侵蝕，在與赫魯庫決戰後連同保存其意識的終端機一同被擊敗
  • 米卡洛斯（ミカロス）
    • 王國所屬賢嫌者
    • 真正掌握王國實權之人，殺死魔王托爾與烏魯姆的元兇
    • 1500年前敗給阿茲杜勒後，接觸了「世界的意志」並成為覺醒者
    • 在期間將國王變成怪物以操控之，並利用禁咒強制使人類變成覺醒者以推動滅世計畫
    • 在魔界軍的進逼下打算發動終極召喚術造出亞空間毀滅一切，不料卻被嗶伊的歌聲破解，最後被阿茲杜勒以樹槍貫穿而死
  • 拉法葉德（ラファエド）
    • 支撐王國發展的元老院貴族，赫魯庫兄弟的教父，夏露雅蜜的父親
    • 本身是擁有轉生能力的覺醒者，在摯愛之人因為戰爭而死去後企圖利用轉生能力阻止戰爭卻未能如願
    • 經歷多次絕望的他接觸到了「世界的意志」，因而參與米卡洛斯的滅世計畫
    • 在米卡洛斯的計畫全面崩盤後為了拯救女兒而前往亞空間，為赫魯庫擋下「世界的意志」的攻擊後死亡，臨終前期許下次轉生後能夠見證真正的和平
  • 克雷斯（クレス）
    • 赫魯庫的弟弟，討伐魔王托爾的勇者
    • 在討伐魔王之際受了重傷，因而被米卡洛斯洗腦成為其屬下
    • 在米卡洛斯死後恢復神志，打算與被困在亞空間的夏露雅蜜一同赴死，最後兩人皆被赫魯庫救出
    • 戰爭結束後成為人類國度的國王，致力於人類與魔族的和平共存
  • 夏露雅蜜（シャルアミ）
    • 克雷斯的戀人，貴族拉法葉德之女
    • 擁有藉由雪傳遞魔力的特殊能力
    • 在亞斯塔潛入人類王國時與其成為好友
    • 終盤一度被困在亞空間內部，後被赫魯庫與拉法葉德救出
  • 艾莉希雅（アリシア）
    • 赫魯庫的戀人，傭兵團長
    • 因為禁咒而覺醒後成為飛翼士兵
    • 戰爭結束後從異變中復原，與赫魯庫踏上拯救世人的旅程
  • 艾迪爾（エディル）
    • 王國所屬飛翼士兵，與修拉曾多次交手
    • 戰爭結束後從異變中復原，並與魔王修拉成為勁敵關係
  • 賽爾西翁（セルシオン）
    • 王國所屬飛翼士兵
  • 哈拉歐爾（ハラオル）
    • 王國所屬飛翼士兵

其他編輯

  • 歐羅羅（オロロン）
    • 在新魔王決定戰的賽馬項目中，為了淘汰赫魯庫而準備的超小型馬
    • 真實身分為世間罕見的神獸，終盤以成長的姿態回歸並再度成為赫魯庫的坐騎

世界觀編輯

魔界國度編輯

  • 由中央的帝都為中心構成的國家
  • 由於北方強大勢力的威脅而派遣重兵防守，卻因此難以抵抗人類勇者入侵

人類國度編輯

  • 由人類組成的國家，持續派遣勇者討伐魔王侵犯帝國領土

世界的意志編輯

  • 操控米卡洛斯等人的幕後黑手，藉由耳語的形式誘導接觸者遵循其計畫毀滅世界的存在
