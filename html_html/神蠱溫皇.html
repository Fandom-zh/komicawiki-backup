基本資料編輯

 
  • 神蠱峰之主，登場之前在此隱居
  • 苗疆出身，與千雪孤鳴、藏鏡人合稱苗疆三傑
  • 智計、武功、蠱毒、醫學、音律、術數、兵法皆修的大能總攻強者，金光四智之一
  • 一人分別使用不同身份包攬甲子名人錄其中三個天下第一之名[1]；分別是：
    • 天下第一毒（神蠱溫皇）
    • 天下第一閣（還珠樓）之主
    • 天下第一劍（任飄渺）
  • 萬濟醫會成員
  • 佩劍：無雙[2][3]
  • 司馬懿[4]

事蹟編輯

  • 黑白龍狼傳故事線的十年前在巫教引爆三途蠱使巫教化為死地。
  • 和藏鏡人、俏如來、靈界多方合作救下被困西劍流多年的史豔文。
  • 以天下風雲碑之局延緩西劍流進攻的步伐。
  • 以極端手法訓練劍無極並逼至瘋癲。
  • 被炎魔幻十郎襲擊後詐死脫離檯面轉至幕後佈局，並在西劍流敗退時傾還珠樓之力大量削減其戰力。
  • 設計讓史豔文與藏鏡人的身世之謎曝光並被中苗雙面夾殺，但同時化解史藏二人半生仇怨。
  • 天允山上引爆三途蠱，使正邪雙方武林人士損兵折將。
  • 與還珠樓代樓主百里瀟湘立下十年之約，後設計除去還珠樓內部反對勢力。
  • 九龍天書之局中遊戲其間，並製造假天書混淆視聽。
  • 與宮本總司決戰不悔峰，成功悟出劍十一艱難取勝。
  • 面對北競王的三步棋殺著，孤身入陣大破苗軍。
  • 以未完成的劍十二重創劍無極與雪山銀燕，自己也經脈盡斷。
  • 癱瘓期間遭到追殺，意外輪椅成精以劍十一殺退玄狐。
  • 鳳蝶被大智慧洗腦，覺得不能再繼續裝死了滿血復活加變身。[8]
  • 復健之後對戰千年功力的缺舟一帆渡，以智計武功取得顛倒夢想。

蠱術&醫學編輯

人稱天下第一毒的神蠱溫皇，所使蠱毒之術變化多端，有時他甚至會以蠱代醫。

除此以外溫皇體表亦有蠱毒，除了百毒不侵的鳳蝶外其他人不慎接觸都會中招。

  • 同命蠱
    • 溫皇自蠱術學成以來就在身上植下，以血養蠱，以蠱代命；可代替宿主承受致命的攻擊，培養時間越長能代命的限度越高。天允山上溫皇被炎魔幻十郎以枯血荒魂斷脈襲擊時便以蠱代命得以脫離死劫，但也失去了辛苦培養多年的同命蠱。
  • 核彈三途蠱
    • 傳說巫教中人欲極力研發的生化兵器。爆發後毒霧席捲方圓三里，凡是人獸，觸及毒霧者瞬間化為血水枯骨。且毒霧如殘餘核輻射般持續時間極為長久，三途蠱爆發之處十數年內仍是死地。
    • 當年巫教便是因為三途蠱爆炸變成了多年來無人敢入的廢墟。後來溫皇在天允山引爆鳳蝶體內的三途蠱，使在場眾多不及逃離的武林人士無論正邪皆中毒身亡。
    • 劇中曾表示三途蠱乃天下最毒的毒物，比這更毒的只有默蒼離的鉅子舌
  • 養血蠱
    • 擁有極強的造血能力。
    • 溫皇將之交予冥醫讓他完成治療失血症的研究。
  • 經驗值鳳蝶受襲防禦系統相思蠱
    • 能釋放劇毒，同時也能刺激經脈成長。
    • 分為子母雙蠱，兩者無法相隔太遠；後經溫皇改良而不再受距離限制。
    • 武功根基不高的鳳蝶因為相思蠱劇毒的刺激而習得飄渺劍法的劍一至劍十，殘廢後的溫皇也因為此蠱得以復健出山。
  • 冰蠶蠱
    • 有冰凍的效果。溫皇曾嘗試以此蠱降低千雪孤鳴身上因元邪皇的燭龍之力造成的高熱體溫但還是被烤熟了好幾隻。
  • 彼岸蟲
    • 可使人產生幻覺，天地不容客為訓練雪山銀燕而向溫皇討要。
  • 金盞蠱
    • 溫皇當年造訪羽國時為釣出雁王而對朝中文武施蠱，中蠱者毫不自知。
  • 千里傳音連命沙蠱
    • 「此蠱雌雄同生。即便分離千里，仍能彼此感應，互通訊息。」by 赤羽信之介
    • 溫皇利用沙蠱偷聽感應遠在東瀛的劍無極安危並接收赤羽的訊息
  • 紫煙蠱
    • 保命蠱的一種，可以感應宿主的危機，在宿主遭遇極端危機時自行噴發紫色煙狀蠱毒，除宿主自身外百丈範圍內無一生者。
    • 白比丘欲救治安倍博雅時被意外引爆

武學編輯

神蠱溫皇多用毒攻，而任飄渺則使劍。但在劇中因為身份限制為了防止破壞強度平衡幾乎沒出現過毒劍並用的情況。

為神蠱溫皇時

  • 藍蝶毒霧、蚳蠱蛺殺
    • 常用的毒攻
  • 風神雷馳、氣走風雷、一指風雷
    • 以琴/羽扇為兵的招式


為任飄渺時

  • 以自創的飄渺劍法奪得天下第一劍之名，劍法特點是無形無相
  • 任飄渺樂於分享劍招之故導致劇中研習飄渺劍法的人一隻手數不過來，但因為根基、適性的不同甚至改造的關係，同樣的劍招效果也會大相徑庭
    • 劍一·破、劍二·空、劍三·飛、劍四·滅
    • 劍五·虛、劍六·絕、劍七·真、劍八·玄
      • 除了劍一·破是比較固定的單發劍氣之外其他幾式特效在劇中較無明顯區分和特色
      • 劍八·玄曾有一段時間被誤植為劍八·極
      • 天下第一劍的稱號來自劍八：｢飄渺劍法，劍八已得天下第一｣
    • 破空飛滅，虛絕真玄
    • 破空飛滅，玄真絕虛
      • 前八式的自由組合招，跟劍九不盡相同。
    • 真玄虛絕，飛空滅破
      • 前八式倒轉逆施
    • 劍九·輪迴
      • 八式往復入輪迴
      • 雖說是八式往復，但劇中特效則以流經地下並刺出地面的劍氣居多。
      • 在與宮本總司對決時自言｢劍九滅巫教全族｣但滅掉巫教的不是三途蠱?
      • 曾為了欺敵而在出招前喊｢劍十一｣但實際出招卻是輪迴此時的特效前所未見的華麗，似乎真的沾了劍十一之光
    • 劍十·天葬
      • 自生而滅謂天葬
      • 任飄渺成為天下第一劍時此招仍未悟出。對決宮本總司時第一次使用：｢天下間，你是第一個見識劍十之人｣
      • 最初在劇中的特效是向四方大量爆發式擴射的劍氣，後來的劇集此招再出時改為多道劍氣合一射出
    • 劍十一·UBW涅槃
      • 極而復始，不生不滅，乃是涅槃
      • 任飄渺與宮本總司在不悔峰決戰時頓悟的劍法並以此擊破宮本總司。
      • 最初的招式特效不固定，自墨世佛劫後統一為拆塔或破壞地形用的擴散式劍氣
      • 劇中施展此招時通常會附帶處刑專用BGM
      • 缺舟一帆渡對此招評價：｢窮究人力極限的完美之招｣
    • 劍十二·殘廢/輪椅（未完成）
      • 不悔峰決戰後任飄渺發現劍十一不敵一劍無悔，開始著手研發劍十二以應對劍無極與雪山銀燕的三個月後重現一劍無悔之約。
      • 劇中使用此招時的旁白：｢劍至盡處無處可盡，唯有全然的劍意｣。由劇中表現來看應該是將劍意集中在雙手(全身?)並以手代劍。
      • 據鳳蝶所述，劍十二已是超越肉體極限的魔技；任飄渺在未完成的狀態下強行使出劍十二最終導致經脈盡斷成為廢人。
      • 附加效果是從此相當長一段時間內溫皇以任飄渺身份出場時都變成散髮造型
      • 任飄渺後來放棄了劍十二的鑽研，將視線從人力移至自然；而劍十一的變招「重生」被認為是相當接近極限而未傷身的劍十二。
    • 劍十三
      • 風雲碑再開，劍神決戰時為破慕容煙雨的招式預判，借助天地萬物之力加上缺舟的饋贈再次臨陣悟招突破極限而使出。
      • 風雲為劍，黃沙為劍，蒼天為劍，九地為劍，萬物為劍；目光盡處，飄渺劍意壟罩；捨無雙，捻指萬物，成就無雙一劍。


夢幻絕技

  • 藍蝶毒霧+劍十一·涅槃，僅在意識空間內不受限制面對大智慧時分身成二人使用的唯一一次毒劍並用。
  • 劇中特效形似小型化的止戈流劍陣。

還珠樓編輯

  • 為任飄渺創立的賞金殺手組織。擁有強大的情報收集能力。
  • 宗旨是標金賣首，只要委託人出得起價，這些亡命之徒就沒有不敢殺的人。
  • 樓址位於中苗邊界的飄渺峰，地形險要，同時樓內外皆佈滿機關，乃是魯家的傑作。
    • 以上數個條件使還珠樓榮登甲子名人錄的天下第一閣。
  • 成員皆以白玉環佩為身份象徵，不同綬帶代表不同等級的殺手。
    • 至於樓主所持的乃是名為「無上令牌」的特製品，成員只聽從持有令牌者的號令。
  • 任飄渺在巫教被滅後失蹤，百里瀟湘代掌還珠樓約十年直至神蠱溫皇回歸。
  • 如何加入還珠樓：
    • 無組織者需要實力證明
    • 先前已加入其他組織者須與前組織撇清關係
    • 擁有還珠樓所需的條件
  • 別名"溫皇飯店"，只招待惡勢力的同志，還收留了正劇目前行蹤不明的殺生鬼言(天恆君)[9]
