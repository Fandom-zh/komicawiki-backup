解說[編輯]

  • 由COMPILE HEART、電擊PlayStation與電擊文庫聯手推出的迷宮探索RPG與其相關跨媒體計畫
    • 相關作品皆以「逃獄」為主題，描寫少年少女們逃離巨大迷宮「神獄塔」的冒險
  • 故事原案為著有《致我深愛的每個妳》與《致深愛妳的那個我》的小說家乙野四方字
    • 在最後關卡都有設置猜黑幕的謎題，回答正確才能進入真結局，否則會進入壞結局並回到解謎前狀態
  • 人設為曾參與同社作品《魔壞神兆力昂》的ナナメダケイ，在角色設計上以童話為原型
  • 2016年10月13日於PSV平台推出一代
    • 在官網有連載前傳小說《獄中童話前日譚》，全12話
  • 一代推出後在電擊PlayStation連載小說版《神獄塔 斷罪瑪麗~光之所在~》，世界觀與本篇共通
  • 2018年7月12日於PS4平台推出二代
    • 遊戲本身包含一代重製版，預約特典為校園風格的短篇小說《戀獄塔 斷罪瑪麗（恋獄塔 めありーすけるたー）》
    • 在二代全破後，於重製一代的隱藏迷宮會揭曉二代的背景為一代前一輪的世界線
    • 在官網有連載前傳小說《獄中童話幻日譚》，全6話
  • 2020年8月27日於PS4 / Switch平台推出系列最終作《神獄塔 斷罪瑪麗Finale》，時間軸接續一代重製版
    • 本篇遊戲包含戀獄塔與前兩作的主線劇情「BEFORE STORY」，預約特典為戀獄塔追加劇情《戀獄塔 斷罪瑪麗 true end》，實際上是本篇劇情的後日談
    • 在官網有連載前傳小說《獄中童話前夜譚》，全10話+EX

系統[編輯]

迷宮探索[編輯]

  • 迷宮構成為《Wizardry》系列開創的第一人稱視角迷宮
  • 探索迷宮途中可以獲得「血晶」供轉職、裝備技能擴充與強化武器等用途
    • 二代與Finale另有種血系統可以藉由種植血晶收取道具

濺血[編輯]

  • 藉由戰鬥的濺血可以使角色的濺血量表累積
  • 累積到全滿時可以發動滅絕模式，擁有各種強大技能
  • 不過若在承受傷害或隊員陷入無法戰鬥等負面影響會導致濺血產生汙穢
  • 累計過高的汙穢會使角色陷入敵我不分的血狂模式
  • 解除血狂模式的唯一方法為傑克的血
    • 一代需消耗血槍的血量發動，在血量不足時發動傑克會陷入昏迷
      • 血槍的最大容量與可使用能力可藉由交付道具給悠來解鎖
    • 二代發動時傑克的精神會陷入不穩，精神崩潰時傑克會暴走並攻擊同伴

殘殺鬼捉人（虐殺鬼ごっこ）[編輯]

  • 在登上迷宮上層摧毀核心之前，無法擊敗迷宮的BOSS「夢魘」
  • 在此之前遭遇夢魘時會觸發「殘殺鬼捉人」，必須在限制條件下逃離夢魘的追殺
  • 逃跑到一定距離前夢魘會持續追殺我方直到全滅，途中如果發生遭遇戰會形成夾擊戰，必須同時承受遭遇敵方與夢魘的攻擊
  • 藉由持續攻擊夢魘可以使其昏迷用來爭取逃跑時間

隊伍切換系統（ザッピングシステム）[編輯]

  • Finale為複數主角同時進行的多主線劇情，序章主角群分為六組各自攻略迷宮，進入第二章後兩兩合併分成三組攻略迷宮
    • 傑克＆瑪麗組：
      • 出擊組：傑克、瑪麗、夏洛特、愛麗絲、鶴、人魚公主
      • 後勤組：上島視子（研究所）、盈＆千昭（物資販賣所）、守＆光（委託所）、賀東（武器製造所）
    • 庫菈菈＆小祝組：
      • 出擊組：庫菈菈、小祝、長髮姑娘、葛麗特、哈梅爾、睡美人
      • 後勤組：樹（研究所）、楓（物資販賣所）、濟（委託所）、拓海（武器製造所）
    • 十＆小紅帽組：
      • 出擊組：十、小紅帽、拇指姑娘、白雪公主、輝夜姬、灰姑娘
      • 後勤組：賈巴沃克（研究所）、宮內（物資販賣所）、門脇永遠（委託所）、悠（武器製造所）

登場人物[編輯]

血式少年[編輯]

  • 傑克（ジャック）【CV：酒井廣大】
    • 角色原型：英國童話《傑克與魔豆》
    • 血式能力：
      • 一代 / Finale：受難血槍（メアリガン）：遠程滅火，隨強化提升射程，最終強化後可以藉由血槍的反作用力後退五格（可跨過洞穴）
      • 二代：仙境洞穴（ワンダーホール）：在迷宮設置臨時儲存點
    • 一代主角，現存唯一的「血式少年」
    • 其血液可以中和血式少女的血狂狀態，加入黎明後受悠贈與受難血槍，作為血式少女們的後盾
    • 因為自身血式衝動而對於登上監獄塔一事抱持憧憬
    • 二代在逃獄時為保護人魚公主而重傷並化為夢魘·傑克，雖然幾乎喪失語言能力卻仍然保有理性
    • Finale主角之一
      • 遭處刑架少女攻擊之際與鶴、視子和守一同被光傳送走，為了找回下落不明的同伴而展開旅程
      • 在擊敗賀東後對魔具許下願望，創造出一個沒有神獄與夢魘的新世界，即是《戀獄塔true end》的世界
  • 庫菈菈（くらら）【CV：中島唯】
    • 血式能力：受難血槍（メアリガン）
    • 悠的徒弟，一代為陣地商人擔當
      • 終盤進入猜犯人環節時一度代替悠成為陣地武器改造擔當
    • 二代在迷宮內擔任流浪商人，並未屬於黎明組織
    • Finale主角之一
      • 遭處刑架少女重傷陷入瀕死之際被視子輸入傑克的血而存活，同時也覺醒了血式少年的能力
      • 甦醒之後被光傳送到哈梅爾與葛麗特身邊，正式展開旅程
  • 十（ジュウ）【CV：酒井廣大】
    • 血式能力：受難血槍（メアリガン）
    • Finale主角之一，記憶喪失的血式少年
    • 為了尋找失蹤的妻子而進入神獄塔，意外發現受傷的白雪公主與輝夜姬後與她們一起行動
    • 實際上是史納克的轉生體，「妻子」則已成為神獄塔的AI

血式少女[編輯]

  • 愛麗絲（アリス）【CV：上坂堇】
    • 角色原型：路易斯·卡羅作品《愛麗絲夢遊仙境》
    • 職業類型：戰士
    • 血式能力：
      • 一代 / Finale：仙境洞穴（ワンダーホール）：在迷宮設置臨時儲存點
      • 二代：兔子跳躍（ラビットジャンプ）：跳躍五格距離
    • 一代女主角，傑克的幼馴染
    • 因為自身血式衝動而對開茶會非常感興趣
    • 二代在序盤就因為血狂而陷入暴走狀態，在終盤才再度出現
      • 實際上加入隊伍的愛麗絲是神獄產生的擬態，真正的愛麗絲早已化為夢魘
    • 在Finale一度重傷但被某人救起，與瑪麗跟夏洛特會合後與她們一同行動
  • 小紅帽（赤ずきん）【CV：大久保瑠美】
    • 角色原型：鵝媽媽童謠《小紅帽》
    • 職業類型：多面手
    • 血式能力：開膛剪刀（ボディニッパー）：破壞鐵網障礙
    • 最初覺醒的血式少女，在團隊中如同大姐頭一般的存在
    • 自小被十島博士收養，將其視為父親般尊敬
    • 因為自身血式衝動而總是在頭上覆蓋帽子
    • 本作PTSD擔當，各作都曾遭遇嚴重精神創傷
      • 一代在拯救人魚公主的任務中意外將對方殺害，本人對於這件事相當懊悔
      • 二代因為目睹謎之夢魘殘殺黎明本隊而導致性格偏執化，一個人獨處時甚至會啃咬帽子
    • 在Finale為主要角色之一，與拇指姑娘和灰姑娘會合後一同行動
  • 拇指姑娘（親指姫）【CV：高橋李依】
    • 角色原型：安徒生童話《拇指姑娘》
    • 職業類型：魔法師
    • 血式能力：迷你公主（プチプリンセス）：將特定道具縮小或回復成原本大小
    • 被太陽教團收養的三姊妹[1]之長女，傲嬌
      • 雖然是長女，不過體型相當嬌小，本人也對此相當介懷
    • 因為自身血式衝動而喜歡躲在狹小的地方
    • 二代因為千昭死亡而成為教團陽司，也因為誤以為襲擊千昭的人是灰姑娘而憎恨她
    • 在Finale與灰姑娘和小紅帽會合後一同行動
    • 在某個異世界的身份是紅魔族的天才魔法使
  • 白雪公主（白雪姫）【CV：高野麻里佳】
    • 角色原型：格林童話《白雪公主》
    • 職業類型：科學家
    • 血式能力：劇毒炸彈（ポイズンボム）：設置可破壞特殊牆壁的炸彈
    • 被太陽教團收養的三姊妹之次女，冒失
    • 因為自身血式衝動的影響而非常害怕照鏡子
    • 二代因為千昭死亡而成為教團陽司
    • 在Finale遭處刑架少女攻擊而重傷，在十與輝夜姬的幫助下得以返回營地接受治療
  • 睡美人（眠り姫）【CV：長久友紀】
    • 角色原型：鵝媽媽童謠《睡美人》
    • 職業類型：射手
    • 血式能力：薔薇箭棘（ローズアロー）：射出箭矢，可以探勘前方地形或是啟動位於遠方的按鈕
    • 被太陽教團收養的三姊妹之么女，無口
    • 因為自身血式衝動而有嚴重嗜睡的毛病
    • 二代因為千昭死亡而成為教團陽司
    • 在Finale與長髮公主遭遇與姊妹失散的小祝，三人一同行動
  • 輝夜姬（かぐや姫）【CV：五十嵐裕美】
    • 角色原型：日本民謠《竹取物語》
    • 職業類型：射手
    • 血式能力：竹子護罩（タケプロタクター）：在十秒內不受地形傷害
    • 乘坐巨型載具「竹子一號」的少女，性格懶散
    • 因為自身血式衝動而對寶物特別執著，一代得幫她收集五個寶物才會加入
    • 因為自小遭養父母虐待而憎恨人類，也是第一位知道血式少女誕生真相的人
    • 在Finale為了拯救重傷的白雪公主而向十求助
  • 灰姑娘（シンデレラ）【CV：今井麻美】
    • 角色原型：格林童話《灰姑娘》
    • 職業類型：戰士
    • 血式能力：12衝刺（12ダッシュ）：在十二秒內迷宮移動速度提升，可以迴避易碎地形
    • 第二位覺醒的血式少女

      • 耳環丟失時會陷入嚴重不安

    • 由於血狂期間年齡不會增長，與主角團會合時仍舊維持年幼模樣

  • 葛麗特（グレーテル）【CV：種崎敦美】

      • 同時也是本作的智囊團擔當，在團隊中總能最先洞察真相
    • 在Finale因為被處刑架少女攻擊而血狂化並攻擊同行的哈梅爾，在庫菈菈的協助下恢復正常
  • 長髮姑娘（ラプンツェル）【CV：小澤亞李】

    • 職業類型：多面手
    • 血式能力：長髮飄飄！（ヘアー！）：伸長頭髮勾取遠處的道具與機關，前方沒有機關時也可以作地形探勘之用
    • 一個人在舊河川沼澤迷宮棲息的少女，初登場時因為全身長髮覆體一度被當成怪物
    • 雖然是血式少女中最年幼的，不過因為自身血式衝動而總是渴望生孩子…
    • 在Finale與睡美人遭遇與姊妹失散的小祝，三人一同行動
  • 哈梅爾（ハーメルン）【CV：今村彩夏（1、2）→河野日和（F）[2]】
    • 角色原型：格林童話《哈梅爾的吹笛人》
    • 職業類型：魔法師（一代 / Finale）→戰士（二代）
    • 血式能力：磁力短笛（磁力ピッコロ）：吸引特定物件，二代可以用來收割種植的血晶
    • 自稱魔王的少女
    • 總是講話咬舌頭的元氣笨蛋，受到漫畫影響對於捉迷藏遊戲相當熱衷
    • 因為自身血式衝動的影響，非常執著於約定跟整齊列隊
    • 一代在隱藏迷宮「地下洞窟迷宮」的門口登場，擊敗後加入隊伍
    • 二代作為序盤BOSS登場，同樣在擊敗後加入隊伍
      • 因為憧憬有同伴的生活而從魔王轉職成勇者
    • 在Finale被血狂化的葛麗特攻擊，幸得庫菈菈協助才逃過一劫
  • 鶴（つう）【CV：阿部里果】
    • 角色原型：日本民謠《仙鶴報恩》
    • 職業類型：天鷹之主
    • 血式能力：竹球編織（バンブーウィーバー）：設置竹球填補特定空洞，僅二代不需消耗SP
    • 二代主角，自小便與人魚公主結為夫妻

相關名詞[編輯]

  • 神獄（ジェイル）
    • 構成本作迷宮的不明生物，襲擊日本後在地下形成迷宮並藉由拷問當地居民以滿足欲望
    • 核心被破壞會導致神獄成長，因此黎明組織打算藉由此現象讓神獄塔突入地面使群眾脫困
    • 實際上神獄成長到最後階段時會與月亮「受精」化為「將願望最大限度擬態實現」的特殊核心「魔具」
  • 童話（メルヒェン）
    • 本作雜兵擔當，神獄利用擬態能力構成的姿態
    • 地上人類稱之為汙染動物
  • 夢魘（ナイトメア）
    • 本作BOSS擔當，生存在神獄中的巨型怪物
    • 在闖入迷宮時會不定時出現追殺玩家
    • 一般而言是不死身的存在，不過在對應的迷宮核心被破壞後就會失去不死的能力
  • 核心（コア）
    • 位於各迷宮頂端，如同生命中樞的存在
    • 因為是維繫夢魘不死的力量來源，在核心周圍會設有童話把守
  • 血式少女
    • 本作可控單位擔當，對抗童話與夢魘的存在
    • 鑑別血式少女的方式為童話的血，在其身上塗抹的話眼睛會變成粉紅色
    • 血式少女的命名各自對應相關童話，多半也有著相關的血式衝動
    • 實際上是由被接觸童話故事資訊的神獄附著在孕婦擬態後生下的嬰兒，因而為繼承了童話故事主角記憶的轉生體
    • 地上人類稱之為滅絕粉紅（ジェノサイド・ピンク）
  • 黎明解放戰線（黎明解放戦線）
    • 持續與神獄對抗的人類組織
    • 在「隊長」為首的調查部隊全滅後，目前以十島博士為領袖，並安排小紅帽等血式少女組成血式部隊進攻神獄
  • 太陽教團（タイヨウ教団）
    • 以大陽女「盈」為中心成立的宗教組織
    • 實際上是千昭為了保護智能低下的姊姊而設立的
  • 處刑架少女
    • Finale登場，蹂躪地上人類的神祕四人集團
    • 擁有凌駕一般夢魘的強悍實力，就算是歷經風霜的主角團隊也未能與之抗衡
    • 實際上與血式少女同樣是神獄附身孕婦擬態生下的嬰兒，也同樣被地上人類稱為滅絕粉紅
