本身為前段班八年級生，一般的高雄市民，表面上有中度自閉症但是知道跟一些熱門動畫相關之特殊祕密的社會下層人、曾經夢想過當社會學家或者歷史學家。

目錄

喜歡的地方編輯

  • 跟火車、電車相關的所有一切。
  • 喜歡觀賞在一瞬間的當下發生的事件與歷史，尤其是新動畫的第一次全台首播第一集的一瞬間。
    • 以前由於不知甚麼原因的經常錯過熱門動畫的第一集，所以我喜歡從頭開始看。
    • 社會議題動畫也多少有關注，曾經在上大學之前多次看過妄想代理人。
  • 更喜歡非常心動的人物設定，例如動物朋友第一期、光之美少女系列的第三代與第五到七代、皿三味等。
    • 曾經深深著迷過KERORO軍曹的人物設計，還有洛克人EXE系列的。
    • 當然妖精森林的小不點和塔麻可吉的人物設計也感覺觸動人心。
      • 所以我對於新世代動畫的態度就不會是針對我們這一家動畫的那種程度了。
  • 親手畫出高彩度的水彩畫。
  • 埔農的書，例如《台灣受虐症候群》系列。
    • 愛爾蘭哲學家Richard Kearney的其中一段名言由此來的。
      • 同時我也因此知道邱新德先生的著作。
  • 觸動人心的印象專輯。
    • 其中《Anne Happy♪》的印象專輯聽了之後很想跳舞。

討厭的地方編輯

  • 我討厭不重要這句話，因為我親眼看過的事實曾被外界認為不重要。
    • 然後也看到2008年的前輩們編輯的多數心血十年後被叫做kerolf666的編輯者給毀棄，在臉書出現之前的一切情況變成不存在。
    • 另外一位叫做A900788的用戶喜歡用維護清理這個理由來把以前記錄到的確切全台首播時間給全部毀掉與抹消、倒退回以前大約2010年之前條目相當落後的樣子。
  • 把未來遠景給徹底覆滅與抹去的強勢維基百科用戶。
    • 因此我才能知曉一些過去很少人知道的跟一些熱門動畫相關之特殊祕密，今後我不會忘掉但不能使其幻滅。
    • 所以凡是在我記憶或者印象中相當深刻的部分我通常會毫無猶豫立刻寫出來，儘管我還是會有不懂的地方，但只要確定是真的都會寫下去。
  • 非常難看但台灣所有電視頻道上卻還是繼續連環重播的動畫，特定主題曲和不合理要求。
    • 我之所以會把當初的《不行不行之歌》當作是黑歷史──現在該稱呼為「黑暗記憶」──就是因為很受不了如同洗腦一般的音樂節奏的關係。
    • 原本我支持海賊王動畫的香菸馬賽克，2012年後我決定要反過來。
  • 中國的歧視性單字
    • 例如生草值，如果沒有那位嘲笑我部落格生草值升高的親中網友的話，我可能永遠不知道中國人的心性有多壞。
    • 當然也不會想看邱新德先生的著作。
  • 狂熱的親共產黨(特別是中國)的網友
    • 別問我為何全世界共產黨百百種，卻只有中國的敢反抗。

編輯方針編輯

  • 一開始是有感而發想填上前輩們沒有寫上去的登場人物與空缺，最初烏龍派出所角色列表的路人是其中一個傑作。
    • 在發生群體場面描寫動畫刪除事件之前我沒有太多激進的想法與用字，而且至少是還沒有被大量抹去之前。
      • 以前因為在烏龍派出所條目發生過重要不重要的編輯事件，不過在看到全台首播時間還在後我不生氣了，決定改正。
      • 一部份來自於《神薙》的美術社社長所說的「把全部的實情一次清楚地表現出來，外界就不會有太多質疑」。
    • 如果Pixiv百科以及維基百科、Nico大百科對於某件事的說法三者完全一致就會寫上去。
    • 然而2017年2月發生群體場面描寫動畫分類刪除事件、外加上一兩個月後烏龍派出所角色列表中辛苦寫出來的許多事件人物被大量刪除與掩蓋的關係，在抵抗不了之下另外創造出烏龍派出所這個條目，然後完全解釋出原本在維基百科上被激進的編輯者企圖掩蓋的一切。
    • 除此之外還有加上我根本不清楚有一些貶抑流行語是在甚麼背景下出現的，所以我想找出用語最初發生的時間點，尤其是臉書根本還未在台灣出現的2001年至2008年之間。
  • 所以現在能查證的我會去查證，因為若不能提出確切證據大家就不會相信。
    • 因為假消息比以前多更多，現在果然一語成讖。

自己編寫的條目編輯

  • 航海王
    • 因為尚未到完結篇，評論有些很敏感還不能完全蓋棺論定，所以暫時把部分負面想法給移除。
  • 來自深淵
    • 少數讓本人想認真寫的條目其中之一，理由是有新番實況資料，而且非常喜歡美術監督增山修的著作。
  • 連環重播
    • 完全非自願創立的條目(自認)。其實當時如果沒有因為烏龍派出所的正確全台首播時間屢次被掩蓋，我不會當場憤而填補該空隙而創立該條目。
      • 另外，除非現實世界中台灣各大電視台播映的所有節目有實質上的改善與變動的話，我會直接將該條目更名，不然的話絕不同意對自己有仇的人擅自更改名稱。
      • 原先我有考慮過要更名的，然而天竺鼠車車在台視頻道開播之後，由於真正意義上不斷連環重播的動畫出現了，無論自身風評多差我已經不可能會有更改的想法。
  • 水果冰淇淋
    • 原為非自願與計畫中的條目，然而這節目中有一半跟台灣ACG發展情況有關，以及小動畫沒有名稱、資料的黑洞比預料中還大的緣故所以記錄下來。
  • リトルウィッチアカデミア
    • 新番實況保存庫有當時的資料所以也是有認真詳實寫，但不是完全本人意願。
  • 大久保瑠美
    • 本身是應景用聲優條目。
  • 台視
    • 兩年前我已經有此計畫了，只是中途受阻而已。
  • 鬼太郎登場人物一覽 PART1、鬼太郎登場人物一覽 PART2
    • 最近鬼太郎第六季播完了，想先整理一下登場人物。

後來認為很屈辱的狀況編輯

  • 以前維基百科具有多樣性的發展，然而我看到的是在未知原因下變質，編輯方針變成強制善意編輯、嚴禁說出一切不利於相關條目的真相(既使已經完全證實)、不准編寫更新情況相差十年以上的落後條目這些(還有所謂的「維基百科不是甚麼」這個現在已經開始變質、落伍的東西)，既使真的不是要惡意反抗的也會受池魚之殃給封禁。
    • 原本當初在2013年我加入編輯時有一些用戶的巡查豁免權與相關頭銜確實是正直的，我並不討厭他們且保持中立。
    • 在上述的2017年發生群體場面描寫動畫刪除事件之後發現他有巡查豁免權並且在日後一年內熱門動畫條目(例如烏龍拜出所、魔法少年賈修、我們這一家)的登場人物列表中以不重要的理由大量抹去僅有一回出現出現的登場人物，我因次產生出厭惡感。
    • 上面寫有詳細播映紀錄的【節目的變遷】模板大量被抹去，我多次要重新寫過但是有個A900788的用戶以「維護清理」為名義給回退然後引起爭論，這就是我為何多次去圖書館的報刊室查找舊報紙節目表的原因。
    • 然後外加上當初親眼看到的烏龍派出所小單元【今日的損壞物品】寫在維基百科上被他以不重要為理由多次刪除，沒有這個情形就不會產生出「不重要效應」單字來。
      • 這個不重要效應就是：只要在某些條目中其中幾段「事實」是有任何一位跟激進的編輯者kerolf666那完全相反的想法一樣或者是更上一層樓的、而且多次有知情的人寫上去卻一直長期被刪除掉，導致在很長一段時間以來就會有相當多的閱覽者就會認為那事實就是完全不重要的效應。
    • 沒有人會在乎說台灣史上第一台智慧型手機是甚麼型號，就算以前的網頁也有大概都消失了。
      • 然後在《復仇者聯盟-史上最強英雄戰隊》明明當初主聲道是英文、副聲道是中文之跟尋常日本動畫完全相反的這種華視當初打破立體雙語常規的不尋常狀況已經成為現實、然而激進的編輯者kerolf666卻以沒有來源或者以莫須有的不重要為由一直刪除和拒絕承認現實，導致了這一個不重要效應所造成的其中一個結果之一。
  • 由於已經有好心人幫忙恢復一部分老舊熱門動畫的【節目的變遷】模板的關係，我想點到為止不再對他生氣。

台灣第一條目的消滅未來發展可能性的行為編輯

  • 本人在維基百科的台灣第一條目中曾經以「未來將會有大大發展與新發現的可能性」之下寫了預定在未來某一天將會填寫的頭銜空格，但是一直被激進的編輯者kerolf666給毀掉。
    • 然後現在激進的編輯者kerolf666卻一直不放棄說要毀掉未來新發現的可能性，使許多人陷入了甚麼都不知道而且還因此認為不重要的情形。
    • 在2018年後還多出了對這個都不懂的outlookxp這個用戶來協助激進的編輯者kerolf666毀掉可以去發現有發展潛力的未來。

本人真正的心聲編輯

  • 因為我實在是不忍心看到應當要公諸於世的真實被掩蓋，所以我決定要如此描述。
    • 還有一個原因就是出現一個神臍小捲毛看太多，然後在那之前又被長期洗腦的觀眾給刪除攤在陽光下的事實之人。
    • 非特別狀況我不會去接近隨便亂罵對方是「私貨」的某人，然而他的性格愈來愈像當時死也不肯接受既定事實的kerolf666一樣。

真憑實據編輯

  • 我在過去的維基百科上編輯在登場人物列表上的資料是因為要讓後來下一個世代的觀眾知悉，然後希望能在這片領域上發展出青出於藍的繪師與跟那些捏他相關的著作，同時在台灣之最與台灣第一條目上找到在過去二三十年間從未發現到的新的頭銜，然後藉此引出後來知道一些新的頭銜的用戶來填上去，我原本在這方面的性格沒現在這樣如此偏激。
    • 主要就是我很喜歡愛爾蘭哲學家Richard Kearney的其中一段名言：
 事實若沒有因被記述而流傳，就等於沒有發生過。任何真實發生過的歷史，都必須以被講述和紀錄的形式流傳下來。
  • 然而後來發現到kerolf666持續跟名言中的本意背道而馳我實在是非常心痛，因為他否定了過去從2007年草創初期以來超過十年很多前輩們的努力，而且他把台灣第一條目未來將要填上的空白頭銜給大量刪除光，若不是因為【群體場面描寫動畫分類】被他刪除不然我可能睜隻眼閉隻眼。
  • 被封鎖帳號的九個月內，我決定不再對以前一些帶有偏見的動畫產生出怨念來。
    • 可是烏龍派出所的事情，以後不能那樣說算了(我打算在華視頻道不再繼續連環重播的一年後放下那一切的，然而…這也讓我確定了未來反烏龍派出所動畫的方向)
  • 然後借用陳破空先生的名言：甚麼群像劇的理論都是假的，只有群體場面描寫動畫的一切才是真的。
    • 即使對方以現代中國人心中的劣根性辱罵我王八蛋，我也不會屈服。
