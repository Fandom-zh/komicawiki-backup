解說[編輯]

  • 平成假面騎士第五作
  • 監製是日笠淳
  • 前期劇本是今井詔二，後期是原作破壞者會川昇
  • 第一套有兩首主題曲的假面騎士作品（現時除本作外有同樣情況者就只有《響鬼》）
    • 前30話的OP是《Round ZERO〜BLADE BRAVE》，而之後的OP是《ELEMENTS》
      • 但以東映的作品來說，年番會替換OP是很罕有的事；因此換OP就是代表前期劇情不受歡迎/玩具賣不出去，高層武力介入要製作組進行劇情大改造的開始…[1]

登場人物[編輯]

假面騎士[編輯]

  • 劍崎一真（演：椿隆之）
    • 本作主角，假面騎士Blade的變身者
    • BOARD破滅之後住進虎太郎的家
    • 因為長期一個人生活，所以料理技術不差
    • 個性明朗快活
    • 因為年幼時父母雙亡，自己卻無力拯救的慘劇、對Undead危害人類的憤怒，與對人類的愛成為其戰鬥的原動力
    • 前期和前輩橘比較起來還顯得不夠成熟，遇到Undead多半都會陷入苦戰
      • 可是隨著多次的戰鬥，已能獨當一面，成為出色的假面騎士
    • 完全沒有任何用錢的計劃，到領薪水前就已經就已經把上個月的錢用得差不多了
    • 和始的關係從開始的敵視，到後來已經將他視為重要的友人，也因此和對始有著不同態度的前輩橘與睦月的關係有些緊張
    • 為了拯救始而不斷使用King Form讓自己成為另一個Joker令Battle Fight繼續下去，然後為了停止爭鬥而消失在眾人面前不再出現[2]
      • 其Joker名為Blade Joker（ブレイドジョーカー）（小說版設定），樣貌部分類似Blade，顏色主要偏藍，武器以長劍為主
    • 劇場版在封印了所有Undead的四年後，成為了一名清潔員。至於Blade的名號由妹妹繼承，開始第二代仮面ライダー剣的傳說
    • 在EX-AID外傳以及Zi-O時回歸
    • 和橘所率領新世代三騎士關係很差，雙方常常吵架

      • 在看到劍崎為了拯救始而不顧一切的努力後，自行去挑戰Joker想將其封印但失敗而受到重傷，最後因此退出戰鬥

BOARD成員[編輯]

  • 廣瀬栞（演：江川有未）
    • 廣瀬義人的女兒，直屬於烏丸
    • BOARD破滅後和劍崎同樣住進虎太郎的家
    • 負責使用Undead搜尋器標示Undead出現位置，並通知劍崎和橘去對付Undead
    • 總是被安上暴力女屬性被改造實驗體B抓住也不會受傷
  • 烏丸啓（演：山路和弘）
    • BOARD所長，以「但我不會道歉」一句而聞名的男人
      • 原因是他堅信成為BOARD一員的橘會以勇氣克服Rider System的缺憾，可是然後還是道歉了
      • 連中之人也吐嘈自己遇到這樣的情況第一時間就會道歉
    • 堅決反對著Undead解放計畫，在廣瀨義人解放Undead後，奉天王路的命令開發了Rider System來重新封印Undead
    • 在BOARD被蝗蟲Undead毀滅後被橘所救，但橘的真實目的是想找出Rider System的祕密，因此被橘用藥麻醉囚禁起來
    • 之後被伊坂帶走，並施以精神控制讓他開發出Lengle
    • 伊坂被打倒後，為了找出更多有關Undead的訊息而出國
      • 期間將強化Blade、Garren的強化裝置寄給劍崎等人
    • 為了不重蹈橘的覆轍，回國後裝成惡魔對新Garren加以試探
    • 最終決戰後，將所有已封印的Undead卡片帶去隱密的地點加以封印
    • 劇場版中和橘一同將所有被封印的Undead卡片帶往隱密地點加以封印時，被白Joker襲擊而亡
  • 桐生豪（演：增澤望）
    • 前BOARD職員，是在橘之前的假面騎士Garren的適合者
    • 因為實驗事故失去右手，在將Garren託付給橘之後辭職
    • 之後，因為對假面騎士的執著產生了心靈上的扭曲，開始以能放出強力電流的義手將犯罪者以私刑殺害
    • 之後趁機搶走了Lengle的腰帶，並依此行使自己的「正義」
    • 最後和為了阻止其暴行而復出的橘對決後敗北，最後被Undead殺害
  • 廣瀨義人（演：春田純一）
    • 栞的父親，BOARD的研究員
    • 數年前解放了被封印的大多數Undead之後失蹤
    • 解放Undead的目的是因為妻子罹患了不治之症而死，為了找出Undead為何能有永遠的生命的秘密而讓這樣的遺憾不再發生
    • 真面目其實是被植入了廣瀨義人記憶的改造實驗體B
    • 廣瀨義人本人已經死亡，現在的他是被天王路控制的傀儡
    • 曾捕捉劍崎，目的是為了採取因為使用King Form而逐漸Joker化的劍崎的細胞
    • 後反叛天王路博史被追殺，最終為保護栞而犧牲
    • 對劍崎有好感甚至有愛意[5]
  • 天王路博史（演：森次晃嗣）
    • BOARD的創建者
    • 在Undead被釋放後引咎辭職
    • 其實是他暗中操控廣瀨義人解放了被封印的Undead
    • 為了減少競爭者，所以要烏丸所長開發將Undead打倒並重新封印的Rider System
    • 同時製作出了改造實驗體系列來進行製作人造Undead的實驗
    • 真正的目的是為了成為Battle Fight的勝利者，讓新的物種稱霸地球，自己則成為新物種的領袖永遠統治世界，因此製作出了人造Undead:Cerberus並與之融合
    • 敗給劍崎等人之後，被金居／長頸鹿鋸鍬形蟲Undead殺死
    • 墮落的老七[6]
      • 假面騎士的末日到了

假面騎士Blade (0w0)[編輯]

  • 變身者為劍崎一真
  • 將黑桃A插入Blade Buckle，音效為Turn Up
  • 武器為劍，必殺技是雷電飛踢Lightning Blast和強化版的Lightning Sonic
  • 強化形態有Jack Form和King Form,使用黑桃Q插入Rouse Absorber再用黑桃J或K刷過
    • Jack Form時劍崎同時與黑桃A和J融合，能夠飛行，並以此強化Lightning Slash
    • 本來King Form是指單獨融合卡牌K的狀態，並沒有將13張卡牌全部融合在一起的打算，不過劍崎本身對Undead的融合系數很高，所以才變成了同時支配十三隻Undead的狀態
      • 因為同時支配多數Undead的緣故，使得劍崎成為了近似於Joker的存在，在維持變身狀態下會大幅度的消耗體力，使用過度會和Undead完全融合而變成Joker。
      • 招式名為撲克存在的五卡組合，包括Straight Flush, Four Cards和最強的Royal Straight Flush
    • 皮套保存狀態相當差:頭冠日漸泛黃，後期劇場版可以見到許多部份有膠帶應急修補的痕跡

假面騎士Chalice (<::V::>)[編輯]

  • 變身者為相川始
  • 將紅心A刷過Chalice Rauzer，音效為Change
    • 真相實為以Joker本身的能力變身成螳螂Undead的樣子，同理以同樣方式可以變身成其他Undead，但一定要在JOKER的狀態下；因此有些認為不是狹義的假面騎士
    • 在螳螂Undead的狀態下刷卡只會附加能力，使用Evolution時卻沒變成幽靈螳螂Undead的樣子，有推測指是和劍崎使用King Form那樣會和13張Undead融合的樣子
  • 以弓作武器但遠近戰皆擅長，必殺技是迴轉飛踢Spinning Attack和強化版的Spinning Dance
  • 強化形態是Chalice Wild，使用紅心K刷過Chalice Rouzer變成，並能有效抑制Joker之力
    • 13張紅心會合為一張Wild，Chalice的最強攻擊卡

假面騎士Lengle (0H0)[編輯]

  • 由伊坂開發的騎士系統，（最初與及最終的）變身者為上城睦月
  • 將梅花A插入Lengle Buckle，音效為Open Up
  • 使用長棍作武器，必殺技有Blizzard Crash和Blizzard Venom，都有效地用Blizzard的效果令對方無法反抗
  • 相較於其他三名騎士，是劇中唯一沒有強化型態的，不過擁有多種的特殊卡牌
    • 十年後（2014）S.I.C.總算設計出Jack Form

Undead[編輯]

  • 地球上所有生命的始祖，無論用任何方式都無法殺死的不死生命體
  • 在一萬年前互相戰鬥、決定哪一個種族能繁榮的53隻怪物
  • 戰敗者會被封印[8]，最後餘下一隻可成為地球的支配者
    • 但若最後的勝出者是第53人的Joker的話，世界就會進行洗牌而被毀滅
    • 最後不論是TV版還是劇場版都湊齊了54隻Undead（撲克牌的數量）

主要Undead[編輯]

  • 伊坂／孔雀Undead（演：本宮泰風）
    • 鑽石Jack，Lengle的製作者之一
    • 利用Garren來完成新騎士系統，不過最後不敵克服了恐懼的橘
  • 蜘蛛Undead（CV：鹽野勝美→梁田清之）
    • 梅花Ace，戰鬥力極強
    • 雖然被橘封印，但因為封印不完全，仍能影響睦月的心智
    • 最後被睦月借助城光和嶋的力量封印
  • 吉永美雪／蘭花Undead（演：肘井美佳）
    • 紅心Queen
    • 利用美人計迷惑虎太郎，並藉機破壞Undead搜尋器以妨礙騎士們的行動
    • 先後利用山羊Undead、鷹Undead與象Undead企圖消滅Chalice，最後仍不敵Chalice而遭封印
  • 矢澤／山羊Undead（演：是近敦之）
    • 黑桃Queen
    • 曾多次使用昭和怪人最常用的挾持虎太郎，但最終在Chalice的協助下被Blade打敗並封印
  • 高原／鷹Undead（演：林泰文）
    • 黑桃Jack
    • 一萬年前Chalice（螳螂Undead）的強敵（とも）
    • 在吉永美雪／蘭花Undead的教唆下為確認相川始／Chalice的真身而偷走其卡片
    • 最終因誤當Chalice是螳螂Undead的情況下被Blade打敗並封印
  • 新名／狼Undead（演：加加美正史）
    • 紅心Jack
    • 可以操控屍體
    • 起初對劍崎等人謊稱自己是前BOARD的成員暨Undead獵人的領導
    • 真正目的為利用他們改造自己的坐騎「黑色獠牙」
  • 大地／象Undead（演：成田浬）
    • 梅花Jack
    • 雖然總是以「太麻煩了」為由迴避戰鬥，不過其戰鬥力卻相當強悍
    • 本篇第一位察覺相川始／Chalice真身的Undead
  • 嶋昇／狼蛛Undead（演：相澤一成）
    • 梅花King，對參戰不感興趣，沒有為種族繁榮而爭戰的本能
    • 烏丸所長的朋友，將Rouse Absorber交給劍崎等人
    • 擁有操控記憶的能力，用於對望美隱藏睦月的身分
    • 為了控制支配睦月人格的蜘蛛Undead，自願被睦月封印
    • 策劃了讓橘遺失自己的卡片，令激戰中的睦月企圖進入King Form，結果成功聯同城光一同把蜘蛛Undead迫出來，結果睦月使用了嶋昇的力量一度和蜘蛛Undead互角，然後劍崎借劍一刀收拾了蜘蛛Undead。

      • 初登場看起來還像美女

    • 其實是個好人[9]

    • 最後被封印的Undead

名詞介紹[編輯]

  • BOARD
    • 故事開始時，劍崎和橘隸屬的組織
    • 正式名稱為「人類基盤史研究所」，創立者為天王路博史，並兼任理事長一職
    • 創設目的是「找出進化史上未解開的部分，也就是人類為何能成為所有物種的霸主」之謎的研究組織
    • 在探索的過程中，發現了生物始祖、有著不死性質的神祕生物「Undead」與一萬年前，53體Undead為了決定地球物種霸主而互相爭鬥的事實
    • 在研究過程中，研究員廣瀨義人為了解開Undead不死的祕密而解放封印放出了大半的Undead，理事長天王路為了負責而請辭下台
    • 為了將Undead重新封印，於是由所長烏丸啟帶頭，開發了Rider System、組成對Undead的特殊隊伍去對抗Undead
    • 不過在故事開始後不久，Undead就襲擊了BOARD，造成BOARD研究所的毀滅，多數成員也因此死亡，陷入了營運停擺的窘況
    • 現階段是由殘存的成員劍崎、橘、廣瀨栞三人繼續維持對抗和封印Undead的行動
    • 雖然是壞滅狀態，不過不知為何還是會每個月定期支付薪水給職員
