灼眼的夏娜

灼眼的夏娜（Shakugan no Shana／灼眼のシャナ）
以下可能透露部份劇情，沒看過原著又不想被捏到請不要繼續看下去。

目錄

解說編輯

  • 高橋彌七郎原作，いとうのいぢ插畫，日本電擊文庫輕小說系列《灼眼のシャナ》。
  • 以持刀的紅髮強氣蘿及普遍認為略需時間參透的世界觀而聞名。
  • 在2006年與同一位插畫負責的《涼宮春日的憂鬱》同時被改編成動畫，也因此成為了輕小說改編動畫黎明期中的知名動畫
    • 是故在萌戰轉成廚戰(無誤)的2007年之前，在日本以外知名度最高，也最受眾人愛戴的數名動畫角色中，身為女主角(兼書名)的夏娜可說一直是名列前茅的知名角色。
    • 不過相對於涼宮春日因為本身的日常調性與京阿尼的改編發揚光大，夏娜卻因為本身的世界觀，還有動畫改編毀譽參半，連帶也使得作品人氣不上不下，這或許正是導致夏娜變成無冕王的最大原因?
  • 雖說因為原作架構完整，本作在同人界可以亂玩的程度較少而顯得注目度不高，不過另一方面卻因誤字灼眼のシャア，以及香港網民秋原據此創作的四格惡搞漫畫「灼眼的薩克」而變得非常有名。
    • 某程度上，該四格惡搞也是夏娜在台港知名度不小的一大助力。

原作及延伸編輯

  • 原作於MediaWorks輕小說連載誌電擊hp不定期連載，本篇全22卷，另有外傳4卷。原作者將之定位為「精彩刺激的動作小說」。中文版由台灣角川購得版權。
  • 漫畫版由笹倉綾人繪畫，於05年4月開始在電擊大王連載，目前出版至3卷，以萌見稱。日版2卷特典GRIMOIRE是由眾多插畫師及漫畫家參與製作。原作小說10卷亦分別將於07年9月開始漫畫化，作者木谷椎。
  • 動畫版由J. C. Staff製作，於日本05年10月至06年3月撥映，第一季共24話，相當成功的表達本作的萌及戰鬥部份。不過臨時直入的原創結局看得很多觀眾露出一副囧臉。
    • 動畫版第二季於07年10月撥出，第三季於2011年10月撥出，已完結。
    • 動畫版台灣中文DVD版權由曼迪傳播取得，已完成發行全卷，香港版DVD由領域映像取得，預計07年末發行DVD BOX SET 1。台灣ANIMAX和香港ANIMAX在07年7月末及8月頭開始撥出。
    • DVD1及DVD5的特典《しゃくがんのシャナたん》及《しゃくがんのシャナたん～リターンズ～》，由迷你版的小夏娜（夏娜炭）帶出一連串故事，充份反映出角色的裡側個性而聲名大燥。
    • 其後連動畫版解說書推出的DVD特典《頂のヘカテーたん》將這個惡搞風氣推至頂峰。
  • OVA於06年12月發行，賣點為泳衣和溫泉，卻因主角被變換而獲得好壞參半的評價。
  • 劇場版於07年4月於日本上映，公佈為原作1卷改編後讓很多人大失所望，但其中使用的戰鬥畫面描寫得到正面評價。DVD採用的Director's cut方式編集，並於07年9月發行。
    • 相較於動畫版採用多本小說連貫改編而使小說第一集劇情顯得面目全非，劇場版則是採用更接近小說第一集的劇情，搭配動畫版比較受到好評的片段剪輯而成。
  • 06年3月發行的PS2遊戲，被移植到NDS上並於07年3月發行。日本方面於06年10月起推出手機用格鬥遊戲，目前共兩代。
  • 另曾推出電擊限定以及動畫版的兩種Drama CD，周邊和Figure也相當多元化。

概要編輯

  • 主角坂井悠二某天被奇怪的生物襲擊，獲一帶有紅髮紅眼的帶刀少女所救，卻因此得知真相：坂井悠二的存在已被平行世界紅世的住民紅世使徒奪去，並成為稱為火炬的代替品，某種定義上已死的他卻身懷寶具而成為紅世使徒的狙擊目標。少女的身份其實是殲滅紅世使徒的火霧戰士，為保護悠二而與他同行。本來無名的少女被悠二取名為夏娜的同時，二人漸漸展開介於紅世戰鬥和校園戀愛的日常生活。

讀懂《灼眼的夏娜》的10個關鍵字編輯

  • 存在之力
    • 世上所有物質存在所需的能量。被奪去存在之力的物質會被視為不曾存在過， 人類的情況為失去在世的痕跡，他人對這人的記憶完全缺失。
    • 存在之力被吞食將會使世界發生扭曲，繼而崩潰。
  • 紅世
    • 這個世界所相對的平行世界．異世界。跟人類所居往的世界有互為平衡的關係，人類世界的扭曲和崩潰將直接使紅世產生相應影響。
  • 紅世使徒
    • 紅世住民的總稱。部份受到存在之力的吸引而來到人類世界，以吞食存在之力而顯現在人世並達成自己的慾望。
    • 其中不少成員被當作人類傳說中的神魔精怪。
  • 紅世魔王
    • 具有較大能力的 紅世使徒的總稱。有部份來到人類世界吞食存在之力，有部份則與人類契約使之成為火霧戰士。基本上越強大的紅世使徒在人世上顯現所需存在之力越多。
  • 火霧戰士
    • 有部份紅世魔王注意到存在之力被大量吞食後對紅世的後果，於是跟人類契約並將之成為火霧戰士，藉以討滅過份吞食存在之力的同胞。
    • 紅世魔王以提供異能作為條件，人類以過去-現在-未來整個存在置換出來，人類化為容器，紅世魔王棲身其中即為火霧戰士。
    • 棲身火霧戰士體內的紅世魔王不需要消耗存在之力即可進行討伐工作，他們經神器向其火霧戰士及其他人表達意志
    • 灼眼的夏娜／火霧戰士三神器：吊墜、書本、女僕頭巾（參見動畫版Drama CD 3）
  • 自在法
    • 使用並操縱"存在之力"所引發，無所不能的現像。了解原理後基本上不論身份均可施展。
  • 封絕
    • 為典型且基本的自在法。造成與外界因果隔絕的空間，基本與結界同義。封絕內造成的損毀可使用存在之力修復。
    • 封絕不是完整的圓球形(因為一半在地底所以看上去是半圓形，根據設定只要邊界上有不可以切割的存在的話，就會把這個存在全部都納入其中。
      • 例如一列多節列車其第一節車廂進入邊界，則會連人帶全體車廂都拉進去其中。
    • 此外如果是河川之類的具流動性、連續性且不可能徹底被納入其中的存在的話，就可以用曖昧的方式呼嚨過去不受影響。
      • 亦即不會有那種所有河川的水都卡在封絕的邊界流不過去而造成大淹水的狀況。
      • 但像是鐵軌的話，就會轉變成「因為發生事故而暫停營運」這樣直接影響到現實世界的狀況
      • 此外封絕外的人類也會潛意識的去避開封絕的區域。
  • 火炬
    • 紅世使徒為緩衝世界扭曲（其實是避開火霧戰士追討），吞食人類的存在之力時使用少量殘餘存存之力作成的代替品。
    • 火炬的表現基本和一般人無異，但不論他人對火炬的關注，還是火炬本身的意識，均會在短期內漸漸變淡薄，最終火炬隨著存在之力用盡而消失。
  • 密斯提斯
    • 載有寶具的火炬。火炬消失後，本來所帶的寶具會隨機轉移到別的火炬身上。
  • 零時迷子
    • 被稱為紅世「秘寶中的秘寶」的寶具，為紅世魔王和人類相戀之下下，讓人類一方成為「永遠的戀人」而製作出來的寶具。意外地落入悠二手中。
    • 能在每日零時自動補充存在之力原始值的永久機關，間接使擁有者有著無窮無盡的存在之力，並且能免於存在之力用盡而消失。
    • 化裝舞會的首腦「祭禮之蛇」，在故事開始前被誘入陷阱，封印於零時迷子中。
    • 故事最後機能停止，但因為新世界有足夠的存在之力且悠二的存在已經被拉米留在「藍天」中的自在式固定下來，因此對悠二而言不成問題。

其他ネタ編輯

  • メロンパン
    • 夏娜最喜歡的食品，被認為是因為作者喜歡而作下的設定。
    • 有因為夏娜這套作品而使日本メロンパン全體銷量上升2%的說法。
    • 2ch夏娜thread裡的經常食品つ（＃）（尤其是給踩中876的人，見下條）
    • 同人糟糕小說貼裡有エロンパン（エ）及メロンパンツ的說法。
    • 台灣角川譯為波蘿面包而非甜瓜面包成為讀者抗議最多的翻譯。
  • 876
    • 因作者名高橋彌七郎發音與高橋876相近，成為作者通用的代號。
    • 876為2ch夏娜thread獨有的キリ番，同理與夏娜近音的487則關注度極低。
  • 小說後記．宣傳語句
    • 每次均用特定模式寫作。每次均會加入超光戰士（超光戦士シャンゼリオン）ネタ。
    • 小說版老編三木（ミキティ）被推定為本作殺必死的始作俑者。
    • 每次宣傳均由老編（？）想出代表性的宣傳語句，基本為「悠二、（下接N字）」，不過也有例外。
  • MMR
    • 原為漫畫「神秘調查班」，主角以抓緊極小的線索解題而著名（換個方法說就是有點強來），被2ch製成AA留傳。
    • 後來AA被使用作解釋「坂井千草處女說」等千草有關命題，自此MMR與千草擁護派成為同義詞
  • 荒川弘
    • 高橋彌七郎與《鋼之鍊金術師》作者荒川弘為未出道已認識的朋友，荒川弘亦曾應邀為小說版解說書畫賀稿。
    • 對高橋彌七郎的金句「燃えも萌えもどっちもすきなんだ！」（燃與萌我同樣喜歡！）回應為「至理名言」。
  • 三冊買い
    • 読書用、観賞用、配布用各卷各買三冊。適合富有人仕。
    • 每次三卷三卷的買。適合初讀者（爆）
  • 魔之35頁
    • 台灣角川中文版小說，連續在第6和第7卷的初版第35頁出現爆笑的植字錯誤。


以下可能含有糟糕內容，純真的未滿16歲青少年或不夠糟糕者不宜觀看。
  • 器（うつわ）
    • 原指作為人類包含的存在之力，所謂命運的容器大小。
    • 火霧戰士的容器容納紅世魔王。夏娜作為「崇高之人」，擁有能包容亞拉斯特爾顯現的巨大容器。
      • 一種程度的主角威能，用以解釋何以夏娜明明讓亞拉斯特爾顯現了自身卻沒掛掉的原因
    • 後來被糟糕引申為能容納某些東西的女性性器，不過這亦反映到男性角色上（見《頂のヘカテーたん》）。
  • 火霧戰士排泄說
    • 2ch夏娜thread定期必會翻炒的糟糕命題。基本針對女性角色。
    • 火霧戰士原為人類，身上保留作為人類的解剖生理，不過因為有稱為「淨化之火」的自在法，作用為對肉體的潔淨消毒，故理論上不需上廁所。
    • 部份住民對此設定抱質疑論，並將命題延伸至紅世使徒，甚至產生有無肛門等等的支派理論。變種命題有「火霧戰士懷孕說」等等。
