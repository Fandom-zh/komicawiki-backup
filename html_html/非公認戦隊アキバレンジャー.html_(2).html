故事概要編輯

  • 第一季
    • 秋葉原被邪團法人傳銷乙入侵，為保護秋葉原原有特色，具有「妄想力」的一男兩女組成「非公認戰隊秋葉原連者」保護秋葉原。←但這一切其實都只屬妄想
    • 但雙方戰鬥催向白熱化之際，一股更強大的勢力正蠢蠢欲動…
    • 每話標題都必定摻有「痛」字
  • 第二季
    • 基本作品名稱一樣，只是作品名稱後面加上了《シーズン痛》（讀音與英文「Season Two」相同）
    • 時間點上為第一季完結後九個月
    • 新惡役組織「新次元改造地下真帝國バロスw」登場
    • 第一話的OP不論歌曲（竟然是Demo roll）還是影片（大刺刺的把分鏡稿貼出來）都是嚴重粗製濫造的未完成版，當然目的是防止破梗（主要為二代アキバブルー及新惡役組織的構成）但オネェダー還是在OP裡出現了，而ED也爆了新アキバブルー的本尊
    • 第一話為「總集篇」（無誤）
    • 你從什麼時候開始產生了世界觀被篡改了的錯覺…？（以下嚴重劇透捏他注意）
    • 在劇組瘋狂吃書下，原第一季6-12話全部被拋入黑歷史
      • 赤木暗戀+妄想對象的さやか因為協助叔父工作關係要離開日本到不丹=踢出戲棚外
      • 「十夜月朱里=葉加瀨博世」的本尊曝露事件因為不重要所以被刪除
      • 原本自第九話以降當妄想及現實世界之間的牆壁被打破後，主角們能在現實世界變身戰鬥的設定被刪，從一開始到第一季結束都只是在妄想世界中戰鬥
      • 傳銷乙科長之一的「下北沢ホヤ」被加上了與瑪露希娜是姊弟關係的荒謬人x獸設定
      • 原本用以扭轉作品走向的都築拓馬/新アキバレッド直接鬼隱，連帶原本被安排作為新アキバレッド宿敵的Deluknight篡改成一個傳銷乙的普通幹部←但他還是個會對小正太出手的變態
      • 都築丈博（博世的父親）改為只是一個普通科學家，在MIT研究機器人，並無失蹤卻也從未登場
      • 當然主角們都被改成了從未意識到「劇組=作品外現實」的存在
    • 第一季末期被追加既不是傳說大戰也不是騎士大戰的妄想大戰，大概是致敬《高速戦隊ターボレンジャー》第一話「10大戦隊集合 頼むぞ！ターボレンジャー」的橋段
      • 與（從名叫「地獄」的謎之洞穴召喚出來的）傳銷乙眾（再再生）科長、Deluknight及瑪露希娜日以繼夜夜以繼日地大戰了三日三夜零7小時
      • 三隊公認戰隊《鳥人戦隊ジェットマン》、《特搜戦隊デカレンジャー》及《轟轟戦隊ボウケンジャー》（除追加隊員外）現身協助
      • 瑪露希娜駕駛流用ドリルサイクロプス皮套的巨大機器人「暗黒ステマ城ロボ」現身，最終被S.H.Figuartsイタッシャーロボ「從頭部那裡射出的光線」變成的碳酸飲料罐擊中自爆按鈕而毀滅
      • 自此傳銷乙被消滅，是一個正統到不行的結局
    • 原本並沒有解散的秋葉原連者被劇組強行拆夥，結局アキバブルー及アキバイエロー因為各自的理由而脫隊，秋葉原連者因而解散
    • 被五角大樓挖角脫隊的人由原本的赤木改為美月
    • 根據劇組在官網上的說法，因為第一季的結局玩太大根本無法續寫下去，故回想劇情裡的第一季是以「假若當初故事沒被劇組亂改的話…」的方向完走
      • 結局在第11話爆梗，上述的一切都是劇組和八手三郎的二次篡改（無誤），除了瑪露希娜以外的所有前作角色都被洗腦並置入了上述的第一季後半劇情
    • 每話標題格式為「妄想○○」，○○可代入任何名詞（大戰、中華、拳劇、女神、輸入…）

主要角色介紹編輯

  • 赤木信夫/アキバレッド（初代）（和田正人 飾）
    • 29歲（第一季）→30歲（第二季）的熱血大叔
    • 在「佐佐木砰砰運送公司（佐々木ポンポコデリバリー，可取羅馬拼音的頭文字簡為S.P.D.←請不要與Special Police Dekaranger或Space Patrol Delta的簡寫搞混了）」工作
    • 戰隊宅，本作中對超級戰隊最了解，自稱為全國第三的戰隊通←第一名是豪快銀，第二名我不能說
    • 每話唱名必定加入歷代戰隊的捏他，但到後面卻出現了「想被哪個女幹部踩」和「想吃什麼料理」之類的問題發言
    • 第二季第4話在スマホモンガー的篡改小葵世界觀及劇透攻擊下讓妄想力見底並失去鬥志，但在優子為他而繪製的同人本「真・劇場版ズバーーン葵 Z-絶倒-」下回復過來，並得到了透過「超妄想」召喚「MMZ-02 ムニュムニュズバーーン」強化變身成超アキバレッド的力量
      • 但他為了替趕稿的優子打氣卻犧牲了許多身為男人的尊嚴…而且超アキバレッド的強化裝甲造型實在是痛得讓人無法直視
    • 小時候十分喜歡於《超獣戦隊ライブマン》登場的支援機器人「コロン」，自此對全身閃亮亮的機娘毫無抵抗力
    • 老婆是櫻田弘的姊姊[2]
  • 青柳美月/アキバブルー（初代）（日南響子 飾）
    • 19歲，高中生
    • 懂得格鬥術，因此在戰鬥中打得最像歷代
    • 最初因為興趣缺缺所以毫不注重變身後唱名，但在經新堀和男的訓練後明白了身為戰士的榮耀，以降的變身唱名改為加入高難度的演武動作
    • 小褲褲是白色，家裡還有收藏印有卡通圖案的保●套
    • 第二季時因為被五角大樓挖角而脫隊，之後會回來的可能性是…？
      • 是說之前赤木就是被劇組用給五角大樓挖角的理由意圖扔出戲棚的大家都忘記了嗎
  • 萌黄ゆめりあ（CN）（本名：山田優子（第一季）→橫山優子（第二季））/アキバイエロー（荻野可鈴 飾）
    • 23→24歲[3]，本職是OL，副業是Cosplayer+女性向人繪師
      • 讓許多同人繪師望塵莫及的達筆，能在半天內趕起64P的小葵本（還連彩色封面…）
    • 每次變身後的唱名都會大談自己的同人之道或者宅知識，但因為太長了直到第一季最後都沒一次能成功說完（？） [4]，第二季能說完的可能性是…？

    • 明明身邊的人都是歌手但自己卻不唱歌[5]

    • 似乎還是學生（下班時有穿水手服離開的橋段）
    • 在戰鬥時協助博世指揮，也擁有一定的戰鬥力但為什麼是拿DX玩具來打
    • 名稱是從身為其演員的愛川來的
  • 都築拓馬/新アキバレッド（第11話）（高木心平 飾）
    • 因為八手三郎武力介入下出現的博世同父異母的弟弟
    • ICPO的特警，為調查ステマ乙而來到秋葉原
      • 實為劇組意圖以他取代掉赤木成為二代目アキバレッド，再將作品扳回正軌
      • 但當劇中人物發現劇組的企圖後就立即被收掉，之後也再沒有出場
    • 其孿生兄弟已擠身於公認戰隊當中（無誤）[7]
      • 因此他成為アキバレッド後整隊變得獸拳味濃厚也是理所當然的
  • 石清水 ルナ/新アキバブルー（澤田汐音 飾）
    • 14歲，代替脫隊的美月成為第二季的秋葉藍[8]
    • 明星夢十分嚴重的腹黑中學生兼不紅的偶像，因為只有東映的特攝片是年番，故為了曝光率而十分希望能成為東映特攝片的女角円谷淚目，甚至還說明了「要用特攝片作為自己下一步的跳板」大概是捏オタギリジョー討厭特攝的流言
      • 為了被東映選上甚至還自製了甄選會攻略，卻把來找新秋葉藍人選的赤木們當成了新版《五星戦隊アキバレンジャー》Pilot回的拍攝人員…
      • 經理人公司沒派工作時甚至會想自己拍片放上Niconicotube搏取人氣，想成為第二個こずこず
      • 啟蒙作是小學時在東映●道看的《五星戦隊アキバレンジャー》（重播），但除此以外的公認戰隊幾乎都不認識
    • 為人極端現實，第3話結末瞭解到自己不是在拍特攝後立即脫隊，第4話更是受博世「聘請」回來幫忙，直至第5話被赤木和優子奮戰的樣子感動才正式加入但依舊是要拿秋葉原連者當跳板
    • 名字部份寫作「美月」讀作「月亮（Luna）」 ←因為姓氏裡有「青」字所以才被選上

客串參演的歷代戰隊成員編輯

第一季編輯

  • デカレッド / 赤座伴番 / 載寧龍二（載寧龍二 飾）
    • 第2話在現實世界登場的是載寧龍二本人，為宣傳日語配音版《Power Rangers S.P.D.》[9]而來到秋葉原
    • 後來因為赤木受到博世用錢聘請跑龍套假扮的デカレッド鼓勵，在妄想世界裡呼應赤木的妄想以正牌デカレッド的身份出現
    • 利用狂妄之力可將他獵奇變成「刑事手銬（デカワッパー）」
  • ボウケンレッド（開運フォーム）[10] （CV：高橋光臣）
    • 第3話時因為喝醉酒的赤木妄想力過高，讓他初出現時穿著一身開運法寶
      • 後來呼應美月的妄想而出現的則是正常版，但授予她狂妄之力時卻順便拈油
    • 利用狂妄之力可將他獵奇變成「冒險鏟（ボウケンスコッパー）」
  • レッドホーク
    • 第6話裡在三人的妄想中出現並協助打退下北沢ホヤ，從出場到最後一句話都沒說過
    • 利用狂妄之力可將他獵奇變成「噴射鋒翼（ジェットウインガー）」

第二季編輯

  • 《中華戦隊チャイナマン》[11][12][13]
    • 本為《五星戦隊ダイレンジャー》，世界被篡改後變成是日本中華料理商協同組合的宣傳用角色
    • チャイナレッド（CV：關智一）
      • 初登場時在送餃子單點外賣…
      • 写メお願いします！！
      • 第三話在秋葉原連者以低成本重現本篇第27話劇情下拾回記憶，回復リュウレンジャー（CV：和田圭市）的身份
    • キバレンジャー（CV：阿部渡）
      • 元《五星戦隊ダイレンジャー》一員，世界被篡改後順理成章地成為了《五星戦隊アキバレンジャー》的追加戰士，但因為作品才播了一季就被腰斬（劇中描述），結果連出場機會都沒有…←豪快銀NDK？NDK？
      • 第三話因為上述的理由怨恨秋葉原連者跑去當瑪露希娜的手下，最後在秋葉原連者以低成本重現本篇第33話劇情下記起自己真正的身份
    • 兩人可以合體獵奇變成龜派氣功波「大連爆擊砲（ダイレンボンバー）」
  • 《Powerful Rangers》
    • 原捏他為美國購入《恐竜戦隊ジュウレンジャー》版權後製作的《Mighty Morphin' Power Rangers》，也是之後的Power Rangers系列的始祖
    • 第5話裡因為ルナ對戰隊起源認知的錯誤（以為PR才是起源）導致世界被篡改，讓《Powerful Rangers》反過來成為了超級戰隊系列的正宗
      • 但與現實世界不同，《Powerful Rangers》的續作是《Wild Force》，連帶讓日本的百獸被美方控告抄襲…
        ↑另外聽說現實中在美國真的有人會這般認為的樣子
      • Powerful Rangers都是很NICE的，這其中一定有什麼誤會
    • Red Ranger（CV：Austin St. John關智一）
    • Green Ranger（CV：Jason David Frank草尾毅）
      • 與原典直接沿用日本皮套不同，兩人在頸項和腰帶上均加入了美國國旗花紋，還操著一口混雜半鹹淡英語的日語襲擊「抄襲」自己的秋葉原連者
      • 在赤木和優子的捨身說得，加上DX獸奏劍的音聲助攻下兩人記起自己的正確起源，回復ティラノレンジャー和ドラゴンレンジャー的身份但聲音沒變回來
