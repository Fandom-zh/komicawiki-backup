目錄

介紹編輯

  • 由ぎゃろっぷ/Konami製作，全146話，以「ZEXAL」作簡稱。
  • 時間設定不明（有說法是5D's扭轉破滅未來後的未來），以「ハートランドシティ/Heart Land City（心之地，又譯心園或心城）」作為舞台。
    • 但從ARC-V和VRAINS的設定來看，從本作開始各動畫均採用獨立世界觀，劇情上沒有關聯。
  • 本作決鬥影像系統採AR決鬥，怪獸不再使用立體投影，而是藉由投射在變形鬥士D・ゲイザー來觀看AR決鬥影像。
    • 雖然遊戲王原作和動畫設定不同，但不論哪一作海馬都早就開發出不須專用鏡片就能觀看怪獸的立體影像投影系統。技術倒退
    • 不過在AR決鬥中可以不依靠超自然力量來實現場地破壞的演出，也不能說倒退
    • 至於巴利安人、星光界人或有關人士則會在每次決鬥前變更瞳孔樣式，讓自身能觀看決鬥影像。異世界人的身體也會支援這系統是ZEXAL一大謎團
    • AR影像只影響決鬥畫面而已，決鬥規則大致上和過去作品一樣。不過表側守備是禁止事項，騎乘決鬥當然也沒有
  • 觀眾群由青年轉向小學生。
    • 早期決鬥也回歸正路，不會有過份變態的效果和戰略。
      • 但到了中期，卡片效果越來越誇張，而為了讓主角勝利也出現大量無效化能力的卡片。
      • 最後Boss戰比前作手牌陷阱還要誇張，陷阱可以直接在牌組發動，甚至修改對方的魔法效果。
    • 第一章在每集結束時會有Q&A環節，教導遊戲王的基本規則，但其實那些基本規則在前三季已經被破壞得體無完膚
      • 第二章則變成搞笑短劇，但每次都是其他角色拿主角來開玩笑……
  • 主角由青年人螃蟹不動遊星轉換成國中生龍蝦九十九遊馬，也是所有主角中最年輕的一位。[1]
    • 同時也是開篇時決鬥技巧最差的一位，連基本規則也不會
    • 另一方面說也是最有成長空間的一位
  • 死人死最多的一篇，在動畫中出過場又幫過遊馬的角色們[2]很多都戰死了，連身為對手的鯊魚和快斗都死了（無誤）
    • 重要角色便當回還會在ED放回憶畫面補刀
    • 又因為官方不自重的總在歌曲唱至「全て壊すんだ」的部份放一些奇怪的畫面而成為話題
      • 128話（玉座兄弟）：全て壊すんだ（顔芸）
      • 133話（梅拉古＆德魯貝＆亞里特＆吉拉古）：全て壊すんだ（カツ丼）
      • 135話（快斗）：全て壊すんだ（ガラス）
      • 143話（米賽爾＆貝庫塔）：全て壊すんだ（無言の手刀）
      • 146話（納修）：全て壊すんだ（皇の鍵）
  • 決鬥以超量召喚與超量怪獸（黑卡）系統為中心。但融合召喚和儀式召喚也有出現，另一方面前作主打的同步召喚和協調怪獸不只沒人使用，劇中也從未提起。
    • 其中一個例子是遊馬初次來到決鬥庵看見了仿照黑魔導、E・HERO新生人、E・HERO火焰翼人等DM和GX的著名卡片做成的雕像，但跟5D's的怪獸有關的雕像卻一個也沒有。
    • 超量怪獸的原文為エクシーズモンスター，但ZEXAL劇中則稱為モンスターエクシーズ。

漫畫編輯

  • 由三好直人作畫，初期劇情遵照著動畫版，不過劇情之後也開始分歧。
    • 值得一提的是，之後三好獲到提拔擔任《遊☆戯☆王 ARC-V》的人設，可喜可賀（？）
  • 與動畫版的差異：
    • 部份「No.」只於漫畫版登場
      • 漫畫沒有巴利安的設定所以也沒有「CX」或「CNo.」，因此不會出現鯊魚妹
      • 緩和了專用牌組促銷的壓力，遊馬比起動畫時更常使用希望皇以外的No.
    • 凌牙童年經歷不同，實力也比動畫強許多
    • 漫畫版的菲卡博士已經身亡

動畫劇情編輯

序章編輯

  • 遊馬在與章魚凌牙決鬥中瀕臨敗北，但是藉由雙親遺留的「皇之鍵」打開了謎樣大門；在獲得力量卻也失去某物的同時，與阿斯特拉爾相遇並召喚出超量怪獸《No.39 希望皇霍普》成功逆轉戰局。
  • 阿斯特拉爾遺忘了一切，只知道要找回記憶就得收集齊99張「No.」怪獸；遊馬雖然不是很願意讓阿斯特拉爾附身，但也只好接受。不過隨著不停的戰鬥，兩人也逐漸變得相信對方。
  • 於回收《No.96 黑霧》時意外使得邪惡意志解放，遊馬和阿斯特拉爾都被其控制；鐵男與之戰鬥卻陷入危機，幸而遊馬只是假裝被其控制，成功婊掉和回收No.96。
  • 遊馬遇上同以No.為目標的弟控快斗，在決鬥中阿斯特拉爾對快斗生出恐懼之情。
  • 阿斯特拉爾克服了對快斗的恐懼，在和快斗手下（自稱，其實只是仰慕者）對戰時喚醒了混沌超量怪獸《CNo.39 希望皇霍普雷》。
  • 快斗搶走遊馬的皇之鍵，直接和阿斯特拉爾決鬥。阿斯特拉爾即將敗北時，遊馬喚醒了「ZEXAL之力」，和阿斯特拉爾疊放（無誤）變身成霸王龍蝦，並同時學會神抽和印卡。最後以霍普雷和快斗打成平手。

WDC初賽篇編輯

  • 為了吸引其他No.擁有者，心之地舉行世界決鬥嘉年華（World Duel Carnival，簡稱WDC），參戰者必須得到五塊「心之碎片」組合成心型才能得到參戰資格。
  • 凌牙為了向Ⅳ報仇而參加賽事，在Ⅲ的手上得到並馴服《No.32海咬龍 鯊魚龍》。
  • 遊馬在賽事打敗了各地的決鬥者並得到四塊「心之碎片」，並成功把神月安娜收入後宮
  • 途中發生晴人被玉座綁票並搶走力量，但最後快斗在晴人手上得到力量，並召喚《超銀河眼光子龍》救回失去力量的晴人。最後晴人昏迷。
  • 初賽最後一日，Ⅲ得到更強的力量並挑戰遊馬。在Ⅲ的力量下遊馬一度失去「勇往翔前！」和阿斯特拉爾，但在其老爸的協助下回復正常，並以「ZEXAL之力」打敗Ⅲ。
  • 遊馬得到最後的心之碎片以及Ⅲ手上的No.。Ⅲ戰後昏迷不醒。

WDC決賽篇編輯

  • 決賽的規則是要乘坐決鬥過山車到達會場，經過淘汰的最後四強是遊馬vs鯊魚、快斗vs玉座。
  • 遊馬在打敗玉座之後，WDC的主辦人菲卡博士露出真面目，其目的為用巴利安世界的力量毀滅星光界，於是遊馬、凌牙、快斗三人聯手打敗菲卡。
  • 最後遊馬與快斗進行場外決鬥，雙方發動相同的卡後由快斗險勝。
  • 至此第一部完，標題也改為《ZEXALⅡ》。

ZEXALⅡ編輯

  • 遊馬遇上了新的敵人—使用RUM的「巴利安七皇」，同時也認識到新的朋友「真月零」。
  • 和吉拉格的對戰中，真月表明自己是巴利安警察，並給予遊馬《RUM-限制型巴利安之力》
  • 操縱菲卡博士的貝庫塔出現並捉走了真月。為了救回真月，遊馬等人只能前往巴利安界。
  • 在和貝庫塔的決鬥中發現真月就是貝庫塔，接近遊馬只是他的詭計之一；最後貝庫塔被遊馬以《RUM-源數之力》打敗。
  • 貝庫塔被打敗不服，擅自喚醒了巴利安之神「千尊」。
  • 因為老爸的留言眾人出發尋找「傳說中的No.」，途中了解到「巴利安七皇」的前世其實是人類。
    • 阿斯特拉爾和No.96決鬥勝出，但阿斯特拉爾被No.96刺殺，為了拯救阿斯特拉爾遊馬前往星光界。同一時間，凌牙在「傳說中的No.」的記憶中發現自己的真正身份。
    • 「巴利安七皇」發動總攻擊，遊馬及時回到人類世界；為了讓遊馬及快斗分別前往巴利安界與月球，Ⅲ、Ⅳ、Ⅴ捨命和七皇決鬥
    • 貝庫塔作反，吸收了七皇中四人的靈魂
    • 快斗捨命和米賽爾在月球決鬥，成功喚出《No.100 源數之龍》。
    • 千尊吸收了貝庫塔能力而變成完全體，但最後被遊馬和凌牙聯手打敗。
    • 凌牙吸收千尊所有No.，為了巴利安界而和遊馬決鬥，遊馬在決鬥中了解自己失去的東西。
    • 為了讓遊馬找回失去的東西，阿斯特拉爾和遊馬作最後決鬥。
    • 小鳥向遊馬告白！

OCG相關編輯

規則變動編輯

  • 開始採用大師規則2
    • 新增超量怪獸，須放置於額外牌組。這也是首次新增沒有等級的怪獸。
      • 但嚴格來說在超量怪獸登場前，陷阱卡《マジカルシルクハット》就具有將魔法或陷阱卡當作無等級怪獸使用的效果。
      • 與前作的同步怪獸一樣都是與等級有關的召喚法，但有以下相異點。
        1. 同步怪獸必須要有協調怪獸當作召喚素材，超量怪獸則不需要。
        2. 同步怪獸對召喚時使用的素材數較寬鬆，超量怪獸則通常是固定的。
        3. 同步怪獸有等級可直接當作其他同步怪獸召喚的的素材，超量怪獸沒等級所以要靠RUM或其他方式才能當超量怪獸的素材。
        4. 超量怪獸的素材須放在怪獸下方，同步怪獸則直接送進墓地。
        5. 超量怪獸使用一部分效果時須把超量素材送進墓地，一旦沒有素材則該效果就不能用。同步怪獸則沒有類似限制。
        6. 代幣可以當作同步素材，但不能當作超量素材。
      • 另外一個協調怪獸可搭配各種素材召喚不同等級的同步怪獸，因此一副牌組採用等級相異的同步怪獸較簡單，但是不同階級的超量怪獸要求的素材等級基本上不一樣，故一副牌組通常不放多種階級不同的超量怪獸。
  • 另外一提，除了「為在場上準備兩張以上等級相同的怪獸，之後以該組怪獸為素材從額外牌組中召喚階級相同的超量怪獸」這種超量召喚外，劇中還有下面兩種超量召喚方式
    1. 以一隻超量怪獸為素材，在其上方疊上另一張超量怪獸
      • 例：CNo.39 希望皇霍普雷
    2. 利用「RUM」魔法卡的效果將場上一隻超量怪獸當作素材，從額外牌組中召喚階級較高的超量怪獸。
      • 例：CNo. 101 沉默的寂靜暗黑騎士
  • 這兩種召喚其實算是卡片效果[4]，不算規則
