作品介紹編輯

  • 超級戰隊系列第33作，全49幕（話）
  • 首播時間：2009年2月15日（日本）；2011年10月9日（台灣東森幼幼台）；2012年3月4日（香港TVB）
  • 劇本主編為小林靖子
  • 旁白：宮田浩德（日本）、曹冀魯（台灣）
  • 繼於2004年播出的特搜戰隊奪得日本星雲獎後，這套亦於2010年奪得日本御宅大獎中的第1名

故事概要編輯

來自冥河的外道眾出現人間，對人類產生危害。
身為志葉家第18代當家的影武者志葉丈瑠將志葉家的家臣們：池波流之介、白石茉子、谷千明、花織ことは集合在一起，為保護人類以及阻止外道眾的破壞，他們運用自身的一種流傳的力量「文字力」，組成侍戰隊對抗外道眾。
第17幕之後，丈瑠的兒時好友「梅盛源太」也加入他們的行列。

角色介紹編輯

真劍者編輯

  • 志葉丈瑠/偽第18代→19代真劍紅（松坂桃李 飾/台灣配音：吳文民）
    • 為志葉家第18代當家
      • 由於是家傳身分的領主，身分亦為侍戰隊的領袖人物
      • 表面冷酷嚴格要求下屬，內心熱心助人處事穩重
    • 第一人稱為「俺」，使用「火」的文字力
    • 雙親是志葉家家臣但已遭外道眾殺害，丈瑠因有優秀文字力之天賦，而被決議在真正的當家練成「封印的文字」前以影武者的身份假裝成當家，並由管家日下部彥馬一手養大
    • 第44幕隨著真正的18代當家「志葉薰」歸位，他的影武者身份也正式暴露；一直為欺騙家臣而感到愧疚的自己在失去一切的情況下接受十臟的單挑，一度險被他拖入外道但被家臣們救出
    • 第48幕被薰收為養子，成為名符其實的志葉家19代當家
  • 池波流之介 / 真劍藍（相葉弘樹（現藝名︰「相葉裕樹」） 飾/台灣配音：胡鎮璽）
    • 第一人稱為「あっし」，使用「水」的文字力
    • 加入前是家傳歌舞伎演員，也擅長女形[1]
    • 片中算是最尊敬丈瑠的人
    • 表面上裝出很堅強的樣子，其實情緒非常容易波動
      • 個性過於認真，但有時像個小孩子一樣
      • 搞笑角色（無誤）
    • 之後報讀暗夜渡鴉學園成為禁果之花宿舍的舍長
  • 白石茉子 / 真劍桃（高梨臨 飾/台灣配音：詹雅菁）
    • 第一人稱為「私」，使用「天」的文字力
    • 擁有除丈瑠外的極高書法本領[2]
    • 加入侍戰隊前是幼稚園老師
    • 平時較為嚴肅，但在其他人遇上煩惱時會展現溫柔一面
    • 做菜十分差，甚至還會把真劍丸拿來當菜刀用…
      • 殺人料理資格者之一，作出來的菜（即使只是負責當中的伴料）除ことは以外的人吃了必定會噴掉，黑子們對此也苦不堪言
      • 大家一直為了顧全她的面子而不敢公表，直到後期
    • 討厭的話是「一生獨身」←但本人還是結婚了[3]
  • 谷千明 / 真劍綠（鈴木勝吾 飾/台灣配音：宋克軍）
    • 第一人稱為「俺」，使用「木」的文字力
    • 加入侍戰隊前是現役高中生
    • 個性開朗而且隨性，但這種性格使他認為現代不應再有君臣關係而和丈瑠經常起衝突
      • 在第3幕自知實力不如人後，終於承認丈瑠為主公
      • 但他似乎還不是怎用主公稱呼丈瑠，亦下決心要在實力上超越丈瑠
    • 後與源太在《豪快者 護星者 超級戰隊199英雄大決戰》登場，個人則在「超級英雄戰記」登場
  • 花織ことは[4] / 真劍黃（森田涼花[5] 飾/台灣配音：龍顯蕙）
    • 第一人稱為「うち」[6]，使用「土」的文字力
    • 關西出身[7]，加入隊前一直住在鄉下務農，代替體弱多病的姊姊接下真劍黃的任命
    • 天真無邪的少女，本性單純善良，同時也是六人之中最年少的
    • 後在「超級英雄大戰Z」登場
  • 梅盛源太 / 真劍金（相馬圭祐 飾/台灣配音：曹冀魯）
    • 第一人稱為「俺」，使用「光」的文字力
    • 第17幕起登場的追加戰士，本職為擺攤的壽司師傅
      • 但功夫十分一般，本篇中會稱讚他作的壽司的只有ことは和十臟兩人
      • 第一次作出來的咖喱卻是異常的美味，好味得會有人來找他洽談開設全國性連鎖店的程度
    • 使用的文字力是由自己鑽研出來的「電子文字力」，被彥馬認為他擁有開發科普的潛質，但因為其出身不正在薰歸位後被其家臣丹波處處針對更一度離隊
    • 個性有時詼諧逗趣，有時太過熱情讓其他成員也稍微感受得到
    • 在隊伍中居為搞笑角色
    • 顏藝
    • 本命是隔壁的Cure Passion
    • 和丈瑠為兒時玩伴，烏賊折神正是丈瑠送他的，不過卻在第20幕被鄰棚的海東大樹盜走，引發出一連串的跨界事件
    • 之後與千明在《豪快者 護星者 超級戰隊199英雄大決戰》登場
  • 志葉薰 / （真）第18代真劍紅（夏居瑠奈[8] 飾/台灣配音：馮友薇）
    • 第一人稱為「私」，使用「火」的文字力的謎之女性
    • 第44幕起登場，其真實身分是志葉家真正的第18代當家
    • 後在封印血祭慟哭失敗後，將丈瑠立為養子，使他能夠以真正的志葉當家的身份去繼續戰鬥
      • 管家丹波歲三懷疑她為什麼會收年紀比她大的丈瑠當養子
    • 之後也有《海賊戦隊ゴーカイジャー》第11-12話、《特命VS海賊》劇場版登場
    • 之後報讀昂星高中

敵組織「外道眾」編輯

    • 侍戰隊的敵人，長居於三途川[9]的魔界之水上
    • 因似乎長居於三途川內，所以只有只有三途川的河水維持生命
      • 在人間久留會缺水，必須返回三途川補充水份
    • 外道眾能夠利用人界裡存在的裂縫進出，相反地人類則不能進入屬於陰界的三途川
    • 怪人設計藍本全是日本傳說中的妖怪
    • 設定上妖怪有兩條命，在等身形態（一之命）被擊破後會用上二之命來再生和巨大化
    • 在片末中隨著慟哭的死，三途川的水隨即逆流把整個外道眾從人界上沖回去
      • 後來在《天裝vs侍》因為血祭のブレドラン的力量將六門船從三途川的河底拖上來，與シタリ一同率領殘存勢力再掀旗幟，最終被完全消滅

首領編輯

  • 血祭慟哭（CV：西凜太朗/台灣配音：宋克軍）
    • 外道眾的首領，擁有絕對的威壓力，能放出麾下妖怪絕對無法抵抗的拘束咒
    • 具飛行能力，使用能放出斬擊波/火焰的「昇龍拔山刀」和「降龍蓋世刀」為武器
    • 過去在與真劍者的戰鬥中被17代志葉家當家以未完成的「封印的文字」封印導致身體四分五裂，歷經長時間後雖然回復過來但後遺症讓他只要一離開三途川就會立即缺水，本篇中長時間都只能留在六門船上
      • 夏季時他的力量會隨著三途川的水位一同上漲，但這種程度的漲潮也解決不了他的缺水問題---
    • 經常喝酒，而且似乎經常憤怒的樣子
    • 對自己麾下的妖怪似乎沒啥好感，即使是對自己喜歡（其琴聲）的薄皮太夫也曾惡言相向，還放火燒壞了她的三味線
      • 但當太夫陷入危機時卻為了她不惜戰到缺水瀕死的程度；當太夫為強化自己而死時亦做出非常感性的發言

幹部編輯

  • 薄皮太夫（CV：朴璐美/台灣配音：詹雅菁）
    • 在船上喜愛拿類似琵琶的三味線來演奏的外道眾
    • 武器為內藏的小刀，嘴巴能吐出光彈
    • 本為名為「薄雪」的花魁[10]，因為愛人「新佐」另娶他人，憤而燒殺了愛人與新妻全家並墮入外道；手中的三味線即是由新佐的軀體所化成，內裡積存了大量的嘆息之力
    • 曾經是鍊金術師
    • 表面上是很冷漠、對其他外道眾都充滿輕蔑的態度，但被慟哭所救時明顯很高興甚至最後不惜犧牲自己去強化慟哭
  • 骨のシタリ（CV：チョー/台灣配音：胡鎮璽）
    • 充滿智慧，為外道眾的軍師
    • 負責為外道眾提供讓三途川的水位升高的作戰計劃
      • 故事後期為了能消滅浮上檯面的志葉家18代當家，更自願獻出自己的一條生命強化麾下的妖怪
    • 武器為手上持有能放出衝擊波/閃電的手杖
    • 本篇最後跟隨著六門船沉入河底，沒頂前遺下了「管它是河底抑或是泥巴裡都能活下去，這就是我的外道！」的話語
    • 後來於《天裝vs侍》裡復歸，但由始至終都對空降的血祭のブレドラン不抱信任；後來當知道ブレドラン要利用三途川的河水直接攻擊護星界後決定謀反，卻被穿越時空的謎之海賊們亂入打敗了…
  • 腑破十臟（CV：唐橋充/台灣配音：曹冀魯）
    • 於故事初期出現的謎之外道眾
    • 平時以墮入外道前的人類身份活動，手上持有名為「裏正」的逆刃刀
      • 一般的武士刀為單邊開鋒，裏正則是兩邊開鋒，背面如鋸齒般的刃部更是它「真正的刀刃」
    • 視丈瑠為對手，但當丈瑠有難時都會出手協助
    • 曾經是劍客，但為追求與強者對戰的樂趣而墮入外道；裏正是惡麻呂暪著他殺害其家人後製成的魔刀，本應是為了讓生死世界逆轉而設的楔子，但十臟卻是一直在知道裏正的真相下仍用它殺人，最終成為了完全的外道讓惡麻呂的咒術失敗
    • 在丈瑠失去一切的時候找到他展開不分晝夜的死鬥，之後在兩敗俱傷的狀況下依然無法壓抑心中與強者戰鬥的慾望，更欲對前來救丈瑠的家臣們出手但被裏正刺穿左腳阻止，最終身體在從地面噴出的火柱中伴隨著咆哮消失
  • 筋殼惡麻呂（CV：堀川りょう/台灣配音：曹冀魯）
    • 於故事中期出現的謎之外道眾
    • 能夠飛行，使用能伸縮和反彈攻擊的手爪、殺人蹴鞠和能發射光束的「削身斷頭笏」為武器，手部能發射光彈/閃電
    • 也能夠以手爪製作出剪紙召喚出巨型怪人「切神」
    • 對一睹「地獄」有著病態般的追求，為此他在日本上施術並給予十臟的裏正意圖發動使生死世界逆轉的「裡見返之術」
      • 然而他唯一的失算就是十臟早就完全墮入外道，在第43幕遭背叛並殺掉一條命，之後的二之命再被サムライハオー用恐龍丸一刀兩斷；然而他在死前的痛苦當中看見了地獄的景象，已經完了心願

    • 本身亦能裝上不同的秘傳碟來發揮其能力

文字力編輯

  • 300年前，在一角笛山上發現，能將文字轉化為實物的神秘力量
    • 根據使用者所書寫能力代表的漢字的準確程度，準確度越高能力越力
    • 嚴格上所有折神也是以這種力量造成
  • 一般發動都是用毛筆手動書寫，不過也能用手機之類的電子設備以打字的方式發動，也就是所謂的「電子文字力」
