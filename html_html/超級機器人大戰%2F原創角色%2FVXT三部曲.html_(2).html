超級機器人大戰V編輯

地球艦隊天驅編輯

  • 本作玩家部隊，以大和號為中心，結合三世界（新正曆世界、西曆世界、宇宙世紀世界）的地球聯邦軍+α組成的艦隊
    • 第40話才正式組成+命名
  • 叢雲總司（むらくも そうじ）【CV：桐本拓哉】
    • 原創男主角，月面特殊戰略研究所防衛隊的隊員
      • 未選為主角時會在途中被葛迪姆拉攏並成為敵機瑪達瓦·迪戈的駕駛，後因主角潛入葛迪姆的戰艦而被救出
    • 原屬月面航空隊第二十五部隊，是該部隊在「冥號作戰」中唯一的倖存者
      • 受到過去部隊全滅的影響而養成及時行樂的享樂主義性格，不過在認真時會相當敏銳
    • 雖然看起來顯老，實際上才23歲
    • 姓氏捏他了驅逐艦「叢雲」
  • 如月千歲（きさらぎ ちとせ）【CV：神田朱未】
    • 原創女主角，第三特殊戰略研究所防衛隊的隊員
      • 未選為主角時會在途中被葛迪姆拉攏並成為敵機瑪達瓦·迪戈的駕駛，後因主角潛入葛迪姆的戰艦而被救出
    • 性格明朗快活
    • 姓氏捏他了驅逐艦「如月」
  • 奈茵（ナイン）【CV：山本希望】
    • 主角乘機「梵格雷」搭載的人工智慧「99系統」，利用駭客開啟大和號的艙門使主角搭乘上去
    • 在主角從因為次元震從新正曆世界跳躍到西曆世界時以少女的型態現身，之後被命名為奈茵
    • 真實身份是為了學習人類行為而由妮凡琳娜系統創造的的隸僕機No.2044
    • 原本在任務完成後就被葛迪姆回收並消除記憶，後在主角潛入葛迪姆艦隊的血淚（物理）勸說下回復記憶並返回隊伍
  • 威爾塔弗·提克斯特（ヴェルターブ・テックスト）【CV：金本涼輔】
    • 尼古拉·威廉研究所所屬兇鳥的測試駕駛
    • 於本篇40話前的外傳中加入，有初回特典的話可以提早加入
  • 夏綠蒂·赫斯汀（シャルロッテ・ヘイスティング）【CV：照井春佳】
    • 尼古拉·威廉研究所所屬古倫加斯特的測試駕駛
    • 於本篇40話前的外傳中加入，有初回特典的話可以提早加入

超文明葛迪姆編輯

  • 本作原創反派，三千年前存在於大麥哲倫銀河的超古代文明
  • 谷瑞（グーリー）【CV：田中一成】
    • 葛迪姆二等武官，性格好戰
    • 真實身份是仿生機器人S3429Y，因為在與主角交戰途中覺醒自我而被清除記憶並被大量造出作為量產士兵
    • 其中一名士兵在主角突入葛迪姆艦隊拯救同伴時恢復自我，在協助主角搭上梵格雷後逕行搭乘瑪達瓦·迪戈與主角決鬥後慷慨赴死
  • 潔美（ジェイミー）【CV：庄司宇芽香】
    • 葛迪姆一等武官，以追求美感為信條
    • 真實身份是仿生機器人C0873K
  • 亞爾弗茲·羅姆·哈赫勒斯（アールフォルツ・ローム・ハルハラス）【CV：中田讓治】
    • 葛迪姆第八艦隊司令
    • 真實身份是妮凡琳娜系統以葛迪姆文明最後倖存者為形象造出的仿生機器人A0012M
    • 本尊在藉由仿生人機器人A0012M研究的時空跳躍技術從三千年前跳躍到現代並將仿生機器人摧毀，最後反而被妮凡琳娜系統視為危害而被其抹殺
  • 妮凡琳娜系統（システム・ネバンリンナ）【CV：鶴弘美】
    • 葛迪姆的文明重建系統，本作最後BOSS
    • 為了學習人類情感以使葛迪姆文明再生而派出隸僕機No.2044（奈茵）潛入大和號
      • 不過最後仍未能了解「愛」的情感，這一點也成為她的敗因
      • 雖然就實際演出而言根本就是被我方好幾對情侶閃到瞎
    • 在機戰T的後日談會與魔獸恩殆合體成為鐵魔獸妮凡恩殆，因為理解了憎恨的情感而想消滅我方，另外亦制造大量自己的分身
      • 其中一個個體和我方交戰後理解了「愛」的情感而加入我方，亦是最後生還的個體
    • 前世是愛的化身，可能因為愛人不在了才變成這個樣子[1]
      • 專屬卡片1：Nebanlinna Das Abscheuliche System[2]
      • 專屬卡片2：Nebanlinna Das Extremer Hasserfüllte Biest[3]

其他編輯

  • 神宮寺辰之進（しんぐうじ たつのしん）
    • 居住在西曆世界的前旋風寺重工業員工，發現從新正曆世界轉移過來的主角並給予保護
    • 定位算是為了導入《勇者特急》的世界觀而登場的路人
    • 千歲本子的常客

超級機器人大戰X編輯

X-Cross編輯

  • 本作玩家部隊，以救世主小渡為中心，為了將異界人們送回其原本的世界與弭平艾爾·瓦司世界的紛爭而組成的部隊
  • 伊歐利·埃歐萊特（イオリ・アイオライト）【CV：新井良平】
    • 原創男主角，魔從教團的「菫青石術士」
    • 本名葵伊織（あおい いおり），被恩殆從「沒有超級機器人的世界」（即地球）召喚到艾爾·瓦司後被教團修改記憶
    • 未選為主角時會作為教團一方的追兵與X-Cross多次對峙，後在回復記憶後加入X-Cross成為捷爾嘉德的副駕駛
  • 安瑪莉·艾奎瑪琳（アマリ・アクアマリン）【CV：佐藤聰美】
    • 原創女主角，魔從教團的「藍柱石術士」
    • 本名天野亞真里（あまの あまり），被恩殆從「沒有超級機器人的世界」召喚到艾爾·瓦司後被教團修改記憶
    • 未選為主角時會作為教團一方的追兵與X-Cross多次對峙，後在回復記憶後加入X-Cross成為捷爾嘉德的副駕駛
  • 霍普斯（ホープス）【CV：野田圭一】
    • 協助主角的鸚鵡型魔法生物
      • 非常照顧女主，不過與男主間的關係極差
    • 真正身份與恩殆同樣是高次元生物
    • 於困難路線會為了激起X-Cross的正面情感而在擊敗恩殆之際將其身體佔據，作為最後的敵人阻擋在大家面前

魔從教團編輯

  • 本作原創反派，信奉智慧之神恩殆，活躍於艾爾·瓦司各地戰場的「法律與秩序的守護者」
  • 賽立克·奧普西蒂安（セルリック・オブシディアン）【CV：真殿光昭】
    • 魔從教團的法師
    • 終盤其野心被恩殆利用而遭後者精神操縱而陷入瘋狂
      • 一般路線會在與X-Cross交戰後戰死
      • 困難路線則會克服黑暗面而存活，為了贖罪而成為新任教團導師，決心使教團重組為真正的「法律與秩序的守護者」
  • 導師齊爾汀（導師キールディン）
    • 魔從教團的導師
    • 終盤被狂化的賽立克殺死
  • 魔獸恩殆（魔獣エンデ）【CV：田中秀幸】
    • 魔從教團信仰的神，本作最後BOSS，惡墮超級進化土台龜
    • 以人類的負面情感為糧食的他，為了與敵對的反螺旋族安基斯拜拉爾抗衡而創造出艾爾·瓦司世界並持續召喚其他世界的居民
    • 為了使步入衰老的自己轉生而打算利用魔從教團、捷爾嘉德與霍普斯建構自身肉體
    • 在機戰T的後日談會與妮凡琳娜合體成為鐵魔獸妮凡恩殆

超級機器人大戰T編輯

VTX特務部隊T3編輯

  • 本作玩家部隊，以VTX特務三課為中心構成的部隊
    • 由艾姆命名，源自「Tread on the Tiger's Tail」
    • The Revengers
  • 才藏·時任（サイゾウ・トキトウ）【CV：草尾毅】
    • 原創男主角，別名「野獸領帶」的VTX特務三課/一課主任
    • 作為主角時為特務三課所屬「暴君旋風」的駕駛，未選為主角時為特務一課所屬「亡靈」的駕駛
    • 對於上班族身份抱持驕傲甚至可以賭上性命的工作狂
  • 沙霧·櫻井（サギリ・サクライ）【CV：庄司宇芽香】
    • 原創女主角，別名「墮天使」的VTX特務三課/一課主任
    • 作為主角時為特務三課所屬「暴君旋風」的駕駛，未選為主角時為特務一課所屬「亡靈」的駕駛
    • 外表甜美但個性相當隨性的殘念美人
  • 拉美·天崎（ラミィ・アマサキ）【CV：加隈亞衣】
    • VTX特務三課新人，天崎課長的女兒，「暴君旋風」的副駕駛
    • 真實身份是來自已滅絕達布尼斯行星的外星人，母星被滅絕後將生父開發的夸克驅動裝置帶至地球
  • 梅莉露·史帕那（メリル・スパンナ）【CV：相澤舞】
    • VTX特務三課成員，負責火器調整
    • 表面冷靜其實個性相當急躁，因而被稱為「鬼火梅莉露」
  • 愛米斯·亞尼斯特（エイミス・アーネスト）【CV：大空直美】
    • VTX特務三課成員，負責行政工作
    • 因為心直口快而有「直接小姐」的別名，本人對此相當困擾
  • 博介·天崎（ヒロスケ・アマサキ）【CV：天神英貴】
    • 特務三課課長，拉美的父親
    • TND專案的負責人，操控綜合戰術支援機「凱瑞克斯」支援暴君旋風
    • 暴君旋風的設計師[4]

UND編輯

  • 本作原創反派，由異星人組成橫跨宇宙各地的傭兵組織
  • 露迪·彼西沙爾特（ルーディー・ピーシーザルト）【CV：永島由子】
    • UND戰務官，負責獵人頭等前線工作
    • 初期勁敵擔當，後期淪為苦勞社畜擔當
  • 瓦西魯蒙·拉茲埃爾森（ワシールモン・ラズエルソン）【CV：置鯰龍太郎】
    • UND執行官
  • 拉古納亞爾·汀哈里修（ラグナヤル・ディンハーリッシュ）【CV：稻田徹】
    • UND艦隊總司令
  • 西格爾·賽格（シゲイル・セイコ）【CV：ボルケーノ太田】
    • 因收賄而被免職的前VTX職員，被UND挖角後成為機動兵器的駕駛員
  • 羅尼·馬布克（ロニー・マーブック）【CV：深川和征】
    • 大學落榜的青年，被UND挖角後成為機動兵器的駕駛員
  • 波爾·達爾朱（ポール・ダルジュ）【CV：関根有咲】
    • 喜歡玩遊戲的小學生，被UND挖角後成為機動兵器的駕駛員

其他編輯

  • 達瑪·哥德文（ダイマ・ゴードウィン）【CV：江原正士】
    • VTX企業社長，過去曾是隸屬地球聯邦的戰爭英雄
    • 在後期為了阻止UND的自殺攻擊而駕駛機兵對其艦隊發動特攻而消散
    • 然而在T3擊退UND後確認生還並駕駛決戰兵器「達蓋安」現身，作為本作最後BOSS宣布實行「銀河統一計畫」
  • 艾姆·普雷茲班德（エイム・プレズバンド）【CV：增谷康紀】
    • 地球聯邦議會議員，改革派「破曉會」代表
    • 實為UND的協助者，終盤為了實行人類士兵計畫而與拉古納亞爾聯手進攻地球
    • 一般路線會在戰敗後被操控達蓋安的達瑪討伐，困難路線則會駕駛另一架達蓋安保護達瑪
