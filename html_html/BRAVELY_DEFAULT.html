解說[編輯]

  • SQUARE ENIX於任天堂3DS開發的角色扮演遊戲
    • 製作總監為淺野智也，劇本為林直孝，配樂由Revo擔任
    • 主要人設為吉田明彥，部分人設則外包給其他插畫 / 漫畫家擔任
  • 以傳統的《最終幻想系列》的轉職為基礎，結合其獨特的「奮勇 / 堅守」而成型的復古風RPG
    • 奮勇：消耗BP進行一回合多次行動，沒有BP時也可以預支，不過當BP為負數時當回合會無法進行行動，直到BP隨回合回復到0
    • 堅守：即一般的防禦技能，不過本作防禦同時也能蓄積BP供奮勇使用
  • 劇情與世界觀也比照傳統FF，描寫在盧森達克（ルクセンダルク）世界的主角四人前往各地解放水晶的冒險
    • 在進入第五章後遊戲會進入無限LOOP，直到最後才會告知其實是平行世界
  • 職業系統為獨特的「晨星（アスタリスク）」系統，在冒險途中將持有晨星的敵人擊敗後可以獲得對應職業的晨星
  • 初版推出一年後發售其完整版《BRAVELY DEFAULT For the Sequel》，更改了部分演出外還預告了下部作品《BRAVELY SECOND》
    • 最主要修改的地方是第七章與第八章的平行世界，原版只是讓BOSS們照原本的陣容再打一遍，完整版則是將BOSS陣容重新洗牌並追加劇情，到了第八章甚至出現全員齊聚公國司令部打總體戰的展開

主要角色[編輯]

  • 提茲‧歐里亞（ティズ・オーリア）【CV：齋賀光希】
    • 男主角，居住於諾恩德村的青年牧羊人，在大穴災難中唯一存活的奇蹟之人
      • 能力是平均型
    • 實際上已經死亡，但因為來自神界的玩家靈魂附體而得以生存，從此開場主角死而復活成為BRAVELY系列名物

      • 能力以知性與精神見長
    • 接受艾雅莉的請託，為了解放水晶而展開旅行
    • 沒有救的大路癡
  • 林格阿貝爾（リングアベル）【CV：竹內良太】
    • 喪失記憶的青年，擁有一本記錄自己未來行動的神秘日記「D之手帳」，在遊戲中還有圖鑑功能
      • 能力以器用與素早見長
    • 上個世界的亞納傑爾，因為在上個世界的伊蒂亞在她面前死亡而進入世界的通路來到這個世界
    • D之手帳其實是他上個世界所寫的日記

  • 伊蒂亞‧李（イデア・リー）【CV：相澤舞】
    • 艾塔魯尼亞公國元帥聖騎士布雷弗‧李之女
      • 能力以力量與體力見長
    • 原空挺騎士團所屬，因為不滿直屬長官歐米諾斯的暴行而叛逃
      • 因為曾是公國所屬的緣故，與公國交手時多半會被對方斥為叛徒或是指責她將會令其父蒙羞等等
    • 因為是騎士子弟的緣故擁有很強烈的正義感，習慣以黑白論來斷定善惡
    • 名字捏他：理想地（Ideally）
  • 艾雅莉（エアリー）【CV：津村真琴】
    • 擁有蝴蝶般翅膀的小妖精，委託亞妮艾絲前往各地解放水晶
      • 跟我簽下契約，一起去解放水晶吧！
    • 本作的真黑幕，欺騙並利用光之戰士們來為自己打通平行世界收集復活銜尾蛇的力量，在這之後成為BRAVELY系列名物之一
    • 當開通下個世界的通道後，會從主角一行人背後現出原形的巨大毛蟲模樣使用重殺把他們殺光
    • 本性可說是極致的惡女，在現出原形後亞妮艾絲原本還一度不想相信…而在劇中也利用過這點嘲諷過亞妮艾絲
    • 「…亞妮艾絲…亞妮艾絲…救救我…我被這個怪物給侵占了身體…！」
    • 「じゃん☆じゃん☆じゃ－ん☆現在才曝光的衝擊事實☆被可愛的妖精姿態揪住自己的少女心嗎？哈哈哈哈哈－」
