凱留

凱留（キャル）/Karyl/接頭霸王
本條目歡迎熱心的騎士君幫忙補完
以下含有部分的劇情，沒玩過遊戲又不想被グリムバースト！/ニューイヤーバースト的話請不要繼續看下去。
沒玩過遊戲還看下去，真的會宰了你喔！

目錄

解說編輯

  • 遊戲《超異域公主連結☆Re:Dive》中的主要角色之一，為主角團公會美食殿堂的一員
  • 陸版翻譯為凱露
  • 由於角色梗太多故足以開單獨條目的典例，也是公連第一個有角色條目的女人
    • 雖然現在用梗時玩家的感情和最初梗產生時變化已經很大了

角色介紹編輯

  • 給可可蘿的外號是「可蘿仔（コロ助[1]）」，也只有對可可蘿有稱呼外號大概是覺得體型太過嬌小，最初可可蘿對此還有一點抵觸，之後也就讓她這麼叫了，可見美食殿堂成員間的關係的確是逐漸親近
  • 美食殿堂的常識人兼吐槽擔當，對於佩可深不見底且異常雜食的食慾和可可蘿的天然總是用力的吐槽。
    • 不喜歡用蟲子和魔物做成的料理，在知道自己常去的餐廳其實是提供魔物料理的專門店時的反應相當壯觀
    • 對於佩可為了尋找未知的美食而把自己拖去一起冒險一事感到相當不滿，但還是會跟著前去，到主線冒險後期進行到後期更是逐漸表現出強烈的探索欲與征服欲
      • 長期處在一群不是很正常的人之間的常識人兼苦勞人
  • 經過各種管道的劇情和描述，可以看出凱留是目前阿斯特朗中有提及的最強魔法使
    • 她同時擁有輔助魔法，防禦魔法，攻擊魔法，恢復魔法，工具魔法的技能，而且對每一種都非常精通，細分可以超過十種（並不都體現在遊戲戰鬥中，也有僅在四格和主線/活動/個人劇情中單獨出現）
      • 所以說她是這方面的天才毫不為過
  • 典型傲嬌
    • 雖然語調非常像不良少女，但其實非常好搞定
    • 喜歡貓的同時也擁有吸引貓的體質，監視主角的時候常常因為野貓的接近而情不自禁的學貓叫，結果就會暴露自己
    • 時常自言自語作出下一步計畫但同樣會被騎士君發現（RY
      • 「宰了你哦！」
  • 言辭毒辣
    • 既然是不良少女的形象，當然是要有所表現的
      • 只要是出現了和凱留一樣說話比較毒辣的角色，兩邊的口水戰就會異常激烈，比如敵對勢力的卡里莎，還有美食殿堂公會小屋裡的涅比亞
    • 而涅比亞和凱留互有稱號：涅比亞稱凱留為「變態奇怪喵咪」
  • 常常被虐
    • 起因是主線中霸瞳皇帝對任務失敗的凱留施以體罰等的懲罰，之後虐凱留就慢慢成為二創中的其中一個主題並得到人氣
    • 除主線內容外，凱留人設的家庭背景也被惡意設定得比較不幸，在現實家中地位很低，在遊戲世界直接成為流浪貓，霸瞳皇帝對她的態度也非常惡劣
    • 引伸出遊戲之外，甚至在官方廣播的小劇場、音樂會「ASL2019」的過場等處都要特別排定迫害凱留的部分
      • 用待遇換強度的女人
  • 實際是個濫好人
    • 雖然言語辛辣，凱留做事卻意外非常可靠，而且她還很難推辭別人的懇求。雖然會找藉口說是其他人要她做的（主要是主角，因為他一直都跟著凱留，而且也是個濫好人），但很多時候她也會主動提出要幫助他人，甚至到後文所述的活動中，能幾乎取代可可蘿去做美食殿堂的大部分內部工作
      • 這其中也有部分來源於主線中對貪吃佩可的寬容的感激和對自己所做錯事的慚愧

經歷編輯

主線/阿斯特朗部分編輯

  • 最初是不知為何倒在森林內的貓娘魔法使，被主角等人救起後順勢加入美食殿堂
    • 其實是「陛下」霸瞳皇帝的手下，接受了暗殺佩可的任務
      • 個人劇情中為了打工掙錢並練習操縱魔物的技能，在主角的介紹下前往一家地下的魔物鬥技賭場工作久生情而無法下手，在動畫版中後來這個任務被霸瞳皇帝改成了單純監視
    • 受陛下賜予了操縱魔物的能力，實質上就是「陛下」的公主騎士。陛下都已經是偽娘了，就別吐槽明明是女的卻是公主騎士
      • 雖然有強大的能力但常常失敗，故事開始時會倒在森林內的原因就是因為控制失敗造成的反饋
        • 霸瞳皇帝失敗後她隨即失去作為公主騎士的所有能力
  • 在第一部第八章被洗腦，代替因使用大範圍破壞魔法及被晶反擊而短期內無法露面的皇帝對王國騎士團等人下命令，對貪吃佩可使用了非常巨大的魔法攻擊並將其羈押，這種狀態的她直到第十三章才得以被解救
    • 第十三章為了給主角傳遞晶給予的力量被迫親了主角的額頭，然而心情久久不能平復
      • 自認為和陛下有親屬關係，無法下決心反抗而在第十五章中，因為霸瞳皇帝消耗過多魔力而顯露疲態之勢，轉而作出讓凱留救自己（實際上是為了取回之前一部分力量）的要求
      • 但凱留知道陛下是為了利用自己達成目的，雖然無法反抗殿下但依舊選擇要同歸於盡，好在最後得主角和佩可的幫助而未犧牲
  • 在第二部中因為王宮的假釋判決與自身的愧疚感，處於貪吃佩可左右給予輔佐，同時維持著美食殿堂的正常公會活動
    • 第一章中與卡里莎互相嘲諷，場面火藥味十足而且兩人間言辭之狠毒相當一致
    • 第四章中作為貪吃佩可的護衛一同前往月光學院成為臨時轉學生，但受潛意識中現實裡對學校的負面印象影響，她並不能很好地融入這種生活，甚至想逃跑。在美空的攻勢來襲後，她和雪菲一起控制了暴走的同學們，但在最後和美空的戰鬥中並不能變成公主型態，心生沮喪
    • 因為對自己不能變成公主型態、之前各種背叛事件的自責以及貪吃佩可和可可蘿突然提到，「另一個世界」，在第二部第五章中凱留一直鬱鬱寡歡。她做了一個夢，正是其泳裝版現實劇情的後續：現實中的她最終沒有機會參與畢業旅行，而且對自己周圍的一切充滿厭惡。一次她和雪菲偶然相遇並互相討論了各自的處境，在此之間還被帆稀插入予以警告。在雪菲和帆稀離去不久凱留就被空間跳躍魔法攻擊而被傳送到未知的地方，卻在彼處看見了霸瞳皇帝
      • 被對方反覆折磨痛擊後，凱留已經失去的所有希望，但遠方的拉比林斯達通過「物件變更」能力將主角傳送到凱留處。凱留在多次被主角拯救後，心中堅定了對他的愛意，終於告白，也同時向自己的命運宣戰。她成功取得了公主型態，揭穿對方是「萬能人偶」變化成的冒牌霸瞳皇帝並將其徹底擊潰，突破了阿澤爾德的陷阱。她隨後使用技能精準擊潰了大量「萬能人偶」，但因為已經負傷太重提前結束公主型態。事後她已經不再迷茫，而且想要終有一天對霸瞳皇帝親口說出自己的自由意志
  • 第六章中為了陪同貪吃佩可穩定王宮和蘭德索爾民心，凱留沒有前往自己很想去的大江戶町，留在王宮陪伴貪吃佩可並為她做文書工作
    • 後陪同佩可前往營救末路的克蕾琪塔，打退了追兵，並鼓勵克蕾琪塔從極度悲傷中振作起來
  • 第七章中凱留幫助貪吃佩可禁錮了大部分豪紳造反所徵集的兵力，從而在王宮騎士團中樹立了威望
    • 雖然自己也是大罪被佩可赦免但強烈希望豪紳被嚴懲，而貪吃佩可自然因為公平性問題相當苦惱
  • 是前作+續作御三家中最後實裝公主形態的
    • 第二部第四章第8話中也試著想變成公主形態過程相當羞恥，但是都失敗，推測可能是自己沒有正視自身的想法而無法使用公主形態的力量
    • 在第二部第五章中最終再次被主角所救，正視了自己對主角的愛意後向其告白於是對不起優衣的人又多了一個，成功解放了力量，變身為公主形態命運脈衝鋼彈凱留
      • 比起以往公主形態給人鋼彈的既視感，凱留的更像是婚紗…凱留本人也有些害羞就是了

個人劇情/現實部分編輯

  • 現實生活中過得並不順遂，主要問題是出於她控制欲強烈的母親
    • 在母親要求下被迫到離家有些距離的補習班上課，而之所以選這種位置的補習班卻是母親出於「風水好」這樣奇怪的理由
    • 如果沒有按照母親的期望，就會被她以「是不是在那裡學壞了」為由大鬧，甚至還會鬧到的打工處，導致自己被開除了好幾次，並被迫與好不容易快交到的朋友分開
    • 雖然母親總是說「這都是為了妳完全由我支配的將來著想」，但凱留本人認為母親只是在乎自己的顏面而已，而不是關心女兒的未來本身
    • 非常希望能早點離開家獨立，所以現在正努力存錢，計畫在大學畢業後正式自立生活
      • 但也因為這樣，平時能用的金錢非常拮据，父母也幾乎不會給予必要開銷以外的零用錢，無法隨心所欲地買自己想要的東西
  • 在現實中崇拜七冠之一的千里真那，這也是她玩《阿斯特朗傳奇》的主因
    • 曾從母親口中知道自己與傳說中的天才可能是有血緣關係的遠房親戚時，感到了一絲振奮，認為有同樣血緣的自己也能有打破現況的可能性，因此積極聯絡並為其工作
    • 故瞞著父母偷偷買了MIMI登入了遊戲，並在輸入姓名時不小心把"キルヤ"打成了"キャル"，也就是凱留名字的由來，而之後也許就沒怎麽進入過遊戲。但可知的是在遊戲中主角也和凱留多次見面，因此能在現實中見面時說出其遊戲暱稱
      • 根據公主版個人劇情可知，凱留在現實中可能從來沒有親眼見過真那
  • 與其他角色在現實世界中與主角是熟人或者有過很長時間的互動不同，現實世界的希留耶與主角交集甚少
    • 初次見面（第4話）時是撿起了她丟失的錢包並交還給她，之後（第8話）在書店再度碰面，最後（第11話）是一起在快餐店用餐
    • 新年版中在得知她想要穿和服卻沒錢買的情況下，主角幫她找到了一家商店並穿上和服作為看板娘進行攝影
    • 泳裝版個人劇情中對應現實中因為憧憬畢業旅行而叫主角為自己挑選泳裝，雖然錢不太夠。而從愛梅斯的話中可以得知她沒有得到母親的同意，不能參加前往海邊的畢業旅行。之後希留耶的表現在上文所述的主線劇情中略有交代
      • 公主版因為非要渲染凱留與主角交情很少還是繼續了這段劇情。經過這件事後，希留耶情緒徹底失控，而且因為其母親的名聲在學校也開始失去朋友。一天主角尾隨著自言自語、自暴自棄的希留耶，最後在公園追到。在耐心聽完她的絕望之言後，主角表示自己永遠會守望她。之後兩人徒步從椿丘區走到了東京都的海邊，讓她的心情得到了紓解
    • 而縱觀上述經歷都未能改變凱留生活基調灰暗的事實，反而因為接觸太少而顯得異類，可謂是整個公連中讓玩家吃胃藥程度最高的

    • 但當對可可蘿離開、對其的記憶也逐漸消失後，凱留也會承擔起可可蘿的那部分工作[2]

遊戲中能力編輯

       

  • 2018年泳裝活動推出泳裝版，在令和元年12月31日和鈴莓一同實裝了新年版，成為第一組有實裝兩種限定版的角色[3]
    • 2020年12月的主線劇情揭曉凱留的公主形態，出人意料的是其服裝以白色而不是本色的紫黑為主軸3週年池子裡有什麼應該不難猜了
    • 次年的公主祭典果不其然公佈了凱留的公主形態實裝消息。有趣的是，作爲主線劇情最後一個擁有公主形態的凱留竟然比怜還要早實裝公主形態
      怜醬大失敗
    • 在4週年活動中緊隨佩可其後，追加凱留（Overload）形象，為全公連第三個擁有4種異色卡圖的角色，且至此美食殿堂的三人全都有6種卡圖形象
  • 遊戲中四個版本都是以魔法攻擊為主體的角色
    • 1星版是在初期玩家缺少魔攻角時的有效選擇，6星版全面提升數值後更追加恐怖效果可以有效抑制對方的UB
    • 泳裝3星版則是特化在單體攻擊方面，成為競技場能迅速秒殺對方前排的恐怖存在
    • 新年3星版由於自身UB「根據造成傷害提高自身魔攻上升的效果量」的效果造成以她及提供TP角色為主體的魔法OTK隊流行，被稱為「唯一神」，更引致2020年2月份的公會戰起追加了傷害上限也就是俗稱的大傷盾，但前兩階段依然只有被一回殺的份；裝上專屬武器之後能力（攻擊範圍及魔法攻擊傷害量）再度大幅提升，攻擊範圍變成以單體為中心的範圍傷害，不少玩家實測已經可以組成僅用一隊便可擊殺EX3 BOSS的效果
      現在應該要叫究極唯一神了
    • 公主版地獄死神鋼彈EW凱留則有至今（2021年3月）最高倍率UB，另外技能還可以減魔防及回復，不過由於角色面版中的HP吸收為0（至RANK 20)，意味不能從攻擊中獲得HP，使用時需要留意。

接頭霸王編輯

 
你又想把我的頭安在別的女人身上了對嗎
 
凱留接頭的例子
  • 源自於2019年年中時，中國大陸那邊有人將凱留的頭PS到別的角色上，基本上只要有手遊出新角或是新Skin都會中彈
  • 「你又想把我的頭安在別的女人身上了對嗎」←話是這樣講，但到後來對象已經不分男女了
  • 隨著時間的過去，漸漸變成「這就是新的身體嗎」「這個身體讚！」「我饞的是你的身體！（因為可以套用很多角色的身體）」並且成為迷因流行梗了
    • DIO只換了一次身體，而她換了數千次
  • 其實本體是食腦蟲
 
凱留的真面目
  • 逆學姐
  • 2020年7月，好微笑公開其黏土人商品化決定的消息，可以光明正大的接頭了
  • 到現在公連出新角色的卡面被凱留換頭[4]在巴哈等社群已經成了定番
    • 看起來更像是一種固定登場的改圖技術比拼大賽了，還會有大量玩家去評價誰的改圖違和感更小

二五仔/背骨貓編輯

  • 二五仔是粵語俚語；背骨是台語。兩者都是背叛者的意思
  • 最初加入美食殿堂時實際上還是間諜，在最初的劇情走向下很多人對凱留這個角色的主要感情還是憎恨，這種情緒到第一部第八章結尾她對貪吃佩可使用了巨大的魔法攻擊後達到頂峰，一時間關於凱留全部都是負面的論調。但在之後的發展下凱留的形象被逐漸洗白，這種仇恨情緒也逐漸下降直到消失
  • 由於各玩家對劇情和角色設定的認知處於不同時期，對凱留的態度還是存在很大的不同，一些人並不喜歡凱留相關的梗，而另一些人則熱衷此道
  • 然而官方也在偶大連動劇情中玩二五仔的梗
    • 未央：一定是這樣！如果不是被控制的話，凱留凱留才不會攻擊自己的夥伴們的！
    • 佩可：這個嘛……
    • 騎士君：呃……
    • 可可蘿：得視具體情況而定……

顏藝擔當編輯

  • 在動畫中表情非常豐富，由此產生了大量相關梗，真是太不妙了
     

凱留凱留☆～編輯

 
  • 最初出自偶大連動中未央對凱留的稱呼
  • 但劇情中凱留被盆栽小花控制，和另外兩個新人偶像的組合「Prosperity」組成了「Prosperity•Cats」，由此換上了偶像服裝登場，而她的出場台詞就是「凱留凱留～♪」
    • 被控制的凱留性格大變，由傲嬌轉為高傲風格，說話結尾也帶上了「喵」
      • 由此揭露了凱留的偶像夢想，實裝希望
    • 另外，這段劇情官方推特還特意發文要大家注意，官方最大手
