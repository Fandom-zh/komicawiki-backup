解說編輯

  • 龍之子公司40週年紀念作，花費了五年時間制作，共有6集OVA
  • CG動畫非常華麗，特別是第一集的戰鬥場面得到各方好評
    • 但有說龍之子亦因這一套動畫搞得將要破產
  • 曾獲得2006年第五回東京國際動畫（東京國際アニメアワード）OVA部門優秀作品獎
  • 由於角色和變身後樣子相似，亦有與特攝片－牙狼-GARO-相提並論的情況
  • 香港的無線電視曾於深夜時播放

故事大綱編輯

為何曾經是街之守護使“鴉”的他會向人類復仇呢？

忘記了夜晚的街，新宿。忘記鄰接的妖怪的存在，流浪生活的人們。

自古以來，互相糾繞的兩個世界的秩序根據“百合音”和“鴉”被守護。不過，這里平衡正在很大程度地發生混亂。

為了阻止殺人案繼續發生，新的“鴉”誕生了，青年乙羽被選作新的守護者，新仇舊恨在新宿街頭開始蔓延…

人物介紹編輯

「鴉」們和有關人士編輯

乙羽/乙羽/おとは

  • CV：和田聰宏
    • 新宿新的「鴉」
    • 本是黑道老大的兒子，自小已經感覺不到痛楚和恐懼，與禮治一起逃跑時身中兩槍再從高處掉下被送進醫院
    • 本名是洋介
    • 劍術相當厲害，可以在人類的狀態用鴉的劍斬殺御座
    • 第五話以人類之身與牛鬼戰鬥，被殺後在自身誕生百合音，再次成為鴉

百合音/ゆりね

  • CV：鈴木霞
    • 都市的意志代理人，每個城市都會有一個，在下面主要是說乙羽的
    • 令人聯想到貓的形象
    • 不同城市百合音的分別只在於衣服，頭髮的顏色，臉上的淚印和數碼寶貝主角必有的護目鏡
    • 良子曾經見過她，當時的百合音是一隻白貓的形態
    • 第3話被土蜘蛛虜走，第4話被迴向消滅，第5話從乙向身體中再生先代鴉
    • 在乙羽之前保護城市的鴉
    • 三年前發現迴向的陰謀與其對戰但失手被殺

鳳春院迴向/鳳春院廻向/ほうしゅんいん えこう

  • CV：櫻井孝宏
    • 原為保護新宿的「鴉」，背叛了城市的意志，意圖控制城市的人
    • 不同其他鴉，他使用了某種方法取回自己的真名
    • 守護城市超過三百年，但對忘記妖怪存在的人類絕望，所以決定背叛
    • 完全不懂BOSS巨大化的死亡Flag

炎/ほむら

  • CV：生天目仁美
    • 某個城市的鴉，為了調查迴向的意圖來到東京
    • 感情豐富，喜歡吃零食的形象與乙羽有很大的對比
      • 鴉冷酷的形象都被這傢伙破壞了
    • 相對乙羽的刀，她是用長槍的
      • 自古槍兵幸運E
    • 因為是ほむら所以能停止時間也是理所當然的

平民編輯

ヒナル

  • CV：涉谷飛鳥
    • 為了達成「在藝能界成功」的夢想而來到東京的少女。為了報導謎樣的連續殺人事件「傳聞的現場」（噂の現場）而來到新宿車站。

吳鳴海/呉鳴海/くれ なるみ

  • CV：鳥畑洋人
  • 隸屬於干涉事件對策課，相當現實主義的警員。上任不久就在新宿車站遇到奇怪的事件。

鷺坂實/鷺坂実/さぎさか みのる

  • CV：後藤哲夫
    • 新宿署干涉事件對策課的刑警，為了證明女兒「看見了妖怪」的說法而處處奔走。
    • 在第五話為保護良子被牛鬼吃掉

鷺坂良子/鷺坂よし子/さぎさか よしこ

  • CV：千葉紗子
    • 鷺坂實的女兒，數年前經歷了神祕的殺人事件後班上唯一的生還者（本人稱是妖怪所為），因為精神上的驚嚇，在大久保東醫院住院多年。

禮治

  • CV：日野聰
    • 洋介的小弟(黑道那種)，認為洋介不應在這個城市，一起逃跑時被抓住
    • 一星期後被乙羽救走，但逃跑時被老大槍殺

新宿署署長

  • ＣＶ：納谷六朗
    • 派遣吳到干涉事件對策課協助鷺坂
    • 實際是數年前吃掉良子全班同學的御座，在第五話展現真身且開始吃掉避難所的人們，後被新生鴉秒殺

妖怪/御座編輯

鵺/鵺/ぬえ

  • CV：藤原啟治
    • 想擊倒迴向，救回弟弟所以回到新宿的妖怪，御座化後能使用電擊彈
    • 鵺是同生共死的妖怪，要求乙羽用自己的生命消滅弟弟拯救城市
    • 亂帥一通的大叔

雨降小僧/雨降り小僧

  • CV：小櫻悅子
    • 跟著鵺的小妖怪，叫鵺做「大哥」
    • 為幫助鵺復原到了大久保醫院取血包，但無意中發現了乙羽的真身

用語編輯

鴉編輯

  • 既不是人也不是妖怪，「保護城市的東西」
  • 一般以一個假面騎士武士的形態出現，但亦能變成戰鬥機，汽車的形態
  • 成為鴉後會獲得一個新的名字，因此「乙羽」和「炎」等都不是他們的真名
  • 平時會治療妖怪的病痛，因此妖怪們都尊稱「鴉先生」
  • 從炎的說話所得，成為鴉要是理解人類悲傷的靈魂

御座編輯

  • 由迴向改造妖怪，令其得以人的形態現身，代價就是必須經常取得人類的血液補充能量
  • 由於是類人的姿態，所以亦能拍照
  • 鵺兄弟的變成御座後的樣子比較特別，原因是迴向的計劃中鵺二人的角色非常重要

妖怪編輯

  • 古代就已經存在的生物，與人類並存，互相幫助
  • 從某時候開始，人類忘記了妖怪的存在，令它們的生命力大大減少
    • 比較類似於小飛俠(Peter Pan)對妖精的設定
  • 沒有人類的話，妖怪亦會消失
