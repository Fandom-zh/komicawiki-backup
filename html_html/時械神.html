時械神

時械神（じかいしん）／Timelord

目錄

解說編輯

  • 「時械神」是動畫《遊☆戯☆王 5D's》的最終BOSS「Z-ONE」所使用的牌組系列。
  • 動畫中Z-ONE使用的牌組由2隻等級1的「時械」下級+10隻「時械神」+1隻最上級的「究極時械神」+代表虛無、無限的3張永續陷阱+4張以塔羅牌為原型的通常陷阱構築而成。
    • 實卡化以後則只有1隻「時械」下級，塔羅牌相關的陷阱則可能是不想跟【秘儀之力】撞元素所以沒有實卡化。
    • 儘管早在2011年的動畫內容就登場，但【時械神】的實卡化則是等到2017-2018年的CP17、CP18才全員收錄（歷經6年），除《時械神 梅泰翁》和《究極時械神 塞菲隆》外玩家多只能去《Tag Force》中體驗到這副牌組。
  • 本家主力怪獸均為等級10的高等級怪獸，擁有著以下共通效果：
    • 「牌組中的這張卡不能被特殊召喚」
    • 「若自己場上沒有怪獸，則可以不用釋放怪獸就召喚這張卡。」
    • 「這張卡不會被戰鬥以及卡牌效果破壞，因這張卡的戰鬥而對自己造成的戰鬥傷害為0。」
    • 「自己的準備階段中必定發動。令這張卡返回持有者牌組。」
    • 進行過戰鬥以後，時械神怪獸都會發動不同的效果，大部分都是以燒血類為主。
      • 因為上述特色，【時械神】對於一些僅有破壞類除去手段、抗性弱的牌組打擊能力極強，且本身不是很需要吃特召，隨隨便便就能通召出等級10以上的怪獸，配合一些削血Combo，成為環境中獨樹一格的牌組系列，也算是歷代BOSS所使用的牌組中強度偏上的一檔。
      • 在官方遊戲《Master Duel》中，因為每次活動都能毫無顧慮地攜帶【時械神】做BEAT向牌局，成為不少新手玩家的噩夢
  • 時械神的原型來自猶太教的卡巴拉生命之樹，因此有分別代表10個質點的10隻時械神，依照生命之樹1-10的順序分別出在CP17跟CP18補充包
    • 後來DT世界觀的【天球】亦是以卡巴拉生命之樹作為原型，有趣的是，作為最早使用這一藍本的【時械神】反而比它還要晚O化。
  • 另有同步怪獸《時械神祖 武加大》，使用者為漫畫版《遊☆戯☆王 ARC-V》中的女性決鬥者夏娃（EVE），但除了致敬以外與【時械神】本家沒什麼關聯。

牌組怪獸介紹編輯

  • 本家怪獸由等級1的《時械巫女》+10隻等級10的怪獸+1隻《究極時械神》構成。
    • 全員均為天使族，其中炎屬性3隻、地水風各2隻、光屬性1隻。
      • 同樣是除了《時械巫女》、《梅泰翁》、《塞菲隆》，同屬性的一組都是在CP17和CP18分開收錄，似乎是有意為之。
    • 由於主力怪獸的戰鬥後效果通常比較強力，因此怪獸多被設定為攻守皆為0。
  • 每隻時械神的造型設定為各種天使寄宿在無機物鎧甲上，胸前都會顯示出顏藝般的人臉，根據Z-ONE所說這些臉代表著他逝去的同伴的意志。

時械巫女（じかいみこ）編輯

 

  • 本家唯一的下級怪獸，自己空場時可以從手牌跳出來，把通常召喚的權限留給上級【時械神】用，通常是釋放自己檢索時械神後拍下來，沒有時械神時抽到這張卡就能補充戰力。
    • 雖然【時械神】不能從牌組特殊召喚出來，但這張卡在墓地時可以無視限制直接從牌組拉出來。
      • 由於整個回合都適用制約，因此用過①效果之後就不能用墓地效果特召牌組【時械神】，通常是用堆墓手段將這張卡送墓，或是留到下回合再用。
      • ②效果幾乎用不上，因為幾乎沒有人會正規用這張卡當祭品召喚時械神

卡片資訊編輯

  • 效果怪獸
    等級1／光屬性／天使族／攻擊力0／守備力0
  • 效果：
    1. 自己場上沒有怪獸的場合，可以特殊召喚手牌中的這張卡。
    2. 在升級召喚「時械神」怪獸的時候，可以將這張卡視為2隻怪獸進行釋放。
    3. 釋放這張卡才能發動。將牌組中1隻攻擊力0的「時械神」怪獸加入手牌。
    4. 除外墓地裡的這張卡才能發動。無視召喚條件，特殊召喚牌組中1隻攻擊力0的「時械神」怪獸。在發動這個效果的回合，自己只有用這個效果才能特殊召喚怪獸。

時械神 梅泰翁（時械神（じかいしん）メタイオン）編輯

 

  • 最早實卡化的時械神之一，除共通效果外擁有戰鬥結束後將全場怪獸返回手牌並給於生命值數值傷害的效果。
    • 不管是燒血還是BEAT功用都比較保守，是時械神唯一一隻守備狀態會吃戰傷的，另外這張卡的⑤效果是唯一沒有發動動作直接回牌組的。
      • 以上因素加上這張卡是最早O化的，可以看出當時K社是出於環境考慮才不想將剩餘的成員一同O化，這張卡的效果也變得跟其他成員截然不同。
  • 原型是指猶太教中的「梅丹佐」，又稱「神之代理者」、「天之書記」、「小耶和華」。
    • 梅丹佐（Metatron）與人類頭頂的大宇宙之間的接點，有創造的泉源、純粹存在、生命力的泉源之意。
    • 數字是1，顏色是白色，寶石以金剛石為象徴，行星以海王星為象徴，代表王的側面。
    • 對應原質為王冠（Crown）：超越、神的本性；亦代表著淨火天。

卡片資訊編輯

  • 效果怪獸
    等級10／炎屬性／天使族／攻擊力0／守備力0
  • 效果：牌組中的這張卡不能被特殊召喚。
    1. 自己場上沒有怪獸的場合，可以不用釋放怪獸就召喚這張卡。

    3. 以表側攻擊表示存在場上的這張卡，因戰鬥而對自己造成的戰鬥傷害為0。
    4. 這張卡進行了戰鬥的戰鬥階段結束時必定發動。令這張卡以外的場上所有怪獸返回持有者手牌，並根據返回手牌的怪獸數量，給予對手其數量x300點的傷害。
    5. 自己的準備階段中必定發動。令這張卡返回持有者牌組。

時械神 拉奇翁（時械神（じかいしん）ラツィオン）編輯

 

  • 除共通效果外，戰鬥結束後會將對方墓地所有卡返回牌組，可以有效打擊需要利用到墓地資源的牌型，但也有幫對方回收卡的危險。
    • ⑤效果是對面抽牌階段扣1000血，通常用來補刀。
  • 原型是猶太教中的座天使「拉結爾」，又被稱為「神之神秘」。
    • 拉結爾（Ratziel）別名為「至高之父」，為「動性」的象徵。
    • 數字是2，顏色是灰色，寶石以綠松石為象徴，行星以天王星為象徴，代表男性原理。
    • 對應原質為智慧（Wisdom）：智慧、純粹理性、創造原點；亦代表著恆星天／原動天。

時械神 米西翁（時械神（じかいしん）ミチオン）編輯

 

  • 除共通效果外戰鬥結束後直接削掉對面一半生命值，而且不屬於效果傷害，是極其強力的削血效果。因此常常在MD各種活動變成限一卡。
    • 配合其他能燒血的【時械神】怪獸，幾回合下來加快燒血的成效，是玩家最希望開局抽到的一張。
  • 原型是猶太教中的天使長「米迦勒」，被認為是最接近神的存在。
    • 米迦勒（Michael）是神話中的天使長，生命之木中央的位置，是補充所有生物之能源的中心。
    • 數字是6，顏色是黃色，金屬是金，行星以太陽為象徴。
    • 對應原質為美麗（Beauty）：慈悲、調停、美；亦代表著太陽天。此原質正是生命之樹的核心。

時械神 亞飛翁（時械神（じかいしん）ザフィオン）編輯

 

  • 除共通效果外戰鬥結束後將對方的後場全部返回牌組，可以打擊依賴後場續航的牌型。
    • 除此以外是唯一具有送墓後遺言效果的時械神，可以抽1張，動畫中的補牌能力則是誇張到一次抽5張。
  • 原型是猶太教中的「亞夫結」，為座天使的指導者，又稱「神之門衛」。
    • 亞夫結（Zaphkiel）有「至高之母」的別名，和拉結爾是相對應的關係，賦予所有事物的形體。
    • 數字是3，顏色是黑色，寶石是珍珠，金屬是鉛，行星以土星為象徴。
    • 對應原質為理解（Understanding）：執行、實質的理性、創造之泉；亦代表著土星天。

時械神 加布里翁（時械神（じかいしん）ガブリオン）編輯

 

  • 除共通效果外戰鬥結束後直接把對手全場的卡洗回牌組，以解場方面的能力來是最為強力。
    • 不過對方會因此抽出對應數量的牌（額外牌組怪獸會洗回額外牌組不會計算在抽牌的數量內），因為幫對手補牌的效果風險太大，採用率不高。
  • 原型是猶太教中的「加百列」，負責傳達神的旨意。
    • 加百列（Gabriel）意味著「星光體（Astral）」，也就是靈魂與肉體之中的靈氣，卡巴拉以此表現「前存在物質」。
    • 數字是9，顏色是紫色，金屬是銀，行星以月亮為象徴雖然它是衛星但古人不懂這個，我們就別糾結這個啦，代表全裸的男性。
    • 對應原質為基礎（Foundation）：基礎、萬物的基礎、神的創造力；亦代表著月球天。

時械神 薩狄翁（時械神（じかいしん）サディオン）編輯

 

  • 除共通效果外戰鬥結束後將生命值（4000以下時）恢復至4000點。
    • 因為原作的LP就是4000點，算是保留原作效果，但是OCG生命值是8000，也因此這樣使用上特別微妙，倒是可以當做關鍵時候的救命手段。
  • 原型是猶太教中的七大天使之一「薩德基爾」，司掌水，象徵「神之正義」。
    • 薩德基爾（Zadkiel）意味著純粹而神聖的宇宙法則——「愛」。
    • 數字是4，顏色是藍色，金屬是錫，圖形是正四面體，寶石是藍寶石，行星以木星為象徴，代表坐在王座上的王。
    • 對應原質為仁愛（Love）：愛、仁慈、恩寵、偉大；亦代表著木星天。

時械神 拉斐翁（時械神（じかいしん）ラフィオン）編輯

 

  • 除共通效果外，戰鬥結束後將與之戰鬥的怪獸攻擊力數值傷害直接給予對手。
    • 如果配合上面幾隻時械神將對手打到殘血，這張卡則具有極強的收尾作用。
    • 或者配合裝備魔法《脆刃之劍》裝備給對方攻擊力超過2000的怪獸，不光只有對方受到戰鬥傷害還會因為這張卡的效果再受到傷害，直接打成OTK。
  • 原型是猶太教中的「拉斐爾」，司掌治癒。
    • 拉斐爾（Raphael）有物質型態的「鑄型」之意。
    • 數字是8、顏色是橙色、金屬是水銀，行星以水星為象徵。
    • 對應原質為光輝（Splendor）：尊嚴、收縮、光榮；亦代表著水星天。

時械神 卡麥翁（時械神（じかいしん）カミオン）編輯

 

  • 除共通效果外戰鬥結束後彈回對面一張卡然後扣500。
    • 燒血功用和解場能力都較為貧瘠，不能被效果對應倒是這張卡唯一的優點。
  • 原型是猶太教中的「卡麥爾」，是能天使的指揮官，「神之力」的象徵。
    • 卡麥爾（Camael）別名是的「天使的外科醫生」。
    • 數字是5，顏色是紅色，圖形是五角形，金屬是鐵，寶石是紅寶石、行星以火星為象徴，也被稱為天空的外科醫生。
    • 對應原質為嚴格（Severity）：法、神的權利、惡的發現、怒；亦代表著火星天。

卡片資訊編輯

  • 效果怪獸
    等級10／地屬性／天使族／攻擊力0／守備力0
  • 效果：牌組中的這張卡不能被特殊召喚。
    1. 若自己場上沒有怪獸，則可以不用釋放怪獸就召喚這張卡。
    2. 這張卡不會被戰鬥以及卡牌效果破壞，且因這張卡戰鬥而對自己造成的戰鬥傷害為0。
    3. 這張卡進行了戰鬥的戰鬥階段結束時必定發動。令對手場上1張卡牌返回持有者牌組，並給予對手500點傷害。不能對應這個效果的發動來發動任何卡牌效果。
    4. 自己的準備階段中必定發動。令這張卡返回持有者牌組。

時械神 海隆（時械神（じかいしん）ハイロン）編輯

 

  • 除共通效果外戰鬥結束後（生命值比對方地時）將對方的生命值變成跟自己一樣，能夠一口氣將雙方的生命值扳平。
    • 因此衍生出故意將自己血量壓低，然後用這張卡將對方的血量拉得一樣低，最後配合任意一個本家時械神或任意燒血效果將對方燒死的戰術。
  • 原型是猶太教中的權天使「漢尼爾」，七大天使之一，力天使和權天使的君主，又稱為「神的榮耀」或「見神者」。
    • 漢尼爾（Hamiel）含有「豐饒」之意，還有有堅實，勇氣的意義在內。
    • 數字是7（來自聖經創世紀之中上帝「創造的七天」），顏色是綠色，金屬是銅，寶石是綠寶石，行星以金星為象徴，代表全裸的女性。
    • 對應原質為勝利（Victory）：永遠、膨脹、勝利；亦代表著金星天。

卡片資訊編輯

  • 效果怪獸
    等級10／地屬性／天使族／攻擊力0／守備力0
  • 效果：牌組中的這張卡不能被特殊召喚。
    1. 若自己場上沒有怪獸，則可以不用釋放怪獸就召喚這張卡。
    2. 這張卡不會被戰鬥以及卡牌效果破壞，且因這張卡戰鬥而對自己造成的戰鬥傷害為0。
    3. 這張卡進行了戰鬥的戰鬥階段結束時必定發動。若自己的生命值比對手少，則給予對手其相差數值的傷害。
    4. 自己的準備階段中必定發動。令這張卡返回持有者牌組。

時械神 桑黛翁（時械神（じかいしん）サンダイオン）編輯

 

  • 除《究極時械神 塞菲隆》外唯一一隻光屬性、具有攻擊力、且只能在對方場上有怪獸時才能直接召喚的時械神。
  • 打點高達4000點，不過不會給予戰鬥傷害，除共通效果外戰鬥結束後給予對方2000點傷害。
    • 通常是收尾時比較會用到，上面幾隻將對方打到殘血後才有奇效，本身也能搭配《技能抽取》，直接變成4000點的大拳頭還不會離場。
    • 如果以命名原型來看，這張卡象徵著地球本身，大概也因此將它設定為異色般的存在。
  • 原型是猶太教中的「桑德楓」。名稱的來源是希臘語「συνάδελφος」，意為「助手」。
    • 另有說法是他和梅丹佐是雙胞胎，司掌天國之歌。這種說法下其名字的意義是「兄弟」。
    • 桑德楓（Sandalphon）含有「物質的王國」之意。
    • 數字是10，顏色是澄黃色・橄欖色・紅褐色・黒的四色，寶石是水晶，行星以地球為象徴，代表坐在王座上的年輕女性。
    • 對應原質王國（Kingdom）：王國、物質、人；亦代表著四元素所合成，即是物質的存在。

卡片資訊編輯

  • 效果怪獸
    等級10／光屬性／天使族／攻擊力4000／守備力4000
  • 效果：牌組中的這張卡不能被特殊召喚。
    1. 自己場上只能存在1隻表側表示的「時械神 桑黛翁」。
    2. 若怪獸只存在於對手場上，則可以不釋放怪獸就召喚這張卡。
    3. 這張卡不會被戰鬥以及卡牌效果破壞，且因這張卡戰鬥而對自己造成的戰鬥傷害為0。
    4. 這張卡進行了戰鬥的戰鬥階段結束時必定發動。給予對手2000點傷害。
    5. 自己的準備階段中必定發動。令這張卡返回持有者牌組。

究極時械神 塞菲隆（究極時械神（きゅうきょくじかいしん）セフィロン）編輯

 

  • 本家王牌怪獸，動畫中必須用《無限光》的效果才能召喚出來。
    • 動畫效果極其霸道，可以盡可能從牌組拉【時械神】怪獸，並將他們攻擊力全部變為4000，自身則是上升全部時械神的攻擊力合計數值。
    • 任何與這張卡戰鬥的怪獸效果都會無效化，除此以外戰階時還能拿其他時械神怪獸給自己擋刀免除戰鬥傷害。
    • 因為這樣霸道的效果，誕生出了遊戲王動畫史上最具魄力的BOSS場面（4隻攻擊力4000+1隻打點高達20000的怪獸）。
      • 最終被遊星用《聚集的祈願》召喚出《星塵龍》一口氣飆到23000點，然後五☆連☆打成功取勝，成就系列動畫史上最燃的最終戰。
  • 最早O化的時械神之一，因為這個原因，他的效果設計除了提及天使族以外幾乎跟【時械神】沾不上邊。
    • 只要墓地躺了10隻怪獸就能跳，對於快速展開型的牌組易如反掌，然而本家怪獸常常跳回牌組＋幾乎場上只會站1隻怪獸，所以反而是個出不來的卡手物。
    • 能拉出手牌和墓地的天使族怪獸，並將它們變成無效＋攻擊力變4000，既體現動畫中的拉怪能力，也能讓【時械神】們的打點飆到4000去揍人。
    • 目前最常用於天使族主題的牌組，例如【墮天使】【代行者】等，曾經的【影依】也有投入，在決鬥後期拉出高打點的《拿菲利》收尾。
  • 原型就是卡巴拉生命樹本身，源於希伯來文單字「ספירה／sefirah／數字」，意指構成生命樹的十個球體。
    • 卡巴拉在西方傳統神秘學中，是從一個被稱為「生命樹」的圖騰開始。
      • 這圖表包含著十個描繪「生命樹（Sephiroth）」的質點（單數請用Sephirah）。
    • 質點從1到10都被精心安排在它們原本的領域裡，每一個質點都有一個原型觀念。也就是說，質點表現出神格與創造這世界的步驟。
    • 在物質界中，根據著古老的觀念，它們表現在神聖的範圍。

卡片資訊編輯

  • 特殊召喚／效果怪獸
    等級10／光屬性／天使族／攻擊力4000／守備力4000
  • 效果：這張卡不能進行通常召喚。必須自己墓地裡存在至少10隻怪獸才能特殊召喚這張卡。
    1. 1回合可發動1次。特殊召喚手牌或墓地裡1隻等級8以上的天使族怪獸。使用這個效果特殊召喚而來的怪獸的效果變成無效，攻擊力變成4000點。

同步怪獸介紹編輯

時械神祖 武加大（時械神祖（じかいしんそ）ヴルガータ）編輯

 

  • 【時械神】唯一存在於額外牌組的怪獸，不過並非Z-ONE所持有，而是由《ARC-V》漫畫版的Eve所持有的BOSS級卡片。
    • 其他兩隻分別是《幻魔帝 托里歐吉克》、《NO.XX 無限黑暗霍普》，即分別對應《GX》、《5D's》、《ZEXAL》的最終BOSS王牌怪獸。
  • 時械神本家的怪獸素材根本出不來這隻卡，因此多半只會到其他牌組打工。
    • 同樣是不會被破壞+不會造成戰鬥傷害的共通效果，但是戰鬥步驟結束時就能一口氣除外對方場上全部怪獸。
      • 雖然回合結束需要歸還這些怪獸，但是這個效果屬於發動類效果，只要將這張卡處理掉就不需要還給對手，是個極其強力的清場卡可惜本家用不到。
  • 《武加大譯本》（Biblia Vulgata）是一個5世紀的《聖經》拉丁文譯本，由耶柔米自希伯來文（舊約）和希臘文（新約）進行翻譯。

卡片資訊編輯

  • 同步／效果怪獸
    等級10／闇屬性／天使族／攻擊力0／守備力0
    • 協調+至少1隻協調以外的怪獸
  • 效果：
    1. 這張卡不會被戰鬥以及卡牌效果破壞，且因這張卡戰鬥而對自己造成的戰鬥傷害為0。
    2. 從額外牌組特殊召喚而來的這張卡，進行了戰鬥了傷害步驟結束時才能發動。除外對手場上所有怪獸。發動這個效果後，對手受到的所有戰鬥傷害減半，直到回合結束為止。
    3. 在發動了這張卡的②效果的，回合的結束階段中必定發動。將自己用那個效果除外的怪獸，盡可能地特殊召喚到對手場上。

相關魔陷編輯

  • 共三張永續陷阱，名字來源均跟生命之樹相關的概念有關聯，且是一張接一張往前遞進的關係。

虛無械 Ain（虚無械（きょむかい）アイン）編輯

 

  • 第一階段的準備卡，動畫中就是為了方便讓時械神不需要釋放進行通召，《Tag Force》起這個效果移植為【時械神】的共通效果，這張卡則變成了本家補牌專用陷阱卡。
    • O化以後保留了這個設定，還多了回收墓地本家後從牌蓋下一階段的卡到場上的效果，給予了在場上只有1次的破壞抗性，提供了本家特化方向。
  • 「アイン(Ain)」是與生命之樹相關的概念，表記方式是「0」，代表無。

卡片資訊編輯

  • 永續陷阱
  • 效果：
    1. 這張卡以表側表示存在場上期間僅1次機會，這張卡不會對手的卡牌效果破壞。
    2. 1回合1次，選擇下述其中1項效果才能發動。
      ● 丟棄手牌中1隻等級10的怪獸才能發動。自己抽1張牌。
      ● 自己的魔法&陷阱區域只有這張卡的場合，以自己墓地裡1隻「時械神」怪獸為對象才能發動。令那隻怪獸返回牌組。然後可以將手牌或牌組中1張「無限械 Ain Soph」覆蓋在自己的魔法&陷阱區域。

無限械 Ain Soph（無限械（むげんかい）アイン・ソフ）編輯

 

  • 第二階段的準備卡，把前面那張送走才能發動。
  • 動畫中用於實現複數【時械神】怪獸在場因此提供手牌特召的效果，《Tag Force》中把前面的補牌效果追加為兩張，
    • O化後把手牌特召的效果改成二速效果，但是補牌能力直接砍掉了，跟前面那張一樣回收墓地本家把下一階段的階段蓋到場面上，以及每回合都有1次破壞抗性。
  • 「アイン・ソフ(Ain Soph)」表記方式是「00」，代表無限。

卡片資訊編輯

  • 永續陷阱
  • 效果：將自己魔法&陷阱區域1張表側表示的「虛無械 Ain」送去墓地才能發動這張卡。
    1. 這張卡每回合各有1次機會，不會被對手的卡牌效果破壞。
    2. 1回合1次，選擇下述其中1項效果才能發動。
      ● 任一方的主要階段中才能發動。特殊召喚手牌中1隻「時械神」怪獸。
      ● 以自己墓地裡1隻「時械神」怪獸為對象才能發動。令那隻怪獸返回牌組。然後可以將手牌或牌組中1張「無限光 Ain Soph Aur」覆蓋在自己的魔法&陷阱區域。

無限光 Ain Soph Aur（無限光（むげんこう）アイン・ソフ・オウル）編輯

 

  • 第三階段的完全體，同樣要把前面那張送走才能發動
  • 動畫中之前的效果都繼承前面那張卡，準備階段不用把【時械神】怪獸送走，只有召喚過10隻以上的怪獸才能除外這張卡召喚《究極時械神 塞菲隆》（O化後此為塞菲隆的自帶出場條件）
    • 《Tag Force》中同樣繼承之前的效果， 召喚《究極時械神 塞菲隆》的效果變成這張卡被送去墓地時發動，雖然放寬了限制但實現又變得困難了。
    • O化後大幅加強，擁有不會被對手破壞掉的抗性，然後是給予時械神效果指定抗性，不會因為自身效果準備階段回到牌組的效果得以保留，如果空場時則是還原動畫的《究極時械神 塞菲隆》的效果，從手牌、牌組、墓地三個區域無視召喚條件拉三隻時械神。
  • 卡名中的アイン・ソフ・オウル(Ain Soph Aur)」表記方式是「000」，代表「無限光」。

卡片資訊編輯

  • 永續陷阱
  • 效果：將自己魔法&陷阱區域1張表側表示的「無限械 Ain Soph」送去墓地才能發動這張卡。
    1. 這張卡不會被對手的卡牌效果破壞。
    2. 自己場上的「時械神」怪獸不會成為卡牌效果對象，且雙方不能令場上的「時械神」怪獸返回牌組。
    3. 1回合1次，自己場上沒有怪獸的場合才能發動。無視召喚條件，特殊召喚自己手牌、牌組、墓地各1隻名字不同的「時械神」怪獸。
