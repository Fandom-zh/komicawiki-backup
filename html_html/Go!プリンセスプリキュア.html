  • 光之美少女系列的第十二輯（第十代），於2015年2月1日接續黑歷史《ハピネスチャージプリキュア!》播出，簡稱「GPPC」、「姫プリ」、「ゴープリ」，又稱「公主組（princess）」。
    • 這代主題是「公主」與「夢」，公主天使表示：
    • 值得注意的是從抓心組之後捨棄的手套配件回歸了。
  • 由於心跳組和幸充組連續兩年的慘淡收場，劇組全面大改革：
    • 光美系列初代製作人鷲尾天以企劃的身份加入。另外根據一些訪談，雖然他是企畫，但實際上至少每一集到劇本定案為止他都會參與。
      • 可能也是因為這樣，本代的敵方組織設定上比較接近前三代的絕對惡，扣除身為追加光美的黛維萊德以及苦命中間管理職夏特還略有點人性以外，大部分的敵方幹部都沒有什麼能被洗白的內容。
    • 製作人除了戰犯柴田宏明外，東映方亦加入神木優。
      • 後來在光美播放總數第555集的賀詞的身分標記中更直接表明神木將是第四代製作人，柴田只做到本季，甚至看起來柴田根本已經不管這棚了
      • 第24集以後柴田宏明正式除名，神木優正式成為第四代製作人[1]。
    • 監督為原光美演出的田中裕太，是首次擔任。
    • 編審為田中仁。
    • 人設為中谷友紀子，是首次擔任。
    • 劇本陣容加入擔任過超級戰隊系列與假面騎士系列的老手編劇香村純子。
      • 因此戰鬥上也出現了一些過往光美不會看到，但在騎士與戰隊很常看到的合理化橋段。
      • 例如第6集，老師與同學用盡各種理由拖延，導致小南與綺羅羅沒辦法第一時間跟小遙會合。
      • 在綺羅羅與母親史黛拉的個人回戰鬥中，利用史黛拉的夢想從中產出兩個絕望獸，好讓另外兩人不會沒戲唱。
  • 因為劇組的大改造，風格完全改變，製作人柴田也發表了這代光美將會「非常不光美」的言論
    • 其實是指柴田在同年的春季電影《All Stars 春のカーニバル♪》慘淡收場後，被丟給《手裏剣戦隊ニンニンジャー》當輔P這一之前從未出現的情況
    • 結果最後指的是劇情與作畫受到好評，但總收視率與銷量卻還輸給幸充組這樣爆冷門的奇怪情況
      • 挫敗帶來的絕望曾經在財報出來後狠狠的打了在最終回說過會不斷跨越絕望的繁花天使的臉
    • OP除字體轉用秀麗的新字體外，整個OP也充滿古早《少女革命》風格，少女革命プリキュア
      • 裡面某些場面還會隨著劇中的登場角色及故事演變而有所改變。
    • 本作是歷代首部粉紅光美登場時並不是國二生的一部，同時亦是第一位不是國二生的粉主。
      • 第二位不是國二生的粉紅在六年後的熱帶組，人設也一樣是中谷
    • 本作是光美首度出現全住宿制校園生活，也因此大部分的受害者學生在出場回後還是偶爾會再次登場。
      • 由於劇情需要，還是有不少外宿戲。
    • 變身畫面應該是光美史上最絢麗之一，能跟當年有錢到令人望塵莫及的音樂組相提並論，不愧是公主
      • 特別受到好評的是變身完畢集合時以千為單位飄出的花瓣（繁花天使）、泡沫（人魚天使）和星星（閃亮天使），以及火焰搖曳背景（深紅天使+後期四人變身）的絢麗特效。
    • 每話結尾會與騎士一樣排列出最為該話重點的變身道具，第38集和41集除外，因為第38集就是傳說中的光之美少女第五個暴言的出處，以全黑背景代替；第41集則以小唯的畫作代替
  • 由於本作強調各光美的成長和個人特色，加上「公主」的主題讓她們絕大部份時間都以禮相待，這代各光美成員之間的距離感可說是歷代最明顯，不過距離再遠也是近在咫尺，這代還是有百合股的存在（主要是「遙×南」和「綺羅×永久」）
  • 四人的變身口號為皆為「プリキュア・プリンセスエンゲージ（Precure Princess Engage）」。
  • 首播時第1集不幸的恰好遇上恐怖組織「伊斯蘭國（ISIS）」處決日本人質的突發新聞，差點就要跟戰隊棚和騎士棚一同觸礁，結果只有烈車中槍
  • 播畢的後日談。

      • 原本預定10月8日出貨的設定資料集也延到10月底補印，小遙表示跨越絕望了。
      • 而且因為本作被公認故事是劣作的心跳+幸充和魔法+甜點夾著，也顯得本作的劇情表現更出色[2]，但是台灣跳過本作……
    • 日本網站「animeanime」在光美15周年時發起了最喜歡的光美系列投票。雖然公主組僅僅以2107票拿下第一，但再次證明公主組的定位是近代光美最佳。

特色編輯

  • 本作全員歷代第二（三）次以粉紅、藍、黃作為初始成員。
    • 同時也是第二次以粉紅、藍、黃、紅為全員。
    • 本作為歷代第三次出現由反派洗白而成的正規光美。
    • 同時暌違兩年，本作的三個強化形態終於都不再偏心了
      • 但最終形態的輕裝版還是粉紅獨有
  • 本作是繼幸充組之後，歷代第二次沒有在劇中將光之美少女稱為「傳說中的戰士」的一代，而是「傳說中的光之美少女公主」[3]。
    • 本作自戰隊組第一季後，暌違九年，再次出現以女性為敵首領的光美，是歷代第二次。
  • 首次有泳裝回（第28集）。
    • 自《小魔女DoReMi》時代以來，泳裝都是此時段的禁忌，小魔女的OVA也有自嘲。
    • 美鬼表示...
    • 以前的光美也有去沙灘玩的情節，但都自動回避泳裝或下水。
    • 本作第16集也是海邊劇情，但當時是利用潛水衣達成神迴避。
    • 根據神木的說辭：「因為這代有海之光美嘛！」真該感謝小南，某KY藍表示：
      • 同時她也表示看回過去的表現，這種積累及迴避演出的理由應該做出思考判斷。
  • 自本作起的光美開始大膽描寫「長大成人後的主角群」。
    • 跟泳裝回一樣，過去因為很多大人的原因，故事結局並沒有明確寫明主角們長大後的模樣。但本作可能是因為主題是「夢想」，因此史無前例地在最終話的後日談簡單提及長大後眾人的狀況。
    • 似乎是評價不錯的關係，之後每一部光美最後一集都放膽寫長大後主角群的動向，甚至還有像魔法組就是靠主角群在長大後的描寫一舉翻轉整體風評的例子
  • 本作是抓心組的月光天使之後，睽違五年，再次出現不露指的手套配件。
    • 自本作起，直到現在，每一代的光美都一定至少會有一位正規戰士穿上不露指的手套。
  • 自本作起，直到現在，团队隊員内的變身服開始變得差异極大，可以說完全幾乎沒有共同點。不放一起完全看不出一隊的了
  • 本作的變身口號為全員（初期+追加）共通，是自微笑組後暌違兩年再次出現此情況。

與新鮮組之間的巧合編輯

  • 本作的設定跟《フレッシュプリキュア!》有不少共通點：
    • 正規全員總共是4人+2妖精，配色分別是粉紅、藍、黃、紅。
      • 初期皆為粉紅、藍、黃三人行，紅是追加。
      • 妖精都是一雌一雄，且都屬於初期成員。
        • 雌獸的頭髮都可以被梳理。
    • 變身道具都有鑰匙。
      • 初期三人的首個大絕都是徒手施放，且在後來第四人加入前都獲得魔法棒作為個人大絕的工具。
      • 兩者喊團體口號時，隊伍陣形的顏色配置剛好是左右相反（含三人和四人）。
        • 水果組是「藍、紅、粉紅、黃」。
        • 公主組則是「黃、粉紅、紅、藍」。
      • 兩者的課金強化道具都是音樂盒。
    • 相比其他隊友，粉紅沒什麼特別的長處。
      • 班上都有一名小時候使認識、常像冤家般跟粉紅鬥嘴、且同時對變身後的粉紅有好感的男性同班同學。
      • 粉紅變身前都屬於短髮，變身後都是金髮+長髮。
    • 藍都是四人之中最高。
      • 變身後都露出肚臍。
      • 同時都被封為ババァ
      • 一個怕章魚，一個怕鬼。
    • 黃變身前都是褐髮，變身後都是橙髮。
    • 藍黃二人的夢想：一個夢想當名模（且都已經是模特兒），一個是夢想當（陸/海）獸醫。
      • 兩代的差別之一在於夢想或顏色互相調換。
      • 夢想當名模的兩者都試過因為光美的使命而放棄過出國發展的機會。
        • 差別在於一個已知在劇終後會出國發展；另一個在本篇中卻沒提及會否出國發展
          • 不過小說中，美希答應爸爸會在國三畢業後出國發展，而且兩者都是前往法國發展
        • 正規私服都是穿短褲和戴上髮卡。
        • 兩者的父親都在全劇中完全沒有台詞。
          • 兩者剛好：一個是四人中唯一已知不是獨生女；另一個是四人中唯一的獨生女。
      • 夢想當獸醫的兩者差別在於：
        • 一個父母已經是獸醫且當獸醫是一開始就有的夢想。
        • 另一個是父親是財團的CEO，且當獸醫的夢想是後來才有的。
    • 關於紅光美：東剎那對紅城永久。
      • 都是從反派洗白過來。
        • 作為反派時的常規造型都是穿著黑色衣服+紅色底色，且都是銀白色頭髮。
        • 而且都有被各自的敵首領賜予高級裝備予以強化，不過待遇就不同
      • 都是來自異世界，且最後都沒留在人類世界而是選擇返回故鄉
      • 變身器都跟隊友為同一規格。
        • 變身後頭髮的顏色都是粉紅色。
        • 武器都是特規的弦樂器，都是在初次變身為光美戰鬥時即擁有且可用以發動個人大絕。
        • 都沒有徒手施放的個人大絕。
    • 主角群最常吃的都是甜甜圈。
    • 全員都試過在粉紅家中團聚。
    • 在前期都至少被一名女性地球人配角知悉主角們的光美身份，且都被觀眾認為是後備軍。
      • 最終都向親友公布光美的身份。
    • 敵役都以異世界（紅光美的故鄉）作據點。
    • 最初三幹部最後全部都活下來，且都包括兩名洗白的男幹部。

故事簡介編輯

強さ・優しさ・美しさのプリンセスレッスン！
（堅強、溫柔、美麗的公主課程！）[4]
  • 女主角春野遙從小的夢想是成為一名公主，在她四歲那年與一位夢幻般的王子殿下相遇，說她一定可以成為公主，並贈予她一個鑰匙作為她的護身符。
  • 數年後，13歲的小遙進入了著名的宿舍制貴族學院「高貴學園（ノーブル學園）[5]」，進入學校的第一天就偶遇了來自「希望王國（ホープキングダム）」的皇家妖精「泡芙」與「阿洛瑪」，還被企圖讓全世界陷入絕望的邪惡組織「迪斯達克（ディスダーク）」襲擊。這時候，皇家妖精們將一瓶公主香水瓶交給小遙，讓他插入換裝鑰匙，變身為「繁花天使」，對抗來犯之敵。

水晶公主魔法棒（クリスタルプリンセスロッド）編輯

  • 初期三人的武器，為希望王國自古流傳下來的皇室秘寶。
  • 由彼方王子於第10集末尾尋獲，11集藉由【將夢想力量傳給彼方】這方法交付給面臨大危機的光之美少女們。
    • 這個方法是被捲入事件的小唯主動代替彼方王子傳達的。
    • 第一次因為克羅茲的阻擋而失敗，第二次加上小唯的夢想力量才成功。
    • 所以劇組為什麼不讓小唯(ry
  • 可裝上優雅或奇蹟換裝鑰匙，放出合體技。
    • 優雅換裝鑰匙能發動【光之美少女 三重聖光（プリキュア・トリニティ・リュミエール）】淨化敵人
    • 奇蹟換裝鑰匙能將水晶公主魔法棒轉換成附有小型心形緞帶裝飾物的模式，並發動【光之美少女 三重爆破（プリキュア・トリニティ・エクスプロジオン）】淨化敵人
    • 此外不使用華美型態的情況下插入優雅與奇蹟換裝鑰匙來發動攻擊或防禦技能，如上述必殺技所示。
      • 大公主模式能夠自由使出前述必殺技的強化版（如閃亮天使的滿月低吟），甚至能像GN Bit一樣同時造出以十為基數的魔法陣複數連續發動（如繁花天使的的玫瑰旋風與百合旋風）
  • 雖說平常是一人用一把，不過在必要的情況下，也可以交給隊友使用
    • 例如39集中，人魚天使和閃亮天使就把自己的魔法棒借給武器被打飛的繁花天使，並由繁花天使同時插入兩把換裝鑰匙發動雙重攻擊技。
  • 於48回中與下述的深紅小提琴一同遭強化過的洛克摧毀。但其碎片化作與人們的夢想連結的小型鑰匙，成為小唯等人脫出絕望牢籠與光美們成為大公主的關鍵
    • 不用等到劇場版上映，兩樣道具已於最終話重生並被王宮妥善保管，等待每年劇場版的使用

深紅小提琴（スカーレットバイオリン）編輯

  • 深紅天使專用武器，是彼方王子的小提琴轉變而成
    • 但是琴弓卻沒有跟著變化，反而出了另一把紅色琴弓。其造型令人有諸多聯想。
  • 每次發招前都會演奏一小段音樂。
  • 讓監督田中頗傷腦筋的支援道具
    • 田中在訪談中曾經表示說追加戰士用小提琴是BANDAI一開始就有要求的事，但是要怎麼自然將小提琴融入故事而不顯得突兀讓他很頭痛。
    • 之後跟負責音樂的高木討論的時候，本來只跟高木單純做小提琴的兩首獨奏，但高木則回說「既然這樣乾脆做二重奏怎樣？」之後田中終於煞到靈感將小提琴相關的劇情設定成現在這樣。

公主宮殿（プリンセスパレス）編輯

  • 第30回由一度被變為絕望獸的希望王國宮殿變化而來的新道具。
    • 以高雅換裝鑰匙發動個人技時可直接使用，發動合體技時必須由泡芙與阿洛瑪換裝成「女僕／執事裝扮模式」之後召喚出來。
  • 分成四部分：打開後能看見公主組圖徽的窗台、附四個鑰匙孔的主舞台、舞台旁的三層台階、帶有四人形象圖徽與小把手的旋轉台。
    • 鑰匙孔與圖徽都只有四個，所以小唯成不了第五人了（流言終結蓋章）
    • 僅能以此道具搭配高雅換裝鑰匙變為第四種華美形態「高級裝扮」（ドレスアッププレミアム，Dress Up Premium）
  • 能夠以高雅換裝鑰匙發動四人合體技【光之美少女 光輝希望（プリキュア・エクラ・エスポワール）】，將敵人封印於主舞台的魔法陣中，於宮殿出現後由四人從掌心中發出四色光芒淨化敵人。還會放煙火慶祝
    • 宮殿升起來的瞬間被網民吐槽根本是 《冰雪奇緣》的艾莎蓋城堡
    • 污たねぇ花火

    • 將敵人封印在主舞台的魔法陣中，同時四人化作七彩光芒從天而降，衝破四色魔法陣突擊淨化對方，並先後喊出「blooming」與「貴安」，不只放煙火還可以把周圍變成繁花盛開的春天景象
    • 發動的台詞「響徹遙遠的彼方」根本在暗示這是告白技吧

  • 意外地也能當一般的音樂盒用。
