電車でD

電車でD

目錄

概要編輯

  • 本作為頭文字D的同人作品，由關西的同人漫畫家きよ○於1999年發表，至今依舊繼續連載
  • 人物與故事主軸基本上都是依照原作頭文字D，只是所有車輛相關設定都換成了電車，有看過頭文字D並對鐵路有了解的人，讀起來會更有意思
  • 雖然本來就模仿不來，但還是警告千萬別模仿裡面的招式，於現實中鐵定出事故

登場角色編輯

關西地區編輯

Project D/D計畫編輯

不同於其他車隊，為專門遠征關西以外地區的車隊，人員也相當精簡，但因為是由神戶線RedSuns的高橋涼介所統籌，故絕大部分都是神戶線RedSuns成員所構成。

藤原拓海

  • 所屬：阪急電鐵
  • 駕駛車輛：阪急2000系

Project D的下坡戰駕駛，隸屬於阪急電鐵，雖然年輕但有著高超的駕駛技術，愛車為相當舊型但經過高速化改裝的阪急2000系電車，每次上場幾乎都沒有輸過。在擊敗眾多關西地區的好手後，被高橋涼介邀請進Project D，開始遠征關西以外的區域。性格上與原作相近，但不同的是對車輛方面稍有了解(但也僅限於阪急的範疇)。

高橋啟介

  • 所屬：阪急電鐵
  • 駕駛車輛：阪急8200系

Project D的上坡戰駕駛，愛車為阪急最新銳的8200系，原本與哥哥高橋涼介並稱兩大王牌，直到遇上拓海才初嘗敗績，由於好勝心強，爾後也不斷提升實力，並被涼介延攬至Project D，展開遠征之途。雖然性格和原作相近，也是橫衝直撞傲氣十足的類型，但卻顯得更缺乏常識，常被眾人戲弄(主要是涼介)，主要來說擔任受害者的局面居多。

高橋涼介

  • 所屬：阪急電鐵
  • 駕駛車輛：阪急7000系

Project D的領隊，愛車為7000系，神戶線RedSuns時期與啟介兩兄弟名震關西，直到看見啟介以及眾多關西好手被拓海擊敗，甚至親自挑戰且以些微之差落敗後，相信拓海就是D計畫的最後一塊拼圖，宣告引退並邀請拓海加入Project D，遠征關西以外的地區。與原作一樣是冷靜的分析派，駕駛實力也相當高超，但性格實在太怪異，各種無厘頭的發想、言行舉動與惡作劇，常搞得周遭無言以對(啟介堪稱最大受害者)，綜合前述原因堪稱本作最危險的角色。

中村賢太

  • 所屬：阪急電鐵
  • 駕駛車輛：阪急8000系

神戶線RedSuns成員，仰慕啟介且駕駛實力亦不差，尤其擅長下雨天的對決。在啟介與中里毅較量完後剛好遇上下雨，便趁機想挑戰拓海，同時打算藉此替啟介報仇，不料太低估拓海的實力而落敗。Project D計畫開始後便較常擔任各種如情報收集等等探勘任務(簡稱跑腿的)。也時常和啟介一起捲入各種麻煩中(多為涼介所發起就是了)。

史浩

  • 所屬：阪急電鐵

Project D的司令員，亦是神戶線RedSuns的老成員，負責情報收集、與其他車隊的各種協調、起跑讀秒等各式任務。

松本修一

  • 所屬：阪急電鐵

Project D的技師，主要負責拓海的2000系的各項維修整備，戲份不多。

宮口

  • 所屬：阪急電鐵

Project D的技師，主要負責啟介的8200系的各項維修整備，拿下眼鏡後是絕世美男子。

寶塚線SpeedStar/高速之星編輯

由拓海的好友阿樹與池谷前輩等人所構成，雖然只是玩票性質，但因同為寶塚線的關係，拓海也常來串門子，甚至會跨刀幫忙出賽。

池谷浩一郎

  • 所屬：阪急電鐵
  • 駕駛車輛：阪急7000系

寶塚線SpeedStar的隊長，同時也是拓海及阿樹兩人的前輩。和原作相同駕駛技術並不特別出色，主要負責前期拓海的各種約戰協調，之後轉為各類型的吐槽(無論是拓海還是神戶線RedSuns的成員們)。

武內樹

  • 所屬：阪急電鐵
  • 駕駛車輛：4050系救援車

拓海的死黨，在本作中相當少見的正常角色，時常和拓海一起當班，同樣也是負責擔任吐槽角色。愛車為4050系救援車，由於算是輔助車輛而無法上場比拼，直到後期換裝發電機後，在拓海的操控下，擊敗從東京來踢館的對手。

健二

  • 所屬：阪急電鐵
  • 駕駛車輛：阪急6300系

寶塚線SpeedStar的成員，池谷的好友同時拓海及阿樹兩人的前輩。但不知為何平常還是負責京都線的班次較多。登場的次數不多，和池谷一搭一唱負責各類型的吐槽。

NightKids/狂夜小子編輯

關西地區的車隊，實力亦有中上階級，與其他車隊不同處是成原來自不同的路線甚至是鐵道公司。

中里毅

  • 所屬：近畿日本鐵道
  • 駕駛車輛：近鐵21000系

NightKids的隊長，擅長上坡戰的駕駛，愛車為改裝RB26馬達的黑色塗裝近鐵21000系，其優勢是對於高出力的車輛依舊能掌握自如進而發揮應有實力，但致命傷是缺乏冷靜，故從拓海以後的幾次敗績都是相近原因所至。性格較為老實，故也容易上當。與同隊的庄司慎吾互看不順眼，但很多地方卻又意外地有著相同論調，私下都認同彼此的實力。綜合來看目前戰績最差的人(加上輸給岩城清次的話共零勝三敗)。

庄司慎吾

  • 所屬：京阪電氣鐵道
  • 駕駛車輛：京阪800系(京津線)

NightKids的第二把交椅，主攻下坡戰，愛車為車廂長度極短的800系，時常為了追求勝利而不擇手段。與隊長中里毅的關係不睦，起初見到中里輸給拓海，便想藉由擊敗拓海讓中里顏面掃地，不料各種卑鄙的戰術激怒了拓海，反而被拓海超車，本想在最後藉碰撞讓彼此雙雙出局，卻被拓海閃過導至自己整輛車衝進了琵琶湖。相對於中里毅擔任笨蛋的角色，慎吾在之後便較常以解說員的身分登場。


無所屬車隊編輯

藤原文太

  • 駕駛車輛：銚子電鐵Deki3牽引車(改標準軌距)

拓海的父親，藤原超音速模型店的老板，雖然表面上不問世事，但卻暗中幫助拓海成長。前國鐵時期的駕駛員(愛車為EF58電力機車，曾經擔任過天皇夫婦專用列車的駕駛)，技術超群堪稱本作最強，連拓海也無法超越。除了常傳授拓海各種電車的駕駛技巧、到正雀列車機場協助阪急2000的整備與改裝外，有時更會以蒙面俠駕駛的身份出現與拓海單挑進行實戰(包含以前國鐵現役時期，除了駕駛Deki3以外都是蒙面現身)。疑似有女裝的癖好。

佐藤真子&沙雪

  • 所屬：京阪電氣鐵道
  • 駕駛車輛：京阪8000系

號稱京阪本線最速的兩人組，愛車為舊款塗裝的京阪8000系，與原作相同由真子駕駛沙雪領航(但沙雪不知為何是穿便服出場)。聽聞拓海與阪急2000系的傳聞後便要求較量，路線則為主場京阪本線，起初拓海陷入苦戰，但真子與沙雪對於不熟路線的拓海依舊能咬得那麼緊，感到壓力十足，最後操作失誤導致落敗。賽後真子邀請池谷前往上野搭乘寢台列車約會，但卻陰錯陽差而無法成行。女角畫風更勝於原作的最佳案例。

秋山涉

  • 所屬：阪急電鐵→東京急行鐵道
  • 駕駛車輛：阪急2800系

阪急電鐵不屬於任何車隊的駕駛，但實力足以和拓海、啟介等人匹敵，愛車為每節車廂都有輔助動力的大改阪急2800系，起初只是悄悄地在一旁觀戰，直到拓海與須藤京一交戰後發電機報銷，換裝新的發電機卻跑不快，經阿樹引線幫忙查看才正式有所交流，對於一開始拓海不懂自己的車輛換裝了那麼高性能的發電機還跑不出速度，以為他在故意裝傻不歡而善，直到拓海虛心請教才協助發現問題所在(主操縱桿的段數過少，無法搭派段速較多的新發電機進行細膩操作)。由於與拓海的阪急2000為同一個系列(2800系為2000系的最高階特急車輛)，競爭意識格外強烈。在拓海換上新的操縱桿後相約進行循環耐力賽，兩人殺得難分難捨，最終因為阿涉體力耗盡分心而由拓海勝出。另外在啟介的回憶中，阿涉與拓海接觸前有表示過想與高橋涼介較量，但被接洽的啟介與賢太婉拒(RedSuns原則上絕不在主場神戶線與人交戰)，不甘心的阿涉趁著啟介與賢太晚上試車時亂入強制開始交戰，佔有地主優勢的啟介完全無法甩開阿涉，最終因為碰上練習甩尾不慎翻車的塚本前輩，三個人撞成一團(啟介的夢境至此結束)。在輸給拓海後，阿涉與妹妹和美一同離開關西，回到故鄉關東並過檔到東急電鐵。極度的妹控，曾有多次爭議性發言，且藉由耐力賽鍛鍊出的強健體魄，就算被和美拿150公斤的鐵軌毆打也能立刻恢復。

小柏凱

  • 所屬：JR四國
  • 駕駛車輛：JR四國2000系

在四國與中國地區號稱無敗的強悍駕駛，愛車為罕見的柴油車JR四國2000系(前後兩節為柴油動力引擎，中間四節為渦輪燃氣引擎的魔改車輛)。儘管柴油車整體速度不如電車是不爭的事實，但憑藉著渦輪燃氣的動力，加上單線軌道磨練出的飛天超車技巧，出其不意的戰術讓小柏凱戰無不勝。繼承了父親的宿願，以擊敗國鐵時期最強對手藤原家為目標，前往關西挑戰拓海使出渾身解數，讓拓海毫無超車的機會，直到最後關頭藉由路線的高低差加速才以些微之差取勝。

塚本前輩

  • 所屬：阪神電氣鐵道
  • 駕駛車輛：阪神9000系

拓海學生時期的學長，因為仰慕飛車的帥氣而開始練習，只是甩尾等技巧似乎還需要多努力(在啟介的夢境中甩尾失敗翻車收場)。本身也是個御宅族，除了鐵道外對於攝影、動漫也有著相當程度的了解，興趣算是十分廣泛。

御木前輩

  • 所屬：南海電氣鐵道
  • 駕駛車輛：南海50000系Rapi:t

拓海學生時期的學長，忌妒拓海與夏樹兩人，便藉機把她約出誘拐上車，得知此消息的拓海立刻前往搭救，起初看到老舊的阪急2000系還不以為意，沒想到兩三下就被拓海追上，而且不會複線甩尾導致最後出軌收場，拓海也成功救回夏樹。常有一些異想天開的計劃(連身旁的小跟班都忍不住吐槽)，或逼迫夏樹一起玩電車遊戲，比起原作完全是脫線型的角色。


其它角色編輯

立花祐一

  • 所屬：阪急電鐵

在正雀基地負責各類阪急電車的維修，與文太是多年好友，曾多次協助拓海維修甚至是改裝阪急2000系。對於電車以及鐵道的知識相當豐富，時常向拓海等人分享情報以及過去的經驗談。

茂木夏樹

拓海的第一任女友，對鐵道有一定程度的了解，初次登場時只是學生，畢業後當上空姐(但實際上的夢想是電車駕駛?)。雖然與原作相同疑似有援交而讓兩人的關係降到冰點，但夏樹被御木綁架時，拓海卻也奮不顧身地前往搭救。最後向拓海表明想前往東京發展，因此選擇分手。

秋山和美

  • 所屬：阪急電鐵→東京急行鐵道

秋山涉的妹妹，雖然不是電車駕駛，但受到阿涉耳濡目染對電車也是相當熟悉。起初與阿樹意外相遇，雖然說是有好感但沒有正式交往。力大無窮，能單手持長3公尺重150公斤的鐵軌毆打阿涉，在阿涉與拓海較量完後決定與阿樹告別，與阿涉一同回到故鄉關東。

高橋緒美

涼介與啟介的妹妹，從與啟介的對話能得知應該是重度的鐵道宅，但是不是鐵道的從業人員沒有相關描述。

小柏健

小柏凱的父親，以前國鐵時期是文太的死對頭，後來敗於文太手下後引退，但暗中訓練兒子成為電車駕駛打算日後復仇。平常總是穿著忍者裝並手持布偶。

中部地區編輯

SEVENSTAR LEAF/七星瓣編輯

對照原作的日鹽七星瓣車隊，為Project D首次遠征的對手，隊上的兩大王牌都隸屬於名古屋鐵道。

末次徹

  • 所屬：名古屋鐵道
  • 駕駛車輛：名鐵1000系改

SEVENSTAR LEAF的下坡戰主力，駕駛風格也相當類似，愛車專為複線甩尾而設定的名鐵1000系改。為了迎戰Project D需要鍛造鋼輪等昂貴的耗材，不得不向社長兼女友的木下奈保低頭，因而約定如果不幸落敗，就要自願接受調離駕駛員的現職並專心準備升等考試。一開始以地主優勢取得領先，但始終甩不掉拓海，一時心急想模仿拓海使用單輪甩尾拉開距離，但不幸失敗以翻車收場。

川井淳郎

  • 所屬：名古屋鐵道
  • 駕駛車輛：名鐵2000系

SEVENSTAR LEAF的上坡戰駕駛，愛車名鐵2000系具備了高出力與傾斜列車兩大優點，駕駛風格也不花俏，以紮紮實實地縮短時間為主。由於徹在前一戰翻車導致本線路線受阻，雙方協議更改到常滑線一決勝負，面對只看過一次路線的啟介，盡管各方面來說淳郎都佔有優勢，但在出征前經過涼介的特訓，啟介儼然已經脫胎換骨。在彎道中不斷被拉近差距，最後在大直線遭到啟介超車而落敗。賽後不知為何代替徹受罰而暫時轉調為站務員。


關東地區編輯

Emperor/帝王隊編輯

對應原作的伊呂波Emperor，來自關東地區的車隊，全員均為京濱急行電鐵的駕駛，且統一車輛為京急2100系，特地遠征至關西進行較量。

須藤京一

  • 所屬：京濱急行電鐵
  • 駕駛車輛：京急2100系

Emperor的隊長，愛車為改裝BOSE擴音器的京急2100系。先前到關西踢館時曾敗於高橋涼介手下，重整旗鼓後再次遠征關西。實力極為強悍，是首次讓拓海吃下敗績的駕駛，但爾後涼介出面應戰並奪勝成功扳回一城。在拓海的阪急2000換上新的發電機後，於主場京急本線迎戰拓海，本以為不熟悉路線的拓海會在最後的連續低速彎減速，自己便能趁機超前，不料拓海卻同樣的速度強行過彎並成功衝線，吃下敗仗後承認拓海的2000系的確厲害，並認為他之後會出人頭地。在之後Project D的許多賽事也到場觀戰並客串解說，加上個性死板反而讓原作的威嚴蕩然無存。實際上雖然未與Project D有過正式的過招(所有交戰都是在Project D正式成軍前)，但也時常在各方面幫忙Project D。東堂塾出身，與猛將館智幸為同期畢業生。

岩城清次

  • 所屬：京濱急行電鐵
  • 駕駛車輛：京急2100系

Emperor的成員，愛車為加裝前檔板大型霧燈(模仿Mitsubishi Lancer Evolution四代)的京急2100系。與原作不同的是是以腹語娃娃的方式出現，故不知是不是真有生命存在，更不知道他是怎麼駕駛電車的。實力僅在須藤京一之下，在拓海與賢太的雨戰中亂入，並代表Emperor車隊挑戰拓海，因為連戰連勝的自負輕敵，不聽京一的耳提面命而敗於拓海(彎中速度慢於拓海，路線亦被識破，遭拓海從外側強行超越)。每當京一生氣時就會被拿來當出氣筒(常被支解、粉碎)，但在某處有秘密工廠進行大量生產，倒也不用擔心。


東堂塾編輯

Project D第二次遠征的對手(原作為鹽那東堂塾)，隸屬於東武鐵道的隊伍，但本質上更像專業駕駛的補習班，除了現役學員的技術不在話下，過去更培育出許多如須藤京一、館智幸等實力頂尖的電車駕駛。

二宮大輝

  • 所屬：東武鐵道
  • 駕駛車輛：東武50000系

現役東堂塾學員中最強的下坡戰駕駛，實力就算是縱觀歷代東堂塾的成員也能排得上前三，愛車為改裝成雙操作桿的東武50000系，以毫無破綻的剎車技巧加上東堂塾極為強悍的駕駛風格，讓首戰上陣的拓海陷入苦戰。一直到最後關頭，被拓海解讀其剎車的時間點並成功反超後，一時心急操作失誤才落敗。在所有東堂塾成員中算是性格較為正常的角色

微笑酒井

  • 所屬：東武鐵道
  • 駕駛車輛：東武200系

東堂塾的上坡戰主力駕駛，對於冷門車輛有著獨特愛好，因此選用改裝高出力發電機的東武200系作為作駕。在第二戰與啟介交手時展現各種糾纏、阻擋技巧讓啟介無計可施，但最後依舊被啟介那絕不認輸的蠻勁突破防禦而落敗，原作中本名不明，但本作中本名就是微笑酒井。其他學員描述酒井認真專注起來表情會變得非常嚇人。

館智幸

  • 所屬：東武鐵道→JR貨物
  • 駕駛車輛：東武1720系DRC

東堂塾的畢業生，在JR貨物擔任駕駛，興趣是新體操，與須藤京一是同期且實力在所有東堂塾畢業生中已經是傳說等級(拓海與大輝在東武伊勢崎線交戰時刷出的最快紀錄，阿智光是試車就再次刷新了)，也因此當東堂塾找來阿智要向Project D復仇時，京一緊急連絡涼介，強烈建議拒絕比賽要不然就要涼介自己出馬才有些許勝算。經東武鐵道社長同意而借到展示車東武1720系DRC出戰的阿智來勢洶洶，但涼介最終決定接下戰帖並派拓海出面應戰，面對難以匹敵的實戰經驗差距，就算拓海天分再高也無法取得領先，直到中後段嘗試關燈好讓自己在難以被察覺的情況下逐漸拉進差距。最後關頭多虧涼介的亂入(涼介用超大型閃光燈拍照，讓稍稍領先的阿智視線一時被擾亂)，拓海才能以些微差距率先抵達終點。雖然對於這次比賽很乾脆地認輸，但由於賽後社長的破天荒舉止(將晴空塔發射升空、完整抄襲京都鐵塔作為新的公司大樓等等)，這一戰未被列入Project D的戰績。


東急電鐵車隊(正式名稱不明)編輯

Project D第三次遠征的對手，相當於原作的西北埼玉賽車聯盟，但在本作中沒有提到隊名。全員皆為東京急行電鐵的成員，隊長為秋山延彥

岩瀨恭子

  • 所屬：東京急行電鐵
  • 駕駛車輛：東急5000系(2代)

重度鐵道宅女，並且極度愛好戰國BASARA裡的伊達政宗。在與友人到關西旅遊時，因要拍攝試車中的阪急8200系，意外邂逅了啟介並一見鍾情。接到延彥的電話要她趕回東京迎戰Project D，發現首戰上坡戰的對手就是啟介時震驚不已，還以為啟介已經歸涼介及拓海所有(腐女視角)，並誓言要將啟介救出腐海。作為隊上最強的駕駛，駕駛性能卓越的新款東急5000系，加上路線又是相當講求技術性的東急東橫線，啟介才發現是不輸給東堂塾的硬仗，雖然最後驚險逆轉獲勝，但要如何面對恭子的熱烈追求才是最麻煩的問題。爾後在Project D遭到京成電鐵的隊伍暗算，導致阪急8200系無法出戰時，立刻決定把自己的5000系借給啟介(加上須藤京一大方出借的京急2100系台車，並由宮口操刀盡可能調整成接近阪急8200系)。總是稱呼啟介為政宗或達令，似乎有暴露狂的傾向。

秋山延彥

  • 所屬：東京急行電鐵
  • 駕駛車輛：東急5000系(初代)

秋山涉的堂哥，同時也是東急車隊的領袖，作中少數不穿制服的角色，設定上比原作更偏像美男子，雖然駕駛技術不算是非常頂尖，但擅長分析及思考(與涼介相似)，看到Project D在東急東橫線練習時就已經知道下坡戰難以贏過拓海，但為了確認自己的戰術正確，仍舊勇敢地以相當老舊的舊款東急5000系出戰(換裝高出力發電機並極度輕量化)，面對經過極端改裝的車輛以及使出渾身解數背水一戰的延彥，拓海反而處於劣勢，雖然最後成功找到機會超車，但是延彥已經完成該階段戰術上的考量，確認輕量級的電車就是擊敗拓海的關鍵，找來坂本並準備好又輕又小的東急300系，並提出在田園都市線進行復仇戰。在涼介被會川及一條暗算時，協助Project D迎戰京成電鐵的隊伍。

坂本

  • 所屬：東京急行電鐵
  • 駕駛車輛：東急300系

在延彥分析完拓海與阪急2000的弱點後，相約於東急田園都市線進行復仇戰的下坡戰駕駛，本身就有相當豐富的駕駛經驗，搭配延彥準備了重量輕且長度短的東急300系，讓沒有這類對戰經驗的拓海根本不知如何招架。原本田園都市線的路線變化大，原本應該是對熟悉環境的東急電鐵有利，但拓海始出與東堂塾阿智交戰時用過的盲目追擊，讓坂本無法明確判斷自己的位置並找到空隙超前，拿下復仇戰的第一勝。

秋山涉

  • 所屬：阪急電鐵→東京急行電鐵
  • 駕駛車輛：東急8500系

離開關西回到老家的阿涉，被堂哥延彥招募加入東急的隊伍，並接收了一輛即將報廢的東急8500系電車，並比照阪急2800系改裝成全M車(每節車廂都裝有發動機)，並做為復仇戰的最後王牌登場贏戰啟介，由於雙方都相當了解彼此的實力(雖然上次在神戶本線交戰只是啟介睡夢中的記憶)，也讓這場上坡戰更加激烈，甚至是同時抵達終點中央林間。由於為了分出個勝負，雙方協調折返跑回涉谷，除了對啟介及阿涉兩人的體力都是相當大的負擔外，下雨更是讓賽況更加惡劣，最後因為阿涉的失誤讓啟介勝出。

京成電鐵車隊(正式名稱不明)編輯

在東急電鐵的復仇戰結束後Project D緊接著的對手，相當於原作的土坂Lancer Evolution車隊，但在本作中沒有提到隊名。行事風格和原作，一樣為了名聲而不惜耍小手段取勝。

會川

  • 所屬：京成電鐵
  • 駕駛車輛：京成AE型(2代)

上坡戰的駕駛，性格略為粗暴，愛車為Skyliner用車2代京成AE型，其實駕駛技術並不算是特別高，多半是倚仗車輛的性能。見到Project D來到關東遠征，便想藉由擊敗他們來一戰成名。盡管這次的戰帖有幾分可疑，Project D依舊赴戰，果不出其然啟介在練習時就中了京成電鐵的招，車輛受損無法出戰。會川也打算以啟介無法出戰為由技術性取勝，但不料史浩卻死纏爛打地交涉，承諾盡快弄台讓啟介能上場的車，並先進行下坡戰以節省時間，雖然會川是勉強答應，但總是認為被小看而氣不過，因此叫了一批流氓混混準備比賽結束後讓Project D吃不完兜著走。

    • 然後與原作一樣，找來的流氓因為是啟介的舊識，最後下場是反被那群流氓給狠狠教訓一頓。

一条

  • 所屬：京成電鐵
  • 駕駛車輛：京成3000系

下坡戰的駕駛，比起會川駕駛技巧高明得多，盡管只是駕駛小改的通勤型京成3000系，卻不能掉以輕心。性格雖然較為溫和，但為了求勝也更會小手段。在第一場下坡戰開跑時，若有似無地向拓海透露會川烙人過來的事，並暗示拓海比賽不用太拼等擾亂其心思；雙方在機場二航廈準備起跑時，更趁機偷跑以拉開差距，各種骯髒招式都用上，足見其對勝利的不擇手段。

    • 最後下場ry。

無所屬車隊編輯

東京來的兩人

  • 所屬：JR東日本
  • 駕駛車輛：JR E233系(京濱東北線)

某天拓海與池谷等人晚上練車時，突然出現的兩人組，雖然嘴巴上說是要交流，但一連串不尊重人的言論，讓池谷等人決定挺身而出捍衛地主的尊嚴。一開始由池谷、健二駕駛阪急7000系出戰，但車輛性能差距太大，還是被E233反超，但隨後拓海開著阿樹剛完工的4050系救援車上前接手，以一連串令人屏息的技術讓東京來的兩人就算施展大決宇宙無敵，超級必殺全力煞車甩尾依舊被狠狠拋在後頭，最後還碰上涼介亂入使得兩人決定棄車逃逸，但不料前方還有啟介烙來了中里毅等關西的電車駕駛們堵人，讓兩人只能乖乖求饒。
