洛克人ZX角色列表

《ロックマンゼクス（洛克人ZX）》系列角色列表
本條目歡迎有興趣的朋友加以補完。

目錄

洛克人ZX編輯

主角編輯

  • 梵（ヴァン）
    • CV：小林沙苗（ZX）→野島健兒（ZXA）
    • 男主角，Model X的適合者
    • 十年前母親被非正規品殺死後成為運輸公司「齊爾威運輸」的成員
    • 名字來源是法語「風」
    • 英文名字非常可疑[1]
  • 艾兒（エール）
    • CV：小林沙苗
    • 女主角，Model X的適合者
    • 十年前母親被非正規品殺死後成為運輸公司「齊爾威運輸」的成員
    • 名字來源是法語「翼」
    • 手遊《X DiVE》中，於2020年9月的ZX聯名活動中實裝，之後同年的萬聖節更是實裝了萬聖版艾兒
      • 原版的艾兒能夠變身成 Model ZX；但萬聖版的無法進行變身
    • 2022年9月的ZXA聯名活動再度實裝ZXA版的Model ZX
      • 梵：淦！難道男角沒人權嗎

生命金屬編輯

  • Model X（モデルX）
    • CV：水島大宙
    • 擁有洛克人X力量的生命金屬
    • 變身後武器為手炮，可以施展二段蓄力射擊
    • 取得ZX型態後便不能使用，使用男女主角各破關一次後開新遊戲時會解除此限制
  • Model Z（モデルZ）
    • CV：風間勇刀
    • 擁有傑洛力量的生命金屬
    • 變身後武器為光劍及手槍，各自有蓄力攻擊
  • Model H（モデルH）
    • CV：緒方惠美
    • 擁有賢將赫爾琵亞力量的生命金屬
    • 變身後武器為雙劍，先後使用主武＞副武＞主武可以追加連擊，非水中地形可以使用空中衝刺，下螢幕會顯示攻擊對象的弱點與剩餘血量
  • Model F（モデルF）
    • CV：中井和哉
    • 擁有鬥將法布尼爾力量的生命金屬
    • 變身後武器為火炮，搭配下螢幕可以改變兩把槍的小子彈軌度，蓄力射擊可以破壞特定地形
  • Model L（モデルL）
    • CV：今井由香
    • 擁有妖將蕾薇亞丹力量的生命金屬
    • 變身後武器為長矛，可以在水中游泳和衝刺，下螢幕會顯示周圍地形，蓄力攻擊可以製造作為臨時立足點的冰塊
  • Model P（モデルP）
    • CV：稻田徹
    • 擁有隱將幻影力量的生命金屬
    • 變身後武器為苦無，可以照亮暗處，下螢幕會顯示隱藏地形與周圍敵人的標記，點擊標記可以對該敵人發動攻擊
  • Model O（モデルO）
    • CV：諏訪部順一
    • 擁有奧米加力量的生命金屬，破關一次後於N區域作為隱藏頭目出現
    • 在普通以上難度將N區域最深處的奧米加擊敗或是使用ZERO 3&4的卡帶連動可以獲得該生命金屬
  • Model V（モデルV）
    • 最原始的生命金屬，也是上述所有生命金屬的起源
    • 毫無疑問這個V代表的就是拜魯，是從諸神黃昏的碎片變成
      • 因此在英文版配合拜魯的拼寫變更（Vile→Weil）改名為Model W

Guardian編輯

  • 普蕾莉（プレリー）
    • CV：廣橋涼
    • 邊境警備隊Guardian第二代司令官
    • 雖然官方尚未證實，不過其真實身分很可能就是ZERO系列登場的愛爾特
  • 齊爾威（ジルウェ）
    • CV：風間勇刀
    • Guardian所屬，Model Z的適合者
    • 主角的前輩，表面上身份是運輸公司「齊爾威運輸」的經營者
    • 被賽爾邦操控而與主角戰鬥，最後為保護主角而將Model Z託付給主角後死亡
    • 名字來源是法語「風向雞」
  • 福魯布（フルーブ）
    • CV：小林通孝
    • Guardian副司令官，科學家

賽爾邦企業編輯

  • 賽爾邦（セルパン）
    • CV：梁田清之
    • 賽爾邦企業社長
    • ZX的最終頭目，Model V的適合者
    • 惡魔鋼彈
    • 名字來源是法語「蛇」
  • 普羅米迪（プロメテ）
    • CV：岸尾大輔
    • Model V代言人之一，別名「死神」，性格衝動
    • ZXA揭曉其真實身份是賢人阿爾伯特開發的新型洛克人「DAN-001」
    • 在海底基地一戰時企圖殺死阿爾伯特，最後被Model V吸收而死
  • 潘朵拉（パンドラ）
    • CV：伊藤靜
    • Model V代言人之一，別名「魔女」，性格冷靜
      • 不過在官方特別企劃的番外短劇中，私下卻是任性的女王系角色，會隨意使喚普羅米迪，甚至可以因為熱茶太熱了喝不下去而鬧彆扭，原諒普羅米迪的條件還是叫他去買GODIVA的巧克力
    • ZXA揭曉其真實身份是賢人阿爾伯特開發的新型洛克人「DAN-002」
    • 在海底基地一戰時企圖殺死阿爾伯特，最後被Model V吸收而死
    • 隔14年後於手遊DIVE首次實裝為玩家角色。

八大頭目編輯

  • 高伏猛禽‧哈伏特（Hivolt The Raptoroid；ハイボルト・ザ・ラプタロイド）
    • CV：伊藤浩資
    • 鷹型機兵，E區域頭目
  • 深淵鮟鱇‧露鈴莉（Lurerre The Abysroid；ルアール・ジ・アビスロイド）
    • CV：淺井清己
    • 鮟鱇魚型機兵，F區域頭目
    • 雖平時語調可愛，但另一人格卻是一回事，不但語調變得凶爆，甚至講出如世紀未風格漫常見的台詞
  • 火狩拳獅‧霍剎里奧（Fistleo The Predatoroid；フィストレオ・ザ・プレデトロイド）
    • CV：松山鷹志
    • 獅型機兵，G區域頭目
    • 由於其弱點位置偏低，在八大頭目中較難以Lv4擊敗
    • 自稱修羅，對弱者毫無興致，會用某勇者機人的大招
  • 紫影狒狒‧朋普魯（Purprill The Mandroid；パープリル・ザ・マンドロイド）
    • CV：高木涉
    • 猩猩型機兵，H區域頭目
    • 雖講話幼稚 ，但十年前是殺死主角母親的真兇
  • 颶刃鐮鼬‧希靳烈（Hurricaune The Wolveroid；ハリケンヌ・ザ・ウルバロイド）
    • CV：柳井久代
    • 鎌鼬型機兵，I區域頭目
    • 平時做事不擇手段並聽從賽爾邦命令暗中綁架市民並處刑，從中獲得電子妖精
  • 足錨水母‧蘭格卡（Leganchor The Gelroid；レグアンカー・ザ・ゲルロイド）
    • CV：龍谷修武
    • 烏賊型機兵，J區域頭目
    • 體型極為巨大，幾乎完全佔滿頭目房間
  • 鑽火鼴鼠‧焚莫洛（Flammole The Moleroid；フランマール・ザ・モルロイド）
    • CV：福原耕平
    • 鼴鼠型機兵，K區域頭目
  • 剛角犀牛‧保特陀（Protectos The Goreroid；プロテクタス・ザ・ゴアロイド）
    • CV：酒井敬幸
    • 犀型機兵，L區域頭目
    • 因為體型厚重也內藏不少軍火

洛克人ZXA編輯

主角編輯

  • 格雷（グレイ）
    • CV：平田宏美
    • 男主角，Model A的適合者
    • 沈睡在神秘研究所的思考型機器人少年，意外中被解放，後被賞金獵人團救出後為了尋找失去的記憶而展開旅程
    • 真正身份是阿爾伯特的後備身體--新型洛克人「DAN-003」，但在完成洗腦前被釋放而視為廢棄品
  • 雅雪（アッシュ）
    • CV：小清水亞美
    • 女主角，Model A的適合者
    • 賞金獵人少女，過去曾在非正規品的襲擊中生還
    • 真正身份是阿爾伯特血統的後裔

生命金屬編輯

  • Model A（モデルA）
    • CV：小林優
    • 被施加封印的神秘生命金屬
    • 變身後武器為雙槍，副武器可以消耗武器能源進行鎖定攻擊
    • 全名「Model Albert」，阿爾伯特為了自身野心而開發的備份系統
  • Model a（モデルa）
    • 通過小遊戲獲得的生命金屬
    • 變身後變成身型細小、8bit版本Model A，性能和本家的洛克人相似，一眾Model中最弱
      • 但某些時候能發揮不俗的效果，例如面對クロノフォス・ザ・トリデンロイド的アイスカーニバル，因為其身型細小，可以確實站在其下方完全迴避冰柱

三賢人編輯

  • 管理聯合政府LEGIONS（レギオンズ）的三名賢人
    • 三位的名字來源分別是元祖三博士萊德、威利、柯薩克
  • 賢人湯瑪斯（マスター・トーマス）
    • CV：柴田秀勝
    • 三賢人的筆頭，擁有律人律己甚嚴的領袖氣質
    • 在困難模式的特殊結局會在阿爾伯特垮台後將四天王收編，企圖重塑全世界⋯⋯
  • 賢人阿爾伯特（マスター・アルバート）
    • CV：安原義人
    • 三賢人的科學家擔當，性格溫厚
    • ZXA的最終頭目，普羅米迪與潘朵拉的開發者，也是生命金屬Model V的作者
    • 平常的模樣是替身，本尊在海底基地一戰時正式出現
    • 殺死普羅米迪與潘朵拉後吸收其絕望與憎恨使Model V合體成為空中要塞「銜尾蛇（ウロボロス）」
  • 賢人米海爾（マスター・ミハイル）
    • CV：長
    • 三賢人的情報管理擔當，講話毒舌

四天王編輯

  • 四位與前四天王(除幻影外)名字原型各取自神話傳說人事物
  • 赫利歐斯（ヘリオス）
    • CV：泰勇氣
    • Model H的適合者，蔑視一切的精神潔癖
  • 阿特拉斯（アトラス）
    • CV：生天目仁美
    • Model F的適合者，主張以戰鬥帶來進化的前軍人
  • 忒提斯（テティス）
    • CV：菅沼久義
    • Model L的適合者，熱愛海洋的環保激進份子
    • 同製作團隊作品《蒼き雷霆 ガンヴォルト》的ニムロド疑似是致敬他（水屬性、槍兵、環保激進等等）
  • 夏爾納克（シャルナク）
    • CV：羽多野涉
    • Model P的適合者，講話聲調如同機械般的殺手

八大頭目編輯

  • 衝焰瞪羚（Deerburn The Gazeroid；ディアバーン・ザ・ガゼロイド）
    • CV：乃村健次
    • 瞪羚型機兵，平時寡言並注重紀律，習慣開頭先吶喊在講
    • 變身後可以使用蓄力攻擊上衝或是飛踢
  • 火花薔薇（Rosepark The Floweroid；ローズパーク・ザ・フラワロイド）
    • CV：安井邦彥
    • 薔薇型機兵
    • 性好男色，用男女主角的戰前對話態度可說是天壤之別
      • 連被擊敗時台詞可是超變態的說
    • 變身後可以吊起垂直鋼索並在鋼索間平行移動
  • 時控冰鱟（Chronophos The Tridenroid；クロノフォス・ザ・トリデンロイド）
    • CV：小原雅一
    • 鱟型機兵
    • 變身後可以使用蓄力攻擊減緩時間，發動後換型態一樣會維持效果，使用得宜能秒殺頭目
      • 在陸地上只能擱淺，但依然能使用蓄力攻擊
  • 赤翼蜂后（Kaisemine The Wasproid；カイゼミーネ・ザ・ワスプロイド）
    • CV：安達麻里
    • 女王蜂型機兵
    • 變身後可以飛翔，以及吊起特定物件
  • 搖滾禿鷹（Condorock The Vulturoid；コンドロック・ザ・バルチャロイド）
    • CV：谷山紀章
    • 禿鷹型機兵，將戰鬥視為開個人演唱會的一面
    • 變身後可以在空中漂浮，以及貼附在牆上
  • 石獅雙子 阿&吽（Argoyle The Shisaroid / Ugoyle The Shisaroid；アーゴイル・ザ・シーサロイド / ウーゴイル・ザ・シーサロイド）
    • CV：水間真紀 / 中山依里子
    • 兩人一組的狛犬型機兵，因吽常閉口不言，由阿負責代其發言
    • 變身後可以製造分身撿拾遠方道具
    • 算是一眾頭目中最難拿取的奬牌[2]
  • 電氣刺蝟（Tesrat The Hedgeroid；テスラット・ザ・ヘッジロイド）
    • CV：菊池心
    • 刺蝟型機兵某知名藍色刺蝟客串
    • 變身後可以在狹縫中衝刺
  • 鋸齒冰鱷（Bifrost The Crocoroid；バイフロスト・ザ・クロコロイド）
    • CV：銀河萬丈
    • 鱷魚型機兵
    • 變身後可以破壞特殊地形
