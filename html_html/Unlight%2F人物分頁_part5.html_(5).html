伊普西隆編輯

  • エプシロン、Epsilon
    • 異名：The Revenant（歸還者）
    • 官方個資：
      • 生日：風月20日（3月10日）
      • 身高：202cm/體重：97kg/血型：？
      • 興趣：遺跡探險
      • 出生地：？
      • 對應角色：雨果
    • 別稱：202、天空樹[1]、圍裙[2]
    • 萌屬性：屁股、人外
    • 幻化為人類姿態的怪物。依靠著像泡沫般的記憶持續鬥爭著。
      • 曾經的連隊隊員，但是被渦吞沒過後，成為異界的科學家的實驗體，變得有些不像人類而得到了力量，長相也和連隊時期變得不同[3]
      • 幾乎所有有資歷的連隊隊員都對他的外表表現出了非常震驚、不可置信的樣子
      • 米利安更是直說出“你這樣子就是被渦給吞噬的戰士最後的樣子嗎”
      • 但單人立繪完全沒有奇怪的地方，不由得讓人產生這樣的理解，“在連隊戰士眼中的他和大小姐眼中的他長相不同，連隊戰士眼中的他確確實實就是個怪物”
      • 沙耶之歌嗎你
    • 名字Epsilon是第五個希臘字母，也是拉丁字母「E」的來源，大寫為E，小寫為ε，數學用語裏表示極小的單位
    • 和雨果是舊識，而雨果表現出了比他人更不敢置信的表情
      • L2卡面接蘋果，官方你是在推什麼
    • 頭髮亂的像長了耳朵，因此也是犬系男子的一員
    • 武器是中東風情的細長彎刀，第二專武徘徊（タジャウル）、第四專武破曉（ファジュル）都是以阿拉伯語命名。
      • 仔細想想，雖然是劍士，他的攻擊技能卻完全沒有用到彎刀
    • 技能則是以“墓地中使用過的手牌數量”為中心進行卡片的回收與使用
      • 由於最終殺招「次元侵奪」是根據墓地中4以上的卡片來決定攻擊力，所以其實實力受到地圖影響，並且會週期性地高攻，總的來說很看使用時機和戰鬥技術
      • 「次元交錯」真是好尻，果然披風就是要拿來掀；「次元侵奪」的胯太不堪入目了，真是個罪惡的新人
      • L3卡面的開腿也色情得不得了，miya老師究竟覺醒了什麼


波蕾特編輯

  • ポレット、Paulette
    • 異名：Skirmisher（斥候）
    • 官方個資：
      • 生日：花月12日（5月1日）
      • 身高：145cm/體重：44kg/血型：A
      • 興趣：攝影
      • 出生地：魔都羅占布爾克
      • 對應角色：艾莉亞娜
    • 別稱：妹妹
    • 萌屬性：妹妹、姐控、病嬌、狂氣、夜襲
    • 導正世界扭曲的集團，卡爾杜斯的一員。為了唯一的心願而奔走。形象色為紫色。
      • 性格單純認真，沒有想太多，會為了變得更強而不斷戰鬥
      • 和姐姐一樣臂力很好。三槍叔嚇壞
    • 雖然立繪和卡面看起乖巧可愛，但只要放招眼睛就會變成紅色，露出十分狂氣的一面。
      • 無法確認究竟是本性如此，還是加入卡爾杜斯的影響
    • 有著特殊的狀態「斷絕」，可以使對手無法回血。
      • 無論是事件卡、技能效果(包括HP對換，EX：靈魂仿製、溫柔注射、迷迭香)還是移階回血都無效。
      • 斷絕狀態能與姐姐的技能產生連動，得到斷絕狀態的對手若是受到「看護的理論」回復HP時，會改為造成傷害。
    • 除此之外，技能以隨機增減傷害為主，配合剝奪回復整整5回合的斷絕，擁有很強殺傷力，並且因為技能移動的原因，能夠較為簡單地卡中距離，是現階段非常強勢的角色
      • 痴狂亂步……是剝奪地板上的摩擦力來迴避掉攻擊，然後……連你的回復力也一起剝奪掉
    • 黑眼圈戰隊成員
    • 立繪中雙持的武器是Vz.61蠍式衝鋒槍（Vz.61 Škorpion）。「亂槍掃射」使用的武器可能是手提式M134速射機槍（M134 Minigun），一種在電影和遊戲中大活躍的高轉速多管旋轉式機槍型號，例如被阿諾德·施瓦辛格在《終結者2》使用。
    • 不知道為什麼，無論是戰鬥立繪還是卡圖都看不出究竟是單馬尾還是雙馬尾，只有技能圖能夠看出來


尤哈尼編輯

    • 異名：Procurator（代理人）

      • 興趣：收集戲畫
      • 出生地：導都潘德莫尼
      • 對應角色：？
    • 別稱：哈尼、Yo Honey、You　Honey
    • 萌屬性：騷
    • 負責執行密令的協定審問官《審判者》。尋求可以改革世界的力量。
      • 第三位審判官，被薩爾卡多提醒要他認真工作。
      • 技能名稱和都和光有關。
      • 夕輝守護者的技能圖被說臉有嬰兒肥。
    • 由於名字、風格和武器和星際大戰相似，實裝時剛好也遇到星際大戰新電影上映，令不少玩家認為是搭星戰順風車。
      • 但在12/11的生放送中繪師表示尤哈尼的設計完全和星戰7完全沒有關係
      • 不過考慮到設計本角色時星戰7根本沒有上映無視在2013年11月，星際大戰七部曲公布於2015年12月18日上映，所以當然和星戰7沒有關係
      • 就跟梅莉和小圓劇場版上映日期一樣剛好
    • 似乎被蕾格烈芙委以秘密的重任
      • 雖同為審判官，卻穿著和布列、老馬截然不同的輕型制服。L5卡面倒是有披上紅色披風
    • 以反彈作為作為防禦技的角色，擁有封印攻擊、防禦階段移動的技能。（來自12/11生放送中官方的介紹）
      • 「拒絕餘光」能夠將等同於盾數值~盾數值+1的傷害無效並反彈無效的傷害給對手。但無效的傷害數值一定要剛好等於盾數值~盾數值+1，超出或低於都無法產生效果，所以其實是相當難用的一招。
      • 「燦爛光刃」無法對攻擊力不足的部份補強，發招條件不易都是缺點，因為是防禦為主的角色才設定成這樣。（來自12/11生放送中官方的介紹）
      • 近期最爐角色

      • 除此之外，也有被擊打到就回復隊友的HP（類似放血療法），和自盡開卡格並回復隊友HP（類似秘密苦藥）的技能
      • 而如此強效的效果卻沒有較高的COST，導致她在中低區間橫行霸道
    • 剛好跟隔壁棚一樣拿雙槍

勞爾編輯

      • 身高：181cm/體重：82kg/血型：A
      • 興趣：讀詩
      • 出生地：尹貝羅達
      • 對應角色：瑟法斯
    • 別稱：
    • 萌屬性：快遞員小哥、打工仔
    • 所屬革命軍的士兵。祈望和平，專注於戰爭之中。
      • 官方在出卡前預告會和一個大家都沒想到的早期角色有關聯，結果竟然是阿奇波爾多
      • 所屬的革命軍正是尹貝羅達革命軍，理解帕蘭達因的志向。但從限定對話來看，他應該去世較早，沒能見證之後的事情
      • 向阿奇詢問革命之後的情況，並對傑多的長相表示驚訝
    • 以在近遠距離發動后設置中距離的陷阱，在對手踩上中距離的陷阱后進行各種技能的聯動為主要戰鬥手段，能夠給予大大小小的直傷
      • 然而設置陷阱非常吃牌，還會和其他技能的要求衝突到，並且中距離異常難卡，陷阱又只有1回合，並且要發動聯動效果還要被丟手牌，就算成功卡到手牌幾乎也用光了
      • 爐角戰隊又增一員猛將
    • 一開始玩家把他看做次世代主機遊戲常見的主角，但有人指出他很像快遞員之後，對他的印象就鋪天蓋地的朝著快遞員的方向偏移但是真的很像
      • 又因為小平頭、軍人、肌肉的形象被認為是非常基佬取向的角色，甚至很快出現了和里斯的配對


潔米編輯

    • 官方個資：
      • 生日：葡萄月3日（9月24日）
      • 身高：155cm/體重：44kg/血型：AB
      • 興趣：程式設計
      • 出生地：導都潘德莫尼
      • 對應角色：？
    • 別稱：喪屍護士、Ｔ病毒
    • 萌屬性：喪屍
    • 想找出人類可能性的工程師，把自己當作實驗體並將研究結果寫在手記中。
      • 皮膚的顏色接近灰色，和N貝姐非常相似，出立繪的時候大家都以為她會是被貝姐的能力影響的僵尸角色，但其實是工程師
      • 作為工程師是和瑪格莉特、羅索這種研究方向很極端的人比較合得來的類型，和他們也有特殊對話
    • 乍看之下風格比較跳脫，但其實是名叫“賽博哥特”（Cybergoth）的一種哥特風格
      • ⊂彡☆))∀`)所以依舊是哥特風遊戲
    • 能夠利用技能在公牌中製造特殊的病毒卡，必須要用病毒卡才能發揮能力，若是病毒卡數量越多，攻擊力會更強
      • 最大的缺陷是病毒卡只能給潔米使用，潔米以外的人物使用會受到兩點傷害，若是沒有潔米的狀況之下因為病毒卡過多造成卡牌狀況。
      • 對方若有病毒卡，且牌組沒有潔米的情況之下使用「海普利斯」，可以讓對方卡手牌，嚴重一點能夠使對方完全不能抽公牌。
      • 由於病毒卡一旦在公牌中製造就會一直存在於公牌之中，因此需要先行積攢一定數量的病毒卡才能發揮實力，戰法是類似帕茉與古斯塔夫的慢熱型


瑟法斯編輯

  • セルファース、Servaas

    • 官方個資：
      • 生日：葡萄月12日（10月3日）

      • 對應角色：勞爾
    • 別稱：色法師
    • 萌屬性：眼鏡
    • 尹貝羅達王國的軍人，關注著腐敗國家的革新。
      • 和勞爾及另外一名媽媽類型的女性是青梅竹馬的關係。對勞爾的對戰台詞感覺很像回到地上後要回老家結婚。
    • 衣著可能會讓部分強迫症患者感到難受[4]
    • 中分男子天團之一
      • 沃肯、布列、凱倫為其他團員中分F4？
      • 有玩家熱切呼吁諾伊庫洛姆加入天團慘遭導正
    • 在近中距離能夠發揮高攻擊力，出牌數比對手多兩張時更是可以擲兩次骰子，近距離的阿修羅。
      • 「飄揚的劍舞」只要盾1以上就能將自己和對手的移動力互換，但對手也沒出移動卡時就可能變成像是雙方麻痺般的狀況。由於條件非常簡單，能夠輕鬆將對手卡在近距離不能動彈
      • 「試探的一步」是把自己的3張手牌送給對手的技能，在生放送中曾說到雖然是隨機挑選，但會偏向先送數值低的牌，能讓對手的手牌塞滿數值小的牌。另外也有給累積異常狀態的對手聖杯或聖水，又或是讓潔米把病毒牌丟給對手等等的使用方式。
      • 由於強大的性能和破格的專武（近攻+6近防-1），能夠輕易在近距離達成40+骰的兩次攻擊，是新的魔角。

維若妮卡編輯

    • 異名：The Expel（驅遣者）
    • 官方個資：
      • 生日：霜月9日（11月29日）
      • 身高：158cm/體重：42kg/血型：O
      • 興趣：茶會
      • 出生地：古朗德利尼亞帝國
      • 對應角色：艾莉絲泰莉雅
    • 別稱：真姬、呸囉妮卡、T-Veronica[5]
    • 萌屬性：
    • 導正世界扭曲的集團，卡爾杜斯的一員。過去被留在地上的工程師後裔。
      • 因為身為工程師後裔，又是最後的卡爾杜斯，原本被認為會是很強的魔角，結果後來卻被認為是舊課金強度。雖說如此，只要使用得當還是能成為了不起的戰力。

      • 「人偶廢棄」能給予對手布偶數×2的傷害，並依據布偶數給予對手5回合的異常狀態。
