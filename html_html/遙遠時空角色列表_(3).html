一代角色編輯

主人公編輯

  • 元宮 あかね（もとみや あかね）/元宮茜/元宮阿卡內（CV：川上とも子）
    • 年齡：16歲（遊戲中依照玩家設定有可能是15歲），身高：160cm，血型是秘密[3]。
    • 初代的白龍神子，高中一年級生，主人公。
    • 在剛升入高中的時候和兩名朋友森村天真和流山詩紋一起從古井中被召入異世界。
    • 受左大臣之女也是星之一族的傳人藤姬的懇求和輔佐，與被龍之寶玉選中的八葉攜手退治怨靈。
    • 基本上是開朗的普通女高中生性格，但也因為周圍的情況超出常識範圍而恐懼不已。
      • 人設兼漫畫作者水野十子女士表示，她是最好畫的角色。因為不知道該怎麼設定，所以只好把她設成普通的類型。
      • 動畫的性格比漫畫要更加堅強一些，但基本上還是半斤八兩。
    • 漫畫版劇情，劇透反黑。
      • 一開始在不知所措和極度恐慌的情況下，迷戀上了召喚她的阿庫拉姆，但後來因為看到阿庫拉姆的殘酷後，轉而對他失望。
      • 初中時代邂逅了天真，粗線條的她主動和因妹妹失蹤而整個人消沉的天真搭話成為朋友，但在不知不覺中使得天真把妹妹的影子投在她的身上。
      • 詩紋也是被她幫來的雖說歸根結底發揮作用的還是天真。
      • 十足的一級フラグ建築士，中後期幾乎把所有的八葉都插上了旗，但也因為沒有太大建樹，淪落到不人姬的地步。
      • 最後的結局是跟（意外）一吻定情的源賴久。
    • 外貌看起來很像是兼差[4]。

八葉編輯

  • 源頼久（みなもと の よりひさ）/源賴久（CV：三木真一郎）
    • 天之青龍。生日：10月9日。年齡：25歳。血型：A型。身高：185cm。五行属性：木。
    • 武士棟樑（武士團的首領）之子，也擔當茜的護衛。
    • 龍之寶玉位置在左耳上部。
    • 傳統類型的忠犬，性格沉默寡言，不善表達，但會在必要的時候提出自己的見解。
    • 在八葉中的戰鬥力屬上乘（不論遊戲中的能力）。
    • 曾有一位兄長，在某次執行護衛任務的時候因保護不成熟的自己而失去性命（遊戲&漫畫設定）。
    • 漫畫中有一位不打不相識的朋友「鵺」（一種居於深山能變成人形的靈獸），但分別後許久的某天鵺被鬼之一族控制了精神。萬不得已的情況下賴久手刃了對方。
      • 很會照顧人。曾經幫助過險些被圍攻的詩紋，與天真是對手和朋友的關係。對茜的保護和照顧十分周全。
      • 在一次生病中迷迷糊糊奪取了茜的初吻其實在此之前早就把茜推倒過了，從此以後兩人的關係發生了本質的變化。
      • 即便失去記憶之後也留存著對茜的思念，最後兩人結為連理共回現代。
    • 水野女士曾經表示過賴久是自己最喜歡的角色。
    • 實際上是一想通就會立刻變成行動派的類型。
  • 森村天真（もりむら てんま）（CV：関智一）
    • 地之青龍。生日：4月2日。年齡：17歳。血型：B型。身高：180cm。五行属性：木。
    • 和茜同為從現代來的高中生，兩人同一年級。
    • 龍之寶玉位置在左臂。周圍的紋身和寶玉是同一時間浮現出來的。
      • 遊戲版是與茜一同到左大臣宅邸；漫畫版是在穿越過程中與茜和詩紋失散，獨自一人在京城生存，不久後與茜一行人合流。
    • 生性爽朗。但因為某些緣故，在學校被傳為不良少年。
    • 其實很擅長吐槽。
    • 草摩夾的兼差。
    • 重度妹控。
      • 實際上比茜要高一個年級，妹妹失蹤後，自己馬不停蹄尋找妹妹導致耽誤了學業。
      • 直至碰上了茜人生才稍微回到了正規，但此後就把茜當成了妹妹的替代品。
      • 據傳遊戲中，玩家攻略蘭的時候如果同時攻略天真，會有加成效果。
    • 腦子很好用，漫畫裡提到不怎麼讀書功課也不會太差。獨立生存能力很強，隻身一人來陌生的時代也能自食其力融入社會。
      • 不過也有不成熟的一面，比較容易衝動。
    • 總之是被大人稱讚可愛被年紀小的人當大哥仰慕的人。
      • 和賴久一開始合不來（實際上是單方面對賴久不順眼），後來關係還算不錯。也曾因為自己的不成熟被賴久警告。
    • 擊墜機一號。
      • 漫畫版來到京都之後對茜告白。等到完全解開心結時，又重新理清了對茜的感情不過為時已晚。
      • 對茜的獨佔慾很強，連永泉都被他警告了。
  • イノリ/夷乃里/伊乃里/祈/海苔（CV：高橋直純）
    • 天之朱雀。生日：8月18日。年齡：15歳。血型：O型。身高：163cm。五行属性：火。
    • 實習的鍛造師。八葉中除了現代人之外，唯一一個出身為庶民的人，所以沒有姓氏。
    • 龍之寶玉位置在額頭。
    • 額頭醬。
    • 心直口快，但因為這點常常跟人發生衝突。
    • 和來到京城不久的天真一拍即合，也發現了天真對茜的感情。
    • 年下，但不喜歡被當做年下。
    • 父母因事故而雙亡，和病弱的姊姊相依為命。
      • 世事難料，姊姊愛上了鬼之一族的人並與其珠胎暗結。因此受到了村民的排擠和欺凌，就此對鬼之一族極其憎恨。
      • 於是金髮碧眼的詩紋也未能倖免於難。
      • 重度姊控，不愧是天真的哥們。
    • 在漫畫中也被插了旗，但比他年紀大的人都開修羅場了，他還能冷靜地叫大家看清狀況鷹通你看看你。
    • 灼眼のイノリ。
    • 名字漢字的正確寫法是「夷乃里」。
  • 流山詩紋（ながれやま しもん）（CV：宮田幸季）
    • 地之朱雀。生日：2月25日。年齡：14歳。血型：O型。身高：158cm。五行属性：土。
    • 茜的好友，初中生，但絕對不中二。
    • 龍之寶玉位置在右手手背。
    • 祖父是法國人，所以天生金髮碧眼，在學校被欺負。後來因為茜和天真出手相助才走出陰影。
      • 據推斷如果沒有茜的相助，靠穿女裝來逃避現實的可能性高達八成。
      • 這般容貌在京城沒少給他帶來麻煩，不過因為他性格純良溫馴，很快就取得了周圍人的理解。
      • 遊戲中玩家在戰鬥時選擇逃跑，好感不會下降的人之一。
    • 雖然有些弱氣，但關鍵時刻會很堅強。漫畫版中不惜捨身保護力量暴走的茜。
    • 擅長做甜點。這一技能在京城也派上了用場。
    • 漫畫版被塞夫魯單方面信任又單方面誤認為背叛，從而被捲入了一系列風波，但到最後還是把塞夫魯當做朋友看待。
      • 所以連載結束的時候暫時沒有回到現代。
    • 麻吉天使。
      • 可能那個年代腹黑正太還沒有流行起來，桐生祟是十年後的事情了。
      • 不過從種種跡象看來，把妹的水準不低，說不定有不輸給友雅的天賦。
    • 草摩紅葉的兼差。
    • 絕對不是偽娘，更因為還沒到高中年齡，所以不是超高校級的程序員。
    • 本來是預定要和友雅一組的，但因為倆人年齡相差太大而作罷。
  • 藤原鷹通（ふじわら の たかみち）（CV：中原茂）
    • 天之白虎。生日：12月22日。年齡：19歳。血型：A型。身高：175cm。五行属性：金。
    • 治部少丞[5]（從六位上）。妥妥的公務員。
      • 漫畫裡不止一次發揮了自己的職業特長不管是正事還是非正事。
    • 龍之寶玉位置在頸部。
    • 本代的眼鏡角色。不准吐槽這個年代為什麼會有眼鏡。
      • 摘下眼鏡之後立刻年輕了不少，在漫畫版格外明顯。不過眼鏡不算是本體。
    • 性格認真嚴謹的工作狂，實際上很年輕在現代還沒到能飲酒的年紀，但看起來很成熟。
      • 百萬遙久裡有愈發年輕的趨勢，終於像個正常大學生而不是大學教授了。
    • 說教吐槽起來毫不留情，對象是友雅也不例外，不過也曾因為缺乏和女性來往的經驗被友雅捉弄。
    • 字面意義的紳士，不要想歪。面對身為鬼之一族的女性，也不忘關心對方的傷勢。
    • 漫畫版設定：本是庶出的孩子，生母去世後被交給嫡母撫養，所幸被嫡母視如己出。為了報答嫡母的養育之恩，才會孜孜不倦靠實力出人頭地。
      • 嫡母病逝後的靈魂被鬼之一族利用囚禁在怨靈中。但在茜的幫助下得以淨化。
    • 漫畫版的擊墜機三號，在弄清楚茜和天真的狀況後直接開了修羅場但被イノリ擺平了。
  • 橘友雅（たちばな の ともまさ）（CV：井上和彦）
    • 地之白虎。生日：6月11日。年齡：31歳。血型：B型。身高：183cm。五行属性：金。
    • 左近衛府少將。和鷹通正好是一文一武。八葉裡身份第二高貴的人。
    • 龍之寶玉位置在胸口鎖骨處。
    • 充滿成熟魅力的男子，風流倜儻。外表上看是個享樂主義者，性格讓人捉摸不透，總表現出一副優哉游哉的樣子。
      • 實際上是個文武雙全又有勇有謀的人，深得聖上信賴的幹將。
    • 女性關係比較複雜，已婚未婚無論身份高低都有可能進入守備範圍。
      • 蘿莉控。
      • 有聞香識女人觸碰肌膚就能辨別出對方實際年齡的能力，看來這位才是現今眾所周知的紳士。
    • 母親是正室出身。目前和雙親以及同胞還有異母兄弟同住。
    • 雖然登場很早，但成為八葉的時間很晚。
      • 動畫為了表現獲得寶玉還大搞了兩秒的脫衣秀，官方自重。
    • 人脈很廣，又才藝雙全。因為擅長彈奏琵琶，和《舞一夜》的多季史也有過短暫交流。
  • 永泉（えいせん）（CV：保志総一朗）
    • 天之玄武。生日：7月6日。年齡：17歳。血型：A型。身高：166cm。五行属性：水。
    • 法親王[6]，當今聖上的異母弟。
    • 龍之寶玉位置在左手手心，在各種宣傳圖裡很少有機會能看到。
    • 與世無爭的溫儒爾雅的少年，風雅之人，喜好吹奏笛子。
      • 但卻有在當今聖上即位前，被大臣們擁立起來爭奪繼承人之位的黑暗過去。最後以出家的方式主動退出。目前和兄長關係良好。
      • 出家前有敦仁親王、中務卿宮、深泉的別稱。最後一個是關係親密之人的特有稱呼。
      • 性格溫和也體現在遊戲裡，從戰鬥中逃跑後他的好感不會下降。
    • 因為擅長吹笛，所以和多季史有過一面之緣。
    • 漫畫版是在茜遭八弦琴詛咒昏迷的情況下被確定的八葉身份。本打算冒著必死的決心以生命為代價彈奏詛咒的琴，但在藤姬的提示下以水為媒介，用極其良好的音感找出了怨靈，最終達成了茜成功得救自己也存活的雙贏結局。
      • 這件事也成為了他找到新的人生意義的契機然後就變成了擊墜機二號。
      • 但是比別人更悲慘的是，他的心意不但茜毫無察覺一無所知，還被天真警告了。
      • 非常害羞，但會開腦內小劇場，某種程度上很有前途。
    • 看似纖弱，實際上也有很男子漢的一面。尤其是在保護茜的時候。
    • 擁有怪力的幾率高達八成。
    • 對酒精的抵抗力極弱，聞到酒味就能醉倒，醉起來酒品很糟。
  • 安倍泰明（あべ の やすあき）（CV：石田彰）
    • 地之玄武。生日：9月14日。年齡：21歳。血型：AB型。身高：179m。五行属性：土。
    • 陰陽師安倍晴明的得力門生。
    • 龍之寶玉位置在右眼下方淚痣。
    • 第一次看到一代宣傳畫的人，會有高達八成的幾率被他的糰子頭吸引。
    • 理智，沉著冷靜。口頭禪是「大丈夫だ、問題ない」。身為陰陽師的能力是一流，但是明顯缺乏人類的感情。被師兄弟們疏遠。
      • 但是也有好奇心很強，像小孩子的一面。偶爾也會有脫線的表現。
    • 真實身份是安倍晴明兩年前用自身的陰氣與其妻的亡骸上殘留的陽氣製造的人造人。臉上的封印會在產生人類的感情之後暫時或永久性消去。
      • 基本上可以說他就是為了神子而生的。
    • 遊戲中因為主人公的幫助而取得了正常感情。漫畫版向茜直球告白。
      • 八葉集體失憶的時候因為出生特殊而很早就取回過去的記憶，不過為了讓茜對他擔心而假裝繼續失憶。
      • 可見很有腹黑天賦。

星之一族編輯

  • 藤姫（ふじひめ）/藤姬（CV：大谷育江、劇場版：興梠里美）
    • 生日：6月15日。年齡：10歳。血型：推定A型。身高：135cm。
    • 左大臣最小的女兒，「星之一族」的末裔（血緣來自母親）。
    • 雖然年紀尚幼還是個小學生，但有著與實際年齡不相稱的成熟穩重。
