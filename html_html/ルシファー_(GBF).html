ルシファー (GBF)

Lucius／ルシファー／路西法
以下含有部分劇情，沒看過原著又不想失去自己唯一認同能比肩的存在，連計劃都因為看不起的不用品失敗的請不要看下去
  • 網頁遊戲《碧藍幻想》世界觀當中的一位男性星之民角色，原型角色於同為Cygames開發的巴哈姆特之怒系列登場。
  • 在三周年活動《どうして空は蒼いのか》中，以天司系統創立者這個身分正式登場。
    • 一般翻譯為《為何天空如此蒼藍》，簡稱空蒼，或天藍藍，因此系列簡稱空蒼三部曲，或天藍藍三部曲。
    • 原本只有第一部，但因為人氣太高所以官方最終決議追加後面第二部的《失楽園》，以及第三部的《０００》，順便補一下GBF的世界觀。

基本資料編輯

  • 名字：ルシファー（Lucius[1][2]/路西法[3][4]）
  • 稱號：星之研究者、墮天司之王、星之民代表[5]、法桑[6]
  • 身高：175cm→186cm
  • 年齡：不明
    • 因為是天司們的創造者，所以年齡自然也是至少2000起跳[7]。
  • CV：櫻井孝宏
  • 形象色：基本上是白色，但是官方周邊方面是淺藍色、深藍色、紅色[8]
  • 興趣：不明，推測應該是研究、探明進化的原因[9]
  • 喜歡的東西：朋友（指ルシフェル）[10]、毀滅世界、巨乳
  • 討厭的東西：不明、但是完全的理科腦讓他對不合理跟矛盾的事物沒甚麼耐性、神（指創世神）、、サンダルフォン（同擔拒否）、ルシオ

故事中經歷編輯

どうして空は蒼いのか編輯

  • 在回憶中以剪影姿態出現，說出製造サンダルフォン的理由。
  • 由於ルシフェル的完美，覺得自己的擔憂太多餘。

失楽園編輯

  • 於歷史VR中以正式的立繪登場，交代了他的野心，以及被自己的創造物處刑的始末。

０００編輯

  • 被ベリアル以ルシフェル的身體復活。
    • 剛醒來就發嚴重的起床氣，順手一刀把某黑衣男子捅重傷扔出去。
    • 不久便理解現在的身體原屬於誰，間接得知ルシフェル的死訊，不禁有些感慨跟悲傷。
  • 把打倒阻礙=主人公陣營的工作扔給ベリアル，自己直接去神之塔，
    • 在那將預言者誤認成ルシフェル，隨後預言者澄清的同時也向他解釋有關他們的世界真相，在得知了更多令他厭惡的，隨後與預言者戰鬥，勝利後將其核心吸收，化為完全的十二翼型態。

拝啓、大切な君へ。編輯

  • 以「不破大黑」這個名字登場。
  • 令人大失所望的官方學園現paro。


角色性能編輯

  • 目前僅有Boss版。
  • HL版關卡是GBF營運以來，第一個需要明確團隊分工合作才能通關的真．超高難關卡。
    • ルシファー的腦袋、ルシフェル的身體、ルシオ的核心，想想這難度也是理所當然的事。
      • 不過前段班騎空士是有可能單人通關的，官方也有限定的單人稱號獎勵。
  • 因為BGM很好聽，而且做為高難關卡需要不少通關時間，會聽很久，被玩家們說是音樂會。
  • 因為Boss本人人氣很高，而且能確實接觸到（攻擊與被攻擊意味），被開玩笑是握手會，還可以拿到紀念品（素材）。
  • 依照會場可容納人數，入場費可能需要30000HP。
  • 因為主流打法還算固定的關係，衍伸出很多相關梗。

簡評、其他資料編輯

  • Cy路西法系列中，GBF世界裡象徵墮天、向神發起反叛的那一面。
    • 用隔壁棚的設定來形容的話，儘管外型等相對原版的ルシオ偏向Alter，但性質更偏向Alter Ego。
  • 初登場時可能是因為官方原本只打算把《空蒼》做完就好，有很多設定與細節不明。
    • 除了當時立繪居然只有剪影外，連櫻井配音的聲線都與之後不同。
    • 因為原繪師虫麻呂設計的輪廓太有標誌性，加上御用聲優，玩家們少數能肯定的只有肯定是路西系列。
      • 一時之間，路西族三人的關係性變成劇情討論跟考據的熱門話題，同人圈也有諸多創作。
    • 即便於《失楽園》中公布正式立繪甚至更多細節設定，當下官方內部其實也還有很多未定空間，另外相關者的ルシオ中途也被修正過好幾次，幾次進出手術室的理由可能不是為了平衡，而是因為官方當時也不斷在考慮該給這三人怎樣的定位。
  • 在星之民中也算特立獨行，且異常優秀的存在。
    • 將ルシフェル製作成有著壓倒性美貌跟力量的特殊存在，導致被元老院忌憚。
    • 被羨慕著、被憧憬著、被忌妒著，即便同為星之民也對他的才華感到害怕，為此還針對其設下諸多不利限制，卻最後依然被耍得團團轉，繞了一大圈還是照著他的計畫走。
  • 於《０００》中公布，他是參照預言者的姿態被模仿、創造出來的存在，記憶跟知識、姿態都不夠完全，因此出於本能創造了跟自己互補的半身．ルシフェル，以達到完美的狀態。
    • 可以說，他將ルシフェル視為「另一個自己」，也因此對其有著絕對的信賴，即便在策畫計畫階段就知道自己大概會死在他手上，也直到最後都希望能跟他同在一個世界，被處決時也不曾怨恨過，僅當作必然事項接受。
      • 可能也是因為如此，對ルシフェル的消逝抱持著悲傷跟憤怒，雖說對ベリアル隨便敷衍帶過，卻對著神之侍者的預言者直言「跟我共享同一視野的存在只需要一個就足夠，但是那個人已經不存在，同時這個世界也要完蛋了。」
        • 僅一句話就展現出他對ルシフェル的特殊感情與定位。
  • 可能有將某階級以上的作品做成互相對應的偏好或某種堅持，雖然可以解釋成為了讓彼此互相牽制、平衡的保險，若考慮到他的出身，或許這種注重對應關係的習慣，是他無意識的某種本能。
    • 低階的天司別說明確的型態，根本只算是結晶狀甚至是無機物（就天使關那意兒），故不在這部分討論範圍。
  • Boss版的十二翼，據說原本是神巴那邊的設定，但是因為作畫困難，所以只好放棄。[15]
    • 不過顯然官方沒有死心，還是在GBF這邊實現了。
  • 自己是路西，也是唯一分別見過另外兩位路西並交談過的路西。
    • 也就是說櫻井孝宏在GBF自言自語（笑）的場合，八成都跟他有關係。
    • GBF三個路西中唯一一個低音聲線演繹的角色，官方給出的說法是因為被斬首又縫合到另一人的身軀上，導致聲帶受影響。
      • 不過《失楽園》的研究所時代回憶中的聲音就已經偏低了，應該是櫻井為了配合角色性格，被要求了相對《空蒼》比較明確的指示的結果。
  • 現在狀態到底算不算真正的活著是個謎。
    • 相較千年前還未被ルシフェル處決前，現時的ルシファー肌膚幾乎沒有血色，讓許多人猜測他現在可能處於算是殭屍之類的狀態。
      • 尤其Cy旗下還有某殭屍當偶像的動畫，神巴也有殭屍角色不說，GBF這邊也有實際稱不上活人的フェリ。
    • 不過就設定上來說，星之民本身也算是近乎不滅的存在，或許本來就是某些條件有達成便能復甦，只是跟原來活著的定義略有不同。
  • 與他有淵源，同被GBF本家宣告不會作為玩家可遊玩角色的ベリアル跟ベルゼバブ先後在GBVS實裝且大肆活躍，讓眾人期待他的登場。
    • 但依照ベリアル所言，目前似乎是失蹤狀態（不過可能已經復活）
    • 作為Boss的關卡似乎已經確定會實裝。
      • 真的要變握手會了。

碧藍幻想編輯

  • ルシオ — 父親，自己是模仿他被創造出來的，GBF世界所有路西族的起源。
  • ルシフェル — 兒子創造物，稱他為自己的朋友與最高傑作，並給予相應的信賴與特權，即使被他如預期般處決，也依然重視他勝過一切。
  • ベリアル — 兒子創造物，ルシフェルー同期同規格，能力與性格都相互對應的作品，對自己有著絕對的忠誠。
  • サンダルフォン— 孫子ルシフェル在自己指示下所創造的天司，原因為ルシフェル太過完美而被自己視為不用品，之後卻作為天司長繼任者破壞了自己的滅世計畫，同單拒否對象。
  • ベルゼバブ — 表兄弟同為星之民，似乎從遙遠的過去就特別敵視自己跟ルシフェル。
  • ミカエル — 火之天司，四大天司之首，創造物之一。
  • ラファエル — 風之天司，四大天司之一，創造物之一。
  • ウリエル — 土之天司，四大天司之一，創造物之一。
  • ガブリエル — 水之天司，四大天司之一，創造物之一。。
