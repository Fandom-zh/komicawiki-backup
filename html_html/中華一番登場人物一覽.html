  • 詳見角色條目

四郎[編輯]

  • CV：坂本千夏 / 藤井幸代（真）
  • 阿昴在西南修行之旅的第一回所遇到的中日混血的小屁孩
    • 動畫版追加仿冒特級廚師的設定（翁聲譽：小子不要學我）
  • 在隊伍上擔任不吉祥物的角色，也是作者最喜歡的角色
    • 很吵，意見一堆的牆頭草，使得各種場合只要四郎一開口就充斥著四郎（銅牌）閉嘴的評論。
  • 雖然前期比當初阿昴還要更死小孩，但是到漫畫尾聲確實是有成長，甚至還把了妹
    • 曾經因為菜被吃掉而塗改了健爺的地圖，差點害他累死不說，還險些引起中法戰爭
    • 把乃哥解魯裝通告費銀票的布袋燒掉
    • 曾比較過梅麗和顏麗的胸部大小
    • 動畫版更是節操蕩然無存，是為隊上最大戰犯，專門性騷擾梅麗（梅麗變成暴力女的罪魁禍首），唯一的功能大概就是在龍蝦三爭霸中選了阿昴的寶山飛龍鍋和幫阿昴亮黨證特廚徽章，並且取得銅器的地圖（動畫並無交代如何取得）
    • 在續作《極》中，煮了回鍋3次的回鍋肉讓守關的官員稱讚可是他老兄一開始還不會煮
    • 在阿昴要被吊到懸崖下鳥葬時很有義氣的衝上去救人，結果變成兩個人一起鳥葬

解魯[編輯]

  • CV：家中宏 / 中村悠一（真）
    • 演員：徐乃麟（五官很像）
  • 因隨身帶著一根鋼棍所以又稱為：鋼棍解師傅肛棍洩濕婦
  • 自稱「大陸第一麵點師」，人稱「鋼棍解」
    • 梅麗：好黑的鐵棒啊！
  • 山西加拿大[2]人，少年時開始遊歷中國學習各種麵點料理
    • 擅長料理為麵點製作的點心類料理，自稱大陸第一麵點師當時怎麼不說中原或大清呢
    • 通常為桿麵的時候會用到，並且可以拆成兩根長度折半，用來拍打製作肉餡，除外老解對於這跟鋼棍還有其他的使用方式，例如：釣魚、搞破壞、打架等
      • 阿昴：每個麵點師都有自己的棒子
  • 戰力不俗，渾身肌肉
    • 和雷恩兩人拆了翁聲譽的魔神殿而被梅麗吐槽乾脆去當武人算了
    • 動畫裡還把陽泉酒家砸了一半
    • 習慣用手抓吃東西，看來也在南亞旅行一陣子
  • 初登場與阿昴對決燒賣料理差點把阿昴逼入絕境
  • 個性豪爽嗜酒如命，也有圓滑逗趣的一面，算是團隊中的甘草因為四郎跟梅麗都是屁孩吵架太難看
    • 被四郎把裝銀票（旅費）的錢袋拿去當火種燒掉，但事後只是強迫四郎幫自己擺攤償還就不再計較
    • 曾因為喝太多所以被偽雷恩偷襲並搶走棒子
    • 有了上述的教訓後，還曾把全部的旅通告費（包含雷恩的錢）買酒一行人只好露宿破屋，差點害自己枉死
  • 鋼棍上的黃色星型符號數量代表著被他打敗的廚師數量[3]，黑色則表示敗場[4]（漫畫版曾被人吐槽過不要竄改戰績），平手是半黃半黑星
  • 第一眼見到「青眼虎」密拉就一見鍾情因為密拉正在綠洲洗澡，但是爭奪傳說廚具地圖中敗給了密拉後被逼跳崖，不過大難不死的他反而救了接著輸給雷恩而跳崖的密拉，最後成為情侶
    • 最初小川是想要畫解魯跟向恩的愛情故事，後來才變成解魯跟密拉
  • 總要懂得敬老尊賢吧？
  • 不錄了！[5][6]

雷恩[編輯]

  • CV：神奈延年、野田順子（幼年、少年） / 杉田智和、青木志貴（少年）（真）
  • 陽泉酒家的天才少年，十五歲（離開陽泉的年齡）
  • 本性善良，處理完動物類食材會在上面貼紅色的符咒超渡
    • 普渡完畢 (雙手合十)
  • 學會了南鮮酒家的技術之後四處遊歷，最後為了追求更多技術加入黑暗料理界
    • 之後了解了真相加上殺死羅歇（其實是殺人未遂，當下羅歇還活著，只是被向恩救活，又被向恩利用後再遭到向恩殺害）的悔恨反悔，於是決定要得到永靈刀來與黑暗料理界抗衡
  • 戰鬥力驚人，在漫畫裡一人就擺平了全部的黃山盜賊，也和解魯兩人就拆了翁聲譽的魔神殿，也因此和解魯一起被梅麗吐槽
    • 動畫被改為被盜賊數量壓制，是副官和其手下相救才逃出
  • 持有全國最好鐵匠羅歇打造的七星刀‧影打（動畫版設定，不過漫畫版羅歇確實說過這些刀還不是最完美的作品之類的話）
    • 在取得七星刀後因為不想讓其他人可以拿到更好的刀而失心瘋殺羅歇滅口
    • 因為最強的「北辰天狼刃」是只能用一次的菜刀北極玄冰製成，所以雷恩持有的七星刀其實只有六把，說是影打也不算冤枉
  • 擅長刀工、解體以及生魚片
    • 神前刀工對決時處理鯛魚竟然用到了湯霜法[7]
    • 雷恩如果去客串將太的壽司的話恐怕會變成最強的壽司師傅
    • 因七星刀的關係，是主角群中與向恩互動最多次的角色，但由於冷面冷心的特質，始終沒有進一步
    • 只會做生魚片，沙西米王
  • 中期轉為解說役，常和阿昴解釋一些知識
    • 你又知道了
  • 戰率：一勝[8]一平二敗
    • 動畫版則是零勝一平兩敗，零勝恩[9]
      • 神前刀工對決第一場平手，第二場輸給小當家，樓麟艦輸給向恩
      • 零勝閉嘴
  • 無口、豆腐心靈、傲嬌、病嬌
    • 初登場時氣燄高傲到一個極點有困難要說啊，跩屁，敗給阿昴後性格大變彷彿變了另一個人
      • 讓人不禁懷疑雷恩是不是輸了才決定跳槽
    • 漫畫得知羅歇被向恩滅口後崩潰，決定殺了她之後再共赴黃泉
    • 救出解魯後得到解魯作為報恩和抱歉用的大量點心，在品嘗完之後刻意說難吃
  • 作者在《極》第12卷發售的賀圖中畫了女裝的雷恩

蘭飛鴻[編輯]

  • CV：阿虜置鮎龍太郎 / 榎木淳彌（真）
  • 曾在大同館修行，與阿昴同時當上特級廚師的天才，暱稱阿飛
    • 名字原捏他是李連杰電影《黃飛鴻II男兒當自強》的提督納蘭元述（甄子丹飾演的那位）
  • 透露自己曾經被阿貝師傅救活，不然自己也將成為黑暗料理界的麟廚師，童年經歷動畫與漫畫有差別：
    • 漫畫裡阿貝與亞刊料理對決平手之後，用地圖交換仍在襁褓中的阿飛
      • 根據《極》中的描述，阿飛打從一開始就是黑暗料理界的人，反而是阿貝硬從黑暗料理界抱走襁褓中的阿飛，所以是橫刀奪愛的阿貝不對
    • 動畫裡黑暗料理界殺死阿飛的父母並強行擄走阿飛，自己從健身房逃出來的阿飛在大雪紛飛中於菊下樓門口前倒下，被阿貝救起來
      • 阿飛好腿力，從梁山泊（山東）走到四川，不然就是黑暗料理界有在四川開健身房
      • 基本上四川只有高山地區才會下雪，平地幾乎都不下雪的
  • 氣質塑造較為陰柔美型。廚師特考初賽時居然以絕代佳人的主題去對仗國士無雙的題考官有很大機率判他個答非所問、文不對題，後期被有瑞鳳紋的迦樓羅刀選中，甚至還男扮女裝化名貝蘭好參加御前料理對決…
    • 也就是說與主角的關係已經不能稱之基友，完全是正宮了
    • 後期和阿昴一起被稱作廣州廚師界的「天與地」（他是天上的那個）
  • 在廚師特考初賽做了給明朝皇帝龍鬚麵給評審吃
    • 你用明朝的麵來餵清朝的官，分明是反清復明
    • 動畫版在御前料理又做了一次，問題更大
      • 你用明朝的麵來餵清朝的帝，分明是（ry
  • 另外特廚複試時作出馬鈴薯冷麵，評價為九分，自評為十分
    • 小韓：我不懂這個麵裡面微微的甜味是什麼（一分）
      阿昴：是南瓜（八分）
    • 被問到為何自評為十分時說了我的答案跟阿昴一樣，也是很大機率判他個毫無創意、不會思考而被刷掉
    • 但是不僅過了還破例和阿昂一起通過以此可見名額是內定的
  • 戰績僅一勝（這唯一一勝還是對方棄權），其餘比賽都是平手
    • 動畫版無節操的劇情硬是讓阿飛吃下一敗
    • 在《極》多了一勝，但那是在裁判被下藥洗腦時做出的判決
      • 下一場也差一點戰敗，只是比賽被中斷而無效
  • 動畫版中自己無腦的向黑暗料理界挑戰，而被抓起來SM play用藥物洗腦而烙上標誌並參加御前料理對決
    • 之後靠著小當家使用傳說廚具的神秘力量煮的湯治好，後被李提督要求和小當家一起蜜月旅行尋找剩下的傳說廚具
  • 在漫畫中成為龍廚師，並在長城決戰後升格為宮廷廚房副料理長，但在之後辭職回到廣州
    • 成為龍廚師的目的是閱覽並複製宮廷秘藏的食譜，夢想是讓這些食譜回歸民間
    • 不過這件事惹毛了內膳房主廚，在《極》中派了人要帶阿飛回宮廷
  • 在西安被太極料理界抓起來，負責代表太極料理界參加西安麵點大賽
