模板:Fate Craft Essence


以下為模板的說明文件（[編輯]、[更新]）
請注意：此模版在某些情況下可能會部份或全體不可見。
檢視模板:Fate Craft Essence/doc編輯文件內容。(如何使用？)
為新模板建立說明文件時，會直接套用範本基底。

Fate Craft Essence是用來顯示 Fate 系列的 Servent 的專屬概念禮裝用的模板。

您也可以使用 {{Fate概念禮裝}} 來使用此模板。

參數編輯

此模板的參數均非必填，參數的說明如下：

參數 預設值 說明
image （禮裝圖像） 寶具圖像。可以是 wiki 內部圖片或是外連圖片
  • 使用wiki 內部圖片的方式： {{Fate Craft Essence|image=[[File:img.jpg]]}}
  • 使用外連圖片的方式： {{Fate Craft Essence|image= http://example.org/img.jpg }}
id 由數字、空格、點以外的符號、英文字組成的任意字串

例如可以是 test123、test-123，但不可以是test.123

若是有設 id 的值，則禮裝模板可以折合。

如果該條目使用多個概念禮裝，則每個模板的 id 不可重複（例如第一個模板的 id 為 test；則第二個不可以是 test）

關於 id 的使用，請看下方#加上參數 “id”

color inherit 禮裝名稱、讀音的文字顏色
bgcolor inherit 禮裝名稱、讀音的背景顏色
name ? 禮裝名稱
explain ? 禮裝敘述文
