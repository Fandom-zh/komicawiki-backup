解說編輯

基本編輯

 
第1＆第2期普通服。第1期見習服。第2期見習服。第2期見習服+女王晉見用披風。第2期皇家使者服。

  • 為魔女見習生的人類少女。

個人設定編輯

  • 1990年7月30日出生，獅子座B型。
    • 這個生日和做人物設定的馬越嘉彦的生日同一天。
    • 在飛鳥桃子加入魔法堂的行列之前，是所有小魔女中最早誕生的。[2]
    • 劇中因為太過繁忙，所以一直忘記慶祝自己的生日。[3]
  • 居住於XX縣美空市美空町的架空小町。[4]
    • 只知道可能位在東京，但公式上沒有透露是住在哪個縣市所以在這以XX縣來標示。
      • 本篇裡的各種訊息都暗示美空市美空町是位在東京都裡頭。
      • 但是OVA第1話裡腳踏車路線的起始點網友研究完他們的出發地點美空神社是位在鐮倉市。其實東映的作品很喜歡把主角的活動區域弄在鐮倉市。
      • 另外OVA第2話的寄給桃子的信的郵遞區號是橫濱市南區港南町。
      • 還有從OVA第13話Doremi的對話裡聽的出東京都在美空市的南方。
      • 總而言之美空市的位置一直是個謎還是說OVA是破壞公式的一種手段。
    • 父親名叫春風溪介(春風渓介)，是位釣魚狂作家，曾跑過去戰隊棚當紅戰士[5]。
    • 母親名叫春風遙子(春風はるか)，是位家庭主婦。
    • 有個能幹的妹妹泡泡。
    • 另外還有住在鄉下春風老家的祖父春風雄介、祖母春風陸。
  • 在小奏鳴曲幼稚園畢業後，進入美空市立美空第一小學就讀。
    • 3年2班→4年2班→5年1班→6年1班這樣升班上去。
    • 之後經歷美空市立美空中學後，進入XX縣立美空高等學校就讀。
  • 魔女見習服的主要顏色是粉紅色。[6]
    • 擁有的妖精叫「多多」。
    • 波龍所擁有的音色為「鋼琴音」。
    • 魔法的咒文是「霹靂卡霹靂拉拉 波波力那貝貝魯多」。
    • 魔幻舞台的咒文是「霹靂卡霹靂拉拉 輕鬆開朗」。
    • 皇家使者的咒文是「霹靂卡皇家使者」。
  • 設定上是扶養小花的真正的媽媽。[7]
  • 最喜歡的食物是牛排。
    • 是小時候的鋼琴獨奏發表會結束後第一次吃到牛排，從此愛上這一味。
  • 「Do-Re-Mi-Fa-So-La-Si-Do」為她的名字由來。

成為魔女見習生編輯

  • 非常嚮往魔女，從幼稚園開始就一直幻想著魔女的存在，那時候自己還學魔女一般唸著魔法咒文打算治療別人的傷痛。[8]
  • 有一天，抱著強烈欲望[9]的Doremi終於來到了「卷機山莉卡魔法堂」，遇上店內的主人魔女莉卡並且識破了他是魔女的身分後就此展開。
    • 被識破的魔女莉卡變成了傻瓜青蛙魔女青蛙，此刻魔女莉卡命令她讓她恢復原狀而要她成為「魔女見習」。
    • 從此Doremi開始進行了最嚮往的魔女的修行之路。
    • 這時候之所以想成為魔女的最直接的動機，就是「想擁有向最喜歡的人告白的勇氣」。
  • 剛成為魔女見習時，除了騎掃帚時搖擺不定外、常忘記自己是變身的狀態而橫衝直撞、魔法使用失敗、常害自己的同伴陷入苦戰等等的。
  • 在晉級測驗中曾出現兩次不合格紀錄，而且每次都是因自己最喜歡的牛排而不合格。
    • 最最最簡單的9級考試中，考試題目是要用魔法變出指定的食物。
      • 卻因為「世界上最好吃的～」的修飾語而自己幻想成牛排，進而擅自變出了牛排來。[10]
      • 魔女莉卡說道，9級不合格的魔女見習已經不知道幾百年沒看過了。[11]
      • 另外批發商魔女萊拉也說Doremi在魔女世界中成為了有名的人物。[12]
    • 3級考試時在終點前出現誘惑Doremi的牛排而讓她測驗失敗。[13]
      • 只不過3級考試被公認非常的困難，曾經讓許多的魔女見習生重挫而紛紛放棄並回歸到一般人類。
    • 另一方面，有藉由特例來晉級魔女見習生的考試。
      • Doremi在魔女世界的忘年會中參加才藝大賽[14]，最後得到優勝。
      • 優勝獎品是女王陛下可以實現她的一個願望，藉此Doremi就從4級越級跳到了2級。[15][16]
  • 最後的最後，和同伴一同放棄成為魔女。[17]
  • 只不過在第6期中以「不對自己使用魔法」的規則為條件，再次成為久違的魔女見習。

擺姿勢編輯

  • 變身完畢後做出了「Pretty Witch Doremi-Chi」(我是漂亮的俏魔女Doremi) 的姿勢，並且以「氣氛啊！」的理由來辯解。
    • 而這個姿勢最初只有Doremi在做，但後來魔法堂的其他成員加入後也學著她開始做起這個姿勢。
    • 也因此標準台詞是「Pretty Witch XX-Chi」(我是漂亮的俏魔女XX)。XX是名字，同時擺出姿勢。

成為小花的媽媽編輯

  • 小學3年級閉幕典禮的當天，成為了必須養育魔女嬰兒小花的母親。
    • 在劇中表現出的是以魔法堂裡的全員作為小花的母親。
    • 只不過第一時間剛好是Doremi接住了小花，也因此Doremi自認為她必須負責擔任她的母親。
    • 但事實上在第2期第1話的對話中，魔女世界的女王陛下是指名要由Doremi擔任小花的母親，而其他人負責輔佐Doremi。
    • 只不過故事的進展演變成魔法堂全員都變成了小花的媽媽。
    • 可是，第3期第28話中小花喊著要找「媽媽」時，卻在魔女世界裡用魔法只把Doremi一個人從人類世界拉了過去。
    • 以及第4期第1話中，在學校裡6年1班的教室裡明明Doremi和桃子同時在場，卻只飛撲抱住Doremi喊「媽媽」。
    • 種種的劇情解釋了在眾多的"媽媽"下，小花比較偏向Doremi為她的真正的媽媽吧。
    • 另外，第6期第3卷開始，小花因為有些事情而和Doremi一同就讀XX縣立美空高等學校，而且還同班。

與孫女的見面編輯

  • 小學5年級生，其孫女Fami(姓氏不明)初次見面。[18]
    • 身為魔女見習生，卻不知道Doremi也是個魔女見習，為了告知Doremi魔女是真的存在而回到過去的世界。
    • Fami的足以萌殺人的髮型、魔女見習的衣服顏色幾乎和Doremi雷同外，「世界最不幸的美少女」的口頭禪、喜歡牛排、非常冒失等等的，就像是Doremi般的隔代遺傳。
    • 雖然不知道Doremi成年後的家庭環境狀況，但以Fami的言語或許可以聽的出一些訊息。

很確定的是Fami繼承了春風家的女兒節娃娃的遺志。

人物編輯

外觀編輯

  • 紅頭髮，頭頂綁著左右對稱，俗稱「丸子頭」的兩顆巨大髮髻為她的註冊商標。
     
    • 在Doremi想像成自己變成魔女青蛙的樣子時(全身粉紅色)那兩顆丸子頭也還在。
    • 在劇中慢慢成長後，丸子頭也相對的漸漸變小了許多。
    • 那兩顆丸子頭從小學3年級開始就有著橡膠球般的彈性。
    • 魔女莉卡曾經氣到要把她的那兩顆丸子頭給吃掉。[19]
  • 很少會把她的丸子頭給放下來。
    • 即使是就寢睡覺也都不放下來。
    • 劇中平常只有在洗澡時才會放下來。
    • 在第2期第49話中因為捨命去摘取「真愛蘇布里姆花」受到巨大的衝擊而脫落。反倒是哭死和萌死一堆人。[20]
  • 在第6期時她還是一樣的冒失，國中2年級時以為是這種原因而被小竹甩了，所以一度把自己的頭髮剪掉。小竹你要負責![21]
  • 後來在進高校之前有舉辦一場小學同學會，她再度的綁成以前「丸子頭」髮型。
    • 只不過因為頭髮不像以前那樣子的長，所以只能綁成半球狀的樣子。
    • 處理了很久還是綁不好，最後藉由媽媽的幫忙才綁好，只是因此而遲到了，被玉木說「上了高中還是一點都沒變啊。」。
  • 使用謎樣的機器(美容院的那種)來做出她那種髮型。只不過在動畫中並沒有直接登場。
    • 在廣播劇CD中，Doremi在綁自己的頭時曾出現了各種難以形容的機械聲。
    • 動畫中只出現在Doremi打算幫小花做出丸子頭，小花從口中說出「那是什麼機器呀」的橋段。隨後產生巨大機器噪音後宣告失敗。[22]
    • 事實上在第2期第2話Doremi和媽媽一起洗澡的畫面中是使用梳子和吹風機來整理出丸子頭的。
    • 第6期開頭也有使用梳子梳理出丸子頭的插圖。[23]

性格編輯

  • 雖然自稱「世界最不幸的美少女」，但本身其實有個好奇心旺盛、樂觀進取且積極的個性。
    • 很少有自私自利的態度出現，常常取得大眾的信任。
    • 但也因此常出現意外或者被欺騙。像是利用她愛牛排的特點拐走小花的老頭阿迪。
  • 幼稚園時，曾在羽月最喜歡的兔子的臉上塗鴉。
    • 第2期第19話。兩人大吵一架完後，又發生了野狗闖入咬死兔子的事件。
    • Doremi隱瞞當時不在場的羽月而將兔子埋葬並且做了墳墓，因為害怕羽月傷心而編了"兔子的媽媽帶牠回家去了"的謊話。
    • 後來羽月知道事實後大罵「騙子」，讓Doremi因自責而很傷心。
  • 在小學2年級時曾把青蛙放在小竹哲也的桌子抽屜裡，看得出小時候也很調皮。[24]
  • 有「只要有妳在的地方大家都會覺得很開心，而且任何人都可以跟妳做好朋友」的評語。
    • 級任老師關老師、同班同學、以及魔女世界的女王陛下都非常認同這個觀點。
  • 一般情況下非常的冒失，連妹妹都比她還要可靠，果然是妹妹比你好(誤)。
    • 但經常會對遇到困難的朋友伸出援手，打動人們的心。
    • 無論做出任何事都不會有罪惡感，而且做出對自己討厭的事也不會感到後悔。但相反的，卻因為這種個性而壓抑了自我感情。[25][26]
  • 非常的積極戀愛，但小學時代卻一直失戀。
    • 第6期矢田優和她的朋友羽月的關係變得更好，連玉木都有不良男朋友而非常的感嘆。
    • 國中2年級時還因小竹而失戀，後來才知道是小竹太晚回信，實際上是沒有失戀的。小竹你要負責!

萌屬性編輯

  • 笨蛋

口頭禪編輯

  • 口頭禪是「我果然是世界上最不幸的美少女了～！」
    • 高校生版本是「世界上最幸福的(15歲)美少女了～」
    • 曾經被音符搶先一步高聲的說「我果然是世界上最幸福的美少女了～」，讓Doremi連自我哭訴的機會都沒有了[27]
    • 從未來來的Fami也會講這個口頭禪。果真是孫女。
  • 有個吸滿氣、臉頰嘟成球狀後臉左右甩，吐出「Bu-BuNoBu！」生氣的習慣動作。
    • 這個「Bu-BuNoBu！」連後來豋場的魔女嬰兒小花也學起來了。
      • 從嬰兒時期就會了，第四期第1話也有出現。
  • 常常出現「なのさ～」(那個啊...)的語癖。[28]

興趣、嗜好編輯

  • 5歲的時候曾學習過鋼琴。[29]
    • 但因為媽媽遙子的嚴格指導下，在首個發表會中因為太過緊張而失敗收場，從此討厭鋼琴。[30]
    • 這次事件後，媽媽為了要讓她心中的陰影消失，就把家裡她最心愛的鋼琴給賣掉了。
    • 後來在第2期因為泡泡也希望能學習鋼琴，所以鋼琴再一次的回到春風家中。[31]
    • 這之後Doremi也終於有想要彈鋼琴的意志，燃起了興趣再度開始練習鋼琴。
    • 第6部有繼續彈鋼琴嗎？小說目前還沒有提到這方面的訊息。
  • 在鋼琴發表會後回家的路上去了一間餐廳吃牛排，從此熱愛牛排。
    • 未來的夢想是「要開家牛排館」。[32]
    • 劇中出現很多次可以吃到牛排的機會，卻完完全全的錯失了所有機會。
      • 第1期第45話的平安夜中吃的「牛排」其實是雞肉做成的仿牛排。(其實是烤雞排)
      • 第3期第1話有提到說晚餐吃牛排，但因為新角色桃子的出現，所以就忘記有這回事而又錯失了機會。
      • 番外篇「ナ・イ・ショ」OVA第13話中的「Fami」的對話中能發現到，在Fami出生後或許也只吃過一次牛排吧～
  • 夏天的時候會有麥茶中毒症，曾用魔法變出麥茶來喝讓魔法堂的人們大怒。[33]
  • 「機動戰隊 正義特攻隊」的大粉絲。後來還模仿組合成「魔女戰隊 小魔女特攻隊」。
    • 曾在第1期第42話飾演魔女粉紅戰士、第4期第24話飾演魔女紅戰士。

與好朋友的互動編輯

  • 主要被稱呼為「Doremi(醬)」(どれみちゃん)
    • 關老師和部分男生都稱呼她為「春風」
    • 魔女嵐直接稱呼她為「Doremi」。
    • 玉木、島倉則稱呼她為「春風(桑)」(春風さん)
    • 小竹稱呼她為「冒失咪」(どじみ)(DoJiMi)[34]
    • 第6期足球社的成員們稱呼她為「經理」、「春風」、「春風學姊」等。
      • 另外後輩柏木千波稱呼她為「Doremi學姊」。
      • 相反的，玉木和小竹等一部份的人把敬語直接給移除掉看起來混熟了。
  • 從幼稚園開始就和藤原羽月是最要好的朋友了。
  • 從幼稚園開始一直和小竹哲同為同窗同學[35] 。這段期間小竹都會玩弄她，他們倆常常因此吵架。[36]
  • 小花事件後開始和FLAT4的小曉產生友好關係。
    • 因為在劇中小曉一直對她很好，所以每次Doremi看到他她都會發情。
    • 小竹因此吃醋，後來對他下戰帖。[37]
  • 同班同學的橫川信子所寫的小說漫畫中她被描寫成「忠犬Doremi」。
    • 忠犬Doremi聽得懂人類的語言，能以2馬赫的速度奔跑。
    • 第3期的忠犬Doremi被羽月博士改造成「忠犬變形Doremi機器人」。
      • Doremi機器人的必殺技是「Doremi機器人丸子頭飛彈」，由駕駛艙前的按鈕按下去就可以發射丸子頭飛彈。
      • 「小花劇團」中的魔女幼稚園才藝大會裡，這個Doremi機器人被當成壞蛋角色登場。[38]
    • 第4期第8話「忠犬Doremi」再度登場並改名為「忍犬Doremi」。只是被壞蛋圍毆而顯得非常的弱小。
    • 最終話信子寫的新小說中Doremi以主角的身分登場，只是被Doremi吐槽說這不是人類而是宇宙人吧。 [39]
