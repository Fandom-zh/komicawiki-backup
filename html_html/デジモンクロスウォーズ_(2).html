介紹編輯

  • 被東映遺忘的數碼獸系列第六輯，之後因為玩具銷量不錯而推出續篇《デジモンクロスウォーズ 時を駆ける少年ハンターたち》
    • 在大陸的WIKI譯為「少年四大名補（字沒打錯）」[1]，正確翻譯應是「電子畜生雜交大作戰（字亦沒打錯）」
    • 不過上一輯的播出年份是2007年，足足隔了三年東映才記起…
    • 因為前兩輯《Digimon Frontier》和《Digimon Savers》已經是東映黑歷史之一，這套弄得如何也沒有太多人理會。
      • Frontier問題是太特攝風且劇情處理不佳，Savers問題則是故事節奏失當外加太過強調大門大的主角威能（而且沒有解釋原因）。
    • 不過當公佈三條陸會是這次的主編劇後，大家的希望又來了
    • 但撥畢後，失望大於絕望
      • 因此本輯主題是「合體」也是理所當然的事
      • 由於本作中獵奇的合體特別多，三條於是被戲稱是被W開啟了某個開關……
      • 不知道是三條習慣這樣操女主角還是如何，女主角陽本明幾乎是每集登場就要顏藝一次，堪稱數碼系列顏藝之后
    • 本作一開始公布的主角獸ShoutmonX4外形嚇到很多人，因為根本是機器人，而動畫播出後OP也確實像機器人動畫的OP，就連聲優也用常配機器人動畫的那一票。
    • 讓人擔憂的是連這個世界也被「地獄兄弟中的弟」米村正二入侵了，おのれ！雖然大家都期待米村在不是騎士棚的地方不會發揮崩壞功力，但最後證明了就算不是騎士棚照樣崩的亂七八糟，偏偏中期有一大半劇本是他寫的……
  • 第一輯的故事分為兩部份，官方定義是兩輯
    • 第一篇為第一集至第三十集，在星期二1930播出，但一般而言30集大約只需六個月左右的時間卻因為停播太嚴重，花了八個月以上才播映完畢主要都是因為朝日電視台要轉播體育賽事......
    • 由於停播過份嚴重，第二篇《デジモンクロスウォーズ～悪のデスジェネラルと七つの王国～》改由星期日0630播出，差一點就能成為「東映御三家」的第四人
      • 整部DXW的人氣也因為停播過分嚴重，一直帶不起來，甚至也被東映半放棄了。[2]
      • 雖然如此，平均收視都有5.5%，在0630這段時間也算不錯
    • 為了拉人氣，這一輯一開始就把前代所有主角數碼獸路過一下（誤？[3]
      • 如果根據Digimemory設定（見下列），也許前代的主角數碼獸全都可以用Memory 叫出來…
      • 除了用Memory外，現時各代的主角數碼獸都會以路人的方式登場，包括只限遊戲中出現的數碼獸
    • おのれ ディケイド!!!
  • 第三輯《デジモンクロスウォーズ～時を駆ける少年ハンターたち～》名目上是續集，但會有主角替換
    • 上一輯的劇本寫到這樣都可以有第三輯，很多人都覺得不可思議...
      • 第三輯是連主編劇三條陸都不知情的情況下決定的，所以第三輯的基本構成和第55集（第二輯第一集）的劇本都是由新導演貝澤幸男[4]負責
      • 可見第三輯跟本是高層的私下決定，暫只能以「玩具銷售」去解釋...
      • 真相是當第二輯做完後，電視台方沒有東西可以播放，所以趕製第三輯用作拖時間之用
      • 估計為縮短製作時間，所以由新人貝澤幸男當導演
      • 前半故事（75集前）異常鬆散，是為了當電視台方下決定後可以隨時完結
      • 後半故事（75集後）直入主線，因為電視台方決定由《聖鬥士星矢Ω》接檔，第三輯要盡快完結讓出檔期
    • 因為只是拖時間用，所以東映開始「另類宣傳」...
      • 由開播至第9集（ep.63），贊助商列表中就經常出現「映画スイートプリキュア♪」的名字，可以說是首次有動畫要其他動畫贊助。到了第6集（ep.60）更成為列表首位。到底是這套蝕錢比較多，還是隔壁棚賺錢比較多？
      • 第10集（ep.64）至第17集（ep.71），贊助商列表更出現了「東映」，看來不太多人想把錢放到這東西之上，直到第18集（ep.72）才有正常的贊助商，不用東映倒錢下去
      • 第7集（ep.61）中出現類似《假面騎士》的特攝宣傳橫額，第9集（ep.63）更出現《假面騎士Fourze》的宣傳海報
      • 現在只差戰隊系列還沒有進場，說不好幾集後會見到海賊戰隊和特命戰隊的宣傳...
    • 除了新的第一主角明石タギル外，預告也講明會有前五代的主角出場，おのれディケイド！
      • 已確定太一、大輔、啟人、拓也和大門大會在本輯中出現
      • 不同於騎士和戰隊，他們並不是以前輩的身份出現，而是以幫助者的身份出現
    • 依照慣例原班後宮主角又消失得不見蹤影，尤其是同校的劍道社學長與女同學
      • 而且還由另一同學麻美繼承苦命零戲份糟糠役
      • 目前音音和明里只出現過一集。劍道社活動更完全無視掉學長...
    • 隨著接檔動畫《聖鬥士星矢Ω》決定下來，全輯在26集下完結，並成為了平成時代《Digimon》系列的終點[5]
  • 因為《Digimon Savers》把慣例完全破壞，這輯的重點是重修慣例，例如六頭身，主角的護目鏡設定。但某程度上也破壞了原設定：
    • 如本輯有很多數碼獸都是等級不明，而Greymon、Starmon等更被重塑
      • 其實本輯中，等級已經不重要，成長期也可以打敗裝甲體
      • 數碼獸也已經不重要，因為人類已經可以痛毆世界樹
    • 在第30集前進化什麼的完全沒有出現過
    • 本輯也是數碼動畫史上第一部沒有ED曲的動畫（AiM與支持者淚目）
      • 應該說近年來東映的動畫除了一些國民番組等級（如光美）以外都不再有ED曲，放在ED的字幕則是都移至OP中所以很多時候光看OP的配音表就捏他了
      • 參與本輯音樂的歌手與製作人以新團隊為主，但插曲仍由和田光司演出，二期時宮崎步與谷本貴義亦再度參加，後來和田因身體出狀況停止活動，插曲改由サイキックラバー負責演出。
      • 第二集與第三集的片頭都是由V家名製作人kz負責、Twill演出
  • 本輯的數碼機名稱為「Digimon Xros Loader」（デジモンクロスローダー）
    • 又名電鬍刀，可以用來剃鬚[6]
    • 手上有「Digimon Xros Loader 」的人類，第一輯中被稱作General（指揮官），第二輯就稱作Hunter（獵人）
    • 此代的數碼機前期所控制的不是進化而是合體，到後期才加回進化機制
      • 工藤大器的紅色Xros Loader可以控制Shoutmon和其他數碼獸合體成Shoutmonx2、Shoutmonx3、和Shoutmonx4、亦可以令Shoutmon 進化成 OmegaShoutmon
      • 蒼沼切羽的藍色Xros Loader可以控制Greymon 和MailBirdramon 合體成2010年版的MetalGreymon，亦能令其進化成Zeekgreymon
      • 天野音音的黑色Xros Loader可以控制SkullKnightmon 和Deadly Axemon 合體成DarkKnightmon；在第24集，黑色Xros Loader變回玫瑰紫色，讓不成材的Monitormon們合體成Hi-VisionMonitamon
      • 除了可以控制和自己的拍檔合體外，也可以控制和其他Digimon合體。如在第四集Shoutmonx2 和烏龜獸合體，得到在海中戰鬥的能力
      • 第21集，Darknightmon以黑暗的力量令Darkness Loader甦醒。在第31集之後，Bagura軍的七個Death General得到複製的Darkness Loader而得到DigiXros的能力。真貨則成為天野悠的Xros Loader
      • 在第54集，因為天野悠的覺吾，讓Darkness Loader變成黃色的Xros Loader。而在第二期中悠可以令Damemon 進化成Tuwarmon
    • 插入稱為「Digi Memory」的記憶卡後，能召喚不同的Digimon作戰。插入Gaia Memory 還可令擁有者進化成傳說中的戰士「プリキュア」[7]
      • 原型是由Omegamon所帶領的聖騎士軍，用盡力量但無法阻止世界分裂而化身的冤念
      • 每張在每一個Zone只能使用一次，使用後會變成黑色，去到下一個Zone 時才能使用
      • 到後期，只要Digi-Memory用盡力量就會變成黑色，回復方法不明
      • 第二輯結局時，所有Digi-Memory變回Digimon。之後也沒有了這個設定。

    • 和大器、悠組成街頭藍球組合Xros Heart，不過總是因為想灌籃而成為扯後腿的一員

  • 工藤大器 （工藤タイキ） （CV：高山南）

    • 本輯的收視保證[13]
    • 組成了名為Xros Heart（LOGO沿用數碼世界冒險時的Shoutmon頭型紅色圖徽）的三對三籃球隊，成員有他自己、激和悠。

前代主角群與數碼獸編輯

  • 本作中登場的前代主角群是同一角色而非平行世界[15]，因此進化方式也與各代原設定相同，而非全部都使用Xros Loader。
  • 八神太一（CV：藤田淑子）
    • 第一輯《Digimon Adventure》的主角
    • 擁有女友被朋友NTR而不憤怒的勇氣
    • 以第一輯小五學生的姿態出現
    • 在第14集（ep.68）中以黑影出現，在第22集（ep.76）中正式和本輯主角碰面
  • 亞古獸（CV：坂本千夏[無印]、松野太紀[Savers]）[16]
    • 同時在第一輯《Digimon Adventure》與第五輯《Digimon Savers》擔當主角的數碼獸。
    • 無印版的亞古獸體型比較小，Savers版亞古獸不僅體型大，手上還裝備著像是拳套的物品。
  • 太刀川美美（CV：前田 愛）
    • 第一輯《Digimon Adventure》，八個被選中的孩子之一
    • 同樣以第一輯小五學生的姿態出現
    • 在第22集（ep.76）中正式出場，在香港與天野音音、蒼沼切羽碰面，未有台詞。
  • 巴魯獸（CV：???）

    • 第一輯《Digimon Adventure》，八個被選中的孩子之一

  • 哥瑪獸（CV：竹内順子）[18]
    • 丈的搭檔，第一輯《Digimon Adventure》中的八隻主角數碼獸之一。
    • 登場就是通知丈決戰地點在東京。
  • 甲蟲獸（CV：櫻井孝宏） [19]
    • 光子郎的拍擋，但光子郎本身卻沒有在第24集（ep.78）中出現
    • 解說役
  • 二手本宮大輔（CV：木内レイコ）
    • 第二輯《Digimon Adventure》的二手主角
    • 在第14集（ep.68）中以黑影出現，於第24集（ep.78）中正式登場。
  • V仔獸（CV：???）
    • 第二輯《Digimon Adventure》的主角獸
    • 第24集（ep.78）中正式登場
      • 令人有點意外的是初登場型態居然是金甲龍獸，而進化型Ex-V仔獸卻沒有出現
  • 一乘寺賢（CV：朴璐美）
    • 第二輯《Digimon Adventure》的真主角（基本上無誤？）
    • 在第24集（ep.78）中與大和一起登場並讓身邊的飛蟲獸與大輔的EX-V仔獸共同進化成帝皇龍甲獸（直接跳過完全體了）
  • 松田啟人（CV：津村まこと）
    • 第三輯《Digimon Tamers》的弱氣主角
    • Matrix進化用零件
    • 在第14集（ep.68）中以黑影出現，第24集（ep.78）正式登場，手上拿的是後期的D-Ark。
  • 基爾獸（CV：???）
    • 啟人的搭檔數碼獸，目前沒有台詞。
    • 第24集（ep.78）與啟人進行羞恥PlayMatrix Evolution進化成紅蓮騎士獸
  • 格拉尼（CV：菊池正美）[19]
    • 第三輯《Digimon Tamers》的外掛基本上無誤
    • 與紅蓮騎士獸合體後，能夠讓紅蓮騎士獸進化成真紅蓮模式。
  • 牧野留姬（CV：折笠富美子）
    • 第三輯《Digimon Tamers》的女主角，人稱爪弾くはたおやかな調べ！キュアリズム！「數碼獸女王」
    • 衣服上的心型圖案並不是後期的完型心型，而是剛出場的碎心型
      • 不過按照表現與第24集（ep.78）的剪影來看，留姬應該也是Tamers後期的狀況
    • 在第22集（ep.76）中正式出場，與美美在香港和天野音音、蒼沼切羽碰面
  • 神原拓也（CV：竹内順子）
    • 第四輯《Digimon Frontier》的主角
    • 所謂最終的Digimon，就是把自己化成Digimon...
    • 在第14集（ep.68）中以黑影出現，第24集（ep.78）正式登場，並進化成火龍獸
      • 火龍獸是拓也使用炎之人型&獸型Spirit進行雙重武裝進化（Double Spirit Evolution）後進化的模樣
      • 在眾人面的羞恥Play
      • 雖然照理講應該是究極體，但是Frontier本篇表現出來的等級也很難說是究極體就是了......
      • 招式對敵人效果不怎麼大（無誤），看來變身成Cure Rouge還比較強
    • 隨後接收來自冰熊獸等人的Spirit進化成金剛武神獸
  • 冰熊獸（CV：渡邊久美子）[20]
    • 第四輯《Digimon Frontier》中主角群之一
    • 冰見友樹使用冰之人型Spirit進化後的模樣
    • 本作中直接以進化後的模樣登場，並與其他同伴一同與火龍獸合體進化成金剛武神獸。
  • 金剛武神獸（CV：???）[21]
    • 拓也與同伴共同使用所有Spirit進行究極武裝進化（Ancient Spirit Evolution）後進化的數碼獸，也是整個《Digimon Frontier》中主角方進化的最強數碼獸。
    • 不准吐槽那很像拼裝貨感覺的設計
    • 必殺技是八雷神與天羽斬。
  • 大門大獸（CV：保志總一朗）
    • 第五輯《Digimon Savers/Digimon Data Squad》的主角
    • 在第14集（ep.68）中以黑影出現，在第22集（ep.76）中和太一一起正式和本輯主角碰面
    • 能夠以肉身毆打究極體的強者，第24集（ep.78）更直接一拳送四隻究極體上路這傢伙到底是何方神聖啊
