清版動作遊戲

清版動作遊戲 / Beat 'em up / Belt Action
本條目內容不足需要擴充，歡迎有興趣的朋友加以補完。

目錄

解說編輯

  • 清版動作遊戲為動作遊戲的衍生類型。
  • 相較於講究地形障礙的橫軸動作遊戲，清版動作遊戲更強調與敵方角色的戰鬥。
    • 地形也會視情況分為多層次橫軸（玩家可藉指令在不同橫軸間移動）或是可以在平面上下移動（XY軸）的地形。
  • 中期隨角色設定深化後，加入輸入簡易指令可打出不同必殺技的性能，可說是格鬥遊戲的雛型。
  • 隨著硬體的進步，清版動作遊戲又可衍生成割草砍殺遊戲。
  • 與STG及FTG同為90年代街機遊戲的代表。
    • 由於玩家多為學生，對貢獻機鋪與磨練遊戲技能的行為將投幣續命戲稱為「繳學費」，沿用至後來的娛樂遊戲文化。
    • 其實還有結合STG與清版動作的遊戲類型： Shoot 'em up，遊戲過程與清版相同，但是角色以射擊攻擊為主

常見要素編輯

跳躍編輯

  • 最基本的指令之一，大部分清版動作遊戲除非是遊戲需要，否則一定會有
  • 甚至有部分清版動作遊戲會有空中必殺技讓跳躍變得相當活用
  • 不過跳躍時還是有機會會被敵人攻擊，所以需拿捏好
  • 也有部分清版動作遊戲有後跳（↓+跳躍鍵）

摔技/投擲技編輯

  • 如字面意義，將敵人投擲出去的攻擊手段，但與格鬥遊戲的設計有許多不同：
    • 通常只需貼近沒有攻擊或移動判定的敵人即可捉住敵人，反之相同，敵人角色貼近我方角色時也可能觸發投擲/扣鎖動作判定
    • 通常分為不按方向只按攻擊的捉住後短打幾下、方向+攻擊的向前或向後摔出，或是更複雜的方向+跳躍翻身換位等變化，不一而足
      • 後來不少遊戲為區分玩家可用角色的不同性能，會出現重視摔技的角色，這類角色通常會設計成攻、防較強、速度慢不過卻有多種特殊摔技，當玩家成功用出特殊摔技後會有充分的爽快感。
    • 另外做為多人動作遊戲的常見設計是投擲出的敵人也有攻擊判定，因此常見以甩出敵人撞翻多個敵人以清場的打法

軸線切換式移動編輯

  • 部分清版動作遊戲只要輸入特定指令即可作出軸切換移動
    • 例子：Treasure的《守護者列傳》（見下）

八方向式移動編輯

  • 有部分清版動作遊戲都會導入此類移動方式，讓角色打起來有方向感
    • 例子：Konami的《忍者龜》清版動作遊戲的部分代數
  • 甚至採用上帝視角的《ガイアポリス》以及後來的3D清版動作遊戲都是採用八方向移動

無敵技/清場技編輯

  • 當敵方單位大量出場時我方角色容易陷入包圍，此時不管做任何動作都容易被來自四面八方的攻擊打斷，不斷循環之下很容易就浪費一條生命。
  • 所以多數遊戲都會設計能夠短時間無敵+大範圍可擊退敵人的攻擊技，用來讓我方角色擺脫包圍。
  • 但為了避免玩家濫用這種技能強行闖關，因此這種技能多半都有使用限制，最常見的就是消耗體力計量，太常使用反會讓角色體力見底。

指令必殺技編輯

  • 部分清版動作遊戲都會導入指令必殺技，使角色攻擊或連招時變得相當靈活
    • 但為了保持遊戲的所有平衡，指令必殺技基本上都只會有1~3個，但也有例外

多人必殺技編輯

  • 前兩者的衍生，部分清版動作遊戲有實裝，顧名思義只要有2位以上的玩家同時輸入特定指令或達成條件即可同時發動多人必殺技

武器編輯

  • 如同其名，每個關卡都會少不了武器，可以直接在途中撿取或是打掉一定物件即可，角色撿起武器時就可使用武器攻擊，但每個武器都有一定的使用次數，打到一定程度就會直接消失，部分遊戲也可將武器朝敵方砸過去
    • 甚至部分遊戲也有所謂「武器必殺技」，只要撿到武器就可使出該武器對應的必殺技

補品編輯

  • 如同其名，每個關卡都會有補品，打掉一定物件都會掉落然後請不要問為何油桶、垃圾桶之類的會掉出一些食物然後讓角色去吃...
  • 初期眾多遊戲的補品是玩家一接觸就生效，但不少玩家在滿體力時為避免浪費補品，常會避開道具進而影響了走位，所以後來改成走到補品處撿拾後才會生效。

短期衝刺關卡編輯

  • 基本上該關卡都會出現在獎勵關卡或是高速關卡，如同其名，關卡相當短

探索性編輯

  • 並不一定是所有清版動作遊戲都是採用一本道關卡設計，有些會導入分歧路線，路線不同，進入的關卡也會不同，甚至會影響結局

世界觀編輯

  • 呼應80~00年代的動作暴力電影與科幻盛世，並為了吸引身處於現代的玩家目光，常有現代背景與暴力橫行的城市世界觀，玩家扮演的角色從街頭攻擊不法集團份子的設定
    • 另外呼應70年代起的東方功夫電影與忍者電影風潮以及多數屬於日本文化圈的開發商口味偏好，常有忍者或武士的元素
    • 但也有像是SEGA的《獸王記》、《戰斧》、CAPCOM的《龍與地下城》系列和《圓桌武士》等都是走中世紀歐洲的劍與魔法風格或以神話為背景的
