GAME OVER

GAME OVER/ゲームオーバー/  GAME GAME[1]
本條目歡迎任何尚未Game Over的人仕補完。

目錄

解說編輯

  • 「GAME OVER」，顧名思義就是指遊戲結束。主要用在電玩遊戲之上。
    • 其實文法有誤，這句子應該補回動詞變成「Game is over.」才是正確，不過因為遊戲業界已經錯了很久的關係，故此大家都習慣了。
    • 不過也有遊戲會用正寫的。
    • 其實就是玩家被發便當（無誤）。
  • 狹義是指玩家正在遊玩的那一局遊戲因失敗性因素而強制中斷時的情況（FTG、ACT或STG裡耗盡所有生命數、RPG或SLG裡主角隊伍全滅、AVG無法達成任何結局或BAD END等），畫面會出現「GAME OVER」的文字（或類似信息）。廣義上則是指遊戲以任意方式結束，包括破關和以保存遊戲進度的方式中斷遊戲。
    • 尤其是破關這一條，因為要節省記憶體容量，故在破關和失敗的場合都會使用同一個GAME OVER畫面。大型機台遊戲中尤其如此。
  • 不一定玩家戰敗才會出現，有時在遊戲過程中選擇了錯誤的選項也有可能GAME OVER。
    • 這種情況在一些遊戲中甚至不會視為一個單獨的結局，也就是比BAD END還不如。有週回設定的遊戲裡，玩家可能無法在遊戲結束後存檔，也就無法繼承該存檔的點數。例子之一就是《幻想水滸傳5》。
    • 也有視為一個結局的，想要收集全部要素的玩家必須要經歷一遭才能填滿收集相冊。譬如《明星志願系列》的破產結局。
  • 在大型機台遊戲裡，操作角色死亡或是扣打耗盡時會強制中斷遊戲，這是則是出現「CONTINUE?（接關）」的字樣搭配秒數倒數等待玩家投幣，如果玩家投幣後，遊戲就會從被中斷的地方復活操作角色或是回覆扣打繼續遊戲（大部分的遊戲會等玩家10秒，少數則是15或20秒或以上不等），但如果不投幣，直接讓秒數倒數歸零時，遊戲就直接顯示「GAME OVER」。
  • 順帶一提，早期的部分大型機台遊戲在GAME OVER後等待玩家繼續投幣的畫面相當驚悚。但在中期後有不少遊戲在「CONTINUE?」甚至是「GAME OVER」畫面加上不少花樣別出心裁，讓不少玩家噴更多錢繼續投幣。（例如：《快打旋風2》在玩家被打敗後會出現操作角色被打得鼻青臉腫的接關畫面，此為其中一個相當經典的例子。）
  • 失敗重來是遊戲乃至於人生常態，因此GAME OVER很少是真正的GAME OVER，頂多接關讀檔除了摔手把砸鍵盤不玩以外；但也有遊戲引入永久死亡（Permadeath）機制，角色死亡會自動刪除所有進度，就像真實人生沒有重來因此請珍惜生命。
    • 永久死亡可以是傳統的高難度模式機制，讓玩家必須謹慎攻略，但也有如類盜賊（Roguelike）遊戲單次流程較短，需要玩家在多次攻略之間累積經驗進而打出更好成績的玩法。

關於遊戲的「結局」種類說明編輯

  • 不同於影視小說等載體，遊戲存在互動性，所以往往會視玩家的行為引領玩家走向不同的結局。
  • 傳統的GAME OVER可以視作「遊戲中斷」，而各種END則是玩家正常走完一輪流程的結果。
    • 雖然也有不少AVG在開局幾分鐘即能完成結局。典型例子就是《河源崎家一族》選擇拒絕接受工作從而逃過一劫。但這比起GAME OVER來講確實更算一般意義上的結局。
  • 一款多結局遊戲中，結局的好壞往往是遊戲本身定義分類的；也有不下定義讓玩家自行感受的例子。
  • 壞的結局往往比好結局打起來容易，但也有例外，譬如《嗜血龍騎士》。

Bad End編輯

  • 通稱BE。一般指主人公沒有能順利解決遊戲中的問題，或是選錯關鍵選項，從而迎來內容沈重的結局。一般是用來和好結局形成對比的。
  • 部分遊戲設置了回收一定程度的BE即可開啟獎勵的機能。
    • 其實隨著成就模式的普及，很多收集愛好者為了成就也會去刻意回收BE。
  • 一些遊戲的BE做的反而比GE更深入人心。
  • 部分遊戲雖然是BE，但在玩家的理解中卻有正面的解釋。這種情況會被稱為Merry Bad End。
  • 個別系列作會延續前作的BE展開劇情。

Dead End編輯

  • 算是BE的一種，明確表明了主人公在該結局中迎來死亡的悲劇。
  • 是部分遊戲的特定稱呼，在打出此類結局後出現「Dead End」的字幕。

Normal End編輯

  • 通稱NE。沒有好事也沒有壞事的普通結局。
  • 許多戀愛遊戲中未能攻略成功任何一人，即會迎來的結局。

Happy End編輯

  • 通稱HE或GE（Good End）。所有事情圓滿解決後迎來的可喜可賀的結局。
    • 有的遊戲還會設置最好的結局（Best End）。
  • 一般來講是最難打的結局。

True End編輯

  • 即真實結局、官方結局。與故事主線最貼近的結局。
  • 首次出現這個概念是在Leaf於1996年發售的AVG遊戲《雫》中，女主人公月島琉璃子的結局。
  • 但TE不一定是最好的結局，上面就是美好與真實相斥的例子。
  • 會引發玩家討論的一個概念。因為玩家們往往會就「哪個結局才是官方正史」這一問題各抒己見。
  • 當然，像是《薩爾達傳說時之笛》這類，不同結局接續系列的不同作品，展開各種平行世界式故事觀也是常見的結局類型。

遊戲 解說
勇者鬥惡龍系列（大多數情況下） 如果主角隊伍全滅，畫面一黑然後出現對話「…再給你們一次機會吧」，然後回到教堂，身上的金錢少一半。
如果玩家事先把錢存銀行裡就連這筆錢都能省了。
POKEMON ［主角的名字］一邊掩護已精疲力盡、動彈不得的瀕死的寶可夢們，
一邊急急忙忙地回到寶可夢中心。
和上者相似，回到中心，身上的金錢少一半。
瓦利歐樂園（2、3代） 遊戲內敵人的攻擊只會造成壞利歐損失金錢或者狀態變化；頭目戰中頭目的攻擊也只是想方設法把壞利歐趕出頭目戰的房間而已。
但是3代的最終決戰是例外，如果被謎之神像的手抓住的話會導致GAME OVER。
阿貓阿狗 包括最終BOSS戰在內，任何戰鬥失敗了樂樂一行人都只會逃跑，而且原地呆著什麼都不做就能恢復。
Second Novel 〜她的夏天、15分鐘的記憶〜 表面上會有GAME OVER畫面，實際上全都是遊戲的一環。玩家要從「故事中的故事」回到「故事中的故事外」進行操作才能把故事進行下去。
街～命運的交叉點/428 同樣，GAME OVER是正常進行遊戲的一環。很多情況下角色A必須GAME OVER，角色B才有路可走，然後把角色A的GAME OVER的FLAG拆掉。當然也有反過來的情況…
俠盜獵車手 You Smoked!
除了任務副本必須重來之外基本上沒有真正的GAME OVER，任務外角色死亡的話會完好如初的從醫院復活並失去一定的持有金謝啦!!歐巴馬健保。
另外玩家被通緝時則可能會被逮捕，逮捕後會失去身上所有子彈跟一定的保釋金。

帽子世界 因為主角身為管理人的權限可以自由穿越各個世界，在多數地圖敗陣時只會強制回城。
不過在終盤突入最終迷宮後該能力便會失效，敗陣時照樣會GAME OVER。
新約追加的洋子路線由於沒有帽子，無論在何時何地敗北都會直接GAME OVER，就算隊友有戴帽子的管理人也是如此，除非到了終盤獲得全世界的帽子之力，不過同樣在最終迷宮無效。
World of Final Fantasy 大多數情況下戰敗時會因為小玉的時間回溯力量而回城，只有特定情況下戰敗才會GAME OVER。
Romancing SaGa 2 在我方全滅或皇帝死亡後會發動傳承密法將皇位傳承給下一個繼承人，直到傳到最終皇帝後因為傳承法失效而無法傳位，此時全滅才會GAME OVER。
