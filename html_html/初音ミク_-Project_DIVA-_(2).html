解說編輯

  • SEGA在取得CRYPTON社和V家創作者的授權後、以大受歡迎的VOCALOID歌曲為基製作的音樂遊戲。除了使用了網上的歌曲外，還有藉由募集活動和找知名音樂人製作新曲。
    • MV大多都是由SEGA自己製作雖然實際遊玩時根本沒人有閒看
  • 遊戲內容是音樂節拍遊戲，跟一般遊戲一樣要在適當的時候按下遊戲要求的按鈕就可以得分。
  • 但失手的話除了會扣體力，初音還會停止唱歌作為警示，生命值歸零則會中斷遊戲。
  • 不同一般音樂遊戲的是，遊戲時初音等角色會在後面的背景跳舞。
  • PV的舞步是對小倉唯的動作進行動態捕捉製成所以相當自然，但後來更新的頻率顯著變少，近年甚至常有PV兼用原曲的情況發生。
  • 系列作遍及PSP、PS3、PS4、PSV、NS等平台，亦有街機版本

初音ミク -Project DIVA-編輯

 
  • 於PSP平台發售的本系列作第一彈，也是VOCALOID首款正式的試玩版官方遊戲，簡稱PjD或PjD 1st
  • 2009年7月2日發售
  • 官方售價6090日圓(含稅)/特典復刻版5980日圓(含稅)
  • 廉價版2010年6月24日發售，官方售價2640日圓
  • 數位下載版2011年12月15日在PSN上發售，售價3000日圓(含稅)
  • 前後三個封面有微妙的不同......(SEGA果然懂騙錢)
  • OP是由神前暁和畑亜貴制作的「The secret garden」
  • 由於是全年齡向，所以PV的舞蹈也很全年齡。(認真)
  • 但還是被島民抓到決定性的瞬間。
 
  • 而且某紳士已經用這瞬間，把所有衣服的小褲褲都有拍下來。(無誤)
  • 在日本NicoNico上也有相關的   「研究」
  • 遊戲看起來上來很簡單，但是因為這遊戲的節奏感跟曲有些微的不對，所以聽節奏按是沒用的[1]，再加上畫面並不是定格背景，所以會容易讓人眼花，但事實是大部份紳士每當看到舞蹈的精彩畫面，手都會自動停下來。
  • STAFF考慮到喜歡V家的人們都喜歡創作，所以設有自行編排舞蹈的系統(即編輯模式エディットモード)。
  • 但明顯輸給MMD，相比起偏日常故事和各類惡搞的MMD，這編輯模式好像真的發揮了舞蹈的功能。
  • 雖然編輯模式只能以內定的動作去編排舞蹈，但因為可供選擇的動作不少，肯花心思一樣能做出好看的舞蹈
  • 只要為舞蹈片段再加上歌曲和按鍵設定就能成為可遊玩的新曲和PV了
  • 本遊戲使用金手指可開啟隱藏表情

初音ミク -Project DIVA- 2nd編輯

 
  • 於PSP平台上的第2作，首度支援陰影機PSPGo，因為比前作追加很多內容被戲稱是完整版遊戲。
  • 也是系列首次推出限定版同捆機，售價26980日圓(含稅)，但只是綠色的主機再配上矽膠套、機袋等配件......
  • 2010年7月29日發售, 官方售價6090日圓(含稅)
  • 2010年8月10日開始在PSN上提供免費的體驗版，可以玩4首歌[4]外，還提供8件服裝[5]
  • 數位下載版2010年8月31日發售，售價5400日圓(含稅)
  • 廉價版(お買い得版)於2011年12月15日發售，UMD版3300日圓(含稅)、下載版3000日圓(含稅)、アーケードスターターパック(Arcade Starter Pack)6,300日圓圈含稅)
    • お買い得版會以同時下載「extendモジュールパック(extend module pack)」方式追加Extend的新服裝(UMD版和正價下載版能夠免費下載？)
  • 主題曲是由supercell的ryo制作的「こっち向いてBaby」, 另外一首是由ちーむMOER[6]制作的「カラフル×メロディ」
  • 刪除按鍵失敗會停止唱歌的系統(所以OP畫面的歌跟第一集不同，會有人唱)，也支援Media Install和Auto Save，此外PV中的舞蹈開始能有兩位角色合演。
  • Edit Mode(エディットモード)也大大的加強，除了可同時編輯兩名角色，動作數量也大大增加，介面也改良過。
    • 最大的差別是加入歌詞字幕和角色口型功能不像1st那樣看PV時沒字幕嘴巴又不會動那麼可怕。
  • 而且能走動的範圍相當大，能做動作也比前作多。[7][8]
  • ↑↓←→按鈕開放加入戰團，不過基本上↑↓←→按鈕只會在同時按時才用到(當然能夠靈活運用的話也可以補充○ X □ △按鍵來增加連打速度或應付按鍵變化)
  • 新增「EXTREME」難度給喜歡自虐自我挑戰的強人去征服[9][10]
  • 會告訴玩家那某一個評價需要多少個Cool+Fine(而不像上一代以分數計評價)[11]
  • Chance Time也由鼓勵Combo變成鼓勵按多點Cool[12]
  • 完成歌曲後除了分數外還可以穫得一定點數，點數多少取決於難易度和最終得到的評價，能購買各種道具(Help Item)幫助遊玩
  • 提供了更多歌曲和服裝以外的元素消耗你的時間和點數，如Diva Room(DIVAルーム)就大大的強化了
    • 此作開始為每個角色準備專屬房間，也有大量的道具可供裝飾(所以DLC中也有大量房間道具)
  • 還有一個不知有何用的機嫌系統，可透過送禮增加與角色的好感度，角色更有機會把你送的禮物裝飾在自己的房間裡
  • 同時收錄部份一代歌曲，總共收錄50首以上，但舊歌卻用回舊PV(Ievan Polkka是教學用的不算)
  • 初音的乳量比前作變小了
