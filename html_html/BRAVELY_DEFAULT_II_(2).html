解說編輯

  • SQUARE ENIX於任天堂Switch開發的角色扮演遊戲，為同名系列的第三部作品
    • 製作總監同樣是淺野智也，人設從吉田明彥變更為生島直樹，配樂則沿用《BRAVELY DEFAULT》的Revo而不是《BRAVELY SECOND》的ryo
    • 同時職業+晨星持有人的設計也從外包插畫 / 漫畫家更改為由生島直樹一人包辦過勞死
    • 說來當初《SECOND》會用ryo的原因就是Revo時程排不上趕鴨子上架，雖然之後ryo的曲風似乎也被Revo影響了
  • 基本上沿用了從初代就存在的奮勇 / 堅守模式，以及脫胎自FF系列的轉職系統
    • 和前作不同，今作從回合制改成一開始是不可見的ATB制，後面變成可見的ATB制
    • 此外本作的必殺技由武器依存變更成職業依存
    • 同時本作也取消了無職狀態的能力差異，改由餡饅頭類道具提供差異
    • 同樣可以取得晨星持有者所擁有的武具，，其性能比起前作更強，除了數值外，可以直接獲得該武器對應職業所有的被動技能
  • 地圖探索從前作的地雷改成明雷制，藉由在地圖偷襲敵方怪物可以先發制人，反之則會被偷襲
    • 地圖功能則被大幅減弱，僅大地圖能查閱地圖，進入迷宮後則不可
    • 歷代慣例的飛空艇更成為敵方專屬，遠距移動的手段只剩下各城鎮之間的馬車而已
  • 本作主要舞台為亞克希蘭特大陸（エクシラント大陸），世界觀基本上跟前兩作《BRAVELY DEFAULT》與《BRAVELY SECOND》無關
    • 不過從初代即存在的水晶與晨星，以及從FF系列就存在的光之戰士稱號等事物皆健在
  • 與前兩作將大量晨星任務放在支線不同，本作的晨星事件除少部分外絕大部分是主線必定發生的任務，並會在事件結束後給部分晨星使用者安排支線任務作為後日談
  • 雖然遊戲本身算是佳作，不過在中文地區卻因為詭異且奇妙的翻譯而造成話題

    • 名字捏他：圓桌騎士加拉哈德（Galahad）
  • 芙麗（フォリィ・モリス）【CV：花守由美里】
    • 神秘少女畫家
    • 第二章的黑幕，擁有「畫師」的晨星
    • 謀殺羅迪夫妻愛女亞夢娜的真正兇手，以完成至高藝術品為由偷走土之水晶，並在恣意屠殺威茲窩魯特的人民後抽出他們的血液當作繪畫顏料
      • 因為其手段殘酷加上玩弄人心且犯後絲毫沒有反省態度，被部分玩家認為是比一代召喚士梅菲莉亞甚至是藥師卡達還要更加惡質的存在
    • 戰敗後隨著潛藏洞窟的崩塌而死亡
    • 名字捏他：愚蠢（Folly）

偏遠之地「恩德諾村」[7]編輯

  • 戈蘭（グラン・ブース）【CV：堀総士郎】
    • 恩德諾村的村長，擁有「藥師」的晨星
    • 雖然是本作少數對主角群完全沒有敵意的晨星使用者，不過在弟弟的夢中還是以魔王形象與主角群交戰
  • 戈麟（グリン・ブース）
    • 戈蘭的弟弟
    • 為了能與亡妻再度見面而向哥哥索取夢見藥，卻在服藥後陷入長期昏睡

冰天雪地之國「拉姆達路」編輯

  • 多莫富伊（ドモヴォイ）【CV：中博史】
    • 拉姆達路正教的最高祭司，擁有「預言者」的晨星
    • 為了成為被水晶選中的光之戰士而搶奪火之水晶
    • 然而火之水晶最後仍選上身為妖精的亞黛拉因而暴怒攻擊主角群，在戰敗後被賀里鄂襲擊而死
    • 名字捏他：斯拉夫傳說的妖精（Domovoy）
  • 賀里鄂（ヘリオ）【CV：落合福嗣】
    • 拉姆達路正教的裁判官，擁有「導師」的晨星
    • 第三章的黑幕，來自荷祿格勒的間諜
    • 慫恿手下的修士進行妖精狩獵，導致無辜群眾冤死的罪魁禍首
    • 趁主角群擊敗多莫富伊後將其殺害並盜走火之水晶，並打開城門引荷祿格勒軍入侵
    • 在戰爭期間嘲諷重傷瀕死的格拉底斯卻被對方搶走火之水晶，最後被哥鐸里祿活活燒死
    • 名字捏他：希臘神話的太陽神海利歐斯（Helios）
  • 古拉底斯（グラディス・ケリー）【CV：齋賀光希】
    • 拉姆達路正教的修道士，擁有「劍聖」的晨星
    • 被賀里鄂欺騙而作為妖精獵人濫殺無辜，最後在戰爭期間被找上門的仇家刺殺，隨後將賀里鄂盜走的火之水晶搶回並交還給主角群後嚥下最後一口氣⋯
    • 死後被安葬在國內的墓園，後續支線劇情還可得知其為加拉哈杜的親妹妹
  • 蘭德勒夫（ランドルフ）
    • 拉姆達路正教的祭司
    • 原本對審判一事深信不疑，直到自己女兒也因為審判被處死後才發現教會濫殺無辜的真相
  • 瑪麗華（マルファ・ランサー）【CV：遠藤綾】
    • 世代傳承的龍洞守護者，亞黛拉的好友，擁有「龍騎士」的晨星
    • 性格溫和的御姐，本作少數對主角群完全沒有敵意的晨星使用者
      • 交手的理由純粹是想跟強者較量
    • 三章後期一度被教會捉去審判，遭處刑之際被展現妖精身分的亞黛拉救出
    • 名字捏他：伯達尼的馬大（Matha）
  • 哥亞梅祿（ゴアメル）【CV：東地宏樹】
    • 與亞黛拉相識的幼龍
    • 終盤在汲取已故哥鐸里祿的記憶與經驗後迅速成長為成龍
  • 哥鐸里祿（ゴードリィル）
    • 拉姆達路的守護神龍，委託主角前去阻止拉姆達路的審判
    • 在拉姆達路與荷祿格勒的戰爭期間抱病出面支援並將賀里鄂燒死，然而最終仍不敵亞達馬斯而被斬殺

軍事霸權帝國「荷祿格勒」編輯

  • 亞達馬斯（アダマス・ホログラード）【CV：小山力也】
    • 荷祿格勒皇帝，擁有「魔劍士」的晨星
    • 在序章打倒奧爾坦後會將其手刃後和主角群對決，基本上是必敗戰
      • 在真結局後的第二輪如果提早在初回戰打倒他的話會解鎖什麼問題都沒解決的特殊END
    • 戰敗後遭到顎丹娜背叛，回絕主角的援助後與墜落的飛空艇共存亡
    • 名字捏他：精金（Adamantine）
  • 朗茲迪魯（マドック・ロンズデイル）【CV：牛山茂】
    • 帝國所屬將軍，擁有「處刑者」的晨星
    • 重視忠義的武者，曾與施隆待過同一個傭兵團
    • 第四章率軍佔領哈爾索尼亞，被主角擊敗後投降
    • 後續被哈爾索尼亞放出來後可以在飛空艇殘骸處看到他祭弔亞達馬斯
  • 摩蕾（モーラ）【CV：青木瑠璃子】
    • 帝國所屬殺手，擁有「幻影」的晨星
    • 第四章潛入莎法隆暗殺主戰派議員
    • 本身是莎法隆人，真名梅雅莉，因為父親西盧拔被迫害而死而企圖對國家復仇
    • 先後偽裝成斯諾評議員與保羅王子暗殺數名評議員，最後在暗殺保羅王子失敗後自殺身亡
  • 比捷奴（ヴィジヌ・アイザック）【CV：西村太佑】
    • 帝國所屬魔法師，擁有「魔人」的晨星
    • 第四章襲擊威茲窩魯特，企圖將全國國民改造成不死族
    • 過去與艾維斯的師傅艾瑪有嫌隙，為了證明自己實力在對方之上而將自己改造成不死族之王[8]
    • 然而艾瑪早已研究出殺死不死族的方法乃是用注入魔力的水曲柳木樁給予其致命一擊，因此在戰敗後被艾維斯用該方法消滅
    • 血量偏少，但因為其不死族特性會復活3次，用赤魔+導師爆發幾波很容易就死了
    • 本作的翻譯受害者之一，明明原文講話很順暢，頂多語尾會用假名，在翻譯中卻活像個口吃的社交障礙
    • 名字捏他：印度教分支毘濕奴派（Vaishnavism）
  • 顎丹娜（エドゥナ）【CV：能登麻美子】
    • 帝國所屬預言家
    • 本作真正的幕後黑手，亞黛拉的姐姐，同時也是BRAVELY系列名物之一的騙人妖精
    • 為了復活黑暗之瞳而協助帝國收集水晶，在亞達馬斯戰敗後背叛他
    • 在5章結束會被黑暗之瞳吞噬，生死未卜
    • 而在進入6章則是會變成石頭沒下文

光輝不再之國「苗瑟」編輯

  • 施隆（スローン）【CV：田中秀幸】
    • 亡國苗瑟的老騎士，前代光之戰士
    • 本作最強助拳人，一刀一個敵人，復活，回復一人包辦，讓諸多玩家在序章就能愉快的練滿凡人
    • 序章最後為了保護眾人而與亞達馬斯交戰並壯烈犧牲，最後被安葬在風之溪谷與其亡妻的墳墓並列
    • 隱藏劇情揭曉其擁有「勇者」晨星的其中一半
    • 中文版的台詞幾乎都是文言文，為翻譯的最大受害者之一，台詞一多會看不懂在說什麼
  • 葛溫達樂（グウェンダル）
    • 葛蘿莉雅的祖父，前代光之戰士

其他編輯

  • 老婆婆【CV：大原沙耶香】
    • 序章被主角拯救的老婆婆，會給予連線尋寶用的船隻
    • 真實身分是妖精王艾琳（アイリーン），過去與葛溫達樂、施隆與艾瑪一同對抗黑暗之瞳的前代光之戰士
  • 福索（フォッソ）
    • 為尋找救命恩人而獨自旅行的魔物，造型是1代的半獸人
    • 在各區域作為傳授地圖技能的NPC登場
  • 黑暗之瞳（ヤミノヒトミ）
    • 本作最終頭目，過去被光之戰士封印的邪惡生命體
    • 真實身分是古代苗瑟王族殷安納，因為貪圖記憶之泉的睿智與記憶而化為怪物
      • 在設定集中提到拿著魔導書的葛洛莉雅恐怕並不是葛洛莉雅，所以在玩家間有著殷安納可能和葛洛莉雅有著同一張臉和造型的說法
    • 司書晨星（不會獲得）的原所持人，藉由將司書晨星化為魔導書保存記憶使自己成為不朽的存在
    • 最後因為玩家使用「手動存檔」的方式覆蓋了她的紀錄導致其記憶被破壞，因而失去了不死之身並導致其本體顯露

世界編輯

  • 四季如春之國「哈爾索尼亞」（春風の国ハルシオニア）
    • 400年前從苗瑟獨立出來的國家，擁有豐富自然資源
  • 沙漠綠洲之國「莎法隆」（渇水の国サヴァロン）
    • 200年前因為採伐與戰爭而陷入沙漠化的國家，現在受水之水晶的影響而四處淹水
  • 魔法學院之國「威茲窩魯特」（魔法の国 ウィズワルド）
    • 50年前由魔法使艾瑪建立的國家，現在受土之水晶的影響植物叢生
  • 冰天雪地之國「拉姆達路」（深雪の国ライムダール）
    • 1000年前被神龍拯救的國家，現在受火之水晶的影響而有暖化問題
  • 軍事霸權帝國「荷祿格勒」（軍国ホログラード）
    • 300年前由英雄在荒地建立的國家，現在在新皇帝亞達馬斯的領導下開始對各國的侵略
  • 光輝不再之國「苗瑟」（宿命の国ミューザ）
    • 位於亞克希蘭特大陸北方的古老大國，因為荷祿格勒的侵略而滅亡
  • 妖精之鄉「木玫里」（妖精の郷マグ・メル）
    • 妖精居住的城鎮
