SSSS.GRIDMAN

SSSS.GRIDMAN
本條目内含有劇透，沒看過原著又不想被アレクシス・ケリヴ變成怪獸的話請不要看下去。

目錄

解說編輯

  • 由円谷Production、TRIGGER聯合製作，改編自1993年特攝作《電光超人古利特》的原創動畫，2017年12月宣布製作，於2018年10月7日起首播
  • 由於製作公司完全是自家人以及製作過《天元突破》《DARLING in the FRANXX》等機器人動畫的緣故，動作戲部分充斥著特攝版力霸王打鬥的影子、也有諸多超級系機器人動畫的經典分鏡如大張一刀流、機體變形/合體等
    • 劇組表示演出分鏡是「給大張的情書」，然而不知道是否致敬得太過份讓大張開始偷酸起來，甚至在微博上說自己的分鏡「被抄襲」然後讓高松在推特上不點名酸他
      • 而播映結束後的玩具包裝設計、以及外傳漫畫的機體設計則真的由大張正己擔任他們給的實在太多了
    • 製作陣容上由參與過超人力霸王系列的長谷川圭一擔任腳本，雨宮哲擔任監督
  • 但比起打鬥部分更多人關注的是劇中兩位女角「宝多六花」「新条アカネ」幾乎搶盡話題，相關的同人創作也是以她倆為中心如雨後春筍般湧出，原本是要來懷念經典的大友們都被鐵壁粗大腿和巨乳柯博文給吸過去了
    • 大概也因此引得円谷發聲明說明有關同人二次創作的界線，隨後預售商品中就推出了上面兩人的抱枕套
  • 世界觀設定上多方表明有與原作《電光超人古利特》為直接續作的關係，原作中的人物或關聯角色亦有直接登場
  • 標題的SSSS取自的美版原片名[1]（唸時略過），到最終話則揭露本作的SSSS是指「Special Signature to Save a Soul（為拯救靈魂而存在的特別署名）」
  • 本作於2019年第85回日本SF大會上獲得媒體部門星雲賞的獎項
  • 本作在機戰30周年紀念作超級機器人大戰30中被選為參戰作，也是首個圓谷製作的作品參加機戰的首例。
  • 本作主要角色的服裝其實都有捏他變形金剛系列中的Shattered Glass系列的角色，因此還吸引了不少沒接觸過變形金剛的人深入研究後入了變形金剛坑
    • 不過Shattered Glass是正邪反轉的世界，而且大多數人物配色和造型是這系列限定，因此沒接觸過這個系列的TF粉有的還會沒注意到這些捏他

登場人物編輯

グリッド同盟編輯

嚮裕太【CV：広瀬裕也】編輯

 

  • 本作主人公，杜鵑台高中1年E班學生。故事一開始在六花家中醒來時便失去了記憶
    • 之後與舊電腦顯現的GRIDMAN相遇，被告知「去完成自己的使命」，通過Access Flash跟GRIDMAN一體化，為守護城市而跟怪獸戰鬥
  • 大部分情況下都是由古利特來掌控意識所以跟GRIDMAN的對話都是自言自語（？），因此沒有自己的記憶
    • 直到結尾古利特離開以後才恢復意識，脫離身體前的古利特有道出裕太本人是喜歡六花的
    • 由於全程沒有以本人的意識說過話，更要命的是整部作品中明明是古利特自己卻沒有表現鮮明的戰鬥意志、後半程更是為了給六花和茜的百合營業讓位而被極度削減戲份，所以被扣上了工具人的屬性，與後作的麻中蓬也形成鮮明對比
    • 所以故事開始前裕太到底和六花說了什麼?
    • 偏偏在這種尷尬的地方致敬了超人力霸王系列首作主角[2]
  • 配色原型是《變形金剛》系列中的Classic跳崖者/飛過山
    • 同時也是少數不是捏他Shattered Glass的角色，因為該作品的跳崖者是從其他作品穿越的
    • 捏他：髮型和跳崖者的臉相似（紅色有角），鞋子和背包的花紋都和該版跳崖者特有的東西一樣

寶多六花（宝多六花）【CV：宮本侑芽】編輯

 

  • 本作女主角，與裕太、內海為同伴同學，廢品店「絢」的老板娘的女兒
    • 故事一開始裕太便是在她家中醒來，发現裕太倒在自己家門前，將其帶回家並幫助了他
    • 當失去記憶的裕太向她問及兩人關系時，回答自己之前並沒有和他說過幾次話。有向對方提及「如果你是假裝失憶就太過分了」
  • 和同班的奈美子、蓮的關系相當好
    • 自稱在裕太失憶前自己跟裕太沒太多的交流，實際上關心著跟GRIDMAN合體而戰鬥的裕太但也僅此而已了
    • 區別於一般ACG作品的男女主角關係，她與裕太與其說是同伴更像是同班同學的相處方式，與後作的南夢芽差別較大
    • 好在時隔幾年依然在雙方聲優辦下來的廣播劇裡有了些進展，這CP還不算完全死掉
  • 由於人設是JK+肥大腿+黑長直的氣質一經開播便獲攬大量人氣，本作的下半身擔當
    • 在最早的原案設計中六花的形象是短髮細腿的人設，但被監督雨宮哲以「要展現JK的形象用富有肉感的大腿會更好」為由改成了目前的形象，髮型也更改為長髮
      • 結果一語成讖引發了現象級效應，還得到了大量同人畫師的青睞→於是二創中粗大腿的形象也深入人心
  • 配色原型是《變形金剛》系列中的SG密卡登
    • 而其母親的配色則是對應G1密卡登
    • 她的同學中則有對應SG天王星和SG聲波
      • 捏他：同學中的なみこ服裝配色和SG天王星相似外，髮夾的形狀和SG天王星所附屬的能量晶片一樣，另一人はっす則是服裝配色和SG聲波相似外，戴著口罩這點也對應了聲波的臉，衣服上寫著的REVAGE則是聲波體內收納的豹型變形金剛的名字（不過SG版是貓，而正好該文字旁有貓的圖案）

內海將【CV：齊藤壯馬】編輯

 

  • 與裕太、六花是同班同學，在裕太失憶前兩人就是好朋友
  • 相當熟悉超人力霸王系列相關作品，經常把作品中的招式用在稱呼GRIDMAN的技能上
    • 在GRIDMAN的第一次戰鬥中找出怪獸的缺點，幫助其取得勝利。之後自作主張地拉著裕太和六花組成グリッド同盟
  • 相比於原作主人公的兩位夥伴能參與建造創造古利特的相關支援裝備，本作中的內海基本上就是啦啦隊擔當，因此也被島民吐槽個沒完
  • 因為都戴眼鏡+內海姓，開播期間很容易跟某位宣誓效忠的科研人員一起被玩梗
  • 配色原型是《變形金剛》系列中的SG橫砲
    • 捏他：SG橫砲因為背叛原本所屬的邪惡博派，因此胸口的標誌有一道刀痕是特徵，和他服服的花紋相似

新世紀中學生編輯

  • 支援GRIDMAN的合體部件化身為人形之後的團體組織，全員都可以在戰鬥時變為和原作一樣的支援機參與作戰
    • 但如果全員都變身進入網路世界會因為信息傳輸過載而被強制彈出，後來調整了出力才得以同時進入網路世界

サムライ・キャリバー【CV：高橋良輔】編輯

 

  • 支援古立特的「新世紀中學生」中的一員，也是四人中最早與裕太等人交涉的
  • 平時是死魚眼造型，但為人態度比較熱忱，但不善於言辭表達還很容易咬舌頭給人一種不易接近的感覺
  • 腰間佩戴著四把劍，進門時都會敲到，持有相當高的戰鬥力。
  • 能變身成支援武器GRIDMAN CALIBUR
  • 配色原型是《變形金剛》系列中的真人版恐龍金剛中的三角龍鐵塊
    • 捏他：配色是跟玩具版較近，此外真人版的鐵塊武器是兩把劍。還有其領子上的花紋和鐵塊的肩膀形狀相似。

マックス【CV：小西克幸】編輯

 

  • 戴著口罩的魁梧男子，冷靜的領隊
  • 人類形態下進行戰鬥時使用帶刺的鐵球為武器。
  • 能變身成支援武器BATTLE TRACTO MAX
  • 原型是《變形金剛》系列中的真人版恐龍金剛中的暴龍鋼鎖
    • 捏他：領子的花紋和玩具版鋼鎖的胸口一樣，口罩的樣子也和他的臉相似，甚至該作鋼鎖的武器正是晨星鎚和帶刺鐵球

ボラー【CV：悠木碧】編輯

 

  • 「新世紀中學生」中是最年輕的少年（沒錯他不是女的），身材比較矮小，打扮也比較中性化
  • 赤口毒舌，吐槽的時候總是無緣無故地會踢內海的小腿
  • 能變身成支援武器BUSTER BORR
  • 原型是《變形金剛》系列中的真人版恐龍金剛中的棘背龍嘲笑
    • 捏他又是領子的花紋和嘲笑本體塗裝的花紋類似

ヴィット【CV：松風雅也】編輯

 

  • 四人中性格比較陽光開朗的形象，雖然是名帥哥，卻總是一副沒有幹勁的模樣
  • 能變身成支援武器SKY VITTER
  • 原型是《變形金剛》系列中的真人版恐龍金剛中的翼龍掃射
  • 捏他還是領子的形狀和掃射的機器人型態時收納在胸口的兩個龍頭酷似


アレクシス勢力編輯

アレクシス・ケリヴ【CV：稻田徹】編輯

 

  • 新條茜的合作者，本作的最終魔王
    • アノシラス2代稱他是「來自外面世界的非常危險的人物」
    • 根據TRIGGER給聲優的裏設定，其真實身分為《宇宙巡警露露子》中登場的OVER Justice本部長
  • 平時寄宿在電腦里，也可以自由地實體化。發動技能實例化可使新條茜制造的怪獸模型實體化
    • 平時雖然看似受限於新條茜的束縛還會被生氣的她踢走電腦無能為力，實際上一直在利用她並在最後反水將其操控
  • 於機戰30老是被梵安、流龍馬記不住自己的全名：「阿格雷希斯甚麼？！」

新條茜（新条アカネ） 【CV：上田麗奈】編輯

 

  • 杜鵑台高中1年E班學生，與裕太等人是同班同學但交涉不深
    • 在班級中人見人愛的偶像般的完美存在，實際上內心相當陰暗，為本作最大問題兒
    • 在アレクシス相助下，制造怪獸將討厭的人殺害，並摧毀不滿意的城市部分
    • 同樣熱衷於超人力霸王系列，但比起超人她更加熱愛其中的怪獸，家中的自製怪獸模型堆裡藏有很多力霸王系列的彩蛋
  • 而她本人也是杜鵑台的造物者，來自現實世界的真正的人類，包括裕太等人生活的世界均由她來創造和調整
    • 結尾離開了電子世界，還把冷氣機搬走了
  • 污部屋，房間裡堆滿垃圾袋
  • 配色原型是《變形金剛》系列中的SG柯博文
    • 捏他：所戴的眼鏡右邊鏡片有裂痕，這是該版本柯博文的特徵（右胸玻璃的裂痕）。耳機則是跟柯博文的耳朵部分相似。
  • 巨乳屬性，本作的上半身擔當
  • 因為人設長相很像《Fate/Grand Order》中的瑪修因而衍生了不少FGO梗
  • 現實中的身分是星野みやこ
    • 星野みやこ出自《私に天使が舞い降りた！》，和茜同CV、同髮型、都喜歡名字裡有「花」的黑長直少女
    • 而且《私に天使が舞い降りた！》剛好在下一季播出，開頭みやこ從床上醒來可以對上《SSSS.GRIDMAN》的結尾
    • 在Pixiv上有tag#私に巨人が舞い降りた!

アンチ【CV：鈴村健一】編輯

 

  • 由茜所創造的唯一一隻有人類形態的怪獸
    • 被設定為專門殺死GRIDMAN的特殊個體，初登場一度戰勝GRIDMAN，但之後敗於強化後的GRIDMAN的戰鬥能力
  • 之後的劇情中再三挑戰過GRIDMAN但無一例外的失敗，被新條茜認為是沒有用的怪獸
    • 後來在跟六花的相遇中逐漸擁有人類化的感情。最後為了尋找自己的意義，選擇跟GRIDMAN共同戰鬥，成為了GRIDKNIGHT
  • 角色原型上服裝配色參考了SG洛迪文（火焰紋路），怪獸型態的原型為原作特攝版中的忍者怪獸シノビラー，角色設定上還參考了《仮面ライダードライブ》中的Chase。
  • 在續作SSSS.DYNAZENON中與下アノシラス成立グリッドナイト同盟，為了調查怪獸的行蹤而四處奔波

其他編輯

怪獸少女アノシラス【CV：高橋花林】編輯

 

  • 突然出現在裕太面前，自稱是怪獸的少女，告知了對方這個世界的真相和與新條茜的關系
    • 據說她的前代受過古利特的恩惠，因此前來報恩
      • 前代為原作特攝版中登場的音波怪獸アノシラス，並於最終話末尾登場
  • 本作中唯一跟新條茜毫無關係的怪獸，屬於原本就存在的怪獸
  • 造型設計上外套配色和髮型分別源自原型音波怪獸アノシラス的背甲和雙角，衣服中央的音符標記源自原作中的音樂精靈胸口的標記
  • 於作品結尾找上了アンチ，兩人在後作《SSSS.DYNAZENON》中組成了グリッドナイト同盟，詳見條目内經歷

登場英雄編輯

GRIDMAN【CV：綠川光】編輯

 

  • 六花家廢品店的舊電腦JUNK上顯現的超人角色，自稱是「Hyper Agent」
  • 最初只有裕太看到他的存在和聽到他的聲音，第一次戰鬥結束後來連內海和六花也能看到
    • 平時是沒有實體的存在，必須和裕太合體才能實體化和巨大化戰鬥
  • 招式、必殺技：
    • 全形態共通：グリッドビーム、グリッドキャリバーエンド、グリッドライトセイバースラッシュ、グリッドキネシス
    • 電光超人古利特：スパークビーム、ネオ超電導キック、グリッドフィクサービーム

形態編輯

  • Initial Fighter
    • GRIDMAN在力量不完全向裕太求助時的姿態，能力較弱，在網路世界的維持時間也很短
  • Primal Fighter
    • 本作新登場造型，為正式作戰時的常用姿態
  • 電光超人古利特
    • 最終話登場，為GRIDMAN取回了真正的力量之後變成的姿態，造型即為原作特攝版的電光超人古利特

GRIDKNIGHT【CV：鈴村健一】編輯

 

「俺は…お前を倒すために生（う）まれ、お前を倒すために生（い）きている。」

「お前を倒すために…俺は！お前と！共に戦う！！！」

  • 由アンチ所變身的的巨大戰士，是アンチ完全複製GRIDMAN的姿態並決心與其並肩作戰後變化的型態，名字有「守護GRIDMAN的騎士」的含義
    • 該角色出自原作特攝版中的廢案「古利特騎士」，原定在劇情中盤由武史先作出複製了古利特數據的卡恩騎士，再由此洗白轉正成友方的古利特騎士
      • 設定上是作為古利特的弟弟，因此外貌會很相似（原案中基本就是改色和換頭）
      • 除了給古利特增加戰力外，也是因為原作中無法實現God Zenon和Dyna Dragon進行合體的皮套製作，原定想要讓同性能的古利特騎士與Dyna Dragon合體，實現Thunder Griaman與King Gridman共同戰鬥的場面
    • 但由於製作經費成本以及贊助商、電視台的多方反對，未能實現這一構思；原定作為彌補的續集《古利特騎士》也未能投拍，於是「與古利特外貌相似的戰士」被設定為「古利特西格瑪」而出現在雜誌連載中，也製作了皮套但也因此未能在TV中登場
    • 上述的相關設定也終於在本作中實現，後作也重現了原案中古利特騎士能與Dyna Dragon合體的設定，雖然名字都大改了不少
  • 小說版針對這個型態有了更進一步的描述：原來這個型態要誕生的話除了アンチ本人的覺悟之外，還需要新世紀中學生和GRIDMAN等人認可アンチ的存在
    • 換句話說アンチ能夠成功變身的原因就是因為GRIDMAN等人認可了他擁有與人類相同的情感並把他當人類看待他才得以蛻變為GRIDKNIGHT
      • 而這點似乎也是アンチ最初變身時サムライ・キャリバー會反駁認為GRIDKNIGHT為偽物的ボラー稱此時的アンチ為真貨的主要原因
    • 順帶一提，不知為何在小說版中アンチ是唯一一個來自不同時間線的角色[3][4]，只是在小說版的大部分時間都處於失去第8話之後所有記憶的狀態，因此一度失去了變身為GRIDKNIGHT的能力，但在小說後期アンチ取回了所有記憶並借用了GRIDMAN在戰鬥後殘留的能量因此短暫的重獲變身能力，只不過官方為了避免對グリッド同盟劇透因此他也在這場戰鬥後回到了他原來所處的時間線，並沒有參與小說版的最終決戰
  • 招式、必殺技：グリッドナイトサーキュラー、ナイトサーキュラーエンド、グリッドナイトストーム、ナイトキャリバーエンド、ナイト爆裂光波弾
  • 在後作中登場的合體形態詳見條目

合體模式編輯

  • 剛力合體超人MAX GRIDMAN（マックスグリッドマン）
     
    • 由GRIDMAN和BATTLE TRUCK MAX合體而成的力量特化型態。
    • 雖然這個型態跟續作的某一型態一樣手臂特別的巨大，但並沒有跟它一樣有重心不穩導致難以站立的問題。
    • 必殺技為Max Grid Beam和Super Lightning Kick明明強化了手臂，但必殺技卻是光砲和飛踢
  • 武裝合體超人BUSTER GRIDMAN（バスターグリッドマン）
     
    • 由GRIDMAN和BUSTER BORR合體而成的火力特化型態。
    • 必殺技為Twin Buster Grid Beam沒用過鑽頭近距離攻擊怪物，連隊友都吐槽
  • 大空合體超人SKY GRIDMAN（スカイグリッドマン）
     
    • 由GRIDMAN和SKY VITTER合體而成的空中移動專用型態。
    • 唯一個沒有用過必殺的形態，但SKY VITTER分離後獨出射出導彈Laser Circus Amp輕易將アンチ打倒

超合體超人FULL POWER GRIDMAN（フルパワーグリッドマン）編輯

 

  • 由GRIDMAN和新世紀中學生全員合體而成的型態，初登場時就被新條茜吐嘈根本就是個機器人
    • 為了降低JUNK的負擔，要合體成這個型態時必須要壓縮出擊五人的資料輸出量
    • 結果導致全員的體型都變得比較小，GRIDMAN的體型更是只有平時的一半，完美的迴避了全員體型比例有差異的問題。
  • 原型來自於原作《電光超人グリッドマン》中的合體超神Thunder Gridman
