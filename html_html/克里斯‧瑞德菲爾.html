解說[編輯]

  • 克里斯‧瑞德菲爾是卡普空的遊戲《惡靈古堡》系列的男主角之一。

基本資料[編輯]

  • 全名Christopher Redfield (克里斯多福‧瑞德菲爾)[1] [2]
  • 身高：181cm（1代）～185cm（截至6代官方的公開資料）
  • 體重：80.5kg（1代）～100kg（截至6代官方的公開資料）
  • 年齡：25（1代，1998年[3]）～47（8代，2021年）
  • 血型：O
  • 演員：
    • Charlie Kreslavskiy（bio1真人片段出現）
    • 溫特沃斯·米勒（Wentworth Miller）（真人電影第四部陰陽界）
    • 羅比·艾梅爾（Robbie Amell）（2021年真人電影重啟版）
  • CV：
    • 井上和彥（DC版[4]、DS版[5]）
    • 東地宏樹（6代以後）
    • 木村昴（2021年真人電影重啟版）

身份[編輯]

  • 美國空軍飛行員（1990年～1995年）
  • S.T.A.R.S. A隊尖兵（1代，1996年～1998年）
  • BSAA特工&創始人之一（5代起，2003年～）

生平經歷[編輯]

  • 《惡靈古堡》
    • 逃出已化為人間地獄的洋館後，前往歐洲蒐集保護傘公司的罪證。
    • 後與吉兒等人一道創立BSAA，為根除生物兵器而在世界各地奮戰。
  • 《惡靈古堡：聖女密碼》
    • 為了救被保護傘公司俘虜的妹妹克萊兒而前往洛克福島，在那裡得知威斯卡還活著和克萊兒在南極的消息。
    • 後前往南極和克萊兒重聚，與威斯卡對戰後，加強了摧毀保護傘公司的決心。
  • 《惡靈古堡：啟示》
  • 《惡靈古堡5》
    • 與搭檔吉兒對戰威斯卡，結果兩人墜崖失蹤、生死未卜，僅剩克里斯歸隊。
    • 爾後克里斯被派往非洲、與新搭檔席娃一道抓捕販賣生化武器的軍火商，並從中獲知吉兒還活著的情報，而展開了千里尋妻記
    • 隨著事態發展失控而被捲入Tricell公司的陰謀中，更發現原來黑幕竟是威斯卡和被他控制的吉兒。
    • 經過奮戰後兩人救醒吉兒，並在一處火山口將威斯卡徹底擊殺。
  • 《惡靈古堡6》
    • 於東歐出任務時遭卡拉陷害、整個小隊除了皮爾斯以外其餘隊員在眼前被感染成生物兵器，如此打擊令克里斯罹患PTSD而失憶、過著整天借酒澆愁的生活，直到被皮爾斯等一干隊友找到。
    • 歸隊後前往中國執行反生化襲擊任務，其間逐步取回記憶，但在遭遇卡拉後便執意要追殺她復仇，因而與里昂發生短暫衝突；之後更與舊日仇人威斯卡之子傑克見面，並向傑克親口坦承自己就是殺死他父親威斯卡的兇手。
    • 最終在海底設施中擊敗巨大生物兵器，但在逃離設施時皮爾斯與生物兵器同歸于盡，僅剩克里斯一人乘坐逃生艙浮出水面。
  • 《惡靈古堡7》
    • 響應新保護傘公司[6]的求援，作為BSAA派出的顧問而率隊前來助陣、與新型生化兵器「菌屍」進行戰鬥。
    • 於遊戲終盤登場救下剛剛與黑幕伊芙琳戰鬥過的主角伊森。
    • 個人在DLC《並非英雄》中活躍，將貝克家的最後餘孽盧卡斯·貝克消滅；並於DLC《柔伊的結局》中照料注射血清而康復的柔伊·貝克。
  • 《惡靈古堡8》
    • 遊戲序盤帶領獵狼小隊突襲溫特斯家擊斃米婭外表的故事黑幕米蘭達，且後者並未死去，將伊森及其女兒蘿絲瑪麗護送轉移。
    • 得知護送伊森父女的車隊半途被襲擊而全滅後帶領獵狼小隊迅速趕往村莊。
    • 出於專業反BOW戰士的責任感一直拒絕讓伊森捲入事件而對他三緘其口，最終在伊森幹掉四大家族最後的海森伯格前終於將真相和盤托出、並同意與他共同行動救出蘿絲。
    • 然而在這之後沒幾分鐘伊森便被米蘭達掏心殺死，克里斯帶領獵狼小隊以絕對的火力優勢殺進村莊、並给米蘭達和E霉菌的力量源「菌主」装上炸彈，之後在米蘭達的實驗室裡救出米婭，並在米婭口中知道伊森在初到貝克家大宅那時已死，並给E霉菌改造成半人半菌屍的真相。
    • 在依靠意志力強行回生的伊森擊殺米蘭達後終於再次遲來一步地趕到並在意識與菌主融合並暴走的米蘭達面前帶著伊森和女兒蘿絲撤退，被油盡燈枯的伊森奪走炸彈的引爆器，並託付照料妻子和蘿絲後獨自帶著蘿絲撤離。
    • 在撤離的飛機上發現BSAA派到村莊的士兵其實是生化武器，怒不可遏之下決定直接前往BSAA歐洲總部興師問罪。
  • 《惡靈古堡：血仇》
    • 參見專門條目。
    • /RED-FIELD!!!\[7]

對人關係[編輯]

  • 吉兒‧范倫婷
    • 舊搭檔，自S.T.A.R.S.時期起便同為A隊同袍，後更有多次聯手合作、並一同創立BSAA。
    • 若兩人共同演出，高達八成會出現互找對方的劇情
    • 彼此有著強烈的信任和合拍度，執行任務時會互相緊張對方。
    • 你們兩個，乾脆結婚算了[8]
  • 亞伯‧魏斯克
    • 從一代開始結怨的宿敵兼真愛。
    • 最終於5代和同伴席娃用火箭筒將其轟殺在火山口岩漿中。
  • 克萊兒‧瑞德菲爾
    • 妹妹，唯一的親人。
    • 大概是怕妹妹被痴漢騷擾，所以有親自傳授給妹妹各種防身的戰鬥技術和槍械運用。
    • 其妹也有著不亞其兄剋隊友屬性的剋夫屬性
  • 貝瑞‧波頓
    • 從空軍時代開始的故交。
  • 里昂‧史考特‧甘乃迪
    • 因克萊兒關係而認識。
    • 同為主角對抗生化兵器的戰友。
    • 雖然克里斯也不是什麼循規蹈矩的人，但相對而言還是和有時候過於我行我素的里昂在一些場面會有分歧，6代更是由於對艾達·王的態度而舉槍相向。
  • 潔西卡‧雪拉瓦特
    • 啟示的搭檔，前FBC特工，兼Tricell公司派來的臥底
    • 喜歡克里斯，而有意無意色誘他
  • 席娃‧阿洛瑪
    • 5代在非洲執行任務時的同伴。
    • 少數沒被克里斯內在的剋隊友屬性剋死的角色，可也一直被官方無視慘變成失蹤人口
  • 皮爾斯‧尼凡斯
    • 6代被克里斯賞識的隊友，槍法了得，不過最後仍被剋死壯烈殉職。
  • 傑克‧謬拉
    • 宿敵之子，孽緣。

能力[編輯]

  • 曾在美國空軍擔任飛行員，故擁有優秀的戰力、射術和生存能力。
    • 常年與B隊的佛瑞斯特‧史拜爾一同霸占射手榜的頭名。
    • 6代中可使用的武器有著標準的軍人特色（戰鬥霰彈槍+標準突擊步槍+連發榴彈發射器等）。
  • 剋隊友
    • 同行的隊友基本都不會平安無事，就連同為主角的吉兒也逃不過被抓去控制一段時間，其餘男性更是幾乎全部陣亡。
      • 而老妹也繼承到他的剋星屬性，在聖女密碼剋了史帝夫。

性格[編輯]

  • 熱情
  • 暴躁（情緒化）
    • 任職空軍時亦因為與上司起衝突而被革職
    • 平素其實還好，在面對某些特定情境時會失去冷靜（比如關於妹妹克蕾兒和搭檔吉兒的問題）
    • 6代有收斂許多。
    • 這點與（4代後成為特工之後、以理性思考優先的）里昂形成對比
  • 容易鑽牛角尖
  • 有強烈責任感
    • 雖然給人以任務為優先的死板感覺，但實際上若遇到某些特定情境時，會以家人和拍擋優先。
      • 如聖女密碼中得知克蕾兒被囚在洛克福島，便馬上前往營救。
      • 啓示中要潔西卡專注於任務，但得知吉兒在任務中失聯便馬上說要找她，把進行中的任務轉交給其他隊友。

萌屬性[編輯]

  • 哥哥
  • 筋肉男這個能萌得起來嗎？
    • 5-6代專屬，本來初代、聖女密碼的克哥這時期還沒有強壯的這麼誇張，但是直到5代開始變成超壯肌肉男，到7代因為改用人臉捕捉技術製作、請不起大猩猩，就又改回去舊世代模樣的折衷版。
    • 8代47歲模樣變成了6代熊男樣的兩倍畢竟克哥都已經中年老了。
    • 5代的體術處決技中可橫向搬起一個成年男性感染者、並像標槍一樣直線扔出去，甚至在跟威斯卡的最終決戰中用拳頭把一個大石打進岩漿之中。
    • 甚至在在《Project Cross Zone》系列中，輕易把巨大機器人打飛
      • 《Project Cross Zone》中透露希望拜肯為師來學會波動拳以節省子彈，還好吉兒以「如果他的手能發出死光的話便和他斷絕拍檔關係」為由而作罷
    • 畢竟以前是美國大兵，筋肉發達到這個地步也大概是理所當然的事。
    • 因此經常被戲謔感染了大猩猩病毒
  • 妹控（半誤？）
    • 在原著中最多是關心妹妹的哥哥
    • 但在同人中則是會對克蕾兒有好感的人全都教訓一頓，而受害者多半是里昂
      • 近年更冒出了強迫里昂娶自己妹妹的惡搞
      • 不過隨著里昂只追求艾達，他的魔掌開始轉向剛亡妻的伊森

其他[編輯]

  • 配槍是Beretta 92。
    • 此槍亦是美軍的制式手槍（現被SIG P320取代），曾經當過空軍飛行員的克里斯會拿這把槍也是（ry。
    • 在CG電影《血仇》中則是携帶其自動版本Beretta 93R（三發點放）。
    • 6代的初始配槍除了手槍外還追加了沒有現實原型的特殊作戰用突擊步槍[9]，《血仇》中則是攜帶裝備模塊化戰術組件的M4自動步槍[10]
  • 自一代以來貫穿整個系列的全勤男主角，與他並列的還有吉兒、里昂、克萊兒、艾達。
    • 就某種程度上來看，他們根本可以湊成四人組+第五人，克里斯是強悍的主力型、里昂是帥氣的輔力型、克萊兒是事件前線後面的支援型、吉兒是多次與死亡擦肩而過的潛力型，而艾達則是危機時拔刀相助的助拳人型。
    • 然而除了六代有讓克里斯、里昂和艾達一同出現外，克萊兒和吉兒到目前為止幾乎沒有同時一起出現的情況。
      • 可能的原因是製作團隊聚焦在克里斯和里昂兩大男主身上，女主似乎因身處不同勢力所以難有交集，其餘的雪莉、蕾貝卡和貝瑞等人總共加起來才露面兩三次，剩下僅僅出現一次的主角們就跟不用說了。
      • 不過就以二代重製版的方向來看，製作團隊如果能利用這由頭發展後續的新作，然後在遊戲中湊齊五人組也沒有不可能。
