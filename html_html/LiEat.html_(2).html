LiEat -以謊言為食的龍-（ライート -嘘喰いドラゴン-）
本條目內容不足需要擴充，歡迎有興趣的朋友加以補完。
本條目內含劇透，沒玩過遊戲又不想被謊言吞噬而成為騙子的話請不要隨意反白。

簡介[編輯]

  • △○□×（Miwashiba）以WOLF RPG Editor開發的免費角色扮演遊戲。
  • 以「吞食謊言」為主軸，描寫詐欺師與龍少女在各個小鎮間的旅程經歷。
  • 雖然有推理要素不過也都是以吞食謊言做為選項，選錯了也不會有懲罰，甚至還能用來練等。
  • 結局通常有BE與GE，分支條件多為最終戰的勝敗，因此也有不小心練太強反而打不到BE的情況發生。

章節[編輯]

  • 每個章節開頭都有一段傳奇故事風格的楔子，通常會在講完一個衰毛爛故事後硬接一個「可喜可賀」收尾。
    • 故事內容通常是劇中要角的實際經歷。

嘘喰いドラゴンと朱色の吸血鬼[編輯]

很久以前，在某個小街道曾經居住著吸血鬼的一族。
他們以十分溫和的性情，和人類們和睦地在街上一起生活著。
然而，在某天深夜裡，溫柔的吸血鬼襲擊了街上的人類。
當回神注意到的時候，已不慎吸了那名人類的血。
之後，攻擊人類的事被其他人類知情，吸血鬼就被處刑了。
可喜可賀，可喜可賀。
  • 系列第1章。
  • 描寫前往紅色小鎮進行青鳥羽毛買賣的主角遭遇吸血鬼傳說的兇殺案件。

嘘喰いドラゴンと紺碧色の夢喰い[編輯]

很久很久以前，在某個地方曾經存在一隻又白又毛絨絨的綿羊。
綿羊以人的惡夢為食糧，靠著吃掉住在附近的人類的惡夢來過生活。
某一天，惡夢的味道讓綿羊吃膩了。
那個時候，在牠眼前的女孩子正作著十分開心的夢。
綿羊將那開心的夢偷偷地吃掉了，真是太好吃了，綿羊非常開心。
夢被奪走的女孩子，在那之後就變得像具空殼一般了。
可喜可賀，可喜可賀。
  • 系列第2章。
  • 描寫主角在蔚藍小鎮的某娛樂設施調查龍族卻遭遇食夢事件。

嘘喰いドラゴンと黄金色の怪盗[編輯]

從前，在某個地方有著一名怪盜。
那位怪盜巡遊世界各地盜取了許許多多光輝華貴的珍寶。
怪盜所瞄準的目標總是那些藉由作盡惡事以獲得大量財產的人們。
怪盜，不曾失敗。瞄準的寶物，必定到手。而且從未被警察逮捕。
怪盜總是將取得的寶物變賣為金錢，並且將那些金錢分送給貧窮小鎮的人們。
接著怪盜…
  • 系列最終章。
  • 描寫主角前往金黃小鎮的某豪宅赴宴卻遭遇怪盜事件。

嘘喰いドラゴンと忘却色の歌姫[編輯]

很久很久以前，有個地方住著一隻青鳥。
相傳青鳥會實現人們的願望，實現了住在那裡之眾多居民的心願。
直到有一天，有個惡人對青鳥許了個願望。青鳥實現了那個人的願望，結果令許許多多的人陷入悲傷。
青鳥不明白自己做錯了什麼。只是繼續實現人們的心願。
漸漸地，人們逐漸離開了那個地方。只剩下青鳥孤獨一個。
可喜可賀，可喜可賀。
  • 小說版，時間軸為系列第0章。
  • 描寫主角前往靛藍小鎮調查青鳥卻遭遇死者復活的怪異。

登場角色[編輯]

主要角色[編輯]

  • 詐欺師 / 特奧巴多·里歐赫茲（Theobald Leonhearts）
    • 男主角，說謊如呼吸一樣自然的詐欺師。
    • 在每一章使用的名字與外貌都完全不一樣。
      • 第1章：里歐（Leo）
      • 第2章：哈爾（Hal）
      • 最終章：席德（Sid）、提奧（Teo）
      • 小說版：阿爾（Al）
    • 雖然總是嫌艾菲娜很煩但又不討厭帶她四處闖蕩，傲嬌確認（蓋章）。
    • 過去為貧民窟出身，為了安頓同伴而接受某幫派的交易，賣掉了自己的一顆腎臟換來大筆錢財。
    • 還為了能讓同伴離開貧民窟而說謊拋下他們，卻因此造成了同伴的不諒解。
    • 冒死逃離了幫派後展開流亡生活，也因此不斷改變自己外貌與姓名以躲避幫派追殺。
    • 因為過去對同伴說謊的愧疚而喚來了艾菲娜，從此展開了兩人的旅程。
    • 最終章因為希瑞兒的計畫而被謊言吞噬成為騙子，最後被艾菲娜救出。
  • 艾菲娜（Efina）
    • 女主角，以謊言為主食的龍。
      • 其能力可以使謊言實體化成為怪物以便將其吞噬。
      • 當目標說出的話語為虛實夾雜時，真假兩邊都會被實體化，但真話的部分對艾菲娜而言會難以入口，因此被男主當作行動測謊機偵測目標謊言的手段。
    • 對外界擁有強烈的好奇心，但因為年紀還小的緣故[1]腦筋不太好。
    • 因為提奧說謊背叛同伴的懊悔而作為吞噬說謊之龍誕生，提奧以剛讀過小說的角色為其命名。
    • 最終章因為希瑞兒的超音波導致能力暴走並使宅邸被汙泥包覆，為了救出被謊言吞噬成騙子的提奧而奔走。
  • 尼爾·梅斯菲爾德（Neil Masefield）
    • 警騎隊隊長，不死之龍。
      • 其不死能力讓他就算受再重的傷都能不死，對痛覺與毒素也有高耐受性。
    • 過去被前警騎隊隊長撫養長大，養母過世以後接下隊長職責。
    • 最終章可以讓他入隊並利用其能力破解特定房間的謎題。
  • 布雷特·葛瑞夫斯（Brett Graves）
    • 警騎隊副隊長，影之龍。
      • 藉由影子的能力可以實施空間穿越。
    • 過去的主人是幫派分子，在其底下持續遭受凌虐，被尼爾隊長救出後加入警騎隊。
      • 因為上述經歷的緣故，對人類尤其謊話連篇的男主非常不信任，只有在同為龍族的艾菲娜面前會卸下心防。
    • 最終章可以讓他入隊並利用其能力破解特定房間的謎題。

第1章[編輯]

  • 卡蘿·黛萊達（Carol Deledda）
    • 帽子情報屋，喜歡可愛的東西。
    • 手邊總是帶著愛莓果實，不是拿來吃的而是防身用（把果實扔到對方臉上）。
    • 在前傳小說委託主角前往靛藍小鎮調查青鳥，自己則為了調查吸血鬼而留在朱色小鎮。
    • 吸血鬼殺人事件的被害者，在主角前往朱色小鎮的當晚在小鎮的暗巷被殺死，屍體被發現時喉嚨被刺穿且全身的血液都被抽乾。
  • 奇斯·瑞格菲亞（Keith Rigufaia）
    • 居住在紅色小鎮的青年，瑞秋的哥哥。
    • 吸血鬼一族的末裔，為了保護妹妹而持續說謊，卻因此被謊言吞噬成為騙子，最後被艾菲娜救出。
    • 從第2章與尼爾隊長的對話可以得知兄妹倆人目前正在更生保護設施接受能力控制方面的再教育。
  • 瑞秋·瑞格菲亞（Rachel Rigufaia）
    • 居住在紅色小鎮的少女，奇斯的妹妹。
    • 吸血鬼殺人事件的真犯人，覺醒了祖先的吸血衝動後因為無法抑制衝動而殺人。
    • 從第2章與尼爾隊長的對話可以得知兄妹倆人目前正在更生保護設施接受能力控制方面的再教育。
  • 羅莎莉·羅絲瑪麗（Rosalie Rosemarie）
    • 羽毛情報屋。
    • 大財閥出身的大小姐，因為不想繼承財產而帶著雷文離家出走。
  • 雷文·布魯姆（Levin Bloom）
    • 羅莎莉的管家。

第2章[編輯]

  • 湯姆·羅得（Thom Lot）
    • 娛樂設施的守門人，馬爾斯的哥哥。
  • 馬爾斯·羅得（Mars Lot）
    • 娛樂設施的守門人，湯姆的弟弟。
  • 奧莉薇亞·卡麥隆（Olivia Cameron）
    • 娛樂設施的發牌者。
    • 舞台殺人未遂事件的真犯人，病態般的愛著盧卡斯，為了報復他的花心而持續攻擊他並企圖利用威廉消除犯案相關的記憶。
    • 被主角察覺真相後將兩人丟進地下垃圾場，隨後卻被陷入暴走的威廉吞噬一切記憶，最後被警騎隊逮捕。
  • 盧卡斯·貝利奧爾（Lucas Balliol）
    • 娛樂設施的酒保，性格風流的色男。
    • 本尊哈爾的朋友，少數察覺到主角偽裝的人。
    • 舞台殺人未遂事件的被害者，在主角前往娛樂設施的當晚被艾菲娜發現被人勒暈並綁在舞台的十字架上。
    • 實際上是因為多次出軌而被其女友奧莉薇亞報復，因為假裝自己被奪去記憶而被謊言吞噬成為騙子，最後被艾菲娜順便救出。
  • 索菲亞·胡安內爾（Sofia Juanele）
    • 娛樂設施的主人，曾為處刑人，目前作為舞台魔術師活躍。
    • 威廉的孵育者，將其視如己出，因為害怕警方介入會給威廉造成威脅而多次隱瞞盧卡斯遇襲的事件，直到舞台事件再度發生後才報警。
  • 威廉（William）
    • 索菲亞的助手，性格軟弱的少年。
    • 真實身分是擁有吃掉記憶能力的龍族，因為索菲亞想忘卻痛苦記憶的心願而誕生。
    • 在終盤因為能力暴走而成為怪物，被主角制服後回復正常。

最終章[編輯]

  • Butterfly / 希瑞兒·里歐赫茲（Cyril Leonhearts）
    • 企圖盜取祕寶的怪盜。
    • 提奧在貧民窟的同伴之一，因為誤以為提奧背叛自己而成為怪盜並與其他同伴聯手展開報復。
      • 最終章的楔子即是她過去所喜愛的怪盜故事。
    • 盜取祕寶只是障眼法，真正的目的是利用超音波讓艾菲娜能力暴走讓提奧被謊言反噬成為騙子。
  • 伊諾斯·里歐赫茲（Enos Leonhearts）
    • 發明家，宴會的助理。
    • 提奧在貧民窟的同伴之一，與希瑞兒一同展開對提奧的復仇。
  • 吉恩·里歐赫茲（Gene Leonhearts）
    • 駕駛員。
    • 提奧在貧民窟的同伴之一，與希瑞兒一同展開對提奧的復仇。
  • 艾瑞絲·阿修貝里（Iris Ashbery）
    • 警騎隊特別調查部隊長，分析之龍。
      • 集中精神時可以對眼前的任何事物進行分析並看穿其本質。
    • 以前的監護人是一名求知慾旺盛的老人，老人死後被警騎隊收留[2]。
      • 雖然對人類的態度更接近尼爾隊長而與布雷特相反，但對兩人都抱持相當的尊敬。
    • 最終章可以讓她入隊並利用其能力破解特定房間的謎題。

小說版[編輯]

  • 米夏（Mischa）
    • 靛藍小鎮的歌姬。
    • 罹患記憶喪失的先天疾病，記憶只能保留三天便會遺失，不論是親朋好友甚至是與自己最親密的菲爾都會在三天後遺忘。
    • 實際上還記得菲爾但也僅限於記住名字卻想不起來相處的種種，為了不讓菲爾傷心而順應他每次打招呼的「初次見面」，卻因此使自己被謊言吞噬成為騙子，最終被菲爾救出。
  • 菲爾（Phil）
    • 照顧米夏的管家。
    • 青鳥的孵育者，為了讓米夏快樂而利用其能力造出了大量快樂鎮民的幻象，卻間接導致原本的鎮民產生猜忌。
  • 青鳥（Bluebird）
    • 相傳能實現願望的青鳥。
    • 實際上是菲爾孵化出的龍族，擁有利用記憶製造幻象的能力，但沒辦法復原喪失的記憶。
    • 事件結束後從男主接下了「阿爾」的名字，並贈送自己的羽毛給他。

世界觀[編輯]

  • 龍（ドラゴン）
    • 擁有不可思議力量的人外生物。
      • 本作的世界觀裡包含亞人的一切人族不論精靈還是獸人都歸類在「人類」的範疇，唯有龍族因為其神祕性與能力而被區隔開來。
    • 因為其能力容易被有心人士利用的緣故，目前也是警騎隊的重點保護對象。
    • 實際上是依照「主人」的心願而誕生，也隨著主人的意識決定其擁有的能力。
  • 警騎隊
    • 偵辦各種案件的警察單位。
    • 在身為龍族的尼爾接任隊長後，開始對龍族（尤其是被惡人利用的）展開保護措施。
