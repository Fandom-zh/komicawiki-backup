簡介[編輯]

Mighty Gunvolt[編輯]

  • 由Inti Creates開發，2014年於任天堂3DS推出的橫軸動作遊戲。
  • 一開始僅作為《蒼藍雷霆鋼伏特》的預購特典，後來開始在eShop一般販售並利用DLC追加四個新關卡。
    • 在PS4推出的是囊括所有DLC關卡的完整版《Gal Gunvolt》。
  • 操作介面為仿元祖洛克人的FC風格，三名可操作角色分別是《蒼藍雷霆鋼伏特》的鋼伏特、《少女☆射擊》的艾可蘿，以及《麥提9號》的貝克。
    • 三人的特性分別是二段跳躍（鋼伏特）、空中飄浮（艾可蘿）與空中衝刺（貝克）
  • 關卡與頭目以《蒼藍雷霆鋼伏特》登場的頭目為主，然而最終頭目卻是《鋼伏特》中被劇情殺的反派角色。
  • 因為本篇不斷跳票的關係，這部還是《麥提9號》主角貝克的真出道作品。

Mighty Gunvolt Burst[編輯]

  • 《Mighty Gunvolt》的續作，2017年推出於Switch及3DS平台。
  • 可操控主角為貝克與鋼伏特，艾可蘿為繼承體驗版存檔的追加角色，除此之外亦可利用DLC購買其他角色。
    • 在PS4推出的《Gal Gunvolt Burst》無需繼承存檔便能使用艾可蘿。
  • 時間點在《麥提9號》結束後，描寫受困虛擬世界的主角各自展開的冒險。
  • 系統層面追加了爆裂連擊，在近距離擊破敵人時可以提升連擊數，連擊數的上升可以提高自身攻擊力甚至影響過關評價。
  • 除此之外在各關卡收集的道具可以用來改裝武器，甚至有部分角色能力也必須藉由武器客製化來解鎖。
  • 與前作相對，本作的關卡與頭目改以《麥提9號》登場的頭目為主，然而最終頭目依然是鋼伏特那邊的反派角色。

主要角色[編輯]

  • 貝克（ベック）
    • 《麥提9號》的主角。
    • 特殊移動法為空中衝刺。
    • 沒有必殺技，不過可以更改射擊子彈的類型。
  • 鋼伏特（ガンヴォルト）
    • 《蒼藍雷霆鋼伏特》的主角。
    • 特殊移動法為多段跳躍。
    • 必殺技為消耗SP施展的各種電擊攻擊。
  • 艾可蘿（えころ）
    • 《少女☆射擊》的女主角。
    • 特殊移動法為滯空。
    • 雖然體力偏低，不過必殺技「女神的祝福」可在一定時間內無敵並提升攻擊力。
  • 可露（コール）
    • 《麥提9號》的女主角。
    • 特殊移動法可選擇滯空或空中衝刺。
    • 必殺技為展開護盾。
  • 希安（シアン）
    • 《蒼藍雷霆鋼伏特》的女主角。
    • 特殊移動法為多段跳躍。
    • 可以變更射擊子彈的類型，但射擊速度或子彈大小等細節只能隨子彈類型改變無法自由調整。
    • 必殺技「絕唱」可在一定時間內解除裝備容量上限。
  • 雷（レイ）
    • 《麥提9號》的宿敵。
    • 特殊移動法為衝撞，可視為有攻擊判定的空中衝刺。
    • 固有技「械胞崩壞」會隨時間過去損失血量，但可以藉由攻擊敵人回復。
    • 必殺技「亂斬」為必定造成爆裂連擊的近身攻擊。
  • 亞裘拉（アキュラ）
    • 《蒼藍雷霆鋼伏特》的宿敵。
    • 特殊移動法為空中衝刺Z，可以往斜角方向衝刺。
    • 必殺技為與「RoRo AI」合力施展的各種攻擊。
  • 庫羅娜（くろな）
    • 《少女☆射擊》的宿敵。
    • 特殊移動法為滯空。
    • 必殺技「惡魔力解放」可在一定時間內提升攻擊力並無視對手防禦。
  • 天造（テンゾウ）
    • 《少女☆射擊》的主角。
    • 特殊移動法為「天使的引導」，可在一定時間內在空中在空中自由移動。
    • 必殺技「心跳不已領域」可以在場上展開領域並利用準心對敵人實施爆裂連擊。

頭目列表[編輯]

Mighty Gunvolt[編輯]

  • 迪德那（デイトナ）
    • STAGE 1「TRAIN」的頭目。
  • 魔界植物君Jr.（魔界植物くんJr.）
    • STAGE 2「SCHOOL」的頭目。
    • 來自《少女☆射擊》的觸手怪物，戰鬥時會將《Double Piece》的女主角神園姐妹關在牢籠。
    • 將伸出的觸手打掉之後本體會出現，攻擊本體方能造成損傷。
    • 偶而會放雜魚襲擊被囚的神園姐妹，此時神園姐妹的呼救也會有攻擊判定。
  • 史特拉斯（ストラトス）
    • STAGE 3「LAB」的頭目。
  • 伊歐坦（イオタ）
    • STAGE 4「TOWER」的頭目。
  • 潘緹拉（パンテーラ）
    • STAGE 5「MIRROR」的頭目。
    • 在本作中成功對本篇擊敗自己的亞裘拉逆襲並將其擊倒在地。
    • 攻擊模式基本上跟後來在《鋼伏特爪》的模式相近。
  • 熱霸（パイロジェン）
    • EX STAGE「FACTORY」的頭目。
  • 艾莉潔（エリーゼ）
    • EX STAGE「BIO LAB」的頭目。
    • 與原作不同，兩人的體力為共享，被打死時也不會用復活技。
  • 劍舞（ブランディッシュ）
    • EX STAGE「HIGHWAY」的頭目。
  • 幸福君（しあわせくん）
    • EX STAGE「GYM」的頭目。

Mighty Gunvolt Burst[編輯]

  • 圓球挖掘機（ラウンドディガー）
    • 序章關卡「市區」的頭目。
  • 熱霸（パイロジェン）
    • 關卡「石油平台」的頭目。
    • 主要會在地上衝刺，衝到對側時可能會對對側牆壁施展拳擊或是對地面使用帶有火球的摔跤。
  • 凍球（クライオスフィア）
    • 關卡「水道局」的頭目。
    • 平時會將冰球圍繞在四周，然後會使用對垂直面的下砸或是對水平面的擊球攻擊。
    • 必殺技疑似致敬G7某下水道人魚會跳起並將冰球對準玩家位置發射冷凍光束。
  • 雷子（ダイナトロン）
    • 關卡「發電廠」的頭目。
  • 震力（サイズミック）
    • 關卡「礦山」的頭目。
  • 軍統（バタリオン）
    • 關卡「軍事基地」的頭目。
    • 主要攻擊模式為在天花板用機槍攻擊，以及放出狼煙召喚雜兵。
    • 必殺技會像G7某甜點師一樣將身體分裂各自出招，最後才合為一體開大砲。
  • 昇行者（アヴィエイター）
    • 關卡「電波塔」的頭目。
    • 必殺技會用錄影機把玩家與頭目的移動位置錄下後播放。
  • 劍舞（ブランディッシュ）
    • 關卡「高速公路」的頭目。
    • 必殺技會跳起並對地砸三次直升機。
  • 擊影（カウンターシェード）
    • 關卡「總統官邸」的頭目。
    • 一般攻擊模式為使用蟲洞穿牆的三連彈或反射彈。
    • 偶而會召喚禮盒丟出具爆破性的假道具或是可回血的真道具。
    • 血量到一半時會用狙擊槍攻擊鏡頭妨礙視野，此後的攻擊還會同時用三連彈與反射彈。
  • 鋼伏特 / 貝克（ガンヴォルト / ベック）
    • 終盤關卡「閘門」的頭目。
  • 灌木守護者（ブライアーガーディアン）
    • 終盤關卡「亞空之海」的頭目。
  • 電漿雷吉翁（プラズマレギオン）
    • 終盤關卡「反烏托邦」的頭目。
  • 偽鋼伏特 / 偽貝克（偽ガンヴォルト / 偽ベック）
    • 最終關卡「電腦競技場」的中頭目。
  • 迪賽歐（テセオ）
    • 最終關卡「電腦競技場」的頭目。
    • 攻擊模式主要與《鋼伏特爪》的模式相近，不過會搭乘大型機器戰鬥。
    • 血量到一半時會將機器的頭部上昇，必須利用落下的針筒當作墊腳石才能攻擊到他。
    • 血量清零時本體會墜落地上，必須與前來支援的貝克 / 鋼伏特協力攻擊本體擊敗他，不然他會坐回機器並回復少許血量。
