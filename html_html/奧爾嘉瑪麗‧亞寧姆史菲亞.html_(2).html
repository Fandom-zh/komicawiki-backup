奧爾嘉瑪麗‧亞寧姆史菲亞

オルガマリー・アニムスフィア／Olga Marie Animsphere
以下含有部分的劇透，如果不想來不及得到他人認可便在故事的序幕就領便當退場，請不要繼續看下去。
  • TYPE-MOON作品《Fate/Grand Order》中登場角色之一。
    • 魔術名門亞寧姆史菲亞家出身，人理續存保障機關菲尼斯‧迦勒底的所長。
    • 在Drama CD中的CV為豐口惠，TV動畫版中的CV則為米澤円[1]。

故事中經歷編輯

《用漫畫來了解吧！Fate/Grand Order》編輯

  • 第一部第1話登場。原本打算好好教訓リヨぐだ子這個新人，卻馬上被對方抓起來恐嚇而嚇哭，從此確立兩人的上下關係。
    • 之後雖然努力地想將漫畫主題拉回遊戲解說，卻因為リヨぐだ子對遊戲解說毫無興趣而失敗。
  • 第二部《更加地用漫畫了解FGO！故事時間軸在序章過後，是所長已經領便當下場的時間點。
    • 因為リヨぐだ子很想念所長（雖然連她長什麼樣子都想不起來），將手伸入？！迦勒底亞斯當中把所長抓了出來，硬生生讓她吐便當。
    • 44話中，リヨぐだ子用烏冬麵糰混合聖杯黑泥製作了從者的雛型，並將麵糰轉交給所長培養長大。
    • 48話中，培育的麵糰順利成長為Assassin的從者，一登場就拿槍指著所長。

《Fate/Grand Order -MOONLIGHT/LOSTROOM-》編輯

  • 出現在藤丸的夢境中，與他對話商談。
    • 途中發覺自己只是一時的殘影，感嘆自己應該早點與瑪修坦承相談，並在離開前鼓勵藤丸。離席後似乎被藤丸遺忘了。

《君主‧艾爾梅洛伊二世事件簿》編輯

  • 此作品中的所長年僅11歲，是FATE系列中登場時間軸最早的作品，劇情發生在《Fate/Grand Order》之前[4]。
    • 劇中所長發動大魔術時特地按照TM的命令追加了額頭浮現刻印的演出，類似FGO第二部中異星巫女前額的刻印。
  • 與家庭教師特麗莎一起乘上「魔眼蒐集列車」參加魔眼拍賣會，目標是藉由特麗莎的未來視預測到的，傳聞中的虹之魔眼。
    • 為了增加得標機率，與二世提議聯手。
  • 在目睹特麗莎被刎首後一度崩潰，甚至在醒來後搧了前來探望的艾爾梅洛伊二世一巴掌。但在二世的自白後願意協助他進行調查。

《Fate/Grand Order》編輯

  • 在序章烈焰污染都市 冬木登場，為48名適任御主講解這次的任務，因為主角打瞌睡而氣得將他趕出中央管制室。
  • 被捲入中央管制室的人為爆炸，之後被靈子轉移到冬木，為了回到迦勒底而和主角、瑪修、庫夫林一同行動。
  • 解決特異點的起因阿爾托莉亞Alter後，原以為終於能夠回到迦勒底，雷夫卻在這時候出現。
    • 雷夫表示奧爾嘉瑪麗的肉體早在管制室爆炸那時便已被炸毀，只有靈魂被轉移到冬木市來。
    • 奧爾嘉瑪麗只能活在這個特異點中，隨著特異點的崩壞，奧爾嘉瑪麗也將真正死去。
  • 雷夫表示應該讓她在死前摸摸自己的寶物，於是將奧爾嘉瑪麗整個人抓起來拋入地球環境模型‧迦勒底亞斯。
    • 奧爾嘉瑪麗在觸碰到迦勒底亞斯的同時，靈魂也被分解成分子層級，在感受無盡痛苦的同時悽慘死去。
    • 但實際上人體進入迦勒底亞斯內部後，會遭遇到什麼狀況還無人知曉，所以廣義來說是生死未卜。

第五異聞帶(下)—《星間都市山脈 奧林帕斯》編輯

  • 以Beast VII「終局」之獸的身份出現，但因為空想樹遭阿特拉斯抵擋和貝里的攻擊導致現在還是幼體狀態。
  • 自稱地球國家元首 U-奧爾嘉瑪麗
  • 雖然在立香等人面前否定並自己為「奧爾嘉瑪麗」這個個體，但自我介紹時自稱U-奧爾嘉瑪麗[5]
  • 依然遺傳所長的遜炮，而且笑聲很沒品
  • 基里修塔利亞的一擊對她沒有造成任何損傷，僅是因為「為什麽一隻蟲子敢攻擊我」和「空想樹也無法消去的光之槍」的驚訝而暫時退回觀察。
  • 似乎打算到南美特異點，找尋替代的身體。
  • 雖然也有玩家認為多半會是被ORT虐到哭奔的遜炮結局
  • 身上的黑金色條紋和遊星從者十分相似。
  • 福爾摩斯推測會以奧爾嘉瑪麗的外貌登場，可能是類似當初蓋提亞利用所羅門屍體的原理。
  • 但立香和瑪修卻不由得感覺「地球國家元首」就是奧爾嘉瑪麗本人。

《非靈長生存圈 通古斯聖域》編輯

  • 前往彷徨海摧毀迦勒底，然而在那之前迦勒底已搭上Storm Border逃離。

能力編輯

  • 魔術師應有的能力和知識一應俱全，但就是沒有御主適性和靈子轉移適性，本人也很在意這點。
    • 庫夫林評價她身上的魔術迴路「質和量都是一流的」，對這樣的她沒有御主資質一事感到訝異與不解。
  • 和遠坂凛一樣，能夠使用Gandr，勉強還能應付骷髏等魔物。
    • 二世事件簿中，即使在是不擅長魔術的格蕾眼中看來，她的戰鬥能力應該在不使用特里姆瑪烏的萊妮絲之上。
      • 於魔眼蒐集列車事件中曾使用過能降下隕石的大魔術用來迎擊腑海林之子。
    • 是藉助天之上（宇宙）的魔力對銀河系內天體進行干涉的魔術，在神秘已經消退的現代需要有很多前置條件才有辦法達成[6]，但是如果是在充滿神祕的神代的話……
  • 二世事件簿裡，身為天體科君主繼承者的她雖然年幼，但仍有遠超年齡的政治手腕與洞察分析能力。
    • 不過因為是純粹的魔術師家庭出身，所以無法理解二世那非魔術師部分的思考方式。
  • 在《用漫畫了解FGO！》的世界中依然沒有御主適性。
    • 她所役使的用漫畫瞭解的Assassin是リヨぐだ子硬塞給她的，不是用正規方法召喚出的Servant。
      • 到底為什麼會有烏冬麵糰這種莫名其妙的召喚方法？
    • 似乎沒有正式的主從關係，連令咒都是自己用筆在手上畫的。
      • 令咒的圖案是「^o^」
  • 在《MOONLIGHT/LOSTROOM》的世界中擁有御主適性。
    • 使役的七名從者（鈴鹿御前、Emiya〔Alter〕、李書文、女王梅芙、梅菲斯特費勒斯、卡蜜拉、庫夫林〔Alter〕）全都是屬性惡的Servant。

簡評、其他資料編輯

 
  • 迦勒底的所長與負責人，也是本次特殊作戰的司令官。
    • 家系「亞寧姆史菲亞」是代代擔任時鐘塔天體科君主的魔術師名門，奧爾嘉瑪麗也在父親過世後接下了天體科[7]當家的位置。
  • 與外表的強勢態度不同，內在其實缺乏自信。為了證明自己的價值而努力，在父親死後仍勉力撐持迦勒底。
    • 但在父親死亡後才得知迦勒底暗地裡進行的非人道實驗，打擊她心目中對父親的理想形象。因此而厭食了一個月，才在幾乎精神崩潰的邊緣下接任職位。
    • 內心恐懼著在實驗中倖存的瑪修，認為她總有一天會向自己復仇。但之後在羅曼的建議下讓瑪修成為迦勒底的員工。
      • 瑪修能在迦勒底之中自由活動可以說是拜奧爾嘉瑪麗所賜，因為即便內心害怕瑪修，性格認真的她也執意做出正確的事。
  • 二世事件簿（時間點為2004年）中登場的奧爾嘉瑪麗是11歲，從這點可以推斷出在FGO序章（時間點為2015年）中登場的奧爾嘉瑪麗的年齡是22歲左右。
  • 在迦勒底沒有任何朋友，除了雷夫以外沒有別的人可以讓她依靠。
    • 雖是不得人緣，但在人理燒卻事件結束一年之後，也有些員工開始認為做為一個剛從學校畢業又失去父親的年輕人而言，她當時的表現已經是很難得的了。替她為迦勒底面臨要被收購的結果感到不甘。
    • 但也有人認為她的能力還不足以當任所長是無可辯駁的事實，即便她還活著也會接受迦勒底被改組的命運吧。
    • 在漫畫《FGOフロムロストベルト》中提及，由於承擔了與自身能力不足的所長職位、被「那位天才的亞寧姆史菲亞的女兒」這身份束縛、周遭都是某方面的天才和專家，所以她給予了自己極大的壓力，亦非常自我厭惡，更甚是把別人的關心腦補成對自己的惡意，最終把自己困在孤立無援的思考。
    • 即使在另一個世界線的二世事件簿也是沒有朋友的狀態，不過在初登場的事件後與萊妮絲有了一定的交情與友誼。
  • 登場沒多久便迅速吃便當下場，引起不少玩家的震撼。
    • 因為便當領得太突然，引起玩家的同情和憐憫，有部分粉絲/觀眾希望她能吐便當復活。
    • 劇情進入第二部之後，這類呼聲也變得越來越多。
  • 在各改編作品中的登場紀錄便當史如下↓
    • 《Fate/Grand Order》：在序章便被炸死，靈魂也被迦勒底亞斯分解成分子層級。
    • 《F/GO-First Order-》：改編自遊戲序章的動畫。如同遊戲劇情，在尾聲悽慘地留下遺言聲優演技注目後退場。
    • 《F/GO-mortalis:stella-》：改編自遊戲序章的漫畫。如同遊戲劇情，在尾聲被悽慘地分解成分子層級死去。
    • 《F/GO-turas réalta-》：改編自遊戲序章的漫畫。如同遊戲劇情，在尾聲被悽慘地(ry
    • 《F/GO-MOONLIGHT/LOSTROOM-》：前半段佔有大量戲份，卻在最後面告訴你這一切只是藤丸立香的夢，夢醒後再度回到所長死去的現實。
      • 所長又死了！！
      • 保安！可以讓所長這樣死了又死死了又死的嗎！

      • 三不五時被性騷擾，還會不斷重播年末特番給所長看
    • 《君主‧艾爾梅洛伊二世事件簿》：由於是迦勒底未能成立的另一條世界線，照理來說是不會像GO那樣慘死了，但本作的奧爾嘉瑪麗年紀還小，之後會怎麼樣還很難說
    • 《型月學園》：輕鬆向作品，不會有人領便當的溫柔世界。
      • 所屬高中部，對不符年齡的制服裝扮感到有些羞恥。戲份並不多。
    • 《FGO QUEST》：可使用角色三人組之一，火力擔當。
  • 雖然在FGO本傳中早早就退場了，卻因為角色塑造夠讓人印象深刻，再加上年末特番和官方四格的幫助之下，至今仍有不少存在感。
    • 尤其是在《用漫畫了解FGO！》中非常活躍，完全可以說是作品的主要角色。
  • 其實有教師屬性，似乎對教學頗有心得。
    • FGO的概念禮裝「個人訓練」的說明文中提到，所長的教學雖然很斯巴達式，但教導卻十分認真仔細，對成績不好的學生也能耐心對待。
      • 唯獨要注意別遲到，對於不守時的學生，所長肯定會大發雷霆。
    • 在《用漫畫了解FGO！》中，所長也負責擔任向讀者講解遊戲系統的角色。雖然教學總是會被リヨぐだ子打斷
  • 進入第二部後，伴隨各種特番與第二部本篇劇情，還有事件簿提到的蛛絲馬跡，很多人都覺得所長乃至整個亞寧姆史菲亞家族說不定背後大有問題。
    • 並不是所長本人有什麼問題，而是其父馬利斯比利對她的態度很明顯不像父親對女兒的樣子。
      • 另外所長有金色的虹膜[8]，可能並非純正的人類；結合首章中庫夫林的診斷結果，她很可能有著相當的特殊性。
    • 事件簿動畫中，所長曾提及父親對她是突然就失去了關注，像是放棄她一樣，讓她從此陷入恐慌。
    • 更有一部分玩家透過MOONLIGHT/LOSTROOM中立香離開LOSTROOM後，異星巫女隨即出現，以及在第二部OP中異星巫女朝著GD伸手似乎想講述什麼這幾點，猜測所長很可能在被迦勒底亞斯分解後變成了異星巫女。然而她成為了終局之惡-異星之神。
      • 更有玩家基於此點推測老所長很可能打從一開始就想把自己女兒推下去當活祭品來實現他的計劃，或者將特殊個體奧爾嘉瑪麗這一計劃中的關鍵要素以女兒的名義掌握在自己手裏[9]，所以才會在確信迦勒底無法在他有生之年完成之後就放棄了所長。
      • 2.5.2章末尾，自稱"地球國家元首 U-奧爾嘉瑪麗"的異星神的登場強化了這一觀點

    • 在烈焰污染都市 冬木中與主角會合之前，受到骸骨們攻擊時的慘叫。


  • 「バカ言わないで！死んでさえいなければ後でいくらでも弁明できるからに決まってるでしょう！？
    　だいたい47人分の命なんて、私に背負えるハズがないじゃない……！
    　死なないでよ、たのむから……！……ああもう、こんな時レフがいてくれたら……！    」
    「別說傻話了！只要沒死以後總有辦法解釋的吧！？
    　再說了，47條人命，這我哪背得了啊……！
    　不要死啊，拜託了……！……啊啊真是夠了，這種時候要是雷夫在的話……！」
    • 面對47名御主適任者的瀕死，奧爾嘉瑪麗當機立斷的要羅馬尼趕緊將他們冷凍保存。
    • 瑪修欽佩地表示「未經本人同意就進行冷凍保存可是犯罪行為。您比起責任更看重人命呢。」奧爾嘉瑪麗則回答了上述那段話。
    • 表現出奧爾嘉瑪麗人性化的一面，以及作為所長尚不成熟的一面。


  • 「いや―――いや、いや、助けて、誰か助けて！ わた、わたし、こんなところで死にたくない！
    　だってまだ褒められてない……！ 誰も、わたしを認めてくれていないじゃない……！
    　どうして！？ どうしてこんなコトばっかりなの！？
    　誰もわたしを評価してくれなかった！ みんなわたしを嫌っていた！
    　やだ、やめて、いやいやいやいやいやいやいや……！ だってまだ何もしていない！
    　生まれてからずっと、ただの一度も、誰にも認めてもらえなかったのに―――！    」
    「不要―――不要，不要，救救我，誰來救救我！
    　我、我還，不想死在這種地方！
    　還沒有人誇獎我……！還沒有認可我啊……！
    　為什麼！？為什麼總是這樣！？
    　沒有人願意對我表示肯定！大家都討厭我！
    　討厭，住手，不要不要不要不要不要不要不要……！我什麼都還沒做呢！
    　出生至今，一次都沒有，一次都沒有得到過任何人的認可―――！」
    • 死前最後的遺言，在被拋入迦勒底亞斯的過程中，奧爾嘉瑪麗不斷地淒慘哭喊。


  • 「あなたには失望したわ。あなたにはもうなにも期待しないから。」
    「我對妳很失望。已經不會再對妳有什麼期待了。」
    • 在《用漫畫了解FGO！》第三部93話中，奧爾嘉瑪麗為了讓Assassin聽話，故意表現出對她很失望的態度。
      • 結果這招對Assassin非常有效，Assassin馬上就變得服服貼貼。
    • 這篇漫畫出來時恰逢2019年5月的超長長草期[10]，奧爾嘉瑪麗的這段臺詞意外符合眾人的心聲，被日本網友們當成梗圖大量使用。
      • 當成梗圖使用時，「あなた」指的便是一直不出新主線或新活動的FGO營運。
