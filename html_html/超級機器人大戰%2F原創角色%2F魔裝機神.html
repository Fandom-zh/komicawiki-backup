魔裝機神 THE LORD OF ELEMENTAL編輯

  • 安藤正樹/マサキ・アンドー(CV:綠川光)
    • 風魔裝機神塞巴斯塔(サイバスター)的駕駛，日本人。
    • 在拉・基亞斯正式的名字為由國王賜予的聖號《ランドール・ザン・ゼノサキス》（ランドール・ゼノサキス是第一代劍皇之名）。
    • 機戰系列原創班底之一[1]，聲優綠川光乃機戰聲優班底之黑幕首席，人氣極高。
      • 綠川光曾經在魔裝機神黑暗期抗議過正樹出場太少
      • α外傳中希洛唯找迷路的安藤正樹非常有效率，不虧是綠川光自己找自己
    • 使魔(ファミリア)為兩隻貓小黑(クロ)和小白(シロ)，搭乘時負責塞巴斯塔的雷達管理。
      • 當初兩隻貓對正樹的命名很不滿意，還是普蕾西亞用鈴鐺和緞帶收買才勉強接受。
    • 超級路癡，轉了地球七圈也去不了日本（元祖機戰劇情）。
      • 後來在F時強化成10圈，α時再強化為20圈。紀錄更新中？
    • 原創班底中最有錢的人。
      • 在魔裝機神LOE第一章的御前試合中獲得優勝，獲得的賞金被普雷西亞拿去保管、投資，換算成日幣約有70億。
    • 原創系列的主角中權力與社會地位最高的人[2]，也是少數有處理外交與政治的劇情的主角。
    • 因為父母被捲入恐怖攻擊事件中身亡，所以對恐怖份子十分的痛恨。
    • 重度妹控，對於義妹プレシア的事情十分緊張
    • 為了在Z出場將自已的名字調轉來用
    • 小時候曾化名為狩屋正樹而轉學到雷門中學踢足球
    • 專用BGM為《熱風！疾風！サイバスター》，角色單曲為BGM改編的同名歌曲
      • α外傳為ReMIX ZERO版（主唱：影山ヒロノブ、水木一郎）
      • OG/OG SAGA為無記版（主唱：JAM Project）
    • 精靈憑依專用BGM為《ポゼッション》
    • 曾在「英雄戰記」客串過
    • 男性角色持有最多數目後宮者
    • 傳說只要是魔裝機神的駕駛者感情路一定死很慘（某些路線要把ウェンディ打倒）
  • 溫蒂·拉斯姆·英格耐德/ウェンディ・ラスム・イクナート(CV:大原さやか)
    • 魔裝機計畫的實行者，將高等精靈降臨於機體上的術式就是她完成的
    • 賽巴斯達與ガルガード的製作者
    • 被雙子姊姊的靈魂寄宿著所以是雙重人格
    • 正樹有送過奧里哈根製的戒指在地底世界那是有求婚定情的含意，雖然送的人沒那意思給她
    • 強吻過正樹也被正樹看過裸體
    • 正樹的大老婆
    • 傳統機戰殺人料理人之一
  • テューディ・ラスム・イクナート(CV:大原さやか)
    • ウェンディ的雙胞胎姐姐，在未出生前便死去
      • 雖然肉體已死，靈魂卻仍然存在，與ウェンディ共享着知識與感情
    • 在ラングラン王都毀滅後，乘着妹妹重傷的時候佔用其身體
    • 擁有優秀的頭腦，單論才能還在妹妹ウェンディ之上，能獨自完成第17台魔裝機「イスマイル」，以及只看一眼「デュラクシール」的設計圖便能完整地製造了2台
    • 對於自己不被世人所知曉而一度憎恨世界，另一方面卻對愛情十分渴求
    • プラーナ數一數二的強大，所以能夠駕駛魔裝機
    • 在魔裝機神2中，與妹妹ウェンディ達成和解
      • 在魔裝機神3中，偶爾與ウェンディ交換意識，並代替ウェンディ把她藏在心裏的話說出來
    • 個性腹黑，卻對於正樹的稱讚的說話顯得動搖
    • 專用機為「イスマイル」、「ネーゼリア」(《魔裝機神F》時)
  • 琉妮·佐爾達克/リューネ・ゾルダーク(CV:日高奈留美)
    • DC總帥比安·佐爾達克的女兒，怪力少女，興趣是存錢和時代劇[3]，自稱正樹的戀人。
    • 原先找上主角群要為老爸報仇，但被正樹不經意稱讚她很可愛後對正樹萌生好感[4]，之後以"跟你們在一起好像很有趣"的理由加入隊伍(比安淚目)。
    • 比安預定給她的機體是瓦爾希翁，可是琉妮覺得外型很遜，比安只好另外開發瓦爾希歐妮，瓦爾希翁留著自己開(比安再淚目)。
      • 其實是本人靠父親送的腰帶變身為巨大化機娘。
    • 地上人召喚事件後留在拉・基亞斯繼續攻略正樹，並且利用拉・基亞斯的科技把座機萌化升級到瓦爾希歐妮R。
    • 結識溫蒂並確認其對正樹也有好感後，約定兩人公平競爭(攻略)正樹。
    • 正樹的二老婆
    • 專用BGM為《フラッパーガール(Flapper Girl)》，角色單曲為BGM改編的《白き鋼の天使 ヴァルシオーネＲ》（主唱：日高奈留美）
  • 迪蒂・諾瓦克/テュッティ・ノールバック (CV：井上17歲喜久子)
    • 水魔裝機神加迪斯(ガッデス)的駕駛，芬蘭人，金髮碧眼的北歐系御姊。
    • 身為水魔裝機神的駕駛員卻不會游泳而被貴家澪取笑，迪蒂則以"正樹(風魔裝機神駕駛)也不會飛啊"來反駁。
    • 情路坎坷，男友理卡路德為她擋子彈而死，懷有好感的菲爾王子也在EX殞命。
    • 甜食派，一杯紅茶加五匙糖還嫌不夠甜，吃壽喜燒的時候也加了一堆糖進去，讓理卡路德吃到流鼻血。
    • 由於一度以蓓爾丹蒂之名當過人類的守護女神，所以使魔為兩隻狼弗雷奇(フレキ)和格里(ゲリ)。[5]
      • 當初迪蒂看到正樹的使魔是兩隻貓的時候，曾說要是自己的使魔也是貓就好了，讓兩隻狼受到了不小的打擊。
    • 來到地底世界的神聖蘭格蘭王國的時間很早。
    • 專用BGM為《水と沼の国から》，角色單曲為BGM改編的《CALM CRYSTAL BLUE》（主唱：井上喜久子）
    • 精靈憑依專用BGM為《決意の大海》
    • 傳說只要是魔裝機神的駕駛者感情路一定死很慘（戀人幫自己擋子彈而死）
  • 黃炎龍/ホワン・ヤンロン(CV：井上和彥)
    • 炎魔裝機神格蘭威爾(グランヴェール)的駕駛，中國人，原本的職業是真憂鬱的體育老師。
    • 特技是拳法和說教(還可以和使魔合體攻擊交互說教)，以及在對話中使用難懂的諺語。
      • 亂入王[6]、經驗值小偷
      • 其實都是中國古代的諺語或成語，只是翻成日文而已，其他人聽不懂也是很正常的。
    • 四位魔裝機神駕駛中最具有常識的人。
    • 對莫妮卡公主抱有好感，但是莫妮卡喜歡的卻是白河愁所以總有一天會NTR回去。
      • 但是跟紅蓮的沙菲妮卻很有緣，曾經私下交換情報和特訓。
    • 在第二部真心喜歡上エルシーネ
    • 使魔是風生獸(海內十洲記中記載的一種神獸，外形類似豹)ランシャオ。
      • 被琉妮吐槽很像巴比倫二世的羅迪姆，之後配音也找羅迪姆的聲優來配，看來官方也是很故意。
    • 專用BGM為《炎の中華体育教師》，角色單曲為BGM改編的《爆焰的紅蓮鬥志！》（主唱：井上和彥，OS：野田圭一）
    • 精靈憑依專用BGM為《炎の覚醒》
    • 傳說只要是魔裝機神的駕駛者感情路一定死很慘（親手打倒自己喜歡的人）
  • 貴家 澪/ミオ・サスガ(CV：かないみか)
    • 地魔装機神薩姆吉德(ザムジード)的現任駕駛，日本人。
    • 個子小兼蘿莉身材，常被取笑(或誤認)是小學生，其實已經是高中生了。
    • 個性看起來不太正經，劇情中也多負責搞笑或吐槽，其實心理層面相當成熟。
    • 興趣是漫才(類似對口相聲)，魔裝機神某路線結局還把ゲンナジー抓來當搭檔。
    • 雖然年紀輕卻知道不少古老的捏他，本身還是合氣道達人。
      • 像是把超振動拳當北斗百烈拳喊，還有惡搞旗本退屈男的諸羽流青眼崩し。
    • 使魔為三隻鴨嘴獸ジュン、チョーサク、ショージ，稱呼澪為師匠。
      • 重製版因為大人的理由改名為ゴクー、ハッカイ、ゴジョー。
    • 蒂法‧阿迪爾
    • 專用BGM為《正調 ミオのじょんがら節》，角色單曲為BGM改編的《新調 ミオのじょんがら節-G》（主唱：かないみか）
    • 精靈憑依專用BGM為《目覚める大地》
  • 理卡路德・席維拉/リカルド・シルベイラ（CV：小山力也）
    • 地魔装機神薩姆吉德(ザムジード)的初任駕駛，巴西人，原本的職業是空軍的飛行員。
    • 頭像看起來還算年輕，其實已經36歲，是個貨真價實的大叔。
    • 雖然身為魔裝機神駕駛員，但在遊戲第一章就為了替迪蒂擋子彈而領便當。
    • 使魔在遊戲中沒有立繪，只在對話中提過是三隻猴子。
      • 名字到重製版的戰鬥台詞中才出現，分別是アメデオ、メルクル和ジョリクール。(來自世界名作劇場的角色名)
    • 過去因為被貓抓過重要部位，所以特別怕貓。因此非常不擅長應付正樹的使魔小白與小黑。
    • 傳說只要是魔裝機神的駕駛者感情路一定死很慘（幫戀人擋子彈而死）
  • 普雷西亞・傑諾薩奇斯/プレシア・ゼノサキス(CV：こおろぎさとみ)
    • 劍皇ゼオルート的女兒，正樹的義妹，現為魔裝機迪亞布羅(ディアブロ)的駕駛。
    • 把射擊機迪亞布羅當成近戰機在開，大概是受到劍皇老爸的影響。
    • 生活技能滿分，因為父親除了劍術外完全沒有生活能力，所以自力學會了應付各種家務，還能從正樹的描述中用拉．基亞斯的食材做出日本料理。
    • 有許多一般駕駛沒有的特權，可說相當受到蘿莉控製作小組的喜愛。
      • 例如其他魔裝機駕駛都沒有的專用BGM，優遇的專屬必殺技，且在α外傳是唯一武裝(くるみ割り人形)有兩段cut-in的角色。
    • 在α外傳的cut-in被許多玩家認為沒有穿內褲。
 
    • 專用BGM為《春風のプレシア》
  • 榭妮亞・格拉尼亞・畢爾榭亞/セニア・グラニア・ビルセイア(CV：皆口裕子)
    • 神聖蘭格蘭王國公主，莫妮卡的雙胞胎姐姐。
    • 由於魔力較弱無法通過測試，故沒有王位繼承權。
    • 乘機為ノルス(魔裝機神LOE第一章)、ノルス・レイ(魔裝機神LOE第二章以降)
    • 你忘騎士是能娶兩位老婆的嗎正樹身邊麻煩事製造者
    • 鋼彈魔裝機デュラクシール及超級電腦デュカキス的開發者，有很高的機械工程天份，蘿蔔控。
    • 曾任職於情報局，並透過デュカキス使用非合法的手段收集情報。
      • 在魔裝機神二更以自己的名字成立セニア機關，根本就是情報頭子。
    • 除了機械和情報，還擅長駭入電腦、打開寶箱鎖，就算魔裝機神改成RPG也還是能當盜賊。
    • 會製作殺人料理。
    • 專用BGM為《情熱のプリンセス》(魔裝機神II追加)
  • ゼオルート・ザン・ゼノサキス(CV:田中秀幸)
    • 劍神ランドール・ゼノサキス的後代，同時是ゼノサキス東宗家的當家。軍階為上校。
    • プレシア的父親、正樹的義父、ファング的師父
    • 擁有「劍皇」的稱號，與「劍聖」シュメル・ヒュール齊名的劍術高手
    • 現時與妻子ピア離婚，持有女兒プレシア撫養權
    • 對於女兒プレシア採取放任主義的教育方針，因此使到プレシア擅長家務以及經營
    • 白河愁駕駛「グランゾン」襲擊王都時，由於其魔裝機「ギオラスト」與「グランゾン」性能相差甚遠而落敗。最終由於逃生裝置失靈，未能逃生而被捲入機體爆炸而喪生。
      • 雖然居於下風，卻成功破壞了「グランゾン」形成「歪曲フィールド」的手腕
    • 專用機為「ギオラスト」
  • 方克/ファング・ザン・ビシアス(CV：中村悠一)
    • 神聖蘭格蘭王國近衛騎士團的一員，劍皇ゼオルート的的弟子、正樹的師兄。後加入安提拉斯隊
      • 身為地底世界人，卻有著不輸給地上人的Prana量，在已知的地底世界出身角色中僅次於テューディ和愁
    • 一開始很討厭正樹，但在老師的提議下和他進行決鬥，不論勝敗最後成了互相競爭的勁敵關係
    • 非常敬重菲爾王子，因此在知道菲爾王子被正樹打倒後跑去バゴニア當傭兵，並在之後多次與正樹對決
      • 直到明白菲爾王子死亡的真相後深感自己玷汙了王子的名譽，但在正樹的怒斥下重新振作，並回歸蘭格蘭王國
    • 專用機為「ギオリアス」，追加裝甲為「ギオリアス・ロイ」
    • 專用BGM為「孤高の牙」
  • ゲンナジー・I・コズイレフ(CV：石塚運昇)
    • 俄羅斯人，本來是奧林匹克游泳金牌選手。
    • 繼正樹後的第二代ジャオーム駕駛員。
    • 論力氣稱的上是男性第一，但依然被琉妮打敗
    • 無口，而且存在感很薄弱。
      • 澪稱他是「會走路的光學迷彩」。
  • アハマド・ハムディ(CV：玄田哲章)
    • 少數在EX就已經登場的魔裝機駕駛。
      • 不過除了愁的路線，其他路線都是以敵方身分登場。
    • 阿拉伯人，ソルガディ的駕駛。
      • 由於出身動亂地帶，因此堅信著「力量就是正義」，因此多數時間都在各地流浪尋找對手並與之對戰。
    • DS版『LOE』只要達成特殊條件，アハマド加入同時可獲得五百萬資金(玩家們都拜アハマド為神)
  • シモーヌ・キュリアン(CV：園崎未恵)
    • 法國人，ザイン的駕駛。
    • 從小是孤兒，一生幾乎是獨立自強
    • 有喝酒的習慣，非常討厭德國人
    • 在魔裝機神Ｉ第一章、王都蘭格蘭被恐怖襲擊毀滅的同時，與シュテドニアス軍交戰後因寡不敵眾而被捕，之後成了シュテドニアス軍的俘虜
      • 不過受到ラセツ的庇護，沒有遭到虐待，還因為對方紳士的舉動而為他傾心，雙方更發展成男女朋友的關係，因此在Ｉ的第二章開始的地底世界三大國相互的對戰中在同伴正樹等人與ラセツ之間一度搖擺不定
      • 最後在正樹說起自己敬愛的菲爾王子被和對王子有好感的迪蒂親手打倒後，終於放下對感情的執著
    • 專用BGM為「移り気パリジェンヌ」
