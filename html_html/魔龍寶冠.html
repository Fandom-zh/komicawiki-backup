簡介[編輯]

  • 由Vanillaware開發，基於《地獄神龍系列》概念的角色扮演遊戲。
  • 與《地獄神龍》同樣使用TRPG的架構，由敘事人引導進入劇情。
    • 預設敘事人的配音為已故聲優谷口節，藉由付費下載可以解鎖由主角群擔任配音的敘事人。

系統[編輯]

  • 撿骨
    • 在地圖上會出現各種過去陣亡的冒險者骸骨，可以領到教會復活成為同伴或是埋葬。
    • 復活後的角色會在酒館待機，玩家可以選擇是否帶他入隊。
    • 同伴是可以拋棄的
  • 裝備道具
    • 冒險途中獲得的寶物經過鑑定完成後可以做為裝備道具使用。
    • 裝備道具皆有耐久值設定，一旦損壞其效能便會打折扣。
    • 玩家可以在道具店維修裝備，但只限主角的裝備可以維修。
    • 裝備也是可以拋棄的
  • 關卡分支
    • 與《地獄神龍》類似，關卡中盤會出現分支選項。
    • 其中A路線為序盤強制發生的路線，後期出現的B路線為取得護身符必經關卡。
      • 必須在限制時間和條件內擊敗B路線頭目才能取得護身符。
      • 請不要用游標調戲對方。
  • 野營
    • 通過特定關卡後進行分支B路線時才會開啟的要素，本作主要的特色飯テロ
    • 要在有限時間內組合食材、調味料作出各種料理
      • 除了一般常見食材外，還有來自怪物的稀有食材吃下那頭龍，他富含蛋白質
      • 迷宮飯的起源
    • 玩家吃下料理後在後半關卡可獲得各種不同能力加成（如攻防上升、HP上升）
      • 料理能力加成可疊加，吃的越多就可獲得更多強化；另調理時間越短可獲得的加成也會提高

世界觀[編輯]

  • 海德蘭
    • 本作舞台。
  • 波爾加
    • 北方的山地王國，與海德蘭為敵對關係。
  • 女神信仰
    • 海德蘭王國以三女神崇拜為中心的信仰。
    • 三女神在過去為封印上古巨龍而將力量宿於九個護身符中。
    • 每突破一個難度可以復原一尊神像，所有難度都破關後可以解鎖三女神畫像。
  • 摩尼翁
    • 使用黑暗魔法的邪教組織。

登場人物[編輯]

主要人物[編輯]

  • 戰士 / Fighter（CV：津田健次郎）
    • 擅長高速劍擊可是圖裡拿的是斧頭，會用盾牌防禦。

  • 亞馬遜女戰士 / Amazon（CV：田中敦子）
    • 揮舞雙手斧的強悍女戰士，每次擊中敵人都會提升攻擊速度。

  • 矮人 / Dwarf（CV：石塚運昇）
    • 雙持鐵鎚可是圖裡拿的是斧頭和強韌肉體是他最大的武器，可以摔擲敵人。

  • 精靈 / Elf（CV：今井麻美）
    • 身著輕裝的弓術高手，所有能力平均成長。

  • 女法師 / Surceress（CV：井上喜久子）
    • 法力高強的女巫，魔法防禦高且擅長輔助魔法。

  • 巫師 / Wizard（CV：安元洋貴）
    • 以高魔力見長，能夠靈活運用各種攻擊魔法的男巫。

次要人物[編輯]

  • 蘭尼
    • 身穿綠色斗篷[1]，精通開鎖技巧的盜賊。
  • 媞奇
    • 因為好奇心而離開森林的小妖精，路肯的知己好友。
    • 酒品不佳[2]

海德蘭王國相關人物[編輯]

  • 迦南神殿僧侶
  • 摩根·麗詩黎
    • 經營魔法道具小舖的女巫。
    • 曾用蜘蛛絲編成絕對不會破損的斗篷，該斗篷目前仍是各路冒險者爭奪的稀世珍寶。
  • 路肯
    • 住在高塔研究符文的魔法師，前往拜訪華萊士後便下落不明。
    • 主角後來在地下迷宮發現其遺骸，後被媞奇帶至迦南神殿復活後回到高塔繼續未完的研究，並會販賣符文給主角。
  • 撒慕爾·約瑟夫
    • 工會會長，負責給予主角委託與技能訓練。
    • 過去曾被波爾加俘虜並被迫成為角鬥士，之後找到機會逃至海德蘭。
  • 薇薇安公主
    • 前代國王的女兒。
    • 結局在主角的協助下繼承已故的父王與王叔登基為王，並封主角為海德蘭的終身盟友。
  • 古斯塔夫首相
    • 服侍公主的宰相，委託主角奪回王室權杖。
    • 真正目的為讓波爾加的馬克尼大公登基而利用項鍊控制公主令狄恩不便出手。
    • 事跡敗露後攜帶王權寶球企圖逃到波爾加卻遭海盜襲擊，就此下落不明。
  • 狄恩伯爵
    • 前代國王的弟弟，為了奪取王位而雇用托密特偷走權杖。
    • 實際上是為了阻止馬克尼登基而出此下策。
    • 在古斯塔夫的陰謀被阻止之後登基為王，不過被魔龍寶冠誘惑的他遭摩尼翁教團欺騙而成為喚醒巨龍的祭品。
  • 神秘老人
    • 出沒在神殿小巷的老乞丐。
    • 真實身分為魔法師華萊士。

迷宮相關人物[編輯]

  • 蠻族戰士羅蘭
    • 隸屬公會的蠻族戰士，為了尋找失蹤冒險者的屍骸而前往神殿廢墟。
  • 神偷托密特
    • 偷走王室權杖的大盜。
    • 會在其他地圖作為小偷的上位互換出場偷開寶箱。
  • 魔法師的學徒里奇
    • 華萊士的老鼠學徒。
    • 過去曾是馬戲團演員，曾經利用魔法操控掃把幫牠提水。
  • 「大廚」大哥布林
    • 隸屬獸人軍團的廚娘，視力不佳。
  • 騎士幽魂
    • 手持卷軸，身穿鎧甲的亡魂。
    • 真實身分為海德蘭前代國王，為了不被當作祭品而自刎。
  • 在淺水區游泳的美人魚
    • 性格好奇的人魚。
  • 女武僧
    • 左腳受傷不良於行的武僧，委託主角使用神聖封印淨化祭壇。
  • 受縛的精靈
    • 被摩尼翁教團捕捉的精靈。
  • 森林隱士魔法師
    • 研究上古巨龍的隱士，與路肯水火不容。

迷宮頭目[編輯]

  • 鳥身女妖
    • 等級：1→17→40→70
    • 擁有女性容貌的怪物，性格凶暴。
    • 幼鳥很可愛，有人試圖飼養幼鳥進行馴化但反被吃掉，人外控表示遺憾。
    • 曾有一名騎士被對方的美色誘惑並打算親近，最後遭其殘殺而亡。
  • 雙足飛龍
    • 等級：3→17→40→70
    • 體形較大的綠色是雄性，灰褐色是雌性；強大的雄性個體會吸引多數雌性跟隨。
  • 末日甲蟲
    • 等級：5→17→41→71
    • 華萊士於地下迷宮培養菌菇時因為菌菇感染而突變的變種昆蟲。
    • 幼蟲對冒險者而言倒是不錯的蛋白質來源。
  • 牛頭怪
    • 等級：6→17→41→71
  • 吸血鬼
    • 等級：8→17→42→72
    • 於村落捕捉少女並將其同化的怪物。
    • 村長亦曾委託獵人企圖討伐之。
  • 海盜
    • 等級：10→17→42→72
    • 擁有能夠召喚魔神的神燈。
  • 魔像
    • 等級：12→17→43→73
    • 玩家可造成傷害極低，需引導我方魔像對其攻擊才能打倒。
    • 我方魔像被破壞的話，可以打倒拿炸彈的哥布林，利用炸彈給予有效傷害。
  • 術士
    • 等級：15→17→44→74
    • 摩尼翁教團所屬成員，擅長使用各種屬性的魔法。
  • 凝視怪
    • 等級：17→45→75
  • 梅杜莎
    • 等級：18→50→80
  • 獨眼巨人
    • 等級：20→51→81
    • 被三女神擊敗而被關在地下牢房的獨眼巨人。
  • 石像鬼之門
    • 等級：22→53→83
    • 由哥布林工匠建造的巨大閘門。
    • 以難攻不落著稱，只有砲擊能給予強大損傷。
  • 殺人兔
    • 等級：23→54→84
    • 棲息於海德蘭與波爾加國界附近的兔子，已經殺害多數波爾加的勇士。
    • 波爾加甚至有公爵企圖利用該兔子借刀殺人。
  • 怨靈
    • 等級：25→55→85
    • 死亡的象徵，其弱點夢之心必須藉由篝火照亮女神雕像才會顯現。
    • 被反對復活的派系視為天使一樣的存在。
  • 海怪
    • 等級：26→56→86
    • 襲擊船隻的巨大海怪。
    • 相傳安娜·邦尼僅靠一把匕首就能殺死海怪，不過至今仍無人證明其可行性。
  • 奇美拉
    • 等級：27→58→88
    • 摩尼翁教團暗自製造的人造生命體，擁有獅子、羊與蛇的頭。
  • 高階惡魔
    • 等級：28→59→89
    • 惡魔之王的屬下。
  • 紅龍
    • 等級：30→60→90
    • 守護寶物庫的巨龍，據說是人類所知最強大的生物。
    • 過去曾有某國公主以自身當作祭品獻予紅龍的事蹟，至今已經成為某種宗教信仰。
  • 上古巨龍
    • 等級：35→65→95
    • 本作最終頭目，過去被女神封印的巨龍。
  • 魔侯
    • 等級：100
    • 本作隱藏頭目，被封印的惡魔之王。
    • 在遺忘聖所會以被封印的狀態出現，但直到混沌迷宮的最下層才會與他正式交手。
