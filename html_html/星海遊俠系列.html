解說[編輯]

  • 由tri-Ace開發，Enix發售的角色扮演遊戲系列
    • tri-Ace社長為前《幻想傳奇》的製作人五反田義治，因此在整體風格上與《傳奇》系列類似
  • 因為受到科幻影集《星際爭霸戰》的啟發，歷代皆以異星探索的科幻未來為主軸，並以西元後的宇宙曆紀年

系統[編輯]

戰鬥系統[編輯]

  • 本系列作品的戰鬥模式以2D平面為主
  • 進入戰鬥後可以操作主角與敵人在平面上即時戰鬥，並可以自由施展攻擊與發動技能
  • AI同伴的行動則可以在戰鬥前設定或是在戰鬥時選擇操控

同伴系統[編輯]

  • 在同伴系統上，本系列在挑選同伴上擁有相當的自由度
  • 到前三代為止的同伴人數最高八人，除了數名同伴綁定之外其餘同伴可以依劇情進行取捨
    • 部分同伴間也有可能有綁定或互斥的情形，基本上只靠一輪無法完全收集齊全部同伴
  • 參加的同伴也會影響下述特定PA的發生與否，有些甚至影響結局發展

PA（Private Action）[編輯]

  • 除了本篇劇情之外，劇情還設置了不少與主線無關的PA劇情
  • 與《傳奇系列》的Skits稍微不同，PA主要發生在各個城鎮，並會安排同伴在城鎮各地引發劇情
  • PA劇情類型包羅萬象，從認真型到搞笑向都具備，某種意義上算是本作最大特色之一

IC（Item Creation）[編輯]

  • 利用各種技術製作道具的系統
  • 於各地的技能屋購買技能後可以消耗SP提升製作道具的能力

遊戲列表[編輯]

本篇[編輯]

  • 星海遊俠（スターオーシャン）
    • 發售日期：1996/7/19，主機：SFC
    • 系列第一作，時代橫跨宇宙曆346年（西元2432年）與宇宙曆46年（西元2132年）
    • 描寫居住在行星洛克的主角在地球人協助下穿越時空尋求治癒石化病的方法
  • 星海遊俠：第二個故事（スターオーシャン セカンドストーリー）
    • 發售日期：1998/7/30，主機：PS
    • 系列第二作，時代為宇宙曆366年（西元2452年）
    • 採用雙主角，男女主角除序章劇情不同外，還會影響可以加入的同伴類型
    • 男主角為前作同伴角色羅尼奇斯與伊莉雅的兒子
  • 星海遊俠：直到時間的盡頭（スターオーシャン Till the End of Time）
    • 發售日期：2003/2/27，主機：PS2
    • 系列第三作，時代為宇宙曆772年（西元2858年）
    • 初版因為有許多BUG的緣故，官方緊急在2004/1/22推出導演剪輯版修正
    • 而近乎超展開的結局演出更讓本作的定位陷入褒貶不一的情況
      • 結局揭曉整個系列的世界觀實際上是在一個虛擬遊戲《永恆星球》之中，最終BOSS路西法更是遊戲營運方的社長
  • 星海遊俠4：最後的希望（スターオーシャン4 -THE LAST HOPE-）
    • 發售日期：2009/2/19，主機：Xbox360 / PS3
    • 系列第四作，時代為宇宙曆10年（西元2096年）
    • 時間軸為系列最早，也有對一代主要角色們的家族譜系進行補充
  • 星海遊俠5：忠誠與背叛（スターオーシャン5 -Integrity and Faithlessness-）
    • 發售日期：2016/3/31，主機：PS4 / PS3
    • 系列第五作，時代為宇宙曆537年（西元2623年）

重製[編輯]

  • 星海遊俠1：初次啟航（スターオーシャン1 First Departure）
    • 發售日期：2007/12/27，主機：PSP
    • 一代重製版
    • 系統比照二代處理，並翻新立繪以及將聲優全面更替，而東地宏樹的角色卻維持原樣
  • 星海遊俠2：二次進化（スターオーシャン2 Second Evolution）
    • 發售日期：2008/4/2，主機：PSP / PS3
    • 二代重製版，同樣翻新了立繪與變更聲優，同樣的東地宏樹的角色依然是本人來配
  • 星海遊俠1：初次啟航R（スターオーシャン1 First Departure R）
    • 發售日期：2019/12/5，主機：PS4 / Switch
    • SO1FD的再重製版
    • 立繪再度翻新之外，並安排初版一代的聲優陣容回歸並開放玩家選擇，東地宏樹依然安定地由東地宏樹擔任

外傳[編輯]

  • 星海遊俠：蔚藍星球（スターオーシャン ブルースフィア）
    • 發售日期：2001/6/28 / 主機：GB
    • 二代外傳，時代為宇宙曆368年（西元2454年）
  • 星海遊俠：回憶（スターオーシャン:アナムネシス）
    • 運營日期：2016/12/8-2021/6/24
    • 時代為宇宙曆539年（西元2625年）

動畫[編輯]

  • 光劍星傳EX（スターオーシャンEX）
    • 以二代劇情（準確來說是基於東まゆみ改編漫畫）為準，2001年播放的TV動畫

主要角色列表[編輯]

一代[編輯]

  • 拉提（ラティクス・ファーレンス）
    • CV：結城比呂（SO） / 宮野真守（SO1FD）
    • 一代男主角，克拉托斯自警隊隊長
    • 武器為單手劍，同時是本作唯二能持盾的角色
  • 米莉（ミリー・キリート）
    • CV：吉田古奈美（SO） / 生天目仁美（SO1FD）
    • 一代女主角，克拉托斯自警隊隊員
    • 武器為杖，擅長施展回復型紋章術，為本作白魔擔當
  • 多恩（ドーン・マルトー）
    • CV：檜山修之（SO） / 伊藤健太郎（SO1FD）
    • 拉提的好友，克拉托斯自警隊隊員
    • 因為收到米莉父親的信鴿而感染石化病，在羅尼奇斯等人來訪不久即遭到石化
    • 武器為單手劍，因為序盤就退場的緣故基本上沒有技術能學…
  • 羅尼奇斯（ロニキス・J・ケニー）
    • CV：岡森諦（SO） / 濱田賢二（SO1FD）
    • 地球聯邦上校，宇宙船卡爾納斯的艦長
    • 為了製作石化病的解藥而帶領拉提等人穿越時空
    • 因為時空跳躍而導致隊伍分散時與米莉同行
    • 武器為弓，擅長施展攻擊型紋章術，為本作黑魔擔當
  • 伊莉雅（イリア・シルベストリ）
    • CV：山崎和佳奈（SO） / 小林沙苗（SO1FD）
    • 羅尼奇斯的副官
    • 因為時空跳躍而導致隊伍分散時與拉提同行
    • 武器為拳套，因為戰鬥回復量較低的緣故，容易有MP耗盡的風險
  • 西烏斯（シウス・ウォーレン）
    • CV：東地宏樹（SO / SO1FD）
    • 阿斯特拉爾騎士團長的兒子
    • 自認實力尚不足，因為不願被認為是靠裙帶關係成為副團長而選擇流浪
    • 完成一開始在哈特發生的運貨任務就能加入，不過如果在完成任務前跑去波特米斯打海盜的話會就此脫離
    • 武器為雙手劍
  • 菲雅（フィア・メル）
    • CV：吉田古奈美（SO） / 豐口惠（SO1FD）
    • 阿斯特拉爾騎士團長的義女
    • 因應西屋斯或阿修雷的加入與否會影響她入隊的事件
      • 兩人不入隊：被指控殺害騎士團長而入獄，協助她越獄可以拉攏她，不過若是在約書亞事件選擇讓他離開的話菲雅也會一起脫離
      • 西屋斯入隊：與西屋斯一同討伐襲擊團長的魔物，進入各國巡迴篇後當隊伍不滿六人時可以在阿斯特拉爾挖角她
      • 阿修雷入隊：在阿修雷討伐襲擊團長的魔物後會與他提起西屋斯的事情，後續同前項
    • 武器為短劍，為本作唯二能持盾的角色，而根據加入路線的不同還有機會習得武神奧義
  • 阿修雷（アシュレイ・バーンベルト）
    • CV：井上和彥（SO） / 若本規夫（SO1FD）
    • 於「魔界戰爭」活躍的戰爭英雄，過去與阿斯特拉爾騎士團長及「真紅之盾」有往來
    • 在西烏斯沒有加入時可以在塔特羅伊的鬥技場找他，之後完成一場鬥技就可以在港口挖角
    • 拉提與他的好感度到達一定程度可以找他學習皇龍奧義
    • 四代在鬥技場可以跟全盛時期的他交手
    • 武器為單手劍，因為斷手的緣故無法持盾，也無法施展皇龍奧義
  • 約書亞（ヨシュア・ジェランド）
    • CV：檜山修之（SO） / 福山潤（SO1FD）
    • 有翼人紋章術士，因為「真紅之盾」的關係導致父母雙亡，為了尋找失蹤的妹妹「艾莉絲」而展開旅程
    • 在巴吉神殿後門可以讓他暫時入隊，突破神殿後可以挽留住他
    • 武器為杖，攻擊與回復的紋章術皆能學習
  • 瑪貝爾（マーヴェル・フローズン）
    • CV：野中希（SO） / 桑島法子（SO1FD）
    • 施展寶珠技術的美女，因為「真紅之盾」的關係導致父母雙亡，為了尋找仇人「真紅之盾」而展開旅程
    • 當隊伍有約書亞時必定會加入，沒有約書亞時可以選擇加入與否
    • 實際上是約書亞的妹妹艾莉絲（エリス，CV：植田佳奈）的轉生，利用禁術將自己轉生成古穆亞人
      • 因為禁術的契約影響肉體被冰封在席爾巴蘭遺跡內，在身分揭露之際與艾莉絲的肉身一同消滅
      • 重製版在達成一定條件後可以迴避消滅事件，作為艾莉絲回歸
    • 武器為寶珠，作為艾莉絲回歸時武器改為杖
  • 佩莉西（ペリシー）
    • CV：山崎和佳奈（SO） / 田村由香里（SO1FD）
    • 精通音律的古貓人少女
    • 因為是古貓人的關係可以自由變換貓型與人型
    • 完成海盜事件後可以撿起她掉落的陶笛，進入各國巡迴篇後於克拉特的PA可以挖角
    • 武器為拳套
  • 提尼克（ティニーク・アルカナ）
    • CV：藤崎卓也（SO） / 鈴木千尋（SO1FD）
    • 精於格鬥技的狼人族
    • 進入各國巡迴篇後當隊伍不滿六人時可以在塔特羅伊的鬥技場與他戰鬥，戰勝後可以挖角
    • 武器為長槍

  • 以保護未開發行星文明而設立的法律

紋章術[編輯]

  • 使用紋章力的技術

凱尼（ケニー）[編輯]

  • 貫穿全系列作品的銀河聯邦軍人世家
  • 除三代以外都有在本篇登場，通常做為與異星人主角接觸的人類方代表，二代更是作為雙主角之一活躍
  • 在年表最早的知名人物是宇宙開拓隊的英雄史提夫‧D‧凱尼，為四代主角方的指揮塔

東地宏樹[編輯]

  • 除五代以外，幾乎每一代本篇作品都有東地宏樹配音的角色登場
  • 甚至在一二代重製版角色CV幾乎全換的情況下，只有東地宏樹的角色仍然由本人配音
