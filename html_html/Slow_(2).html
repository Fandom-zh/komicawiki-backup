Slow Start

スロウスタート/Slow Start/慢活開始[1]/慢啟動[2]
本條目內容不足需要擴充，歡迎有興趣的朋友加以補完。
以下含有部分的劇情捏他，不想被捏到的話請不要反白

目錄

概要編輯

  • 由篤見唯子創作，於2013年6月7日開始在《Manga Time Kirara》連載的四格漫畫作品。
  • 於2017年7月號《Manga Time Kirara》宣佈動畫化，由當時是A-1 Pictures的子公司CloverWorks[3]負責製作並於2018年1月正式放送。
    • 然而因為同期有另一套同社露營作品改編動畫番組播放的關係注目度被壓下去，加上話題和銷售不足的關係動畫二期似乎難以實現
  • 芳文社角色手遊《きららファンタジア》中作為初期作品登場。

故事簡介編輯

一之瀨花名因為某些原因比其他人晚了一年上高中，不過這件事沒有告訴同學。
雖然別人可能會認為沒什麼大不了的，但對她來說是個重要的問題，並想要追趕上其他人。

一之瀬花名（いちのせ はな）編輯

  • CV： 近藤玲奈
  • 年齡：17歲/ 班級：1年2班/ 生日：4月6日/ 星座 - 白羊座 / 血型 - A型 / 擅長科目 - 數學 / 學號 - 1號
  • 本作主角。
  • 在高校入學試前因患上流行性腮腺炎和及後引發的持續高燒而錯過所有的入學試。
    • 曾因此非常失落幾乎自暴自棄，在母親的建議下獨自一人從東京前往位於「長乃縣爽井澤」，志溫擔任管理員的「てまりハイツ」公寓裡生活，一年後就讀星尾女子高等學校。
  • 入學前的一年間幾乎在房間學習的關係體力非常差、跑步幾乎要生要死。
    • 社交能力也不行，在入學一段時間後才認識班上的同學。
  • 胸部似乎在慢慢成長中。似乎有朝一日會長到志温那麼大
    • たまて：「原來胸部會成長不是都市傳說啊﹗」(打擊)
  • 對自己身為浪人這點非常介意、自責及有罪惡感，至今也不敢在要好的朋友表白這點。
    • 因此好幾次作夢也夢到這個話題。但拜其浪人經歷所賜完全不怕苦味的食物。
    • 亦因為這點和朋友非常要好的關係，私底下非常不安甚至害怕碰見舊初中同學而經歷曝光，然而在Step 92碰見曾經有一面之緣的同級生億果実並因為她的無心之言導致負面情緒爆發並逃走。
      • 最終在三位好友追上去跑得很慢的她並開解下最終解開心結並開始和億果実作為朋友相處。
  • 實際是兄控小說家，間中會去283事務所兼差
  • 長大後變成爆乳級艦船

百地玉手（ももち たまて）編輯

  • CV： 伊藤彩沙
  • 班級 - 1年2組 / 生日 - 5月23日 / 星座 - 雙子座 / 血型 - B型 / 喜歡的食物 - 金針菇 / 學號 - 25號
  • 愛稱為「小玉（たまちゃん）」，個性開朗有點我行我素，和朋友說話時多用敬語的雙馬尾女孩。
    • 名字由玉手箱（たまてばこ）而來，她本人非常不喜歡這名字，堅持要別人叫她「小玉」。
  • 家住「爽井澤」的「言農追分站」[4]附近，家中有兩位婆婆，平時會穿和服。
  • 御宅族，玩戀愛遊戲已有10年以上。經常將「事件CG」（イベントスチル）」或「死亡フラグ」之類名詞掛在口邊。
    • 相比起和其他人解釋她更喜歡自己一人在房間玩戀愛遊戲。然而房間可以自出自入的關係一早被家人知曉了
    • 黃金週時會去東京參加同人誌即賣會之類的活動。
  • 和花名非常要好，榮依子說過「她們兩個在一起時有說不盡的話題」。
  • 和榮依子讀同一初中但不同班，對於檢查時藏東西有自己的一套心得。
    • 但榎並老師的手段更高明於是就拜師學藝了
  • 擅長料理，主要負責家中的伙食。自己的零錢就是一個月伙食費剩下的，有時因為想給婆婆們嘗試新的美味料理而花太多錢在食材上導致剩下不多。
    • 然而從動畫中由東京國際展示場返回住家那段路程來看，交通費用是另外計算而且非常充裕
  • 像平板一樣的貧乳，得知花名胸部有所成長時大受打擊
  • 作者另一套作品過來兼差的角色[5]
  • 樂團的鍵盤手，興趣是修剪盆栽
