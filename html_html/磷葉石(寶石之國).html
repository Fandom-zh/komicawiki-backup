磷葉石 (寶石之國)

フォスフォフィライト/磷葉石/Phosphophyllite
本條目內容不足需要擴充，歡迎有興趣的朋友加以補完。
本條目含有大量劇透，如沒看過原作又不想四分五裂並因此換掉自己的肢體，最後被作成超度機器的話請不要隨意反白。
無垢的薄荷天使現在已經死的差不多了

目錄

解說編輯

  • 由市川春子所創作的漫畫《寶石之國》及其衍生作品的主角

個人資料編輯

  • 本作的主角，年齡為寶石人之中最年幼（約300歲）
  • 硬度3.5、靭性最低。
  • 愛稱「フォス/小磷/法斯/3.5/萬碎爺」
  • 寶石顏色為月人最愛的薄荷色。

聲優編輯

  • 黒沢ともよ（原版）
  • 陳貞伃（Animax中文配音版）

性格及形象編輯

  • 性格天真爛漫、相當冒失
    • 做什麼事都相當笨拙。
    • 雖然如此，卻一直想要幫助自己的夥伴
    • 在經歴各種事件後戰鬥技巧和精神上大幅成長，同時身體與性格也跟著改變。
    • 後來卻因為各種不顧後果的作死大膽行為還有幻聽先後換掉雙足、雙手、頭部及左右眼，體型也從原本中性的面貌變成近似人類男性的樣子
    • 隨著一系列的劇情發展而逐漸壞掉（精神與物理上）最後成佛了

經歷編輯

  • 因為非常脆弱又沒什麼特長而一直沒工作。
    • 在金剛先生要求下，接下了完全不重要的編寫博物志的工作
  • 在第一天工作時為了寫博物誌而接觸了辰砂，説出了要幫助辰砂，找到他能做到又想做的工作。
    • 可是之後又一直想不到如何幫助他。
  • 被アドミラビリス族女王出賣作救弟弟的條件而被月人襲撃而失去雙足，可是知道アドミラビリス族的景況又沒生女王的氣
    • 事後金剛大怒，因此被停止博物誌的工作，並因失去三分之一身體而失憶。
      • 之後利用女王留下的石英及貝殼部份造成了新的腳，歩速大幅提升。
  • 在冬天沒像一如以往冬眠，而是跑去跟着南極石工作和學習找他麻煩
    • 途中卻在浮冰的「話」的影響下失去雙手，在和南極石一起到緒之濱找可以當手的材料時，看着南極石被月人帶走。
    • 之後利用找到的黄金和白金形成的合金造成新的手，髪型也改短了，一人完成冬天了的主要警備工作。
    • 但曾出現忘記辰砂的現象(一秒)。因為新的手有伸縮及變形能力而戰力大幅上升，可是卻不時幻覺見到南極石，而且因為合金過重令内部碎裂而要靠合金填補「加固」。
  • 因為對老師與月人的關係相當在意，而開始査閲月人資料，並嘗試與月人對話。
    • 其間在某場戰鬥中被腰斬，幽靈水晶為了救他而被月人剝掉表層。
    • 精神不安定的情況惡化，曾經突然把黑水晶困起來「保護」，之後合金手還捏碎了他自己的頭部，要別的寶石們制止(物理)。
  • 振作起來之後繼續和黑水晶組隊，為了撿黑水晶的斷手而被月人帶走了頭部。
    • 黑水晶決定把幽靈水晶最看重的青金石的頭部接在小磷上，手術後睡了102年。獲得了極強的觀察力與細膩的思考力。
    • 小磷與青金石在夢中交流，在他的助言和黑水晶的「幫助」下故意被月人帶上了月球，從月人的王子口中知道了部份老師的秘密。
      • 青金石的「金」其實是黃鐵礦，別名愚人金。
  • 在月人口中了解月人行動的原因後，認為老師不是同伴並為了讓同伴不再被抓而決定幫助月人。
    • 左眼被換成了月人製造的人工合成珍珠，並得到月人的防水技術及衣飾武器的支援。
    • 説服(試探)部份寶石人自願上月球，想刺激老師令他願意祈禱，然而沒有成功。
  • 另一方面留在地上的寶石人們，得到老師的自白而變得更能接受老師的存在。
    • 磷葉石似乎沒有理解到辰砂不願上月球的原因。
    • 向月人交渉讓他們展開月面的中至高硬度寶石砂回復工程。
以下是71話起的重大劇透，請做好心理準備再點開
  • 71話特攻學校失敗並被黑水晶打碎帶回月球。
  • 72話先是注意到自己不再被地面的寶石人視作同伴，再被前來看自己的黑水晶否定了過去的羈絆及友人關係。
    • 與其他角色的關係開始變得「裡外不是人」，特攻學校失敗後沒同伴來探望，還被新生的黑水晶嘴炮提醒他的態度(又冷淡又自以為在帶領同伴)會讓身邊同伴慢慢消失。
    • 然而當他得知寶石砂回復工程很順利，又覺得即使沒人認同自己還是沒有做錯...
  • 73話被告知寶石砂回復工程即使成功了，復活的寶石人也沒法回復記憶，而且需要殺光食砂為生的アドミラビリス族[1]以取回殼上的高純度寶石碎，被牠們視作皇族的小磷只要一句命令就能讓牠們交出貝殼(滅族)。
    • 當小磷向アドミラビリス族要求取得貝殼時，不忍看到殼和肉連在一起的個體痛苦地拔殻並決定先再考慮一下（因此整個寶石砂回復工程進入了死胡同），後來王子表示再下一世代不一定會長殻所以等吃砂的世代自然死去再回收貝殼就好。
  • 74話決定下次單人回地面拜託老師為月人祈禱，並表示他已經只可以想到這個辨法了。
  • 76話與月人王子了解月人的「身後事」如何處理後，不帶武器的一人向地球出發。
  • 77話不但被打碎監禁，還被其他寶石警戒着；看到金剛祈禱中斷後，被誤以為想攻撃金剛而被場上（除了翡翠和藍柱石以外的）所有寶石打到粉碎。
  • 78話在辰砂的意見下小磷的碎片被分散丟去不同地方收藏。
  • 79話在被打碎220年後，在其他寶石人及月人忘記下小磷被金剛重新組裝起來，然而只剩下一副皮包骨架的樣子。
    • 在看到金剛展開祈禱模式後又中斷的情況下發出了嚇人的怒吼。
    • 結果在第80話時一度強迫金剛祈禱，但之後81話因其他寶石人介入而失敗，並與其他寶石人展開混戰。
  • 82話被救回會後表示金剛仍愛着自己，認為只要其他寶石人不在一定迫能使金剛祈禱，下次要把阻事的寶石人殺掉，結果刺激到黑水晶身上疑似小白的寵物月人而被打爆。
    • 被王子交給バルバタ修復，並提醒他要「慎重」處理小磷這二百年的記憶。
    • 修復期間夢到其他寶石人對自己説出各種過份的話。
  • 84話時以尚未修復好的破爛姿態[2]出現，要求借用月人軍隊及其他月面寶石人的力量進攻地球，並決定放棄寶石砂回復工程。
    • 然而因為其構造的問題，出撃前的防水銀加工做得不完善，敗北flag?
  • 87話時看着碎光光的寶石人們傻笑，還一路走一路故意踩著寶石碎片
  • 根據第88話中王子的解釋，磷葉石獲得了三族的能力，又在月人刻意下得到模擬人類的契機。
    • 肉體的能力無法讓他産生重大改變，但青金石的頭的智慧令他飛躍性的進化。
    • 在故事推進下，他得到了足以讓他求死的絶望與仇恨，並在所有人與物的「幫助」下成為了人類。
    • 到91話時已經打爆了試圖交涉的藍柱石和對他感到愧疚的翡翠。
      • 磷葉石：理解？理解什麼啊？早就來不及了啊。
    • 92話時遇上了辰砂並展開交戰，雙方的身體在戰鬥中化成一片片碎片，但是他們用來戰鬥的合金及水銀逐漸交融在一起……[3]
    • 93話時打倒了辰砂，在身體吸收了大量的水銀的狀態下走向金剛老師的所在位置；途中還看到以前模樣的自己的幻覺。
      • 身體吸收水銀後明顯增高，且恢復了些許理智
      • 在遇見金剛老師後再次請求他的祈禱，結果在金剛表示沒辦法後氣得叫他粉碎
    • 結果94話時金剛因小磷已成為「人類」而真的開始碎裂，他露出安祥的笑容感謝小磷讓他終於能停機並得到解脱
      • 金剛把右眼交給磷，叫他祈求自己的幸福，話沒説完後就碎光光了
      • 同時，王子到達只剩磷一人的學校，要索取金剛給予磷的右眼；拒絶的磷被一撃打飛
      • 磷看着被撃飛到空中的右眼，想要伸手抓着「幸福」，並馬上把它放進自己的右臉
        • 得到右眼的磷回頭一看，只見破爛的校舍空空如也，昔日的「同伴」們被月人大批大批的搬走，並幻見到過去人類城市被隕石破壞的情景
    • 95話時證實這一切都是月人為了製造下一代超度機器的計畫，而磷就是那台被培養出來的機器。
      • 裝上右眼的磷被灌入金剛的資料和記憶，預計時間要一萬年之久。而這時全部被轉化成月人的寶石人和金剛正要在王子的歡迎下度過愉快的一萬年。
  • 98話捏他
    • 在這一萬年裡讀取了金剛與博士的回憶，終於昇華成佛並迎來以前所認識、現已成為月人的各人
    • 一反讀者期望經已變得相當平靜並認同月人想要化作虛無的願望，在金剛與艾庫美亞的道謝下合十禱告，畫面隨即爆炸變成一團白光......
    • 然後又要休載兩個月了幹
 
磷葉石進化史（截至84話）
  • 因身上有很多部份已經不再是單純的寶石，有讀者提出意見認為小磷可能會在多次修補下成為融合月人(魂)、アドミラビリス族(肉)、寶石(骨)的「人類」，引發金剛祈禱。
    • 在上月球後，部分寶石人實現了各自的願望，唯獨小磷一直被打臉到精神崩潰，不但一次又一次的忘記初衷，還變得越來越像人類。
    • 結果一如所料，磷連身軀都變得像人類一般，並能指令金剛「停機」。
  • 經過一萬年後道出了自己其實由始至終都是孤獨的存在。

人際關係編輯

  • 金剛老師
    • 對小磷特別照顧，一開始小磷對金鋼也是抱著敬愛的態度，但後來隨著小磷不斷挖掘秘密而逐漸產生裂痕。
    • 實際上和王子互通有無。
  • 辰砂
    • 在故事初期救下磷葉石，在過程中小磷逐漸發現辰砂的孤獨；事後小磷說要幫他找到比現在更好更開心的工作
    • 磷葉石最初開始行動的契機，但之後兩個人卻漸行漸遠……
    • 後來隨著身體的更換，磷葉石逐漸忘記自己的初衷，還一度忘記了辰砂
    • 諷刺的是因為小磷不斷搞事作死，間接讓辰砂反而被寶石人夥伴逐漸接納
    • 在被磷葉石打倒後感謝他的行動實現了彼此之間的約定
    • 動畫版OP根本就是他們兩個人的世界
  • 鑽石

    • 和小磷一起上月球的寶石人之一，一直都蒙着眼睛。

    • 因為此事而給了小磷重大的影響和PTSD。還一度把黑水晶誤認成南極石

    • 因為小磷把夥伴帶上月球而開始不把他當做夥伴，因此再見面時二話不説的把對方打個粉碎。

  • 神臍小捲毛的小捲毛
    • 同樣都是以寶石為題材，同樣在相近時間點推出的作品，同樣都是樂於助人的天真主角，但是兩人的境遇卻是完全相反
    • 在磷葉石不斷行動作死卻一次又一次被打臉時，隔壁棚的小捲毛則是一次又一次克服困難並成長

      • 隨著時間進展小磷越來越像人類，而且在吸收辰砂的水銀後樣貌還和金剛老師有幾分相似……最後直接走上成為第二代金剛的路……與此同時其他月人化的寶石人們正要在小磷被轉化的一萬年內過上幸福生活……

  • 由於其在劇中的一連串遭遇，被粉絲吐槽作者為甚麼不給他一個痛快（死掉）
    • 求求作者放過小磷，讓他趕緊死掉不好嗎史上第一部讀者希望主角快點死掉的漫畫
  • 更因此在對岸衍生出了一個成句：「上帝才有憐憫之心，我沒有」。
    • 原出處為美國動作片《絕煞刀鋒》
    • 隨著漫畫的劇情發展，對岸的寶石之國粉絲開始將市川老師的自畫頭像P在了圖中神父的頭上，開始在粉絲群內傳播。
    • 而隨著《鬼滅之刃》等一部分熱門漫畫的劇情發展愈發殘酷，越來越多P上其他漫畫作者的頭的版本出現。
    • 目前加入被P圖行列的作者有鱷魚吾峠呼世晴（鬼滅之刃）、變態土筆章人（來自深淵）、究極生物荒木飛呂彥（JOJO的奇妙冒險）、小高和剛（槍彈辯駁）等。
  • 在單行本第10卷末尾附贈的小冊子裡出現了以早期故事設定概念為基礎的草稿漫畫。在早期設定中，磷葉石是會老老實實負責博物志的性格，還能幫忙搞醫務工作，且與黑鑽的關係很好（黑鑽在早期概念故事中是主角）。
