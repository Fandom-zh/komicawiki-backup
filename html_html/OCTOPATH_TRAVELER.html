介紹[編輯]

  • 由Square Enix和Acquire開發的RPG，目前發售於任天堂Switch平台。
    • 於2019年6月7日推出PC版並與NS版同步發行官方中文版。
  • 遊戲使用Unreal Engine 4開發，主要特徵為復古風格的點陣圖結合3DCG的「HD-2D」。
  • 製作人為開發《BRAVELY DEFAULT》系列的淺野智也與高橋真志，而在戰鬥系統上也挪用了該作的BP制。
    • 有一點不同的是，本作的BP為隨著回合數自動增加且無法預支。
  • 將於2019年推出手機版續作《歧路旅人：大陸的霸者（オクトパストラベラー 大陸の覇者）》，時間軸可能是本篇的前傳。

系統背景資料[編輯]

  • 主線劇情
    • 開始新遊戲時可以從八位主角中選出一位展開冒險，未被選中的主角則會作為同伴出現在各自的新手村。
    • 完成四章主線任務後可以解除主角的綁定。
  • 轉職系統
    • 八名主角除了預設的基礎職業之外，還能藉由轉職神殿變更戰鬥職業。
      • 戰鬥副職業在小隊中不能與隊友重複。
    • 另有數種隱藏職業可以在擊敗隱藏頭目後入手。
  • 地圖技能
    • 在地圖上，主角可以對NPC施展地圖技能，八名主角共有四種技能×正邪兩道。
      • 正道技能必須達成一定等級（トレサ的買斷則是金錢足夠）才能施展，不過必定成功。
      • 邪道技能則有一定失敗機率，如果發動失敗五次（ハンイット的教唆則是在戰鬥中落敗）的話必須去酒店繳費才能再度啟用。
  • 支線任務
    • 本作除了八位主角的主線任務之外，另有設置多數支線任務，玩家需要活用地圖技能以完成支線任務。
    • 有些支線任務的解法不只一種，不同的解法也會改變該支線的發展[3]。

主角[4][編輯]

  • オフィーリア・クレメント / Ophilia Clement / 歐菲莉亞（CV：茅野愛衣）
    • 在冰霜地區的聖火教會「フレイムグレース」擔任神官的女性。
    • 為了代替養父與親友完成傳火儀式而前往各地教會。
    • 地圖技能為帶領NPC的正道技能「引導（導く）」。
    • 固有行動為讓被帶領的NPC參戰的「加勢」。
      • 有次數限定，次數耗盡後會回到被引導前的位置。
  • サイラス・オルブライト / Cyrus Albright / 賽拉斯（CV：梅原裕一郎）
    • 在平原地區的都市「アトラスダム」擔任教師的學者。
    • 在經過圖書館藏書失竊事件後因為與公主關係過度密切，被嫉妒的另一位學生テレーズ向校長告密而被放逐。
    • 大木頭、童貞魔法師[5]
      • 不過對別人的感情糾紛倒是很敏銳。
    • 地圖技能為詢問NPC的邪道技能「探求（探る）」。
      • 部分NPC的情報可以增加該地商店的販賣品或是部分折扣。
    • 固有行動為得知敵方弱點屬性的「預習」。
  • トレサ・コルツォーネ / Tressa Colzione / 特蕾莎（CV：青木瑠璃子）
    • 於海岸地區的港口「リプルタイド」經商的商人之女。
    • 經歷了海盜來襲的風波後為了拓展視野而離家展開旅行。
    • 地圖技能為獲得道具的正道技能「買斷（買取る）」。
    • 固有行動為在地圖探索時獲得金錢的「撿錢（歩いてリーフ）」。
  • オルベリク・アイゼンバーグ / Olberic Eisenberg / 歐爾貝克（CV：小西克幸）
    • 委身於高山地區「コブルストン」的戰士，過去是ホルンブルグ王國內被稱為「剛劍騎士」的強者。
    • 在討伐山賊時意外得知弒君叛徒エアハルト的下落而展開旅行。
    • 地圖技能為攻擊NPC的正道技能「試合」。
      • 可以將一部份擋路的NPC打昏以繼續前進，另外離開該區域再度進入即可復原。
    • 固有行動為增強防禦的「防禦推進（防御ブースト）」。
  • プリムロゼ・エゼルアート / Primrose Azelhart / 普里姆蘿潔（CV：桑島法子）
    • 沙漠地區的歡樂街「サンシェイド」的舞者。
    • 為了尋找殺父仇人而展開旅行。
    • 地圖技能為帶領NPC的邪道技能「誘惑」。
    • 固有行動與オフィーリア同為讓被帶領的NPC參戰的「加勢」。
  • アーフェン・グリーングラス / Alfyn Greengrass / 亞芬（CV：關智一）
    • 在河濱地區的村落「クリアブルック」行醫濟世的青年藥師。
    • 過去曾被旅行的藥師拯救，受其影響的他為了救助更多無助人民而離鄉背井展開旅行。
    • 地圖技能為詢問NPC的正道技能「聆聽（聞き出す）」。
    • 固有行動為調配藥劑的「調合」。
  • テリオン / Therion / 泰里翁（CV：松岡禎丞）
    • 前往斷崖地區的「ボルダーフォール」企圖竊取龍石的盜賊。
    • 在到手前一刻失風被レイヴァース家的執事捉住，為了換取自由之身而答應對方提出「協助奪回其餘三尊龍石」的要求。
    • 地圖技能為獲得道具的邪道技能「竊盜（盗む）」。
      • 一部份非賣品必須使用竊盜才能獲得。
    • 固有行動為打開紫色寶箱的「開鎖」。
  • ハンイット /H'aanit / 海茵特（CV：甲斐田裕子）
    • 居住在叢林地區的村落「シ・ワルキ」的獵人。
    • 為了尋找下落不明的師傅而啟程。
    • 地圖技能為攻擊NPC的邪道技能「教唆（けしかける）」。
      • 此時觸發的戰鬥只能使用怪物相關技能。
    • 固有行動為捕捉怪物的「捕獲」。
      • 除了初期的怪物同伴「リンデ」以外都有使用次數限制，且一次只能同時持有六隻。

    • プリムロゼ篇三章頭目。

  • シメオン
    • プリムロゼ的戀人。
    • プリムロゼ篇最終頭目，少數有兩階段的頭目。
    • 殺害ジェフリー的三名刺青男子之首，刺青位置在脖子。
    • リブラック旗下組織「黑曜會」領袖，受其指使殺害得知「フィニスの門」存在的ジェフリー。

    • 在ゴールドショア治療熱病的旅行藥師，不過不願向同行透漏自己的祕方。
    • アーフェン篇二章頭目。
    • 為了賺取暴利使用療效雖好但是使用過多會讓病患產生強烈副作用的藥物後，再高價販賣治療藥物以收取高價治療費。
  • オーゲン
    • 出現在セントブリッジ的藥師，認為「惡人不應接受治療」而拒絕救治ミゲル。
    • 過去曾因治療通緝犯導致愛妻喪命，因而秉持上述主張，而這主張同樣包含著他自己。
    • 完成主線後有專屬後日談。
  • ティム
    • 在セントブリッジ的酒場猝倒的少年。
  • ミゲル
    • 在セントブリッジ因故受傷的男性，因被オーゲン拒絕救治轉而向アーフェン求救。
    • アーフェン篇三章頭目。
    • 真實身分是惡名昭彰的強盜殺人犯，被アーフェン救助後仍毫無悔悟之心並挾持ティム，最後被アーフェン擊敗。
  • テングワシ
    • アーフェン篇最終頭目。
    • 少數在路線中沒有被各種劇情殺的最終boss之二。

其他[編輯]

  • クリス・クロスフォード
    • 尋找父親的旅人。
    • 作為遊戲中遇見的第一位支線任務委託人出現在新手村附近，後來出現時加入了某個旅行劇團。
  • グラム・クロスフォード
    • クリス的父親，本篇開始前就下落不明。
    • トレサ獲得的日記的真正主人，也是當年アーフェン的救命恩人。
    • 妻子過世後受リブラック慫恿打開「フィニスの門」，最後在被リブラック當作復活魔神的容器，成為了怪物「赤目」。
  • アルファス
    • 追求真命天女的傭兵。
    • 原打算追求リブラック未果，後來自己寫的情詩被フローラ看上而促使兩人交往。
    • 完成主線後和收留クリス的劇團團長有專屬後日談，該任務也是開啟「フィニスの門」的重要任務。
  • リブラック
    • 尋找「命運之人」的謎之女性。
    • 真實身分是魔神ガルデラ的使者，貫穿全篇的幕後黑手。
  • マイルズ
    • 於冰霜地區各城鎮出沒的聖火騎士。
    • 為了洗刷父親的冤屈而四處旅行。
  • テラキア
    • 於平原地區各城鎮出沒的教師。
    • 為了創建學校而四處奔走。
  • ル・マン
    • 於海岸地區各城鎮出沒的冒險家。
  • ノエル
    • 於高山地區各城鎮出沒的歷史研究家。
  • リア
    • 於沙漠地區各城鎮出沒，自稱旅行家的少女。
    • 真實身分是カリム王之女グローリア，為了揭發大臣的惡行而在國內巡迴。
  • メリル
    • 於河濱地區各城鎮出沒的少女。
    • 意外發現來自生母的信息後，為了尋根而離家出走。
  • アメリ
    • 於斷崖地區各城鎮出沒的少女。
    • 基於某個契機獲得了龍之卵，為了使其孵化而展開冒險。
  • アシラン
    • 於叢林地區各城鎮出沒的魔物使。

世界[編輯]

  • フロストランド（冰霜地區）
    • 所屬王國：聖火教皇領
    • 旗下都市：フレイムグレース、スティルスノウ、ノースリーチ
    • オフィーリア的起始區域。
  • フラットランド（平原地區）
    • 所屬王國：ウォルド王國
    • 旗下都市：アトラスダム、ノーブルコート、ウィスパーミル
    • サイラス的起始區域。
  • コーストランド（海岸地區）
    • 所屬王國：グランポート王國→亡國
    • 旗下都市：リプルタイド、ゴールドショア、グランポート
    • トレサ的起始區域。
    • 雖然盛行海外貿易，卻也有大量海盜猖獗。
  • ハイランド（高山地區）
    • 所屬王國：ホルンブルグ王國→亡國
    • 旗下都市：コブルストン、ストーンガード、エバーホルド
    • オルベリク的起始區域。
  • サンランド（沙漠地區）
    • 所屬王國：トライブリア聯合王國
    • 旗下都市：サンシェイド、ウェルスプリング、マルサリム
    • プリムロゼ的起始區域。
  • リバーランド（河濱地區）
    • 所屬王國：フェリエン王國→リーヴェン王國
    • 旗下都市：クリアブルック、セントブリッジ、リバーフォード
    • アーフェン的起始區域。
  • クリフランド（斷崖地區）
    • 所屬王國：フェリエン王國→エドラス王國
    • 旗下都市：ボルダーフォール、クオリークレスト、オアウェル
    • テリオン的起始區域。
    • 擁有豐沛的礦脈資源，在商業操作下形成相當大的貧富差距。
  • ウッドランド（叢林地區）
    • 所屬王國：各城市自治
    • 旗下都市：シ・ワルキ、ヴィクターホロウ、ダスクバロウ
    • ハンイット的起始區域。
    • 不屬於任何國家的自治區域。

用語[編輯]

  • 聖火教會
    • 信奉聖火神エルフリック的教會，過去曾協助阻止グランポート王國對平原地區的侵略。
    • 每隔二十年會舉行一次「式年奉火儀式」。
  • 邊獄之書
    • 十五年前自アトラスダム圖書館亡佚的書籍，記載了人體練成等禁術。
  • 龍石
    • レイヴァース一家擁有的四尊龍石，目前除青龍石之外皆散失。
    • クリス的祖先オージン・クロスフォード與ホルンブルグ初代君主ベオウルフ過去在封印魔神時所使用。
