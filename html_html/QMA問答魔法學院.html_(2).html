概說編輯

  • QUIZ MAGIC ACADEMY，簡稱QMA，為KONAMI公司出品的大型益智機台遊戲。該遊戲以眾多題目以及豪華聲優群為主打，題目之多已獲金氏世界紀錄認證。
  • 台灣地區於2013年5月15日，由SAINT-FUN公司以代理商身分引進台灣，除了原有的日版題目翻譯成中文外並追加在地化題目。
  • 香港地區於2013年8月30日正式稼動開始。
    • 但是於2014年12月開始，陸續有撤機報告出現，又以2015年二月湯姆熊多數店家撤收完畢或者正有撤機打算為重要事件，而且絕多數還是玩家自行發現，官方自始至終從來都沒有為此事情做出回應，把玩家當成白癡耍
      • 至2015年3月底，只剩熱門店繼續營業，但是不少店家有的是只擺一台，連店內對戰都只能跟npc打。
      • 題庫更新也停留在2014年12月，看來代理商真的已放棄...
      • 更慘的是，部分熱門店也出現撤機潮...[1]
    • 2016年1月，正式宣告大台北地區全撤光
    • 2017年三月已證實全台全部撤光，台版正式走入歷史，該是考慮更換代理權的時候了吧
  • 遊戲搭配e-AMUSEMENT PASS使用，可以紀錄戰果以及使人物升級，以達到最高級賢者為目標。
    • 只有日版在登入e-AMUSEMENT PASS後才能開啟人物資料的查詢，先前台版連排名都要去跟日方要資料
  • 智慧型手機也有推出iOS版與Android版(QUIZ MAGIC ACADEMY SP)[2]，如果e-AMUSEMENT PASS卡片有日版遊戲資料，檢定模式可將大型機台版的檢定模式進行資料連動。
    • 但可惜的是，已在2015年3月31日後停止QMA SP的配信。
  • 然後於2017年聖誕節，公布新的APP作品--The Lost Fantarium。
    • 於2018年2月9日正式稼動，但是會鎖非日本以外的IP下載[3][4]。
    • 但是一樣營運不佳，已在2019/5/30停止伺服器運作。
  • 也曾經在NDS主機上出過兩款遊戲，一代的クイズマジックアカデミーDS[5]以及二代的クイズマジックアカデミーDS〜二つの時空石〜[6]，也各自存在者原創角色。
    • 只可惜隨者NDS風潮退燒，原本可以透過wifi更新題庫的服務也已經終止；此外因為NDS版本只有純日文版，當時在台灣反而沒有受到重視，因為光是文字輸入題就打趴一堆非日本玩家，更不用說還有DS版獨有的文字填寫題[7]更讓非日本玩家連寫個日文字就會讓時間用盡。

出題方式編輯

  • ○×是非題
    • 根據敘述選出對或不對的答案，要小心有的題目關鍵字會比較慢才顯示出來。
      • 專門裱拼手速的玩家用(無誤)
      • 此題型有衍生出圖片二選一題。
  • 四選一選擇題
    • 四個答案挑一個符合題義的正確答案。
  • 聯想題
    • 根據逐一出現的線索，盡快推理出答案的問答題型，往往第四個線索才會是答題關鍵。
      • 若是第一個線索就可以得知答案的話，稱之為見破。
  • 順序題
    • 根據提問，將答案依序排列的問答題型。
  • 配對題
    • 從左右選項，將有關聯的選項連線起來，不會有一對二的情況發生。
  • 一問多答題
    • 答案不只一個（最少有一個正確，最多全部正確），選取答案後要按下「OK」，才算完成解題。
      • 一定要完全正確才會得到分數，即使差一個選項也是判定錯誤，與台灣學測計分方式不同[50]，所以對新手很嚴苛
  • 先前台版僅這六種出題方式，因硬體設施問題，日版的鍵盤輸入答案類型不會引進[51]，否則玩一次的收費會更貴...
    • 此外日版於天の學舍版本之前，玩家等級到達見習魔術士10級之前，出題模式只有 ○×是非題與四選一選擇題還有聯想題這三種，由此可見部分題目如一問多答題[52]太早出現會對初入門玩家來說，會是個不小的負擔，造成台版推廣不易的原因之一
    • 日版於曉の鐘之後則是把○×是非題、四選一題、聯想題分到easy難度，一問多答題分到hard難度，更加證明一問多答題太早出現會對入門者的負擔很大。
      • 至於normal難度就出現對非日本人很不友善的文字輸入題就是了...
    • 軌跡の交叉版本於連線模式下會出現一方通行四選一題，按下顯示下一個選項之後就直接放棄上一個選項。
    • 手遊版The Lost Fantarium則是出現文字SLOT題與文字替換題還有文字填字題等專婊海外玩家的出題方式。

    • 當然因為僅一人遊玩，所以魔法石結算是以預習合格數x3來結算所獲得之魔法石，不會因為預習全對甚至拿到100分而有額外的魔法石。一樣在結算後可以抽到任一配件。

日版專屬之限定模式編輯

魔龍討伐編輯

  • 日版於Season1時代開設之限定模式，整體來說與(全國對戰)類似，不過科目限制在特定科目，如天青魔龍討伐就是全部都是動漫類的題目。
    • 絕對沒有所謂的金剛魔龍討伐這回事，那就與全國對戰沒啥兩樣。
  • ROUND 1與ROUND 2各別出8題共計16題，每4題替換一次問題形式，FINAL ROUND共12題，每題隨機挑一種問題形式出題
  • 與全國對戰不一樣的地方在於，最後前四名的平均分數必須高於該場設定之指定分數(隱藏)才算討伐成功，只有一個人爆強但其他三人拉低平均分數導致失敗也層出不窮。
  • 唯一可確認推測魔龍的傷害值只能從機台的「副螢幕」進行確認，共計三階段－「無傷」、「兩角破壞」、「四角破壞」
  • 後來不再出現是因為有玩家反應每次能打魔龍的都是那幾位強者，新手想見魔龍都無法突破強者的分數而在第一第二戰被剔除在外，沒有事先分級造成之殘酷結果。

邪神封印編輯

  • 日版於天の學舍開設之限定模式，走的是FF6風，四人合力將邪神封印為最終目標。
  • 一共有24題，答對會增加自軍的AP數，當然答錯會快速提升邪神的AP。當任何一方AP值滿就會採取行動，當然邪神方是採取(二回行動)。
  • 當自軍HP都歸零或者24題結束後沒有打倒就算封印失敗；反之在24題之前成功把邪神血量削弱到一定值以上就會出現封印的動畫。
  • 於HARD難度封印成功或者達成特殊條件亦可取得白金徽章(功能見下述)。
  • 初登場時為金剛邪神，近期日方改成限定期間的xx邪神討伐，魔龍表示憤怒。
  • 結果日版於2015年3月竟然有正式的金剛邪神封印戰......
  • 此後的協力戰鬥全都採用該模式，雖然難度與對手有很大的不同，要求的題目數量也有不同。
  • 戰鬥開始前會要求玩家選擇職業，各職業會有不同的能力值，而部分職業需要把其他職業的等級提升到一定程度才能選擇。
  • 在高難度協力戰，不光只是全員正解而已，職業與戰術的搭配也很重要，只要一個人職業選錯/技能選錯就無法順利打倒Boss。
  • 選擇職業功能在MAXIVCORD版本取消，變成看最終分數是否大於討伐成功的指定分數。
    • 但到了-Xross Voyage-版本變成マジック☆コロシアム(HP制)與協力對戰(積分制)這兩種模式交互出現。

檢定模式編輯

  • 為單人遊玩之模式，有分期間限定與常時進駐之檢定模式，常時比如戰國檢定。
  • 全部25題，一樣有很簡單的是非選擇題，越到後面除了對非日本人無法應付的文字輸入題外，還有即使是四選一也是難度很高的題目。
  • 依照積分決定等級判別，最高可以到SSS級[68]，除了對應的魔法石，也會有特殊道具入手。
  • 要是當初亞版有辦法引進檢定模式，只要能搞出{古文檢定}等教科書題目，至少就能讓玩遊戲=壞小孩之觀念的家長閉嘴......
  • 部分期間限定會是由當年度的KAC冠軍負責提案[69]，只要別太扯多半就可以通過。比如2017年6月的期間限定--深夜動畫檢定，就是第六屆冠軍的提案。

リコードアリーナ(Record Arena)編輯

  • 作為MAXIVCORD版本的主打招牌，就是1 VS 1的特殊卡牌決鬥。卡牌比較貼近暗影詩章那種卡牌呈現方式。
    • -Xross Voyage-之後的版本繼續沿用。
  • 而卡牌除了每玩一道會取得一張以外，可在最後接關那邊按下購買リコード處購買(需使用Paseli扣點功能)。
    • 但是在 -Xross Voyage-版本後，只在リコードアリーナ模式才能免費得到一張；輝望の刻之後只在該模式開頭下才能購買。
  • 每回合會出現二到四題的問答，每回合召喚的卡牌不能超過當回合最高限制的cost數(中間數字)。
    • 旁邊的魔法格等級亦同，可放置消耗型魔法卡與設置型魔法卡，差別是設置型不會自動消失(只會被對方的消費型效果給破壞)
  • 每回能夠開啟的魔法格是依照該回的答對題數來決定，若是全錯0分的話就會無法開啟魔法格對手就可以直接攻擊。

協力對戰職業編輯

  • 若是該協力對戰採取HP制的マジック☆コロシアム，會在確認隊伍後進行職業選擇。
    • 這就考驗玩家對於該職業的認識度，若是隊伍搭配極糟的話很容易滅團。
    • 除了基本的攻擊全部都有外，依照職業不同會有不同的技能。
  • -Xross Voyage-版本以後的職業，一開始有三種可以選擇，會依照遊玩次數解禁其他職業。
  • ファイター/戰士(紅色)
    • 技能：自身攻擊力提升兩階、集氣、消除敵方AP(70%)
    • 初期的攻擊主力，集氣後的傷害力是最高的。
      • 戰術：集氣->攻擊。
  • ソーサラー/巫師(藍色)
    • 技能：隊伍攻擊力提升一階、隊伍防禦力提升一階、吸收敵方AP(最大減少50%--最大增加自軍25%AP)
    • 輔助職業，提升完隊伍攻擊力後改吸收敵人的AP，傷害力最低。
      • 戰術：提升隊伍攻擊力->吸收AP->攻擊(若沒有AP能吸時)。
  • プリースト/牧師(黃色)
    • 技能：隊伍回復力提升一階、回復、解除異常狀態
    • 回復職業，同時是唯一解除敵方給予的異常狀態的要角。
      • 戰術：提升隊伍回復力->補血or攻擊(補血優先)。
  • シャーマン/薩滿(綠色)
    • 需要遊玩三次才能解禁。[70]
    • 技能：降低敵方攻擊力一階、放毒、回復
    • 唯一可以給予敵方異常狀態的職業，攻擊中毒狀態的敵人傷害會增加。
      • 戰術：放毒->攻擊，若隊伍只有薩滿可以回血時要視情況補血。
  • パラディン/聖騎士(橘色)
    • 需要遊玩六次才能解禁。
    • 技能：隊伍回復力提升一階、隊伍防禦力提升一階、回復
    • 具備隊伍HP被打到紅血時，暫時自動提升防禦力；攻擊兼回復職業。
      • 戰術：提升隊伍回復力->攻擊or補血。
  • マジックナイト/魔騎士(紫色)
    • 需要遊玩九次才能解禁。
    • 技能：隊伍攻擊力提升一階、反動攻擊(會損失最大1/4HP)、消除敵方AP(70%)
    • 反動攻擊若是隊伍HP不到1000時無法發動，傷害力僅次集氣後的戰士，不過沒有兩回攻擊一次的限制。
      • 戰術：提升隊伍攻擊力->攻擊or反動攻擊(攻擊後有補師補血時可以用)。
  • メサイア/舞者(金色)
    • 需要遊玩12次才能解禁。
    • 技能：吸血攻擊、回復、防護盾
    • 防護盾可以減半兩次傷害，與聖騎士一樣具備被動技能--隊伍HP被打到紅血時，暫時自動提升防禦力；吸血攻擊回復量與隊伍回復力有關；完全的回復輔助角色。
      • 戰術：防護盾->吸血攻擊or回復。
  • ベルセルク/角鬥士(紅色)
    • 需要遊玩15次才能解禁。
    • 技能：爆擊加護、反動攻擊(會損失最大1/4HP)、消除敵方AP(70%)
    • 爆擊加護會在自己隊伍接下來的四次行動中，提升打出會心一擊的機率；具備被動技能--隊伍HP被打到紅血時，暫時自動提升攻擊力；完全的攻擊角。
      • 戰術：爆擊加護->攻擊or反動攻擊(攻擊後有補師補血時可以用)。
  • クロノマンサー/時之賢者(藍色)
    • 需要遊玩18次才能解禁。
    • 技能：吸收敵方AP(最大減少50%--最大增加自軍25%AP)、增加自軍AP增加速度、狀態全部重置
    • 除非敵人有暗黑的波動才能用狀態全部重置打消，否則此招放下去會連同自軍的增益效果也一併重置。
      • 戰術：增加自軍AP增加速度->吸收敵方AP OR 攻擊(無AP可以吸時)

玩家編號編輯

  • 就是玩家的學號，由一個英文字母搭配7個數字組成，如Z0000001。
    • 除亞版全部都是以Z開頭之外，日版玩家編號依照開卡當時的QMA版本來決定英文字母，也可以證實QMA歷經多少次改版。
    • 先前日版{天の學舍}開卡的玩家就是J0xxxxxx，SEASON2因為是SEASON1加強版故SEASON2與SEASON1合計都是I字頭，同時也是日版目前最多的開卡版本[71]。
      • 日版先前版本「MAXIVCORD（マキシブコード）」是N0xxxxxx時代，同時亦為系列作15周年的紀念作品，亞版...R.I.P。
      • 日版上一版本「軌跡の交叉」則是P0xxxxxx，因為o與0太相像容易搞混。
      • 現行版本「輝望の刻」就是Q。
      • 不過日版人氣也出現退燒之現象，2016年十月中旬編號僅到L0067xxx，2018年十月中旬編號更是只剩到N005xxxx，對照2015年{曉の鐘}12月中旬編號有到K0131xxx來看，也是靠老玩家在撐場面了。手遊版出現後更確定機台版玩的人會剩下老玩家[72]。
    • 所以若是遇到A字頭的日本玩家即表示從初代就開始遊玩的資深玩家，堪稱老妖怪，如某些賢將賢王這樣的強者（又有人敲門了，我再去看看）。這種玩家通常比起新手擁有更強大的答題能力，掃把模式的強敵[73]但同時也是協力作戰的強力隊友。

      • 優先度：金剛>天青>紅玉>翡翠>黃玉>琥珀>瑠璃>紫寶
    • 一旦升為寶石賢者之後，即固定為該類型的寶石不會再變更其他類型的寶石。
    • 台版全撤機前最高只到寶石賢者{初段}，就COUNTER STOP無法繼續升段，因現時點台版沒有任何取得白金徽章的方式換句話說就是封頂。
      • 就算真的有辦法換成日版現行的制度，先前台版最高紀錄只會到軌跡的交叉大概白金賢者的地位。
