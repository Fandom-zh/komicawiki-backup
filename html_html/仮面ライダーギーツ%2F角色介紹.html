登場角色編輯

參與遊戲的假面騎士變身者編輯

  • 從ツムリ手上得到Desire Driver參加DGP的玩家們，戴上Desire Driver後會被傳送到遊戲大廳集合點「Desire神殿」
  • 每輪DGP遊戲後隨著世界重置所有人都會失去前一輪的記憶，唯獨玩家的記憶會連同遊戲記錄數據一起儲存在各自的ID Core裡，當在新一輪遊戲裡被選上後就會重新憶起
    • 一旦ID Core被破壞就會被視作戰敗而消滅，能否利用許願的力量將這些人復活則尚未知
      • 因為規則而被淘汰消滅=失去了假面騎士資格的玩家則只會被消除有關DGP的一切記憶，回歸平常生活
    • 不能告訴任何第三者有關DGP的情報，被發現的話會失去資格而被淘汰
    • 當玩家因為傷病等臨時理由無法繼續參加遊戲的場合，會被強制淘汰這什麼超有良心的規則，神崎士郎和檀正宗都表示不滿
    • 除非特別情況[1]玩家們不能互相攻擊，若有類似行為發生會扣分處置
    • 決定參加遊戲的玩家名單的權限在ゲームマスター手上；過往輪次遊戲的生還者與及被淘汰的玩家，不一定會在新一輪遊戲中再次被選上
  • 每一輪遊戲開始時都須把自己想實現的理想世界在デザイアカード上提交，當成為該輪遊戲的最終勝利者「Deza神（デザ神）」時便會自動實現該願望，但凡事都有例外

主要登場騎士編輯

浮世英壽【演：簡秀吉】編輯
  • 黎明篇參戰話數：第1話（Deza神）
    邂逅篇參戰話數：第2話-第9話（Deza神）
    謀略篇參戰話數：第10話-
  • 本作主角，變身假面騎士Geats的謎之男子
    • 性格好勝，擁有天才般的戰鬥觸覺，在DGP裡號稱無敗
      • 曾參加過多輪DGP遊戲，在第7話被景和問到是從哪時開始參加時自稱是「西曆元年」
    • 為了取勝和活下去毫不介意欺騙別人，但也有私下幫助他人的一面
    • 小時候從母親手上得到一枚雕有凱撒大帝樣貌的金幣，現今仍會隨身攜帶著、儘管嘴上說已忘記了家人但其實對她念念不忘並想找到她
  • 第1話成為了上輪DGP的Deza神實現了願望成為「スター・オブ・ザ・スターズ・オブ・ザ・スターズ」，從第一輪遊戲起以身穿燕尾服的模樣出場
    • 之後也成為了第一輪遊戲的Deza神，實現了「讓DGP營運成為自己家人」的願望，成為了ツムリ的弟弟與ギロリ的兒子
    • 由於第一次成為Deza神時實現了「直到自己死掉為止都能參加DGP」的願望，讓DGP營運方對他的存在感到頭痛、甚至不惜安排有力人選試圖將其「清除」掉
      • 過去某輪還實現了「自己不工作也可以過活」的願望，讓一些觀眾想起某位自言「打工是不可能打工的，一輩子都不可能打工的」的對岸網路紅人
  • 令和版天道總司+桐生戰兔
  • 口頭禪：
    • 戰鬥開始時「さあ ここからがハイライトだ！」
櫻井景和【演：佐藤瑠雅】編輯
  • 邂逅篇參戰話數：第2-8話（最終關卡因傷被強制淘汰）
  • 求職中的大學生，興趣是當義工
    • 父母在10年前因被捲入DGP當中而遭Jyamato殺害，但在相關記憶被消除後只以為他們是意外身亡，自此與姊姊沙羅相依為命；似乎自那時開始便已想成為守護世界和平的厲害大人
      • 當天他並不在場，因此直到沙羅在第8話意外恢復記憶才知道真相
    • 喜歡吃麵衣蕎麥麵，喜歡得會選它為自己在世界末日前的最後一餐
    • 性格直率，是個會認真地許下想世界和平的願望、即使自己面臨存亡危機也不想給別人製造麻煩的好人
  • 故事開首被捲入了上輪DGP的最終關卡，期間初識了祢音並得英壽所救；世界重置後於第一輪DGP獲選為假面騎士Tycoon的變身者
    • 靠著變身後的加持經常獲得Boost Raise Buckle卻拱手相讓而迅速被網友調侃
    • 因得聽聞其身世後理解其想世界和平理由的的英壽鼓勵，在第7話終於能自己變化成Boost Form想一夫當關挑戰Knight Jyamato，但失敗被重創後才發現原來是Geats為取得隱藏於最終關卡舞台的Ninja Raise Buckle而晃點自己，身心受創下憤怒地吐出不會再相信對方的話語後昏了過去
      • 得知姊姊被捲入遊戲裡後負傷出戰想救她，惜勢孤力弱加上在知道父母去世的真相時大意被偷襲而讓她被Pawn Jyamato抓走
      • 為救出姊姊向其他玩家提議合作對付最終魔王，然而不但大家都不願讓出勝利的機會而拒絕，還被英壽反嘴他只管自己幸褔而要犧牲他人的幸福傲慢到不行、著他想抓住幸福就去戰鬥，於是之後再戰サボテンナイトジャマト時一夫當關選擇以自身作餌，將打倒大魔王及救人的工作交給另外三人
      • 其不畏於自我犧牲的「忍（耐之）心」引起了Ninja Raise Buckle的共鳴，從Geats手中脫手飛到其身上救他出サボテンナイトジャマト的殺招、助他摧毀巨大植物救出被抓的姊姊和其他人們並為其他騎士製造出踢罐的機會
      • 然而因為自身傷勢過重在サボテンナイトジャマト撤退後被ツムリ強行淘汰而消滅，消失前感概要守護世界實在不簡單
  • 在謀略篇的世界中似乎失去了使世界更美好的欲望，寧願將錢用在彩票上也不拿去捐助他人，喜好的食物也變成壽司
  • 令和版城戶真司+葛葉紘汰+火野映司（變身動作）
  • 本作的旁白，退場前的每一話都是以他用手機錄音的獨白作開場前情提要
鞍馬祢音【演：星乃夢奈】編輯
  • 邂逅篇參戰話數：第2-9話（最終關卡生還）
    謀略篇參戰話數：第10話（第一關卡中途參戰）-
  • 名流財閥家族的獨生千金，線上追隨者眾的網路評論名人
    • 儘管擁有財富和美貌但家教十分嚴厲，自小便已經沒有交友的自由逞論戀愛，因此有著捨棄這一切當個普通人的夢想
      • 但外人眼裡她就是個已經擁有一切的名門千金，就連其母親「伊瑠美【演：遊井亮子】」也以為她是世界上最幸褔的女孩
      • 被其他玩家認為她在DGP裡許下想遇上自己命中注定的白馬王子的願望，但實際許的願是想得到真正的愛（本人對外表示是秘密）
        • 幼時【演：松岡夏輝】曾被綁架勒索的經歷讓父母對她管教異常嚴厲、也不願放她離開自己身邊；其母親扭曲的愛也是她下述多次想離家的原因
    • 儘管不擅長戰鬥，但在DGP裡為實現自己的理想世界而戰反而更有活著的感覺
  • 在捲入上輪DGP的最終關卡時正於網路上直播自己離家出走；世界重置後於第一輪遊戲DGP獲選為假面騎士Na-Go的變身者
    • 從第3話再次直播離家出走的影片標題可知已嘗試49次，而每一次最後都會被保鑣架回去
    • 在第二關卡裡因遭Da-Paan晃點而被Zombie Jyamato咬傷逐漸喪屍化，中場休息期間離開遊戲時發現自己頻道的影片被人擅自全刪、再次被保鑣找到後還發現母親在她身上及行李裡裝了許多GPS，回家對母親抗議換來的卻是對方愛的巴掌
      • 該晚馬上再次離家出走並遇上擔心她狀況的景和與英壽，在得兩人鼓勵下決定不放棄再次投身戰場，對Zombie Jyamato大開無雙通過了關卡
      • 戰後回家時出手架住了母親要再次揮下的巴掌，並對她聲言終有一天自己會離開這個家
  • 在謀略篇的世界中似乎失去了追求愛的欲望，不再離家出走甚至會接受母親安排的相親
    • 直到ツムリ把Desirer Driver送來後重新回憶起了一切，於第一關卡中途參戰救下被豬隊友拖累而苦戰中的Buffa
吾妻道長【演：杢代和人】編輯
  • 黎明篇參戰話數：第1話（最終關卡生還）
    邂逅篇參戰話數：第2-9話（最終關卡生還）
    謀略篇參戰話數：第10話-
  • 假面騎士Buffa的變身者，平日是建築工人
    • 與英壽同是上輪DGP的最後生還者，在之後的遊戲裡繼續以Buffa身分參戰
  • 尚未成為假面騎士時曾被捲入某輪DGP遊戲之中並目擊參賽者之一的透被其他假面騎士暗算並消滅，自此對所有假面騎士都抱有怨恨
    • 為了得到能擊潰所有假面騎士的力量而參加DGP，並以此作為自己每一輪遊戲的共通願望
    • 而在透消滅後因被Geats勸最好把這段悲劇忘記，於是特別視英壽為自己的宿敵
    • 認為在DGP裡要實現理想的世界便必須把其他人踢下去，然而儘管視其他所有玩家為敵人還是會偶然不情願地對他們伸出援手
  • 在謀略篇的世界中似乎失去了鬥爭心，待人處事變得平和，不過隨著ツムリ把Desirer Driver送來後重新回憶起了一切，再一次以Buffa的身份參賽
    • 因為很快就取回記憶平和的一面沒怎展示過
  • 令和版花家大我，喪屍道長
晴家Win（晴家ウィン）【演：崎山つばさ】編輯
  • 邂逅篇參戰話數：第5（第三關卡中途參戰）-6話（第三關卡被淘汰，任務完結）
    謀略篇參戰話數：第10話-
  • 假面騎士Punkjack的變身者，在第5話因應第三關卡的組隊戰人數不足而由DGP營運方派來填補空缺的工作人員
    • 身分關係不能表露自己的真身也不能說話，但當英壽成為第一輪遊戲的Deza神時在戰場遠處解除變身，獨自暗語期待下一輪DGP
    • 行事自我中心雷到不行，在需要合作的關卡裡完全不聽從搭檔玩家的指令合作抗敵
  • 在謀略篇（被安排？）正式以玩家身分參戰；表面性格是狂放不羈，愛好龐克搖滾的吉他手，但從其暗地裡的行動中可見為人城府頗深
    • 被問及是否第一輪遊戲的Punkjack時直接表示他們認錯人
    • 在Desire神殿集合時向「一見鐘情」的ツムリ求婚，並指摘欲勸阻的英壽別擋別人的戀愛之路
    • 第一關卡跟英壽組隊時還故意以上廁所為由溜掉，在旁用跟0M0相同的鏡頭看著並期待對方因此出局，結果Geats還是獨自擊敗了來襲的海盜Jyamato們讓他大吃一驚
      • 關卡完結後就此對ゲームマスター抱怨時被回「這是你的工作」

豪德寺武【演：金城大和】編輯
  • 參戰話數：第1話（最終關卡戰敗消滅）
  • 十分有正義感、勇於救助他人的消防員，假面騎士Shirowe的變身者
  • 與英壽、道長同為晉身上輪遊戲最終關卡的玩家，並在關卡裡救了景和與祢音，但最終遭城Jyamato突襲而消滅
  • 以前曾經營過萬事屋

邂逅篇登場騎士編輯

平孝人【演：長谷川朝晴】編輯
  • 參戰話數：第2話（第一關卡戰敗消滅）
  • 景和應徵的IT企業的人事部長，在第一輪遊戲被選為假面騎士Ginpen的變身者後於第一關卡與景和重逢
  • 在參加遊戲前許下了想讓身患不治之症的兒子回復健康的悲願，可惜在第一關卡被Rook Jyamato擊敗而消滅
    • 一旁知悉其遺願的英壽雖向景和訛稱自己的願望是「要幫助不幸的孩子們」藉此拗來了對方撿到的Boost Raise Buckle擊敗Rook Jymato，但戰後還是以匿名捐款的方式資助孝人的兒子的手術費
墨田奏斗【演：宮本龍之介】編輯
  • 參戰話數：第2-4話（第二關卡被淘汰）
  • 高中生，假面騎士Da-Paan的變身者
    • 性格孤僻冷酷，拒絕和其他人交流的獨行俠
    • 過去曾是校內籃球部員但一次車禍斷送了成為籃球員的機會，在對世界絕望之下於DGP裡許下後想讓人類滅亡的願望
      • 因此十分嫉妒明明已經擁有財貌卻想將之捨棄的祢音，不但在關卡裡坑她甚至不管扣分對她攻擊
  • 第4話因被Zombie Jyamato偷襲咬中而喪屍化，自暴自棄下用先前得到的Magnum Raise Buckle跟道長交易，假裝被他綁起來放置PLAY後再關卡中襲擊其他玩家盡情胡鬧企圖多拉幾個人墊背
    • 但隨著Na-Go奮起把最後一波Zombie Jyamato一掃而空完結第二關卡，雖然喪屍化狀態消失但得分最低的他因而被淘汰消滅，消滅前還大言不慚地笑說期待下一位消失的玩家會是誰
小金屋森魚【演：あべこうじ】編輯
  • 參戰話數：第2-6話（第三關卡被淘汰）
  • 不務正業的男子，假面騎士Mary的變身者
    • 過去曾於非法賭場當荷官，自詡因職業關係手指很靈敏
    • 後來因捲走大筆賭場的收益逃亡海外而上新聞，被問及時雖表示自己已贖完罪洗心革面，但英壽隨即吐槽不曾看過他被逮捕的新聞
  • 看似平易近人實則笑裡藏刀，為求勝利不擇手段，在面對有利或不利的場合會毫不留情地出賣其他玩家
    • 第一關卡時主動要與祢音聯手戰鬥，但在第4話知道對方喪屍化後直接把她拉去奏斗被綁起來的地方
    • 在第三關卡用ギロリ提供的搭檔交換券讓自己能跟道長組隊，為了確保景和必定被淘汰還把英壽交還的Boost Raise Buckle偷走當作伴手禮送給道長
      • 東窗事發後被迫參加道長主持的Boost Raise Buckle爭奪遊戲，儘管被景和先一步搶到但他卻直接動粗硬是「搶回來」然後宣稱是自己勝利
    • 其為人就連嘴上總是說著不介意與任何人組隊的道長也覺得讓他活下去會很麻煩要率先排除，在第三關卡最後一波開始前偷偷用搭檔交換券抽籤再次與景和組隊，讓他最終因得分最低而被淘汰消滅，消滅前恐嚇其他玩家下遍再遇上時絕不饒過他們
  • 明明是個大叔變身後卻是粉紅咩咩羊騎士…這年頭連騎士都開始流行バ美肉了

謀略篇登場騎士編輯

丹波一徹【演：藏內秀樹】編輯
  • 參戰話數：第10話-
  • 假面騎士Keiro的變身者，年近古稀的老人，在第二輪遊戲中許下了想回復年輕的願望
    • 一開始因為動一下便腰痠骨痛根本不敢下場戰鬥，後來被Na-Go的音樂激發上Buff才願意站起來對抗Jyamato
八木沼雪繪【演：大田路】編輯
  • 參戰話數：第10話（第一關卡戰敗消滅）
  • 假面騎士Letter的變身者，在第二輪遊戲中許下了想減重十公斤後跟自己的推結婚的願望
    • 儘管想實現願望但還是因為怯戰企圖臨陣脫逃，最終先像《國產凌凌漆》裡的鐵腿水上飄般被Jyamato先用火箭筒從天上轟下來後再遭多位海盜Jyamato圍爐擊敗而消滅

  • 在第5-6話道長的回想中初登場的朋友，儘管有懼高症但有著想在老家建一個大地標的夢想
  • 在過去某輪遊戲中遭其他假面騎士襲擊並搶走Zombie Raise Buckle，最終被Jyamato擊敗消滅，在道長懷裡消滅前還呢喃著DGP不該如此

DGP編輯

ツムリ【演：青島心】編輯

  • 在每輪遊戲開始前親自把Desirer Driver和ID Core交給被選中成為玩家的人，並於每個關卡為玩家說明規則並進行遊戲的領航員
    • 由於在第1話結尾於各個場合亂入出現並恭喜人們被選中成為假面騎士，讓鄉民想起30分鐘後的拉霸機型妖怪而替她起了個「妖怪無差別當選女」的外號。
      • 也因為遊戲的願望達成部分而跟キュゥべえ扯上了關係
  • 令和版ポッピーピポパポ；伊茲二號
  • 以前曾經撿到過龍蛋

ギロリ【演：忍成修吾】編輯

  • 常駐於Desire神殿的休息室，在關卡中場休息時間為玩家們提供協助的門房
    • 所有玩家都能自由使用休息室，禁止在裡面進行任何暴力或妨礙行為，違者將馬上失去資格
    • 玩家可在休息室消費關卡中得到的金錢添購外觀服裝飲食等物品，但不包括任何P2W關連道具
  • 另一個身分是DGP的「ゲームマスター」，每一輪遊戲的參賽玩家都由他以此名義「經過嚴正審查」去挑選
    • 平常遊戲進行時會在某個地方觀察進行狀況，絕少在台前出面
  • 代理奇樂，持有死亡筆記[2]
    • 長得有點像劉德華
    • 偽裝真差，ゲームマスター的面具一看就知道是《地獄怪客》的克焰人
      • 大概因此在被網友發現後數日官方重新上載線上公開版的第1話並替換了他的正面鏡頭，到第4話再次出場時已換上了別的面具

Jyamato（ジャマト）編輯

  • 出現於DGP的遊戲關卡舞台「Jyamar Area（ジャマーエリア）」，襲擊被困在關卡裡的人們的謎之戰隊雜兵怪人集團
    • 若無法達成通關條件/玩家們在關卡中被全滅導致GAME OVER的話該Jyamar Area與及被困在內的人們會全數消滅，消失在人們的記憶之中
  • 因應關卡的不同風格會於基礎造型「Pawn Jyamato（ポーンジャマト）」之上予以變化，能力及行動模式也視外觀風格有所不同
    • 只要能活著撐到通關，玩家在關卡中被Jyamato攻擊造成的受傷、異常狀態與懲罰都會在關卡完結同時重置

  • 與景和同住的姊姊，十分理解弟弟的想法，同時是單推祢音的超級粉絲

    • 每位主役假面騎士都有各自相性較佳的Raise Buckle組合，會於變身形態列表中以粗體顯示
