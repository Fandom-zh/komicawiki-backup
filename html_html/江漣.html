Ein／Elen／（吾妻）江漣
以下含有部分的劇情，沒接觸過原作又不想被江漣用反戰車步槍、狙擊槍、手槍、刺刀什至腳踢爆頭請不要繼續看下去。

出處[編輯]

  • 原作 十八禁遊戲《Phantom -PHANTOM OF INFERNO-》
  • 改編動畫《Phantom 〜Requiem for the Phantom〜》

名稱[編輯]

  • Ein（アイン）
    • 原意為德文的「一」。
    • 但實際上德文的一是Eins，Ein是一個的意思，相當於英文的a。大概是原作認為s太難讀所以無視了
  • Elen（エレン）
    • 男主角吾妻玲二為了給予其生存意義而隨口改的。原因大概是因為都是E字頭吧。
  • 吾妻江漣 （あずま エレン）
    • 跟玲二在日本生活，報稱是兄妹時改的(實際上明顯是夫妻)，把玲二的姓氏加上由Elan強制翻成的漢字構成
  • 聲優：遊戲：南央美 動畫：高垣彩陽

外表[編輯]

  • 十來歲的東方女孩子
    • 動畫版比遊戲版看來成熟一點點
  • 個子比較小，但身材相當不錯要不然怎當H Game的女主角，是正統的蘿莉體型
    • 動畫版的個子一樣小，但因為明顯地變得圓潤粗壯，所以顯得更嬌小萌。但也能更好地解釋戰鬥力高強的原因，好像也比較符合戰鬥民族蒙古人的血統
  • 灰黑色的半短髮，似乎沒怎麼整理過，也符合殺手的設定
    • 動畫中毛色偏墨綠，而且髮型也變得比較不自然。頭髮在幾年間不斷變長可見真的不常打理
  • 眼睛在遊戲中是棕色，在動畫變成跟頭髮一樣的灰黑色
  • 在故事最後找回記憶之前都是木無表情的，在這之前的表情幾乎都是裝出來的，但恐怖的是都不眨眼，真的給人像人偶的感覺
  • 皮膚非常之雪白而且幼滑

身份[編輯]

  • 地下組織「煉獄」(Inferno)，被冠以最強 Phantom 稱號的殺手，目的是統一全美的地下組織
    • 因此，在ED2的開頭，會聽到「我們是，無法存在於任何地方的亡靈，什麼人也不是的Phantom」的獨白
      • 除了表達神出鬼沒的殺人手法外，也顯出的身份被抹消、殺孽深重的悲哀
  • 為變態賽斯教授的第一名訓練出來的殺手，故名為Ein
    • 賽斯的得意技為先向對象洗腦，然後再以死亡為威脅強迫對方接受訓練，成為殺手。男女主角和之後的數字姊妹也受過這種對待，而且還認為自已訓練出來的殺手是自已的「作品」，完全不當做人
  • 在逃亡到日本的時候，以吾妻江漣、玲二妹妹的身份行動，過著普通的高中生活。動畫中的學校是可以自選校服的類型，可能是心境問題選上一件暗色的水手裝
    • 連帶OP2也有大量的日常生活的溫馨畫面 但加上Ali Project 的歌和故事背景卻使人異常心寒
  • 其實是蒙古人，不知為什麼留落在香港的孤兒院中，數年前被賽斯收養了

能力[編輯]

  • 殺人，在主角三人中資質是最低的，但可能因為資歷較長的關係，始終應該是最強
    • 狙擊槍
      • 狙擊技術高超，殺人於無形。也試過在被敵人發現的情況下單用狙擊槍就殺敵大半 雖然自已最後還是中槍了。最厲害是以反戰車步槍作超遠距狙擊
    • 手槍
      • 不用狙擊槍時一般只用手槍，使用方法有時十分粗暴，例如強行轟爆防彈玻璃，而且也少用滅聲器。似乎習慣瞄准心臟多於腦袋
    • 刀
      • 曾在暗殺行動中用過剃刀。戰鬥刀則只正面戰鬥時才使用。同樣技術高超，可以用飛刀爆頭
    • 格鬥技
      • 粗壯的肢體帶來強大的攻擊力，以四招殺掉一名數字姊妹
    • 自動步槍，自動手槍
      • 動畫中只各用了一次，總之也很強大就是了
    • 瞬間轉移
      • 正如其稱號 Phantom一般神出鬼沒，會化成幻影，瞬間轉移等等，看來只要是賽斯訓練的都會，大概只是動畫中用以表現動作靈活的手法
  • 追蹤/反追蹤
    • 懂得在追蹤時以被追蹤的方式思考，反之亦然
      • 這能力好像也是主角三人中最高的
  • 演戲
    • 強大的演技使她成為明顯比玲二專業的殺手
    • 曾為暗殺而演過女僕，為了調查暗殺的環境而演過玲二的女朋友
    • 最後在日本的學校裡更是整天在演玲二的妹妹，可見其表現不受時間地域的影響
      • 最厲害是「就算披著羊皮，也不要忘了用狼的嗅覺」
  • 變裝
    • 演戲當然少不了變裝，而且很簡單當然也難不了她 只是可能是髮型和身材的關係，總覺得一整個違和，尤其是穿著晚禮服

武器[編輯]

  • 槍，很喜歡用大口徑的左輪手槍 明明手掌應該很小才對
    • Colt Python 357
    • S&W M29
    • H&K U.S.Socom Pistol
    • TDE 44 Auto Mag
    • Vz61 Scorpion SMG
    • Gepard M1 (就是上面題到的反戰車步槍)
  • 刀
    • Chris Reeve Project 2

性格[編輯]

  • 原來的性格在洗腦和調教之後已不得而知了
  • 故事前半部給人的感覺完全就是賽斯教授的人偶，可見賽斯非常成功
  • 故事後半在被玲二冠名和情訂終生後，逐漸變得人性化，但因為始終所有的記憶也是當殺手的記憶，加上正在逃亡，性格還是相當死板，感覺是一名相當冷漠而且認真的殺手
    • 但其實只要細心留意表情細部的變化，會發現其內心的轉變
  • 飾演的女朋友是有點嬌蠻而活潑的類型 這集應該萌倒不少人
  • 飾演的妹妹則是乖巧卻帶點淘氣的類型

萌屬性[編輯]

  • 失憶
  • 無表情
  • 無眼神
  • 不眨眼 (細心留意之下，其實在心理波動時有眨過數下)
  • 說話無感情
  • 剛強
    • 失去生存重心會變得相當軟弱
  • 變臉
  • 可憐
  • 悲涼
  • 人間凶器
  • 小個子
  • 短裙 (動畫限定)

其他[編輯]

  • 曾說過作為一個道具，作為一個裝置，我們只要能正常運作即可，不能思考多餘的事，如果去思考為什麼殺人的話就會崩潰
    • 可見她殺人時也會有相當的感覺，只是藉由不去思考來壓抑，所以外表看不出來
  • 以只作為Ein，而不是作為「自已」的思想過活
    • 所以一直對賽斯言聽計從，也從沒問過原由
  • 在她的表情看來，玲二的出現在她心中造成極大的動搖，雖然當時還是絕對聽命於賽斯
    • 也難怪，畢竟自己一個過著地獄般的生活兩年多
  • 早期以賽斯，後期則以玲二為其生存目的
    • 早期曾冒死替賽斯擋掉玲二的子彈
    • 後期則有：「如果說這世界並不是無限的地獄，那正是因為你（玲二）還活著啊」、「我不想活在沒有你（玲二）的世界裡」這樣的閃光發言
    • 但在殺掉賽斯後大哭的表現來看，賽斯始終在她的心靈中佔很大地位
  • 極之為玲二著想，尤其希望他能過著一般人的生活
  • 玲二把自已的護照撕毀時是她全劇最激動的時刻
    • 雖然都很平淡，但我看來感覺是出奇的閃
  • 有時做出一些出奇的行經，如買熊寶寶和種花
    • 雖然最後都表現出沒所謂的樣子，但應該是還留有一些少女情懷的緣故
  • 在第一集險些被玲二殺死時，露出很嫵媚的表情，使玲二萌翻下不了手，這很難說到底是裝出來的還是真實的表情
  • 始終也是一個殺手，為了生存可以不擇手段
    • 如綁架同班同學
  • 好像可以以精神力壓低痛楚，但在遭遇玲二後好像失效了
  • 在夢中經常出現蒙古的藍天和大草原，還有燿眼的光與強勁的風，是被洗腦前的記憶
  • 在賽斯被坑，開始逃亡時，曾試過自殺，但失去生存重心的她連開槍的膽量都沒有，因此玲二才給他冠名
    • 不過正如開頭所說，她是被賽斯以死亡威脅才成為殺手的，所以現在想起來，會怕死也是理所當然的，只是她看來連這個恐懼都忘記了，果然被調教得很慘
    • 在這角度來看，賽斯的失憶殺手方針其實是很失敗
