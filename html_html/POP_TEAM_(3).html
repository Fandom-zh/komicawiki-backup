POP TEAM EPIC

POP TEAM EPIC[1] / ポプテピピック（Poputepipikku） / POP子和PIPI美的日常[2] / PPTP / 點對點隧道協議 / ☆色ガールドロップ
本糞條目因編者太糞而內容不足，請各位熟識此糞作的糞朋友加以補完。
本糞條目可能有毫無屁用的劇情捏他，如未看過不補也沒差的原作又不想被糞劇情捏到的話請不要繼續往下看。

簡介編輯

  • 由大學中退、島中作家和糞漫畫卿大川ぶくぶ於2014年開始連載創作的四格糞漫畫。
    • 作者早期畫過東方、灰姑娘等作品的二創，所以在部分同人圈小有名氣。從當時就開始在玩梗了
    • 目前出到第五集（季）但只有四期單行本，曾有去夏威夷養病而休刊的經歷
    • 作者大川ぶくぶ於2022年4月30日結婚，對象保密不公開。
  • 原本要在2017年10月動畫化[3]，因為動畫唯一贊助商KING RECORDS的錯誤延期至2018年1月，全12話。
    • 延期當時漫畫也同步提到「すまねぇ……おそ松さん……うまるちゃん……ッ」「おめぇらと……覇権争いしたかった……ッ」[4]。
      • 不過當時更多人吐槽「這傢伙誰啊」
    • 動畫播出後因為實在太糞演出手法太新奇而打響了名聲。
    • BD/DVD第一集的評論音軌竟然還放原作者與マフィア梶田在看動物朋友動畫第1話的實況。
    • 到2019年4月1日一舉播出第13與14話的特番，而且……（下述）
  • 以下為第一話的情況：
    • 開幕就播出意味不明的萌系美少女青春偶像動畫《☆色ガールドロップ》，找來近年當紅女聲優為女角配音，還有頗為精美的OP。
      • 在小倉唯喊出第一集標題時，由江原正士配音的POP子撕掉畫面。
    • 一開始官網介紹的POP子和PIPI美的聲優沒出來[5]。
    • 在很短時間内玩了包括你的名字、Chrono Trigger、純愛手札、蒼之流星SPT雷茲納、動物朋友、神劍闖江湖、機械巨神 地球靜止之日、星光樂園、豆豆龍、東離劍遊紀、神奇寶貝、烙印勇士、上古卷軸、超時空要塞甚至星際異攻隊等各種新舊梗讓觀眾腦袋爆炸。
    • AC部的恐怖作畫讓沒見過的人SAN値暴跌。
    • 莫名的有個實寫老外在對觀眾講法語（沒有字幕）。
    • POP子和PIPI美講法語（也沒有字幕）。
    • 才播了一半就放ED。
    • 剩下的一半時間是將以上內容重播。
      • 不同的是POP子和PIPI美的聲優換掉重配，法文部份追加字幕，有一幕BGM晩了播及ED換人唱。
    • 甚至播放的內容與先行上映會的完全不一樣[6]。
    • 撥完後NICO上負評率達67.7%[7]，很多島民表示糞到看不懂決定再看幾次。
  • 本作獲得「第十三回声優アワード」＜シナジー賞＞。
    • 此獎項是頒給最能夠激發聲優們自身潛力與魅力的作品。
  • 2021年10月起突然推出將原本的聲優全部更換但内容沒變的重置播版動畫，本篇受害聲優還不夠所以要再來一輪。
    • 與首播版不同的是AC部和法語環節也跟著重新配音，打馬賽克的部分在重播版中都會變得清晰一些；後半播出時畫面上會出現兩個「再放送」字樣。
    • 第1話男女CV順序變成先女後男，到第2話甚至連原先沒畫好的背景都重置了。
  • 2021年12月26日第12話（重）播完畢後於結尾宣布動畫將有第二季讓更多聲優被迫害，2022年10月1日起播放。

主角編輯

 

POP子（ポプ子）編輯

  • CV：下述
  • 隨處可見的14歲女子中學生，比較矮、臉比較圓的那個。
    • 特徵為金色短雙馬尾，繫有黃色波堤髮圈。
    • 毒舌，基本除了PIPI美以外的人都是完全敵意般的沒幾句好話。
    • 對指定暴力團竹書房[8]異常厭惡。
  • 名言：
    • おこった？（生氣了嗎？）
      • 出自漫畫1話（Season 1連載第1回）
    • 今日も一日がんばるぞい!（今天一天也要加油ZOY！）
      • 出自漫畫23話（Season 1連載第3回）
      • 直接用《NEW GAME!》女主角青葉的台詞然後馬上就忘掉了，三格的記憶力
      • 動畫第3話播出後被《NEW GAME!》原作者得能正太郎看到，回文馬上被洗一堆「おこった？」
    • もう見た（看過了）
      • 出自漫畫39話（Season 1連載第5回）
    • あーそーゆーことね 完全に理解した（原來如此 完全理解了）
      • 出自漫畫74話（Season 1連載第7回）
      • 動畫第10話播出後柯南劇場版官推發文表示「完全に理解しました」
    • 進路相談でパラディンになりたいって言ったら怒られた（志願討論會上說要當聖騎士結果挨罵了）
      • 出自漫畫97話（Season 1連載第9回）
    • さてはアンチだなオメー（那麼你這傢伙是反廚呢）
      • 出自漫畫129話（Season 1連載第12回）
    • もしもしポリスメン？（喂喂警察先生？）
      • 出自漫畫137話（Season 1連載第13回）
      • 喂喂喂，我幹你娘

すしポプ子（壽司POP子）編輯

  • 動畫第13話OP曲登場，由13話OP人設兼人物作監「壽司（ことぶきつかさ）」所繪製的ポプ子，身形變成普通中學生但多了一雙相當大的黃色耳環。
  • 於スパロボX-Ω作為角色參戰，發招時發出的光其實是用來遮掩比中指的雙手。

PIPI美（ピピ美）編輯

  • CV：下述
  • 14歲，比較高、臉下巴比較長的那個。
    • 特徵為藍色直長髮加上紅色蝴蝶結。
    • 性格冷靜，相較POP子基本沒啥表情。
      • 相對POP子的吵鬧暴力，PIPI美時常是無聲地嗜殺。冷面殺手的意思
    • 基本是吐槽役擔當，不過大多數都是跟POP子一起瘋。
    • 喜歡POP子。
      • 所以兩人結婚了
  • 名言：
    • おこってないよ♡（沒生氣唷♡）
      • 出自漫畫1話（Season 1連載第1回）
    • いっぱいちゅき♡（超喜歡哦♡）
      • 出自漫畫10話（Season 1連載第1回）
    • あぁ、音もなく殺せるな（能無聲的殺掉呢）
      • 出自漫畫86話（Season 1連載第8回）
    • あぁ、ごめん。ヘルシェイク矢野のこと考えてた（不好意思，剛剛在想HellShake矢野的事情）
      • 出自漫畫97話（Season 1連載第12回）
      • 動畫第7話播出後一度成為推特話題榜的第一名

スーパーピピ美BARIモード編輯

  • 動畫第13話OP曲登場，由大張正己繪製的ピピ美，變成機器人外觀。
    • 平時是約2公尺大小的人形機器人，攻擊模式就變成有M尺寸的戰鬥機器人。這是怎麼變的…
    • 參戰機戰系列的主因
  • 於スパロボX-Ω作為機體參戰，基礎武裝為腰間的磁軌砲，必殺技則是持龍劍發動。銀色火鳥功

評價與迴響編輯

  • 在Niconico動畫上成為史上最快達成100萬播放動畫，並在最終話播畢後成為首套所有集數播放數量超過100萬的配信動畫。果然觀眾們都是M
  • 總體的笑點和主題雖意義不明，但總體評價依然是一部捏他動畫，因此流出大量二次（三次？）創作，官方於14話動畫中則自我吐槽「梗玩的太明顯會引發版權問題，但太高深普通觀眾看不懂」的自黑評價。不過在資深廚面前其實都沒差
  • 而根據一些有心人士的統計，雖然本作觀眾在同期新番之間流失率偏高，但觀眾基數比其他同期新番多出最少一倍，所以根本沒有影響，每集播放後的twitter相關推文數量也在trending前列。
  • 因為有人拿第1話POP子奔跑的姿勢去製作BB素材，惡搞某個慘死前不忘叫人不要停下來的團長而引發另一波PPTP惡搞潮。
  • 同一時期發生的任天堂控告白貓也被拿去混在一起玩。
  • 演出的聲優都以參加這套作品為樂，像古川登志夫就十分喜歡這套作品，說『這套作品證明即使是在同一套教學體系下，演同一個角色不同聲優也有不同的風格。』
    • 喜歡到甚至看到網民說糞動畫會生氣的程度，很多網民立即安撫他說叫糞動畫是原作者要求的。
  • 另外有人結合假面騎士系列製作出一系列仮面ライダーポプテ。
    • 結果就是幾乎每一集Eternal都會因為唱歌而被W逮捕
      • 唯一一次沒被逮捕是因為成為了改編自第7話「HellShake矢野」的「Eternal克己」的主角，因此改由W的師傅Skull代替他被逮捕
      • 不過在第10話被抓了兩次
    • 每次被捕時W的型態都會改變[50]
  • 因為在對岸是以將原文對白譯成粵語髒話的改圖方式流傳，動畫版一出在網路上跑出很多帶著各地口音的粵語惡搞配音，也加了很多額外的動畫及香港電視劇梗。
  • 因為作者也在玩到一塊，似乎有進化成聲優梗圖的趨勢
    • 梶裕貴和竹達彩奈公佈結婚當日，大川ぶくぶ就在Twitter上公開了兩個PIPI美結婚的圖。
    • 水樹奈奈公佈結婚後，不少網民都回想起中村悠一在動畫Event上調侃水樹的名場面，結果他又以漫畫畫風再現名場面。
    • 小野花澤公佈結婚後當日也畫了圖，不要吐槽為何一人拿刀一人拿槍。
