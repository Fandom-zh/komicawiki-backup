機皇帝

機皇帝（きこうてい）
這是一個編輯中的條目，歡迎陷入絕望的決鬥者加以補完。

目錄

解說編輯

  • 《遊☆戯☆王 5D's》的反派角色「普拉西多」、「盧奇亞諾」、「荷西」與「阿波利亞」的王牌怪獸群
  • 一致為機械族怪獸，主力怪獸群大多具備針對同步怪獸的效果，因而人稱「同步殺手（笑）」
    • 附屬字段有「機皇」、「機皇帝」與「機皇兵」三者
    • 這之中，「機皇兵」全員皆可通召，很長一段時間是「機皇」牌組裡僅有的可通召怪獸
  • 其中三隻機皇帝在原作中是由「機皇帝○○」、「○○攻擊部」、「○○防禦部」、「○○頭部」與「○○載具」五張怪獸卡組成的，OCG化之後皆改成僅1張怪獸卡的模樣
    • 但也因此通通變成不能通召的怪獸，要跳出來還得先想辦法炸掉自己的怪獸
    • 有趣的是，機皇帝的零件怪獸其實可以互通，普拉西多就曾經偷拿盧奇亞諾的《神空載具》裝在《機皇帝 神智無限》身上
    • 另外，零件怪獸們共通的效果為「這張卡不能攻擊」以及「自己場上沒有『機皇帝』怪獸的話，這張卡將會被破壞」
      • 不同部位各自具備專用輔助效果，但不同機皇帝的相同部位效果也一樣
      • 此外，有些零件怪獸還存在上位互換版，將場上的對應部位怪獸卡送去墓地就能跳出來，概念上類似升級機體零件
    • 除了五大部位，還各有一張長得像機械蛋的「○○核心」，被效果破壞就會一口氣特召出對應機皇帝的本體與零件
    • PSP遊戲《Tag Force》系列中同時有收錄五部件機皇帝以及OCG的單卡機皇帝，也是少有地能讓玩家體驗到與五部件機皇帝對戰的遊戲平台
  • 手遊《Duel Links》中，除了OCG版的機皇帝怪獸卡，也有收錄動畫中的部件怪獸，而且因為本來就是看「機皇帝」字段來配合的，所以效果原封不動
    • 看得出來官方應該很想在手遊重現動畫裡的機皇帝五部位場面，但礙於規則限制只好折衷
    • 而且DL的AI根本不懂官方的用意，結果就是AI常常空場通召機皇帝部件，放著給他自爆再跳機皇帝出來
  • 原作中是未來永動機展開反轉後釋出的怪獸，大量機皇軍團遍布全世界，地球生靈塗炭，除了世界本源滅四星外無人倖免
    • 明明設定上應該是阿波利亞不共戴天的仇敵才對，為什麼你能用得那麼開心那麼得心應手啊這位大哥
    • 另外阿波利亞也是目前未來四人組中（包含後續所有新卡強化、非本篇登場的牌組系列）都堅持沒有使用同步怪獸的
      • 雖然ZONE的時械神是風評被害[1]

  • 普拉西多的機皇帝，是第一個登場的機皇帝，也是唯一曾給雜魚使用過的機皇帝
  • 機皇帝中唯一在動畫中屬性被改過的，最初登場時為地屬性後來再次登場時才改為闇屬性
  • 雖然打點不上不下，但卻是少有的可以壓制一次魔法卡發動的追加效果，與反而是最先會被踢出牌組構築的下面那位形成鮮明對比

卡片資訊編輯

  • 特殊召喚／效果怪獸
    闇屬性／機械族／等級1／攻擊力2500／守備力2500
    • 這張卡不能進行通常召喚。必須使用這張卡的效果才能特殊召喚自身。
  • 效果：
    1. 自己場上表側表示的怪獸被破壞並送去墓地時才能發動。特殊召喚手牌中的這張卡。
    2. 1回合1次，以對手場上1隻同步怪獸為對象才能發動。將那隻怪獸變成這張卡的裝備卡。
    3. 只要這張卡存在於怪獸區域，自己場上的其他怪獸就不能攻擊，這張卡的攻擊力上升使用自身效果獲得的裝備卡的原本攻擊力數值。
    4. 1回合1次，對手發動魔法卡時才能發動。無效那張卡的發動，並破壞那張卡。

機皇神 機械編年史無限（機皇神（きこうしん）マシニクル∞（インフィニティ））編輯

 

  • 阿波利亞初登場時召喚的怪獸，是「超越機皇帝的機皇帝」可是這傢伙不叫機皇帝
  • 雖然擁有高達4000的攻擊力，但召喚條件極為苛刻，尤其是沒有了專屬陷阱《機皇創世（きこうそうせい）》後基本不可能正規出場，所以一般是冷板凳位置
    • TF系列中，《機皇創世》不但強化了動畫效果還增加了代破效果，所以為什麽不照著這個效果O化啊

卡片資訊編輯

  • 特殊召喚／效果怪獸
    光屬性／機械族／等級12／攻擊力4000／守備力4000
    • 這張卡不能進行通常召喚。必須將手牌中的3隻「機皇」怪獸送去墓地才能特殊召喚這張卡。

    3. 自己的準備階段中，將1張因這張卡的效果作為裝備卡使用中的怪獸卡送去墓地才能發動。給予對手那隻怪獸的攻擊力數值的傷害。在發動這個效果的回合，自己不能進入戰鬥階段。

    2. 這張卡被送去墓地的場合才能發動。這個回合的結束階段，根據自己墓地的「機皇」怪獸數量，給予對方那個數量×100點的傷害。
