「深海棲艦」編輯

 
みずち繪製的深海棲艦群像
  • 多數非活動（イベント）Boss的敵艦/敵艦娘都有3個等級：普通、精英（Elite）和旗艦（Flagship）。
    • 普通的眼睛為青綠色；精英為紅色，身體和武裝纏有紅色火焰；旗艦則是金色。
    • 自Extra Operation（X-5）實裝後開始出現旗艦以上的「改」級，眼睛會像B☆RS一般噴出藍色火焰，其餘外形與旗艦級同樣。
      • 其實當初運營曾打算推出在旗艦級之上的「夢魘級（Nightmare）」，最後改以現在的「改」級呈現
    • 2014年夏季活動地圖出現冠以「後期型」之名的新種強化版
      • 反正關卡越出越難是常態，以後有改二或新的字尾也不奇怪
  • 一般的敵人以伊呂波歌[1]的順序來命名，特殊的敵人則以鬼／姬來命名
  • 深海方的裝備和艦娘的裝備不同，主砲與副砲的口徑皆以英吋來表示
  • 因為後來的限定活動BOSS的原型幾乎都會以通關獎勵的方式出現，加上單色調外觀以及得到救贖的擊破台詞，造就了所謂「虛化」的某某艦的說法
    • 歐美討論區則是稱為「達斯」某某艦
    • 另，在同盟國艦娘出場以前深海棲艦在設定上是同盟國船艦的化身，後來才改變成戰死者怨念的集合體

  • 在次年夏季活動中也開始出現精英級後期型，擁有超越旗艦級同型艦的資料，近乎變態的閃避能力以及威脅巨大的夜戰二連攻擊。
    • 但不是動畫裡出現的精英級イ級後期型的樣子，因為眼睛沒有變紅
  • 動畫裡出現的似乎都是イ級後期型，在第1話甚至出現當時遊戲中還沒實裝的精英級イ級後期型
    • 吹雪初出戰就要面對這種對手，好殘忍
      • 結果被一航戰放出的艦戰給打爆了，好弱
  • 自pixiv上出現將イ級給萌化的同人創作後，深海驅逐艦一般在二次同人中以深海方的寵物狗吉祥物役登場

驅逐ナ級編輯

   

  • 於17夏E-2初次登場，是14秋的ネ級以來首次出現的新量產型棲艦
  • 比起上面的前輩們，ナ級不太像魚型，更像是被插了兩枝槍的洋蔥砂鍋魚頭
    • 實力除了雷擊值較高一點以外還算是正常雜兵，但是...
      • 到了2020初夏出現了裝甲等數值破百，還會開幕雷擊的II型看來是搭載了回天？，加上強大的防空能力已經成為甲作戰的最終煞星
      • 最慘的是營運實裝（或追加）更多對小型艦攻擊特化的陸航以及新的開幕雷擊艦娘，意味著這類單位將在高難度的後段作戰Boss點繼續攔路
      • 考量特定陸航多有航程限制，支援砲擊被認為是對此類單位的最佳解不考量資源損耗下

      • 19秋起甲乙難度常出現的へ級flagship除了有150血還有百點以上的雷裝值，已經成為水雷戰隊的新夢魘

    • 根據官方四格，ホ級有可能是12五十鈴的怨念
    • 從造型上看，雷巡チ級似乎是由ヘ級改裝而來

輕巡ツ級編輯

 
  • 在2014/9/12更新後開放的3-5王點中初次登場
  • 目前擁有輕巡中最高的人型度
  • 高達88（精英級是96）的防空數值以及武裝配置，讓大多數玩家一度推定為捏他亞特蘭大級巡洋艦（但亞特蘭大級未配備三連裝魚雷發射管）。然而後來亞特蘭大也有了對應的虛化型態。
    • 其強大的對空能力有時甚至能把放在較小格子裡的艦載機全部一次擊落，不禁令人懷疑她是否能發動對空Cut-In
  • 但是武裝配置的方式以及那殘念的身材也讓人懷疑有混合一些夕張的元素，其實是夕張虛化
  • 迴避能力很高，然後把火力都吸過去了還會Miss

重巡リ級編輯

 
  • 在1-3初次登場的單位
  • 夜戰地圖（5-3、2013年秋活E-4、E-5等）的惡夢，尤以旗艦級及改級黑岩莉莉為甚，人稱リ害
  • 身上的衣服是用昆布做的（by官方）

重巡ネ級編輯

     

  • 在2014秋季活動E-1王點初次登場的單位，直到2015/6/12更新後才首次在活動以外的地圖（4-5）出現
  • 重巡リ級的上位強化版，經常出現於活動地圖的王點中作為比較硬的肉盾吸收火力
    • 但形像上的人型度卻不如リ級，仍有一部分人外的特徵
    • 高達70的運使她理論上有比リ級更強大的夜戰實力，但目前有ネ級的夜戰點只有4-5下路的那一個而已
  • 據說是虛化的鈴谷
    • 17夏的旗艦級改成了航巡配置，你果然是虛化鈴谷吧
    • 19秋的改級換了新立繪，同時裝甲也暴增至鬼姬級數的255，我錯了，鈴谷才沒那麼變態
      • 該次活動在E6門神點必然出現兩個，導致勸退率暴增，自此成了提督們的惡夢。
      • 後來登場的改型還有先制雷擊能力，與ナ級II同為甲級作戰尾關的攔路虎，導致大量的提督退坑
  • 二創加入貓科的元素，似乎是從某討論版不小心打成重巡ネコ級開始的…
  • 2020年2月在和三越合作的線下活動出現了改型的購物mode立繪和DJ一樣立刻變廢宅
  • 2020初夏活動成為了第一個有（線上）限定立繪的量產型棲艦（夏mode）都準鬼姬級了還裝菜

雷巡チ級編輯

 
  • 臉上帶著白面具，在1-2的Boss點初次登場
  • 雖然是雷巡但不會開幕雷擊，即使是旗艦級也一樣
    • 但旗艦級有著和我方大北不相上下的雷擊值，如果不小心讓她完好地活到雷擊戰的話幾乎必定會中大破一艘艦娘
  • 虛化的大井和北上
    • 二戰時的美軍根本沒有雷巡←不要緊，還有魚雷艇←然後PT小鬼群出現了
  • 好不容易出了旗艦級結果被紅レ級搶走新裝備[2]
  • 據說面具下是個大美人

戰艦編輯

戰艦ル級編輯

 
  • 在1-3的Boss點初次登場的單位，低速戰艦，俗稱魯爺。
    • 從初期的普通型到後期的黑岩改級，自登場以來就一直以強敵之姿攔在玩家面前，出擊時被勸退的主因之一
  • 身著暗色襟裝、兩旁大型機砲為主要特徵 ，妳確定這不是鐵○貝的mega進化嗎？
  • 在初期的地圖經常於門神點或BOSS點登場[3]，對新手提督而言的最強守關人
  • 擅長唱歌
    • 黑歷史動畫版OP歌手「AKINO」在PV中的裝扮便是一身黑色襟裝，外加其本身又有著一頭飄逸長髮，PV開場揮手的瞬間便被鄉民們開玩笑說她是ル級
  • 2017年春活期間一度加入我方
    • 事實上是官方忘了調代碼，讓甘古特的立繪變成了ル級[4]
      • 所以是甘古特虛化

戰艦タ級編輯

 
  • 在一期4-1初次登場的單位[5]，高速戰艦，俗稱塔姐
    • 與ル級相比火力及裝甲較低不過被打中都是中大破根本沒差，但有著較高的回避和命中率。出擊時被勸退的主因之二
  • 白髮、上身水手服為主要特徵
  • 目前旗艦級的副砲換成了水偵，夜戰攻擊從Cut-in變成威脅更大的二連擊，通過配有旗艦級タ級的夜戰點即成為攻略5-3時必須面對的最大難關之一
  • 有提督發現タ級的內褲和島風的內褲很像，於是有タ級是島風虛化的傳言
    • 但島風妳不是驅逐艦m…清霜妳別望過去
      • 事實上是島風失蹤的媽媽
  • 夕級太太

戰艦レ級編輯

 
  • 在5-5初次登場的單位，分類是航母戰鬥群重雷裝航空巡洋戰艦，俗稱雷爺。
    • 深海棲艦中首位能全程參與日戰的5輪攻擊的全能單位所以其實是重雷裝航戰，妳的真面目是宇宙戰艦或者海底軍艦吧
      • 一說指其原型可能是近代化改修後的愛荷華號戰艦但是四條愛荷華級都沒沉沒所以不太可能，或者是小說家幻想出來的米利蘭土型航空戰艦[6]
    • 精英級以下時不會參加開幕雷擊意思是精英級以上就會
  • 除了各種攻擊樣樣都會之外，最犯規的地方是其能力和裝備幾乎都高到規格外
    • 誰來解釋一下為什麼戰艦會有我方最強正規空母1.5倍以上艦載量這件事[7]
    • 唯一的弱點是夜戰其實只是沒日戰恐怖而已，レ級發動Cut-in的機率是很高的
    • 通過配有一艘精英級レ級的門神點以及之後的隨機溝是攻略5-5的最大難關
  • 更可怕的地方是目前僅出到精英級而已…也就是還有旗艦級和改級未出現請期待將來的活動實裝成為新一代夢魘
    • 所幸在5-5之後所有新出的地圖均沒有她的身影=被下門禁令了，也請繼續吧
      • 結果在2017年冬季活動的B點出現，沒有六潛的提督大崩潰
      • 2019年冬季活動[8]還有稍後的春季活動甚至來個雙鬼拍門，大概是跟院長學的
      • 2019春季活動尾關和2020初夏活動E-4（高難度）也有出場
    • 伴隨2015年冬季活動更新當提督們忙著對理事長哀號的同時，有人挖出遊戲內部資料發現旗艦級的圖片已偷偷實裝，只是還沒正式出場
  • 實際做成戰艦的話…
  • 提督們給航改前的千歲姊妹上甲標的和反潛機也能變全能，只是性能慘不忍睹便是
  • 帽子下的造形很像雷（喂！清霜別看過來！）
  • 唯三有推出黏土人的深海棲艦之一。

  • 本遊戲最強大的隱藏Boss，在各鎮守府繁忙的時段就會頻繁出現襲擊提督們的白色惡魔。擁有反轉時間的能力，只要遭遇上，全艦隊就會被強制傳送回過去。此攻擊無法被防禦、迴避或反擊，因此是提督們最害怕的敵人。
    • エラー娘表示一切都是貓的錯，她只是代表運營來消滅牠們的[42]
  • エラー娘現有4代：同人創作最中常見的是黃蝴蝶結，雙手提著貓前爪的初代，俗稱「妖怪猫吊るし」，拿著的那隻貓則稱為「抜猫」；水藍色蝴蝶結有呆毛作土下座狀的2代目則為朝雲「妖怪猫土下座」
    • 3代目（紫髮）俗稱「妖怪猫パンチ」和4代目（綠髮，以貓遮臉）俗稱「妖怪猫垂らし」嚴格來說並不算是エラー娘一族，因為她們主要是出現在搶佔新提督名額的島民面前（3代目：F5太快時會出現，4代目：當看見她時就代表你今天搶不到名額），與上兩者的共通點其實就只有水手服+都有貓而已。
    • 甚至所謂的「初代」其實只是運營當初忘了製作連線錯誤的引導頁面，急就章下於是就拿了新手導覽裡面出現的立繪圖來兼用，所以真的要說的話真正的エラー娘其實只有2代目一位。
      • 雖然如此二次創作認受度最高的還是初代。因為初代機動性高才可以四處出擊襲擊各鎮守府和艦隊，而且土下座就沒辦法襲擊提督了
  • 就某些因素來說エラー娘是目前可以藉由自己的因素去做部分掌控，例如加快自家網速、VPN換比較快速一點的線路、清除快取和cookie、迴避路線繁忙的時段等等，如果以上都做到了還是貓妖頻頻就當我沒說就是伺服那頭我們無法掌控的因素了…
  • 傳說在繁忙時段用稀有驅逐艦公式建造會有極低機率入手圖鑑編號000的運營型驅逐艦1號艦「妖怪猫吊るし」
  • 不要嘗試反抗エラー娘，因為只會變成「妖怪提督吊るし」的悲慘結局
  • 不過自從官方不斷的改進遊戲的UI之後，再加上新玩家的人數開始慢慢減少，不止遊戲介面有明顯的改進，エラー娘不只在離峰時間絕跡、就連尖峰時期眾提督被貓襲的事件也開始減少了。

羅針盤娘編輯

  • 實際主宰每次出擊的隱藏Boss，指引艦隊前進方向的妖精惡魔
  • 雖然編成可以一定程度地調整航路的大致方向，主要的航向是由羅針盤娘隨意隨機決定的，所以只要羅針盤娘不樂意運氣不好的話就是一直轉圈或者快到Boss門前又來個栗田U字回頭的悲慘局面（尢其是2-4和3-4）
  • 與エラー娘僅在繁忙時間出沒不同，羅針盤娘每次出擊都會讓提督提心吊膽抓狂不已。高達八成的提督翻桌事件都可以看到羅針盤娘的ドヤ顔
  • 羅針盤娘有4位，每次出擊都會隨機抽選1位。注意以下名稱並非官方設定：
    • ショートボブ：茶色短髮有呆毛戴髮卡，幹勁十足的類型。
    • ヒヨコ：深黃色長單馬尾，頭上坐著一隻有呆毛的黃色小雞，元氣型，最愛亂轉羅盤
    • ショート：綠色短髮，一副隨時會睡著的表情跟聲音。可能跟這樣的角色設定有關，她轉的羅盤會轉得少減速比較慢，指向南北的機率也有點高
    • 魔女っ娘：長雙馬尾戴魔女帽的惡魔羅針盤娘，手裡拿著根路標。黑魔法是讓羅盤轉向提督最不想去的方向
  • 徹底暴走的提督決定讓羅針盤娘體會下被羅盤支配命運的滋味
    • 不過也存在著可以打敗羅針盤娘的提督
  • 自從2015年春季活動增加能動分岐的功能，她們的的阻礙變的小但是其中兩個該死的關卡沒有受影響
    • 另外Arcade版因為完全由玩家決定前進方向，所以羅針盤娘不會出來搗亂

工廠的妖精娘編輯

  • 實際主宰提督每次建造和開發的入手物的隱藏Boss，每天為提督辛苦忙碌的妖精惡魔
  • 雖然統計學上使用某些特定公式獲得特定類型艦娘和裝備的可能性會很高，實際上還是會由妖精娘系統從該類型列表內隨機抽選的。所以只要工廠的妖精娘們不高興，提督每次建造都會出\燃2彈4鋼11/、開發出企鵝，大建造出超大型扶桑、金剛足球隊、DMM陸奧
    • 多向間宮買冰淇淋收…獎勵她們就會連造出稀有艦或是稀有裝備
    • 大鳳跟隼鷹不都一樣是會載飛機的船嗎(ﾟ∀。)？
  • 不要嘗試性騷擾工廠的妖精娘或是在她們面前騷擾其他艦娘，否則後果自負
