阿貓阿狗

阿貓阿狗（TunTown）
本條目經常遭受不明人士清空刪除，敬請各位愛好者注意並即時回復
本條目內含劇透，沒玩過遊戲又不想被貓派氣功扁的話請不要隨意反白。

目錄

解說編輯

  • 《阿貓阿狗》是以《軒轅劍》系列出名的大宇資訊DOMO小組所製作的一款RPG遊戲，於1998年2月15日發售。
  • 算是小品遊戲，雖然有些子供向，但是也有嚴肅的一面。
    • 遊戲過程中充滿對社會畸形面的諸多批判。
    • 別忘了尾聲的蝦格里拉火山爆發，雖然沒交代但肯定發了許多便當。
  • 遊戲中共有92首音樂，光碟可以直接放進音響裡聽，因此可以說遊戲光碟本身就是原聲帶。
  • 2005年7月7日推出了3D版續作《阿貓阿狗2》，但已被玩家們視為黑歷史[1]。
  • 2013年以懷舊版為名義在iOS上推出，但操作非常困難。現已下架。
  • 2017年出手遊，唉....半年內倒了。
  • 目前可以在方塊遊戲平台上免費下載正版遊玩（用的是DOS-BOX模擬器）。

特色編輯

  • 各角色（包括貓狗等動物）的造型、名字都很特別，對話也非常的豐富有趣，連NPC也不例外。
  • 引入類似冒險解謎遊戲的勘查機制，尋找道具不必再走到寶箱或是容器裡，在移動滑鼠時看到游標變成爪子的時候，就代表那裡有道具或機關，可以說到處都有隱藏道具。
    • 小孩：「馬麻妳看，有小偷！」
    • 媽媽：「乖，不要亂說。那可是勇者大人喔！」
  • 時間雖然設定在使用高科技的2020年，但是也有一些超自然的事物。
    • 遊戲一開始的鬼屋就有天使和幽靈的對話。
    • 中間有段劇情是和隱居在圖書館的前鎮長學法術對抗惡魔。
    • 遊戲後期還可以使用瞬間移動。
    • 當然也別忘了能和動物對話的主角。
  • 因為是DOMO小組製作的，所以裡面也有DOMO工作室在電影院螢幕後面。

故事背景（引自遊戲包裝）編輯

一位能和動物溝通的12歲男孩──樂樂，回到闊別已久的故鄉「木桶鎮」，發現鎮上的貓群正四處作亂！ 樂樂決心挺身而出，為鎮上解決貓的問題，便努力四處奔走，想號召被貓打得落花流水的狗兒們一起來對抗貓。 不料就在這時，他發現在貓群作亂的背後，一個更邪惡的陰謀正在鎮上悄悄展開……

系統編輯

  • 合成系統：
    • 即《軒轅劍》系列的煉妖系統，可以將兩個看上去像垃圾的道具合成裝備、恢復道具或是帶屬性攻擊的消耗性道具。
    • 一樣有合成等級限制，不過頂多是無法合成，倒沒有會合成出大眼蛙的狀況。
    • 不乏黑色幽默的要素：「鉛筆」+「紙」=「政客簽名」是貓狗用防具，但卻沒有防御力。只有更高級別的「政客證明書」和「主席的駕照」才有實際用途；唯一沒有道具圖片的大米特有的防具「國王的新衣」也沒有任何防禦加成。
  • 時間流動：
    • 遊戲中的一小時等於現實中的一分鐘，會有日夜變化。
    • 到了半夜時段時會出現宿屋指令進行睡眠。
    • 鎮里的居民的在不同時間內會出沒在不同位置，亦有每日固定的走動區域。
    • 因此要找某些NPC便是遊戲中另一項挑戰。
    • 當玩家站著不動時，流失的生命和體力會高速的逐漸恢復雖然說長時間移動也是有恢復效果。
  • 開發系統：
    • 可利用戰鬥或完成任務所得到的「創意點數」，依據不同屬性來開發各種裝備和道具。
    • 總共分基礎科學、電力、冷氣、熱能、生化、時空、金屬、合成、幻想、機器、醫學和料理共12種項目。
    • 《軒轅劍肆》的天書系統很可能是從這衍生來的。
  • 支線任務：
    • 除了主線任務之外，也有支線任務可以接，用空心的五角星表示。完成任務後基本會獲得一些獎勵。
    • 遊戲中有兩個支線任務是無論如何也完不成的。這不是BUG，而是有意而為之。
    • 某個後期的支線任務裡有選項是把自己的PDA交出去，千萬不要點錯……
  • 戰鬥：
    • 使用一碰到敵人，敵我雙方就分站左右兩邊的半即時制戰鬥。
    • 戰鬥並不會另外切換成「戰鬥畫面」，而是「就地開打」，此時有路人的情況下還會挺識相的快速繞跑。
    • 選擇攻擊目標時為無差別判定，一不小心還會打趴自己人XD。
    • 人物升級之後可學會各式特技，少數特技要在特殊情況下才能學。
    • 由於等級上限不算高[2]＆透過本作特殊的「繞跑密技[3]」情況下，到後期的戰鬥主要是為了累積「創意點數」...
    • 打輸了不會GAME OVER，而是全員夾著尾巴逃離此區。

角色介紹編輯

主要角色編輯

  • 樂樂
    • 本作主角，喜歡大自然及動物的12歲少年，擁有與動物溝通的能力
    • 因父母長期在國外工作不回家，才回到木桶鎮找他叔叔，正好遇上了貓群作亂，才有了集合眾人之力解決貓害的想法
    • 使用武器為雨傘和玩具槍，不裝備武器的話還可看到他赤手空拳攻擊
    • 不務正業[4]
    • 在旁人眼中看起來就是一個帶著狗到處虐貓的變態少年，不過因為當時鎮上貓害嚴重，所以沒人阻止
    • 穿著疑似從秦朝以來就存在的藍格條紋大衣，但本人卻使不出「龍颿」[5]
    • Davis[6]
    • 口才很好，劇中曾跟樂樂吵嘴過的幾乎都反而被婊回去
      • 唯一辯不過的大概只有銀狐
      • 2代有變弱的跡象
    • 戰鬥全勤角色，遊戲末期五人眾的其一
    • 故事結束時率領木桶小隊前去大都市救災。
  • 阿康
    • 樂樂的兒時玩伴之一，能言善道，擅長交際，崇拜擅長暴力的英雄人物
    • 怕鬼，樂樂要到鬼屋裡的時候站在外面不敢進去
    • 武器為拳套，使用部份特技時還會換裝成消防員、隆、小天使與神槍手
    • 遊戲末期五人眾的其一
    • 2代造型變成紅色上衣配迷彩褲，增加把妹失敗跟大食屬性
  • 阿吉
    • 樂樂的兒時玩伴之一，天才發明家，PDA裡大部份的程式都是他寫出來的
    • 「真搞不懂你們，相對論有什麼難的……」
    • 絕大部份時間都待在做為小隊基地的鬼屋二樓裡做研究
    • 曾經加入隊伍，不過只有挨揍的份…
    • 帳面能力全99 可是一出手攻擊就會死機
    • 無法逃跑
    • 2代可以戰鬥
  • 鬼屋之鬼──約翰
    • 樂樂的叔叔，在木桶鎮無親友，一個人住在大屋裡，於故事開始前幾年過世，死後因為他的關係讓房子變成鎮民口中的鬼屋
    • 死不瞑目的原因是他出資建造的路被取做「不是約翰的路」，所以賭氣不肯上天堂
    • 非常疼愛姪子的叔叔，還把自己的房子借給樂樂當基地，遺產寶藏本要分給樂樂[7]
    • 年輕時曾和多福一起到大都市留學
    • 曾經是拜金教徒的偶像。
    • 故事結束的時候終於想通上了天堂
    • 全能力都是99 但沒機會讓他戰鬥

    • 統領著貓群作亂、身份以及目的皆不明的怪人，戴著南瓜面具與紅披風，行動像小丑
    • 跟Elvis Presley完全沒有關聯，部分動作卻很像Michael Jackson

  • 紅珊瑚

    • 雖然看起來不好相處，不過心地不壞，和樂樂成為同伴後，慢慢學會如何關心朋友。

路上遭遇的敵人編輯

  • 野貓
    • 僅在早期出現，有十幾個種類，越後登場的越會使用各種道具。
    • 早期在西區徘徊，北區大戰時轉移到北區。
  • 玩具師清潔車
    • 玩具師發明的機械，皮厚血多但獎勵超少。
    • 怕電
    • 性能隨著劇情不斷地加強。
  • 捕狗隊員
    • 擁有自身專屬招術。
  • 下水道生物
    • 怕熱，但若對它們用像冰水球、阿康的消防栓等水屬性能替它們"回血"。
  • 搶匪
    • 會使用全體攻擊道具"跳蚤特攻隊"搶匪身上怎麼會有比貓狗還多的跳蚤？
  • 輪鞋黨
    • 戴著傑森面具、身穿紅色和藍色緊身衣、腳穿輪鞋、手持曲棍球桿的不良少年。其中藍色的會使用全體電系的道具"天電導雷管"。
    • 除了在街道上，在大都市辦事處裏也時常見到。
  • 大都市噪音團
    • 由主唱、鼓手、吉他手及鍵盤手組成。
    • 在大都市辦事處走劇情路線時必定會交手一次。
  • 保鑣
    • 會使用"震撼炸彈"像是丟免錢似的，皮厚血多，普通攻擊也不算差，若沒有充足準備或等級不足時就對上要有被痛宰的準備。
  • 玩具兵團
    • 除了玩具熊"刀疤老大"事件中作為主要的路邊敵人外，後期也會再出場
    • 通常會大批出現，後期在路邊戰鬥累積創意點數幾乎都是打這類敵人
    • 怕電，但小丑盒，以及有一種會用電系攻擊的機器人階除外
  • 野狼
    • 在山路出現。據其他動物所言，很懂得如何團體行動。
    • 會使用全體攻擊道具"跳蚤特攻隊"。
  • 山豬
    • 在山路出現。
