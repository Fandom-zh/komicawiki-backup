故事中經歷[編輯]

  • 1967年12月28日出生，當時已年老的父親言峰璃正為他取名「綺禮」，這個名字有祈願美麗的意思。
  • 1981年自曼雷沙的聖伊那裘神學院跳滿級兩年，以首席生的資格畢業。
  • 但後來加入了教會，並在教會中多次轉職。曾任為第八秘跡會的司祭（璃正的部門），亦曾擔任教會中的代行者。
  • 言峰無法安居一個職位，是因為對自己內心歪曲的本性感到困惑，而試圖尋找自己真正想要的正確道路。他的內心無法和一般人一樣，視美好的事物為快樂，而是對相反醜惡的事物為樂。言峰對教會的虔誠也是希望上帝能解答他內心的困惑。
  • 為了改變自身的異常並感受到常人所擁有的幸福實感，言峰做了許多的嘗試，成為代行者也是其中之一，而他的最後一次嘗試，是與一名叫做克勞蒂亞·赫特希亞[12]的女性的婚姻。
    • 妻子是已經身患死病，沒有未來的女子。言峰說「因為這樣的女人才選，這個女人以外的人無法選擇，這個基準，自己怎麼想都無法判斷。」這樣敘述
    • 在這段婚姻中，妻子理解言峰的扭曲去愛著言峰，言峰也很努力的去愛妻子，兩人間也生有一女。
    • 可到了最後，言峰依舊只能以妻子的痛苦感受到快樂，無法透過妻子的愛獲得常人的幸福，他查覺自己無法正常的去愛人而認為生來就是錯誤的自己應當抹殺。
    • 所以言峰在自殺前到妻子的房間向她坦白。
      • 言峰：我沒能愛你。
      • 赫特希亞：不對，你是愛著我的。
    • 赫特希亞為了證明言峰內心是能愛人，有生存價值之人，而在他面前自殺。但言峰卻只是一言不發的轉身離開了病房，同時向自己以來所信仰的神的教誨訣別了。
      • 言峰：我內心確實很難過…因為沒能親手殺死她（病嬌？）。
    • 言峰發現了自己想親手殺死妻子的感情，也不再嘗試改變自己的異常，就此接受自己的扭曲。
      • 以本篇HF線的描述來說，言峰在四戰之前就認清自己的惡性並一直在尋找對應自身扭曲本性的方法，而妻子是他最後一次的嘗試，這段婚姻讓言峰對自己的本性感到絕望，試圖自殺過，而他老婆的自殺，就是為了阻止言峰的自殺。自從妻子死後，言峰便查覺並接受了自己的扭曲，正式走上外道之路。
    • 但是，妻子的死亡也成為言峰學習治癒魔術的契機。
      • 自己的本性沒有改變的事實會讓妻子的自殺變成無意義的舉動，但不知為何，言峰不希望她的死毫無價值，所以才要去證明自己的生是有價值，也放棄了自殺的念頭。
  • 不久後，言峰便加入了第四次聖杯戰爭，最後敗在衛宮切嗣手下。
    • 言峰的妻子在Zero開始的幾天前過世了，或許是虛淵玄想加強故事中的情節轉折，Zero中把本來應該透過妻子領悟的「我想親手殺了他」的感情轉移到父親身上連帶對自己惡性也變成不瞭解的狀態，並讓吉爾加美修去說服他。
    • 見下方Fate/Zero欄目。
      • 本篇只是略為提及，後由虛淵創作了Fate/zero補完，不過ZERO中也有不少和本篇不符的地方。
  • 第五屆聖杯戰爭開始前六年，言峰在一次代行者的工作中，結識了魔術協會的魔術師巴婕特‧法迦‧克米茲。
    • 當時言峰和巴婕特都失去了所有同伴，雖然雙方所屬的組織不同，但因為目的一樣，所以兩人還是進行聯手。
    • 之後言峰和巴婕特也數次合作過，而在這過程中，巴婕特對言峰這位可靠的前輩產生了好感巴姐真是少女心。
      • 巴婕特在HA中試圖尋找言峰的蹤跡，最後被卡蓮斷言「言峰綺禮從來就沒有喜歡過妳」。

Fate/Zero[編輯]

注：Fate/zero和Fate/stay night的設定上略有一些不同，閱讀時需要留意，然而諷刺的是本文愉悅的發祥地卻是源自於此處。

  • 在第四次聖杯戰爭開始的三年前，言峰被聖杯挑選成為Master並獲得令咒。但他本人不明白為什麼他會被選為Master，因為空虛的他沒有任何所要的願望。
  • 因為父親言峰璃正和遠坂時臣是至交好友，而指示他加入魔術師協會，拜時臣為師。在聖杯戰爭中全力協助時臣。
  • 這三年中言峰向時臣學習各種魔術，但這些行為仍未能能填補他的空虛。
  • 在聖杯戰爭中，言峰的Servant為Assassin，而時臣的Servant為Archer。言峰故意派遣一個Assassin去襲擊Archer，然後任由Archer殺死他，以此引起言峰已經戰敗的錯覺。
  • 實際上言峰一直隱藏在教堂中，並派出剩餘的Assassin偵查和搜尋情報幫助時臣。
  • 其間言峰曾主動計劃襲擊衛宮切嗣，但只打傷了他的妻子艾莉絲菲爾和助手舞彌，未能和切嗣交手。
    • 期間艾莉絲菲爾把他捆在一棵大樹上，但他在雙手無法移動的狀態用寸勁將樹上打斷。
  • 後來為試探Rider，言峰動用了所有的Assassin，但Assassin們全被王之軍勢所殺。
  • Archer因不滿時臣而對言峰未曾注意到的本性感興趣，而勸言峰仔細觀查其他Master參戰的動機，並指出言峰其實有著對他人的痛苦為樂的興趣，但被言峰否定。
  • 言峰因為下意識的對間桐雁夜的不幸而對他感興趣，在雁夜被時臣重傷後暗中將他救下。
  • 回到教堂後發現父親被殺，並在遺言中留給他所有剩餘的令咒。
    • 而言峰內心和妻子死時有同樣的感覺：遺憾自己沒能親手殺了父親。
      • 不過zero的言峰由於被設定成未理解自身本質的狀態所以有關妻子的回憶與stay night是完全不一樣的。
      • 最大差異stay night是言峰當時是理解自身本質，妻子是為了證明他能愛人而自殺，zero是言峰當時不理解自身本質，而妻子未注意到他的本質就病死了。

  • 在Archer的唆使下，言峰決定背叛時臣參加聖杯戰爭。
  • 事後言峰告知Archer時臣計劃在聖杯戰爭結束後，用令咒殺死Archer來到達根源。因此Archer決定和言峰合作。
  • 臨行時送了言峰一把Azoth短劍為禮物。但在他離去時，言峰在他背後用劍肛刺死了時臣。
  • 言峰告訴他如果和他合作，就可以幫助雁夜得到聖杯。

    • 言峰在觀賞雁夜的悲劇後感到以前喝過的酒有著美味的感覺，開始了惡的悟性。
  • 在聖杯戰爭最後和切嗣交戰，但因為切嗣內置了外掛Avalon而被他所傷，最後發現聖杯本質的切嗣將言峰槍殺。

  • 目睹聖杯引起的大災害後，言峰這時終於察覺到自己的本質，並高興自己終於找到了自己所要的目的。

  • 代表教會擔任第五屆聖杯戰爭的監督。
    • 推薦自己的舊識巴婕特作為魔術協會的代表參戰。
    • 在聖杯戰爭開始前，言峰去拜訪來到冬木市的巴婕特。由於巴婕特對言峰抱持好感與信任，所以對他完全沒有提防，結果就這樣遭言峰暗算，並被奪去了右臂與令咒，言峰因此取代巴婕特成為Lancer的Master。
  • 聖杯戰爭開始後，言峰讓Gilgamesh待命[13]，而派出Lancer去偵查其他每一組參戰者的行動[14]。
  • 在士郎因為Lancer的襲擊而成為Master後，凜帶士郎到教堂找他。
    • 言峰知道衛宮士郎為切嗣之子，所以勸說他為避免聖杯落入惡人之手而加入聖杯戰爭。
    • 在這次談話中，由於言峰一次次挖苦的話語，使士郎對言峰這個人物抱持了強烈的排斥。

 黑鍵 Black Keys
教會人員所使用的長劍，但其實也是魔術禮裝。

根據衛宮士郎的分析，此劍大約一公斤重，外型和刺劍類似，但奇怪的是劍身的重力主要集中在劍尖，類似Assassin的投擲短刀。此劍本身不是用來斬擊，而是和士郎分析的一樣為投擲武器。投出後如果擊中，會將目標的身體定身。

雖能傷到英靈，但對他們的效果有限。對幽靈和死徒卻很有效。

教會的殺手可以同時扔出很多把劍。高手甚至可以只攜帶劍柄，而在需要時再將劍身投影，如此即可將劍隱藏於衣中來突擊敵人。

      • 這句話顯示言峰還記得克勞蒂亞女士，並與卡蓮做為比較
  • 「正直事後処理がめんどくさい」
    「說白了，事後處理實在太麻煩！」
    「啊呀…事後處理…」
    • 分別是幻想嘉年華中「第5次魔術師大激突！聖杯戰爭」和「格蘭披治聖盃賽車」對於事後處理的形象崩壞抱怨。
  • 「あたためますか?」
    「溫熱嗎？」
    • CCC中購買物品時的其中一句話。
    • 如果只是買食物時說還好，問題是買回復藥和禮裝時也會說出這句話……要穿戴的東西被你溫過還敢穿上身嗎！？
      • 男主人公討厭的東西：温められた商品（被溫熱的商品）。
  • 「────フ。英雄王に剣を向けるなど﹑そんな莫迦はそこの犬で十分だろう。」
    「────哼。對英雄王刀劍相向，這種笨蛋有那邊的狗就足夠了。」
    • CCC中在和Lancer一起與主人公和金閃閃對打後，對金閃閃下次直接動手對打的邀請的回答
    • 在說完這句話後，在一邊的Lancer就當場發飆了
  • 「令呪を持って命ずる。紅洲宴歳館・泰山の麻婆豆腐を10皿、1分で完食してくるがいい」
    「以令咒命令。紅洲宴歳館・泰山的麻婆豆腐，在1分鐘之內吃完10盤。」
    • CCC中慰勞Lancer。Lancer你果然是幸運E。平均要每6秒食完一盤，比自殺更慘
    • 閃閃聽到這“獎勵”後也被嚇倒了……
  • 「ラーメンという名が全くの飾りだ
    麻婆の海の底に申し訳程度に沈んでいている！」
    「拉麵這個名字完全只是裝飾
    以很遺憾的程度下沉沒在麻婆的海底之中!」
    • 在魔法少女伊莉雅3Rei中，面對伊莉雅吐槽拉麵在哪兒的時候，而作出的回答
  • 「食い逃げとは舐められたものだな。
    だが、ちょうど豚骨が切れていたところだ。文字通り身体で支払ってもらうとしよう」
    「心臓よりも肝臓や腎臓の方が高く売れると知っているか?
    最後の晩餐が私の麻婆だったことを幸運に思い逝くがいい……！」
    「吃我霸王餐，何等看不起人。
    嗯，正好豬骨用完了，就按字面含義用身體來償還吧。」
    「比起心臟，肝臟和腎臟要值錢得多，客倌您知道嗎？」[23]
    「作為最後一餐，能夠吃到我的麻婆，勢必也能含笑九泉了吧……！」
    • 魔法少女伊莉雅中煮拉麵給伊莉雅和田中吃後，要伊莉雅和田中付錢，結果得知兩人都沒錢時的發言。
    • 當然，作為主角怎麼可能會因此領便當，所以之後兩人就在路過的小金閃幫忙付錢的情況下得救了。
  • 「令呪を以って命ずる、台風の上陸を阻止せよ！！」
    「以令咒命令。阻止颱風登陸！」
    • 其實是中田讓治在2014年10月9日發的一篇推特中模仿綺禮的語氣發言
      • 文中的颱風指的是2014年10月初登陸日本造成2死5失蹤90多傷的黃蜂颱風。
  • 「投影，開始」
    • 中田讓治在HF劇場版橫濱見面會(2017年10月22日)的開場白
