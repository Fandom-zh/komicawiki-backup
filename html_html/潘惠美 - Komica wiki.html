解說[編輯]

  • 東京都出身的日本女性聲優，1989年6月3日生，母親是同業的老牌聲優潘惠子。
    • 不過和井上喜久子的女兒井上ほのか（井上穗乃花）的狀況不太一樣，潘惠美是因為父母小時候離婚而冠母姓。
    • 順帶一提，有著中文姓氏並不是偶然，而是潘惠子的父親（即潘惠美的外公）是台灣人。
  • 所屬事務所為於Atomic Monkey，該事務所知名聲優有關智一、折笠富美子、木村昴、榎木淳彌...等。
  • 成為聲優契機是在她童年時期憧憬母親替角色配音，並從中得知聲優這份職業並以此為目標。

人物特點[編輯]

  • 聲線廣泛，由活力十足的正太、開朗少女、弱氣系及動物都有配過;也可以以低音聲演冷酷系、熟女及反派角色。
  • 不過相對地欠缺個人風格，故出道以來配角佔了演出角色的大半。
    • 於2017年獲得聲優賞助演女優賞，曾經與同場得獎者花江夏樹合照。
    • 可能一出道已經跟資深聲優共演的關係，潘以出演子供番及黃金番數量遠勝於深夜番新人聲優哪會有這麼好的待遇
  • 與特攝很有緣份，除了假面騎士系列外，三大特攝系列中已有兩個系列有份演出。
    • 如手裡劍戰隊的十六夜九衞門，超人力霸王Geed等的貝加。
    • 在戰隊特番中擔任歷代首位女性綠戰士綠忍者[1]九重ルナ[2]，這種情況跟戶松遙及內田真禮不同的是前者分別飾演反派及助拳人，潘直接飾演戰隊成員，是戰隊史無前例的。
    • 非常受到円谷的寵愛，當怪獸娘擬人化計劃動畫化時更直接找她為艾雷王配音。
  • 除特攝片之外和鋼彈系列也頗有緣分，經常在不同系列的作品裡擔任配音，從主角到配角甚至路人都出演過。
    • 而在搞笑作品的鋼彈桑中也接演了母親曾演出的拉拉‧遜一角。
  • 相對以工作為主的女聲優，她對有份參與配音的作品相當有愛。
    • 她的推特有不少デジモン的周邊及童年時期的照片，數碼獸的周邊更是一直留到現在。
    • 《Hunter x Hunter》試鏡中因為說出自己從小時候買了第一本漫畫的說話令在場審查員感動人心而得到小傑演出機會，其推特與小傑皮套合照的橫幅一直未被換過。
    • 甚至得到Happiness Charge 光之美少女！白雪姬/公主天使一角，在劇場版宣傳時感動落淚。
      • 2021年2月2日「光之美少女日」在推特上會有很多推特網友PO出對光美的喜愛推文，且身為配音白雪姬/公主天使的潘惠美也有發推文表達出對光美系列和自己配音的角色的熱愛，並把這篇推文設為釘選頭版推文，在IG上更是發了一篇對光美的長篇心得文感動不少大友們。

人際關係[編輯]

  • 潘惠子
    • 母女關係、潘惠美吹替代表作為2013年版魔女嘉莉的嘉莉懷特，而潘惠子聲演1976年版吹替的嘉莉懷特。
    • 她的主役為《Hunter x Hunter》的小剛，小剛的姨媽米特一角由潘惠子演出。
    • 於早年的《燃燒吧！小露寶》最終話曾經以非正式童星身份客串，而當時潘惠子有份參與聲優主要演出。
    • 之後《鋼彈桑》短篇動畫化時擔任拉拉．遜的聲優，而原作中擔任聲優的正是潘惠子。
    • 但自從得到《Happiness change 光之美少女》白雪姫一角後減少了捏他，而當時動畫評價較為負面，在她出道早期動畫連帶本人被婊，但往後演出評價有所回升。
  • 伊瀨茉莉也
    • 自新全職獵人與她一起演出後，經常在不同作品共演。
    • 於後來的《絕對絕望少女》中為故事中的反派組織——希望戰士的隊長大門大配音。而在《Hunter x Hunter》中奇犽的聲優伊瀨茉莉也同样在本作出演希望戰士。不知是巧合還是有意捏梗。
    • 二人都已經當上了光之美少女，但伊瀨茉莉也在更早之前就已經當上了
    • 最初生放送經常跟伊瀨一起，目前正在跟種崎一起直播。
  • 澤城美雪
    • 承上，值得一提的是，她也同樣有在新全職獵人和《絕對絕望少女》（不過不是希望戰士，而是前作就已經登場的配角）參與演出，甚至在翌年當上了追加光美
  • 本渡楓
    • 後輩本渡楓在雜誌訪談中提到與潘關係很好，知道《碧藍幻想》的貞德達魯克是她獻聲後相當興奮。
