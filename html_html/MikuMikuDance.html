MikuMikuDance

MikuMikuDance/MMD/萌萌達/咪咪蛋死

目錄

解說編輯

  • 台灣電競戰隊閃電狼的前隊員，因為表現差強人意時常被鄉民痛罵「咪咪蛋死爽沒」
  • 由樋口優自2008起開發的Windows動畫製作軟件。因為壓縮方法的關係程式檔案相當小，此外免安裝即可使用。
    • 美中不足的地方是日系軟體慣例的開發者眼裡世界只有日本人語系問題使得字體時常變成亂碼，所以要修改Unicode程式的預設語言設定[1]。
      • 此問題在Windows 10更新後更加嚴重[2]，迫使原本已經休息的樋口P再次出山，將程式一路更新至9.26版本。
      • 但Windows 10升到1903版之後又出現UI跑掉、輸出AVI用解碼外掛消失等問題......不過那些影響不大且容易解決[3]，同時這些問題已在在9.32修正樋口P辛苦了
    • 此外相較於3DSMAX、Maya、Blender等專業的3D動畫軟體，MMD欠缺Render Pass[4]的輸出功能，因此要將輸出的影像放到Photoshop/After Effect等軟體進行更專業的特效加工時會變得比較麻煩不然就是得手動放綠幕模組或直接用MME在MMD軟體產生
      • 同時現行的版本必須安裝微軟的3個runtime可轉散發套件[5]才能啟動
        • 原本樋口P提供直連官方下載的連結，後來因為三個套件相繼結束配布之後改成樋口P在自己的VPVP官網提供下載
  • 正如其名主要是讓Vocaloid角色跳舞的軟件，但在2010武力介入P推出MME(MikuMikuEffect)插件後，玩法和質素大增，日常劇、戰鬥甚至飆車都成了短片內容
  • 目前民間流傳最廣的中文化版本是天知道P翻譯的7.39版本
  • 原本MMD內建模組只有初音未來且只能使用該模組編輯單人舞步，更新後除了追加鏡音連、鏡音鈴、巡音流歌等VOCALOID角色的MMD模組之餘，還開放使用者自己做模組並放在MMD上運行
    • 起初大部分的模組都是作者細心自己捏出來的，當然模組的作者基本上都會貼出使用規定讓其他玩家遵守
    • 不過還是會有一堆害群之馬違反作者給出的規定，通常違反規定的原因是二次配布、R18改造、洩漏密碼提示、騷擾作者以及在VRCHAT使用，其中第三者～第五者是近年最常見的問題
      • 結果就是製作者一怒之下往往會「鎖國」：設置複雜的日文密碼、放置在NICO社群甚至需要日本IP的網站，更有甚者明文禁止非日本玩家使用、極端的情況下還會退坑並停止素材配布
      • 然而真正的害群之馬難以被懲罰之下，形成作者退坑—更多的創作者（尤其是海外）因為缺乏素材卻步的惡性循環
    • 同時有部分MMD愛好者會利用黑科技將電玩遊戲中的3D模組[6]提取出來並做成MMD用模組並放在網站上公開[7]
      • 這些從遊戲提取出來並做成MMD專用的模組雖然都是遊走在灰色地帶，不過基本上都被大部分MMD玩家及作者(尤其是日本地區的玩家)視為禁忌雖然還是有人會用就是[8]
      • 所以使用這些提取出來的模組之前必須三思，不過當然還是不建議使用
      • 有部分模組的作者甚至因此給出禁止讓其模組跟從遊戲提取出來製作的MMD模組混用
    • 另外並非所有MMD模組都是隨處可得或是免費下載，有些特殊的MMD模組得必須購買特定的電腦雜誌或是同人誌並在其附送的特典找到其特典代碼並到特典網站輸入代碼下載或是直接把特典CD放入電腦並入手，或是去Melonbooks DL或DLSite等專門提供數位版同人作品的網路商店付費購買
      • 基本上特典附贈的MMD模組其實也比網路購買途徑獲得的MMD模組比較難獲得，加上近年因為部分雜誌停刊或是部分商品停產的緣故，那些只能在特定雜誌或產品獲得的特典模組的入手難度也相對更高
      • 雖說如此，還是一樣有害群之馬無視規定直接把特典附贈或是用購買管道獲得的模組無斷二次公開或是跟有拿到這些模組的玩家伸手要

基本操作流程編輯

  • 解壓縮檔案時，請記得先把解碼器的語系設定好，否則模組很可能無法運行!
  • 1.開啟程式後，代入人物模型的PMD或PMX檔案以及場地物件的(ry
    • 初期的模組專用格式為PMD檔，後來推出了放寬較多限制的PMX格式，因此近年角色配件和場地物件多半已改使用PMX檔。
    • 某些場地或配件會使用X檔作為檔案格式，必須在配件欄匯入。然而因為該檔案格式不便調整大小外的參數，近年模型製作者多已改用PMX檔。
  • 2.替角色導入舞步的VMD檔案並對動作逐一做修整
    • 基本常識:30畫格等於一秒[9]
    • 通常只要將動作尺度依照角色身形做修整，就不會有太大的問題
  • 3.導入攝影機的VMD檔案
  • 4.輸入濾鏡或特效的X檔以及fx檔
  • 5.導入WAV音檔
  • 6.輸出成AVI檔
    • 輸出影片的畫面尺寸要先在上方工具欄設定(預設為640x360)，FPS以及編碼器則在輸出前設定即可
      • 初期編碼方式為無壓縮，但無壓縮輸出的影片容量相當大，因此也建議選擇想要使用的編碼器之後再進行輸出即可（愈能輸出不會有明顯失真的影片的編碼器愈好），不過有些編碼器(如UT Video Codec)則是需要另外安裝，但安裝之後可能需要數次的測試才能正常使用

相關軟體與插件編輯

  • MikuMikuEffect
    • 作者：武力介入P
    • 2010年9月4日公開，讓MMD可以使用的特效種類從此大為增加，質素也因為使用fx檔而非x檔作為插件而提升。
  • PMD Editor/PMX Editor
    • 作者：極北P
    • 2008年12月22日公開，簡易的MMD模組編輯程式，主要是可以給模組進行換衣服、修改權重及尺寸、更改格式等簡易修改。
    • 不過如果要進一步修改就必須加裝一些外掛補丁。
    • 最主要的功能是製作紳士MMD
  • ray-mmd
    • 全名Raycast Shader，讓MMD模組套上類似UE等3D引擎的等級的高品質特效。
      • 不過操作方面比其他特效更為複雜(雖然有好心人士提供入門版)，且很吃效能，若電腦配備不好不建議使用。
  • iKenoPollish
    • 簡易版的ray-mmd。
  • AutoLumious
    • 使場上物件發光的插件。
  • nanoem
    • 對應MAC的版本，然而許多功能有問題。
  • MikuMikuMoving
    • 由Mogg製作的版本，操作及介面有做一些特化，更有內建一些預設特效。
    • 相較原版在建立動作上簡易不少。
      • 由於使用率不及MMD，已在2016停止更新。
  • MikuMikuVoice
    • 樋口P親自開發，能建立同時對應VOCALOID以及MMD兩邊的發聲設定檔
  • MMBG
    • 系統更新後令背景仍能播放AVI的腳本。
  • MikuMikuTelop
    • MMD的字幕插件。
    • 但是操作性比起影片剪輯程式的字幕插件複雜，使用率較不普及。

常見出場系列編輯

  • VOCALOID本家
    • 終究是製作舞步、演示技術的主流載體，目前的角色模型大多以Lat式以及Tda式、Sour式的人物為基製作（包含亞種）。
      • 同時又是改造模組最為氾濫的一個系列，甚至因為太過氾濫或是連打擦邊球而逼到讓原作者修訂使用規範。
  • 東方Project
    • 東方的MMD創作中二創劇中劇佔了不少，甚至有一些MMD的影片是以還原東方遊戲中那些絢爛彈幕為主的。
    • 此外最有名的還有ベホイミProject製作的以重現TAS影片的「變態」玩法的TAS魔理沙系列，甚至還有在YouTube開設專屬頻道
  • 偶像大師
    • 雖然偶像大師本來就是出自家機，也可以自己用輸出舞步的錄影影片(甚至在某款作品還有密技可以去背)剪輯成需要的跳舞MV，但MMD的攝影機角度自訂度很高，因此也有一些P會另外以更換衣裝的MMD模組來製作MV或甚至做出劇中劇的演出。
    • 以本家(765Pro)為主，其他系列的模組也有人製作。
  • 艦これ
    • 目前以日常劇、演示舞步為主流。
    • 也有部分創作者會讓艦娘重現遊戲中的戰鬥，甚至使用鋼彈系列的武器作戰。
      • 不過製作戰鬥動畫的人終歸少數，很多作者到後來索性不做儀裝只做角色。
    • 最受VRCHAT問題困擾的圈子，多位模型製作者如へな羊以及槭樹因為該平台限縮自家模型的配布。
    • 還有一說是無需渲染和太多動作調整，就能輕鬆獲取大量點閱的紙芝居（連環靜畫）導致市場萎縮。
      • 其實其他系列也有上述兩個問題。
  • 碧藍航線
    • 由於主題與上者相似，也有MMDer會拿來與上者Crossover
  • 《ブルーアーカイブ-Blue Archive-》（蔚藍檔案）
  • 義呆利 Axis Powers
  • Code Geass 反叛的魯路修
    • 也是紅極一時的系列，但主要的魯路修等人的MMD模組現在已經結束公開。
  • 刀劍亂舞
  • 美少女花騎士
  • 銀魂
  • JOJO的奇妙冒險
  • 進擊的巨人
  • 少女與戰車
  • 動物朋友
    • 監督降版事件後痛毆咕嚕嚕還有天藍怪的二創劇情成為主流。
      • 動物朋友3推出後也陸續開始有其他的Friends模組。但因為上述的事件人氣不如當初。
    • 有部分模組為特殊模組，入手途徑基本上是動物朋友MMD同人本的特典CD或是追蹤作者推特。
    • 此外也有玩家因應官方的VTuber企劃《けもV》製作相關VTuber版動朋模組，但為私人使用，原因是不希望有違規者濫用。
  • 戰國BASARA
  • Fate/Grand Order
  • 佐賀偶像是傳奇
    • 目前最有名同時唯一現有的模組系列為ISAO製作的版本，均有在Melonbooks或是DLSite平台公開，為付費下載模組。
      • 所有成員有分制服、舞台服、便服版本模組（後兩者為同捆另外購買）。
      • 巽幸太郎淚目
  • バンドリ!
    • 因為受眾主要在華語圈，除了AtAt0307專門製作舞蹈外，其他人多半拿來作搞笑靜畫。
    • 缺乏RAS,Morfonica的素材。
    • 模組多數由タマ製作，停止配布後基本上已經消亡，大致僅剩粉專「武士道迷因」有在更新靜畫。
  • LoveLive!
    • 曇花一現，模組主要製作者被廚盜用名義後停配模組而走向衰落，大致僅剩前兩代部分角色有模組可用。
    • 虹團的模組則在リスト配布狀態[10]，因此有和沒有一樣如果不考量遊戲提取。
    • 因為音樂版權的影響，作品數難免受限。
  • 機動戰士鋼彈系列
    • 常有黑科技模組出現，可謂臥虎藏龍的一組。
  • 賽馬娘 Pretty Derby
    • 動畫推出後慢慢有起色的系列，一開始只有Jean製作的校服版本，手遊推出後更陸續有其他MMDer開始製作數個勝負服賽馬娘模組，現今與艦收、hololive總是霸榜NICO立體。
    • 因為本作對於二創的規範，使用規範比其他系列模組相對嚴苛[11]，因此使用前請先仔細參閱Readme檔。
      • 同時MMD相關wiki：VPVP wiki方面則是不記載賽馬娘相關模組，也有部分作者以リスト配布形式公開
        • 但近日因為有廚不斷挑戰賽馬娘官方和模組作者底線，無視二創規範和模組使用規範直接對外公開或私下公開賽馬娘R-18改造模組，且即使被轟殺也還會不斷復活，令整個MMD賽馬娘圈陷入創作危機
        • 而後在2022年10月中旬也有部分網友發現自己精心製作的賽馬娘MMD動畫被版權砲下架（目前已知niconico開始出現相關災情），使整個圈子更加惡化，甚至部分MMDer也為了規避版權砲只能忍痛將賽馬娘MMD動畫暫時切換成非公開（連MMD模組也暫停公開），同時更有網友把矛頭指向惡質廚並且不斷用檢舉砲開炸
        • 之後經過一部分玩家的觀察才發現應該並非前述的惡質廚在作亂，而是影片中使用的MMD賽馬娘模組（即上述的Jean製作的校服版本）不符二創規範才會遭到版權砲[12]，因此本次的版權砲風波才總算平息
        • 雖然某種方面應該算是巧合，但此次事件也算是留下了一個大污點，從此事件之後幾乎所有MMDer和玩家都對此事變得相當謹慎
  • 虛擬YouTuber
    • 起初先是キズナアイ、ミライアカリ等個人模組，後來Hololive也開始陸續推出0期生等的官方模組不過蘿蔔子等Vtuber卻最早登場。
    • 當然也有粉絲自己弄的二創模組，包括尚未3D化的Vtuber。
  • UTAU
    • 和V家角色都是技術演示的常見平台。
    • 最常見的是和V家血緣最近的重音テト，此外還有一小群專門製作獸男的玩家存在。
  • ハッカドール 駭客娃娃
    • 為DeNA推出的ACG企劃之一，起初為分享ACG相關的新聞資訊平台，後來動畫推出之後MMD模組也同步公開
      • 其中以偽娘屬性的3號最受歡迎
    • 後來DeNA將駭客娃娃收攤之後雖然還是有一陣子繼續提供MMD模組下載，但後來隨著網站消失也還是在2021年底結束公開
  • 原神
    • 可在官方活動網頁及模之屋下載（但後者需要註冊會員）
    • 也是少數為官方提供的官方自製+提取模組
    • 但因為檔名和編碼問題，可能會出現部分圖層無法讀取，因此可能需要額外花時間修改資料夾名稱、檔名甚至用PMD／PMX Editor修改材質名稱
    • 雖然說明書（以及二創規範）有明確表明禁止使用此系列模組製作紳士向MMD影片，不過嘛...
  • 唐澤貴洋
    • ニコ站的專門tag為「尊師ＭＭＤ」，會亂入各種題材的日常劇，是日本網路暴民不滿東方、艦收的產物。
    • 亂入率在14屆MMD盃達到高峰甚至洗到該屆前三，間接導致了MMD市場的衰落。代表作為一般男性脱糞シリーズ。
      • 因此之後的MMD盃更是開始對此加強取締，但上述廚的破壞已經難以挽回。

主要的檔案載點編輯

  • Bowlroll
    • 從角色模型到特效插件都找的到，可說是MMD的大寶庫
    • 多數的vmd數據檔以及場景模組都存放在此
    • 部分配布物資訊（包含密碼）必須要從NICO靜畫或是影片連結得知
  • NICO立體
    • 目前多數角色模型的存放點
    • 標籤使用較Bowlroll普及，基本上只要點標籤就可以找到同系列的所有模型
  • uploader.jp
    • 部分作者會使用的配布物存放網站
    • 密碼同樣必須由靜畫或製作者的個人網站取得
  • DeviantArt
    • 歐美創作者偏好使用的網站，簡稱DA
    • 也是目前資源豐富的網站之一，但有數成MMD模型有包括上述提到的無斷二次配布、無斷改造以及從遊戲提取出來的提取模型，因此該網站也被部分玩家冠上「大魔窟」等汙名
      • 不單只是MMD相關問題，DeviantArt也已經有數次無斷轉載他人作品的案例[13]，而且幾乎防不勝防
      • 撇除那些問題不說，其實也還是有細心製作模型的歐美MMD玩家[14]
    • 其他細節請參考pixiv條目的DeviantArt一欄
