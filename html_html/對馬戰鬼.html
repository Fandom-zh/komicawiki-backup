對馬戰鬼

對馬戰鬼（Ghost of Tsushima） / 對馬懦夫（Coward of Tsushima） / 對鬼戰馬[1] / 刺客教條：對馬（Assassin's Creed Tsushima）

目錄

解說編輯

  • 由以《惡名昭彰》系列聞名的Sucker Punch Productions製作，由索尼互動娛樂於2020年7月17日在PS4平台上發行的開放世界動作冒險遊戲
    • 為製作團隊受黑澤明電影影響而首次採用日本風格的作品，在畫面表現上還可以調整成黑白電影風格
      • 同時也支援英語與日語的雙語配音，不過不管是遊戲畫面還是字幕都是以英語為主進行對口與翻譯，因此使用日語配音時可能會產生嘴型與翻譯上的落差
    • 而為了還原符合當時時代的刀劍戰鬥，製作組還請了天心流古武術的傳人鍬海政雲與井手柳雪擔任顧問[2]
  • 因為受到最後生還者2炎上事件波及的影響，本作是第一批Metacritic採用新評分制度「遊戲發售後36小時才開放玩家評分」的作品
    • 然而本作最近卻疑似慘遭TLOU2支持者洗差評⋯⋯[3]
    • SJW們所使用的理由是大部分時間都是偽命題的文化挪用，全世界只有美國不准做其他文化圈的遊戲（笑）
      • 雖然SJW一向以亂開槍著稱，不過這次倒是正中紅心⋯⋯有日本人批評，製作組刻畫世界觀的時候還是傾向依照自己的想像，而非參考實際的歷史或是已有的時代劇，造成劇情的空洞和人物互動不真實，造成本作既不是「忍者殺手」一般荒誕式世界，也沒有達到真實系日本時代劇的兩頭不到岸的水準；遊戲裡頭甚至有些部分表現出製作組似乎對日本文化完全沒有興趣[4]
      • 不過本作在日本剛上市時，實體版就供不應求到必須等候數周才能拿到，並且於7月時銷售突破33萬份，超過了之前的記錄保持者《漫威蜘蛛人》，成為日本地區最暢銷的PS4獨佔遊戲。
      • 因為韓日對馬島領土爭議，本作實體典藏版在韓國被禁售，但是一般實體及數位板仍可入手
      • 而在中國雖然因為涉及元日戰爭，而在剛發售時被人舉報辱華，不過目前尚未有更進一步發展，同時也有一派網友以當時南宋才是正統中華為由支持本作
  • 遊戲背景為十三世紀末的元日戰爭，描寫年輕武士境井仁為救出舅父並收復對馬島而隻身面對元寇的冒險
    • 後續追加新多人模式-奇譚並分為2人合作劇情、4人合作生存，但該劇情是獨立

系統編輯

  • 本作系統採用開放世界，在完成序章後可以在地圖上自由探索完成任務
    • 任務類型有金色標記的主線任務「仁之道」、白色標記的支線任務「浮世草」與藍色標記的特殊任務「傳承」三種
    • 當標記任務目標地點後，會藉由刮風的方式指引方向
    • 完成任務可以獲得名聲，藉由名聲的提升可以獲得能力的加成
    • 除了任務之外還有多種收集要素在地圖上供收集
  • 本作戰鬥類型有擅長正面對決的武士與專精暗殺的懦夫戰鬼兩種
    • 生命低落時可以藉由消耗決意回血，重傷倒地時也可以消耗大量決意再起三十元接關
    • 正面面對敵人時可以觸發對峙模式，在敵人出招前施展強攻擊可以一刀必殺敵兵並回復大量決意，不過施展失敗時會陷入瀕死狀態
      • 強化後可以在對峙勝出時吸引其他人前來對峙
      • 這使你充滿了決心( -_- )
    • 當未被敵人發現時可以使用暗殺技偷襲敵人，暗殺成功後同樣可以回復大量決意
      • 強化後可以進行連鎖暗殺
      • 志村：只有懦夫才會從暗處發動襲擊！
    • 藉由討伐或觀察各地的蒙古將領可以累積經驗值獲得新的架勢，不同的架勢可以應用在不同場合
    • 除了刀劍之外，還有弓箭、苦無等多種副武器可以使用

主要角色編輯

境井仁（Jin Sakai）編輯

  • 演：辻大介
  • 日語CV：中井和哉
  • 本作主角，境井家第八代當主，志村的外甥日本刺客兄弟會的刺客大師，埃及浴巾俠傳人[5]
    • 由於戰鬼之名頗受島民信賴，境井家在戰後被幕府解散，本人則隱居山林中繼續對抗蒙古
  • 在小茂田一戰敗陣之後，為了從蒙古手中拯救對馬島而展開冒險
  • 自幼父母雙亡而被志村收留，本身也嚴格恪守著志村教導自己的武士道
    • 不過隨著在對馬島的冒險以及在島上結識的人們影響，原本信奉的價值觀也因此產生變化⋯⋯
    • 在金田城第一次面對赫通汗時還相當堅守武士道，但是被打敗後領悟到赫通汗為了征服日本已對武士的戰鬥方式了解透徹，因此不得不退守身為武士的底限[6]
    • 同時在結奈的調教引導下為了擊敗蒙古人與生存而不斷做出違背武士精神的作為，同時官方相當惡意的在玩家做出這些舉動時放出與志村舅舅間的回憶以及武士的信仰，但是惡魔般的玩家才不會理會這些，隨著劇情推進還會學會並使用各種外道的戰鬼武器
    • 志村大人：仁，快說，快說你是個武士，我不當武士啦！舅舅！
  • 在小松鍛造場之戰因為如惡鬼般的砍殺蒙古軍，驚呆一眾島民，被結奈稱呼為從冥府中再起的復仇之鬼，之後戰鬼之名開始在島民與蒙古軍之間傳播
    • 為了擊敗蒙古開始不擇手段，除了偷襲跟潛入外，更學會了使用毒藥
    • 在收復志村城之戰中目睹了武士萬歲衝鋒蠢死的景象，讓他決心違背軍令，卻也讓自己成為幕府之敵
  • 相較於隔壁棚整天都在當戰士的忍者，是個整天都在當刺客的武士
  • 只拿一把刀的索隆
  • 之後失去了一隻眼睛，拿起了六把刀，成為了奧州筆頭
  • 很喜歡看出包王女
  • 酒品不太好，喝多了會發酒瘋且亂脫衣服
  • 對馬窮鬼
  • 高瘦版的香川照之[7]

志村大人（Lord Shimura）編輯

  • 演：Eric Steinberg
  • 日語CV：大塚明夫
  • 對馬島地區的地頭[8]暨當地的領袖人物，很堅持身為武士而必須遵守的相關精神。
    • 同時也是仁的舅舅。由於自己沒有子嗣，所以希望收養仁成為自己的接班人。
  • 在小茂田一戰後被赫通汗俘虜並囚禁在金田城舅舅請再等我一下，還有支線沒解完，被仁救出後派人聯絡幕府前來救援並驅逐對馬島上的蒙古人
    • 囚禁期間從汗王口中聽說的仁的戰鬼之名，但本人堅決否認外甥會如此無恥
    • 獲釋後由於戰鬼聲名遠播加上其不同於一般武士的戰鬥方式，跟仁開始漸行漸遠
    • 志村城一戰親眼目睹仁使出各種外道戰術後大失所望，在爭執中怒扇對方一巴掌，兩人自此分道揚鑣這個才叫做立場跟情緒對立好嗎
    • 在將蒙古人趕出對馬島之後，因為將軍忌諱仁的存在而要求志村將仁處死，在境井家墳前寫完辭世之句後會與志村決鬥，擊敗志村後可以選擇是否按照志村要求將其介錯所以說比起說教重點還是能讓玩家選擇
  • 本人曾是傳說中的傭兵、索羅門的惡夢

結奈（Yuna）編輯

  • 演：Sumalee Montano
  • 日語CV：水野ゆふ
  • 女性盜賊，為了拯救被蒙古軍捉去的弟弟「孝」而四處奔走
  • 原先救出孝後便打算與弟弟搭船前往本島遠離戰亂，但在孝被汗王所殺後便心灰意冷繼續留下幫助仁
  • 實為鑓川出身，因為鑓川謀反被志村討伐而憎恨志村

赫通汗（Khotun Khan）編輯

  • 演：Patrick Gallagher
  • 日語CV：磯部勉
  • 來自蒙古帝國的汗王，對馬島侵略戰的指揮官
  • 初見戰鬥必敗就算了，勉強打下去還會鎖血，比屑一郎還屑
  • Makikaka[9]
  • 武器是雙頭長柄戟（第一型態，一邊是戟頭一邊是大刀）跟刀+盾牌（第二型態），跟他的決鬥打完後，會變成亂戰的戰鬥模式，他會烙小兵一起打，可以用各種戰鬼武器玩他

石川定信/石川貞信[10]（Sadanobu Ishikawa）編輯

  • 演：François Chau
  • 日語CV：千葉繁
  • 通稱石川老師（Sensei Ishikawa），弓術達人，前長尾家家臣，因為某起事情而被迫辭退。
  • 在過去的女弟子「巴」背叛並加入蒙古軍時決心清理門戶而追捕她。
  • 個性非常機掰，許多玩家表示他們更樂意幹死石川師傅。
    • 在劇情後續認識巴後玩家紛紛表示很能理解她為何背叛
      • 石川：我對待她就像親生女兒一樣！
        政子：有很多女人都受不了自己的父親
    • 也因此被網友起了個「對馬內鬼」的藐稱
  • 在結局過後正式收仁為徒。仁：我這是自找了什麼麻煩啊？

堅二（Kenji）編輯

  • 演：James Hiroyuki Liao
  • 日語CV：佐藤せつじ
  • 結奈的朋友，販酒商人，對馬島亂源
  • 仁：為什麼你老愛把我裝進桶子裡？

安達政子（Masako Adachi）編輯

  • 演：Lauren Tom
  • 日語CV：安藤麻吹
  • 安達家當家安達晴信的妻子，劇情內通稱「政子夫人/政子殿」
  • 丈夫及兒子在海灘戰死，留守的族人又遭到背叛而滅絕，成為安達家唯一的倖存者
    • 因為家變而對於追殺凶手抱持強大執念，甚至曾無視仁活捉逼供的建議直接將對方砍死。
    • 政子：她殺了我全家，今天就要她血債血還
  • 真正的對嬤戰鬼
    • 強烈的復仇心讓她不管是劇情還是戰鬥都顯得鬼氣逼人，上陣時更是大吼大叫的往前衝，作為復仇者的魄力比主角強烈許多
    • 加上政子在遊戲中是屬於會積極衝到敵人面前對刀的類型，沒有拿盾的敵人很容易直接被政子劈死
    • 隔壁那款戰神指的是主角兒子，這邊的戰鬼不是指主角好像也是可以的
  • 在她的任務中暗示她與過去被趕出家門的女僕舞有戀情，所以政子在她面前不但表現得像是希望挽回戀情的男友一樣遜炮，並且最後只將兩人的定情物還給她後就放她走了。
  • 害她滅族的真兇是其親姊「花」，對於當年被搶走心上人和被安排嫁給一個酒鬼的事耿耿於懷，一恨就是幾十年

典雄（Norio）編輯

  • 演：Earl T. Kim
  • 日語CV：かぬか光明
  • 對馬島最後倖存的僧兵，杉寺的護寺僧「燕淨」的弟弟

龍三（Ryuzo）編輯

  • 演：Leonard Wu
  • 日語CV：多田野曜平
  • 仁過去的好友，浪人集團「斗笠幫」的領袖
  • 過去曾在比武大會中輸給了仁並喪失了出仕的機會，本人對此耿耿於懷
  • 原本接受仁的號召攻打金田城，但因為蒙古重金懸賞仁的首級而與仁決裂
  • 後來在志村城與仁對決敗北之後請求仁殺死他

孝（Taka）編輯

  • 演：Eddie Shin
  • 日語CV：山口勝平
  • 結奈的弟弟，鐵匠
  • 被仁救出後為了協助其攻打金田城而開發鐵鉤
  • 與仁一同被赫通汗捕獲之後被要求殺死仁，孝拒絕之後反而襲擊赫通汗，之後被赫通汗殺死
  • 在結奈的支線任務中，可以得知他在小時候與結奈一起逃出時，曾被拐騙他們姊弟的奴隸商人性侵過
  • 跟刀刀齋學習打鐵的犬夜叉
  • 流落到和之國的騙人布
  • 差點砍出結局的男人

百合子（Yuri）[11]編輯

  • 演：Karen Huie
  • 日語CV：田畑ゆり
  • 境井家的下人，在仁的母親過世後長年照顧著仁，在仁出征時繼續守候在家中等他回來，擅長製作草藥甚至是毒藥。
  • 在幫助仁開發吹箭的毒針後因衰老而安詳過世
  • 疑似與仁的父親‧境井正產生感情並發生關係，並將把這份愛深藏在自己內心中，直到臨死前，誤認仁為其父親下才娓娓道出

信 / 空 / 影（Nobu / Sora / KageCage[12]）編輯

  • 境井仁所騎乘的戰馬真老婆，一路的旅程中都會有牠的陪伴，同時仁騎乘牠也會有很多關心的語音直到主線後段因為傷重身亡為止。
    • 這匹馬的便當比喬爾夫球要動人不知道幾百倍
  • 顏色以及名字可以根據玩家的喜好所挑選，選擇之後便不能再更改
  • 支線任務完成時會看到阿仁與馬的互動，例如仁會躺牠身上睡覺或是會從阿仁的背後戳他一下。
  • 部分沒良心的玩家很喜歡騎著牠從懸崖上一跳而下
  • 第三幕前期會騎到一匹不怎麼會聽呼喚的普通馬，直到後期會再換一匹專屬戰馬給你，專屬戰馬的差別是可以取名字跟裝備不同的馬鞍

敵人單位編輯

人類編輯

  • 蒙古人
    • 遊戲中主要敵人，依照主線任務推進，服飾越華麗的越強
    • ＼內褲鬆太大！！／
  • 刀兵
    • 分為單刀兵跟雙刀兵兩種，基本的雜魚單位
    • 持刀強盜跟浪人也屬這一類
  • 盾牌兵
    • 盾牌的防禦很難攻破，除非切換到相剋的架式
    • 後期會出現體格巨大的血牛盾兵，可以把仁打飛出去
  • 槍兵
    • 攻擊距離長，正面對決不用相剋架式打會不好打
    • 橫掃絆倒攻擊要用跳躍躲過
    • 拿著斧頭的血牛也是屬於這類
  • 暴兵
    • 清一色是體格巨大的血牛，而且幾乎不會陷入震攝狀態，大錘攻擊強到可以把仁打飛出去，但攻擊速度慢
    • 後期會出現拿三眼銃的暴兵，開火時會有瞄準動作
  • 弓兵
    • 理所當然是遠距離射箭攻擊，強盜中也會出現
    • 射箭前會喊叫，特色是血少皮薄，衝過去亂砍可以輕鬆解決
    • 後期會出現射火箭的弓兵
    • ＼都秀！都秀！／
  • 擲彈兵
    • 拿小短刀，會投擲火焰彈或閃光彈的敵人，跟弓兵一樣血多皮薄可以輕鬆砍死
  • 將領
    • 小BOSS級敵人，特色是穿著把臉遮住的全身盔甲
    • 對他用暗殺也不能一刀殺死，直到後面學會戰鬼型態後用暗殺可以一刀把他的頭砍下來
    • 拿到長弓後用重型箭矢可以輕鬆射殺
    • 部分將領會跟你決鬥
  • 強盜
    • 戰鬥力跟最弱的蒙古兵差不多，防禦很低，有強化過的話幾刀就可以解決
  • 浪人/武士
    • 比上面的強盜強一點，攻擊速度稍快，解支線與傳說任務的時候有數個都是會決鬥的BOSS

動物編輯

  • 殺掉可以取得掠食動物的獸皮，可以去跟狩獵商人升級攜帶的暗器/弓箭攜帶量
  • 蒙古獒犬
    • 黑色的大狗，會隨蒙古兵一起戰鬥，其攻擊無法阻擋，而且會咬住仁的手拖行幾秒，就算是戰鬼模式也會
    • 之後拿到犬神使護符時，可以趁獒犬沒有發現時進行撫摸，就能讓其成為同伴幫忙戰鬥。
  • 鷹
    • 有時會在蒙古軍營旁出現，如果沒有早點射死的話會暴露行蹤
  • 野豬
    • 通常成群結隊三隻一起出現，看到人會衝撞或逃跑
  • 黑色大熊
    • 攻擊方式是咬住仁甩出去
    • 有時會看到蒙古弓兵在野外跟他戰鬥，或是關在蒙古人的牢籠裡
    • 堅二：熊、熊要跑出來啦
    • 玩家在野外進行探險時，能夠經常看到大熊和蒙古兵之間"快樂地玩耍"。

武器裝備編輯

近戰編輯

  • 太刀
    • 境井家的祖傳寶刀，也是唯一的近戰武器
    • 真正的武士刀結構非常脆弱，若是像遊戲裡那樣從高處跳下刺的話，可能敵人還沒死光就先折斷了
  • 短刀
    • 就是所謂的脇差，由於現實中的太刀是設計用在騎兵戰上的，所以武士都會配有短刀以利近身戰
    • 遊戲中僅在暗殺時用上
    • 懦夫刀

暗器編輯

  • 苦無
    • 投擲用短刀，被擊中的敵人會陷入破防狀態，非常適合用來對付重裝兵
  • 煙霧彈
    • 投擲後發出濃煙遮蔽敵人視線，此時即便已被敵人發現也能進行暗殺
    • 等級上升後煙霧附加回血效果
  • 瀝青炸彈
    • 會黏在目標身上爆炸的炸彈，爆炸傷害不分敵我所以須注意是否會殃及自身
    • 等級上升後爆炸不會傷到玩家，但NPC似乎還是有影響
    • 「戰鬼劍法是無敵的」→ 丟炸彈
  • 油

遠程編輯

  • 半弓
    • 完成主線任務「石川師傅的故事」後取得
    • 拉弓速度快，但相對的射程跟威力就略差，適合混戰中使用
  • 長弓
    • 完成傳承後才可獲得，除一般箭矢外還有爆破箭可用
    • 穩定度跟射程都非常優秀，但拉弓時間較長，適合遠程狙擊
  • 投擲物
    • 有鈴鐺、鞭炮跟炸彈三種，前兩者只能吸引敵人而沒有傷害
    • 炸彈雖可攻擊大範圍的敵人，但投擲到爆炸有些許時間差，使用時須注意
  • 吹箭
    • 完成主線任務「揮之不去的過去」後取得
    • 對敵人射出粹有毒藥的的細針，有劇毒跟幻覺兩種，前者可一擊殺死敵人，後者會讓目標發狂攻擊同夥
    • 被毒死的敵人有機會讓周遭同伴陷入軟腳狀態

指標編輯

  • 引導之風（Guiding Wind）
    • 在設定好目標地點之後，會出現引導之風引導玩家目標的方向
    • 藉由向上滑動觸摸板可以即時吹起引導之風
    • 配合升級過的旅人衣裝跟技能可以找尋隱藏收集物
    • 老爸的化身
  • 黃金鳥（Golden Bird）
    • 於各地出現，引導玩家前往各種隱藏秘境的黃金鳥
    • 老媽的化身
  • 狐狸（Fox）
    • 於各地出現，引導玩家前往稻荷神社的狐狸
    • 其實真正的對馬島上面並沒有狐狸
    • 可以摸的狐狸，10/10 GOTY

收集要素編輯

  • 溫泉（Hot Spring）
    • 浸泡溫泉後可以回想目前的冒險經歷並提升生命最大值，泡溫泉時，可以看到阿仁的美尻和他的一些妄想
  • 斬竹（Bamboo Strikes）
    • 在竹場通過一連串QTE按鍵後可以提昇決意上限，砍完後可以獲取一定數量的竹子資源
  • 神道神社（Shinto Shrines）
    • 通過一連串考驗登上神社頂端後可以獲得特殊護符與木材
    • 由於通往神社的路都剛好毀了所以需要翻山越嶺，參拜變成媲美極限體能王的大挑戰
  • 榮耀之柱（Piller of Honour）
    • 放置刀造型的石柱
    • 部分的收集品(頭帶，面具，斗笠)部分放在各種超難跳上去的地方
  • 稻荷神社（Inari Shrines）
    • 追蹤狐狸找到稻荷神社後可以參拜並提升裝備護符數目的上限或是戰鬼之花，但大多數人都喜歡拜完後擼狐狸
  • 俳句（Haiku）
    • 在俳句定點可以吟詠俳句並獲得紀錄俳句內容的頭帶
  • 燈塔（Lighthouse）
    • 設置在各地的燈塔，登上燈塔後可以點火照亮四周
  • 戰旗
    • 收集給某位和尚可以換到新的馬鞍
  • 蟋蟀籠
    • 都放置在各墓地，找到一定數量可以學會新的吹笛曲(可以改變天氣)

相關編輯

  • 最後生還者2
    • 另一個由索尼第一方工作室推出的作品，因為造成炎上的緣故一度讓本作上市前蒙上陰影
    • 順帶一提，《對馬戰鬼》的實體版遊戲盒內一律附有本作的廣告單張，想當然是慘遭被撕爛的命運，但這張廣告單實際是雙頁，背面是另一款遊戲：《終極戰士：狩獵戰場》
    • 在各種意義上來說都是被本作打臉的作品
  • 傑克武士
    • 同樣在2020年暑期檔推出改編遊戲，但一反動畫版的惆悵結局，只要蒐集散布於各關的家徽就可以達成完美結局。
