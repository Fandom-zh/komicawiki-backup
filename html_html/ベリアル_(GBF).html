基本資料[編輯]

  • 名字：ベリアル（Belial/彼列）
    • 也有「貝利亞」、「貝利亞爾」等翻譯。
  • 稱號：司掌「狡智」的天司、變態墮天司[1]、店長的典範[2]、第68柱魔神
  • 身高：182cm（但是作為星晶獸的一員，可以變成巨大模式）
  • 年齡：不明
    • 身為ルシフェル的同期，理所當然有2000以上。
  • CV：細谷佳正
  • 形象色：基本上是黑色，但是官方周邊方面是深紫色
  • 興趣：購物、逛雜貨、享受岩盤浴
    • 被玩家吐槽到底哪來的OL
  • 喜歡的東西：極其扭曲的人事物（公式）、ルシファー、調戲小朋友（如サンダルフォン）
  • 討厭的東西：完美、光明磊落、無私無欲、ルシフェル（根本是討厭事物的集合體）

萌屬性[編輯]

  • 美青年
    • 容貌俊美性感、身材高挑精壯，連GBVS模組都有不少動作是強調他帥哥的部分
  • 翅膀
    • 收起來時是毛毛裝飾，以前副官時代似乎是跟ルシフェル脖子上那條相同的紅色布條
      • 考慮到ルシファー研究所時代的造型也有同樣的紅帶、而且ルシフェル展開翅膀時紅帶沒有不見，那條紅帶可能只是裝飾或是ルシファー特別給的某種象徵物而已。
  • 白色軍裝（副官版限定）
  • 皮衣皮褲
  • 角（Avatar版本限定）
  • 黑化
    • 包含讓人黑化、墮落
  • 多才多藝
  • 設計師
    • 雖然本人是唬爛的，但是他確實出於很莫名的理由設計了天司武器（闇屬性天司武器解放劇情）
    • 還順手駭了別人設計的天司武器（他屬天司武器四階以上解放劇情）
    • 可是在2021グラブルフェス販促環節卻自稱室內設計師顧問（可是推銷的物品卻是茶包）
  • 魔術師
    • 雖然本人是唬爛的，但是他確實具有相關的技術。
  • 糕點師
    • 雖然本人是唬爛的，但是官方確實有推出形象甜點跟飲料，還附上他的性騷擾感言卡，還說是他親手準備的，要美味的享用（能感受到多重含意）
      • 順帶一提，那個甜點的名稱跟說明都非常糟糕。
  • 監護人
    • 對サリエル非常相當好，能陪他坐一整天螞蟻觀察都不膩，也總意無意地幫他。
    • 從態度能感受到對サリエル的特殊關照與情感。
  • 巨乳
    • ルシファー製造的人型天司，除了ルシフェル似乎有額外追求外，全員無論造型男女老少都有著豐滿的上圍
  • 騷想X
  • 啊嘿顏
  • UR等級的細谷佳正
    • 詳見聲優個人條目
  • 腐男子
    • 官方惡搞四格裡曾兩度去同人展看ルナール（GBF裡知名的腐女兼同人作家）的新刊
      • 第一次覺得沒有達標，糾結很久還是沒買（1067話）
      • 第二次是進次元夾縫前，ルナール成果大幅進步有達標，就爽快付錢還說期待下一本新刊（1263話）
      • 因此同人也有作為讀者是ルナール老師理解者的梗。

故事中經歷[編輯]

碧藍幻想[編輯]

失楽園[編輯]

  • 出現在アウギュステ島的復興慶典上，主動找ルリア搭話，以設計師想要欣賞美的事物的理由借來ルシフェル的羽毛，說了一大堆感覺非常有問題的讚美詞後，順手拿普通羽毛掉包被シェロカルテ抓包，只好悻悻然的還給ルリア。
  • 不久後帶著ベルゼバブ來到カナン偷襲ルシフェル，破壞他的遺體並奪走身軀的部分，只留首級跟一部分殘骸，之後就躲在附近等待ルシファー的遺產封印鬆動。
    • 期間聊了一點接下來的計畫，順便講講參雜個人哲學的幹話。
  • 好不容易等待結束，卻被覺醒繼任天司長的サンダルフォン找來尋仇，雖然サンダルフォン力量還不穩定，但是拚上性命還是可以殺死自己，為了逃命只好加減應戰。
    • 開始戰鬥前，バブさん丟下他就先跑了。
    • 趁著アバター的干擾順利從サンダルフォン手上溜走，順利錯開四大天司領軍而來的追捕。

０００[編輯]

  • 將ルシファー的頭與ルシフェル的身體接起來，使ルシファー復活。
    • 不是跟法桑一起的終末我不要。
    • 就說創作中跳涯的都不會死。
    • 可以跟法桑一起住雙人小套房，真爽。
    • 可以跟心愛的法桑住在一起這樣多棒。
  • 不知道為甚麼依然可以到處刷存在感，你真的有被關嗎？

拝啓、大切な君へ。[編輯]

  • 以「織部明彥」這個名字登場。
  • 令人大失所望的官方學園現paro。

Granblue Fantasy: Versus[編輯]

  • 基本上跟本家的《空蒼》差不多，但是更忙了，每一條個人路線都有事要做。
    • 墨鏡架子
  • 似乎正在尋找失蹤的ルシファー，可能這個世界的ルシファー已復活[4]。
  • 劇情中表明，GBVS世界是沒有天司的「新世界」，嘲笑ベルゼバブ沒有柔軟的羽翼卻想翱翔於天空是非常可笑的事，並直言ベルゼバブ的世界意外的無聊。
    • 但是先不說圖鑑說明，ジータ的前置劇情有明確提到打倒作為災厄的サンダルフォン的事，考慮到GBVS的世界似乎是多個平行世界相互干涉、衝突的狀況，可能是同時存在有天司系統（ジータ世界線）跟沒有天司系統（グラン世界線）的，如果是這樣，就算出現還活著的ルシフェル或許也不是不可能。

性能[編輯]

碧藍幻想[編輯]

  • 未角色化，目前有2019年12月31日實裝闇屬性召喚石。
    • 效果非常強但因為副作用很機車，所以主要都拿來當副召。
      • 針對サンダルフォン必定發動的效果：奧義雙發和發動奧義時自身扣除最大HP 49%。
        • Another版也沒逃過，被強行餵蘋果。
        • 那個奧義雙發效果絕對是因為氣到腦溢血了吧？
    • 放首頁時BGM會切換成專屬歌曲「Parade's lust」，還疑似會各種暴雷和丟出下流梗
      • 其他有同樣特權的召喚石包括中之人作為歌手，官方成員粉絲眾多的アオイドス、Jehuty、光美淫獸等
      • 至今語音中提到的天司長到底是指前任還是現任，玩家間依然爭論不休。
  • 於2022年7月31日實裝火屬性泳裝版召喚石，同原版召一樣，放首頁會有專屬BGM跟台詞。
    • 這次的版本可以天井了，第一次卡池實裝期間還有免費10連三天。
    • 從立繪到敘述、甚至演出效果再次讓人懷疑遊戲分級。
      • 在敘述中提到他毫不懷疑偉大的創造主親手做出的自身魅力。
        • 那個三角泳裝真的不是情趣服嗎？[5]
  • 有不少玩家相信Another版L角是為他角色化鋪路，但到底要等多久？讓我們繼續看下去
  • 《失楽園》的Boss版本後來實裝成為高難度多人戰之一，但剛實裝時因為Bug能被コルワ一個綑綁瞬間送下場[6]（現已修正）。

Granblue Fantasy: Versus[編輯]

  • DLC2的首位收費追加角色。
  • 《失楽園》中以Boss身份登場的「アバタール・ベリアル」先是以免費更新的Boss角色參戰，及後再以DLC形式成為可使用角色。

簡評、其他資料[編輯]

  • 既サンダルフォン之後，又一個原意是要讓玩家討厭、結果人氣卻高得官方措手不及的奇妙角色。
    • 原本想著前一個阿芬可能因為最後的反差太激發玩家父性母性了，就索性乾脆弄個滿嘴黃腔（姑且有好好調查過法律允許範圍）跟謊言、卑鄙惡劣無恥愛調戲的壞人。
      • 遊戲營運以來各種意義上尺度最大最放得開的男性角色，在劇情中甚至有跟路人女性同居並暗示做過了的橋段。
        • 大概也拜此索賜，同人CP也是亂到不行，配上官方劇情中的演出，所以也有不少抹布本[7]。
          • 因為GBVS他DLC贈送的主角專屬skin太色，導致主人公無論是葛蘭還是姬塔的暗墮梗大量增生中，Avatar版本增加的紋身(?)更是被直接說是淫紋。
    • 結果以上問題反而因為獨特的人格魅力，使得一票玩家不分男女老少直接暈船，證實了壞男人的魅力。
      • 登場隔年直接上了巧克力榜第一名，之後也基本沒怎麼掉出前三。
    • 少數在官方預料之中的，只有CV是那個幾乎都是配好人的細谷引發的討論。
  • 在《０００》中，對於將ルシフェル的身體用於ルシファー的復活一事，直言「創造物的所有權是屬於造物主的」。
    • 不知道是單純用來宣張自己的行為正當性，還是因為對造物主的忠誠與著迷使他打心底這麼認為。
  • 自登場以來每年都在刷新玩家的認知。
  • 被GBF本家宣告不會作為玩家可遊玩角色，卻跟同樣本家宣告不可遊玩的ベルゼバブ先後在GBVS實裝的且大肆活躍，幾乎每個角色的個人路線都有他擔當幕後黑手的影子，讓玩家完全相信本家遊戲中暗示他的過勞程度，有過之無不及。
    • 跟到2021年底都還沒有實裝的サンダルフォン同是神巴那邊的宣傳角色之一，負責GBF本家的登入獎勵宣傳，每天也是一天一調戲。
    • 武器是格鬥，skin體現結果是墨鏡[8]，本來戰鬥模組就很帥氣了，特寫變得更潮了，除非玩家專門選惡搞武器的skin。
    • Avatar版本在2021/12/14（グラブルフェス結束隔天）實裝。
    • 開場無一例外都會向對面開黃腔或調戲對面，開場台詞大多帶有「正面上我啊！」的意思，相對勝利台詞則是相對應的「該換我來調教你囉~」的內容。
      • 被揍的時候不但顯得很爽，連戰敗姿勢顯得相當有餘裕，讓人懷疑他是否是故意輸的。
      • 果然是變態
  • 明明登場的活動沒有任何十天眾的事，同人卻常常有跟劍王相關的創作，之後還逆流回官方實體活動2021グラブルフェス，在オフィシャルキャスト環節有不少針鋒相對的互動。
  • 在官方實體活動グラブルフェス裡每次都造成話題，直衝推特熱門。
    • 2020年以3D模組登場，公然在舞台上調戲主持人白石稔。
    • 2021年則是擔任官方周邊販賣部的店長，猝不及防的促販宣傳，隔天還偷把團員的便當吃光。
      • 宣傳台詞理所當然全程都是下流梗+調戲阿芬+滿溢出對法桑的愛，以下為例：
        • 兔子娃娃：「也就是說，你們可以每晚抱著我入睡呢」
        • 戒指：「首先最下面是Sandy，中間夾著我，最上面是ルシフェル，這種的也不錯吧？」
        • 官方角色房間門匙：「順帶一提，沒有我房間的鑰匙，因為一直是開著的」
        • 徽章：「好大啊～後面附有支架，可以放在桌上裝飾？好耶，我也拿Sandy的來裝飾好了」
        • ルシファー的T恤：「如果早出個2000年就能當墮天司制服」
  • 2022年グラブルフェス跟進之前的四騎士VR公開天司VR，確定由サンダルフォン與ルシフェル和ベリアル擔任。
    • VR情境中的已知表現有：
      • 公然調戲團長、並理所當然要撩爆（通常運轉）
      • 當然沒有忘了調侃隔壁的サンダルフォン
    • VR結束後，會贈予參與者黑色的羽毛飾品。
  • 2022年FES的オフィシャルキャスト[9]堪稱神還原到全場最吸睛，導致有登場期間推特標籤幾乎被霸占。
    • 除了基本的模仿官方本家的立繪動作外，其他基本多為模仿GBVS的模組動作[10]。
    • 幾乎每一次登場（不是單純指場次不同，而是即使同一場，兩天也）都有不同的新花招。
      • 專門準備了打光燈（兩支），並且親自扛著（其中一支）到定點，開燈動作很挑逗。
      • 準備了執事眼鏡合作的ベリアル眼鏡，跟八成是模仿之前FES官方賀圖中ルシファー（或著後來泳裝召）的橘色太陽眼鏡。
        • 後來還追加蘋果拋著玩。
      • 公然調戲團長，不論是在定點，還是移動中都不忘記飯撒，導致大量團長被飯殺(?)
        • 連工作人員都沒被放過，在某次定點時無預警被摸臉調戲。
        • 偶爾移動中遇到落單的遊客，還會給一些像是轉圈、假裝摸臉/摸頭、突然靠近的飯撒，引得團長們驚呼連連。
      • 在舞台環節從觀眾席入場，還隨機拿觀眾的手機自拍。
        • 仙台場在ビィ登場時從後面繞出來摸頭，結果被踹了。
        • 大阪場第一天，無預警在自己的環節途中以一個響指換BGM，ルシファー隨後驚喜登場，之後就幾乎都一起行動[11]。
        • 大阪場第二天直接挑釁後跟サンダルフォン來一場打戲，雖然看得出是套招，不過運用踢技的動作相當確實，最後在對方交接退場前還抓著領口公然摸臉調戲溜走，隨後悠哉從另一頭再登場。
          • 名古屋場在打完逃走後，從舞台另一邊再登場時，換成帶著墨鏡騎滑板車登場，引發眾人集體爆笑，前面摸サンダルフォン臉調戲的橋段居然還增加趁隙比讚嘲諷。
      • 遇到遊客cos成ルシファー時，會露出驚喜的反應，當對方離開時也會顯得依依不捨，直到對方走遠再恢復。
      • 有跑去召喚石布景跟案天司武器的區域玩。
      • 遇到舞台サンダルフォン演唱角色歌[12]時，即使在別的定點也會特別轉過身去看，並且略帶嘲諷地像是看了場好戲的拍手。
  • 於活動《拝啓、大切な君へ。》中登場。
    • 翻譯過來就是「拜啟 致重要的你。」，標題的背景跟配色很明顯是要讓人聯想到《０００》，所以很早就被猜到可能會出場。
    • 因為立繪是用輔佐官版本改的，還跟ルシファー一起登場，玩家間期待值非常高。
    • 結果卻是個被Cy官方同期實體展覽《蒼の追想》特別聲明（切割）「現在進行中的劇情活動與本展覽的內容沒有連動或關聯」的災難。
  • 日本有一名知名的ベリアル控太太，毎次發廚都會製作充滿觸覺跟梗的高質搞笑MAD，人稱「ベリアルが好きすぎる人」
    •   例子
  • 雖然2022年9月就開通了推特卻是10月4日才公開，有玩家推測可能是為了配合「天司之日」的關係[13][14]。
    • 才開通沒多久就去調戲木村唯人等官方帳號，聊天內容讓人擔憂是否沒問題。
      • 可是卻以「還不能被BEN」為由表示遺憾不能發自拍照，讓玩家們浮想連翻懷疑他到底想發甚麼樣的照片。
      • 但是卻繼續用雙關語持續性騷擾。
    • 還公開用google表單限時徵求企劃意見，所以連google帳號都確定有了。
      • 結果馬上被大家當客服許願亂玩亂問。
    • 毫不意外迅速上了關鍵字趨勢。
    • 發的照片馬上被注意到是在Cygames本社拍的。
      • 除此之外也非常愉悅的找理由發些周邊或出去玩的照片。
    • 從自介到貼文一如既往走著虛實混雜，踩著法律允許尺度邊界卯起來玩，活像真正的煩躁系網紅在分享生活。
      • 連同其中一篇使用的tag「だって狡知だもん」也上了推特趨勢。
    • 被玩家開玩笑表示「我們正身處一個可以黑單或騷擾ベリアル的世界線」。
    • 發推勤奮，被懷疑要是兩千年前就有SNS，這傢伙會不會一路發文到某人復活還停不下來。
      • 因為是天司，所以就算用詞出現十幾年前的死語，也不會有問題。
      • 於10月5日表示留下「還請繼續維持追蹤」的訊息後，暫時停止發文。

名台詞[編輯]

  • 「OK~」
    • 有各種語氣、聲調的版本。
  • 「ヤバイ、達する達する！」
    「不好、要高潮了高潮了！」
    • 糟糕意義上的同字面意味。
    • 基本上是性♂奮起來時的口頭禪。
  • 「どうして空は蒼いのか」
    「為何天空是藍色的？」
    • 儘管他本人可能不知道，但是ルシフェル也說過同樣一句話，可以看出兩者基礎思維的相似處。
  • 「蒼でなければいけない理由などない」
    「不過本來就沒有人規定天空應該要是藍色的」
    • 接上句，同時也難得用明確抱持著批判的語調在陳述自己的價值觀。
      • 劇中也曾表明覺得象徵終末紅色的天空也很好。
    • 隨著泳裝召喚石實裝，玩家們才發現這句話可能不是隨口說說而已。[15]
  • 「では良い『終末』を」
    「（祝各位有個）美好的世界末日」
    • 因為發音跟劇情開放時間，有玩家認為也有著「（祝各位有個）美好的週末」的意思。
      • 第四面牆的存在感
  • 「愛は矛盾するモノだぜ」
    「愛是很矛盾的東西的呀」
    • 面對ルシファー的責問，帶著感慨如此回應。
    • 同時也跟ルシフェル之後的「心とは矛盾するモノだよ」剛好相互對應，再一次顯示兩者相似的部分。

相關人物[編輯]

碧藍幻想[編輯]

  • ルシオ — 不知道其存在的祖父。
  • ルシファー — 父親創造主兼單戀對象，非常著迷於他的人格魅力與才華，可以為他做任何事。直言對於由他親手創造的自己擁有能使人墮落的魅力這點，毫不懷疑並樂於大方展示，好誘惑某個誰。
  • ルシフェル — 兄弟ルシファー同期同規格，能力與性格都相互對應的作品，自己不太喜歡的前上司，對他有相當複雜的感情，用自己的計畫而使理應不滅的他消逝了，用他的身體復活他們的造物主。
  • サンダルフォン — 姪子ルシフェル的創造物，天司長的繼任者，覺得逗他很好玩，每次見面不調戲一下會死，以前明明很純真可愛的，繼任後越來越像ルシフェル很討厭，不過該調戲的還是沒停手，以前曾慫恿他對ルシフェル發起叛亂使他被關。
  • ベルゼバブ — 其實算叔叔的存在合作對象，負責殺死ルシフェル，似乎從遙遠的過去就特別敵視ルシファー跟ルシフェル，因為瞧不起自己結果被自己順利的暗算。
  • ミカエル — 火之天司，四大天司之首，現任副天司長，自己推薦的繼任者，過去利用她對自己的信任策畫陰謀，導致現在被她深深的憎惡著。
  • ラファエル — 風之天司，四大天司之一，冷靜的前部下。
  • ウリエル — 土之天司，四大天司之一，熱血的前部下。
  • ガブリエル — 水之天司，四大天司之一，細心的前部下。
  • バレンティン — GBF的糟糕人物之一，残虐三兄弟的成員，ドM
  • ロベリア — GBF的糟糕人物之一，肉醬賢者，ドM
