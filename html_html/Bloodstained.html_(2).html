血咒之城：暗夜儀式

Bloodstained: Ritual of the Night / 血咒之城：暗夜儀式[1] / 血污（赤痕）：夜之儀式[2]/血虧：夜之義工
本條目內容不足需要擴充，歡迎有興趣的朋友加以補完。
本條目內含劇透，沒玩過遊戲又不想被鍊金術公會改造成碎片人或是屢遭跳票的話請不要反白。

目錄

簡介編輯

  • 由Inti Creates與Artplay製作、505 Games發行的動作遊戲
  • 製作人為經手過惡魔城系列多部遊戲的「IGA」五十嵐孝司，遊戲系統也是採用系列後期主流的「Metroidvania‎」
    • 在IGA製作完《被奪走的刻印》後想要製作3D惡魔城，其企劃案於是與同想製作3D惡魔城的西班牙開發商「Mercury Steam」競爭；最後KONAMI選擇了Mercury Steam，IGA就此喪失了對惡魔城系列的主導權
    • 此後IGA轉任KONAMI的攜帶遊戲部門，但他心中還是想開發正統的家機/掌機作品
      • 正好2013年稻船敬二的《Mighty No.9》募資大成功給予他一個啟示，於是在Digital Development Management的經紀人兼仲介Ben Judd的協助之下，IGA在2014年離開了任職逾20年的KONAMI並開始著手新遊戲企劃
    • 起初IGA向世界各地的遊戲發行商推銷提案但卻處處碰壁；在Judd的努力下最終確保了450萬美元的開發資金，並效法《Mighty No.9》於2015年5月起在Kickstarter上募資籌集餘下的50萬鎂、同時藉此測試外界對提案的反應
      • 正巧當時在網路上正盛傳KONAMI對小島秀夫的不公待遇，IGA的提案正好給想幹KONAMI的玩家們一個宣洩點，加上惡魔城系列的老玩家們也對正統的Metroidvania遊戲餓了很久，結果募資超額達成並達到當時Kickstarter遊戲類最高紀錄的550萬美金
  • 配樂由負責多部惡魔城系列遊戲的山根ミチル擔任，美術與人物設定則是經手多部Inti Creates作品的夏目裕司
    • 雖然負責月下等作品人設的小島文美未直接參與作品製作，不過還是有找她來繪製典藏版封面
  • 原定於2017年發售，但由於Inti Creates的技術力無法好好運用UE4遊戲引擎，導致開發進度不斷拖延
    • 在2016年10月宣布遊戲由505 Games發行、負責全世界的遊戲營銷，並投入更多的開發資金，加入DICO、Monobit、WayForward等開發團隊協助遊戲開發
  • 經過數次延期後終於在2019年6月18日於PS4、Xbox One與PC平台正式發售，並於一週後的6月25日推出Switch版
    • 雖然開賣後初期還是被發現數個會影響遊戲遊玩的Bug和對白亂翻，但各國遊戲媒體基本上都給予了正面評價
      • 但Switch版本相當糟糕的優化、和與其他平台版本差太多的畫面品質還是引起了不小的爭議，甚至搞得製作組得特地花時間處理Switch版優化，導致斬月DLC推遲半年時間才正式更新。[3]
      • 而到現在Switch版還是有切換地圖延遲和遊戲崩潰問題，OP動畫甚至還會音效不同步……但已經比首發好些。
    • 而當初在Kickstarter募資目標中剩下來沒有兌現的內容，預定會放在後續的13個免費DLC上
      • 目前更新進度：7/13
      • 然後官方在3月初的情報更新提到因為團隊技術力不足，決定閹割掉原本會實裝的Rougelike模式，改以改變遊戲參數設置的Randomized模式替代。
        • 結果導致參與眾籌的玩家們大炎上，不論是官方論壇還是Kickstarter的留言區都是罵聲一片。看起來米莉安想不落入和前輩Beck一樣的悲慘下場還有得等。
      • 等了將近一年，斬月DLC和Randomized模式終於宣布預定於2020年5月7日於PC、XBoxOne、PS4平台發佈。至於Switch則是得等到5月底-6月初才能更新。
      • 2020年在6/10號的時候宣布全世界破100萬份，另外也公開了後續的更新企劃[1]，應該是最近幾年離開大公司之後的日本遊戲製作者製作的募資遊戲中成績比較不錯的。同時也推出由網易負責的手機移植版。

系統編輯

  • 武器
    • 玩家可以使用的武器共有九種。
      • 武靴：以足技為主的攻擊，同時可以提升滑踢和跳踢的威力，必殺技可於空中使出，相當於過去惡魔城的拳套超絕強化板。
      • 短劍：特色是射程短但攻速較快，必殺技可於空中使用。
      • 突刺劍：射程長於短劍，範圍侷限於前方。
      • 劍 / 棍棒：射程與範圍中等，屬性豐富。
      • 大劍：基礎攻擊力高，有攻速慢及破綻較大的缺點，同樣的也能攻擊到自己的背後和上方。必殺技很難用
      • 刀劍(日本刀)：破綻略小，攻擊範圍稍短，但必殺技非常豐富。
      • 長矛：射程長，難以擊中過近的目標。
      • 鞭子：對周圍攻擊，容易擊中背後的目標但射程較短且硬直長，運用上須掌握跳攻技巧。
      • 銃槍：消耗子彈的射擊武器，子彈耗盡時會變更成威力低的無限子彈。
    • 武器各自有秘技，當熟練秘技後可以套用到同類型武器上。
  • 碎片
    • 惡魔之力的結晶，米莉暗在吸收碎片後可以將其化為自身的力量。
    • 碎片依照顏色與用途分類共有六種類型。
      • 觸發（紅）：藉由消耗MP發動的攻擊技能。
      • 效果（藍）：藉由持續消耗MP發動的輔助技能。
      • 定向（紫）：確認方向後消耗MP發動的魔法。
      • 附魔（黃）：不需消耗MP，多與自身能力變化相關，等級提升至最高後可以變成技能碎片。
      • 使魔（綠）：不需消耗MP，召喚使魔協助作戰。
      • 技能（白）：不需消耗MP，取得後可持續發動。
  • 煉成
    • 獲得素材後可以找約翰尼斯進行道具煉成。
    • 煉成的道具亦可以使用溶解劑將道具轉為素材。

劇情[4]編輯

工業革命迎來了一個新時代，也大大地改變了人的價值觀。這意味著精神文化的逐漸衰退。
1783年，樂奇火山的噴發的火災使整個歐洲大陸陷入黑暗之時……
自英國的一部分城市中，一大群惡魔趁著這片黑暗突然出現在人世。

一切的起因，是鍊金術公會擔心崛起的工業主義會讓他們失去富有的投資者，最終造成了力量的失控。
他們通過秘術創造出了碎片人：在人體中移植了晶體，使其能夠與惡魔之力產生共鳴。
鍊金術公會獻祭了碎片人，向人世間召喚出了大量的惡魔。無數無辜的人們死於動亂。
雖然教會成功地征服了地獄，但也付出了高昂的代價。
人們以為碎片人全部已經死於獻祭的儀式。

唯一從惡魔召喚的儀式中奇蹟般生還的青年「吉貝爾」，與另一個因為陷入沉睡而沒有參與儀式逃過一劫的女孩「米利暗」。只有這兩人逃過了一劫。
在那之後經過了十年。而人們也漸漸遺忘了那個悲劇時，惡魔歸來，與城堡一起再次現形。
就在這時，米利暗醒來了。
在得知吉貝爾就是這場新的血腥儀式的幕後黑手後，米利暗和鍊金術師約翰內斯，一同向被詛咒的城堡進發。
登上船後，在那裡等待他們的，將是無法逃避的命運…

登場角色編輯

主要人物編輯

  • 米莉安米利暗[5]（ミリアム / Miriam）【CV：小清水亞美 / Erica Lindbeck】
    • 本作主角，被改造成碎片人（シャードリンカー / Shardbinder）後沉睡了十年、並因此逃過惡魔召喚的獻祭，精神年齡似乎是JS程度。
      • 在《月之詛咒》作為同伴登場時使用聖鞭戰鬥，擁有滑行技能；同時也具備可以跳三格高度+可以蹲下滑動進而取得道具。
      • 於《月之詛咒》若是被斬月殺掉，斬月會得到跳斬，攻擊判定會增加甚至有機會一刀有兩下攻擊判定
    • 甦醒之後發現吉貝爾征服世界的野心，為阻止他而聯合約翰尼斯前往詛咒之城。
    • 貪吃，料理成功會轉圈跳起，拿東西給老奶奶吃會也想跟著吃，吃越多種食物越強。
    • 除了戰鬥外還會彈鋼琴。
    • 除乳量外造型與夏諾雅類似，不過裡面的人卻是賽法（審判）。
    • 在Speedrun中和各前輩一樣不會好好走路。
      • 能夠坐在椅子上揮動真空刃/導向劍並化為光線穿牆、用功夫鞋舞空、用居合取消進行超高速移動的變態
    • 名字的中文翻譯最為玩家詬病，有卡片遊戲的話一定是闇屬性。
      • 但是鍊金術合成最高級的武器，有兩把都是冰屬性，其他有屬性都只有各一種。
      • 不過大概是因為受到抱怨，現時(2020)版本的中翻已經被改成『米莉安』了，然而各種名詞的譯名還是一樣讓人無言例如某條硫磺味的圍巾......甚至還有簡體字。
    • 斬月模式下的最終Boss，精神被黑暗力量給控制，斬月會盡全力送她最後一程。
      • 斬月模式下使用的武器正是冰屬性的單手劍與雙手劍交換用，使用的技能全都是Boss掉落的，並會用逆轉。



  • 吉貝爾（ジーベル / Gebel）【CV：白川周作 / Ray Chase】
    • 米莉安的好友，碎片人，因為造型緣故，OD登場前常被戲稱為阿卡德2號。
      • 在《月之詛咒》作為同伴登場時招式類型也與阿卡德類似。
      • 於《月之詛咒》被斬月(ry，斬月會取得衝刺能力，唯一可以用跑的角色。
    • 在惡魔召喚的儀式中奇蹟生還後，為向人類復仇而召開新的儀式。
    • 本作最終Boss（偽），在完成特定條件前擊敗他的話會進入Bad Ending。
      • 未持有斬月刀的Bad Ending：被米莉安重創後，在死前恢復意識後死去，城堡仍舊留在原地，而約翰內斯則是把事情丟包給多米尼克和其他人後帶著米莉安走人。
      • 持有斬月刀的Bad Ending：被米莉安重創後，在死前恢復意識後死去，而在一旁的失去了吉貝爾的格莫瑞則轉而控制了米莉安成為新的城主。
      • 不過就算完成了真結局條件，他依舊會全身水晶化死去，結果還是死了。
    • 其實有驅魔師血統，過去能夠生還便是因為這個原因。
  • 約翰內斯（ヨハネス / Johannes）【CV：興津和幸 / Ben Diskin】
    • 米莉安的好友，前鍊金術公會成員之一。
      • 會在村內設置工作室製作道具與強化碎片。
      • 日常對話會隨著時間帶而改變。
    • 另一個身分是討伐身為好友的吸血鬼的波紋戰士
  • 多米尼克（ドミニク / Dominique【CV：水橋香織 / Fryda Wolff】
    • 教會方派來支援米莉安的驅魔人，會留在村落提供補給。
    • 本作的幕後黑手兼第三個碎片人，全身的黑緊身衣便是為了隱瞞自己是碎片人的事實。
    • 以各種方式操縱米莉安和斬月等人的行動好讓事情向自己希望方向發展。
      • 雙親為驅魔人，在十年前的惡魔召喚事件中被殺，令她懷疑自己對神的信仰從而改信惡魔，並暗中推動復活巴力的計劃。
    • 奸商，賣東西給她只有一成價格，還會叫米莉安別太依賴碎片要賣碎片給她淨化，但遊戲系統上碎片是越多越強。
      • 實際上賣給她的碎片會被她吸收所用。
    • 預計在月之詛咒2成為同伴登場一樣具備高跳躍能力，手持長槍戰鬥，一樣具備各樣花招的技能攻擊
  • 斬月（Zangetsu）【CV：安元洋貴 / David Hayter】
    • 來自東洋的退魔士，憎恨鍊金術士以及他們創造出的碎片人。
      • 在《月之詛咒》系列為主角，《暗夜儀式》透過更新成為第二個可操控角色。
      • 同時也是《月之詛咒》Nightmare模式下的最終Boss，被其他三位夥伴解救其靈魂後於另一個世界重生。（場面惡搞洛克人2的片頭）。
    • 需要米莉安完成真結局後才能解鎖，有舊紀錄的需打完最終Boss才能解禁。
    • 雖然討厭身為碎片人的米莉安，不過在必要時還是會提供協助，ツンデレ（蓋章）。
      • 經過數次與米莉安的對立與合作後逐漸消去對她的敵意，之後更將自己的佩刀「斬月刀」送給米利暗用以斬殺格莫瑞。
    • 中途會兩次與他戰鬥，主要使用劍術作戰，血量低下時會將佩刀賦予魔力。
      • 第一次戰鬥時只會賦予火屬性，第二次戰鬥時會依序使用火→雷→冰屬性。
      • 難度在Hard以上且血量低於一半時會發動即死技，若是操作角色被困在圓球之中就會受到殺傷力2xxx的即死攻擊
    • 原本受多米尼克委託對抗阿爾弗雷德，得知其陰謀後與米莉安聯手追擊之。
      • 最後在與格莫瑞交戰時中了對方的觸手落穴陷阱、生死不明。
      • 在True Ending最後米莉安準備離開時會先聽到「別人」的腳步聲，似乎是暗示斬月生還。[6]
    • 哈密瓜假面騎士
      • 真名是友哈巴赫，可以卍解，最強的一招是自身化為月牙這件事。
    • 傳說中的傭兵
    • 會用阿修羅閃空還會冒出色即是空的字樣，你是尤里烏斯假扮的吧！
      • よろしｸｳｿｸｾﾞｼｷ…
    • 斬月模式一開始出現的招式表請務必記熟，之後無法透過任何方式再次出現。
      • 與上面不同，屬性轉換是無->火->冰->雷->無屬性變換，招式與技能也會受屬性不同而有變化。
    • 武器綁定斬月刀，前半段Boss都能迅速解決，但是後半就得依靠技巧才能取勝，斬月只有在存檔點 or 吃到HP最大值增加道具才能回血。
      • 請善用發動招式時的無敵狀態。
  • 阿爾弗雷德（アルフレッド / Alfred）【CV：楠大典 / Christopher Swindle】
    • 約翰內斯的師傅，為了奪取魔法書「羅格艾斯之書（ロガエスの書）」以重組公會而潛入城堡。
      • 在《月之詛咒》作為同伴登場時與賽法同樣主打多花樣的魔法技能，當然也繼承她成為耐久度最脆弱的角色。
      • 《月之詛咒》中的魔法名似乎是拿Inti Creates遊戲蒼き雷霆 ガンヴォルト主角Gunvolt的必殺技名改的[7]。
      • 於《月之詛咒》被斬月ry，斬月會取得二段跳能力。
    • 在隱藏沙漠作為Boss出現時主要會使用瞬移搭配各種賦有異常狀態的魔藥戰鬥。
      • 前期使用中毒與詛咒為主，血量低下時會使用減慢速度的魔法。
    • 原本與多米尼克共謀從吉貝爾手中搶走魔法書來拆城，卻在最後遭對方背叛而被殺。
    • 隱藏的大功臣，十年前把米莉安冷凍睡眠不止救了她，也避免她身上的碎片喚來更多的惡魔，在死前也在城裡脆弱的部分埋好了術式來準備拆城。
      • 說是這麼說，也知道他是擔心徒弟和米莉安，不過你都不跟徒弟們和米利暗商量一下的嗎？

主要雜兵編輯

  • 卡拉波斯（カラボス / Carabosse）
    • 出現在靜寂的庭園的妖精，擊敗她並獲得碎片後可以成為同伴。
    • 和月下的妖精一樣，在主角重傷時會使用回復道具治療之外，還會提示隱藏道具的位置。
      • 但是必須為妖精的xx(恢復藥)才會給予，不會主動給一般的回覆藥；除了道具掉落以外，拿到妖精之書後也可以用鍊金來合成。
      • 雖然會攻擊，不過火力和攻速上是全使魔最低……
      • 在主角坐下時會坐在她身旁，主角彈鋼琴時會唱歌。
  • 殺人理髮師（キラーバーバー / Killer Barber）
    • 受詛咒的理髮師，擊敗後有一定機率掉落「路哈巴拉爾」。
      • 與月下的真空劍（ヴァルマンウェ）類似，攻擊時會出現連發的刀刃特效，上位板也需要四把來作合成。
    • 在狄恩斯赫特大教堂有一位與主角友好的理髮師托德（トッド / Todd）[8]會幫主角變更造型。
      • 除了把髮型拉長剪短還有衣服與頭髮染色外，最神奇的就在於他還能幫你改膚色，其中幾個顏色還根本不是人類的顏色。
  • 鏟鐵鎧甲（ショベルアーマー / Shovel Armor）
    • 與知名橫版動作遊戲鏟子騎士（Shovel Knight）的合作角色，其實就是血咒之城版本的鏟子騎士。
      • 在本作當中以敵人的身分登場，穿了他掉落的裝甲米莉安可以變身為鏟子騎士，連操作方法都跟原作相同。
      • 同時劇情對話的人物立繪也會變成鏟子騎士。
    • 模組基本上兼用遊戲裡面同樣拿長武器的鎧甲惡魔。
  • 莉莉（リリム / Lili）
    • 本作以兔女郎的外型登場，當然一些掉落物與兔子有關。
    • 於邪惡的橋梁地區擊敗她並獲得碎片後，米莉安可以變身成其樣貌。主要是用腳踢攻擊，等級上升能追加各種攻擊方式。
      • 與鏟子騎士一樣，變身後劇情對話的人物立繪也會改變。
    • 變身後攻擊力是看INT，取得該碎片能快速踢死前面幾個Boss。
  • 庫尼庫尼（くねくね / Kunekune）
    • 邪惡的橋梁的隱藏怪物，在車上唯一能坐下的火車座位坐下觀看窗外一陣子即會出現。
    • 本作有名的初見殺怪物，因為對視時的連續大傷害附帶詛咒，還有登場時的場景給諸多玩家留下了印象。
      • 而且等級非常高，沒有事先作好準備一定會被其玩死。
      • 會掉落詛咒抗性碎片。
    • 又名「扭來扭去」，為日本知名的都市傳說。
  • 索尼克（ソニック / SonicHarrier）
    • 出現在巨獸區的藍衣忍者。
    • 在賽跑中擊敗他可以獲得加速器碎片[9]。
    • 捏他同名電玩遊戲主角的敵人。

Boss編輯

  • 威沛（ウェパル / Vepar）
    • 第42柱魔神，襲擊帆船「米涅瓦號」的海中惡魔。
  • 工藝品（クラフトワーク / Craftwork）
    • 寄宿在彩繪玻璃的怨念，米莉安擊敗後獲得抓握能力。
    • 似乎搶了格莫瑞在《月之詛咒》的登場方式。
  • 安德雷斐斯（アンドレアルフス / Andrealphus）
    • 第65柱魔神，擁有漂亮羽毛的孔雀惡魔，米莉安擊敗後獲得二段跳能力。
      • 此能力可以透過煉金另外取得高跳躍的能力，但是重要素材會在開啟二周目時無法繼承以避免惡性Bug。
    • 承襲了二段跳傳授者瑪法斯的優良傳統，對初見而言算是相當親切的Boss，不過太過貪刀還是會被踢飛的。
    • 復仇模式的可用角色之一，斬踢可以令任何敵人硬直。
    • 在《月之詛咒》為STAGE 5的Boss，因為兩邊有無底洞、難度比本篇高上許多。
      • 畢竟這場景在《血之輪迴》是死神用的。
  • 華列克（ウァラク / Valac）
    • 第62柱魔神，盤踞在高塔的虺蜥之帥[10]雙頭魔龍。
    • 擊敗牠後可以從寶箱中取得溴化銀，為拍照必須的道具。
      • 無血模式在相同地點則是取得吸血碎片
      • 洗出照片後可以找OD交換火車票。
    • 和《被奪走的刻印》的螃蟹ブラキュウラ一樣有著地圖進度限制，就算將牠的血量打光之後仍要移動到一定距離才會被擊破，有望成為Speedrun玩家眼中的「強敵」。
    • 在《月之詛咒》為STAGE 4的Boss。
    • 外傳時兩個頭一起被米莉安的斧頭砍到怕了，本傳改為左右夾攻。
    • 利用bug穿透地面繞到其背後，會發現兩端的模型並非連在一起，斷面也沒有做特別處理。
  • 暴食火車（グラットン・トレイン / Glutton Train）
    • 與火車頭融為一體的惡魔，戰鬥有時間限制[11]。
      • 斬月模式及無血模式下取消時間限制。
    • 在《月之詛咒》為STAGE 1的Boss。
  • 巴欽（バティン / Bathin）
    • 第18柱魔神，擁有光速移動能力的惡魔，擊敗後獲得反射能力。
    • 復仇模式的可用角色之一，不能跳躍但可以光速移動，且衝拳威力驚人。
    • 在《月之詛咒》為STAGE 7的Boss。
      • 外傳中死時的自滅攻擊會極限加速至超越時間產生分身，可惜本傳沒了。
  • 深淵衛士（アビスガーディアン / Abyssal Guardian）
    • 位於里弗人工意識圖書館，守護通往終焉之間通路的巨龍。
    • 之後在巨獸區成為雜魚，與月下圖書館的レッサーデーモン命運類似。
      • 也因為這樣在里弗人工意識圖書館打倒它不會100%掉落碎片。
  • 無血（ブラッドレス / Bloodless）[12]
    • 操縱血液的吸血鬼，擊敗後獲得吸血能力。
    • 由於是女性外型人氣很高。
    • 作為遊戲難度分界點的BOSS存在[13]，本身因一如字面的回血能力也非常硬。
    • 因衣服也是血構成，血量下降時衣服也會減少，不過不會脫光(超長裙->長裙->褲裙)。
      • 衣服顏色與玩家在功能設定裡的血液顏色同步，若是顏色選白色那就是白衣。
    • 復仇模式可操作，跳起來能滑翔，亦在2020年11月確認為額外追加的可操作角色(Switch版則是12月；更新後在創角人物名稱大寫輸入BLOODLESS即可使用)。
      • 這時候衣裝的變化是依據MP的存量來決定，攻擊時的聲音也會改變。
      • 需要到城中各處取得與衣裝同顏色的碎片來強化能力及增加攻擊招式；與原版不同因是從原版打她的房間開局，部分怪物的血量與傷害力有改；在取得沖天跳之前沒有兩段跳增加一些難度，而且還致敬了月下拿到特定能力之前碰到水會受傷這個設定；最後通關後就是坐在王座之間成為該城的新城主。
      • 雖然沒有特別大幅度製作專屬過場動畫，在一些場合會出現特別的口氣，如進入村落時見到空無一人時會說人都到哪裡去了。初見格莫瑞不但有短的過場動畫且會說找到妳了，打倒後會說出去死。
    • 在《月之詛咒》為STAGE 6的Boss。
  • 分身（ドッペルゲンガー / Doppelganger）
    • 以要殺死對象的相貌出現的惡魔。
    • 和月下出現的不同，裝備與碎片能力為固定(武器武士刀)，不會受到玩家裝備的狀態影響。
      • 而且就固定米莉安的樣子，斬月模式與無血模式對上它並沒有一起改變。
      • 但要注意因為拿武士刀，被奧義打到超級痛。
      • 從碎片的升級素材可以看出就算是分身，還是跟本尊一樣貪吃。
  • 歐若博斯（オロバス / Orobas）
    • 第55柱魔神，可以改變引力的惡魔，擊敗後獲得重力操控能力。
  • 華利弗（ヴァルファール / Valefar）
    • 又名「瑪列法爾」，第6柱魔神，從硬幣中汲取能量的惡魔。
      • 第二階段的血量和金錢掛勾，錢越多HP越高，而且會超越原本的HP上限。
        • 若身上的所持金額是0則會當場掛掉。まさか！無一文なのか……

      • 此作中本體其實是整個寶庫，戰鬥中那堆錢塔其實也是他的身體一部分，攻擊錢塔其實比攻擊人形部份還更有效率。

    • 包含這個的所有隱藏Boss，可以重複挑戰+會掉落鍊金合成最高級裝備的素材，代價就是不會100%掉落碎片。

  • 8bit霸主（8ビットオーバーロード / Eight Bit Overlord）

  • 復仇戰士（レブナント / Revenant）

    • 另外自林賽那接來的討伐任務，死去的人名字叫作里希達[14]，職業是解放鎖匙的變態無賴。
