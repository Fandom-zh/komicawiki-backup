略歷編輯

  • 安妮‧伯尼
    • 愛爾蘭出身，家世顯赫，與情夫私奔到巴哈馬後結識有「印花布傑克」外號的海盜傑克‧瑞克姆[2]。
    • 與傑克私通並上了他的船，展現出剽悍本性，以女海盜的身份活躍於加勒比海。
  • 瑪麗‧瑞德
    • 英國出身，自幼被當作男孩扶養。後來成為軍人，退伍後與軍中同袍結婚經營酒吧，然而丈夫後來卻因意外身亡。
    • 後來取得私掠船資格，成為海盜，最後因為遭到傑克的打劫而不得不以男裝打扮繼續加入印花布傑克旗下。
  • 起初安妮與瑪麗感情並不是那麼好，安妮甚至還把男裝的瑪麗當作可以誘惑的美少年應對，逼的瑪麗只好向她表明自己是女性。
  • 因此不久後兩人便發展到如膠似漆的關係。
    • 為了確保自己不會被誤以為是安妮的外遇對象而使傑克吃醋，瑪麗也對傑克表明了自己的女性身分，之後便作為傑克的情人與船上唯二能打的人留了下來。
    • 但船上只有安妮與傑克知道瑪麗實為女性，除此之外的船員仍以為她是一名男性。
  • 在傑克的船遭到牙買加統治者巴奈特的攻擊時，安妮與瑪麗為了船上這群派不上用場的男人奮勇抵抗，最後仍寡不敵眾遭到逮捕。
  • 同船的海盜都被處刑了，兩人因懷有身孕而得以緩刑。瑪麗最後死在獄中，而安妮得到父親保釋，最後平穩的度過餘生。

故事中經歷編輯

  • 在第三章《封鎖終局四海》登場，被黑鬍子招攬加入海賊團。
    • 因為看到噁心的黑鬍子[3]而讓瑪麗很想把他一刀劈了，但被安妮阻止，雖然她也很想轟掉他。
  • 在與德雷克第二次交戰時率先迎擊，落敗後消失。
  • 終局特異點時再度登場，做為黑鬍子的船員一起攻打魔神柱。

泳裝Archer編輯

擁有技能編輯

Rider編輯

  • 航海： A ─ 操舵能力，船專用騎乘，不適用於馬或戰車等。
  • 接舷突擊： B+ ─ 在六周年從者強化活動由航海強化而成。
    • 遊戲中除了原先的打星率提升外，追加1回合的嘲諷狀態、3回合的根性狀態、3回合NP獲得量提升的效果，能夠更適合承受敵方致命攻擊後以極低血量以寶具反擊。
  • 射擊： B ─ 安的射擊技巧，能在搖曳的船上準確命中對手，無論直射還是跳彈皆擅長。
  • 絢爛的海賊姬：B+ ─ 在《F/GO》強化任務中由射擊強化而成。
  • 除了原先的暴擊傷害+100%持續1回合外，並追加獲得星星25顆。
  • 聯合： C ─ 與特定人物並肩作戰時得到加成，C級能靠一個眼神傳達想法，對二人的寶具幫助甚大。

泳裝Archer編輯

  • 海濱花： A+ ─ 在海灘走過時會吸引四周的目光，A+級可以說是名人。儘管不是著名的大英雄，安的肉體對男女都有相當的吸引力。
  • 探險家（海）： C → A[4] ─ 要說海的話當然是尋寶啦！
    • 只有以海為生之人才被允許擁有的技能。有中概率能發現沉入海底的財寶。
  • 海盜的榮耀： C+ ─ 海盜的獨特價值觀，包括精神污染、勇猛和戰鬥續行的效果。因為並非船長，等級稍稍下降。
    • 但是若是船長不中用的話，反過來這榮耀也會提高。

簡評編輯

 
  • 兩人都可算是最有名的海盜之一，但相比黑鬍子、基德船長或黑色准男爵之類的狠角色，她們倆只能算是在三腳貓集團中顯得傑出的二流小海盜。
    • 當初受審時兩人的故事震驚歐美，其中一個原因是兩人與傑克間的風流逸聞，另一個原因為兩人的女海盜身份。
    • 在過去帶女人出海可算是東西方共通的禁忌，雖然對海盜來說這並不是什麼金科玉律，但帶女人出海仍算少數，更遑論作為海盜了。
    • 也因此，她們倆也被某些人視為打破傳統藩籬的女性代表之一。
  • 兩人在搶劫與戰鬥多穿著男裝，日常生活則穿著女裝。
    • 而她們在殺死對手前會露出前胸，藉此嘲笑對手連個女人都打不贏。有夠費工夫的。
  • 一開始的設定為五星，正式實裝後減為四星。
    • 不過跟兩星、卡面偷工減料、完全變成搞笑角色的黑鬍子比起來，待遇簡直好到不行。黑鬍子UCCU。
  • 沒有想用聖杯實現的願望，單純只因海盜追求財寶的本能回應召喚。
  • 兩人對待Master的態度都是以不分上下的同位階身分來看待，然後偶爾會有些叛逆的表現
    • 對將自己視為上位者的Master，叛逆行為會更加嚴重，所以如時臣這樣會將Servant視為使魔的標準魔術師來講，相性是最差的。
  • 遊戲中在戰鬥時只有瑪麗會待在場上，安妮則是只有施放寶具時才會出現。
    • 最有名的被害者（？）大概就是中村悠一了。抽到時很興奮的喊道「金髮姊姊prpr」[5]，結果發現戰鬥時只會出現瑪利，怒喊道「別開玩笑了」[6]
    • Archer職的泳裝型態時剛好相反，以安妮為主。終於可以實現兩人同時上場的夢想了
  • 二人一組的Servant，其中一方被擊破另一人也會立即倒下。
