解說編輯

  • 香港電影星爺系列作品，另有《白面包青天》的第二片名，於1993年上映，改編當時的著名古裝劇《包青天》以及借用了部分翁虹主演電影《滿清十大酷刑》、《清宮秘史》的橋段。
  • 此片可謂星爺系列中成句產出量最高的一部，主要是因為電視台不停重播下造成的內化記憶，加上鄉民在網路上討論經常引用，使得笑果歷久彌新、老而不爛。
    • 甚至不只是成句，連劇中人物所提到的角色、物品名稱都已經被賦予了特殊定義
      • 尚書大人還真機靈
      • 天橋底下說書的
      • 尚方寶劍－鹹魚
      • 蘋果(菠蘿)心－吃完就甩啦
      • 拿明朝的劍斬清朝的官
  • 本片在上映時因不同地區要求有刪減，例如一些性暗示、不雅文字與搞基行為的片段，但仍以有刪減過的台灣配普通話版流傳最廣。
  • 劇中有數段以高語速進行的對罵場面，相當考驗配音員的功力

角色編輯

包龍星編輯

  • 演：周星馳
  • 包青天的清代子孫，生父包不同（跟金庸《天龍八部》那位純粹撞名）。
    • 龍星可能是包拯的稱號－包龍圖（稱號:龍圖閣直學士）改來的。
  • 由於其父包不同原先貪瀆，遭到天譴，12個兒子都早逝，最後包不同散盡家財離開官場，才保住這第十三個孩子。
  • 母親則在正式出場時老人痴呆，似乎也是報應之一。
  • 從小立志當清官抓貪官，卻被娘親訓了一頓。（國語版有刪剪）
    • 原因是當時包不同仍為貪官，作清官等於是跟包不同作對。
  • 後來花大錢捐了個芝麻小官，但是不夠狠又不夠奸，弄得撈不到油水還被民眾臭罵兼丟菜不要玩食物！
  • 某次因為運氣好巧擒豹頭而受到奸商戚家的青睞，也因為這次為民除害而逐漸轉向正派。
  • 對於戚秦氏有好感，所以戚家慘案發生後決心要公正嚴審。
    • 但方唐鏡為了串證放給他許多銀兩讓他在貪官與清官間掙扎，最後還是銀兩獲勝。
  • 由於常家布局得宜，因此一覺醒後發現此案已轉由被收買的上司陳知縣審，所以戚家慘案成為一樁冤案。
  • 因戚秦氏入獄加上六月雪[1]，使他決心為戚秦氏申冤。
    • 戚秦氏在被收押前因誤認包龍星出爾反爾的怨毒眼神，成為包龍星擺脫貪官的關鍵。
  • 結果不但被陷害，家中又逢變故，到京告御狀的途中更是一波三折，在途中共有兩段姻緣，但很難得的仍選擇將重心擺在告御狀上，沒表現得花心輕浮或是見色忘本。
    • 在雜耍團學到飛刀和吞劍但都只學了個皮毛，辯駁技巧是從妓院的老闆娘身上偷學並青出於藍，更獲得「吵架王(國語版)」的稱號。
  • 後來偶然得到皇上的幫助，受封八府巡按[2]重審此案，將旅途中學到的技能威力全開。
  • 最後成功審清戚家血案，令冤罪平反、真兇伏法。隨後辭官轉行開藥鋪，主打商品正是戚秦氏給的生子秘方。
  • 很好吃
    • 「包大人！好吃的油炸包大人，油炸包龍星呀！」
  • 「貪官奸，但清官要更奸」

包有為編輯

  • 演：吳孟達
  • 包龍星的姪子。按照包父與包母的年齡推算，可能是包龍星頭幾個兄長所留下來的，才會比包龍星更老。稱其十三叔。
  • 性格上如同吳孟達與周星馳初期搭檔時的三叔，有時候會做出些自私行為，但也不算太可惡。
    • 「我還有點餓...」
  • 在調查屍體被逮後，自己提報為汙點證人，事後帶了如花前來劫獄。
  • 似乎是為了掩護十三叔的福利與吳廣得搞基，事後真的定情（國語版中刪除）。
  • 被餵餅後自己跑去當了乞丐，事後被包龍星惡整到曬得跟非洲雞[3]一樣

豹頭編輯

  • 演：徐錦江
  • 本名雷豹，隸屬於京城六扇門[4]中的天下第一神捕。
  • 功夫非常了得。
    • 盜賊甲：豹頭，你就那把刀厲害，有種把刀放下。（被擒拿+一掌拍死）
    • 盜賊乙：豹頭，你就那雙手厲害，有種就不要用手。（被一腳踹死）
    • 盜賊丙：豹頭，你就那雙腿厲害，有種手腳刀都不用。（被獅子吼震死）
  • 經常做些霸道行為，業績可能也是這樣衝上來的。
    • （召妓）「兩個怎麼夠？我要二十個！」
    • 「我玩完她，不給錢就不算賣囉～」
    • 「我全都要」
  • 似乎學過韋小寶的絕招抓奶龍爪手（受害者：有為）。
  • 雖然手、腳、刀、吼聲都能殺人，但輕功似乎不是很好…
    • 不小心掉到縣衙後門被鄉民挖出來的坑還因為鄉民丟的菜太多轉眼間被活埋。
    • 追包龍星到碼頭邊，沒武功的包龍星跳船成功，有武功的他自己卻不敢跳。
      • 連跳個桌子都會被拐倒，還說是高手
    • 皇帝親封英明神武高手高手高高手(笑)。
  • 形象很像鹿鼎記中的鰲拜因為裡面根本都同一個人，不過跟鰲拜相比性格帶有較多喜劇元素在其中。
  • 起初因被包龍星用計擒住而站在敵對立場，後來在鳳來樓的床底下和包龍星、皇上共謀脫身後和解。

如花編輯

  • 演：李建仁
  • 包家的下人，曾經在江南當搶匪，後來跑去香港開美容院。
  • 負責去幫包龍星留種和逃獄，絕招是無敵撞穿牆。
  • 第一個被丟菜的
  • 此角是李健仁首度以如花之名串場。

吳好緹[5]（莫再提）編輯

  • 演：鍾麗緹
  • 上京表演的雜技團女團長，收容包龍星與有為。
    • 因為耳聞包龍星為了幫戚秦氏申冤而丟官的義舉深受感動，兩人結下良緣，最後嫁給了他。
  • 在重審時與兄長吳廣得協助有為扮演被腰斬的藥行老闆。

吳廣得[6]（莫再講）編輯

  • 演：梁榮忠
  • 上京表演的雜技團團長，吞劍與飛刀技巧一流。
    • 誤會了包龍星是女扮男裝，但從後來的表現再看，應該是個基佬。
    • 粵語完整版中，與包有為湊成一對。
  • 好像還是個笨蛋。
    • 「以你的智慧我很難跟你解釋呀！」
  • 另外本身還有縮骨功，再小的洞都可以鑽進去。
    • 曾與人比誰最慘

方唐鏡編輯

  • 演：吳啟華
  • 別名荒唐鏡，屬於專幫富商惡霸欺壓良民的訟棍，廣東民間傳說人物－四大狀師之一[7]。
  • 奸詐又足智多謀，在公堂上能玩弄包龍星於股掌之間。
    • 「大人剛才說抓手就是強姦，我現在不是在強姦大人嗎？」
    • 「我是跟鄉民進來看熱鬧的，只是往前站了一點」
  • 按照劇中表現，其實是害戚秦氏蒙受冤屈的最大主謀，常家只負責出人力財力執行計畫。
    • 但後期因為包龍星經過吵架特訓早已今非昔比，在公堂上被包龍星壓制。最後被揭發他收買證人而被革除功名。
  • 抖M
    • 「打我啊笨蛋！」(被包龍星與包有為暴打一頓)「大家都聽到了，是他叫我打我才打的」「像這種要求我這輩子沒見過」

常威編輯

  • 演：鄒兆龍
  • 水師提督常昆之子，拜任內務總管五品官的李蓮英為乾爺爺（國語版誤譯）乾爹。
  • 名字疑似參照粵語「常威（經常作威作福/耀武揚威）」
  • 天生神力（自稱），學過武功，再加上面所述，十足的富二代跋扈子弟。
    • 天下第一神捕豹頭跟他對打時還一度居於下風[8]。
    • 也曾和葉師傅還有尼歐交手過
  • 花花公子，和自己的表妹（戚秦氏的婆婆，但是續弦）有染，也不能怪他啦，娶的老婆比人家後母還老。
  • 在戚家的喜宴見到戚秦氏（秦蓮）後不但侵犯了她，還把戚家十三口+一隻狗全殺掉。
  • 而之後在包龍星重審戚家血案被包龍星以滴血驗親的騙術騙到，因慌亂而被套出話來。
    • 「一定不是我的！那天晚上我玩完她後第二天就有了，哪有那麼快的啊！」「喔~那晚（消音）」
    • 隨後才知剛才和自己滴血驗親的嬰兒是自己和自家妻子生下的小孩。「你這個兔崽子，你敢陰我！」
  • 事跡敗露後挾持著李蓮英，最後被豹頭扔到虎頭鍘，當場被包龍星處決。
  • 雖然鄒兆龍在出演本片時仍有些不情願，但由於本片的多年口碑，常威也成為鄒兆龍的生涯代表角色[9]。

皇帝編輯

  • 演：黃一山
  • 雖然劇中沒有明顯表明，但根據劇中提到老佛爺和李公公還有最後的死因，應該是同治皇帝。
  • 痴漢（各種意義上）。
  • 在鳳來樓中認識包龍星，用鬥獸棋的理論建議由包龍星出面擺平協理大臣[10]。
  • 因為嚇掉了的內褲被包龍星撿到而被識破身份，答應幫助包龍星將戚家血案發還重審。
  • 最後因花柳病不幸駕崩，嚇得包龍星立刻把他的龍內褲扔掉。

李蓮英編輯

  • 演：劉洵
  • 內務府五品總管太監，角色原型為慈禧太后寵宦李連英。
    • 雖然因受慈禧太后寵信而常在後世作品中被醜化，但史實上的李連英個性謹小慎微、圓滑周到，也從不仗勢欺人作威作福，與本作中展現出來的形象大相逕庭。
  • 常威的義父，常昆的義兄（國語版升等為乾爺爺與乾爹）。
  • 為了幫常威脫困而帶著黃馬褂前來公堂助陣，雖被包龍星的嘴砲壓制，但仍使出不少妨礙。
    • 你好大的官威呀～包大人！
  • 實際上只因被蒙蔽不知實情才偏袒常威、與包龍星作對；最後得知真相正欲離去卻遭到常威挾持，在豹頭的搭救下撿回一命。
    • 剛來時就說過若常威真的作奸犯科也饒不了他，而在案情水落石出後也氣得丟下常威拂袖而去雖然沒成功，可說性格還算公正，並非有意包庇奸惡。

常昆編輯

  • 名字疑似參照粵語「經『常』『昆』人（經常騙人（昆一字經常單獨使用且同義））」

  • 吩咐家丁在戚家現場亂畫了許多春宮圖(國語版刪除)
  • 第一個被提老母的人
  • 最後跟花尚書、陳知縣一同被送去問斬。

戚秦氏（秦蓮）編輯

  • 演：張敏
  • 戚家剛過門的媳婦，但新婚三個月就被常威侵犯，隨後還被陷害為戚家血案的主謀，過程中更受刑多次，本作最大苦主。
