TEXT IS 解說編輯

  • 由芬蘭人Arvi Teikari開發的益智遊戲，在2019年3月在Windows，Linux，macOS和Switch發行。
    • 雖然是傳統走格子的解謎遊戲，但是因為幾乎所有的物件的規則都可以被修改而導致遊戲的自由度接近無限大。
      • 這也是為什麼本條目的告示欄全都是這種規則
    • 玩家如果按著平常的思維遊玩的話會無法通關，極度考驗玩家的想像力、推理力和洞察力，因此也相當燒腦。
    • 由於其自由度高到無可比擬，目前已經有許多開發者成功將之變成一種程式語言並且用來重現一些經典的遊戲例如是俄羅斯方塊、Pong和吃豆人等等。BABA IS YOU MAKE GAME
      • 編者絕對不會講開這個條目的原因只是因為獅白白有玩過這個遊戲，BOTAN IS CUTE

TEXT IS 遊戲規則編輯

  • 以下內容中，屬性 IS 帶有該屬性的物件。例如：YOU=帶有YOU屬性的物件。
  • 每關會有不同的佈置，只要操控YOU到達WIN就算勝利。
    • WSAD控制。
  • 遊戲中會有不同的文字方塊，每一行構成規則的文字方塊都代表著現在關卡的規則。
    • 沒錯，雖然遊戲名字叫BABA IS YOU，但並不代表BABA永遠都會擔任YOU。
      • 有時候不只要到達WIN，甚至需要改變物件WIN代表的事物來取得勝利。
    • 遊戲內會有各種的障礙和陷阱來阻止玩家到達WIN，玩家需要想辦法來通過，破壞或者是無視掉。
      • 如果因為各種原因導致場上沒有YOU存在，就算玩家失敗。但好在遊戲可以隨時按Z來返回上一個行動以調整策略，或者是直接按R來重設關卡。
      • 另外，在遊戲的關卡選擇畫面其實也符合以上的規則，因此如果想辦法把既有的框架打破的話......？
      • 順帶一提，編者卡關了，到現在都還沒打贏第二區的第二關

TEXT IS 物件解說編輯

  • 此處將會解說場上經常出現的物件。

ITEM IS BABA編輯

  • 一隻像是兔子或是倉鼠之類的白色生物。通常是玩家操縱的物件，也就是本作的標題BABA IS YOU。
    • 不是爸爸。

ITEM IS FLAG編輯

  • 一面黃色的旗子，終點的象徵。
    • 不過隨著關卡越變越難，不時會被作為誤導之用(例如先拚好FLAG IS WIN給你看，但實際上根本無法達成，解答是另一個物件)。

ITEM IS WALL編輯

  • 磚牆，屬性STOP的代表。
    • 玩家大部分時候要克服的困難來源，如何把物件或文字從牆壁圍繞的狹小空間中帶出是常見的難題。
    • 除此以外，FENCE，HEDGE等等都是常見擔任屬性STOP的物件。

ITEM IS ROCK編輯

  • 一塊大石頭，可推動物件的代表。
    • 有時會作為障礙物卡住窄路，有時是玩家越過水漥的踏腳石，為多數機關的構成者

ITEM IS KEKE編輯

  • 二足步行的橘色有腳的可可豆生物。
    • 常會做為2P定位讓玩家能分別或同時操作兩個物件以在不同地方作業。
    • 偶爾也會單獨出場。
      • 除了KEKE，ROBOT（一隻灰色的機器人），GHOST（一隻粉紅色的阿飄），ME（一隻二足步行的紫色的衰小臉）等等都是常見2P角色。

ITEM IS SKULL編輯

  • 骷髏頭，幾乎都跟DEFEAT成雙成對出場。
    • 禁區擔當，常常一次出現一大堆並附上DEFEAT把地圖分成兩塊，要玩家想盡各種辦法越過界線或是把它消滅。

ITEM IS WATER編輯

  • 水體。屬性SINK的代表。
  • 通常會以河流和湖泊的形式出現，這時候就要想辦法消除之。

ITEM IS LAVA編輯

  • 熔岩。屬性HOT的代表。
  • 通常會配合BABA IS MELT來阻止玩家前進。

ITEM IS BOX編輯

  • 一個木盒子。
    • 規則HAS的代表，經常需要破壞盒子以取得其產物（例如是屬性WIN的FLAG，或者是屬性OPEN的KEY）。

ITEM IS DOOR編輯

  • 一堵門。屬性SHUT的代表。
    • 通常會配合STOP來擋住玩家的去路，玩家需要想辦法取得屬性OPEN的物件並開啟（破壞）之。

ITEM IS KEY編輯

  • 一根鑰匙。屬性OPEN的代表。
    • 通常作為破壞SHUT物件的關鍵，但是大部分時候不是少了PUSH推不動就是沒有OPEN，玩家需要想辦法湊齊之。

TEXT IS 屬性解說編輯

  • 注意：RULE IS 擁有該屬性的物件。

RULE IS YOU編輯

  • 玩家操縱的物件。可以多於一個，也可以不只一種。
    • 如果關卡中一個YOU都不存在的話就算失敗，必須返回上一個行動或者是重設關卡。
    • 另外如果沒有物件為YOU的話也算失敗。
    • 推成YOU IS WIN的話也不會直接過關的，請多用點心思。

RULE IS STOP編輯

  • 會擋住除了FLOAT以外的其他物件的移動。
    • 牆壁大部分時間都以STOP存在。

RULE IS WIN編輯

  • 當有此屬性的物件和有YOU重疊在同個位置時即過關。
    • 若是YOU跟WIN指向同一個物件也算是過關。
    • 判定優先度較低，若是該物件同時有DEFEAT等會造成物件消失的其他屬性的話，碰觸後會因為先判定消失而過關失敗。

RULE IS PUSH編輯

  • 可以推動的物件，最基礎的可互動屬性之一。
    • 雖然是解關的常用便利屬性，不過有時候關卡會故意安排WIN也附上PUSH，如此一來就沒辦法直接走上該物件，讓你看的到吃不到。

RULE IS DEFEAT編輯

  • YOU物件碰到即消滅。
    • 同WIN，若是將YOU附上DEFEAT也會直接消滅。

RULE IS SINK編輯

  • 會和除了FLOAT以外所有經過的物件一起消滅。
    • 只要有東西碰到就會被消除，為比較簡單的障礙。

RULE IS HOT編輯

  • 會消滅所有經過的MELT。
    • 只要不是MELT就可以通過，為比較簡單的障礙。

RULE IS WEAK編輯

  • 當碰到任何沒有PUSH，FLOAT的實體即消滅。
    • 除了用來阻止玩家前進，也會配合HAS來取得其他物件。

RULE IS TELE編輯

  • 當TELE有兩個或以上存在時，走上去就會傳送到另一個TELE的位置。
    • 可以利用這點來消除一些障礙，例如是DEFEAT的牆壁。
