簡介[編輯]

  • 遠藤達哉所創作、2019年3月25日起於日本《少年Jump+》上定期連載的漫畫作品，台灣則於《寶島少年EX》上連載。
    • 2021年11月1日宣佈動畫化，由WIT STUDIO與CloverWorks製作，2022年4月起分上下兩季出。
    • 台灣衛視中文台頻道於2022年7月9日引進，每週末晚上6點整開播，每次2回。

劇情簡介[編輯]

西國一流間諜「黃昏」奉命調查東國政治家唐納文·戴斯蒙德之政治陰謀。

然而，戴斯蒙德深居簡出，唯一公開露面的場合只有他兒子所就讀的東國名校——伊甸學園舉辦的懇親會，故只有讓「自己的小孩」進入伊甸學園就讀，並以家長的身份參加懇親會，才有機會與其接觸。

為順利執行任務，黃昏以「洛伊德·佛傑」的身份掩人耳目，並先後找到「女兒」安妮亞和「妻子」約兒組建一個臨時家庭

殊不知安妮亞其實是有讀心能力的超能力者「實驗體007」、而看似溫柔善良的妻子，另一身份竟然是代號為「睡美人」的職業殺手。

三人互相隱瞞真實身份，就此過起虛假的家庭生活。

登場角色[編輯]

佛傑家[編輯]

洛伊德·佛傑（ロイド・フォージャー）[編輯]

  • CV：江口拓也
  • 本作男主角，在東國從事諜報活動的西國頂尖間諜，代號「黃昏（たそがれ）」，為了成為間諜已經捨棄自己的本名
    • 接收了「行動代號：〈梟（Strix）〉」。現在以伯林特綜合醫院精神科醫生「洛伊德·佛傑」的身分行動著
  • 曾是戰爭孤兒，為了「打造一個不會讓孩子哭泣的世界」以及讓平民們過上幸福的生活而成為間諜
  • 戰鬥力、記憶力、情報處理能力、偵查和反偵查能力都很優秀，同時是個易容高手，每次任務都使用了不同的面貌和名字
    • 長相端正帥氣，本身性格沉著冷靜，即使不做間諜在外人看來無疑是個居家好男人的形象，也因此讓尤里完全挑不出刺
  • 故事最開始已經是精英間諜，被委派執行「梟」作戰後被要求「請在一周內結婚生子」而先後安妮亞和約兒組成家庭
    • 雖然一直強調自己作為間諜已經拋棄了感情，但在與安妮亞和約兒的相處間會體現出人情味
  • 本作苦勞人之一，做事認真，不管做什麼都會在腦中思考多個方案去執行
    • 但由於安妮亞和妻子的脫線行為，經常在不擅長的事上與預期不同而不得不補救，常為了兼顧計劃和其餘任務搞到心力交瘁，開始想戒掉咖啡因
      • 原本是黑咖啡派，但現在為了減少對胃的刺激而開始加牛奶
      • 聽聞安妮亞得了第二個 雷電（Tonito） 時甚至震驚到當場昏眩過去
  • 算上間諜的外勤任務報酬與精神科醫生薪水，年收入大約有兩千萬日幣左右
    • 雖然間諜的收入會隨任務難易度而有所波動，但基本上都會比精神科醫生的薪資還低
  • 比企谷八幡[3]
  • 妖精王[4]

約兒·佛傑（ヨル・フォージャー）[編輯]

  • CV：早見沙織
  • 本作女主角，27歲。 隸屬於暗殺組織「花園」，代號「睡美人（いばら姫）」
    • 舊姓布萊爾，嫁給洛伊德後改姓佛傑
  • 外貌姣好但卻是有著高度反差的殘念美女
  • 表面是玉山銀行職員[5]伯林特市政府的女性公務員，實際上是技藝高超的殺手
    • 在被稱為「店長」的人物指使下進行暗殺行動，專用武器為針狀的匕首
    • 目前的任務成功率接近百分之百，擅長直接了當的攻擊，不擅長用毒或構造複雜的武器，槍械與弓箭都不會用
  • 本作最強戰力（無誤），有著凌駕於「黃昏」之上的身體能力，達到了非人的境界（但對「黃昏」聲稱是學習瑜伽和防身術）
    • 因身為殺手所以身體具備一定的抗毒性，喝下超量河豚毒的酒僅只是全身酥麻並緩解了傷口的疼痛，也能無視足以放倒熊的毒霧
    • 被子彈射穿過臀部也只是暫時不能坐著的程度，之後兩三天內就恢復了
    • 打網球時全力揮拍會把球切碎，即使減輕揮拍力道，其發球的球速甚至超越音速
    • 打排球時把球打到地上會留下大坑，打向空中時球甚至會突破大氣層飛上太空
    • 她的戰鬥力若定義為100，那洛伊德大該有60~70左右，但因為洛伊德的綜合能力較強，也會避免正面戰鬥，所以兩人算是在伯仲之間
    • 技能點數幾乎全都點在這，因此除了戰鬥外都有點問題（見下）
  • 有個和自己同樣是公務員的弟弟尤里，在雙親過世後為了扶養弟弟而成為殺手（但她一直對弟弟說自己是在打工）
    • 兒時擁抱尤里的時候用力過度把他弄到骨折（折斷兩根肋骨），以至於之後觸碰小孩都格外小心，但尤里反而更加喜歡姐姐。
  • 性格極度天然呆，腦袋不太好，容易相信他人的說詞，也因此從來沒懷疑過洛伊德的身份明明是心理醫生卻身手了得，還經常毆打「患者」
    • 平時性情溫順可人，但涉及到家人的情況就會變得相當嚴肅和殺氣十足
    • 知道殺手是絕不能曝光的黑暗工作，但相信花園的作為是剷除壞人、保衛國家的正義之舉，所以深感自豪
      • 除了工作外絕不濫用力量，即便腦袋冒出一個使用暴力的想法也會自我克制
      • 雖然很關心安妮亞，但內心不經意間暴力的想法通常會被安妮亞用讀心術知道，因而嚇個半死
    • 基本上什麼事情都會聯想到「殺」有關的術語，算數時會想到如何肢解比較好，志願理由也會聽成「死亡」理由
    • 顏藝威懾力很強，曾嚇跑一隻兇狠的牧羊犬。
  • 酒量差，一旦醉了言行就會變的比較奔放，而且很難控制力道
    • 具本人所知，她醉酒後還沒惹出大麻煩，至少沒因此進警察局
  • 故事剛開始時因為殺手身份而獨居，不過從同事口中聽聞中年女性原來這麼嚴格到27就是中年了嗎沒有對象會遭到國家懷疑後感到擔憂，再加上弟弟的推波助瀾促使她急於找到對象
    • 在常去的服飾店偶遇佛傑一家，被洛伊德的無心讚美[6]跟安妮亞的助攻連段成功插旗，最後因利害一致而合作
    • 之後洛伊德本來準備順手偷了一只不錯的戒指向她求婚，卻在被敵人追殺時弄丟，最後是拿手榴彈的插栓代替這個「戒指」有點大
  • 家事方面了除了「打掃」以外都不行，料理方面則是黑暗料理的毀滅程度（只有尤里不會覺得難吃）
    • 動畫第9集做出了足以媲美英國料理[7]的驚悚早餐，被網友戲稱是「死有魚菇派」
    • 漫畫24話曾經在市政府工作的後輩卡蜜拉的秘密指導下，成功做出自己老家尼盧堡口味的南方燉菜，在洛伊德跟安妮亞吃了後洛伊德評價是可以撫慰人心的味道
      • 不過很快又做出足以讓安妮亞和洛伊德昏厥過去的菜品
  • 雪之下雪乃[3]

安妮亞·佛傑（アーニャ・フォージャー）[編輯]

  • CV：種崎敦美
  • 6歲[8]，洛伊德從孤兒院中領養的女兒[9]
  • 真實身份是某組織代號007號（被検体（ひけんたい）007）的實驗體，因為無法忍受組織不斷逼迫自己學習和沒有童年的生活而逃離研究所
    • 自此使用「安妮亞」的假名並隱瞞自己的能力，輾轉於多個收養家庭與孤兒院之間
  • 擁有讀取他人內心想法的能力X教授，人類和動物的思想都可以讀取，但自身無法完全控制，也不清楚讀取範圍，過度使用能力會流鼻血
    • 每逢新月會失去能力，變成正常的小孩
    • 雖然有著讀心術，但是由於自身心智不成熟+表達能力不足，經常會作出與讀出的想法無關的舉動或是話語，讓黃昏相當苦惱
  • 喜歡的食物是花生
  • 很喜歡觀看間諜動畫《SPYWARS》，崇拜裡面的主角「彭德曼」[10]（實際上是把他當作跟「黃昏」一樣帥氣的人）
  • 因為能力的緣故，是佛傑家唯一一個知道名義上父母和舅舅真實身分的人
  • 成績很差，但也一直在努力，而成績最高的考卷科目為古語，「黃昏」推測她過去曾居住在使用古語的國家。
    • 因為上述在組織的經歷而討厭讀書，剛開始想透過讀心來作弊應付考試，但在〈黃昏〉的提點和過去展現能力而被同儕排擠欺負的經歷，決定不依靠讀心認真讀書準備考試
  • 在黃昏的培養下入讀了伊甸學園，在學校生活中經常和貝琪一起行動
    • 為了幫父親完成任務一直想和達米安打好關係，結果在開學第一天就因為打達米安一拳而拿了一個雷電（Tonito）（集滿8個會直接退學）
      • 後來有對達米安就此事道歉，結果反而讓達米安對安妮亞產生好感，常在本人不知情的情況下，偶然讓達米安對她的印象有所好轉
    • 後來在醫院做親子義工期間用讀心能力救了在泳池溺水的小孩，而成為在伊甸學園同級中第一位拿到星星（Stella）的學生。
    • 在風紀委員唐娜·施拉格老師出場時面臨嚴酷的物品檢查，發現達米安正因為沒帶手帕而面臨不合格的危境，為了避免他得到雷電（Tonito）而將手帕借給他
      • 但因為自己到最後才發現沒有帶第二條手帕，雖然讓對方欠下了人情，但代價卻是收穫第二顆雷電（Tonito）

彭德（ボンド）[編輯]

  • 第四卷起登場，安妮亞的寵物，擁有預知能力的大型犬
  • 原本是東國某個組織以創造高智力動物為目的、「蘋果計劃」下誕生的第8個實驗體
    • 與安妮亞一樣，曾經遭到計劃研究員的非人道對待，後落入由大學生組成的恐怖組織手中，被當作狗體炸彈使用（組織內沒有一個人知道這隻狗的能力）
  • 因為實驗的關係，能夠看見未來發生的事
    • 登場時由於看見了被佛傑一家收養的未來而主動幫助安妮亞（安妮亞也因此知道這隻狗的能力），在恐怖組織的炸彈襲擊計劃失敗後，正式成為佛傑家的一員
  • 名字取自安妮亞喜愛的諜報動畫角色「彭德曼（ボンドマン）」[10]
  • 很害怕約兒的料理，因為透過預知未來得知一旦吃下必死無疑
  • 看起來體型很大，但其實是因為體毛蓬鬆的緣故，被水淋濕後會發現牠其實是隻很瘦的狗。

達米安·戴斯蒙德（ダミアン・デズモンド）[編輯]
  • CV：藤原夏海
  • 唐納文·戴斯蒙德的次子，安妮亞的同班同學
    • 其父親正是黃昏的任務行動的關鍵人物
  • 有一位名叫德米特律斯的哥哥，為皇帝學徒優待生之一
    • 旁人多叫他為達米安少爺，安妮亞則叫他次子
  • 初登場時形象為一個依仗父親權勢的富二代，待人高高在上
    • 在和安妮亞第一次見面那天，因為對她出言不遜、同時汙辱她的養父洛伊德，被憤怒的安妮亞打了一拳，也讓她因此收穫了一個雷電（Tonito）
      • 在安妮亞道歉以後，為她哭泣時的行為所動容，即使想要發火也不知道該說什麼已被攻略，此後倆人成了歡喜冤家，日常相處中常和安妮亞互相吵架
      • 因為安妮亞的態度不同於其他只會奉承自己的學生，平常反而在意對方的態度（讓安妮亞有時會感到一絲惡寒）
    • 自帶濾鏡（對安妮亞專用）
  • 傲嬌屬性
    • 根據作者訪談，角色個性就是《流星花園》裡的道明寺司，難怪身上散發著少女漫畫男主角氣場。
  • 實際上本性不壞，只是出於對家族的榮譽感而使他的姿態比較高
    • 希望父親能在意自己，而努力想成為「皇帝的學徒」，其父母在開學典禮當日並未出席
  • 成績優異，擅長歷史，但國文成績差，曾靠歷史成績取得一顆星星（Stella）
  • 根據作者在單行本的附錄所言，原本與父親唐納文相貌一樣的設定被責任編輯認為太醜而不獲通過，無奈之下才變成現時這副模樣。
貝琪·布萊克貝爾（ベッキー・ブラックベル）[編輯]
  • CV：加藤英美里
  • 安妮亞的同班同學兼好朋友和養母二號，是軍火商CEO的女兒，家境十分富裕。
  • 最開始只想把安妮亞收為跟班而對她友好，後來知道安妮亞打達米安是替自己被踩了一腳而打抱不平（實際是安妮亞情急之下掰出的藉口）後相當感動，將安妮亞視為真正的朋友。

  • 不知道約兒的真實工作是殺手，同樣約兒也不知道弟弟是秘密警察

      • 在這次探訪中因為喝酒、用上東國諜報慣用的偽裝說詞，被洛伊德看穿了他其實是秘密警察的事
