  • 本條目列舉機動戰士鋼彈SEED中的人物名單以及其相關惡搞物。

主要角色編輯

  • 煌‧大和／キラ・ヤマト（CV：保志總一朗）
  • 拉克絲‧克萊因／ラクス・クライン（CV：田中理惠）
  • 阿斯蘭‧薩拉／アスラン・ザラ（CV：石田彰）
  • 卡佳里‧由拉‧阿斯哈／カガリ．ユラ．アスハ（CV：進藤尚美）
    • 生日：C.E.55年5月18日
    • 種族：自然人
      • 小說版中穆和瑪琉在閱讀孟德爾帶出來的資料時推測生父響博士刻意不調整的原因是為了顧慮母親的感受。
    • 歐普代表首長烏茲米‧那拉‧阿斯哈的養女。人稱公主大人。過去受到歐普的正規軍人訓練，因此在歐普相當有人氣。
    • 因為懷疑歐普在秘密為地球軍製造新型兵器，在烏茲米不知情下潛入赫里奧波利斯，剛好札夫特作出襲擊，果然發現G兵器（攻擊鋼彈等五台機動兵器），對烏茲米違背不使用武器維持和平的承諾感到憤怒。但是很快被煌送進救生艙後輾轉回到歐普。
    • 因為烏茲米嗆她根本不知道世界是怎運作的，一氣之下她跑到阿拉斯卡成為反抗軍的參謀（烏茲米倒有默許），對抗著安德列‧渥特菲德的軍隊一段時間。
    • 在煌和阿斯蘭的死鬥後，率領搜救團找到攻擊和神盾鋼彈激戰的小島，找不到煌卻找到被沖上岸，重傷的阿斯蘭，在一番說教(?)後把受治療的阿斯蘭交回札夫特軍。
    • 在大天使號來臨時協助擊倒安德列的軍隊，然而抵抗軍的戰力也所剩無幾，結果搭乘大天使號和煌等人一同行動。
    • 真實身分為煌‧大和的孿生姐姐，這一點直到歐普的質量投射器被爆破前一刻才從烏玆米口中得知。
    • 在嬰孩時由於雙親被藍波斯菊派人殺害，和煌被歐普的人救起，歐普對外宣稱他們已死亡，卡拉里成為烏茲米的養女。煌就被烏茲米安排的另一自然人家庭領養。雖然有保密協議避免大和家庭和烏茲米再度接觸，可是煌和卡拉里變得熟絡使他不得不在歐普被武力接管前暴露這真相
      • 雖然同胞弟弟煌‧大和是最強調整者，但她的受精卵並沒有受到基因調整。與煌相同，具有覺醒為SEED持有者的能力，但事實上用途不大。
      • 官方人物關係圖已明寫為姐弟，並非只是本人自稱而已。
    • 之後與煌‧大和的摯友阿斯蘭‧薩拉發展為戀人。
    • 曾駕駛「FX-550 Skygrasper」與「MBF-02 Strike Rouge」。

地球聯合軍（Oppose Militancy and Neutralize Invasion Enforcer, O.M.N.I.）編輯

  • 或稱反戰反侵略主張者。為地球方最大的陣營。
  • 「哥白尼慘劇」後，「地球聯合」（Earth Alliance）替代崩潰的聯合國成立，旗下包括大西洋聯邦、東亞共和國等數個聯邦制國家。
    • 實際上地球國家中也有如歐普一般保持中立的國家，但往往在以大西洋聯邦爲首的地球聯合軍威逼下傾向地球陣營。
  • 地球聯合軍為其下轄國家軍隊組成的聯軍。

大天使號（原屬，後叛離）（LCAM-01XA Archangel）編輯

  • 瑪琉‧雷明斯／マリュー・ラミアス（CV：三石琴乃）
    • 生日：C.E.45年10月12日
    • 種族：自然人
    • 階級：大西洋聯邦宇宙軍第8艦隊大尉→同少佐。
    • 原為大天使號的副艦長，因為原本的艦長戰死沙場而接任艦長一職。職責是搖乳
    • 在大西洋聯邦的天堂島技術研究所中以技術軍官的身分協助開發相轉移裝甲。後隨技術團隊秘密轉移至歐普的殖民衛星海利歐波里斯。
    • 之後與穆‧拉‧福拉卡發展成戀人關係。大天使號首席閃光彈
    • 比起組織戰鬥更擅長與部下溝通，和副艦長娜達爾正好相反，由於管理方式不同與其有時並不和睦。本人也承認不如其適合擔任艦長職務。
  • 穆‧拉‧福拉卡／ムウ・ラ・フラガ（CV：子安武人）
    • 生日：C.E.43年11月29日
    • 年齡：28歲（MIA）
    • 種族：自然人
    • 地球軍中首屈一指的菁英飛官，曾在月球的安迪米翁隕石坑以MA梅比烏斯‧零式擊落五台基恩，而被稱為「安迪米翁之鷹」，亦是軍中屈指可數能以MA跟MS對抗的王牌駕駛員。
    • 擁有「空間認知能力」，成為軍中少數能單人靈活操作「有線式砲筒」的駕駛員。
      • 然而官方表示他並不是NT。
    • 與ZAFT軍中的拉烏‧魯‧克魯澤是多次在戰場交手的死對頭。
    • 真正身份為地球聯合軍政治家亞爾‧達‧福拉卡的兒子。因為自然出生，繼承了部分母親的基因而不能完美地繼承父親擁有的空間認識能力，因此亞爾尋找科學家尤連．響（即煌的生父）資助後者製造亞爾的複製人，那就是拉烏。
    • 在亞爾死於拉烏的縱火之前，亞爾一直把穆當作真正的兒子照顧，暗地裡卻想著之後把穆捨棄，只是亞爾的急死粉碎整個計劃，結果令他繼承了父親的遺產而且對拉烏和他的關係毫不知情。
    • 名言是：「我是化不可能為可能的人」
    • 之後與瑪琉‧雷明斯發展成戀人關係。
    • 先後駕駛「TS-MA2mod.00 Moebius Zero」（宇宙）、「FX-550 Skygrasper」一號機（大氣圈）與「GAT-X105 Strike」（煌‧大和改乘Freedom後）。
    • 片末挺身擋住主天使號擊向大天使號艦橋的「羅安格林」砲火，導致攻擊鋼彈全毀，本人也消失在了宇宙之中。
      • 原片結局中有他的駕駛服頭盔與攻擊鋼彈碎片一起漂浮于宇宙中的鏡頭。在總集篇中相同的鏡頭刪除了他的頭盔。這麽大一個便當都能吐掉
    • 某歐洲小國的遺孤
  • 芙蕾‧阿斯達／フレイ・アルスター（CV：桑島法子）
    • 生日：C.E.56年3月15日
    • 年齡：15歲（KIA）
    • 種族：自然人
    • 「大西洋聯邦」外務次長喬治‧阿斯達的女兒，煌的同學。
    • 由父親在逃難時搭乘之太空船被ZAFT軍擊墜身亡後，一心就利用煌當作復仇工具，開始親近煌；以身體為交換迫使煌作出守護她的承諾，以誘導煌與調整者自相殘殺為樂。
    • 一開始與賽伊有婚約，之後片面與他分手。但是不小心漏了不是真心愛煌的口風，令到他心中對她產生不信，但是因為實際上煌一直保護所有人，尤其還填補她的心理缺口，之後反而是她對煌產生感情。
    • 卡佳里等人的攪局對煌的支持之下漸漸揭露芙蕾的詭計，煌的心理狀態結果穩定下來，結果反而是芙蕾自己被孤立，自業自得。在之後開始反省但是直到煌MIA為止也沒有把握道歉和修補感情的機會。
    • 在煌MIA時喪失了依賴的對象，但是把他的消失當成價值已經用盡了，把賽伊當成煌的替代品（？）並企圖投回他的懷抱，卻被賽伊看穿，理所當然的被狠狠的嗆了。
    • 不時有暴走的行為，不但嗆拉克斯，還幾乎射死了迪安卡。
    • 在地球軍指令下本應脫艦移送去當政治宣傳，卻在試圖逃回大天使號時遇上阿拉斯卡基地遇襲，用槍指嚇ZAFT特高階將領拉烏‧魯‧克魯澤，卻因為形象和父親相似（也是關俊彥配音），錯愕之下被打暈俘虜。
    • 之後拉烏把刻意洩漏「反中子干擾器」的情報給她並將她送回主天使號。
      • 在被俘虜時似乎受到了不知名狀的遭遇[1]使得伊扎克不得不忠告應該以俘虜的方式對待她
    • 最後她所搭乘的逃生艇被拉烏擊墜，陣亡。她的意念向煌謝罪，但是煌可能沒法理解。理解了可能沒有種死這回事了[2]
    • 惟一（？）成功上過煌的女人
    • 志願成為軍人前經常穿著粉紅的連身裙，而且因為非常腹黑，可以稱為粉切黑。和種死的拉克絲‧克萊因腹黑程度不知鹿死誰手
      • 在機戰時的補正很不一致，有塑造成稀代惡女，有和煌和解甚至以乘員身分隨隊的場合…
    • 桑島魔咒受害者之一

    • 生日：C.E.55年2月17日

    • 煌在海利歐波里斯的同學，因為ZAFT攻打海利歐波里斯而逃到大天使號，之後擔任艦上的飛航管制員。

    • 曾負責照顧被俘的ZAFT駕駛員迪亞哥‧艾爾斯曼，有用手術刀攻擊他的嗜好。
    • SEED時有疑似暗戀迪亞哥的傾向，但DESTINY時就徹底遺忘
  • 賽伊‧阿蓋爾／サイ・アーガイル（CV：白鳥哲）

    • 原與芙蕾有婚約，但被煌‧大和NTR芙蕾片面中止。

    • 擔任大天使號上的副操舵手。
    • 自告奮勇操作「FX-550 Skygrasper」二號機升空迎戰ZAFT鋼彈部隊，遭到阿斯蘭甩開神盾鋼彈的防彈盾砸中駕駛艙陣亡。
      • 在HD Remaster內可發現主要死因是防彈盾衝擊駕駛倉引致頭部被猛力推離頸以下的身體，雖然之後的爆發很可能也燒掉身體了。這樣血腥的畫面果然不能在電視播…
  • 娜達爾‧巴吉路／ナタル・バジルール（CV：桑島法子）
    • 生日：C.E.46年12月24日
    • 年齡：24歲（KIA）
    • 種族：自然人
    • 大天使號的副艦長；出身於軍人世家，嚴格遵守上級的命令以及規條，難以通融，常與瑪琉的立場相左。擁有極高的戰艦指揮能力及統率能力，若果不是直至歐普她一直在大天使號，它一早被擊沈了。

    • 和大天使號展開艦對艦砲擊時，由於芙雷反抗喪心病狂的阿茲萊爾，終於決心不再聽命他人而下令所有人棄艦，將阿茲萊爾與自己一同鎖在主天使號艦橋，自己也遭到阿茲萊爾槍擊受傷。

主天使號（LCAM-02XD Dominion）編輯

  • 娜達爾‧巴吉路／ナタル・バジルール（CV：桑島法子）
  • 穆達‧阿茲萊爾／ムルタ・アズラエル（CV：汚いシロー[4]檜山修之）
    • 年齡：30歲（KIA）
    • 種族：自然人
    • 自然人激進派系「藍波斯菊」的首腦，同時亦經營大手軍需產業，在大西洋聯邦中發言權極大，形同地球軍實質領導人。
    • 在回想中似乎在少年時期聯群結黨，卻被煌一個調整者輕易的扳倒全員，那調整者在他在同一學校讀書，不但受教授賞識還受很多女同學喜愛。穆達對於這種歧視作出投訴卻被像是家長的女性掌摑。
    • 在漫畫版確定為其母親，還對他說了「為什麼你不去成為調整者」的暴言
      • 「やめてよね…本気でケンカしたらムルタが僕に敵うはずないだろ…」
    • 激進組織「藍波斯菊」為嫉恨調整者的自然人組織，與調整者不共戴天。經常進行抹殺調整者為目的的恐怖行動。為了令自然人也能達到調整者的戰鬥力而秘密捕捉兒童進行強化人實驗，手段慘無人道。
    • 把戰爭比擬成商業競爭，同時完全採取非黑即白、非勝即負的策略管理戰場；因此全劇三小強除了波阿斯攻防戰以外，幾乎每次返航都會被予以斷藥懲罰。
    • 最初在地球戰艦上作為客席觀戰和作全局指揮，後來乘搭主天使號，跟蹤大天使號企圖奪取自由和正義鋼彈，以取得夢寐以求的黑科技——重新起用核武的方法。

      • 本人也被劾以破甲者插穿駕駛艙而亡。意想不到的是，兩年後他竟然以影印人的方式「復活」...

      • 蜜娜對歐普本國的默認態度示以感謝的同時，在宇宙中靜靜守望著卡佳里領導的新生歐普，被杜蘭朵議長評為「歐普的影之軍神」

    • 千葉大嘴巴的瑪莉蘇前傳
      • 千葉曾經表示劾是CE中「最強的角色」（無誤），但馬上被森田繁打回頭指「不可能比煌他們強」
