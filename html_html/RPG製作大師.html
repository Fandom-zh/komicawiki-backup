RPG製作大師

RPG製作大師（RPGツクール/RPG Maker）

目錄

解說編輯

  • RPG製作大師是日本ASCII公司製作的軟體系列，最初是於超任上推出，後來亦有推出適用於PC的版本。
    • 在ASCII公司於2002年退出遊戲市場後，系列的開發移轉至Enterbrain公司底下。
    • 因為單獨販賣（還要是平價）的遊戲引擎在推出當時實在很罕見，也有不少網友會直接歸類本系列為一款遊戲。
  • 系列的原名「RPGツクール」的「ツクール」是「ツール（工具）」和「作る（製作）」的合成字。
  • 由於使用容易，加上自訂高，而且使用這程式製作出來的RPG不僅可以脫離RPG製作大師的環境[1]，甚至還可以拿來販賣[2]，因此成了許多人製作遊戲的入門方式。
    • 所以實際說起來本作比較接近一款「附上許多輔助工具讓使用者方便運用的遊戲引擎」[3]
    • 順帶一提，大部分的商用遊戲引擎在授權費上都非常的可怕[4]，根據遊戲引擎不同，可能會需要付數百萬元美金不等的費用，而RPG製作大師雖然本身成像引擎不開源，但日文版不過一萬到兩萬日幣，中文版更只需要數百塊台幣即可購買自由使用，這也是本系列受到不少玩家喜愛的原因。
  • 雖然名字標記著RPG，不過稍加修改後，要做成AVG（特別是視覺小說型的AVG）、STG甚至RSLG都是可以的。
    • 模擬城市之類的SLG甚至是世紀帝國之類的RTS就不要太強人所難了
      • 這也是RPG製作大師與Unity3D等軟體有所差異的一點，RPG製作大師是在RPG的人物、技能以及操作介面等特化的遊戲引擎，拿去做跟RPG關係比較不大類型的遊戲就很容易卡住；但Unity3D、Unreal以及Shiva等新型的3D遊戲引擎中，編輯器基本上只負責創造地圖[5]，其他的遊戲邏輯、使用者介面仍要使用者自行編寫，因此除了寫RPG與STG之外也可以做其他類型的遊戲。
    • 在2017年同公司更推出了針對視覺小說型AVG特化的「ラノゲツクールMV」。
    • 此外該軟體還被諸多開發者用於製作未成年人不得遊玩的小黃油，Steam上DLsite上都能碰到不少。較為知名的例子就是露奈藥店。
  • 不僅是預設素材，網路上也有許多人製作了可以使用的素材，當然也可以自己製作。
    • 只是因為遊戲使用的素材是以點圖為主，如果沒有一定的功力其實做不出來，也造成了一定的難度。
      • 特別是預設的素材是以西式奇幻背景為主，和風系[6]、中國系與新大陸系甚至是科幻的作品要用RM製作的話就會先在素材上碰壁了。

RPG製作大師MV Trinity編輯

  • 2018年11月15日於PS4、NS發售，XBOX ONE版則預定明年冬季發售→中止發售，首次中文化

家用電腦系列編輯

RPGコンストラクションツール Dante編輯

  • 最早的作品，台灣沒有代理。

RPG製作大師2000編輯

  • 目前日台都仍為主流之一的版本，台灣由會宇多媒體（已倒閉）代理。
  • 畫質偏低，但是自訂性比看起來的還要高，雖然沒辦法做出隊伍中換人的功能，但是也有許多人製作了遊戲。
  • 角色屬性分為生命值、法力值、攻擊力、防禦力、敏捷力與精神力六個屬性。
  • 著名作品：屍體派對（Dante98版）

RPG製作大師2003編輯

  • 2000的修改版，遊戲資料庫與素材規格基本上沒有改變，但是戰鬥系統從原本的DQ式換成了FF式的橫向對戰模式。
  • 在台灣由光譜代理。
  • 值得一提的是HP和傷害的上限更改為9999，技能的屬性也容許作出吸收XX屬性的效果
  • 狀態異常的功能也從單純的異常狀態，變成強力BUFF/DEBUFF的程度[7]
  • 另外從日版就存在著精神力和防禦力的能力值提升技能的嚴重BUG[8]

RPG製作大師XP編輯

  • 2004年推出，台灣由英特衛代理。
  • 畫面細膩程度提升許多，解析度也提高到640X480，音樂部分亦支援MP3與OGG格式，不過比較不便的是資料庫和使用素材規格大改，很多2000/2003使用的素材都沒辦法直接無痛轉移到XP裡面，除此之外，事件指令中一些指令也被移除（但可以使用RGSS重現）。
    • 這個似乎也是很多日本玩家不願意轉移到XP的原因
  • 角色屬性分為生命值、法力值、力量、敏捷、速度、魔力、攻擊力、物理防禦和魔法防禦共9個屬性，但是後三者是只能透過裝備來增加。
  • 這個版本加上了「腳本」功能，可以藉由撰寫奠基於RUBY程式語言的RGSS來自訂功能。
    • RGSS語法和RUBY一模一樣（包括模組化、繼承等物件導向程式語言都有的撰寫方式），但是程式提供了許多RPG遊戲專用的模組[9] 讓撰寫功能變得比較方便
    • 不過因為是初創期，一些腳本的寫法並沒有考慮到擴充性，如果不加以考慮的話是蠻容易讓加進來的腳本互相衝突，導致程式出錯跳出。
  • 在台灣也是較多人使用的版本（主要是因為XP之後很長一段時間都沒有代理新版本......）
  • 因為使用RGSS可以動用到系統指令，不少防毒軟體會以為遊戲主程式是病毒而把程式給殺掉算是另一個不便之處。

RPG製作大師VX編輯

  • 2007年推出，台灣由光源媒體代理。
  • 畫面基本上同XP版的全彩規格，但解析度下修為544X416。
  • 遊戲資料庫與事件指令，還有素材規格等基本上全面回歸2000/2003規格，包括角色的臉圖顯示以及角色屬性。
  • 但地圖除了使用預設的五頁以外便無法新增，也造成使用上的不便，同時RGSS也因為修改成無法與XP相容的2版，使得XP製作的遊戲完全沒辦法移植。

RPG製作大師VX ACE編輯

  • 2012年推出的VX更新版，台灣代理商於2013年五月推出中文版
  • 畫面規格和素材規格都與VX相同，不過這個版本不是單純的更新版，因為遊戲資料庫和RGSS又做了一次修改。
    • 角色屬性修正為生命值、法力值、物理攻擊、魔法攻擊、物理防禦、魔法防禦、速度和幸運八個屬性
    • 技能和物品可以自訂攻擊公式（雖然還是有簡易設定，不過可以准許製作者直接輸入公式）
    • 2000/2003做不到或需要很麻煩步驟、XP與VX需要修改RGSS的系統在ACE中直接內建，例如更換隊員的系統。
    • 最大程度化整合角色、敵人、武器裝備和遊戲狀態的特性並統稱為特徵，有些原本需要修改RGSS才能追加的特性亦直接內建
      • 本作的武器防具設置上幾乎相同（差別在武器可設置動畫/防具可設置裝備部位），過去武器不能加防禦力/防具不能加攻擊力的設定在本作中已經取消。
      • 也因為上述更改，VX的RGSS在轉移時就會碰上問題。
    • VX令人詬病的地圖編輯圖塊太少也在本作修改成與先前相同的分地圖類型追加圖塊（素材規格則是與VX相同，因此可以直接把VX的地圖資料移植過去）
  • 雖然程式裡內鍵角色生成器，但人物頭像造型整體而言評價相當差或是又名怪物生成器。
  • 某個程度上算是RPG製作大師的里程碑。

RPG製作大師Unite編輯

ツクールシリーズ素材集 和編輯

  • 以VX規格製作的額外素材集，沒有代理進來
  • 因為RTP內部和風的素材佔少量，於是為了騙製作者的錢而推出
  • 付有範例遊戲『長一郎剣風伝』

RPGツクール音素材集 Vol.00編輯

  • 這次是音效和音樂的素材包

RPGツクールMV編輯

  • 官方在2015宣布的新版本，本作開始推出Mac版，另外製作出的遊戲不僅可以在PC與Mac上遊玩，甚至也可以產出iOS/Android與網頁版，能夠直接上架或者是上傳至網路上給眾人遊玩。
    • 英文版在2015年10月23日透過Steam販售，日文版則是在12月上市。
  • 因為VX ACE設計上已經夠完美了所以操作介面基本上與VX ACE差不多
    • 數據屬性上則是跟VX ACE幾乎一模一樣，所以也有人寫出把VX ACE的數據庫轉換成MV用數據庫的RGSS劇本給使用者用。
    • 技能/道具/武器/防具/敵人/敵群組最大數量由過去的1000提高為2000（其他仍為1000）
      • 另外防具過去皆限定為四種，本作開始可以增加數量（但能不能同時裝備兩個飾品之類的目前未知）
      • 同時過去複製不容易的問題也獲得了改善
    • 新增搜尋變數/開關與事件的介面，管理大型遊戲時面對的難題應該可以得到不少改善。
  • 由於必須跨平台，因此本作的腳本改採Java Script編寫也就是說舊有的Ruby腳本又要重寫了
    • 數據庫格式也從舊有的Marshel加密格式更改為容易讀取、可編輯的JSON格式
    • 本作的可執行檔架構採用node.js客戶端獨立運行框架nw.js，而Android/iOS版則是產出網頁版，再自行利用Phonegap/Cordova包裝成Android/iOS上架用檔案。
      • 不過當初官網講的太模糊，不少人發現沒辦法一鍵輸出之後都覺得官方在騙人。
      • 另外雖說宣傳只有提Android/iOS，但因為實際上是可包裝的網頁版，所以Win8/Win10也是可以包裝成APP的。
      • 搭配Phonegap/Cordova與Win8/Win10 UWP功能，甚至製作出課金道具也不是問題。
    • 另外本作開始導入腳本管理功能，可以自由開關腳本
      • 但是相對的官方拿掉了腳本編輯器，所以要編輯腳本必須自己使用記事本或其他Java Script程式碼編輯程式編寫。
      • 腳本的設計上也與過去不同，可以在開頭設計讓使用者自行輸入可使用的參數運用，減少直接修改腳本可能造成的錯誤問題。
  • 首次可以自由選擇要使用橫向或者是縱向的戰鬥介面
  • 解析度提升至816X624
    • 並且取消RTP設定，製作使用的素材會直接匯入至遊戲裡面。
    • 但是內建素材數量為歷代最多，從SF到和風都有，甚至還有200X的重製素材
      • 不過這也連帶發生匯入素材不便的麻煩。
    • 另外導入自動放大縮小功能，縮放視窗的同時也會重繪畫面讓畫面不至於過小或過大。
  • 過去的系列作都只能使用鍵盤操作，本作開始內建滑鼠與觸控操作。
    • 從上面的跨平台來看，過去必須自己寫腳本的輸入介面應該也可以直接輸入而不用另寫腳本了
