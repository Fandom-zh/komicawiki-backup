名稱編輯

 
  • 佐倉杏子（さくら きょうこ／Sakura Kyōko）
    • 主要角色中唯一全名是漢字。
  • 常見暱稱有「紅毛」、「あんこ（餡子）」、「槍妹妹」[1]等等。

基本資料編輯

  • CV：野中藍
  • 於動畫第四集才登場的隱藏角色，因此於各種宣傳海報上都鮮少出現。
  • 在巴麻美身亡後，因為盯上了見瀧原町這塊地盤，而從別的城市到來的魔法少女。
  • 以對自己的利益來當做魔法少女的行動方針，是個不顧周圍受害者的利己主義者。
  • 出場時總是在吃著什麼東西。

經歷編輯

 
  • 身世
    • 出生於見瀧原隔壁的風見野一個並不富裕的宗教家庭，與父母妹妹一家四口一起生活。
    • 雖然家人間相處和樂，但因為父親所推廣的優性人類生存說[2]新式教義並不被人接受，使得他們的家庭越來越貧寒，乃至三餐不繼。
    • 因為無法接受父親這樣受人否定，所以杏子為了支持父親的理想，而向QB許願，讓人們認真的去傾聽父親的話，而她也就這樣成為了魔法少女。
    • 因為這樣，杏子家的家計終於有了起色。她也相信只要自己在世界暗部討伐魔女、父親在世界表面推廣教義，這樣父女倆就能一同拯救這個世界。
    • (The different story補完)之後遇上麻美並成為麻美的徒弟，和麻美搭檔組合了好一段時間
    • 然而當杏子的父親知道人們是因為魔法才來聽自己的教義時，卻絕望的帶著妻子與小女兒自殺，留下了杏子一個人。
    • 受到這件事影響，杏子捨棄了過去的理念，化為了一個只為自己而戰的利己主義者。
    • (The different story補完)也因此有了心理創傷，無法再使用固有的幻惑魔法。
    • 而也是在這時候和麻美決裂，麻美打算挽留杏子，而和杏子展開對戰，最後由杏子勝出。
  • 第五集捏它注意
    • 來到見瀧原的杏子，遇見了一味追求正義的美樹沙耶香，兩人交談後因為理念不合而大打出手。在杏子以老練的實力壓制住沙耶香、打算給她最後一擊時，被突然出現的曉美焰用奇異的能力阻止。
    • 之後杏子在遊戲廳玩跳舞機時，焰前來拉攏，兩人為了對抗即將到來的魔女之夜而組成臨時同盟。杏子也向焰示好，將手上的Pocky分給她。
  • 第六集捏它注意
    • 在沙耶香默默的站在恭介家前聽他演奏時，杏子出現在她面前，嘲諷她的願望與做法，建議她去打斷恭介的手腳來徹底得到他。沙耶香因此而被激怒，杏子也約她到天橋上戰鬥。
    • 橋上的戰鬥因為焰和圓的亂入而被打斷，之後由於圓將沙耶香的靈魂寶石丟棄，而使魔法少女的真相顯露在杏子面前。得知真相的杏子憤怒的將QB抓起來斥罵，但在之後她還是接受了自己這樣的下場，認為一切是自己自作自受。
  • 第七集捏它注意
    • 杏子去找因為得知真相而失落不肯上學的沙耶香，將她約了出來。她帶沙耶香到昔日父親的教會，將自己的過去告訴她，希望她能看開，並與自己成為同伴。不過堅持自身理念的沙耶香，雖然承認自己以前誤會了杏子，但卻沒有領情。
    • 在沙耶香與影之魔女戰鬥時，杏子決定將魔女讓給她。但由於沙耶香陷入苦戰，所以她還是出手相助。然而已逐漸絕望的沙耶香拒絕了杏子的幫助，以捨身的戰法獨自打倒魔女，並因為不想欠杏子人情而將得到的悲嘆之種扔給杏子。
  • 第八集捏它注意
    • 杏子到焰的家，和她一同討論對於魔女之夜的戰法。這時QB出現告訴他們沙耶香因為沒有淨化靈核，情況逐漸惡化的事情。焰為此而試著去幫助沙耶香，在被拒絕之後，決定將沙耶香除掉，但被杏子及時阻止。之後杏子在車站找到獨自離開了的沙耶香，然而她卻在這時因過度的絕望而化為了魔女。
  • 第九集捏它注意
    • 杏子在焰的幫助下，帶著沙耶香的屍體脫離了魔女展開的結界。與圓相遇，並對這件事與焰發生爭執後，杏子將沙耶香的屍體暫時帶到旅館中保管，並持續以魔力維持屍體的鮮度。
    • 杏子在與QB對沙耶香的事情進行討論後，隔天找上本來正準備去學校的圓，邀請圓一同去試著拯救變成魔女的沙耶香。圓答應之後，主動向杏子示好，而杏子也微笑的向她遞出美味棒。
    • 杏子領著圓進入沙耶香的魔女結界，兩人終於見到沙耶香變成的魔女。杏子放出屏障保護圓，讓圓試著喚醒沙耶香，而自己則盡力的抵擋沙耶香的攻勢。最後杏子在單方面的防禦戰中受了重傷，而小圓挺身保護杏子後，雖然被杏子救下來，但還是暈了過去。
    • 這時趕到的焰接住暈過去的圓，杏子請她帶著圓離開，而自己則留下來對付魔女化的沙耶香。杏子就這樣用自己最後的力量，引發爆炸和魔女化的沙耶香同歸於盡。
  • 第十集捏它注意
    • 三周目時來到見瀧原，並與眾人一同迎擊人魚魔女，但隨後遭到目睹魔法少女的真相而崩潰的麻美擊碎靈核身亡。
  • 第十二集捏它注意
    • 在圓許下願望，讓魔女從這個世界上徹底消失之後，由於沙耶香變成魔女這件事已經不存在，所以杏子也得以生還。但她依然是魔法少女，以後也將為了自己願望的代價，繼續和世界上的絕望戰鬥下去。
  • 巴麻美的平凡日常
    • 已婚，育有一女由真。家境不寬裕需要在超市打工貼補家用
    • 靈魂寶石被小孩貼了貼紙撕不下來

能力編輯

 
  • 訂立契約的願望是希望人們能認真的傾聽父親的話
  • 以資歷與實力來說算是魔法少女中已相當熟練的老手。
    • 本身擁有不錯的中近距離戰鬥能力，戰鬥技巧也十分純熟，但相較之下防禦力就顯得薄弱。
  • 武器是正三角式槍頭的長槍，槍柄能化為多截棍[3]，以此來增加攻擊範圍或是綑綁目標
     
    • 漫畫版中槍頭為十文字槍，此外該槍是以變長成為鞭子而非變成多節棍的方式增加攻擊範圍。
  • 能製造出大量由方塊拼造成的網狀結界，通常用來將需要保護的對象隔離開來。
     
    • 漫畫版沒有出現這項能力。
    • 黃昏Frontier製作的「Griefsyndrome」遊戲中，此能力是杏子的獨有魔法，但是功用只能阻擋小怪跟彈幕，相對另外四名魔法少女而言較不好用，幸好杏子這角色其他能力都算中上
  • 根據虛淵玄所說，杏子許願成為魔法少女時，獲得了幻惑的能力，然而因為家人被自己的願望害死，於是從此不再使用那個能力。
    • 在廣播劇3中，這個能力體現為分身，最多能分身到十三人。
      • ロッソ・ファンタズマ！（麻美學姊親自命名）[4]
  • 職階是Lancer，所以幸運是E
    • 所以繼承了槍兵自殺的傳統
       
    • 在PSP遊戲中的最強殺招是自爆（汗），幸好就算自爆了也還救得回來（默）。

其他編輯

  • 失去家人的杏子似乎是以非法手段獲取物資來維持生活，例如利用魔法偷竊或是破壞ATM盜取現金等[6]。
  • 根據廣播劇和小說版，杏子的妹妹叫做佐倉モモ（小桃）
  • 在官方廣播劇3中，杏子在一年前作為魔法少女的新手，曾經成為巴麻美的弟子，和她一同行動，討伐魔女。但之後由於家人死亡的那件事，理念改變的杏子也與麻美決裂。
    • 然而比較矛盾的是，在動畫第七集杏子的回憶中，能看出當時的她比現在嬌小許多，實在不像是一年前的事情。[7]
      • 不過由虛淵監修的PSP版則是採用了廣播劇3的設定，所以這應該已經被承認是官方設定。也有可能是當時劇本根本沒設定，才會讓動畫的製作群以及小說版作者搞錯而把杏子畫/寫得年紀太小
 
魔法少女おりこ☆マギカ
  • 在外傳漫畫《魔法少女おりこ☆マギカ》中，杏子遇見了父母被魔女殺害的受虐少女千歲由麻。在無可奈何的狀況下開始照料她的生活，因此而被稱為帶子狼。
  • 在第五話副音軌中，小圓的聲優悠木碧做出了「杏子たんは俺の嫁」發言，也因此得到「杏子が好きすぎる人」之稱號。悠木碧又喜歡美少女戰士的木野真琴，所以是馬尾控。愛的宣言已被MAD化
    • 就像那些藍愛[8]與悠木勇氣[9]勝利的故事[10]（詳見連結）
  • 本篇的演出與喜怒形於色的表現提供MAD職人大量的素材，贏得「萬能的杏子」之稱。
  • 2011年秋季日本「導盲犬普及支援協會」將杏子選為宣傳海報的看板娘，希望能徵招更多的志工來參予培育導盲犬的工作。
