勇者鬥惡龍XI

勇者鬥惡龍XI 尋覓逝去的時光 / ドラゴンクエストXI 過ぎ去りし時を求めて / Dragon Quest XI: Echoes of an Elusive Age
這是一個編輯中的詞條，歡迎有愛者加以補完以及補圖
以下含有部分的劇情，沒玩過原著又不想被霸王斬+大十字=大星雲轟成渣滓的話請不要繼續看下去。

目錄

解說編輯

  • 由Square Enix開發的勇者鬥惡龍系列三十周年紀念作品，發售於PS4與3DS平台，為系列作中首次有官方中文版的本篇作品
    • 3DS版可以切換2D / 3D模式，主要會影響其遭遇戰的發生模式與移動模式，2D模式為地雷遇敵與十字鍵移動，3D模式則是明雷遇敵與類比墊移動
    • PS4版則是使用類似十代的3D操作系統
  • 於2019年推出Switch版《勇者鬥惡龍XI S》，追加了全語音對話與隱藏頭目
    • 分為日本版和海外版，海外版名為《勇者鬥惡龍XI S - Definitive Edition》，有多國語言字幕但就沒有日文字幕
    • 而日本版雖然要豪華版或追加下載才有英語字幕，但有廣播劇和特典服裝，亦有連主機版本，讓很多玩家不得不買日本版的元兇
  • 破關後才是遊戲的真正開始（無誤
    • 本篇的遺撼、悲劇都需要在破關後才能解開，也只有破關後再開始才符合遊戲的副題。
  • Komica wiki上，勇者鬥惡龍系列中第一個有專屬條目的作品

系統編輯

章節編輯

  • 本作採用了類似四代的章節制推動劇情，章節的推動會影響可攻略地域以及支線任務
  • NS的2D / 3D切換也並非無接縫式切換，而是選擇過去已完成的章節切換
    • 另外寶箱會重生，所以東西可以重複獲取
      • 而也因此導致本作的時光倒回道具複製技巧可以複製的範圍變得更大，除了冒險之書世界獲得的裝備和勇者之劍系列外都可以複製

任務編輯

  • 延續九代的系統，本作在攻略期間可以從NPC處接受任務
  • 主要有道具交付、使用特定聯手技或是討伐轉生怪物等類型

技能盤編輯

  • 角色在等級提升後可以獲得技能點數，藉由消耗技能點數可以開啟技能盤上的技能
  • 在儲存點可以花錢將技能還原以回復消耗的點數，作為點錯技能的補救措施

定神&連攜技編輯

  • 藉由持續戰鬥可以使角色進入定神狀態，當特定角色進入定神狀態時可以發動連攜技
  • 與一般技能不同，連攜技除了特定技能外不須消耗MP，發動之後會結束定神狀態
  • 不過若在其中一位發動角色陷入無法行動的狀態時會無法發動連攜技

神奇鍛造編輯

  • 獲得鍛造配方後，玩家可以使用神奇鍛造套組製作道具
  • 製作道具時會消耗素材，素材不足時可以花錢購買（限商店有販賣的素材）
  • 開始鍛造後須消耗專注力敲打或是使用技能，鍛造成功後可以獲得重鑄寶珠強化已鑄造完成的裝備
    • 鍛造的+值只會上升不會下降，所以寶珠夠多的話可以亂打一通的弄上去

賽馬編輯

  • 於薩瑪迪王國舉辦的一系列賽馬活動
  • 難度由簡單至困難依序為青銅、白銀與黃金三項盃賽，勝出或是挑戰滿三次後可以獲得稀有道具
  • 終盤在通過尼爾森的考驗之後會新增最困難的盃賽「黑色盃」，該項盃賽除了視野極差之外，施維亞與葛雷格兩人還會做為對手出場

賭場編輯

  • 位在索爾提科鎮飯店的地下
  • 主要項目有紙牌、拉霸與魔霸
  • 古洛塔鎮在遊戲中盤也會設置賭場，除了前述三項遊戲之外還有新增一項輪盤
    • 一如之前的賭場，想要穩賺就是拉霸，只要拉的次數夠多就一定贏，所以有玩家用不需緊按的連打手掣來賺金幣。

連武討魔行編輯

  • 設置於度魯達鄉的挑戰模式，依據完成的回合數可以獲得不同道具

弩力冒險（PS4限定）編輯

  • 薩瑪迪王國的事件結束後，玩家可以獲得弩，並從弩大師處接受弩力冒險的任務
  • 弩除了可以吸引怪物之外，各地皆設有箭靶供玩家尋找
  • 將所有箭靶擊破後，可以向營火附近的弩大師徒弟領取獎賞
  • NS版被刪除

約奇族（3DS和NS限定）編輯

  • 類似九代的擦身通信藏寶圖
  • 可以開啟位在約奇族村落的時渡迷宮，可以攻略過去作品的迷宮
  • NS版改成尋找在世界各地出沒的約奇族以開啟迷宮

登場角色編輯

勇者與其同伴編輯

  • 勇者
    • 名稱自定義，宣傳影片的名字為十一哥伊雷文（イレブン），而五代作為廢案名的唐努拉（トンヌラ）在本作依然是廢案名
    • 因為中文化了，所以會出現一堆奇怪的名字，比如我幫你生孩子，義彥，你他媽的...等
    • CV：齋賀光希
      • 戰鬥時會有喊聲外，兒童期也會有台詞登場
    • 特南克斯
    • 本作主角，生活在伊希村的少年
    • 在迎接成年禮時得知自身勇者的身分並展開冒險
    • 真實身分是尤格諾爾王子，王國在他出生之際遭到兀爾諾加所滅
    • 職業類型為勇者，後期可學習專屬對全體技能霸王斬
      • 終盤獲得勇者之劍後還能習得好幾個對全體技能，有一半是神龍之謎的阿邦劍殺法
    • 歷代勇者威能最不自重
      • 一直以來DQ系列的主角都沒什麼威能，有的也沒什麼機會使用，但這勇者之力卻三不五時就開一次，變成過關道具，和前輩們差很遠
      • 不過考量到這次整體對勇者不算友善的緣故，這點也算是能接受的
    • 繼一代與四代之後另一個一登場就是勇者的主角
  • 卡繆（カミュ）
    • CV：內山昂輝
    • 林克/索尼克
    • 被關押在迪爾卡達王國地下牢房的青年，按照預言者的預言追隨勇者行動
    • 過去曾與妹妹在克雷莫蘭周圍定居
      • 妹妹因為詛咒項鍊而變成黃金後，為了拯救她而按照預言者的預言尋找勇者
    • 在生命大樹一戰後一度被魔王軍所擒，在脫獄期間為了活下去而捨棄自己的記憶和預言者交換力量
      • 後在回到過去生活的克雷莫蘭王國時回復記憶
    • 職業類型為盜賊，不過比起偷竊更適合活用短劍技能的二刀流，或是施放魔法陣給予延遲攻擊
      • 習得會心必中後更是成為我方物攻輸出和殺金屬史萊姆的頭號人選
    • 官方欽定的帥哥，走到各地都會吸引女性路人關注，前往徽女時還會被學生遞情書…
      • 因為髮型和臉型相似還有個妹妹的關係，經常被玩DQ6的主角的梗，甚至官方自己也在六代的時渡迷宮玩這個梗
    • 勇者的官配
  • 薇若妮卡（ベロニカ）
    • CV：內田真禮
    • 阿什莉/惠惠
    • 雪妮雅的姐姐，來自聖地拉姆達
    • 因為遭到魔物襲擊而成為幼兒體型，本人倒是當成是變年輕了
    • 性格極為強勢，基本上到一個新地區都會與當地人發生爭執然後觸發主線任務
    • 在生命大樹一戰為了保護主角一行人而冒險發動傳送魔法卻因此犧牲性命，在主角回溯時間後得以生還
    • 是DQ史上第一個明確的以死亡離隊的角色
    • 職業類型為魔法師，體力極低卻能夠施展多樣攻擊咒語
      • 和某大胸部暴力女一樣會雙龍打，肯去刷力量種子的話也可以打打物理
  • 雪妮雅（セーニャ）
    • CV：雨宮天
    • 薩爾達公主/阿克婭
    • 薇若妮卡的妹妹，來自聖地拉姆達
    • 性格溫柔天然，與姐姐相反
    • 職業類型為阿克西斯教神官，隊伍主力回復擔當，另外可以施展巴基系的風屬性攻擊咒語，可惜是純僧侶傷害不太好看
      • 或者可以使用短杖之中傷害以回復魔力來看的神聖之光來攻擊
      • 薇若妮卡死後會繼承她的咒文和技能盤
  • 施維亞（シルビア）
    • CV：小野坂昌也
    • 李龍/Smile 光之美少女的小丑
    • 遊走各地的男大姐旅行藝人
      • 真實身分是索爾提科領主的兒子歌利亞
      • 因為是名門騎士之後的緣故，對騎士道可說是相當精通，終盤通過父親的試驗後更能習得保護隊友的聖騎士技能，專用技女士優先更是可以讓各個女角能夠進行二次動作
    • 在生命大樹一戰後為了拯救群眾而組團展開巡迴演出，甚至還把團員都掰彎了
    • 職業類型為超級明星，擅長賦予多種異常狀態的技能，同時解除棘手的狀態異常的各種回復技也很充足，在物理隊中甚至可以用加攻魔法幫忙輸出
      • 擁有男性陣最高的魅力，由於部分技能看魅力決定成功率和傷害，全力堆效果不錯
  • 瑪蒂娜（マルティナ）
    • CV：小清水亞美
    • 戀姬無雙的關羽
    • 與羅結伴行動的美女武鬥家
    • 真實身分是迪爾卡達的公主，開場動畫抱著嬰兒時期主角逃難的小女孩就是她
    • 在生命大樹一戰後，為了解放古洛塔鎮而隻身潛入敵營，卻被妖魔軍王布基以人質要脅施予詛咒而變成魔物，被趕來的主角擊敗後復原
    • 職業類型為武鬥家，無法使用魔法卻能施展多種物攻技能
      • 雖然不會魔法，但擁有女性陣最高的魅力，由於部分技能看魅力決定成功率和傷害，全力堆效果不錯
    • DQ系列新一代本子王，拜某位軍王所賜幾乎都是兔女郎洗腦本居多
  • 羅（ロウ）
    • CV：麥人
    • 龜仙人
    • 與瑪蒂娜結伴行動的神秘老人
    • 真實身分是尤格諾爾前代國王，主角的外祖父
      • 雖然是以賢能著稱的明君，不過也有喜歡收集色情書刊的色老頭紳士一面…
      • 在克雷莫蘭魔女的事件結束時還因為不小心把收集的A書洩漏出來而被同伴白眼…
    • 過去曾在度魯達鄉修行，在生命大樹一戰後為了修行而自行墜入冥界，在接受尼瑪大師指導後習得專屬技能大十字
    • 職業類型為賢者副修武鬥家，黑暗系攻擊咒文與回復系咒文都可以學習，很早期就能學會復活咒與持續補血的治癒之雨，也擁有爪技能可以打打物理
    • 另外也持有雪妮雅並未持有的回覆魔力增加技能

魔王軍編輯

  • 魔王兀爾諾加（魔王ウルノーガ）
    • CV：黑田崇矢
    • 侵略羅德賽塔西亞大陸的魔王
    • 真實身分是勇者羅許的同伴烏拉諾斯，因為渴望邪神的力量而殺死勇者並吸收邪神之力
    • 攻擊主角的祖國尤格諾爾之際操控了迪爾卡達王並宣稱勇者為惡魔之子，趁主角在生命大樹被荷梅洛斯擊敗時出手搶奪勇者之劍使其魔化成魔王之劍
      • 獲得魔王之劍後為避免邪神阻礙而將其軀體斬碎
    • 時間回溯後因為荷梅洛斯戰敗的緣故而將其處決並打算親自下手，最後卻因為邪神的阻撓而失敗，在臨終前意識到邪神的存在後嘲諷主角並死去
    • 在天空魔城與其決戰時會有兩階段，第一階段主要會使用所有六軍王的寶珠技能搭配大劍技能戰鬥
      • 第二階段會與魔王之劍變化而成的魔龍交錯出擊，由於魔龍除了會封印我方行動之外還會使用復活咒復活魔王，建議優先擊破
    • 時間回溯後會在迪爾卡達王城現出真身與勇者交戰，主要會召喚魔杖施放群體咒文
      • 除此之外還會使用凍結波動消除我方強化或是發動冥界之霧反轉我方回復能力，治癒之雨等持續回復技反而會成為死亡倒數的催命符…

    • 過去討伐邪神的勇者
  • 雪妮卡（セニカ）
    • CV：桑島法子
    • 與勇者一同討伐邪神的賢者
    • 羅許死後為了與他再次見面而前往被遺忘之塔，因為力量不足以破壞寶珠而失去記憶成為時光看守
  • 烏拉諾斯（ウラノス）
    • CV：關智一
    • 與勇者一同討伐邪神的魔法師
    • 被邪神力量吸引的他，為奪取邪神之力而殺死了勇者羅許並化身成魔王兀爾諾加，殘存的良知為了阻止魔化的自己而化為預言者等待新生的勇者
  • 尼爾森（ネルセン）
    • CV：三宅健太
    • 與勇者一同討伐邪神的戰士
    • 在勇者死後建立邦德爾馮王國，後世尊其為英雄王
    • 為了給予後世勇者考驗，在自己王國地下設立了迷宮，在勇者完成迷宮並戰勝恐懼後會給他許願的機會
      • 紳士程度和自己的後裔有得比，許願時還能跟他拿A書…

其他編輯

  • 愛麗絲（アリス）
    • CV：立木文彥
    • 施維亞專屬船隻「施維亞號」的航海士
    • 過去是邦德爾馮王國的海軍將軍，在王國滅亡陷入絕望之際與施維亞結識，從此與他形影不離
  • 預言者
    • CV：富永美衣奈
    • 擁有好幾種樣貌的神秘預言者
    • 真實身分是魔法師烏拉諾斯魔化成為魔王後殘存的良知，對於殺死同伴一事感到懊悔
  • 邪神尼茲傑爾法（邪神ニズゼルファ）
    • CV：宮田幸季
    • 過去被勇者羅許一行人封印的邪神，勇者之星為其身軀
    • 在魔王兀爾諾加降臨之際打算將自己過去被封印的軀體喚回地上卻在途中被兀爾諾加阻撓
    • 後在主角回溯時間之際與他一起回到過去，並趁兀爾諾加被討伐之際將自己的軀殼喚醒並化為黑色太陽
    • 戰鬥之際會將本體包覆在闇之披風中，使用勇者之劍可將其破解
