九十九遊馬

九十九遊馬/龍蝦
本條目歡迎有興趣的朋友加以補完。
以下含有部分的劇情，沒看過原著又不想跟他一飛衝天的話請不要繼續看下去。

解說編輯

出處編輯

  • 《海產王遊戯王ZEXAL》
  • 男主角

基本資料編輯

  • CV：畠中祐
  • 初登場時為13歲，歷代主角中年齡最小。[1]
    • 歷代主角中起始實力最弱，連規則也不懂，但是潛力最大而且心理出乎意料地堅強。
    • 意外地秒殺小角色的平均速度是歷代主角中最快。
      • 遊戲約要兩話，十代約要一話半，遊星只要一話內（有些會分開兩話播放），遊馬至今只需半話
    • 同時在黑暗決鬥和正式比賽中，卻未嘗一敗。
      • 不過在WDC結束後，第73話與快斗的決鬥中落敗。
  • 與傳說中的奶奶—九十九春、姐姐—九十九明里居住在未來城市「心之地」。
  • 父親-九十九一馬、母親-九十九初音未來在探險中失蹤，第49話父親卻出現在皇之鍵內
  • 故事開始時讀中學一年級，接過父親交代的皇之鍵，在與宿敵神代凌牙決鬥時喚醒了皇之鍵內的阿斯特拉爾。
    • 初期決鬥往往依靠阿斯特拉爾的指示，甚少憑自己取勝。
      • 在決鬥中有時會吃飯糰重燃鬥志到底這傢伙是否認真決鬥？
  • 有痔瘡[2]
  • 阿斯特拉爾在和千尊在古代戰鬥後所失去的半身

共同身分編輯

  • 決鬥王4號
    • 幾經70多回的考驗，終於在世界決鬥狂歡節奪得冠軍，但知名度仍不及鯊魚、快斗和Ⅳ
  • 敗北王
    • 敗北紀錄在於ZEXAL第44話的第10次敗陣，成功超越喪家犬城之內的9次敗陣，成為《遊☆戯☆王》系列內敗陣最多的角色（不計被No.96附身有11次敗蹟，要計上劇情外的話會多到數不清）
  • 龍蝦
  • 「No.」獵人
  • 遊戲王系列的頭號作弊王
    • 主要是因為當眾將手上的牌變成另一張牌，比十代的和遊星的印卡還要過份
    • 不過比起蹦太和別人交換心靈，還差得遠就是了
    • 之後的千尊更是過份，空場時可以改寫對方發動和召喚的卡
    • 後輩的遊矢也不惶多讓，直接改規則創造新的召喚法
    • 到了遊我甚至直接自創新的決鬥規則...
    • 而且牌組中有兩張當時仍是限制卡的《死者甦醒》（無誤）
  • 在M78宇宙中自稱片桐大介的弟子

性格編輯

  • 義且勇
  • 熱血
  • 信賴同伴
  • 笨蛋[3]
  • 死小孩
    • 事實上這主要還是因為遊馬初期對阿斯特拉爾的態度帶來的錯覺
      • 畢竟阿斯特拉爾莫名其妙就附身在自己身上，還被強制要求要幫他蒐集No.卡。阿斯特拉爾還會在決鬥中糾正遊馬錯誤，而觀棋不語真君子，導致遊馬在初期會對阿斯特拉爾感到反感也是很正常的
        • 這點在事後真的體會到阿斯特拉爾的性命和決鬥勝負相連後才改變態度
    • 實際上遊馬還挺為他人著想的
      • 最好的例子是19話《約束のかっとビング》的教學參觀，遊馬發現姊姊跟奶奶當天都要參加同學會，就決定隱瞞教學參觀讓兩人能好好去做自己的事
  • 遲鈍
  • 口吃（我我我魔術師、我我我女孩）
  • 影帝（受害者no.96）
  • 精神強韌
    • 從最初的DM到現在為止，所有主角中唯一沒黑化的（黑化的反而是背後靈的阿斯特拉爾[4]）
      • 闇遊戲在漫畫連載初期受到一同被封印在千年積木裡的黑暗大祭司（阿克納帝）的靈魂影響而黑化，不過隨著劇情的發展，這部分影響已經漸漸淡化，在遇到夏迪時黑暗大祭司對闇遊戲性格上的影響幾乎已經完全消失；在動畫原創的杜馬篇中，闇遊戲在對上拉斐爾時，曾一度因為受到「奧雷卡爾克斯的結界」影響而黑化，不過決鬥結束後就恢復正常了（不過因此付出了「表遊戲的靈魂成為召喚奧雷卡爾克斯之神的祭品」的代價）
      • 十代則是受到同伴消失加上丸藤翔對他的不諒解的打擊下黑化成「霸王十代」；
      • 至於遊星則是在漫畫版中受到《閃珖龍星塵》的力量影響而黑化，不過只黑化一兩頁就恢復正常了；
      • 遊矢則是被在內心深處的扎克的意識影響而黑化
      • 遊作則是本來就有點黑了
    • 強韌的精神根本不像是13歲的孩子，甚至在日本得到了「遊馬先生」[5]的稱號
      • 第一季時雖然看到因為玉座的復仇而消亡的Ⅲ、Ⅳ、V……等人，但依舊原諒玉座
      • 第二季遭到真月的背叛，因為受到小鳥的鼓勵而沒有失去相信他人的心，依舊沒黑化
      • 之後阿斯特拉爾和No.96殉情一同死亡後，雖然中途無法振作，甚至去Cosplay普拉希多，但仍決定要振作
      • 面對為了自己犧牲的Ⅲ、Ⅳ、V、亞里特、吉拉格……等人，以及快斗的死亡，雖然哭了很久，甚至讓觀眾覺得站不起來也很正常時，不但整理好情緒振作起來，還說出「這場戰鬥結束前，我不會再哭了。這就是我的勇往翔前！」這種話來
        • BGM：折れないハート
      • 在貝庫塔即將被千尊吸收之際，還是奮不顧身地跑去拉住貝庫塔，即使對方在這個時候再度背叛卻仍舊相信著他，精神強韌度和寬容度已經達到聖人的等級了……

愛好編輯

  • 一飛沖天/勇往翔前（東森幼幼台譯）/かっとビング
    • 遊馬精神強韌的象徵，也是遊馬從頭貫徹到尾的精神。
      • 意思是「勇敢的踏出一步，無論多危急都不能放棄，無論多困難的事物都要挑戰」。
    • 雖然這句話從一開始因為劇情風格轉變的原因變成笑柄，但隨著遊馬的表現和成長，這句話變成了ZEXAL的代名詞。
      • 序盤視聴者「かっとビングｗｗｗ」
      • 中盤視聴者「かっとビング…」
      • 終盤視聴者「かっとビング！」
  • 飯糰
    • 在決鬥中吃過飯糰後，鬥志提升至頂點，成為勝利Flag（無誤）
  • 魔法卡《雙倍機會》
    • 效果是當自己怪獸的攻擊被無效時，攻擊力倍增後再攻擊一次，經常配合霍普的效果
    • 發動次數之多連敵人都吐槽又是這招
    • 根據觀月小鳥所述，雙倍機會代表了遊馬的人生觀：遇到挫折不會放棄，再接再勵

能力編輯

  • 跳躍（可以跳過數十層）
    • 不過其實本作中遊馬的同學的體力都不正常。
      • 比較不普通的小貓能憑手指就把雕像抓壞；
      • 普通人的鐵男、等等力孝都能和巨大機器人互打；
      • 比較弱的小鳥也是拿著武器[6]就能和一整群機器人互打還占優勢
  • 跟阿斯特拉爾合體成超級遊馬人ZEXAL
  • 印卡
    • 阿斯特拉爾出現後，遊馬的額外牌組就出現了《No.39 希望皇霍普》（首次印卡）
    • ZEXAL形態時的閃光抽牌（跟遊星同樣光明正大），主要是印ZW有關的卡
    • 同時也能夠使卡牌從額外牌組內消失
      • 當阿斯特拉爾躲回皇之鍵內，「No.」卡片便從額外牌組內消失
    • 亦可當眾改卡（《限制巴利安之力》變成真正形態時）
    • 但在121話前都是靠阿斯特拉爾的力量才印出卡來的，第121話終於自己印卡了（雖然依舊是閃光抽《RDM－源數降落》）
  • 心理治療

牌組編輯

  • 由父親交托，起初沒有任何超量怪獸，直至遊馬開始收集「No.」卡
    • 遊馬第一隻超量怪就是《No.39 希望皇霍普》
  • 以超量怪獸作為王牌（融合、同步怪物淚目）
  • 超量怪獸中以「No.」卡為強
    • 最常用的「No.」是《No.39 希望皇霍普》；其他「No.」只在遊馬決鬥中最多登場兩次，多數被遊馬入手後便無戲份
      • 39這數字之意義有多種推測
      • 一是源自怪獸名Hope
        HOPE → EPOH → 39OH → 39王
      • 二是源自於遊馬初期狀態：未熟（不成熟）
        39 → 三十九 → み　じゅうきゅう → 未熟（みじゅ）
    • 另一超量怪獸王牌是【けいおん！魔人】系列，針對超量怪獸
    • 在和葛許的戰鬥中用了新的超量怪獸《我我我槍手》
      • 雖然名字寫是槍手但攻擊方式和葉問沒什麼兩樣……
  • 基礎牌組主要為重複字的怪獸（《隆隆隆石人》《熱辣辣弓箭手》等)，魔法和陷阱基本就是初期卡組
    • 擁有魔法卡《死者甦醒》（疑似在牌組放入了兩張，大約是因為動畫沒有禁卡表的關係）
  • 戰術級數較低，主要召喚同星數怪獸作超量召喚。另外很常使用無效攻擊+《雙倍機會》（攻擊被無效時可以再攻擊一次，並且攻擊力變2倍）的戰術
  • 常以《我我我魔術師》與《我我我女孩》進行超量召喚素材
  • 因不想怪物進入墓地而受苦，會阻止怪物留在墓地，與遊星的戰術相反（但後期已將這概念忘得一乾二淨）
    • 幸好多加了數張墓地能使用效果的怪獸，不然貝庫塔的牌組破壞就能完全擊破他
  • 在第144話和阿斯特拉爾進行最終決鬥時，使用的牌組依舊是以「我我我」「隆隆隆」等疊字怪為主，但額外牌組沒了「No.」

經歷編輯

  • 誕生
    • 被一馬發現，並寄宿在他身上轉生成遊馬
  • 回憶
    • 小時候跟隨父親探險，立志成為最強的決鬥者
  • 序篇
    • 為幫助被鯊魚——神代凌牙奪去牌組的朋友 武田鐵男，與鯊魚決鬥
      • 在決鬥時皇之鍵內的靈魂——阿斯特拉爾出現，並提供協助
    • 在阿斯特拉爾的協助下多次取勝，並奪得散失「No.」卡片
    • 在第一次與天城快斗決鬥，險些因戰敗而被奪取靈魂
    • 其後到決鬥庵進行修練，重拾信心
    • 第二次與快斗決鬥中打成平手
      • 在阿斯特拉爾與快斗決鬥時中途加入，並與阿斯特拉爾融合疊放超量成ZEXAL
  • 世界決鬥狂歡節篇・預賽篇
    • 參加世界決鬥狂歡節，在心城內收集心之碎片以獲取決賽資格，同時繼續收集「No.」卡片
    • 期間No.96再度發難，在霍普的幫助下險勝葛許和特洛瓦
    • 得知快斗的過去，為幫助天城兄弟重逢，與快斗組隊對Ⅲ、IV
    • 與Ⅲ對決後，獲得最後的心之碎片，成功晉身決賽
  • 世界決鬥狂歡節・決賽篇
    • 決賽剛開始後被玉座派來的三個刺客圍攻，不過在葛許的亂入幫助和後宮神月安娜的犧牲下成功進入地下區
    • 之後不看路亂開無意的把過山車行駛到熱帶雨林場地然後觀看了特洛瓦和玉座的決鬥
    • 遇見正在尋找快斗的軌道7，在軌道7的帶路下找到了正在宇宙場地和V決鬥的快斗
      • 這時V把遊馬的父親九十九一馬、快斗的父親「Dr.菲卡」以及V、Ⅳ和Ⅲ的父親玉座以前的關係與及發生了的事告訴遊馬和快斗
      • 在快斗離開後，和阿斯特拉爾討論了遊馬的父親把阿斯特拉爾送過來這個世界的原因，並得到了其原因很有可能是為了幫助遊馬阻止Dr.菲卡和玉座的結論
    • 抵達了熔岩海洋場地後觀看了Ⅳ和神代凌牙的顏藝秀決鬥，從Ⅳ口中得知了更多內幕
      • 多次勸凌牙別被復仇心所淹沒，但是卻因為玉座的干涉而失敗了
    • 在危險峽谷場地和葛許決鬥
      • 因為之前看過了三場十分殘酷的決鬥所以採取守備戰術，讓葛許對遊馬十分失望甚至對他破口大罵
      • 在恢復了原來的自己後，因為軌道偷工減料已到盡頭所以和葛許一同掉落到地上
    • 在之後的王牌對戰中勝利，得到葛許的王牌怪獸《英豪勇士EX咖喱棒王者之劍》
    • 得知Dr.菲卡的目的是要統治世界
    • 準決賽時和被玉座慫恿的凌牙對戰
      • 得知凌牙被操縱的原因是《CNo.32 海咬龍 鯊龍-白煞》後，利用陷阱卡把海咬龍的操縱權轉移為自己，可是卻受到了玉座力量的侵蝕（海咬龍附有玉座的紋章力量）
      • 不過凌牙自願認輸並使用《深海雪原封印》[7]把海咬龍除外，所以恢復了正常
    • 在快斗和玉座决鬥時跳上了他們決鬥的平臺上觀戰，幫助陽斗解開了和玉座的連結以便能夠讓快斗專心決鬥
    • 決賽跟玉座在Dr.菲卡的球型場地進行決賽，最後使用《ZW-獸王獅子武裝》的效果以再度攻擊戰勝玉座
      • 王座在被吸入黑洞前釋放所有被他囚禁的靈魂
    • 被阿斯特拉爾推離球型場地後與快斗、鯊魚跟Dr.菲卡（後為巴利安[8]）決戰
      • 擊敗巴利安後被突然出現的玉座送出心之塔
    • 依照願望與快斗進行最後的決鬥
      • 中途快斗投降，但因遊馬的說話快斗繼續決鬥，Ex咖哩棒交給遊馬後終於登場
      • 最後被快斗擊敗，但決定要終有一天要打倒快斗
  • ZEXALⅡ
