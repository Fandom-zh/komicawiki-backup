賽爾號

賽爾號/Seer/小破船

目錄

解說編輯

  • 由上海淘米網絡科技有限公司開發運營的子供向抄襲網頁遊戲，於2009年6月12日起在中國發行，台灣服已停服。
    • 整體遊戲基本上抄借鑒自神奇寶貝的捕捉養成進化和戰鬥的概念，是中國在古早的抄襲遊戲時代的其中一個代表作之一。更早的代表為出自同公司的企鵝俱樂部摩爾莊園。
      • 然而這款抄襲作因為入手門檻低加上中國政策因素，還是吸引了不少中國玩家海外沒人理
        • 推出時適逢網路開始於台灣國小生間普及的時代，因此與摩爾莊園一度相當有人氣。
      • 但在營運各種無限榨取玩家錢包商法和無能營運之下，此遊戲也走入了衰退之路。
    • 後炮製借鑒大法，於2011年發行同類型新作賽爾號2，述說賽爾號飛船偶然被吸引到阿卡迪亞星上，與當地住民、精靈之間所發生的曲折離奇的故事，後為了推出課金精靈並迴避舊營運的良心宣稱改名為「約瑟傳說」然後比本家先倒了，死因是走回1代老路無限榨取玩家錢包商法和無能營運。

精靈編輯

  • 此處只聊比較耳熟能詳的部分給予各位有印象的玩家髒自己手公開處刑的機會
點擊展開

初始精靈編輯

點擊展開
  • 布布種子系列
    • 屬性：草->暗影·草（暗黑）
    • 基本形象與妙蛙種子系列相似，飛葉風暴、地震、寄生種子等樣樣齊全
    • 初期進化鏈根本看不出是雌性
    • 進化鏈：布布種子->布布草->布布花
      • 再進化：布布花-（融合）->黃金布布->蒙娜布布->麗莎布布->蒙娜麗莎->生靈之主·蒙娜麗莎
      • 亞種（暗黑）：暗黑黃金布布->暗黑蒙娜麗莎
  • 小火猴系列
    • 屬性：火->戰鬥·火（暗黑）
    • 基本形象為隔壁老祖DP的小火猴
    • DP的小火猴得讓他叫兒子，一個06一個09年
    • 進化鏈：小火猴->火焰猴->烈焰猩猩突然切換物種
      • 再進化：烈焰猩猩-（融合）->炎火猴->熔火靈猴->魔焰猩猩->熾焰金剛->烈焰之主·熾焰金剛
      • 亞種（暗黑）：暗黑小火猴->暗黑熾炎金剛
  • 伊優系列
    • 屬性：水->超能·水（暗黑）
    • 唯一角色形象上跟隔壁老祖水屬性御三家對不上的，但是從技能組看來有他們的影子（高壓水槍、泡沫光線、水流噴射）
    • 擁有對戰利器「克制」，效果為「以翻倍傷害反彈對方攻擊」
    • 進化鏈：伊優->尤里安->巴魯斯
      • 再進化：巴魯斯-（融合）->伊優達->利爪尤里安->魯斯王->利爪魯斯王->利刃之主·魯斯王
      • 亞種（暗黑）：暗黑伊優達->暗黑魯斯王

稀有精靈編輯

點擊展開
  • 特指星球上特別難捕捉的特產
    • 隨著版本更替已再無更新

    • 屬性：草
    • 克洛斯星特產

      • 再進化：魔花仙子->魔域仙子
      • 再進化：魔花仙子->魔靈仙女
  • 莫比系列
    • 屬性：地面
    • 雲霄星特產
    • 最初的地面屬性精靈因此雷伊初推出很多玩家就是全靠他毆打到死
    • 進化鏈：莫比->格格爾->魯加斯
      • 再進化：魯加斯->魯迪諾斯->魯迪諾加斯
  • 利利系列
    • 屬性：電
    • 赫爾卡星特產
    • 曾在官方的一次更新中使其直接絕跡
    • 進化鏈：利利->綿綿->電擊兔
      • 再進化：電擊兔->狂暴電擊兔
      • 亞種（閃光）：閃光利利->閃光綿綿->閃光電擊兔
  • 林克系列
    • 屬性：冰
    • 塞西利亞星特產
    • 居住在冰洞內，需要點擊冰塊以切換照明
      • 當然僅為照明作用，不用的情況下依然可以點擊明雷
    • 進化鏈：林克->林斯奇->布林克斯
      • 再進化：布林克斯->布爾札克->寒冰之魂/裂冰之魂
  • 巴多系列
    • 屬性：火
    • 火炎星特產，需要攜帶Nono才會出現
    • 進化鏈：巴多->巴奈特->巴法爾
  • 閃光皮皮系列
    • 屬性：飛行
    • 克洛斯星特產
    • 第一個閃光精靈，為皮皮的亞種
    • 進化鏈：閃光皮皮->閃光波波->閃光波克比
  • 尼爾系列
    • 屬性：超能
    • 隨機出現，會偽裝成其他精靈的模樣，明雷狀態下無法辨認
    • 會在戰鬥開始5回合後逃跑，因此捕捉難度大大增加
    • 進化鏈：尼爾->菲斯利->艾斯菲亞
      • 再進化：艾斯菲亞->艾菲德斯
  • 尼奧系列
    • 屬性：超能
    • 尼爾的哥哥，隨機出現，會偽裝成仙人球跟利牙魚的模樣，明雷狀態下無法辨認
    • 會在戰鬥開始5回合後逃跑，因此捕捉難度（ry
    • 進化鏈：尼奧->菲斯德->艾斯菲格
      • 再進化：艾斯菲格->艾格諾斯
  • 格林系列
    • 屬性：地面
    • 大耳朵精靈，旅行到各個星球上，指定時間下才會出現
      • 一出現就會在地圖圍個水洩不通，不僅使遊戲變得緩慢，也會難以點擊明雷，成為捕捉的一大難關
    • 同樣會在戰鬥開始5回合後逃跑，配合上面的因素而使捕捉難度更進一步飆升
    • 進化鏈：格林->格林姆->格魯奇高
      • 亞種（閃光）：閃光格林->閃光格魯高
  • 布魯系列
    • 屬性：水
    • 旅行到各個星球上，指定時間下才會出現
      • 一出現就會在地圖圍個水洩不通，但情況比樓上格林好點可能是因為地圖夠大
    • 同樣會在戰鬥開始5回合後逃跑（ry
    • 跟格林是好友配對，曾差點經歷生離死別
    • 進化鏈：布魯->布魯布->布魯克克->布魯卡卡
      • 再進化：布魯克克->布魯卡卡
      • 亞種（暗黑）：布林->布林多->布林克克
      • 進化亞種：布魯克林

任務精靈編輯

點擊展開
  • 奇洛
    • 屬性：地面
    • 都市傳說等級，當初流出圖鑑情報時為大地星產物，然而時隔多年依舊沒有任何相關更新
    • 更多時候是海盜艾力遜的專用精靈，在星系未曾發展完全的時期玩家只能望梅止渴
      • 後終於推出相關任務，一隻被其遺棄的奇洛作為獎勵贈送予玩家
    • 進化鏈：奇洛->傑拉特->塔基拉頓
      • 亞種（暗黑）：洛奇->拉傑特->基拉塔頓
      • 亞種（閃光）：閃光奇洛->閃光傑拉特->閃光塔基拉頓
        • 再進化：閃光塔基拉頓->閃光塔克洛頓
  • 西塔、奇塔、卡塔
    • 屬性：機械
    • 通稱「赫爾卡三塔」，全是赫爾卡星土產機器人
      • 當中西塔與奇塔可融合成「多塔」
    • 西塔：
      • 初登場於「雲霄星戰役」任務，為艾里遜所綁架，後救出並修復，作為獎勵而贈送予玩家
        • 後因任務關閉而絕版，到拜倫號登場時可以在閉鎖空間中捕捉，5回合後逃跑且捕捉難度很高
      • 在塞西莉亞星的飛船上有一個被冰封的西塔NPC，完成其任務後可得到記憶晶體傢俱
      • 進化鏈：西塔->鐵達斯->雷吉歐斯
        • 再進化：雷吉歐斯-（融合）->多塔->托雷亞->托雷塔斯
    • 奇塔：
      • 完成「遺跡中的精靈信號」後獲得，作為任務精靈的存在感比前輩低
      • 進化鏈：奇塔->鋼塔斯->雷吉姆斯
    • 卡塔：
      • 完成「卡塔尋找」後獲得，千年前的赫爾卡星上全是他的進化體
      • 進化鏈：卡塔->赫卡特
    • 大哥大->EVE->薩克頭，從排場到出身都有點隔壁老祖三巨像的味道
  • 尼布
    • 屬性：水
    • 海盜佐格的專用精靈，完成「站長歸來」後獲得明明「尼布保衛戰」也有戲份，咋那次不能拿
    • 以進化體初登場在海盜相關任務裡並爆殺一堆玩家
    • 進化鏈：尼布->尼斯克

SPT編輯

帕諾星系編輯

點擊展開
  • 蘑菇怪
    • 屬性：草
    • Boss等級：10
    • 形象就是個長了腿的蘑菇
    • 需要以玩家裝備火焰噴射器射擊並破除其護盾又不會主動驅趕玩家，是加個心安理得的盾牌啊
    • 進化鏈：小蘑菇->蘑菇怪
      • 再進化：蘑菇怪->巨蘑怪
  • 鋼牙鯊
    • 屬性：水
    • Boss等級：25
    • 從只有二段進化看來，參考來源為隔壁老祖的同名精靈
    • 新手殺手，水屬性的克制面加上冰之牙能擊殺大多數不懂事的步步種子玩家
    • 進化鏈：利牙魚->鋼牙鯊
      • 再進化：鋼牙鯊->巨齒鯊
  • 里奧斯
    • 屬性：火
    • Boss等級：35
    • 早期為長了懶覺藍角的雌火鹿，後更改畫風
      • 早期形象不知道從哪個商標摳下的
    • 進化鏈：胡里奧->里奧斯
      • 再進化：里奧斯->里奧傑斯
      • 進化亞種：火炎巨鹿·里奥斯
  • 提亞斯
    • 屬性：飛行
    • Boss等級：40
    • 人妻，建築物上放的就是她的蛋
    • 樣子及設定可能來自摩斯拉
    • App版那個布偶是三小
    • 進化鏈：米拉米->提亞斯
      • 再進化：提亞斯->提亞戈斯
  • 阿克希亞
    • 屬性：冰
    • Boss等級：60
    • 雌老妖狐，站在塞西利亞星的滑梯冰涯上
    • 跟小雷伊是救命恩人及良師關係，雷伊落在星球上後受其照顧，並作為其鍛鍊對手，助其習得「雷神覺醒」
      • 後來玩家們炒作之下出現了跟雷伊組配對的現象
    • 初推出時為一眾當中極其破格的高等級，前後相差數十等級下成為許多玩家冒死挑戰然後直接被冰天雪地碾走的的高牆
      • 在當時遊戲尚未發展良多時，即使在屬性克制的場合下依舊難以跨越，當時網上流傳高等級火屬性精靈也要幾乎用換血打法才能對應之
    • 進化鏈：索拉->赫拉絲->阿克希亞
      • 再進化：阿克希亞->阿克妮絲
      • 進化亞種：女皇·阿克希亞、冰之契約·阿克希亞
  • 雷伊
    • 屬性：電->聖靈·電（聖霆）
    • Boss等級：45->50->70
    • 較上者晚推出（08至09年間），且初推出等級僅45，直接坐鎮赫爾卡星
      • 後來在官方拉拔下不僅技能開始擴展，等級也上調，更加入了組織，獲得宛如吉祥物兼親兒子的地位美國隊長
    • 初推出時學習了地震的布布花都能弄死，拉拔過後技能「白光刃」開始展露殺傷力，即使地面屬性也要注意隨時暴斃
      • 基拉大和，版本前無人問津，版本後絕世無雙
    • 跟蓋亞是勁敵關係，後成立「戰神聯盟」並加入了卡修斯、布萊克、繆斯等戰友
    • 經典台詞「我要秒殺你！」為攻打宇宙海盜時發飆所言
    • 與眾多ACG雷電相關的內容似乎有所關聯
      • 驚雷切（卡卡西「雷切」）、極電千鳥（佐助「千鳥」）、形象（索尼丘）
    • 沒有進化鏈
      • 再進化：雷伊->雷神雷伊
      • 進化亞種：雷神·雷伊、聖霆·雷伊、王·雷伊自然進化受阻症候群
  • 納多雷
    • 屬性：大地
    • Boss等級：70
    • 阿爾法星SPT，藏在地表下的三眼岩石巨像
    • 專用技能「屬性反轉」可以與對方交換屬性
      • 但是沒有明示所以根本不知道有沒有跟交換的精靈跟著換一次
  • 雷納多
    • 屬性：機械
    • Boss等級：80
    • 貝塔星SPT，樓上的哥哥，標準的山大王
    • 專用技能「屬性複製」可以複製對方的屬性
      • 但是沒有明示所以根本不知道（ry
  • 尤納斯
    • 屬性：暗影
    • Boss等級：70
    • 拜倫號SPT
    • 原本需要技能「貫穿水槍」擊穿其減傷屏障，後來則被修沒了
  • 魔獅迪露
    • 屬性：普通
    • Boss等級：50
    • 斯諾星SPT，藏在斯諾巖洞裡，拉動藤蔓會使之出現
    • HP高達300萬，需要運用異常狀態才能有效率地攻破
    • HP虛有其表的廢物，算好回合就可以玩死他
    • 加個鬃毛就「畫羊說是獅」了
    • 進化鏈：露露->魔獅迪露
      • 亞種（暗黑）：迪莫->狂獅迪卡
  • 斯賓塞
    • 屬性：暗影
    • Boss等級：50
    • 阿爾法/貝塔星SPT，邪靈組織的軍師，藏身在海盜能源站裡
    • 通常狀態下被命中時傷害只有1，只有異常狀態下才能造成正常輸出
    • 進化鏈：斯肯->斯賓塞
  • 克萊芬
    • 屬性：火
    • Boss等級：55->80
    • 露西歐星SPT，稱號「酒鬼」
    • 初戰時較脆弱，再戰時大幅增強醉酒發狂
    • 進化鏈：克迪->克萊芬

卡蘭星系編輯

點擊展開

    • 專屬技能「藥劑反噬」可使恢復效果轉變為扣除生命值
    • 進化鏈：Nike奈姬->奈尼芬多
  • 塔克林
    • 屬性：龍
    • Boss等級：80

      • 後回到哈莫星，在哈莫成為龍王後稱臣鑒於子供向而沒有處死

      • 塔克林的機器版，海盜仿照其製作的機器龍，外貌相差無幾

遊戲内容爭議編輯

劇情及系統設定爭議編輯

  • 故事各種吃書，且地圖上的劇情觸發條件很謎
  • 用完即棄的人設與活動故事
  • 活像中了面目全非腳的電影人設
  • 掛名邪惡組織海盜集團
  • 無用的收集能源系統
  • 無人性的精靈合體人體錬成

抄襲爭議編輯

  • 初期遊戲設計、玩法左抄右抄基本參考寶可夢系列，例如屬性設定、御三家、個體值、稀有種及閃光亞種的設定等
    • 卡通片主角是光屬性皮卡丘
    • 部分精靈的原設名字甚至改都懶得改，比如小火猴就是小火猴
  • 由於本遊戲上線時中國還未正式代理寶可夢系列，於是養成精靈的"創新"玩法得到了大量小學生玩家的推崇，在正值暑假結束的開學季新增大量玩家
  • 如果是初期還只是抄襲基礎設定，到後期人形精靈大量推出後躺槍角色可謂越來越多，包含《魔獸爭霸》的伊利丹、《Fate》系列的金閃閃、貞德均被盜用人設

後期的精靈設計編輯

  • 出到停不下來的屬性和雙屬性
    • 跟隔壁老祖甚至同行不一樣地是完全單位化的分類而非互相拼湊，因此種類比同行的奧拉星、洛克王國更多，非常多，多到不行
  • ご都合主義(各種意味)，精靈設計慢慢貼近近代手遊，各種人化長出奶子穿上衣服
    • 隨著時間過去，現今形象不僅不再注重非人形部分外觀，甚至直接把人的形象拿出去賣林肯要從棺材裏跳出來了
    • 我們是來抓精靈的？是來抓老婆的
    • 這他媽是明擺著的特殊種族人口販賣，你賽爾號不做點事？

課金爭議編輯

  • 從2010年起採用年費精靈機制，即充滿100元可以獲得強力精靈明擺著不課金拿不到
    • 初期的猛虎王和十二年兔強度充其量是收藏精靈，但從2013年開始開始該類精靈強度極高，甚至有很多關卡BOSS不依靠年費精靈根本打不過去因此明擺著騙課的目的更重了
    • 同年在已經推出年費精靈斯塔克的情況下又推出第二個年費精靈哈迪斯一年一次大騙課已經不能滿足了更是在玩家之間引發爭議，被稱為13暑假事變
  • 2014年開始對遊戲的刻印系統作修改，原本強化精靈可以用兩個能力刻印和全能刻印（前者只要玩家投入精力便能獲得，後者是課金產物），修改后變成了只能裝備一個能力刻印和兩個全能刻印直接廢掉了大半部分肝帝玩家的努力轉而逼課，此事件又被稱為"722事變"
  • 2015年則出現扎克斯挑戰暗改事件：
    • 原本玩家需要通過挑戰BOSS精靈成功後獲得精元才能養成精靈，但扎克斯的設定是挑戰時必定先手且一擊致命，無論等級多高的精靈都難逃一死
    • 但有玩家發現可以利用金色火焰（VIP Buff減傷害）在扎克斯面前存活，提高了玩家的挑戰成功可能性所以還是要課金
    • 但之後官方卻通過一次不關服的停服暗改了扎克斯的數值，變成即使用上述做法也會如前述一般被一擊斃命，導致扎克斯變得幾乎無法被攻略，想要獲得只能通過998鑽石（100元）進行一鍵通關
    • 雖然標明可以免費獲取但明擺著告訴你不課金是不可能的這種官方做法在玩家之間再度炎上，被稱為206事變你們這遊戲怎麼這麼多事變
