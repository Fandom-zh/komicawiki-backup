解說編輯

  • 英文單字；意為：技巧、鍛鍊。
  • 第2次超級機器人大戰α的主題曲，由影山ヒロノブ填詞、須藤賢一／河野陽吾作曲，主唱為JAM PROJECT
  • 不知何時開始成為了寵物部的版歌：
    • 只要有人張貼動物跳躍、騰空、飛翔時的圖片時，就會有人留「I can fly！」的歌詞，之後就會有人開始用之後的歌詞接唱，而且通常是唱第一段副歌。
      • 命運石之門的主題曲《ファテイマ》副歌首句也是「I can fly」，不過沒有引起上述現象
      • 不過日本人英文L、R不分，所以有時候會聽成I can fry、You can fry
