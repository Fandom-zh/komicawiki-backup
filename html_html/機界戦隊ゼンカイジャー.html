解說編輯

  • 超級戰隊系列第45作（46）紀念作暨令和超級戰隊第2作，2021年3月7日播映，全49話
  • 標題中的「ゼンカイジャー」根據讀音有「全開」「全界」等含義；主角的口頭禪是使用前者，機器人合體的官方宣傳語則使用後者
  • 本作各種突破可謂完全粉碎系列特色：
    • 主角戰士首次不用紅色而是白色，長得還幾乎和隔壁棚的某騎士一模一樣[1]，造型極具衝擊性地致敬《秘密戰隊五連者》的赤連者和《JAKQ電擊隊》的Big One
    • 另外四位初期成員則是捏他歷代超級戰隊機器人的機械生命體連緊身衣都不穿直接皮套上陣，更能從等身大小巨大化進行巨大戰
      • 雖不如過去作品的前輩們般會換上統一的戰鬥服，四位キカイノイド變身前後造型上仍有所區別
      • 雖然知道是受當時疫情關係要注意三密，但直接變成一演員+四皮套也未免太狠
        • 部分拍攝時還會像鄰棚一般採用室內實時合成特效背景，連室外接觸機會都避免了
        • 然而會染疫的還是會染疫的[2]
    • 因為同期的假面騎士戰隊化，所以戰隊便假面騎士化了
    • 不少島民驚呼這根本就是一個騎士帶著四台支援機或勇者王
      • 呼呼呼，你其實是從金屬英雄棚過來的吧
    • 除了造型跟隔壁的主角一樣是白+彩虹之外，連一話出現一個初期戰士的展開（除了第1話兩個）都光美化了
  • 由於《遊☆戯☆王》中也有【機界騎士（ジャックナイツ）】牌組且配色完美符合戰隊常用色，故本作剛公開名字時也有不少遊戲王梗
    • 因為又機體合體方式和勇者王的龍神系列相同，難道大反派的目的是機界昇華！？
  • 監製是近年曾執導《ZI-O》的鄰棚大魔王白倉伸一郎和武部直美，主編劇是香村純子大概也因此取消了戰隊系列恆例的ED與騎士看齊
    • 音樂由曾給《秘密戰隊五連者》、《宇宙刑事》三部曲、《無敵鐵金剛》等昭和作品作曲的老牌音樂人渡邊宙明，與曾參與戰隊和光之美少女系列多部作品作曲的大石憲一郎共同擔任
    • OP主唱是從隔壁棚穿越過來串場的つるの剛士
    • 道具音效由在日美國演員Lenne Hardt負責終於連戰隊也要找純正外國人配音效了
  • 本作每一話的標題均會有帶「カイ」發音的部分

劇情簡介編輯

五連者戰鬥的世界、獸連者戰鬥的世界、キラメイジャー戰鬥的世界…
無數的世界被キカイトピア的邪惡王朝トジテンド封印在細小齒輪之中消失了。
最後被盯上的是平行世界之中唯一的這個世界。
然而當トジテンド襲來時，感覺可以好好相處的キカイノイド蜂擁而至。

五色田介人以雙親留下的秘密道具「Geartrigger」與凝縮了超級戰隊力量的センタイギア變身成ゼンカイザー！
與擁有正義之心的4位キカイノイド一起迎戰トジテンド！
一位人類與四位機器人，成為守護所有世界的英雄！
其名字就是機界戦隊ゼンカイジャー！

角色介紹編輯

全界者（ゼンカイジャー）編輯

  • Zenkaizer（ゼンカイザー） / 五色田介人【演：駒木根葵汰、上野山夢輝（小時候）】
    • 在祖母的粗點心咖啡店打工，不論任何領域都想成為「世界初」的青年，2000年12月12日出生，20→21（第39話後）歲
    • 平常是個十分有活力超級樂天的笨蛋，但只要「開關啟動」就會變身成為熱血笨蛋，口頭禪是「全力全開」
    • 把每個人都看作自己的朋友，但時常越界闖入別人的私人領域
      • 見到因為世界融合而突然出現在人類世界的キカイノイド的第一個反應是主動上去交朋友
    • 第1話在トジテンド大舉侵略世界時雖然一度為キカイノイド並非全都是善良一事感到沮喪，但回想起父母的教誨後決定要保護世界
      • 上述舉動意外觸發了セッちゃん的啟動開關，受其指引得到了Geartlinger和センタイギア
    • 在所有平行世界被「神」再次封印後是唯一憑藉自己的力量回復記憶的人；於精神世界裡與「神」對峙，向祂訴說自己從各個世界的連繫之間交到了不同的同伴，並展示出決心要奪回所有世界的覺悟
      • 面對變身Zenkaizer→Super Zenkaizer的「神」自己則以集結在心裡的同伴之力召喚他們出來一起應戰，雙方交手多輪依然平手後改以猜拳一決勝負
      • 在一盤決勝負的猜拳中擊敗了「神」後摧毀了隱形於其世界之中的トジルギア裝置，再一次解放所有平行世界
    • 世界回復和平後本來想繼續留在自己的世界，但在家人的鼓勵下帶著セッちゃん、Juran、Gaon、Majine與Vroon一起開始穿越所有平行世界之旅
    • 戰隊版常磐ソウゴ
    • 跟隔壁的Cure Summer一樣捨棄掉多年常用的C位顏色，造型為白色+彩色組合


  • ゼンカイジュラン / Juran（ジュラン）【CV：淺沼晉太郎、演：高木勝也[3]】
    • 隊伍裡最年長的キカイノイド，力量（連腦袋也長肌肉）型，十分自來熟喜歡開派對和跳舞
    • 有如大家的大哥般照顧和支援另外四人，自以為是冷酷帥大叔，但實際上卻是有點吵耳又頑固和怕熱
    • 十分重視友情，與介人初次接觸後很快意氣相投
      • 第1話因為保護了被クダック襲擊的小孩而被介人看上，相當快地接下對方給予的Geartlinger加入戰鬥，收到的センタイギア又恰好是獸連者的
    • 對トジテンド有所認識，表示只要繼續受ボッコワウス統治キカイノイド們不論住哪都不會有幸褔
    • 原捏他是《恐龍戰隊獸連者》的大獸神，代表戰隊機器人類型的恐龍系
    • 傳說中的蟲師兼龍印者，還在某個櫻花樹開不停的小島上大享姊妹丼
    • 人間體之後跑去兼差當防衛隊的成員


  • ゼンカイガオーン / Gaon（ガオーン）【CV：梶裕貴、演：笹森裕貴[3]】
    • 比起冷冰冰的機器更喜歡地球既柔軟又溫暖的生物和人類的天然系キカイノイド，擅長烹飪
      • 會把包括介人在內的所有生物當成寵物看待和逗弄，相對地對於同族的キカイノイド則態度冷淡，尤其與Juran很不合
      • 第7話與介人的談話中也表示討厭自己身為キカイノイド的身份，初期只認為其他同伴是能夠一起保護人類的同志
        • 但在第18話受レンアイワルド的影響而與Juran產生了本人無法忘記的奇妙互動
      • 在第22話ゼンカイジュウギア「離家出走」事件中被Juran責難是否因為對方是機械而非人類所以才把它趕跑，重新審視自己過往的做法和觀點時回想起受大家需要的時光，於是一人扛下所有被トウギュウワルド變成鬥牛人的人們的攻擊讓Zenkaizer能趁機將前者擊破
        • 戰後跟Juran坦白承認自己太小孩子氣，從Flint處得知真相的Juran也向他為自己先入為主道歉，兩人冰釋前嫌
    • 看到介人在街上想用10張カラフル的零食券作報酬努力招募全界者成員的可愛模樣後對他起興趣，並想與他作朋友，後來成為繼Juran後第二位加入的成員
      • 起初因為不想與キカイノイド扯上關係而婉拒加入，但之後目睹自己最愛的人類和動物因キノコワルド的作戰受苦，於是一怒之下加入全界者
    • 原捏他是《百獸戰隊牙吠連者》的牙吠王，代表戰隊機器人類型的動物系
    • 偽裝真差，身體一看就知道是ゼンカイジュラン的改色
    • 前世也能巨大化，曾經暴肥過一段時間並和Juran是幼馴染
    • 跟隔壁棚的異魔神泰迪和怪盜BN團的巴蘭斯曾經在杜王町一起打過強龍者的前上司
    • 老婆是愛用人體改造手術的瘋狂醫生
    • 不論CV還是人間體演員的名字都是裕貴


  • ゼンカイマジーヌ / Majine（マジーヌ）【CV：宮本侑芽、演：新田さちか[3]】
    • 十分迷戀占卜、奇幻及超常現象等非科學事物的宅女性型キカイノイド，十分容易害羞，本性喜良但一旦自己喜歡的事物被拿來嘲笑就會爆氣
      • 小時候是Juran的鄰居，稱他為叔叔
      • 會用隨身攜帶的水晶球占卜，占卜時會唸出「ヌヌヌ、マジーヌ」的咒文（平日説話則會帶有「ヌヌ」的口癖），但似乎因為本人沒有才能結果往往不怎準確
        • 開始接觸占卜的契機是小時候想加入小孩子們的遊戲裡，猶豫不決時得占卜相助推了一把
    • 因為是宅女所以不擅長整理也是理所當然的
      • 房間沒幾下便會被她搞得髒亂不堪，甚至只是讓她顧店或打掃也能搞得一團糟
      • 相對地因為在垃圾堆中生活習慣了所以十分擅長在雜物中尋找特定東西
    • キカイトピア與介人的世界融合後流落當地彷惶，在第3話捲入全界者與コオリワルド的戰鬥中被因地滑到處亂撞的他們撞飛並因而與Juran重逢
      • 雖得ヤツデ收留但起初因為自己對粗活沒有信心+Juran表示讓她上場戰鬥太危險而婉拒加入，然而得介人鼓勵記起自己當初喜歡占卜的原因後萌起了想成為其伙伴的想法，在得占卜「推一把」下成為第三位加入的成員
    • 原捏他是《魔法戦隊マジレンジャー》的魔法王，代表戰隊機器人類型的奇幻系
    • 寶多六花


  • ゼンカイブルーン / Vroon（ブルーン）【CV：佐藤拓也、演：大橋典之[3]】
    • 有如班長般十分嚴肅認真還（連自己本人也）不知為何戴眼鏡的キカイノイド
    • 好奇心十分旺盛，只要是有興趣的事情便會一直想追問到底、完全不看場合和時間甚至會給大家和自己帶來麻煩
      • 起初因為想學習一般庶民不能知道的知識和技術而加入トジテンド，結果卻是被派去イジルデ身邊擔任トジテンドパレス的清潔工
        • 但從幹部們身邊不但什麼都學不到，當他聽到幹部們討論自己好奇的話題想插嘴時都會被他們電（物理）回去
      • 兩度在城內觀察到本被封印的平行世界在トジルギア粉碎時解放後為究明原因詢問イジルデ，結果招致殺身之禍被後者派兵追殺
    • 上述被イジルデ派兵追殺時亂入到全界者和ボクシングワルド的戰場，對全界者及センタイギア充滿好奇之下不斷打斷戰鬥問問題，在快要被ボクシングワルド帶回去時得他們相救並帶走
      • 來到現世接觸到了許多新鮮事物而且不論自己問什麼問題介人都會盡量回應，讓其瞭解到自己之前待的世界是多麼狹隘，於是決定與トジテンド訣別並加入介人一方要跟隨他學習更多知識
        • 是第一個加入後主動問要如何變身的成員所以不像上面三人般沒事先開一槍再問
    • 因為曾擔任清潔工所以十分愛整潔和擅長整理，不僅效率極高還會做好垃圾分類
    • 原捏他是《轟轟戦隊ボウケンジャー》的大冒險但怎樣看都比較像柯博文，代表戰隊機器人類型的載具系
    • 機械版川口名人
    • 前世跟克里斯塔利亞的國王是舊識，還會用肥皂泡攻擊敵人


  • Twokaizer（ツーカイザー） / Zocks Goldtsuiker（ゾックス・ゴールドツイカー）【演：增子敦貴】
    • 第8話起登場，為尋找寶物與來自海賊トピア的弟妹們一起穿梭不同世界的「界賊（世界海盜）」
      • 家族信條是「對有興趣的東西要一股腦地上」，然而海賊トピア是個十分動蕩的世界，父親便是在保護他們時喪命；此後他接下父親的遺志以一家之長的身分努力守護家人
      • 因為受前世影響[4]和出身的世界裡所有人都是當海盜所以一切都用搶的可謂常識，初次見面時被介人帶到カラフル招待後不但不付錢還與弟妹一起直接洗劫店子一空
      • 在與介人比拼誰先打敗カシワモチワルド的決鬥中敗陣[5]後答應對方條件不會在這個世界作亂，並為之前搶劫カラフル付帳然而給的不是這個世界流通的貨幣
      • 戰鬥時不擇手段，毫不理會旁人的死活
    • 作為家族的兄長雖然相當看重弟妹們的性命安全，在知道Flint受到カシワモチワルド影響後馬上放下與介人的決鬥趕到現場
      • 自身激進的做法也是為了解除Cutgunner和Ricky身上的詛咒追尋著SDトピア的トジルギア，對其他ギア提不起興趣
        • 知曉其目的的Stacy於是在第14話提出要他打倒Zenkaizer去換取帶其去トジテンドパレス取得想要的トジルギア的交易外包再外包的概念；他表面上應諾、約介人出來決鬥並打倒了他，但其實早已猜到Stacy會翻臉不認帳而與全界者聯手演戲引蛇出洞
      • 相對地有著自己出事時要自行解決不能讓弟妹來救的信條，即使與他們一同作戰也必定要由自己決勝負
    • 一出場便邊尬舞邊作自我介紹，變身動作也是在跳舞，還能用跳舞時間長短判斷他有否認真（by公式）
    • 到第44話尋覓以久的SDトピア的トジルギア終於以ワルド的模樣在他面前出現；儘管在戰鬥中Geardalinger被變成SD大小而失去變身能力也依然決心單刀赴會，要家人們與介人不要插手介入
      • 但介人還是出於「不會阻礙你，我們只是因為想守護大家而順便」帶領全界者參戰，Cutgunner與Ricky也跑來當保護他的肉盾，Flint還給他製作了配合Geardalinger大小的センタイギア讓他能再次變身
      • 擊破SDワルド後為解除弟弟們的詛咒舉家離開介人的世界往獲解放的SDトピア出發，但在航行途中發現海賊トピア被解放後開始心生懷疑於是折返，並在第47話攻入トジテンド與Stacy一同支援介人
    • 原捏他是《海賊戦隊ゴーカイジャー》的豪快金紅，變身後造型與裝備外觀都是Flint見過豪快者後抄過來的，所以比起全界者更像是豪快者的追加戰士也是理所當然的
      • 定位像是ソルダートJ


  • ゼンカイレッド【演：伊勢大貴】
    • 《ゼンカイレッド大紹介！》登場，因某種理由「領隊就應該是紅色的吧」與介人槓上的謎之戰士
      • 變身後的外觀完全是紅色版的Zenkaizer變得更像赤連者了
    • 内心有股想要成為紅戰士和超級戰隊隊長的心而被ノーマル惡者ワルド引誘，獲得了センタイギア和Geartlinger
      • 上面的四位キカイノイド也因為被ノーマル惡者ワルド的能力影響變成人形姿態，五人一度組成了全新的戰隊

    • 必殺技：スーパーゼンカイキック

合體機器人編輯

  • ゼンカイオー
    • Juran、Gaon、Majine，Vroon四人裡其中兩人互相對稱全界合體成的機器人，官方稱之為「ゼンカイコンビネーション」
      • Juran+Gaon：ゼンカイオー ジュラガオーン，1000萬馬力
        • Juran主導時擅長以ジュランソード亂砍，Gaon主導時則擅長足技和以ガオーンクロー亂抓
        • 必殺技：ジュランソード　円月クラッシュ
      • Vroon+Majine：ゼンカイオー ブルマジーン，1000萬馬力
        • Vroon主導時會把左手的ブルーンピッカー當成能射出光束的弩使用，Majine主導時則能以右手的マジーヌスティック詠唱咒文
        • 必殺技：ブルーンピッカー　アクセルストライク
      • 上面兩個有皮套，下面兩個只有純CG
      • Juran+Majine：ゼンカイオー ジュラマジーン，900萬馬力
        • 能以背上的マジーヌウイング飛行和攻擊，高速回轉時還能藉助風のエレメント的力量變出防護罩或真空刃
      • Vroon+Gaon：ゼンカイオー ブルガオーン，1100萬馬力
        • 力量型+力量型的組合，能以雙腳的タービンホイール高速衝刺並以剛力無雙的ゴーゴーパンチ毆打
      • 不要問為什麼沒有ジュラブルーン或ガオーマジーン
    • 合體時的背景會根據組成左邊的成員來決定：左邊是Juran時為棒球場、左邊為Vroon時則是拳擊擂台
    • 戰隊版左右合體的龍兄妹
    • ご唱和ください我の名を！ゼンカイザー！
  • ツーカイオー
    • クロコダイオー與Cutunner和Ricky界賊合體成的機器人，是兩人為更能協助Twokaizer戰鬥拜託Flint改造追加的合體形態
    • ツーカイオー カッタナー：由Cutunner組成頭部、Ricky組成左臂的カッタナー刀，1100萬馬力
      • 必殺技：カッタナー刀　烈火大斬
    • ツーカイオー リッキー：由Ricky組成頭部、Cutunner組成右臂的超リッキーガン，1100萬馬力
      • 必殺技：超力バルカン、超力クラッシュ、超力カンフー
  • ゼンカイジュウオー
    • 巨大Super Zenkaizer與Super Twokaizer SD超全界合體成的機械獸，1100萬+40萬=1200萬馬力
    • 擅長如怪獸一般大肆破壞，全身上下所有火器同時射擊可發揮強大火力
      • 必殺技：カイジュウマックスブラスター、ゼンカイジュウダッシュ、ゼンカイジュウドリル、ゼンカイジュウオー・ツインブレイカー
        • 使出ゼンカイジュウドリル時會變形成類似豪獸鑽頭的飛行（？）形態[8]
      • 不知甚麼原因牠出場次數多
  • スーパーゼンカイオー ジュラン
    • 巨大Super Zenkaizer與Juran全界合體成的機器人，1600萬馬力
      • 因為合體機構關係理論上也能與Vroon合體，但從現時資訊看似乎只會被當成玩具原創合體形態
    • ゼンカイジュウオー時的高火力特色幾乎全數保留，但同時也擁有與重武裝不相稱的靈活機動性
    • 兩次出場都被對手隨意攻擊便炸至解除合體，劇組大概潛意識討厭這形態
  • スーパーツーカイオー
    • Super Twokaizer SD與クロコダイオー和Ricky界賊合體成的機器人，1250萬馬力
    • 合體後外觀與ツーカイオー カッタナー相似，但胸口和頭部均換成Super Twokaizer SD的部件，原本胸口的「炎」字則與巨大Super Zenkaizer的ゼンカイテンランス和胸甲合體成剛龍槍拿在右手使用
  • ゼンリョクゼンカイオー
    • Zenryoku Eagle與Juran、Gaon、Majine和Vroon全力全界合體成的機器人，3150萬馬力
    • 以大家的武器合體而成的劍ゼンリョクゼンカイソード為武器，轉動胸口的ゼンリョクギア能召喚出對應組合的超級戰隊巨大戰力（組合與ゼンリョクゼンカイキャノン時不同）助戰
    • 必殺技：ゼンカイジャー・ゼンリョクファイブスラッシュ、ゼンカイジャー・オール戰隊ファイナルビッグバン
    • 其實本作未有預定推出全合體機器人，但在劇組的強烈願望下硬生了出來
