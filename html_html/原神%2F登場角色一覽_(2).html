解說編輯

  • 本條目為《原神》的子條目，為減少條目冗餘，這裡專門介紹收錄遊戲中登場的角色。

主角編輯

「旅行者」空/空（そら）/Aether、熒/蛍（ほたる）/Lumine編輯

  • 日語CV：堀江瞬（空）/悠木碧（熒）；漢語CV：鹿喑（空）/宴寧（熒）
  • 神之眼：無 / 使用武器：單手劍 / 命之座：旅人
  • 因自己故鄉被毀而在無數世界中流浪的孿生兄妹，在一場事件中被迫分離後踏上了尋找親生手足的旅程。
    • 序章過場動畫可以看到失散（選擇性別時沒有選到）的那方疑似成為深淵教團高幹並主導風龍武器化計畫，是引發蒙德城龍災的黑幕。
    • 一幕四章更透露遊戲開場動畫劇情就是五百年前的坎瑞亞。當時兄妹兩人隨隕石墜落提瓦特，醒來時發現整個世界已經陷入一片火海，隨即準備撤離，卻遭到自稱天理調停者的女神攔截。兄妹失散後，玩家沒有操控的那方成為「王子/公主」，領導深淵教團繼續「對抗天理的戰爭」。
  • 沒有神之眼，但可藉著跟不同屬性的七天神像共鳴而使用對應的元素力量。
  • 是遊戲開局送的五星角色。
  • 由於是系統送的角色，等級突破素材跟命座解鎖道具會經由提升冒險等級或完成任務時作為獎勵入手，不需另外蒐集。
    • 聽起來很佛心，但角色技能的升級素材還得經由挑戰不同的副本蒐集，並消耗大量的金錢才能封頂。
    • 而且每種屬性的技能等級不共用，需要分別培養。不愧是米坑遊。
  • 兩位中之人均是本遊戲玩家。
    • 配妹妹熒的悠木碧的遊戲進度更相當靠前（世界等級8、冒險等級55）。本遊戲三大重度玩家聲優其之一。
    • 悠木的角色性癖似乎相當廣泛。一開始培養正太隊，最近表示被魈打中好球帶，更曾在推特上表示萌到打了一堆亂七八糟的推文，但想想覺得不太妥當就在發推前全部刪掉了。
    • 配哥哥空的堀江瞬則是會跟其他聲優友人組隊鋤大地，也是公式廣播節目提瓦特放送局的三位主持人之一。
      • 堀江因為是田村粉所以是七七推。
      • 在神里綾人的角色公佈後曾在推特上大喊即使世界毀滅也要抽爆綾人的豪邁發言。可見是石田粉。

派蒙/パイモン/Paimon編輯

  • 日語CV：古賀葵、漢語CV：多多poi
  • 為了報恩而主動擔當起嚮導任務幫助主角的小精靈。
  • 除了當嚮導之外，多數時間都只是一個廢萌吉祥物，甚至在對話內容中被說成是應急食品。
  • 意外地是個吐槽役，甚至還會透漏不少玩家對遊戲的觀感。
  • 由於外表的關係，因此有不少人認為祂跟開頭就露臉的「天理調停者」是同個人，而且祂不只和塵世七執政與其他故事提及的各路魔神同樣命名自《所羅門的小鑰匙》，至上四柱之西柱，在原典中更是高居王族二位，僅次於首席巴力。
    • 更奇怪的是，整個提瓦特似乎沒人對派蒙這個名字有任何反應，甚至對這個沒見過的廢萌吉祥物的存在沒有任何懷疑與不自然的感受。
    • 從頭上頂著像是環冠帽飾，應該是象徵理之冠的執政官。
  • 在宿怨裡面有一群信仰祂的邪教徒。
  • 公式廣播的三位主持人之一。
    • 因為本不玩遊戲，所以在她主持的回數基本上都有讓她體驗遊戲的專欄，也能藉此吸引新手玩家入坑。

蒙德編輯

西風騎士團編輯

「代理團長」琴·古恩希爾德/ジン・グンヒルド/Jean Gunnhildr編輯

  • CV：齋藤千和
  • 神之眼：風 / 使用武器：單手劍 / 命之座：幼獅 / 五星
  • 西風騎士團的蒲公英騎士兼代理團長，蒙德三大舊貴族之一，古恩希爾德家族的後裔。
    • 該家族世代維持風神祭儀傳統，在五百年前對抗勞倫斯家族暴政的起義中選擇放棄貴族身份與人民站在一起，並在戰後協助建立西風騎士團。
  • 有著對職業盡忠職守和為人民福祉的強烈責任感。
  • 本身並無天賦，而是透過自己的勢力刻苦訓練後奮鬥出來的。
  • 對自己要求嚴格，在自己的傳說任務中即使無法承受負荷依然堅持崗位，事後騎士團的成員私下幫她扛工作還為她辦了驚喜派對。
  • 常駐池的五星角之一。

「偵察騎士」安柏/アンバー/Amber編輯

  • CV：石見舞菜香
  • 神之眼：火 / 使用武器：弓 / 命之座：小兔 / 四星
  • 有著無盡的活力和熱心腸，但同時因急性子又太容易相信別人而顯得像傻瓜。
    • 反之在執行任務會表現出機靈的一面，是個懂得利用周遭環境做出適合戰術的天才。
  • 擅長使用風之翼，已連續三年繼承飛行冠軍的連霸。
  • 四分之一璃月人，祖父是璃月港出身的跨國商隊保鏢，商隊遭怪物全滅時被西風騎士團救下，為報大恩毅然加入騎士團並在蒙德安身立命，是偵查騎士小隊的創辦人。
  • 會隨著主線進行而得到一隻，跟凱亞及麗莎被玩家通稱為御三家。
  • 遊戲實裝迄今從未曾pick up過，只能從遊戲商店換取或是等抽卡歪出來，所以至今能把她抽滿命的玩家不多。
  • 雖然戰力不算高，但是遊戲初期有不少地方需要用到她的火屬性和弓箭射擊解謎。

「圖書管理員」麗莎·敏茲/リサ・ミンツ/Lisa Minci編輯

  • CV：田中理惠
  • 神之眼：雷 / 使用武器：法器 / 命之座：沙漏 / 四星
  • 嗜睡成性的知性魔女，負責圖書館的管理工作。
  • 雖然看起來一副遊手好閒的樣子，卻總是能以最有效率的方式處理眼前的問題。
  • 被她留學的須彌教令院譽為「兩百年來不曾有過的奇才」，甫加入騎士團就被團長親自延攬擔任第六連隊長，連不喜歡騎士團的迪盧克老爺都稱讚她入團必成大器，但她婉拒所有邀約，只願意擔任圖書館員，且不但準時到退，還把比較吃力或者需要出門的工作全部「外包」給了其他同事。
    • 她的許多人設細節都讓人聯想到跟死亡有關的象徵，官方更一度親口承認她命不久矣。
  • 主線劇情免費贈送的御三家之一。
  • 爬山的聲音特別好聽，不愧是田中理惠。

「騎兵隊長」凱亞·亞爾伯里奇/ガイア・アルベリヒ/Kaeya Alberich編輯

  • CV：鳥海浩輔
  • 神之眼：冰 / 使用武器：單手劍 / 命之座：孔雀羽 / 四星
  • 酒業大亨萊艮芬德家族的義子，同時也是琴團長最信任的副手，只要是委託的任務總能夠順利解決。
  • 其個性風趣幽默又不拘小節，有著行事方式和隨意而為的人生觀。
    • 雖然其行事風格招人非議，但他本人並不在意，不管是與同伴並肩作戰還是背水一戰，他都會展現出玩世不恭的態度。
    • 每次只要可莉給蒙德帶來不必要的麻煩時，都是他幫助可莉脫罪。
  • 隱約有著不為人知的秘密，據信是坎瑞亞派來的臥底。
  • 御三家之一。
  • 提瓦特男神團其之一，凝冰渡河真君。
    • 其元素戰技cd短範圍廣，能輕易在水面製造能夠行走的冰路，成為過河渡海必備的工具人。

「逃跑的太陽」可莉/クレー/Klee編輯

  • CV：久野美咲
  • 神之眼：火 / 使用武器：法器 / 命之座：四葉 / 五星
  • 西風騎士團的火花騎士，有研發火藥的天賦。
  • 從小就被作為冒險家的父母託付給了西風騎士團，尖耳朵的特徵疑似是亞人種。
  • 行動時總伴隨火藥出現而造成許多損傷，雖然騎士團都得收拾爛攤子但依然心疼她。
    • 每次闖禍過後就會受到琴團長的「關照」下關在禁閉室。
  • 遊戲開服後第一個pick up的限定五星女角，也是目前為止營收最高的角色。FBI警告
    • 該記錄於2021年九月正式被雷電將軍超越。

「無勳之花」諾艾爾/ノエル/Noelle編輯

  • CV：高尾奏音
  • 神之眼：岩 / 使用武器：雙手劍 / 命之座：心護 / 四星
  • 儘管沒有獲得騎士資格，但仍以女僕的身份在騎士團認份的工作兼努力學習。
  • 性格單純善良到誇張的地步，作為騎士團不但沒有要搶攻功勞的想法，更在自己的邀約故事中被人欺騙而沒自覺。
  • 新手玩家首個10抽必出一隻。
  • 大概是白銀諾艾爾在提瓦特大陸的分身。

「白堊之子」阿貝多/アルベド/Albedo編輯

  • CV：野島健兒
  • 神之眼：岩 / 使用武器：單手劍 / 命之座：白堊之子 / 五星
  • 西風騎士團的鍊金術士，年紀輕輕卻有令人折服的知識。
  • 為人氣質高雅又彬彬有禮，表面上雖然疏於交際，其實上他會不吝於為朋友伸出援手並繪製精美的肖像畫。
  • 非常注重研究"生命"的課題，對財富和人脈視若無睹，唯一感興趣的東西就是寫生。
  • 身上掛著一個非常罕見的「四角星」徽章，其他持有同款徽章的角色有凱亞和戴因斯雷夫，似乎是坎瑞亞人的身分證明。
  • 真實身份是五百年前創造魔龍烏薩引爆大災厄的罪魁禍首，大煉金士「黃金」的徒弟。
  • 1.2版本實裝的限定五星角。
  • 雖然中之人自己不是玩家，但因為兒子是，所以曾經跟兒子一同前往聯乘原神的甜點店。還被兒子吐槽沒有玩遊戲的才能。
  • 每次開池的豪華程度都會忍不住讓人懷疑米哈遊高層裡有很多人廚阿貝多。

「無害甜度」砂糖/スクロース/Sucrose編輯

  • CV：藤田茜
  • 神之眼：風 / 使用武器：法器 / 命之座：燒瓶 / 四星
  • 好奇心強烈又盡心盡力學習的鍊金術士，為阿貝多的助手，研究方向為「生物鍊金」。
  • 有明顯的獸耳特徵，幾乎是蒙德城裡唯三的亞人種隊伍成員。
  • 是目唯一沒有專屬的傳說任務或邀約事件的蒙德實裝角。

「浪沫旋舞」優菈·勞倫斯/エウルア・ローレンス/Eula Lawrence編輯

  • CV：佐藤利奈
  • 神之眼：冰 / 使用武器：雙手劍 / 命之座：波沫 / 五星
  • 「浪花騎士」，西風騎士團游擊小隊隊長。
  • 蒙德三大舊貴族之一，勞倫斯家族的末裔。因為祖先的獨裁暴政而飽受蒙德居民的有色眼光，加入騎士團前是城內所有商店的拒絕交易對象。
    • 本就不被家族疼愛的她，更因為加入了推翻貴族統治的西風騎士團而在家中受盡白眼。
    • 在自己的傳說任務中更親自逮捕了私通愚人眾意圖謀奪蒙德城的家族成員。

西風教會編輯

「祈禮牧師」芭芭拉/バーバラ・ページ/Barbara Page編輯

  • CV：鬼頭明里
  • 神之眼：水 / 使用武器：法器 / 命之座：金杯 / 四星
  • 西風騎士團的祈禮牧師，也是蒙德城的知名偶像。
  • 西風主教西蒙·佩奇的女兒，亦是琴團長的妹妹。
    • 對於比自己優秀的姐姐面前感到自卑，而為了跟上腳步決定努力學習，但因天賦的原因只選擇當後勤。
  • 從小就樂觀開朗，儘管有些傻里傻氣的，但總是能振作起來再度嘗試。
    • 給自己制定"失落時間"三十秒的規則，時間一到就會重新打起精神。
  • 之所以成為牧師是因為不善戰鬥，另一方面是想要治愈他人並讓希望他們愛惜自己的同理心。
  • 玩家冒險等級到達20級時免費入手一隻。如無意外會是玩家第一隻入手的奶媽。

「棘冠恩典」羅莎莉亞/ロサリア/Rosaria編輯

  • CV：加隈亞衣
  • 神之眼：冰 / 使用武器：長柄武器 / 命之座：荊冠 / 四星
  • 雖說是色氣褲襪修女，但其言行舉止冷淡又有些無所事事，絲毫沒有神職人員該有的風範，不過也有不凡的本領。
  • 角色故事透露她來自某個不確定是否在蒙德境內的小山村，被毀掉村子的強盜撿走，年僅十歲便覺醒冰屬性的神之眼，之後被西風騎士團救回蒙德並安排進入教會，因其特殊才能而在修女身份的掩護下擔任教會的「劊子手」，負責暗中處理蒙德境內的不安定份子，可以說是類似埋葬機關的人物。

  • 神之眼：火 / 使用武器：雙手劍 / 命之座：夜梟 / 五星

  • 雖然是酒業的大少爺，他本人卻對酒沒有太大的興趣。

    • 璃月魔神之一，岩神的紅顏知己，擅長鑽研機關術，三千年前與岩神結盟建立「歸璃原」治理璃月人民生活。
    • 在魔神戰爭中身亡，岩神承續她的遺願與契約掃平璃月的魔神戰爭，成為岩神維持璃月和平治世。
  • 「鹽神」赫烏莉亞/ヘウリア/Havria
    • 璃月魔神之一，女性，建立地中之鹽城鎮治理子民，魔神戰爭時雖然努力庇護子民四處奔波，但權能不夠強大的她無力對抗魔神襲擊，只能一昧避戰，最終被部份子民背棄而殞歿。
    • 命名自《所羅門的小鑰匙》72柱魔神之一，公爵64位蔽魔「浩瑞士」（Haures，又名Flauros/佛勞洛斯）。可通曉古今，預言未來，形態為豹。

  • CV：喜安浩平
    • 璃月魔神之一，曾幫助岩神討平魔神戰爭，但對待人類和平共處一事有所糾結而被岩神封印。
  • 「爐灶之魔神」馬科修斯
    • 璃月魔神之一，曾幫助岩神討平魔神戰爭，為平息魔神殘渣對璃月人民的傷害，犧牲神力平息魔神殘渣而衰退隱沒。
    • 真身即香菱身邊隨行助戰的熊型元素生物「鍋巴」，原型為所羅門72位魔神第35柱馬可西亞斯（Marchosias），位階為侯爵，別號「第七座天使侯爵」。

    • 出現在璃月港玉京台的老婦，滌塵鈴的主人，旅行者與派蒙受到鍾離委託找她索取滌塵鈴時，讓旅行者與派蒙進入塵歌壺內的小洞天世界取出滌塵鈴，讓旅行者與派蒙認為她是仙人的猜想。
