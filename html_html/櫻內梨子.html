櫻內梨子

櫻內梨子（桜内（さくらうち） 梨子（りこ））

目錄

解說編輯

  • 《LoveLive! Sunshine!!》的主要角色
  • CV：逢田梨香子
  • 生日：9月19日（處女座）
  • 血型：A型
  • 身高：160 cm
  • 三圍：80/58/82
  • 代表色：櫻色（サクラピンク）
  • 象徵圖案：鋼琴

簡介編輯

  • 16歲，浦之星女學院的2年級生。
  • 學園偶像團體「Aqours」的成員，所屬分隊為「Guilty Kiss」。
  • 由音乃木坂學院轉學而來，但在動畫中完全不認識μ's及校園偶像活動。觀眾表示失望和不科學
  • 對自己長得地味而感到自卑所以起初拒絕成為偶像，但在高海千歌的強行勸誘下加入。
    • 可是各作品中均有「長得可愛」、「美人」之類的評語，或許只是缺乏自信而已。
    • 動畫版不但把自卑屬性鬼隱，而且似乎對自己的相貌還有點自信。
  • 常識人，弱氣受，所以很自然被認為會成為隊中的吐槽役中之人也是，會被千歌等人耍得團團轉。
    • 動畫版則比較強勢攻，敢怒敢言。這不就是中之人的作風嗎，而且一出場便脫衣跳海到底哪兒正常了
    • 因逆流的關係似乎漸漸也浮現出糟糕以及腹黑屬性…果然粉紅切開都是黑
    • 和善子混熟後被帶壞習得了對方的顏藝技能和中二口號。大怪獸梨子、梨子宇宙射線
  • 又一位因動畫版而形象大崩壞的角色。天國的初設
    • 怕狗，尤其是千歌家的香菇，第3及第5話裡與香菇的對手戲均讓她淪為惡搞素材
    • 喜歡以壁咚、壁咚抬下巴為題材的同人本，令不少觀眾尤其是梨子推跌破眼鏡。
      • 疑似中の人的性癖逆流回角色上
  • 擅長美術的室內派，大城市出身的她不太適應鄉下環境。
    • 然而中之人的作畫能力堪稱史詩級，超越小鹿的天才終於姍姍來遲了
    • 甚至連自己的角色是美術部的設定都忘了。
    • 筆下作品由於太過魔性，被粉絲笑稱為逢田文明
      • 最知名的作品是大象（無誤）
    • 可能因為這個原因，動畫版索性把這一點刪去了
  • 與某前輩一樣不太擅長站於人前（雖然用了半集的時間便克服了），也對短裙子也頗為抗拒。
    • 動畫中梨子的校裙比其他成員的長。
  • 擅長中提琴，亦會彈鋼琴而且又是紅髮的，這既視感…
    • 據說是因為不滿某土豪學姊霸佔著學校音樂室所以才憤然轉校
    • 動畫版自稱自小學習鋼琴，轉校前也參加過發表會，亦會作曲，順理成章成為Aqours的作曲擔當。
    • 中之人在第二回Live中彈奏想いよひとつになれ曾因為錯音一度造成表演中斷，不過後來表演還是重新開始，次回表演更在中斷的地方走下來擔任Center
  • 官方前期推的是千歌×梨子，動畫版中甚至曾經向千歌表白（無誤），因此千梨一度是熱門CP，同人作品中可攻可受，甚至會被描寫成千歌廚
    • 第二季後還出現曜×梨子的配對，由於二人的關係被戲稱為邪教配對。
  • 會被善子喚作「梨梨（リリー）」，梨子則稱呼對方為「夜醬（よっちゃん）」（G's/SIF設定）。
    • 由於兩人在雜誌環節時代已經有互動，彼此之間限定的暱稱這一點也非常耐人尋味，再加上中之人感情很好，善梨/夜梨CP在同人作品中亦擁有極高的人氣這時梨子通常被畫成超S、ヤンデレ或痴漢，第二季第五集確定成為官推CP
    • 還有禁斷的梨菇人獸配（設定出來前）
  • 獲得稱號：あざと医（心懷歹醫）、腹黑梨、工口梨、畫伯、梨子老師、腐梨子、本子魔
  • 擁有偶像以外的才能，擔任過鋼彈尖兵，還成為了魔法少女

經歷編輯

  • 為了做出有大海感覺的樂曲，脫下音乃木坂的制服跳海聽海的聲音，被千歌拉住後雙雙落入海中。
  • 次日正式轉學到千歌班上，經過一番勸誘後成為最初的團員之一。
  • 日漸與器重她且互為鄰居的千歌友好，而後也在她與曜的鼓勵下，在上京參加鋼琴比賽時成功克服心魔演出。
  • 第二季時與合力收養狗的善子也成了朋友，並克服懼犬症。
  • LL決賽前借用母校的鋼琴演奏。
