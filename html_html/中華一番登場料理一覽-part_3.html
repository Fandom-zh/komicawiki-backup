中華一番登場料理一覽-part 3

中華一番登場料理一覽-part 3
本條目是一個編輯中的詞條，歡迎有興趣的朋友加以補完。
為安全起見，本條目內記載的準空想級及以上之料理均不建議讀者在閱讀後試圖於現實中重現。

目錄

必殺七色餃編輯

  • 烹飪者：阿昴、解魯
  • 特色：用了多種食材並做出多樣形狀的餃子
  • 秘密：口味不同，且有不同顏色，在視覺上有不同感受
  • 空想度：非空想菜色
    • 西安飲食三絕之中的「西安餃子宴」
    • 難度不高，難的地方就只有捏出那麼精細的造型要很費功夫當然如果用趙瑜的翡翠餃子的做法來打發就不用費那麼多功夫了
      • 口味上也是依照廚師的調味方式像是阿昴喜歡吃麻辣，有些人也是受不了川味的辣度的（像梅麗就吃到噴火）
    • 種下了解魯冤獄的開端

變色龍碗塔編輯

  • 烹飪者：阿昴
  • 特色：
    • 以雞蓉麵、翡翠麵等各式麵條，和辣椒湯、椰汁湯等各色冷、溫湯頭加以組合
    • 緩兵料理，延遲官差上法場的時間
  • 秘密：組合高達數十種，並以少量的方式促進食慾
  • 空想度：非空想菜色
    • 根本就是日式小碗蕎麥麵了
    • 不要討論麵條本身和調味手法的空想成份了，大家一起來用這自助吧組個巴比倫通天碗塔吧

小籠包塔編輯

  • 烹飪者：解魯
  • 特色：量很多，蒸籠量可以堆滿一整間房間，湯汁吃起來像雞汁，但比雞汁更爽口鮮美
  • 秘密：使用的肉餡是滋補性好的眼鏡蛇肉，其他還加入了對身體有益的配料
  • 空想度：非空想菜色
    • 除了眼鏡蛇肉不是能隨便買的食材就是了
    • 雷恩：真難吃，我喜歡味道淡一點的東西。
      • 引起解魯暴走：你這個臭小子！
        • 手上拿著好幾個空蒸籠的四郎：嫌難吃還吃這麼多？

鮑魚濃湯編輯

  • 烹飪者：翁聲譽
  • 特色：濃郁的香氣，好喝到骨頭都要酥了，會上癮
  • 秘密：用傳說中的廚具「魔聖銅器」把要好久才能泡開燉熟的乾貨鮑魚瞬間回復到新鮮的狀態，即使是超大塊的鮑魚也可以
  • 空想度：超空想菜色
    • 因為魔聖銅器有多神在現實中是無法驗證的，所以…

奇蹟彗星炒飯編輯

  • 烹飪者：阿昂（動畫版追加雷恩、解魯）
  • 特色：從天而降的神蹟，富含太陽與大地的芬芳
  • 秘密：
    • 先準備好蔬菜炒飯，再用投石機丟送至眾人面前
    • 利用飲食均衡的觀念，彌補村民因長期吃鮑魚造成的營養不良
  • 空想度：強空想菜色（使用投石機） / 非空想菜色（不使用投石機）
    • 如何將鬆散的炒飯不用外物包起來，在一般環境下使用投石機丟出數百公尺遠而不會散開就是一個極大的問題
      • 除了真空環境下，一般環境中因為有空氣阻力使得炒飯每個單位食材受到的阻力都會有差距，在受力不一致的情況下結構鬆散的炒飯自然會散開
      • 扣除投石機這個空想因素，蔬菜炒飯是一個很平常的料理
    • 如同其他料理，在吃下的瞬間就能治療好疾病的確是太誇張，還需要醫生嗎真是

羊肉蔥湯編輯

  • 烹飪者：凱由
  • 特色：引起大汗
  • 秘密：蔥葉溫和，可用之散熱以為功；羊肉大熱，配以酒食，就能迷亂人心
  • 空想度：非空想菜色
    • 雖然現實可作出來，不過強烈建議不要食用
    • 吃下大熱食物發汗後臉上會很容易起挫瘡進而留下疤痕，所以讓身為女性的向恩吃下這道菜真的是一個很毒辣的懲罰

大爆回鍋肉編輯

  • 烹飪者：阿昴
  • 特色：主料的高麗菜在極致高溫的翻炒下獲得了脆爽口感，和回鍋豬肉入口即化脂肪味道融為一體
  • 秘密：將鍋子埋入爐灶加熱至整個通紅，再拿來炒菜
  • 空想度：強空想菜色
    • 炭火可以達到的最高溫度在1200℃左右，而埋於其中的鍋子因為沒法散熱，所以其最終溫度應該與炭火差不多
      • 這個溫度下不管是高麗菜還是豬肉，扔進去都會轉瞬間便變成焦炭，調味料也一樣
      • 漫畫中阿昴只墊了一條濕毛巾就徒手去抓這樣的鍋子，根本是不想要那隻手了
    • 扣除這點就是普通的回鍋肉

火焰烤乳豬編輯

  • 烹飪者：阿昴
  • 特色：
    • 色澤鮮美、味道香濃，外加彈性恰到好處的豬皮
    • 搭配牛油的油脂，讓烤乳豬擁有與眾不同的風味
  • 秘密：
    • 將乳豬包在牛裡面，然後把整個料理船給炸了
    • 因為牛肉可以控制火候，所以乳豬沒有烤焦
  • 空想度：非空想菜色
    • 即乾坤烤乳豬，技術好的人可以連外面的牛都不會變成黑炭，阿昴你果然未夠班

天雄炙青魚編輯

  • 烹飪者：亞刊、李可
  • 特色：
    • 將五、六尾140公分左右的大青魚劃圓甩開，在爆炎河中灸烤，並在同時淋上醬汁
    • 成品後的青魚口感細膩，用筷子一碰就會散開，且魚全身上下所有部位的灸烤程度完全一致
  • 秘密：
    • 利用離心力和魚身上的割痕令醬汁既深又快速的滲入魚身，並能同時甩開多餘的醬汁和泥沙
    • 靠超觸覺完美控制青魚各部位的灸烤時間
  • 空想度：強空想菜色
    • 正如阿昴所言，在大火下烤魚仍會有部分焦黑，沒法呈現漫畫敘述的均一性
    • 一般火場各處溫度不同是因為引發火勢的可燃物有很多（燃油、瓦斯、木製品、塑料等）的緣故，而造成爆炎河的可燃物就只有亞刊撒於江面的燃料這一種，所以整個爆炎河的溫度應該是均一的，根本沒有溫度的差異供超感覺感知
      • 加上火勢的範圍遠達大青魚的體積且不受控制，只要身處爆炎河中，不管怎樣精密調整青魚的入火角度也無法做到讓整條魚均勻受熱
    • 不做任何防護衝入如此猛烈火勢的話，姑且不論可以靠體溫熔斷鐵鏈的美漫超級英雄亞刊本人，作為料理協助人必不可少的李可顯然會和青魚一起被烤熟
    • 此外少許焦香焦苦本身就是灸烤類菜色的賣點，完全沒有燒焦的烤魚其實屬該歸類於烤失敗的範疇
    • 最後將長江變成天雄爆炎河會引來長江兩岸居民的抗議甚至會跟你拚命

七色月餅編輯

  • 烹飪者：朱七
  • 特色：用七種食材做成七種月餅
  • 秘密：餅皮跟餡料不同的染色搭配
  • 空想度：非空想菜色
    • 中秋時大家都可以買的到的常見餅

向日葵擔擔麵編輯

  • 烹飪者：朱七
  • 特色：以鴨掌排成向日葵形狀，搭配豆瓣醬作成川式口味
  • 秘密：將皮蛋和入麵糰，搭配鴨肉高湯以及柔軟又有彈性的鴨掌，再加入鴨胗及鴨心以整隻鴨子製作
  • 空想度：非空想菜色
    • 但排成向日葵狀還要寫字耗費時間，等排完麵又涼又爛，朱七你確定要端出來嗎？

常夏北京烤鴨編輯

  • 烹飪者：阿昴
  • 特色：山菜餡填鴨翅，製成餃子型
  • 秘密：調味的糖水加入了阿昴在廣東和梅麗共享的水果以掩蓋鴨肉腥味
  • 空想度：非空想菜色
    • 用粵菜的做法製作北京烤鴨沒問題嗎？[1]

不死鳥蛋編輯

  • 烹飪者：阿昴
  • 特色：連味覺白痴都讚不絕口的不滅之味
  • 秘密：以摻鹽的蛋白包裹鴨身，以老酒直接點火蒸熟
  • 空想度：非空想菜色
    • 和叫化子雞的作法類似，只是將泥土換成鹽酒而已
    • 不死鳥與中國人熟知的鳳凰其實不是同一種生物這點還是不要計較了

罈子烤肉編輯

  • 烹飪者：朱七
  • 特色：蒙古烤肉與肉湯香味的二重奏
  • 祕密：沒什麼秘密，只是善用自己的超嗅覺做菜而已
  • 空想度：非空想菜色
    • 連味道、吃起來的感覺都忠實重現了
    • 即使採取阿飛的建議煮一下也不會讓味道變好，炒過或烤過的菜泡到湯裡面會好吃才有鬼
    • 雖然製作過程沒什麼問題，但請不要只穿著布鞋就爬上蒙古烤肉的鐵板上烤肉
    • 蒙古烤肉其實與蒙古一點關係也沒有而是北京小鍋子烤肉的改良版，改良者為台灣知名相聲家吳兆南，出現時代也是現代

清蒸大河鰻編輯

  • 烹飪者：阿昴、蘭飛鴻
  • 特色：在清湯中游泳，如同白銀一般的龍肉
  • 秘密：以豬肉片吸收鰻魚的腥味，再以五重湯熬煮
  • 空想度：非空想菜色
    • 畢竟該被吐槽的不是料理，而是那條巨大如龍的食材

麻婆革命編輯

  • 烹飪者：蘭飛鴻
  • 特色：無比細膩的口感，可以用叉子像分食蛋糕一樣食用
  • 秘密：將豆腐乾切成頭髮粗細的細絲後重新擺成豆腐外形，再夾上藉助燕窩膠質而凝結成凍的麻婆醬料做成千層餅外形裝盤上桌
  • 空想度：超空想菜色
    • 豆腐乾真切成這種粗細根本就無法維持形狀了，更別說拿來擺造型
      • 雖然現實裡有一道菜「文思豆腐」可以做到近似的效果，但那用的是嫩豆腐，而且切好後必須放入水中才能保證豆腐絲不會碎掉，當然也沒法用豆腐絲擺造型
    • 即便做出來了這也是一道涼菜，作者自己提出的麻婆豆腐特色之一「燙」在哪裡？
    • 十九世紀末的清國人會知道蛋糕是什麼就有鬼了
    • 另外當時屬於民智未開的年代，直接提到革命這個名詞可不太妙啊…

麻婆維新編輯

  • 烹飪者：阿昴
  • 特色：春捲狀、讓麻婆豆腐可以帶著吃，等同於帶著走的麻婆蓋飯
  • 秘密：春捲皮是疊起來的米紙做的
  • 空想度：強空想菜色
    • 基本上這種春捲是吃不出米味的，因為都被油香蓋掉，要吃出跟飯一樣的感覺大概要用一本週刊的量才有可能辦到
    • 命名為「維新」的日本明治維新運動則剛剛開始，中國版的光緒新政則因為本作屬於架空年代而不必深究有無關連

玉衡麻婆豆腐編輯

  • 烹飪者：梅麗
  • 特點：極度火辣的口感，吃過後食用者會不由得發出「らりるれろ啊咿嗚耶喔」的連串讚嘆聲
  • 秘密：加入自家特別培育的極品川椒（花椒）使麻度上升
  • 空想度：非空想菜色
    • 即是最正宗加入花椒的麻婆豆腐
    • 就算可以照書中的說法靠常吃這道菜抵抗風濕的侵襲，但總吃的這麼辛辣腸胃病又鐵定跑不掉了

四郎拉麵編輯

  • 烹飪者：四郎
  • 特色：
    • 真是泡得漂亮，不過麵條本身煮得太爛了
    • 無巧不成書地順便把瓷器給賣出去了
    • 順便讓四郎把到妹
  • 秘密：湯裡加入大量的寶蓋菇（長裙竹蓀）
  • 空想度：非空想菜色
    • 畢竟四郎本身廚藝還不到家，基本上只要肯把錢花在材料上鐵定能比四郎作得更好

無影麵編輯

  • 烹飪者：解魯、密拉
  • 特色：麵條通體透明仍不失口感
  • 秘密：
    • 麵粉中加入甘藷，揉成麵糰後浸入水裡，再以優異的技術與集中力拉製而成
    • 密拉則是完美的複製了解魯的作法，再以香料調味更勝解魯的作品一籌
  • 空想度：超空想菜色
    • 小川你把粉條與麵給混在一起了
    • 麵粉的顆粒比澱粉粗，所以麵條拉的再薄也不可能有寬粉那種半透明視覺效果
    • 如果只是泡在清湯裡看不到也就算了，超越寬粉達到「夾出碗後不仔細看甚至看不見」的程度更是無稽之談

印度式青椒羊肉絲編輯

  • 烹飪者：雷恩、密拉
  • 特色：將炒過的青椒羊肉絲混合雞油蔬菜湯
  • 秘密：
    • 炒過的食物混在湯裡怎麼可能會好吃？即使用了香料也除不掉那黏答答的口感
    • 用來陰密拉用的障眼法
  • 空想度：非空想菜色
    • 同上述的罈子烤肉，無論是作法本身還是最後的味道都一點不空想

扒羊肝（羊肉什錦煲）編輯

  • 烹飪者：雷恩
  • 特色：將羊的內臟雕花，然後用香菜湯汁與調味料悶煮
  • 秘密：利用內臟的腱子肉的嚼勁、腦和脊髓的甘甜、眼、胃、腸的質樸順口和細心的調味
  • 空想度：非空想菜色
    • 同樣是作法、還有味道本身都沒有空想成份的一道菜
    • 但一些內臟要像書中般，以魔術障眼法的速度在他人眼前處理完畢並不讓人發覺就需要相當的技術

秦始皇炒麵編輯

  • 烹飪者：亞刊
  • 特色：在養魚池放火炒麵，成品後調味醬汁會被包裹在麵條中間
  • 秘密：將牛肉和入麵糰，包覆六種食材以表達秦始皇統一六國
  • 空想度：強空想菜色
    • 就算是切再碎的牛肉也沒辦法均勻和入麵糰，造成口感的缺陷
    • 因為燴料並不是直接做好再揉入麵條而是僅僅放了原料進去，所以在炒的過程容易滲出，牛肉麵條本身也會因此散架
    • 雖然這次是燒自己的魚池，但是即使最後贏了收拾殘局的人絕對會非常辛苦
      • 所以凱由是因為怕收拾麻煩才炸壩放水毀掉整個梁山泊的？

恆河炒麵編輯

  • 烹飪者：阿昴
  • 特色：每一口吃起來都不一樣，具有無限種味道的可能性
  • 秘密：每一條麵雕刻成不同的形狀，並搭上不同的材料
  • 空想度：強空想菜色
    • 基本上炒在一起只會讓味道混在一起，作出的成品毫無重點，連是否美味都成了問題
    • 況且不同形狀的麵條炒在一起，部分麵條可能會受熱不均，導致類似青椒肉絲的悲劇重演
    • 更大的問題是：明明主題是大陸英雄卻命名為恆河？

破魔八陣編輯

  • 烹飪者：凱由
  • 特色：操控人心的究極毒膳料理
  • 秘密：以農民曆背面記載[2]相剋的食材，藉由自身運氣改變其味，卻不改其本質
  • 空想度：超空想菜色
    • 支持整道菜成型的原理是「氣功」這種奇幻力量
    • 相剋食材被處理完後並不是就這麼直接端出來食用，而是依然需要烹飪，這期間會再加入諸如老酒之類的東西調味，這些東西難道不會攪亂食物的相性嗎？
    • 號稱以食物中的陰之氣來操縱人心，但是做出來的每一道料理也都用火處理過，熱乎乎的東西也能算作【陰】[3]？
      • 再說明明是黑暗料理，卻取名「破魔」？

母親太陽球編輯

  • 烹飪者：阿昴、梅麗
  • 特色：
    • 哀家好像吃了一口太陽耶
    • 特殊的球狀蛋包炒飯，口感蓬鬆
    • 恰好解破魔八陣的毒
  • 秘密：
    • 運用陽光與重力，在萬里長城上製作出的球狀炒飯
    • 因為吸取陽光精華，所以無巧不成書地成了剋破魔八陣毒的料理
  • 空想度：究極超空想菜色
    • 炒熟食材的鐵板熱度號稱僅靠太陽光直曬，這種激烈的日照度別說北京長城段所處的氣溫帶，就是熱帶沙漠正中心也不一定有；更別提清朝時地球還沒有溫室效應、城市熱島等會使氣溫騰高的異常氣象
      • 食品曬了陽光也不會加重陽氣，例如西瓜曬了大半天的陽光還是性寒
    • 這種料理理論上不會是一個完美的球型，因為含了許多空氣，且食材的密度大約等於水，所以最後作完形狀會跑掉
    • 阿昴好腳力，可以跟上自由落體的加速度[4]正常人的話當球體完成時大概才跑到梅麗的位置除非跟著食材一起滾下去
      • 即使真有天生神力追得上自由落體的太陽球，即使用了什麼黑科技讓食材固定成球形，依照漫畫中這麼龐大的太陽球是會砸傷人的，就算像是亞刊、解魯這種明顯有強健體魄的人也是一樣，但漫畫竟然若無其事地將太陽球接起來
    • 所以被評為中華一番中最終究極不可能做出來的料理，雖然食材上比破魔八陣可靠得多，但是製作手法與成型絕對是會讓破魔八陣與其他三洨料理黯然失色的天外奇想
      • 看到母親太陽球之後，突然覺得其實破魔八陣似乎是可以作出來的…

湯編輯

  • 烹飪者：阿昴
  • 特色：
    • 幫阿飛解除黑暗料理界的洗腦還有除刺青
      • 雷恩：能除刺青的話也給我一些啊
    • 動畫最終回的機械降神，沒有細講食材和調理方式，也沒有取名字
  • 秘密：使用傳說廚具的神秘力量
  • 空想度：超空想菜色
    • 同破魔八陣，料理的核心是超自然力量
    • 現實除刺青的原理是用雷射破壞色素粒子至身體能代謝的狀態，只要這道料理能在體內產生同等的能量就可以除刺青
