「物」[編輯]

ゆっくり/油庫里[編輯]

  • 來自ゆっくりしていってね!!!的AA東方角色，簡稱「ゆっくり」「油庫里」「饅饅來」
  • 即東方飛饅頭，長相滑稽的一頭身生物
  • 原型是「紅豆大福」所以可以吃
  • 活躍在使用Softalk與AviUtl等軟體所製作的"ゆっくり實況"中，使得一些人印象中ゆっくり的聲音就是Softalk所模擬出來的那樣，雖然也有人使用人型立繪代替ゆっくり
  • 除了饅頭外，還有其他從AA文字轉變來的類饅頭生(?)物
  • 隨著有關的創作增加，ゆっくり已自成一個體系，能夠在東方世界觀以外獨立存在
    • 按作品內容大致可以分為：
      • 作為東方二次創作的「東方系」，內容以幻想鄉為背景、和東方原作角色互動為主
      • 把ゆっくり從東方世界觀抽離的「生物系」，把ゆっくり視作現實存在的生物，幻想鄉和東方原作角色可以完全不出場；其中還可以細分為「愛」「虐」「制裁」「觀察」等分支
        • 在這種情況下，下述提及的ゆっくり會或多或少給予惡劣的性格，各種分支的創作也根據不同性格來延伸，其中靈夢種、魔理沙種和愛麗絲種中一般被視為大自然或都市裡廣泛分佈的種群，大多性格惡劣，而其他種類的ゆっくり則品性會更溫順一點；此外大部分ゆっくり都設定成智力低下的種群。
        • 生物系ゆっくり會牽扯到建立家庭以及繁殖的概念，若是家庭題材則有很多倫理內容，若繁殖題材則會有很多H內容[1]。
        • 「虐」題材的ゆっくり作品甚至會往獵奇發展。
      • 兩者都不是的「混沌系」，也就是上面提到的ゆっくり實況之類的，ゆっくり只是個符號，不用深究其世界觀設定為何
    • ゆっくり的設定因作品而異，不過仍然有一些公認的設定：
      • 本體是可以吃的饅頭，內餡因種類而異
      • 會呼叫「ゆっくりしていってね!!!」，以「能夠ゆっくり」為生存目標，非常愛吃甜食
      • 認為頭飾／帽子是重要的東西，一旦被拿走就立刻要求還回；部分二設中是分辨個體的重要標誌，若頭飾被拿走會立即被判定為不同個體甚至被驅趕。
      • 比⑨還要笨
  • 2022年遭到版權蟑螂登記商標，在ACG界掀起軒然大波；幸在各方抗議下告一段落

ヤゴコロ[2][編輯]

  • 代表永琳自身的詭異吉祥物
  • 原本是2ch一個討論版的AA
  • 把它戴上永琳的帽子以後，就被拿來當作二創的一個惡搞角色

すくすく白沢/白澤球[3][編輯]

  • 一種毛絨絨的生物
  • 由upg氏創造，代表慧音（白澤狀態）的吉祥物
  • 衍生出來的其他物種通稱すくすく(角色名)

「其它」[編輯]

  • 東方LIVE A LIVE
  • ゆっくり板
  • 幻想少女大戰
    • 推出四代的超級機器人大戰風格遊戲，還有原創角色以及舊作人物出場，普遍獲得好評
    • 不同代數可接關，2020年初推出四部合輯

「動畫」[編輯]

  • 東方M-1漫才（2007-）
    • R-note製作的同人動畫，現已推出15話
    • 模仿吉本興業M-1漫才比賽的動畫，主打多種類的捏他與改編曲
  • 夢想夏鄉（2008-）
    • 舞風製作的同人動畫，現已推出4+1話
    • 以知名聲優，品質優秀為特長但是工期也甚長幻想鄉奧運
  • 幻想萬華鏡（2011-2021）
    • 滿福神社製作的同人動畫，全16話，還有推出藍光版
    • 雖然沒有聘請職業聲優，然而作畫等級以及配音都獲得肯定
    • 主題為本篇主角等人解決異變的經歷，結尾總會為下一次主題埋伏筆
    • 礙於經費收尾在《風神錄》的妖怪之山決戰
  • 秘封活動記錄（2015-）
    • 京都幻想劇團製作的同人動畫，現已推出2話
    • 以秘封組的活動為主題，描寫幻想鄉眾人幻想入之前的經歷
