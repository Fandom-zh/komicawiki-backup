比那名居天子

比那名居天子/ひななゐ　てんし
本條目內容不足需要擴充，歡迎有興趣的朋友加以補完。
本條目可能含有部分劇情內容，沒看過原著又不想遇到地震的話請不要繼續看下去。

目錄

解說編輯

  • 東方Project重要角色，俗稱M子，二次創作最大被害擔當之一
  • 本名比那名居地子，在經吼陀選拔成為天人後改名為天子

基本資料編輯

  • 種族：天人
  • 職業：NEET無
  • 住所：天界
  • 能力：操縱大地程度的能力
  • 聲優：
    • 二次創作遊戲《不可思議的幻想鄉》：榊原ゆい
    • 東方CB：雨宮天和某人的代表色一樣

稱號編輯

  • 非想非非想天の娘 （緋想天、非想天則）
  • 有頂天のお嬢ちゃん （ダブルスポイラー）
    • 佛教天界中三界二十八天最上層的位置別名「有頂天」
  • 有頂天のお嬢ちゃん（文花帖DS）
  • 緋想的で浮世離れした天人（憑依華）
  • 暖衣飽食の天人（茨歌仙）

簡介編輯

  • 有如溫室的花朵，行為我行我素，情緒高昂
  • 會成為天人是因為她族人侍奉的名居守一族成為了神靈，所以和族人因功到天界居住
  • 會對人做個性診斷（在他人眼中是天氣預報），從而引發氣質發現（天氣變化）
  • 由於沒有天人的品性，所以看到地上的妖怪和人類引發騷動並作樂感到很羨慕
  • 對自己有絕對的自信，實際上也相當的強，在與妖怪們交手時皆有留手，能力被人瞧不起就會很激動
  • 延繼先祖的職業，具有控制要石達成鎮壓或引發地震的能力
  • 手持天人才能使用的武器「GN-緋想之剣（終極黃金斬）」，能精準洞察他人的弱點並具現作紅色的霧狀氣體「天氣」

萌點編輯

  • 不良
    • 如上所述，因為進入天界的緣由是一人得道雞犬升天，修為不如其他天人，此外也沒有經過修行，所以沒有天人應有的水準
    • 自己也不滿天界環境，這成了在幻想鄉掀起異變的背景之一
  • 搞事
    • 因為太無聊所以利用能力興風作浪，射命丸文還見過他以性格診斷為由改變天氣
  • 受
    • 此二創屬性的來由有三：一是在緋想天中引起異變的原因只是因為無聊想要有人來推翻她；二是在The Grimoire of Marisa中，魔理沙對気符「無念無想の境地」的評價是M度五顆星，三是渴求接近性格殘酷的風見幽香
  • 開無雙

主題曲編輯

  • 有頂天変 ～ Wonderful Heaven(緋想天より、憑依華)
  • 幼心地の有頂天(緋想天ラストスペルのテーマ)

符卡編輯

  • 緋想天(自機)
    • 地符「不譲土壌の剣」
      出自李秦國宰相斯的「諫逐客書」，規勸秦王贏政不能隨便驅逐他國的賢者：「泰山不讓土壤，故能成其大，河海不擇細流，故能就其深，王者不卻眾庶，故能明其德。」
    • 非想「非想非非想の剣」
    • 天符「天道是非の剣」
      典出司馬遷在「史記．列傳」開頭反駁「天道無親，常與善人」的說法，並在列舉了一些善人滅亡、惡人昌盛的史實之後，提出了這樣有名的疑問：「余甚感焉，倘所謂天道，是邪非邪？」是司馬遷經歷種種不測後，對天地正義的一種質問。不過在道家裡面，天道指的是自然的道理，所以道家認為天道本無是非，如同人無所謂善惡。
    • 地震「先憂後楽の剣」
      典出北宋．范仲淹「岳陽樓記」：「先天下之憂而憂，後天下之樂而樂歟」，呈現出一種古代士大夫重負大任的胸襟。
    • 気符「天啟気象の剣」
    • 要石「天地開闢ブレス」天地開闢 Press：天地開闢重壓
      天啟：天的啟示，觀天以知雲雨之意。幾乎世界各地的創世神話一開始，都是一整個混沌，接著有神將其分為天地兩者；這就是天地開闢。
    • 気符「無念無想の境地」
    • 「全人類の緋想天」
  • 緋想天(敵機)
    • 要石「天空の霊石」
      要石為日本用來鎮壓地震、埋入地中的靈石，茨城的鹿島神宮跟千葉的香取神宮都有。
    • 乾坤「荒々しくも母なる大地よ」荒蕪母親般的大地
      八卦中乾為天坤為地，乾坤則衍申為指代宇宙、全世界。將孕育作物的大地看做母親，是原始人類都有的信仰，就是地母信仰或蓋亞信仰；英文是「Mother Earth」，拉丁語是「Terra Mater」。
    • 霊想「大地を鎮める石」鎮壓大地之石
      承接上方要石的解說，鹿島神宮祭祀的武甕槌大神（通稱鹿島樣），被奉為武神、劍神，而要石是一把長石劍；日本人相信地震是大鯰魚引起的，而鹿島樣用要石，將鯰的頭插在鹿島神宮、尾巴插在香取神宮，就這樣鎮住了大地震。
      題外話，艦隊Collection唯二實裝的練習巡洋艦皆由此得名。
    • 天地「世界を見下ろす遙かなる大地よ」俯瞰世界的遙遠大地
    • 「全人類の緋想天
  • 非想天則(新增)
    • 剣技「気炎萬丈の剣」
    • 天気「緋想天促」
  • ダブルスポイラー
    • 気性「勇気凛々の剣」勇氣凜凜之劍
    • 地震「避難険路」
    • 要石「カナメファンネル」Kaname FangFunnel
      Funnel原意為漏斗，後來在機動戰士鋼彈系列指稱小型腦波傳導兵器感應砲
    • 「全人類の緋想天」
