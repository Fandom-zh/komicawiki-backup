AQUARION

創聖のアクエリオン / Genesis of Aquarion / 創聖機械天使[1] / 創聖的亞庫艾里翁[2] / 創聖大天使[3] / 水神世紀[4]
本條目是一個編輯中的詞條，歡迎有興趣的朋友加以補完。
以下含有部分的劇情，沒看過原著又不想被無限拳打上月球或是和狗交合合體的話就請不要繼續看下去。


目錄

概要編輯

  • 河森正治繼《超時空要塞》系列後推出的機器人動畫作品。
    • 超級系マクロス
  • 系列皆以合體作為主軸。
    • 在機戰Z中常與合體系統相似的蓋特相提並論。

系列作編輯

TV版編輯

  • 《創聖のアクエリオン》(以下稱無印)
    • 2005年4月4日～2005年9月26日播放，全26話。
    • 描寫12000年間天翅與人類之間的愛恨情仇。
    • 同名主題曲阿姨洗鐵路由AKINO演唱。
  • 《アクエリオンEVOL》(以下稱EVOL)
    • 2012年1月8日～2012年6月24日播放，全26話。
    • 以無印結束後12000年的世界為主軸。
    • 主題曲依然由AKINO演唱，前作主題曲則以插曲形式呈現。
  • 《アクエリオンロゴス》(以下稱LOGOS)
    • 2015年7月3日～2015年12月25日播放，全26話。
    • 雖然有在劇中出現EVOL的手辦與創勢的片段，但仍未清楚表明兩者關聯。
    • 本作主題曲改由May'n演唱。

OVA編輯

  • 《創星のアクエリオン》(以下稱創星)
    • 無印的平行世界故事。12000年前的歷史與創聖相同。
    • 分為前篇《裏切りの翼（背叛之翼）》及後篇《太陽の翼（太陽之翼）》。
    • 與壹發逆轉篇一同於劇場上映。
  • 《壱発逆転篇（壹發逆轉篇）》
    • 劇場版短篇。
  • 《創勢のアクエリオンEVOL》(以下稱創勢)
    • 在10周年紀念活動及LOGOS上映前播放，有無印及EVOL角色跨時空共演和合體。

登場機體（無印）編輯

Vector編輯

  • Vector SOL
    • 增強「氣」與「特異能力」的戰機。
  • Vector LUNA
    • 增強「感受性」與「回復力」的戰機，通常由女性元素人搭乘。
  • Vector MARS
    • 增強「智力」、「指揮能力」與「思考力」的戰機。

強攻型Vector編輯

  • Vector ALPHA
    • 新國聯軍開發的Vector。
  • Vector DELTA
    • 新國聯軍開發的Vector。
  • Vector OMEGA
    • 新國聯軍開發的Vector。

Aquarion編輯

  • SOLAR Aquarion
    • 以Vector SOL為主體的形態。
    • 重視格鬥能力的形態，展開羽翼時可引出其真正的力量。
    • 最接近原本的Aquarion的形態。
  • Aquarion LUNA
    • 以Vector LUNA為主體的形態。
    • 以弓箭為主要武器。
  • Aquarion MARS
    • 以Vector MARS為主體的形態。
    • 擅長劍技與足技的形態。

強攻型Aquarion編輯

  • Aquarion ALPHA
    • 以Vector ALPHA為主體的形態。
  • Aquarion DELTA
    • 以Vector DELTA為主體的形態。
  • Aquarion OMEGA
    • 以Vector OMEGA為主體的形態。

特殊合體編輯

  • Aquarion SOLUNA
    • SOLAR Aquarion失去雙臂的情況下，將背部的Vector LUNA改為頭部的特殊形態。
    • 同時有SOLAR Aquarion的力量和Aquarion LUNA的能力，可以使用拳術和弓箭。
      • 由於是以機體受到破壞的前提下的合體，只登場過一次。
    • 命名者是紅麗花。
  • Aquarion ANGEL
    • 席魯斯奪取Vector MARS後，以Vector OMEGA代替Vector MARS作為主體的形態。
  • 強攻型Aquarion．Armageddon形態
    • 由Aquarion OMEGA變形而成的六足步行形態。
  • 強攻型Aquarion．Assault Walker形態
    • 由Aquarion ALPHA變形而成的飛行形態。機體下方有腳。
      • 外觀像Macross系列的Gerwalk。
  • 強攻型Aquarion恐龍形態
    • 在創星中登場。外形似恐龍的四腳形態。
  • Aquarion四足形態
    • 在創勢中登場。合體模式和Aquarion恐龍形態相同。
      • 比起恐龍，這邊的合體比較像狗？
    • 沒有正式名字。

聖天使學園男子宿舍編輯

  • 阿瑪塔‧空（アマタ・ソラ）（CV：梶裕貴）
    • 男主角。
    • 元素能力為「重力干涉」，當高潮情緒激昂時會不由自主的飄起。
      • 其實是六神合體
    • 阿波羅12000年後的轉世。
    • 曾被未影一分為二，現在的阿瑪塔是弱氣的一面，並沒有阿波羅的記憶。
  • 凱炎‧鈴白（カイエン・スズシロ）（CV：鈴村健一）
    • 御子乃的兄長。
    • 元素能力為「絕望預知」，該能力會將未來發生的壞事以影像呈現。
  • 安迪‧W‧霍爾（アンディ・W・ホール）（CV：鳥海浩輔）
    • 喜好挖洞的樂天青年。
    • 元素能力為「穴掘力」，可以在多種物質或介質間挖洞。
  • 修雷特‧艾蘭（シュレード・エラン）（CV：浪川大輔）
    • 罹患絕症的鬼畜眼鏡美青年。
    • 元素能力為「精神演奏」，可以利用演奏進行精神干涉。
    • 被神樂指出有和御子乃相似的氣味。
    • 為了讓同伴突破次元狹間，使用自身全部的力量發動「神融奏」，最後在「雙星之書」上用血寫下自己創作的旋律後消逝在次元狹間。
      • Z3時獄篇在進入IF路線後可救活，天獄篇則無條件吐便當。
  • 諸井‧多里薩（モロイ・ドレッツァ）（CV：佐藤健輔）
    • 南國出身的青年。
    • 元素能力為「脆弱力」，可藉由破壞分子間鍵結將該物質脆化。

共通設定編輯

  • Aquarion
    • 作中登場的機器人。
    • 可以合體成Aquarion的戰機被稱為Vector。
    • 無印登場的Aquarion是本篇24000年前由墮天翅族製造的巨型機器人，本身卻極難操控。之後12000年來被稱為「禁斷之翼」並存放於亞特蘭提斯地下，只有最強的墮天翅有能力使用。
      • 值得一提是本來的Aquarion是不具備變形合體功能的獨立機器人，被阿波羅尼亞斯偷走的只是其中一台。後來被改造成變形合體機器人。後世稱之為神話型Aquarion。
      • 在EVOL結局及街機版中登場的古代形態就是Aquarion本來的形態。
      • 阿波羅尼亞斯死後，其羽翼被埋葬在核心水晶內，並存放在Aquarion中。
    • 無印中人類側開發的Aquarion名為強攻型Aquarion。
    • EVOL登場的Aquarion是強攻型的後繼機。
      • 雖然同樣是量產機，強攻型是大量生產，EVOL型只有兩組Vector和一組後備。
    • LOGOS中的Vector機是NESTA基於創聲之書記載的神話而開發。
      • 劇中登場的只有餓號機是神話中的機體。
  • 無限拳（MUGEN PUNCH／MUGEN KEN）
    • 貫穿三部作品的必殺技及衍生技能，基本上由主角發動。
    • 正如其「無限」之名，發動時機體的手臂會無限延長並追擊對手然後打向月亮。
      • EVOL初發動時沒打月亮，反而打穿分隔男女的柏林圍牆。月亮逃過一劫
      • LOGOS中的無限拳讀音不同，使出的無限拳也不是無限延伸而是無限量的拳。
  • Element
    • 通稱元素人。Vector的駕駛員。
    • 在創聖中的大異變令磁場變化，使一些年輕人得到了特殊能力。這些能力可以在Aquarion上增幅。
      • 在創星中沒有發生大異變，所以元素人也沒有特殊能力。
    • 在無印和EVOL中可以用Element Change以瞬間移動（Teleport）的形式遠距離交換Vector的Element。傳回DEAVA的Element會全裸送回去，戰鬥服留在駕駛倉，而傳送到Vector的Element會自動穿上原來的戰鬥服。
      • 由於阿波羅不會穿戰鬥服，所以不能被傳送回去司令部。
      • EVOL中的傳送可以在合體後使用，傳送後也不會全裸。
    • LOGOS中的駕駛員為創聲Element，使用能力是創聲力而非超能力。由於駕駛員與Vector的組合是固定，所以也不會有Element Change。
  • 合體
    • 透過合體，Vector會合體成Aquarion。
    • Element之間的合體是身體和心靈都會合而為一，所以合體時的精神狀態十分重要。
      • 合體後Element能以類似「靈魂」的形式轉移到主Element的駕駛倉，也可以協助操作。
    • 合體的時候會產生快感和精神恍惚。絕對不是高(ry
      • 有機會成癮。
      • 不過EVOL特典CD反而出現過對合體厭倦的情況。
    • LOGOS中的合體並沒有身心合一，Element也是物理地轉移到合體模式中的駕駛倉。
      • 所以對精神的要求並不重要。但不知道為甚麼同樣有合體時的快感。
  • DEAVA
    • 司令部。
    • 無印中是對抗墮天翅族而成立的機構，全名Division of EArth Vitalization Advancement（地球再生機構）。
      • EVOL中改名Neo DEAVA，沿用無印中的基地。
    • LOGOS中同樣有DEAVA，但名字的意思是Division of EArth Verbalism Ability（世界創聲力機構）。
  • 轉世
    • 本作中的轉世和一般人的理解差不多。比較特別的是一個人可以有多個前世、多個來世。轉世關係並不是一對一的。
      • 據河森所說，EVOL的御子乃、不動也不是完全等同於他們的前世。
    • 雖然故事是以12000年為輪迴單位，但似乎並非12000後才能轉世一次。只是劇中的事件都相隔12000年。
  • OO之書 / ソウセイの書
    • 記載過去歷史以至預言未來的神秘書本。
    • 一共有7本。
      • 最初是在河森訪談提到，後來在創勢的Aquarion EVOL中明確說明自神話的大戰後世界分裂成7個。
      • 每一本書也是讀作ソウセイの書(sousei no sho)，漢字寫法不同。河森你可以考慮一下日本以外觀眾的感受嗎？
    • 目前以動畫形式登場的有「創聖」、「創星」、「雙星」、「創勢」、「創聲」。
      • 按照這個說法，EVOL有可能是處於無印以外的世界，但有和無印相同或相似的歷史。
      • 暫時也不知道LOGOS的「創聲」和其他書/世界的關係。創聲之書聲稱是由真理族人以劍嵜總的前世桐生的身體製造。
    • 如果河森想用動畫以外的作品補完世界觀，目前也有一些候補。不考慮機戰的話目前最少已經有7個世界登場。河森你寫的世界觀會不會大了一點？
      • CR版(柏青哥)是其中之一，因為Aquarion在遊戲中可以還原成「創聖」沒有登場的古代形態。漢字不明。
      • 無印漫畫《虛空的天翅》是本篇的平行世界，漢字是「創世」。
      • 超級機器人大戰中出現過「蒼星」、「總星」，但不太可能是本篇的7本書。

  • 無印及EVOL的歌大部份由AKINO及bless4[7] 主唱。
