解說編輯

登場人物編輯

主要角色介紹編輯

  • 門矢士 / 假面騎士Decade （井上正大 飾）
    • 本作的男主角，詳見主條目
  • 光夏海 / 假面騎士Kivara（森カンナ飾）
    • 本作女主角，詳見主條目
  • 光榮次郎（石橋蓮司飾）
    • 夏海的祖父，攝影館「光寫真館」的店主
    • 真正身分不明，幾乎每次都是因為他讓照像館跑到另外一個世界去。
    • 在電影版完結篇中指榮次郎其實是被姆斯卡大佐 園咲琉兵衛 植入死神博士的記憶體才會變成死神博士，但本人全不知情。
      • 所以即使老頭也不要和古怪的爺爺去喝兩杯哦！
  • 海東大樹 / 假面騎士Diend（戶谷公人飾）
    • 本作的第二女男主角，詳見主條目
  • 小野寺雄介 / 假面騎士空氣我（村井良大飾）
    • 本作的第三名主角，平行世界的五代雄介，詳見主條目
  • 鳴瀧 / 假面騎士響鬼（奧田達士 飾）
    • 以阻止和殺死Decade為目的神祕男人，可以召喚其他世界的騎士，詳細資料仍然未明
      • 在電影版2中已說明其真正身份是大修卡的佐爾大佐
    • 稱Decade為「破壞世界的惡魔」，但Decade可以令其他騎士骨折變型，某方面來說的確是惡魔
    • 因被拍攝到裡面的人偷偷去看《フレッシュプリキュア！》的舞台劇，這是真的，而在《ネット版 仮面ライダーディケイド オールライダー超スピンオフ》的預告中偷學夢原望美，因此被眾人視作「Pretty Cure 控」
      • 其實他的真正目的就是阻止Decade破壞Pretty Cure世界
      • 在《フレッシュプリキュア！》第30話終於如願以常到達Pretty Cure的世界(誤
    • 第30話中終於開口請求Decade（士）拯救世界，所以是傲嬌
    • 在結局篇劇場版中，亦擔任佐魯大佐角色，然後不敵時就撤退。
    • 在《假面騎士x超級戰隊 超級英雄大戰》中鳴瀧的身份變為大修卡的大干部——G博士，與大殘格古一同實施巨大的計畫，企圖將雙方的要塞融合成巨大的邪惡機器人。試圖利用門矢士，但反而中計，在騎士的圍攻中變身為蟹激光，通過使用強大的激光攻擊令騎士與戰隊們陷入苦戰，最後被得到天裝戰隊護星者力量的假面騎士Decade、假面騎士龍騎與假面騎士劍擊敗。蟹激光爆炸後顯示出鳴瀧的原貌，宣言只要門矢士還存在他的旅途就不會結束，並表示還會再來對付士。
      • 接連在《スーパーヒーロー大戦》和《昭和VS平成》登場，於後者劇場版見到兩派騎士和解後，放懷大笑（疑似是解脫了？）但S.I.C還是沒放過他
  • キバーラ（CV：澤城美雪）
    • 鳴瀧派去監視門矢士的迷你蝙蝠怪獸，キバット3世的妹妹。
      • 雖然如此，但他正逐漸被士等人同化中……
      • 偶爾是電腦CG，偶爾是實體玩具。
    • 本作品唯一的萌角（無誤）
    • 可惜被她咬是不能變成kiva的。（亦無誤）
      • 第31集將雄介變成了黑眼究極空我。（同雄介部分，31集特別版刪除該幕）
      • 而在結局篇劇場版中，夏海可以使用她變身成假面騎士Kivara，但是變身過程中沒有被她咬過
    • 真紅
  • 鈴村健一
    • 在一系列的假面騎士大百科節目中和奧田達士一起當司儀。
      • 由於是騎士迷，所以主持節目時往往比奧田達士還要情緒高漲。

夏海的世界編輯

  • 根據第一話的場景，該世界為一個龍騎的平行世界
  • 紅渡 / 假面騎士Kiva（瀨戶康史 飾）(Kiva)
    • 本作原創[1] 假面騎士Kiva的主角
    • 第1話引導士去拯救所有的騎士世界，最終戰中也會出來，變身成Kiva和劍崎Blade一起攻擊DCD。真是成也Kiva，敗也Kiva

電王的世界編輯

  • 野上 良太郎 (溝口琢矢 飾)[2]
    • 電王的主角，雖然和電王的角色是同一個人，但卻因為時間異變而成為小孩。
  • 野上 幸太郎 (櫻田通 飾)
    • 電王劇場版3的主角，是本作中少數幾個(不算聲音演出)全部由原演員飾演且設定相同的角色。
  • 桃塔羅斯（CV 關俊彥）
    • 因為時空異變而不能實體化，失去了「自我」
    • 初出現時附在士身上變成電王，後來附身在雄介身上。
    • 在超劇場版中附在士身上變成DCD
    • 使用FAR時『アッー！！』了雄介(空我)。(無誤)
    • 由於不像其他騎士需要找回本人來演，所以不論是劇場版一還是二都有大量對白和戲份，聲音更明顯比其他騎士大。
  • 浦塔羅斯（CV 遊佐浩二）
    • 附到士身上。
    • 附到夏海身上(服裝為OL制服)，終於變成了女人
    • 變身成電王Rod Form和DCD戰鬥。
  • 金塔羅斯（CV 寺杣昌紀）
    • 附到士身上。
    • 附到夏海身上(服裝為和服)
    • 變身成電王Ax Form和DCD戰鬥。
  • 龍塔羅斯（CV 鈴村健一）
    • 附到士身上。(極大違和感)
    • 附到夏海身上(服裝為可愛的服飾(!?))
    • 變身成電王Gun Form和DCD戰鬥。
  • 齊格（CV 三木真一郎）
    • 於電王世界完結後和KABUTO世界開始前之間的時空出現，本人好像也不知道自己怎麼出現了。
    • 超劇場版中附到士身上。
  • 車長 (石丸謙二郎 飾)
    • 時空列車Den-Liner的車主。
    • 喜歡吃炒飯時維持飯糰中央的牙籤紙旗屹立不倒。
    • 新一代亂入王
  • 直美 (秋山莉奈 飾)
    • Den-Liner的餐車服務員，會沖出非常有特色的咖啡。
    • 擁有一個引人犯罪的美臀
  • 小花(Hana) (松元環季 飾)
    • 因為時間異變而變成小孩子。
    • Den-Liner中最強的人，擁有一拳KO異魔神的實力，甚至可以在車上以火箭炮攻擊敵人，因此在觀眾之間得到『小花女王』的外號。
    • 中華聯邦天子[3]
      • シンクー！シンクー！
    • 健次一黨的成員[4]

響鬼的世界編輯

  • 斬鬼（松田賢二 飾）
    • 少數和原作相同的演員飾演，而除了響鬼和明日夢之外其他角色和原作相同。
      • 同時也是平成騎士系列中演出次數最多的演員[5]
    • 本作中為斬鬼流的宗師。
    • 露體狂
    • Kiva的後宮其之一[6]
  • 轟鬼（川口真五 飾）
    • 斬鬼的徒弟，後來繼承了斬鬼成為宗師。
    • 純情派笨蛋。
    • 在騎士大戰中被SAGA擊殺，消失。
  • 威吹鬼 （涉谷讓二 飾）
    • 威吹鬼流的宗師。
    • 燕尾服假面[7]
    • 不知為甚麼弟子全是女的
  • 天鬼（秋山奈奈 飾）
    • DCD中第一位出現的女騎士
    • 原名天美明，亦是威吹鬼的徒兒，但原作是只變身一次就失敗，最後放棄鬼之修行離開。因此「天鬼」這名字在Decade中是為第一次出現。
    • 本作中後期繼承了威吹鬼成為宗師。
    • 在騎士大戰中被SAGA擊殺，消失。(SAGA你這個不懂憐香惜玉的混帳！)
    • 絕大多數觀眾對沒有天鬼解除變身的場面十分不滿

負片世界編輯

  • 紅音也 / 假面騎士黑Kiva（武田航平 飾）
    • 假面騎士Kiva另一主角，紅渡的父親，
    • 在暗影世界（21，22）中擔任首領黑Kiva，但當看到士得到新型PSPK-Touch發現不敵後逃掉。
    • 騎士史上最強奸夫
    • 不論是原作還是DCD他的戲份和風頭都要比兒子強，甚至比kiva世界的正太版渡強，果然是爸爸比你好。
    • 擁有五條腰帶的男人

Diend世界編輯

    • DCD中另一名女騎士

侍戰隊世界編輯

  • 志葉丈瑠 / 真劍紅(松坂桃李 飾/台灣配音：李世揚(本作)/吳文民(原作))
    • 第一人稱「俺」，使用烈火大斬刀，但也可以用被FFR過的BLADE當武器。
    • 志葉家19代當家，真正18代當家志葉薰的影武者。
    • 沒有任何專用道具，所有的強化道具都是全體真劍者通用。
    • 外冷內熱的悶騷型。
  • 池波流之介 / 真劍藍 (相葉弘樹 飾/台灣配音：李景唐(本作)/陳余寬(原作))
    • 第一人稱「わし」，使用水之弓。
    • 歌舞伎世家出身，舞蹈與劍法都是真劍者中最強的。
    • 嘴巴上說丈瑠不是真當家而不打算去救他，最後還是去了。
  • 白石茉子 / 真劍桃 (高梨臨 飾/台灣配音：錢欣郁(本作)/詹雅菁(原作))
    • 第一人稱「私」，使用天堂扇。
    • 書法專長（本人與劇中皆是(高梨臨的書法有五段)），隊中僅次於殿下最擅長使用文字力的人。
    • 人妻志願，不過廚藝則還遠遠不到可以當人妻的水準。
  • 谷千明 / 真劍綠 (鈴木勝吾 飾/台灣配音：何志威(本作)/宋克軍(原作))
    • 第一人稱「俺」，使用木之槍。
    • 侍戰隊中唯一的學生，剛從高中畢業準備考大學。
    • 非常的隨性，對於侍戰隊中的武士傳統非常不滿，認為現代不應該在有君臣關係，因此是隊伍中少數不叫丈瑠殿下的人。
    • 極端搶眼，戰鬥時候經常擺出和其他隊員格格不入的動作，但也令他的討論度頗高。
    • 第27集的進階型態變身「超級真劍綠」是目前所有超級真劍者中必殺技最帥的一個。
    • 其實很關心殿下但又故做叛逆。
  • 花織琴葉 / 真劍黃 (森田涼花 飾/台灣配音：林美秀(本作)/龍顯蕙(原作))
    • 第一人稱「內」，使用土之迴旋鏢。
    • 是六人中最年幼的一位，往往也是代替體弱多病的姊姊加入侍戰隊而來的。
    • 具備柔道專長，將來可能會繼承轟轟之冒險黃的怪力女特色。
  • 梅盛源太 / 真劍金 (相馬圭祐 飾/台灣配音：梁興昌(本作)/曹冀魯(原作))
    • 第一人稱「俺」，使用魚之丸。
    • 丈瑠的兒時玩伴，原先是位壽司師傅。
    • 在原作第20集中被海東大樹偷去了烏賊折神。
    • 把海老藏(其實是海老折神(蝦折神)，但被取名成這個名字，另外，市川海老藏是有名的歌舞伎演員)養在自己的水族箱裡頭。

    • 在劇中由門矢士Cosplay登場。
    • 當年由於Black的故事太灰暗，所以續篇加給光太郎的同伴，在RX世界中調查古萊斯帝國時失踪。

各個騎士世界中非原班角色介紹編輯

空我的世界編輯

  • 八代 藍 (佐籐寬子 飾)
    • Kuuga世界中的警察，知道雄介是未確認生物體4號=空我並協助他，相當於原作中的一條薰。
    • 使用iPhone
    • 跟八雲藍沒有關係。
    • 除了車頭燈以外戰鬥力比一條薰明顯遜色甚多。

KIVA的世界編輯

  • 渡 / 假面騎士Kiva（深澤嵐 飾）
    • 本作原創的KIVA世界主角。[13]
    • 正太一名，雖然是FANGIRE的「王」，卻對於自己的混血血統感到憂慮，認為FANGIRE和人類共存的可能性極低。
    • 和KIVA本傳的紅渡不一樣，身為FANGIRE的是父親不是母親。
    • 於騎士大戰世界跟劍立一真戰鬥，最後因為和KIVA世界一起消失
    • 消失後靈魂轉世為漢堂將

龍騎的世界編輯

  • 辰巳 真司 / 假面騎士龍騎 (水谷百輔 飾)
    • 龍騎世界的主角。起初誤會蓮背叛了自己和桃井社長，最後冰釋前嫌。
  • 羽黑 蓮 / 假面騎士夜騎 (北村榮基 飾)
    • 跟真司曾經是ATASHI雜誌社的最強組合，但因為妒忌真司的才能，桃井社長希望他能出去冷靜一下，在桃井被殺後以夜騎的身份參與騎士裁判，目的是要取得TIME VENT找出真正的兇手。
  • 鎌田 / 假面騎士Abyss (入江雅人 飾)
    • ATASHI雜誌社的副編輯長。真正身份為劍世界的紅心K的Undead。[14]，大企業Board的理事長。
    • 於OP曲的PV中亂入
  • 桃井 玲子 (高橋佐衣 飾)
    • 「ATASHI雜誌」的總編輯。

BLADE的世界編輯

  • 劍立 一真 / 假面騎士BLADE (鈴木擴樹 飾)
    • BLADE世界的主角。後來為了救睦月而無視作戰，由黑桃A被降級到黑桃7成為廚房雜務員，後來因為運餐時發生意外再降級至黑桃2，被要求取去劍的變身資格。
    • 後來得到士的鼓勵(?)重新振作，和士一起打敗了Joker和鐮田。
    • 不會說オンドゥル語
    • 在騎士大戰的世界中，雖然擊敗了與大修卡聯手的Fangire和Undead，卻因為阿波羅蓋斯特的操作隨著Blade的世界一同消失。
  • 四條 始 / 假面騎士Chalice (累央 飾)
    • Board的社長，和鐮田合作企圖支配劍世界。
  • 菱形 朔也 / 假面騎士Garren (成松慶彥 飾)
    • 與一真和睦月同為Board的員工。
    • 在騎士大戰的世界中跟著Blade的世界一同消失。
  • 黑葉 睦月 / 假面騎士Leangle (川原一馬 飾)
    • 在一真被降級後成為假面騎士Leangle的變身者，與一真和朔也同為Board的員工。
    • 在騎士大戰中被渡變身的Kiva消滅。

AGITO的世界編輯

  • 蘆河 翔一 / 假面騎士Exceed Gills / 假面騎士AGITΩ (山中聰 飾)
    • 本是G3的測試員，但因為自己身體發生了異變（變成Gills），在害怕自己的異變會傷害到淘子的情況下，選擇逃離並一直躲著淘子。但最後終於成功由Gills進化成為AGITΩ，回到淘子身邊。
      • GILLS在原作中不會進化成AGITΩ，而是AGITΩ的亞種。
    • 煌鬼
  • 八代 淘子 (佐藤寬子 飾)
    • 外貌與空我世界的八代藍一樣，但兩人存在於不同的世界。喜歡翔一。
    • 相當於AGITΩ的小澤澄子。
    • G3、G3-X及G4系統的創作者。
    • 在海東打算偷走G4的控制晶片時，自己動手毀掉晶片。

Kabuto的世界編輯

  • 總司 / 假面騎士Kabuto (川岡大次郎 飾)
    • 因為Clock Up系統失控而被困於Clock Up世界中。
    • 跟原作一樣是個妹控，但比原作來的有擔當(或許是因為模樣的關係？)
  • 弟切 想 / 假面騎士TheBee(川岡大次郎 飾)
    • ZECT的隊長，真身是Phylloxera Worm[16]。
    • 擬態總司後差點被變身為Kabuto的正牌總司所殺，且交戰時造成KABUTO的Clock up系統暴走
    • 後來利用ZECT開發Clock Down系統向Kabuto報復，並企圖支配世界。
  • 真由 (菅野莉央 飾)
    • 總司的妹妹，真身是囧臉Sisyra Worm。
    • 目擊Kabuto重傷冒牌總司的一刻，因此誤認Kabuto為殺兄仇人。
    • 目前為止唯一一位認為士的照片拍得很好的人
      • 而且更穿上不同服裝(COSPLAY?)當士的模特兒
  • 傳說中的奶奶(佐々木すみ江 飾)
    • 總司和真由的奶奶，與真由一起經營著家族的食店「天堂屋」，疑似除了黑輪外啥也不賣。
    • 不少島民都相信這位奶奶能夠單手抓住clock up中的kabuto（無誤）

      • W連Fang Joker（第二形態）甚至Maximum Drive（必殺技）都未用就把影月打飛了
