略歷編輯

  • 生於天曆2年（948年），卒於治安元年7月19日（1021年8月29日）。
  • 那日，源氏府上明珠入掌，鎮守府將軍源滿仲的妻子懷胎三年三個月，在丑年丑日丑刻迎來夫妻倆的第一個女兒。
    • 這名牛頭天王化身的女孩丑御前，因那異質的力量被視為鬼子，因而被寄放在大和國的寺院中。
    • 但是，滿仲對女兒展現出的才能感到可惜，於是在她十五歲時從寺院將她作為兒子、丑御前的弟弟接回來。這就是武人‧源賴光誕生的瞬間。
  • 賴光在父親的教育下成長茁壯，成了個足以背負源氏家名的武士。
    • 同時，賴光將源自牛血的魔性人格丑御前深藏心中，用盡全力抑制下來。
  • 以這種方式活著的賴光，在與與自己同樣是混血，並同樣脆弱的、需要保護者的少年金時相遇時，獲得了些許的救贖。
    • 為了不讓金時體會到如自己童年般的孤獨，賴光暗自起誓，下定作為母親守護金時的決定。
    • 對於未曾自父母那獲得愛的賴光來說，只有作為母親鍛鍊金時的那段日子，是她「稍微像個人」的寶貴時光。
  • 既是天下無雙的武人、亦是文采斐然的歌人，賴光率領麾下的四天王，奔走各地斬妖除魔，博得美名。
  • 然而，在日復一日的殺伐中，賴光對自己不合理的人生產生疑問。苦思許久而鑽牛角尖的她，下了透過自裁來消滅自身魔性的決定。
    • 賴光來到東國的某座寺廟，開始進行冥想。在漫長的冥想後，丑御前作為牛頭天王的化身顯現出來，賴光與丑御前開始進行「自殺」。
    • 四天王中的一員從賴光出發前的言行發覺些許異常，而趕到賴光的身邊。
    • 與丑御前的戰鬥極其慘烈，在戰鬥中金時哭了、下跪了，一把鼻涕一把眼淚地求賴光別輕易結束自己的生命。最後，放棄了的丑御前一頭撞上金時的閃電，回到賴光的深層意識中。
  • 被金時說服的賴光接受自身的魔性，回到京都。與視為愛子的四天王一同，作為京都的守護者活下去。

階級固有能力編輯

身為Berserker時編輯

  • 狂化： EX － 以理性作為代價換取能力的上升。
    • 賴光的情況是，不會喪失理性，而是與生前保有理性的自己一樣。但因為被鬼之血污染，言行會因變得異常深厚的母愛而出現道德上的破綻。
    • 也就是，為了深愛之物排除一切，並像狂信般的守護。
    • 一臉認真地述說著愛，為了愛的話連無視社會道德的行為也能許可……也就是說，在精神上已經病入膏肓。
    • 這種精神破綻無法因外力而清醒，因此無法將之與其他Berserker相提並論。基本上要說服她、使她改變想法是不可能的。
  • 神性： C － 北野天神、牛頭天王化身的神靈適性。賴光具有島產、國產的權能。
  • 對魔力： D － 能夠使一工程的魔術行使無效化。相當於驅魔護身符程度的抗魔力。
    • 若是以原本的職階接受召喚的話則有B等級的對魔力，但因屬性變化而導致等級降低。
  • 騎乘： A+ － 源自牛頭天王的騎乘能力。只要是獸類，就算是幻獸、神獸亦可以駕馭。但是，不適用於龍種。

身為泳裝Lancer時編輯

  • 對魔力： C － 能夠將詠唱少於兩節的魔術無效化。但無法防禦大魔術、儀式咒法等大規模魔術。
  • 騎乘： A+ － 源自牛頭天王的騎乘能力。只要是獸類，就算是幻獸、神獸亦可以駕馭。但是，不適用於龍種。
  • 狂化： C － 以理性作為代價強化身體能力的技能。
    • 其精神雖然成為Lancer也還保有一定程度上的問題，但與作為Berserker時比起來有相當大的差異。
