概要[編輯]

特異點編號 人理崩壞域
時代 德川家光時期的江戶時代[1]
人理定礎值 -
地點 大奧
導航角色 大奧的御年寄
通關後贈送角色 -
劇本作者
  • 2019年4月舉辦的活動，僅限制通過第2部第3章玩家方可參加。基於製作組表明是次活動為第2部第4章的前日談，所以被視為第二部主線之一。
  • 是次特異點取名為「人理崩壞域」，為過往未從出現的新特異點稱呼。
    • 按希翁的說法
「特異點」是汙染周圍的人類史，是把稱為人類史這種卷軸的強度降低的「污漬」。
「異聞帶」是「因為就算有也沒有意義」，所以應該被捨棄掉的另一個卷軸。
「人理崩壞域」則是兩者混合體，在是特異點的情況下，試圖對汎人類史進行干涉，更接近新宿和下總國的「亞種特異點」
  • 本次活動的御主技能新增了「印籠スキル」的全新功能，是取代御主禮裝、隨著劇情推進技能亦會強化的強力技能
    • 但一直使用會加速主人公德川化，最後面對伽摩時更為困難
      • 當然官方最後有安排救濟方式，所以任用也沒差。
  • 本活動於2022年7月13日起成為常駐活動，可籍由0個金方解鎖。
    • 一旦通過活動會固定在主線劇情最終戰，繼深海電腦樂土 SE.RA.PH後另一個新角色試驗場。

故事[編輯]

攻略印度異聞帶前的某一天，新迦勒底遭受來自外部的謎之攻擊，幾乎基地裡的所有人都消失。
逃過一劫的御主與唯一悻免的從者，得悉基地眾人被逆召喚至另一個時代。
他們向著一切謎團中心的時代探索，目標是17世紀半的日本、江戶時代。
在該時代等待他們的，同樣是人影全無的江戶城，還有瀰漫詭異氣氛，化為大型地下迷宮的「大奧」──

本節登場主要角色[編輯]

迦勒底[編輯]

  • 殺生院祈荒
    • 在新生迦勒底遭受攻擊中，唯一沒有消失的從者。
    • 按其說法，大概是因為擁有「單獨顯現」甚麼的技能才倖免
      • 但同樣擁有相同技能的這個和那個為甚麼沒有倖免就不曉得了[2]
    • 最先發現大奧迷宮的異常性質，點明五戒[3]的存在並減慢主人公受迷宮的異常影響
    • 「然後，日本嗎……什麼時候出發呢？也讓我一起同行吧。」「殺生院」
      • 事實上這個梗已經用第三次了。[4]
    • 在迷宮底層面對具壓倒性實力的伽摩時將主人公一行送到迷宮入口，自身留下斷後而被伽摩消滅，遺下要主人公去收集花札的遺言花生院祈荒
      • 其實並未被消滅，而是停留在伽摩宇宙中等伽摩被打飛過來時可以報復
  • 雪山神女
    • 最後一位被逆召喚至江戶的的迦勒底從者
    • 與春日局共用同一個靈基
    • 在終幕提到本來偏於惡神的摩羅應該因為濕婆神的看管而無法浮現影響伽摩，認為這是代表有「什麼東西」在影響印度神祇全體
      • 在繼後第四章得知是因為阿周那吸收了接近所有印度神祗而造成此空隙
  • 春日局
    • 大奧的掌權者(御年寄)
    • 在特異點事件中差點被消去時靈魂得到松平信綱和天海僧正的救助而留存，之後被雪山神女發現收留，與她共用同一個靈基
      • 在劇情中的立繪是穿上和服的櫻
    • 因為想永遠守護德川的血脈而吸引了伽摩的注意，引起是次危機
    • 最終決戰時，憑籍被強化的「大奧御年寄」概念，把伽摩驅逐離開大奧，瓦解了她在大奧所建立的宇宙
  • 柳生但馬守宗矩
    • 被逆召喚至江戶的迦勒底從者之一，靈基差不多和一樓的柱子完全同化
    • 後得御主手上的謎之花札所救，作為御主的守衛同行
  • 瑪塔‧哈里
    • 被逆召喚至江戶的迦勒底從者之一，被迷宮所洗腦而化成大奧的女中
    • 後得御主手上的謎之花札所救，作為御主的守衛同行
    • 藉由其寶具模擬出醉酒狀態，讓御主得以在不破戒的情況下突破第三層
    • 在最終決戰中以寶具讓取回大奧所有權的春日局認為自己擁有從者等級的強度，從而能以大奧掌權者的權限將Beast III/L「趕出」大奧。
  • 山魯佐德
    • 被逆召喚至江戶的迦勒底從者之一，被迷宮所洗腦而化成大奧的女中
    • 後得御主手上的謎之花札所救，作為御主的守衛同行
    • 藉由其寶具召喚以四十大盜為首的小偷角色代為偷竊，讓御主得以在不破戒的情況下突破第四層
    • 在最終決戰中以寶具將各種故事中「剛強的春日局」形象加諸在「春日局」本人身上來強化她作為大奧御年寄的概念，使主人公一行人得以在Beast III/L的宇宙中，建立了「春日局的大奧」，逆轉戰局。
  • 德川戈爾德魯夫
    • 被逆召喚至江戶的迦勒底所長
    • 因在迷宮內屢次犯戒而被迷宮洗腦，進而德川化
    • 在被春日局打了一巴掌後因為回想起被杜爾斯巴達式教育的心理創傷而一度解除了德川化，不過很快被伽摩洗腦回去，新所長你沒用時真的很沒用
    • 結果最後因為在大奧暴飲暴食太多而被達文西醬強逼執行地獄鍛鍊防止生活病出現

敵方勢力[編輯]

  • 伽摩
    • 是次異聞帶的幕後黑手，引誘主人公進入大奧迷宮而誘使其德川化
    • 為了使自己羽化成真正的Beast而把迦勒底的成員逆召喚至江戶時代，並以他們及江戶時代的人民作材料製成大奧迷宮
    • 最後在概念合戰中敗北而被趕出大奧，被祈荒調整靈基後登錄到迦勒底的紀錄上
    • 馬拉大人、巨型螃蟹、野獸後輩
  • 松平信綱
    • 協助伽摩進行陰謀、江戶時代唯一的活人
      • 奇怪的是，本來在家光當政時期沒有患病記載的他，出現在主人公一行面前時卻是一副病秧秧的樣子
    • 實際上知道凡人是無法正面戰勝伽摩而假意投誠，暗中支援主人公一行人
      • 主人公在迷宮內撿到的花札正是由以他的內臟作材料、用於對抗德川化的魔術術式
    • 為了保護德川而與天海僧正一起挽救春日局的靈魂
    • 同時假意和德川化的主人公敵對，使自己的存在與「與德川敵對之人」相契合，並以此啟動了淨化術式
      • 最後讓柳生將自己斬殺使術式完成，將自己變成有對「偽」德川特攻的「偽」村正
