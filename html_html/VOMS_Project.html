VOMS Project

VOMS Project / ボムズ / ボンズ
本條目內容不足需要擴充，歡迎有興趣的朋友加以補完。

目錄

概要編輯

  • 由GYARI基於想看無限的遊戲實況的個人私慾而創立的個人勢虛擬YouTuber計畫。
  • 最初以聲音（VOX）跟怪物（MONSTERS）概念取名為「VOX MONSTERS」，後正式定名為「VOMS Project」。
  • 以「直播怪物」為核心概念，旗下的藝人皆被稱為「怪物」，各自有屬性設定以及去除掉特定部位就會死掉的「特色」設計。

團隊成員介紹編輯

GYARI編輯

 

  • VOMS Project創建者，同時也包辦了怪物人設等事務。
  • 過去作為ボカロP活躍在niconico，在啟動VOMS Project後將重心移到VTuber。
  • YouTube頻道
  • Twitter帳戶

⚡天野ピカミィ⚡編輯

 

  • VOMS Project一期生，雷屬性怪物，特色是頭上的天線。
  • 2020年3月19日出道。
  • 擅長日英雙語，最大特徵是如海豚音般的笑聲。
  • 初配信時以成為VTuber頂點為目標。
    • 然後想登上Getting Over It的頂點但就是登不上去，過了九分鐘後裝死重開直播。
  • 在2022年3月26日出現了由しらたま繪製、Jujube製作的「異世界ピカミィ」。
    • 為了讓觀眾確實的轉生到異世界，還特地安排了トモシカ開卡車把正在看直播的觀眾撞死，然後被ミウネル女神與リューゴン小天使送到異世界。
  • 目前活躍的成員中訂閱數最高的。
  • 參考了精靈寶可夢的代表寶可夢皮卡丘而設定為雷屬性，老任法務部警告。
  • YouTube頻道
  • Twitter帳戶
