解說編輯

  • 2007年由GAINAX製作的動畫作品，以鬼之作畫和熱血劇情著稱；全26+1話（總集篇）。
  • 共分為四部，分別是：立志篇、風雲篇、怒濤篇、回天篇，各篇都有特定角色台詞包含標題
    • 立志篇(鑽頭的底部)：卡米那
    • 風雲篇(物種級王都テッペリン攻略戰)：妮亞
    • 16話（總集篇）：維拉爾
    • 怒濤篇(衛星級月球落下阻止作戰)：羅修
  • 回天篇(銀河級最終決戰)：西蒙
  • 「鑽頭是男人的浪漫」
  • 雖然熱血老梗，但感動無數島民
  • 動畫結束已有多時，不過GAINAX仍繼續把騙錢計劃劇場版推出來插幾張新咭就把大家騙倒了阿魯，漫畫版同樣龜速連載了數年後順利完結
    • 編者表示雖然故事都一樣，劇場版便當還派少了很多很多（重點），不過其實還是不錯的，漫畫版則有更多機會描寫更仔細的劇情，畫風也和動畫無違和，值得一看
  • 已參戰超級機器人大戰Z2 破界&再世篇，目前在OE、Z3 時獄篇&天獄篇持續登場（後續參戰紀錄待補）

故事提要編輯

住在地下村落ジーハ村出生的少年西蒙，每天不停地做著挖掘的枯燥生活，跟很多同村的孩子一樣，從沒看過天空.....
跟西蒙一起成長情同兄弟的卡米那，卻立志追隨年幼時告別的父親見識地上的世界
就在某天，西蒙掘出了一個小鑽頭和一個大臉孔，就在此時，村落受到地上的怪物的襲擊
然後與從天而降的少女．庸子相遇
為了迎擊怪物，西蒙坐上了那個大臉孔－－Lagann，改變了三人的命運.....

  • 螺巌/ラガン(LAGANN)
    • シモン専用ガンメン，漢字為「螺巌」，由アニキ親自命名
    • 跟其啟動器コアドリル一起被シモン挖到從此占為己有
    • 為當初螺旋族開發的對アンチ=スパイラル用最強兵器之一，可以任意跟其他機體合體，改變其結構，進而發動螺旋力

    • 曾經被他揷合體過而同化的機體代表為グレン，SIZE最大者為ダイガンザン
    • 可以從一顆頭的身體伸出手腳，嘴巴還會跟著動，具有非常神奇的比例以及連動的機體
    • 可以短時間飛行
    • 擁有螺旋力護盾,不過必須先發動嘴砲才會有，嘴砲護盾威到可以把エンキ的光束彈開

    • 可視情況裝配不同的武裝，其中又以機槍最為普及
    • 第三、四部的主力部隊
    • Kimmy、Darry的ACE專用機分別被塗裝成藍色與粉色作為標示，除了這兩台之外其他同型機種被打爆是經常有的事
  • 方舟紅蓮/アークグレン(ARC GURREN)
    • 螺旋王殘留在王都的超巨大宇宙戰艦
    • 約可收容36萬人於艦上，被Rossiu視為人類最後的希望
    • 動力來源是螺旋王的生體電腦所產生的螺旋力，也會隨著艦上人員的士氣提升出力
    • 可形成螺旋力防護罩
  • 方舟紅蓮螺巖/アークグレンラガン(ARC GURREN LAGANN)
    • グレンラガン與アークグレン合體而成的超巨大顏面
    • 駕駛艙裡其實坐著的是グレンラガン而非駕駛員，也就是駕駛員操控著グレンラガン來操縱アークグレンラガン
    • 也曾經由Kimmy、Darry的グラパール來駕駛
    • 合體名為「怒涛合体」
    • 必殺技︰
      • 時空烈断バーストスピニングパンチ
    • 以拳擊打出時空的裂縫，直接把敵人毆飛到時空的另一頭去的招式
    • 總而言之，就是把敵人從這個世界上轉學
  • 超銀河ダイグレン(Chouginga Dai-Gurren)
    • 螺旋王所殘留下來的超巨大級顏面宇宙戰艦
    • 在被解放之前一直偽裝成月球在地球上空盤繞，換句話說就是這艘戰艦的大小和月球不相上下
    • 以次元跳躍的方式在宇宙中移動
    • 備有超高性能的生產機能，在短時間內製造出各式比普通顏面大一號的空間顏面(スペース・ガンメン)
    • 艦長由Dayakka回鍋擔任，其他艦上成員︰Leeron、Leite、Attenborough、Tetsukan、Gabaru、Shibera、生體電腦螺旋王
  • 超銀河グレンラガン(Chouginga Gurren-Lagann)
    • アークグレンラガン與超銀河ダイグレン合體後的超巨大顏面
      • 劇場版則是從月球變形，成為超銀河大紅蓮的原體，曾經因陷入死亡螺旋海溝中而變形回戰艦型態，最後回復人型。
    • 吸收了デススパイラル次元的海溝裡的螺旋力，擁有小宇宙級的能量
    • 在合體變形時需要大量的螺旋力，由Simon旋轉グレンラガン的鑽頭來填充
    • 擁有螺旋力護盾，除了敵人以神等級的因果律操作攻擊外幾乎是無人可破
    • 在顏面上出現星型的紅色墨鏡，作為對戰友們信念的繼承
    • 必殺技︰
      • 波動時空軸一斉射撃
    • 克制敵人任意薛丁格方程跳躍(ランダムシュレディンガーワープ)的鑽頭彈砲擊
    • 針對以多次元確率變動制御在時間軸裡移動的敵人，超銀河グレンラガン本身預測出敵人的所在時間位置並加以砲擊
    • 炮擊手由Yoko、Attenborough、Kimmy、Darry四人擔任
    • 部分砲台名稱︰「メイルシュトローム砲」、「メガボルテックスキャノン」等
    • 瞬間消滅無量大數的敵人，超銀河級地圖炮
      • 超銀河大切断
    • 將胸前的墨鏡投擲出去斬斷敵人，當然威力提升為超銀河級
      • 超銀河ギガドリルブレイク
    • 將肩上背負的兩只巨大鑽頭於手腕重疊後使出的ギガドリルブレイク
    • 簡單來說，就是超銀河級威力的ギガドリルブレイク
  • 天元突破グレンラガン(Tengen Toppa Gurren-Lagann)
    • 最後一戰，在超螺旋宇宙中出現的最強最大最後的顏面機器人
    • 集Gurren團所有成員的螺旋力於一身，威力已經不是強不強的問題
    • 體積大於銀河，但是由於是在只要假想便可成真的超螺旋宇宙中出現，所以計較大小是無意義的行為
    • 移動速度依照比例已經遠超過光速，但是由於……（以下略
    • 各個團員的拿手招式皆可使用，如︰Yoko的電擊槍狙擊、Viral的雙刀、Attenborough的彈幕砲擊等
    • 必殺技︰
      • 俺の嫁は宇宙一スイング
    • 直譯︰我老婆是宇宙第一之旋轉
    • Dayakka以閃光戰術欲閃瞎敵人的招式，需要舵手Gabaru的配合
    • 總之就是以超次元錨(超次元アンカー)將敵人綑綁住之後再旋轉投擲出去
      • 確率変動弾
    • Attenborough神之領域的攻擊，操縱因果律以飛彈無視防禦直擊敵人
    • 雖然看起來很威不過對反螺旋王無效
      • ラゼンガン・オーバーロード
    • 螺旋王的最後閃光
    • ラゼンガン自天元突破グレンラガン的腹部冒出，並藉由敵人的攻擊量子化
    • 其後凝聚成巨大鑽頭，由天元突破グレンラガン吸收其螺旋力
    • 螺旋力爆表並化為彩虹色，最後一擊的ラガンインパクト能量來源
  • 天元突破機組
    • 僅在劇場版中登場，由於劇場版中天元突破紅蓮螺巌遭到反螺旋族解體[5]，在確認彼此意念之後由大紅蓮團成員[6]各自駕駛的機體，基本上就是其駕駛機體更加巨大化而已。
    • 部分TV版中天元突破紅蓮螺巌使用的招式在劇場版中也變成天元突破機組使出，像是「我老婆是宇宙第一大旋轉」、「機率變動彈」這兩招改成天元突破大紅蓮(天元突破化的大紅蓮戰艦)使出。
    • 除了TV版的機組天元突破化外，妮亞在本作中也開了一架原創的手持鑽頭型長槍的天元突破機組出場。
  • 超天元突破紅蓮螺巌
    • 僅在劇場版中登場，由所有天元突破機組合體而成，模樣近似於由藍色火焰能量構成的卡米那，應該說這根本不能算機器人了。
    • 比天元突破紅蓮螺巖還要巨大得多，銀河在它身邊也只是一個細小的光旋渦而已，而且必殺技「超天元突破ギガドリルブレイク」還是一個比機體本身大很多的鑽頭...

  • 第四回作畫崩壞
    • 雖然作畫整體維持高水準，但是監督對於作畫監督完全放任，所以風格本來就很特殊的小林治(包辦第四話作畫監督及分鏡)於第四話與整部人作品格格不入的風格受人非議。
    • 第四話引發的2ch鄉民大肆評擊，對此不滿的GAINAX女社員在自家的mixi[7]發表了「素人が知ったような口きくなっつーの（私もアニメ素人だけどさ）（門外漢居然用那種好像很清楚的口吻說大話（雖然我也是動畫外行啦）)的言論
    • 然而引發事件的是GAINAX取締役赤井孝美的回應—「オール匿名だし、ひとつの板で書いてる人って同じ奴が数人でグルグル書いてるパターンが多いから、だんだん中毒っちゅうかね、他人よりすごいこと書きたくなる。『オレはオマエらとは違うんだぜ。』と。（そのようなものを読むのは）肛門に顔を近づけて深呼吸するようなもんです。必ず屁を吸ってしまいますぞ」*** 翻譯：畢竟是完全匿名，一個板都不知有多少人自演，(2ch那些人)不知不覺就沉迷於表現「我跟你們是不同的喔」而去寫比別人嗆的內容吧。（如果去看2ch的話）就好像把臉貼著肛門深呼吸，也只會吸到屁一樣吧。
    • 以上發言不知為何外流至2ch鄉民討論串，一眾鄉民殺上官方blog攻擊，赤井孝美被指責侮辱觀眾。
    • 最後GAINAX公開謝罪、赤井辭退取締役位，事件才告一段落。
