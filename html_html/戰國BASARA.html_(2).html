戰國BASARA

戰國BASARA/戦国バサラ/戦国BASARA/Devil Kings[1]/戰國婆娑羅[2]
本條目歡迎各位島民繼續補充

目錄

解說編輯

  • Capcom社製作的動作遊戲，以大量惡搞、超時代科技、作品捏他，美化古代人物為賣點。
    • 人設方面同樣都在戰國時代，因此角色列表以及主要系統都與戰國無雙以及其他日本戰國作品大同小異。
    • 不同於光榮較貼近寫實與正史的做法，Capcom以輕鬆、美形、偏向漫畫風格為優先，另外也替每位武將加上了封號，詳見武將列表。
      • 除了某些名將因為固定形象太難扭轉之外，大致上都做出了個人的捏造獨創設定。
    • 將無雙亂舞改為BASARA(婆娑羅)技，實際上差別不大。
      • 動作系統差別在於除了一般的連段外，還可選配兩種固有技，但到三代改成四個固有技[3]可同時使用，再從三個固有奧義中擇一使用，另外ai弱化，不過三宴時，又再把ai改強而變成既會閃又會防。四代時則是讓玩家遊玩時，能夠依情況選擇固有奧義使用
    • 遊戲地圖簡單化，玩家只需要從起點打到終點就結束，但每個關卡卻有不少機關或事件。
  • 除了玩家可操控的角色外，還有在遊戲中登場卻無法讓玩家使用的角色，稱為『敵武將』
    • 一代時的敵武將:德川家康、長曾我部元親、毛利元就、北條氏政、今川義元(二代後長曾我部元親與毛利元就更改成可用角色)
    • 二代後新增的敵武將:片倉小十郎、風魔小太郎、淺井長政、阿市、本願寺顯如
    • 二代敵武將除了固有技之外，一般攻擊都是兼用可用角色的動作(只有今川義元擁有獨立普攻和空中攻擊動作)
      • 德川家康、北條氏政→前田利家動作
      • 片倉小十郎→伊達政宗動作（左右相反）
      • 阿市→阿松動作
      • 風魔小太郎→猿飛佐助動作
      • 本願寺顯如→武田信玄動作
      • 淺井長政→真田幸村動作
      • 今川義元→上杉謙信動作
    • 英雄外傳後將上述敵武將全更改成可用角色並擁有個人動作，並再新增一位敵武將：松永久秀
      • 但除了片倉小十郎、淺井長政、阿市、風魔小太郎擁有完整的固有技之外，其他角色卻都只有兩種固有技，德川家康與本願寺顯如更是繼續使用兼用模組偸懶的起源...(連身為敵武將的松永久秀的攻擊都比他們豐富阿....)
  • 除了正常的武器外，每位角色都有專屬惡搞武器，大部分都是日常生活用品，而一二代中伊達政宗、真田幸村、濃姬的惡搞武器則是惡魔獵人主角但丁的武器藉機打自家廣告
    • 一代時一共有六把武器，惡搞武器則是第六把
    • 二代與英雄外傳後更新成八把，惡搞武器為第七把(伊達政宗、真田幸村、濃姬為第八把)
  • 光榮將戰國、三國等作品進行crossover，創作出無雙OROCHI等斬草大作專攻本作已經在名稱上定名為"戰國"的死穴，再加上光榮近年各種外包無雙及本作自爆人氣冷卻，有變成時代的眼淚的可能性。

屬性解說編輯

  • 這裡的屬性指的是遊戲中各武將的屬性，於各武將有提到其屬性
  • 大體而論分成風、雷、火、冰、光、闇六大種屬性，四代足利將軍為凸顯其主角格而給予特殊屬性：震
  • 若攜帶的武器附帶有屬性，攻擊時會有一定機率發動屬性攻擊與效果
    • 風屬性沒有其他附帶效果，為了平衡其屬性攻擊力比其他屬性還要高很多
    • 雷屬性是攻擊附帶麻痺、火屬性攻擊附帶幾秒持續灼傷、冰屬性攻擊附帶結凍、光屬性攻擊附帶暈眩、闇屬性則是直接擊破沒有名稱的小兵[4]+自身HP些許回復
    • 三代以前武器都是只有固定武器才附帶屬性[5][6]，四代改成武器上的銘若是有道出、六道、六道極就會賦予角色主屬性
  • 原則上武將屬性是固定的，僅少數主角格的DLC武器會出現其他屬性

英雄外傳編輯

  • 戰B一作分兩份賣及致鬱劇情的起源
    • 新增可操作角色：片倉小十郎、阿市、淺井長政、風魔小太郎、德川家康、北條氏政、今川義元、本願寺顯如
    • 新增敵武將：火燒野原廣志松永久秀
  • 追加片倉小十郎、阿市及淺井長政的故事及伊達政宗、真田幸村、前田慶次、風魔小太郎、長曾我部元親5人的外傳故事
    • 當中阿市的劇情   爆幹鬱到連官方也叫玩家先玩阿市再玩淺井長政
  • 追加可雙人的天下統一及大武闘會等模式
  • 整部作品評價都很不錯......除了那個動畫
    • 本作的立繪改為2D賽璐璐式畫風，讓小部份玩家無法接受，但這不是問題
    • OP曲BLADE CHORD配上下巴寬得活像來自クロマティ高校的片倉和松永的臉部大特寫讓很多人再起不能
      • 往後只要播出此OP必定有人刷顎/AGO/ドヤ/低予算等字眼，之後的的OP及動畫不包括第三作更是把本作作畫壓在地上磨
    • 劇情動畫不流暢、用色微妙，被批評品質像是歷史教材
      • 例如武田信玄長出了美男子般的五官，阿市的頭髪活像昆布貼在臉上不會動
      • 伊達政宗的六爪變成了七爪，阿市的劇情結末中臉朝下倒地的信長則被笑像蟑螂
 
 

戰國BASARA3/宴編輯

  • 時間上剛好也與光榮發表戰國無雙3的時間相近，互爭人氣市場意味濃厚
  • 由於遊戲內容和二代相比較為簡略，且敵武將太多以及CG動畫過少，而被眾多玩家認為一定會出個外傳資料片來騙錢然後就説中了
    • 時間背景前進到主導日本天下大勢的"關原之戰"，除了伊達政宗與真田幸村等主要角色還保留之外，德川家康也在更改人設後與新角色石田三成升格成主角
    • 三代新增角色：整容成功的德川家康、暗屬性上杉石田三成、大谷吉繼、濃姫雜賀孫市還要變成女人、黑田官兵衛、森蘭丸鶴姬
    • 三代新增敵武將(「宴」中為可操作武將)：小早川秀秋、明智光秀天海、最上義光、呂布立花宗茂、大友宗麟
    • 三代新增地方領主永遠的NPC：宇都宮廣綱、佐竹義重、南部晴政、尼子晴久、姉小路賴綱
    • 三宴追加可操作武將：松永久秀、武田信玄[7]及把3中收起當NPC的片倉等人補回去
  • 由於部份角色人設太黑暗、專用道具佔四格兼不實用、大量本來可操作的武將NPC化等原因而受到許多玩家批評
    • 石田及毛利的其中一條路線嚴重致鬱，新角大谷吉繼性格極為負面[8]，並因史實皮膚病及人設的木乃伊造型引來日本痲瘋病學會的關注[9]
    • 宴更因松永的劇情而成為唯一CERO:C的作品
    • 阿市廚小林P的問題發言：「系列成功是因為不懂史實的人們買」、「阿市要如何續投登場是最重要的課題」
  • 宴的其他新要素：
    • 補完新増8名角色劇情，當中一半為搞笑劇情、兩個為嚴肅劇情、剩下兩個為超黑暗劇情
      • 但無角色劇情的復歸舊角不但造型只有一個，連八把武器造型都完全一致，包括無銘跟山吹武器連個特效都懶做
    • 戰極Drive、BASARA抽獎、天下統一模式復活並追加日本遊戲模式
    • 專用道具改回一格並増加實用性
    • 追加難易度「婆裟羅」、角色性能調整等

戰國BASARA4/皇編輯

  • 因為作不下去把世界觀設定在平行世界[10]中的室町時代末期其實這才是戰國時代，以室町末期的足利義輝推行「天政奉還」及「戰國創世」為主題
    • 除了既有的戰國時代武將外，亦新增室町時代的武將參戰
    • 四代新增角色：島左近、柴田勝家、山中鹿之介、後藤又兵衛、井伊直虎
    • 四代新增敵武將(「皇」中為可操作武將)：足利義輝、京極瑪麗亞
    • 四皇新增的角色：千利休又一個非武將
  • 四代追加戲畫BASARA技，只要雙方(玩家和戰友)的BASARA計量表全滿，並呈現紅色計量和印章，就能發動戲畫BASARA技[11]
    • 另追加小兵的立體陣形疊羅漢雜技團系統
    • 由於四代的武器系統變動太大加上取消專用道具而且賣腐賣過頭跟劇情暴走，受到許多玩家ry
      • 主角太多了只好讓家康和幸村把OP位子讓出來
        • 慶次？沒有這個人
    • 很多故事跑認真路線跟再次黑過頭而讓人感到胸糞，例如
      • 因為秀吉一掛三成就會爆炸島左近跟家康在某條路線都得制止他，結果付上沉重代價
      • 柴田勝家不是自己瘋掉就是害人瘋掉
      • 一如以往地黑到爆的松永除了爆點減少還害風魔賣點之一的神祕度扣分了
  • 皇的其他新要素：
    • Production IG繪製的高水平OP
    • 新増及追加劇情路線把風魔救回來，但慶次仍是沒動畫路線
    • 專用道具回歸(以專用銘名稱形式)、充滿變化性的合戰輪盤(出陣前可隨時開或關輪盤模式)
    • 追加固有技 改(用天貸兌換)，指令是L1+正方[12]
    • 推出萬代南夢宮的熱情傳奇合作角色服裝[13]，並錄續推出快打旋風、魔物獵人、惡靈古堡、新日本摔角聯盟、西川貴教等DLC服裝

戰國BASARA：烈傳系列：真田幸村傳編輯

  • 黑歷史爆死GAME，不用一個月就60%OFF
    • 瘋了這麼久才喊要重現歷史結果毀了系列
  • 為了跟上2016年大河劇《真田丸》的風潮[14]而便乘推出的新系列(寫作系列但只有一作)，故事以真田一族的壯烈歷史為主
  • 新增角色：幸村爸真田昌幸、幸村哥雙馬尾真田信之、酷正太梵天丸及小正太弁丸
    • 系列製作人表示真田幸村傳將不會有女角可供操作，完全腐向...
      • 僅在練武模式才能使用含女角的前作40多位角色，其他模式就只能用特定角色且全都是男性，腐向確定
    • 因為故事又是平行世界而再次吃書，慶幸的是北條老爺爺又回來了但是不能用、柴田勝家則壞腦變成梗角
      • 武田大叔老早掛掉轉行當旁白，武田勝頼連CV跟臉都沒有
    • 一如所料的流用前作地圖、新武將技能少還跟舊角特性重疊
      • 劇情上出場的可使用角色只有20多名，佐助戲份亦少得可憐，部份角色台詞對不上場景
      • 明明PV有出現海量雜魚但實際卻只有很少PV詐欺
      • 不知為何還要改惡把主畫面撃殺人數鬼隱，廢除2P模式和戰友系統

  • 唯一一部OPED都沒西川貴教獻唱的作品，但卻讓教主為酒井忠次（動畫原創）配音，後期還穿成HOT LIMIT的樣子
  • 人設跟前作相比較粗曠人物小腿變粗而引起部份女粉不滿，但最致命的是作畫不流暢跟品質欠佳，戰鬥畫面只比英雄外傳好一點
    • 甚至出現人物線條粗幼都出包的謎之作畫

學園BASARA編輯

  • 年份：2018
  • 製作公司：Brain's Base
  • 原先是一系列的公式同人漫畫集，在2018年宣布動畫化
    • 人物設定主要以三代為主，但動畫有出現四代的角色
  • 因為是校園劇，比較偏向日常搞笑風，作畫也很隨便
    • 但與本家相比之下本作猶如清泉般和藹可親
  • 角色年齡大多集中在高中因而出現大量老聲老臉的高中生
    • 本多連校服都穿不下
      • 阿市觸手Play第四作，本作觸手出現率比之前都要高
    • 德川石田鬥爭十二集只是為了當學生會主席
      • 最終回更是讓鬼隠多年的薩比（加上大友及疑似保安叔叔的立花）亂入並以看起來很趕時間的學生全體衝向鏡頭大喊BASARA方式完結
  • 移植至自家手遊《戰國BASARA 戰鬥派對》時，某些登場角色會有別於家用遊戲的四字肩書（稱號）

配樂製作編輯

  • 遊戲系列的配樂早期是由Capcom Sound Team主導，中後期則由T's Music接手。
    • 從系列開始以來至今未中斷作品參與紀錄的樂師為T's Music的Chamy.ISHI（伊師正好）
    • 二代開始則有近藤嶺的加入，四代部分樂曲也是由他進行重新編曲與製作。
    • 三代則邀請了在美國因為《汪達與巨像》獲獎的名作曲家「大谷幸」參與劇場模式的樂曲製作，是唯一有C社與T'S Music成員以外作曲家參與的作品。
  • 動畫版除了「Judge End」、「學園BASARA」分別相繼由「得田真裕」與「立山秋航」製作外，其餘作品則全部由「澤野弘之」包辦。
