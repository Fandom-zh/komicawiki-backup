無夜國度系列

無夜國度系列 /よるのないくにシリーズ/Nights of Azure Series

目錄

解說編輯

  • 由GUST（現KOEI TECMO轄下GUST品牌部）開發的似無雙式3D動作RPG
  • 非常百合的作品，大部份主要角色都是女的估計男的都戰死了
    • 因為是百合作，所以女角的內衣/睡衣/泳裝/半裸畫面經常出現
      • 二代甚至可全員穿泳裝出動
  • 因為是延期闇榮家的，所以兩作都被延過發售日
    • 1代只有21天，二代…足足曬足將近180天

特色編輯

  • 如上，本糸列主打美少女們的戰略級胃痛百合救世故事。
    • 主軸是『犧牲親友才能拯救這世界，妳幹不幹？』
  • 作品名已經說很清楚了，本作是被妖魔之王威脅下失去戶外夜生活的世界
  • 本糸列作是CERO：C的、又是糟糕的GUST部門的…，所以該有的擦邊球一定有
  • 似無雙式3D動作RPG
    • 但是要在家機才能感受到那種華美的畫面
      • PSV的玩家角落畫圈圈中…
  • 使魔和主角共鬥
    • 只是使魔有點太強了…

共通設定編輯

  • 邪妖
    • 原本是流著蒼色血液的惡魔，後來在夜王被討取時所噴向全世界的蒼血影響下追加淋到蒼血而轉變成魔物的分類
      • 任何東西都有可能變成邪妖，那淋到地上豈不是整個地球都要變邪妖了？
      • 老實說戰力很強，非教皇廳聖騎士以上的騎士都需要用圍毆才能打贏；即使那是隻最弱的邪妖。
  • 純血妖魔
    • 指全部血液都是蒼色的妖魔種之中的高等種類
      • 幾乎都是BOSS級擔當
      • 來自異世界的高智力生物
    • 討厭因蒼血而變成邪妖的存在
  • 夜王(夜之君/夜之君王)
    • 妖魔/邪妖種之中的王，統領全部的妖魔
      • 傳說中教皇廳的初代聖女和夜君決戰後雙雙隕落。
      • 事實上兩人並沒有死亡，夜王的部份是力量大部份塞到血劍裡；本人則是成為雅娜絲身邊的黑貓
  • 教皇廳
    • 教皇廳的最高領袖是教皇一個人(1代)，二代則是分成三人合議
      • 主要業務就是全世界討伐邪妖
  • 聖騎士
    • 教皇廳下的強力戰力單位，可以和邪妖單挑
      • 據說初代聖女只帶了一萬人的聖騎士就可以拖住夜王的大軍不去干擾夜王和聖女的單挑
  • 半妖
    • 一半是人血，一半是邪妖蒼血的存在
      • 目前所知天然半妖只有雅娜絲一人，其他的半妖都是人工製造的
      • 持有吸收蒼血強化自己的力量和使用血液化形武器的能力
    • 缺點是吸太多蒼血會變妖魔種
      • 對付方法就是吸人血…蕃茄汁無效喔！(BY雅娜絲)
  • 聖女
    • 能封印夜王（1代）/月之女王（2代）的存在
      • 簡而言之就是對大BOSS專用寶具。
      • 但封印時間大多不長，也不太能徹底封死對象
    • 有趣的是聖女名字全部都是L字開頭的巨乳少女(兩代共通)
      • 聖女的L是Large Boobs的L
      • 相對地，聖女的姬友（=主角）名字也全是A字開頭。

登場人物編輯

1代編輯

  • 以下的中文譯名取自《無雙☆群星大會串》的Steam正體中文版（僅限事典中有相關條目者），英文譯名則取自英文版。

雅娜絲（アーナス/Arnice）編輯

  • 1代主角，天然型半妖
    • 曾經淋到夜之王的蒼血，當年的玩伴變成邪妖。而雅娜絲推測是覺醒變身成炎魔模式大鬧，後來被教皇廳的聖騎士們『圍毆』到力盡解除變身撿回家研究
    • 所以真實年紀很大…
      • 無雙大亂鬥中躺在按摩椅上時會倚老賣老說「畢竟是幾百年來的痠痛了……沒那麽簡單就……啊，好像有點舒緩了耶？這個真棒！」，所以歲數很大無誤
    • 後來是由某個自稱師兄的傢伙在近幾年用信把雅娜絲教育回正常一點的人，還丟去上學……（喂，這樣的學生沒問題嗎？ ）
      • 和琉莉迪絲是學校認識的閨密，也是負責護衛琉琉去封印夜王的聖騎士
      • 巧克力飲品派的愛好者
      • 前陣子還和1代另一名人氣王去KOEI大亂鬥出場，和部份登場角色的關係很不錯[1]。
  • 2代確定已經是成為夜之王本人，只是記憶被封印掉了
    • 雅娜絲妳的睡衣和沒穿根本一樣！

琉莉迪絲（リュリーティス/Lilysse）編輯

  • 1代的女主角，也是封印夜王的夜之巫女/聖女
    • 被雅娜絲昵稱「琉琉（リュリュ/Lily）」
    • 會叫雅娜絲「雅娜絲大人」。
      • 猜是當年在學校時雅娜絲運動萬能、又太帥的關係
    • 學校畢業後在教皇廳直屬旅館「エンデ」當侍者
      • 整理房間以外的工作根本是『災難級MAID』[2]
    • 會作出能一口必殺所有使魔的瑪芬/杯子蛋糕
      • 就連夜之君化身的那位也說吃下去會見天國的等級

サイモン編輯

  • 教皇廳直屬旅館「エンデ」的管理者兼咖啡MASTER
    • 喜歡安靜的地方和解謎
      • 所以讀過很多偵探小說
    • 拿琉琉沒辦法
    • 過去一團謎，但故事中能片面知道他和教皇廳的因緣很深

ロイド編輯

  • 來到旅館「エンデ」的房客之一
    • 是個很輕浮的愛錢行商人。
    • 能搞到一堆來自世界各地的貨物的實力派商人，但貨物是怎麼進入1代那座被封鎖的島啊？
      • 與另外一個房客有角教授兩人經常吵架
      • 這兩人其中一個是教皇廳派來的幫手，但到最後也沒說明是誰
    • 大人的黑咖啡派(不加糖)

有角教授編輯

  • 來到旅館「エンデ」的房客之一
    • 邪妖的研究者，也可以說是邪妖通
      • 老是想拖雅娜絲下水一起研究邪妖
    • 紅茶派……只是教授你一杯紅茶加7顆方糖沒問題嗎？！
    • 跟ロイド兩人之中有一個是教皇廳派來的幫手。
      • 根據2代的消息，他應該才是教皇廳派來的那個

師兄（ブラザー/Brother）編輯

  • 雅娜絲的上司，常用黑色信紙下命令或是配送一些雅娜絲根本用不到的高價商品
    • 無可救藥的師妹控
    • 真身是夜王，也是常跟在雅娜絲身旁的機車(BY雅娜絲)黑貓。
      • 基本上已經沒有要滅掉這世界的主意了。
        • 不過觀察雅娜絲反而成為主業了。

初代聖女編輯

  • 傳說中討伐夜王阻止『永夜』發生的聖女
    • 事實上從夜王時代開始就擔任教皇至今(1代)，會利用各代聖女在封印儀式時用石化搶奪當代聖女的生命力延命
      • 同時也是把夜王召喚到地面上的真正元兇，其目的是搶奪夜王的力量去停止爭戰
      • 1代最終BOSS，二代已確定被討伐

克莉絲特弗蘿絲（クリストフォロス/Christophorus）編輯

  • 簡稱克莉絲（クリス/Chris），當然不是隔壁棚的某肌肉男。
  • 以歌劇院為根據地的純血妖魔，會操控樂器攻擊。
    • 話雖如此，但她對狩獵邪妖的半妖（=主角）們很有興趣。稱雅娜絲為「半妖小姐」。
    • 而且明明跟雅娜絲（以及2代的雅爾潔）是妖魔/教皇廳這種對立的身份立場，她跟主角群的關係卻並沒差到哪去，甚至兩代都會主動跑來告知主角們消息。
  • 雙面人格，依戴著的一張遮半臉的面具而變換。
  • 1代的人氣王，還和雅娜絲一起出差去同社的亂鬥無雙參戰。
    • 兩人也是唯二有跑到2代的1代角色。

2代編輯

  • 以下的中文譯名取自Steam中文版。

雅爾潔（アルーシェ・アナトミア）編輯

  • 昵稱雅爾（アル）。
  • 2代主角，開頭就被月女王貫心而死。後來在卡蜜拉的幫忙下以人工半妖復活。
    • 和莉莉雅娜、露恩海德是同學兼玩伴。
    • 封面三主角之中的笨蛋役。
      • 是個筋肉笨蛋。
      • 是個笨蛋。（重要的事情要說三遍。）
    • 一樣愛喝巧克力飲品。
    • 除了跳樁時是抱所有LILY；此外在使用滑車時，莉莉雅娜、繆貝爾隊長、雅娜絲和卡蜜拉她們是抱雅爾潔然而雅爾潔本人卻是一副死魚眼的表情，剩下的3個LILY都是被抱方。
      • 習慣裸睡。

莉莉雅娜（リリアーナ・セルフィン）編輯

  • 昵稱莉莉雅（リリア）。
  • 2代巫女役，又稱『刻之新娘』。
    • 巨乳。
      • 封面三主角之中的和事佬役。
    • 持用時間減速的力量。
    • 本作其實都是她的夢境，差別只是雅爾潔有沒有發覺。

露恩海德（ルーエンハイド・アリアロド）編輯

  • 昵稱露恩（ルーエン）。
  • 2代第二女主角（笑）。
    • 不算巨乳，但原因是莉莉雅娜胸部比她小，所以是爆乳。還不一樣是大胸嘛！
    • 本作最傲嬌的角色。
      • 持有能將感情化為力量的光之力（看起來像是光之劍氣）。
    • 為了拯救莉莉雅娜和笨蛋雅爾潔的關係加入和教皇廳對立的『ルルド教団』。
      • 本作最慘人物。[3]

卡蜜拉·有角（カミラ・有角）編輯

  • 將雅爾潔以人工半妖身份復活的天才研究者。
    • 同時也是教皇廳的另一個旅館『艾提爾娜（エテルナ）』的管理者。
    • 和前作的有角教授似乎有血緣關係(官方未說明白)。
    • 武器是槍支。
      • 疑似是改造人。
      • 曾經看過雅娜絲和邪妖戰鬥的過程，並有拿到雅娜絲的血。

薇露希卡（ヴェルーシュカ）編輯

  • ルルド教団派來的刺客，目的是奪回不回教團的露恩海德
    • 但還是被雅爾潔加LILY扁昏&收入後宮
    • 本人是ルルド教団作的人工半妖，只是因為全方面強化到過極致；所以生命並不長。
      • 會量子化隱身，出戰時常會看到她衝最快

艾蕾諾雅（エレノア・エルネスト）編輯

  • 教皇廳的商人，主業是巧克力料理師
    • 能用巧克力攻擊，甚至還能作出武器
    • 準備好了嗎？你將面對的是無窮的巧克力武器
      • 作出來的巧克力飲品，據說是特供等級的

繆貝爾（ミュベール・フォーリン・ルー）編輯

  • 雅爾潔的前輩，也是雅爾潔她們學校的以前學生會長
    • 原卡蜜拉作出來的人工半妖，只是現在是完全妖魔化了
    • 加入前總是想拖雅爾潔加入妖魔方的半妖
    • 和雅爾潔一樣愛喝巧克力飲品
      • 泳裝只遮右胸…
    • 曾在敵方幹部的虐待下硬撐了100天，但還是墮落了，標準的小薄本劇情。
      • 在那之前有和雅爾潔約定如果她墮落了，那就由雅爾潔殺掉她。

雅娜絲編輯

  • 2代最強同伴
    • 本人就是新的夜王，只是立場變成保護世界對抗邪妖
      • 基本上在遊戲中算是抱雅爾潔最多次的女性[4]
    • 同伴(愛人)琉莉迪絲還活著，沒有登場
      • 不過琉莉迪絲現在也3X多歲了…(1、2代中間隔十年多以上)
    • 「……我説，如果我猜得沒錯，雅爾潔你是不是喜歡莉莉雅娜？」雅爾潔你個笨蛋

從魔們編輯

  • 這邊是二代的，一代的特色沒二代多
尼祿（ネーロ）編輯
  • 雅爾潔她們小時侯在ユーラルム遇到的黑色野貓型善良邪妖，雅爾潔變成半妖後在戰鬥中重逢，作為第一個從魔收入隊伍。
    • 本作唯四(DLC不算)有正常語音的從魔，其他都是叫聲
      • 叫雅爾潔她們『御主人』
    • 專用燒城鎮巨蔓用從魔，對邪妖用縱火狂
    • 究極進化會變噴火的天使白貓
フィーユ編輯
  • 在聖者之森的花田解放的從魔
    • 超會睡的，一天到晚都在睡
    • 輔助型的從魔
      • 語音意外的可愛
シャルフ編輯
  • 在學園解放的雷狼型從魔
    • 團體意識很強，所以很能幫上忙
    • 專門電爆電門機關的從魔
      • 官設原本是要配給傲嬌女露恩的專用從魔，但廢案了。
スポラ編輯
  • 礦山解放的從魔之一
    • 台詞全部是片假名，魏延
      • 本作從魔技能威力最恐怖的一個(能一擊幹掉強力種邪妖整條血)
      • 難怪從魔大家都會自動躲她…
カエデ編輯
  • 旅館『エテルナ』的經理和風日本人偶從魔
    • 在旅館工作的式神手下會跟著進化變樣子
    • 會對偷賴的式神使用修正咒術(無誤)
      • 認識アリス和シュネー(雜誌DLC)
      • 法米通雜誌DLC給的那個從魔ヒナギク是她的師父
        • 和アリス合稱本傳有配音台詞最多的從魔組
アリス編輯
  • 在聖者之森救到的西洋風慠嬌人偶從魔
    • 本人的形象是照那個非常有名的童話故事アリス作的
      • 據カエデ說她其實很怕一個人獨處。
      • 和カエデ一樣是操作手下(兔子兵)打人
      • 台詞好多
    • 究極進化會變真・アリス
プニ編輯
  • 初回特典附贈的從魔，來歷…大家都知道的。
    • 第一次登場是從旅館泳池的出水口流出來的
      • 對式神來說是種天敵的存在(濕氣)
    • 進化會變黃金色，究進是那個經驗聚合體的金屬色
