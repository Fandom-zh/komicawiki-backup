賽馬娘 Pretty Derby/登場人物介紹-2星賽馬娘

賽馬娘 Pretty Derby/登場人物介紹-2星賽馬娘
本條目介紹賽馬娘 Pretty Derby收錄的賽馬娘，歡迎有愛的訓練員們幫忙補完。
本條目及角色條目的馬娘中文翻譯基本採用香港賽馬會的官方翻譯，如沒有官方翻譯則採用常見翻譯（使用時請標明暫譯[1]）或遊戲繁體中文版官方翻譯（使用時請標明遊戲中譯[2]）。
本條目或許帶有些二次設定。

馬娘一覽編輯

草上飛（グラスワンダー）編輯

  

  • CV：前田玲奈
  • 美國出生、但受喜歡日本文化的父母薰陶下十分有大和撫子氣息的馬娘。
    • 性格溫柔善解人意，在同學之間人氣很高，但由於無論什麼事情都能靈活應對會讓人感覺有點腹黑。
    • 據本人說法，對茶道、插花、書法、日本舞，甚至薙刀術都有相當程度的造詣，即使是長時間的跪坐也不會因為腿麻而動不了。
    • 心底有著不論在什麼事情上都絕不認輸的頑強個性。
      • 對自己被稱為丸善的後繼者「怪物二世」非常在意，這個名號就意味著永遠的第二，因此將丸善斯基視為超越的目標。
    • 丸善斯基兜風行程的受害者，隨機幹勁下降的其中一個事件是意外答應了丸善的兜風邀約，結果被丸善的開車技巧嚇得半死，回來後叮囑訓練師要小心不要輕易上了丸善的車。
    • 因為下述的動畫劇情影響在二創中經常被冠上病嬌屬性，更與角色歌「Secret GRADUATION」的歌詞完美對應
      • 在某些扭曲二創中和露崎真晝成為CP
      • 「那我呢」
    • 有一位住在美國的妹妹，目標是要超越草上飛。
      • 原型應來自草上飛的全妹「Wonder Again」，雖然成績沒有草上飛那麼耀眼，但亦有GI2勝的成績，現實中的賽事「Wonder Again Stakes」便是以她為名。
  • 動畫裡剛開始由於骨折需要養傷而跟特別週經常有交流，之後對特別週的事情特別關注。黑化的開端
    • 復原後本來期待與特別週的對決卻因為當時她一直在想著無聲鈴鹿的事而感到生氣，在寶塚記念中用實力碾壓了特別週獲得冠軍。
      • 有趣的是歷史上草上飛的騎師與在1993年終結目白麥昆天皇賞（春）三連冠的米浴同為的場均，而特別週的鞍上人與目白麥昆一樣是武豊（動畫第2季第8話有稍微捏他了這段關係）
      • 另外兩馬的祖父一樣是被稱為「世紀反派（世紀の悪役）」的Roberto，背刺的血脈。[3]
    • 於特別週回復狀態後在有馬記念同樣把特別週壓在2位獲勝。
      • 原型也是兩度戰勝特別週，直到特別週生涯的謝幕賽都未能反勝牠的一生之敵，大概也是動畫劇情的由來。
      • 近年鄉民提到牠們時大都會提及2014年的   兩雄再♂會
    • 然後便成為了特別週的飯友，第二季經常和特別週一起吃大碗飯
      • 草女士：屆到了
    • 其實在廣播劇中就時常陪特別週去吃甜點，也因此體重增加不得不晚上外出運動減肥，此舉動害得亞馬遜擔心找富士奇石討論而鬧出笑話。
      • 原型很會吃、身型壯碩體重也不低，團隊常常為替牠減重而費盡心思（與敵手特別週剛好相反）。
        • 最後一次有馬勝利後又再一次受傷陷入低潮，日經賞復出時為其生涯最重的體重，最後還是因傷、體重難減跟狀態無法回復而退役。
  • 退役後成為種牡馬，生涯20年一共產出了5匹GI級產駒。2020年正式退休，現於北海道明和牧場漫渡餘生。
    • 種牡馬時期曾與黃金船及愛麗數碼當鄰居。
    • 退役後依然盡顯吃貨本能不停在牧場內吃草，特別喜歡吃蒲公英（所到之處的蒲公英都會被牠吃到一棵不剩），連中の人也見識過
    • 即使在被週日寧靜系統治的日本賽馬界依然能靠著「銀幕英雄（スクリーンヒーロー）」→「滿樂時（モーリス）」→妙發靈機讓其父系血脈傳承下去到第四代、繼續產出GI級產駒。
  • 馬名的來源是馬主的冠名（グラス）[4]+ Wonderful的縮寫Wonder（ワンダー）
    • 嚴格按照來源翻譯的話會得出「玻璃奇蹟」這某種意義上可以總結其職業生涯的名字
  • 高凌風
  • 除一般勝負服外，還有幻想世界造型（3星，2021年6月追加）

神鷹（エルコンドルパサー）編輯

  

  • CV：高橋未奈美
  • 來自美國的開朗馬娘，喜歡觀看格鬥比賽和超辣料理。所以是歸國子女屬性
    • 總是戴著繼承自父親的半面罩為其特徵。
      • 脫掉時會變成原本內向害羞的個性，變成普通的清楚美少女
    • 會偷偷在宿舍裡養寵物，這一點讓室友草上飛感到困擾。
      • 從育成劇情可得知寵物是名為「マンボ」[5]的安第斯神鷹[6]
        • 雖然神鷹飼養鷹（Cóndor）看起來是一件很理所當然的事情，但無論如何在宿舍裡飼養一隻翼長3公尺的大鳥且不被發現也太不可思議
        • 況且安第斯神鷹屬於日本現行的動物愛護管理法明令禁止作為寵物飼養的物種
  • 經常說西班牙語，有在句尾加上「デース」的口癖。
    • エル、日本語でおk
    • 教壞特別週的元兇之一
  • 原型是在美國出生、於日本訓練的外國產馬。
    • 以無敗之姿贏得NHK一哩盃（NHKマイルカップ），同時作為三歲馬首次增程2400公尺即贏得日本盃。
      • 遊戲中如果未在東京草地2400公尺的比賽中獲勝，然後在日本盃以第一名衝線的話會觸發特殊實況對白。所以想聽特殊實況的話就得小心之前的德比
    • 四歲轉戰法國獲4戰2冠2亞佳績，挑戰凱旋門賞憾負望族獲得亞軍，從此開啟了日本賽馬業界對凱旋門賞近乎瘋狂的衝擊
    • 遠征法國後退役並力壓特別週成為1999年度代表馬，在多年的纏鬥後又於規則變更後的2014年力壓特別週被選為JRA顯彰馬，令特別週成為無冕王的元兇
      • 特別週：Why always me?
    • 在日本只有無聲鈴鹿曾在1998年每日皇冠中戰勝過牠。
    • 當時JRA制度限制外國出生賽駒不能跑部分GI比賽，動畫和遊戲裡則超越歷史之壁讓她跑日本德比和天皇賞（秋）。
      • 在動畫第一季出賽史實中並未參加的天皇賞（秋）並因無聲鈴鹿骨折而獲得冠軍，可能是因為史實中該場的冠軍「越位陷阱（オフサイドトラップ）」與神鷹是同馬主的關係而作為替代角色上場也間接讓越位陷阱幾乎確定無法馬娘化。
  • 馬名來自著名秘魯民歌「山鷹之歌（El Cóndor Pasa）」
  • 除一般勝負服外，還有幻想世界造型（3星，2021年6月追加）
  • 一天配種20多萬次的大種馬
    • 2021年12月15日更新後，育成時可以無限次借用排名前100名的玩家登錄的殿堂馬娘。
    • 而第一天（即15日當天）可借的馬娘中只有其中一位神鷹是三星速因子，這位神鷹因此成為了各大玩家爭相借用的對象，截至排名更新時已被借用逾20萬次。

氣槽（エアグルーヴ）編輯

  

  • CV：青木琉璃子
  • 特雷森學園學生會副會長，魯鐸象徵的左右手。
    • 基本上學園內的大小事務都會由學生會處理，而氣槽還會另外負責照顧有個人內情的馬娘，常常公務纏身。
      • 在魯鐸象徵的育成中就曾因為過勞而倒下，個人故事中亦因為過於繁忙而影響到比賽訓練。
    • 在舊版介紹裡提到有非常討厭男性的潔癖[7]，後來可能是為了避免跟目白多伯的人設重複而取消了
  • 中性化馬娘其之二。
    • 動畫中因為打扮風格很像所以與好歌劇、富士奇石一起作為管家咖啡廳的看板。牛郎
  • 幼名是ベロちゃん
    • 源於原型馬幼時會常常舔人，頻度之高讓牠得了個ベロちゃん的別名。
  • 其原型於1996年在橡樹賽奪冠，與母親實現了兩代制霸橡樹賽的實績。
    • 1997年得到天皇賞（秋）冠軍、日本盃亞軍及有馬記念季軍，被選為該年年度代表馬。也因為與牡馬競爭下仍有佳績（大部份賽事名次均在3甲之內[8]，1997年跑出3連勝）而有「女帝」的稱號。
  • 弱點是閃光燈
    • 動畫第2季第1話裡被遊客的手機的閃光燈閃到眼睛。
    • 原型在1996年參加秋華賞時於賽前被閃光燈嚇倒，結果只得第10位之餘還骨折；自此以後日本競馬場內禁止使用閃光燈。
    • 在育成中亦有機會發生相關劇情。
  • 在榮進閃耀和東瀛佐敦的育成賽事中會出現帶有負面技能「ゲート難」的氣槽，被認為是捏他其產駒「統治地位（ルーラーシップ）」。
    • 其活躍時間為2009～2012年，與榮進閃耀、東瀛佐敦等馬相近。
    • 黃金船勝出的2012年有馬記念也是牠的退役戰，但牠卻開場就在閘內站起來以至於前期跟隨黃金船吊車尾，可惜黃金船的育成賽事裡沒有ゲート難氣槽
    • 因為統治地位的馬主為Sunday Racing，大概因此大人的理由未成為賽馬娘故只好由親媽代子出征了
  • 馬名的來源是馬主的冠名（エア）[9]+ Groove，其中Groove取「令人興奮」之意。
    • 雖然是完全偏離了馬主原意的錯誤譯名，但氣槽至今仍被廣泛使用。
  • 除一般勝負服外，還有新娘造型（3星，2021年5月追加）
  • 夢女四天王之三

重砲（マヤノトップガン）編輯

  

  • CV：今村彩夏→星谷美緒
  • 憧憬著成人女性，但遇上令她興奮的事物就會露出小孩子一面的天才馬娘。
    • 父親是飛行員，所以說話時會帶點飛行術語。
    • 有著驚人的直覺和超人般的學習能力，能馬上理解事情的答案，但也因此很容易感到無聊。
      • 為了不使重砲幹勁下降，訓練員總是在調整訓練清單和跟重砲約會而疲於奔命。
  • 原型曾以4種不同跑法贏下4次GI，被稱為擁有變幻自在的腳質。
    • 因此遊戲內4種跑法適性都很高（領放A、前列A、居中B、後追B），專屬稱號「変幻自在」更是需要使用4種戰術各贏出一場GI賽事。
    • 其騎師田原成貴在比賽開始後都會讓牠自行決定怎麼跑，自己只會在重要關頭和最後階段給予指示。
    • 最有名的一戰是GⅡ阪神大賞典，該賽事裡與成田拜仁於最後600公尺並排領前，最終輸頭位得第2名，兩者衝線時已經拋離第3的ルイボスゴールド9馬身。
  • 原型也是一匹相當任性的馬。牧場人員山崎努表示，重砲就像一個很任性的小孩，一旦不喜歡某件事就什麼話都不聽，尤其討厭打針，看到獸醫就會直接逃跑。
    • 而重砲能夠變換自在的駕馭四種不同戰術，除了本身聰明外最大的原因也是這個任性的性格。賽馬的腳質適性與本身性格有頗大的關係，重砲的任性使得於牠在賽場上的心情與狀態極為不穩定，若是事前基於某種狀態而訂立了戰術，可能會因為實行當下重砲不願配合導致表現不佳。
  • 動畫裡是東海帝皇的室友。但每次都在睡覺，從來沒被叫醒過
  • 馬名的來源是馬主的冠名（マヤノ）[10]+ 電影《捍衛戰士》
    • 勝負服是飛行夾克也是這個緣故。
      • 夾克背部被頭髮遮著的花紋在遊戲中也有忠實呈現，然而上面的國旗和圖案不知是否因為要考慮到日後陸服的營運或者可能出現的版權問題而在2021年8月29日更新後全數被刪除或更換。
    • 大概也因為此一關係而獲選為電影續作《捍衛戰士：獨行俠》於日本的宣傳大使。
    • 摩耶的好大一支槍
  • 除一般勝負服外，還有新娘造型（3星，2021年5月追加）

超級小海灣（スーパークリーク）[1][11]編輯

  

  • CV：優木加奈
  • 十分有包容力的馬娘，比起自己更喜歡看見對手的成長。
    • 由於老家開托兒所，影響了小海灣的性格喜愛照顧並帶給他人幸福。
    • 母性爆發時甚至會連自己也會跟著被照顧對象一起變得母湯。
      • 連帶造成二創中非常容易成為母性暴走的角色，受害者通常是玉藻十字、成田大進、小栗帽以及訓練員
      • 走る西松屋、でちゅねの悪魔
    • 人設是巨乳+溫柔成熟、散發著母性光輝的形象因此也被稱為超級大海灣
      • 勝負服的色氣度（不論原案抑或實裝版本）各種意義上也很糟糕
      • 大灣區
    • 喜歡的類型是愛撒嬌的人。[12]
      • 就連選拔賽結束被挖角的時候，也是無視了其他主動搭訕的訓練員，走向在人群中卻弱氣不敢開口的新人訓練員來個逆挖角。
  • 與成田大進是室友，反叛期的女兒和母親
  • 設定上不擅長畫畫，然而2021年天皇賞（春）的賀圖的畫師是小海灣
  • 是少數能承受丸善飆車的馬娘，曾因訓練員說「需要清醒一下頭腦」而帶著訓練員參加丸善斯基的兜風行程。
  • 原型是第一匹讓日本傳奇騎師武豊獲得GI優勝的賽馬。天才賽馬遇上天才，吃了武豊GI童貞的女人[13]
    • 不過那一場菊花賞差點因為出賽順位排名第19而無法參賽，幸好其中一匹可參賽的賽馬避戰才能令牠可以參加。[14]
    • 與小栗帽、稻荷一號是同世代宿敵，並稱「平成三強」。
    • 有腹瀉的老毛病，準備出道時因為腹瀉發作要延遲出道，退役後也因為腹瀉不止急速衰老而病故。
      • 在遊戲的3月堇錦標賽後會發生事件「細小的裂痕（小さなほころび）」：小海灣身體出現明顯的不良狀態，使所有訓練失敗率+5%，可能即為對此的捏他。
      • 一旦勝出菊花賞便會被治療，取而代之的是提升訓練成功率的「大輪の輝き」。
  • 關於馬名的來源暫時看到兩種解釋：
    • 其一是「今は小川でも、いつか大河になって欲しい」，即馬主寄予了今天的小川將來能成為大河的期望。
      • 有香港報章譯作「超級溪流」，遊戲中譯「超級小溪」大概也是取自此說。
    • 其二是馬主在購得該馬時正在打高爾夫，便靈機一動以第五木桿（Creek）來命名。
      • 但上述兩種說法都跟小海灣無關，而且馬娘化後乳量有夠大的
  • 除一般勝負服外，還有萬聖節造型（3星，2021年9月追加）

詩歌劇（マチカネタンホイザ）[1]編輯

  

  • CV：遠野ひかる
  • 性格十分認真正面、總是全力前進的高材生型馬娘。
    • 然而競賽成績不振，也總是會在重要的地方凸鎚，有著天然的一面。
      • 本人對自己的評價是普通的孩子，因此總是對個性強烈的馬娘抱有憧憬。
    • 人設特點是淺褐色的頭髮以及套在耳朵上的帽子。
    • 遊戲中與待兼福來是室友，吵鬧的待兼房
    • 因為名字太長所以被雙渦輪和優秀素質簡稱為「マチタン」，待兼福來則叫她「おマチさん」不過嚴格來說待兼福來也是おマチさん
  • 於一周年卡池與北部玄駒一起實裝，是遊戲開始營運後首次新增2星角色並引起玩家猜測狄杜斯與雙渦輪會否同樣以2星實裝
    • 北部玄駒以性能和角色故事吸引訓練員，她則負責賣萌萌倒訓練員
  • 動畫中是Team Canopus的一員，運氣相當差，在拜年敲鐘時祈禱自己以後一年運勢變好卻直接被鐘砸到臉而流鼻血。
    • 之後經常會有被意外事件牽連的演出，厄運纏身
    • 原型曾經在日本盃前流鼻血而被迫退賽，不久後又因為誤吃蜘蛛引發蕁麻疹[15]取消參加有馬記念（動畫裡有相關捏他台詞）
      • 實裝後此事跡如實反映到育成事件中，在第三年11月後半日本盃開跑的回合會發生詩歌劇流鼻血而被迫退賽的事件，並使該回合無法出賽。
  • 馬名的來源是馬主的冠名（マチカネ）[16]+ 華格納創作的歌劇《唐懷瑟（Tannhäuser）》
    • 關於譯名的問題，唐懷瑟作為實際存在的歷史人物以詩人之名著稱，以他為主角的歌劇當然就是詩歌劇了幹編不下去了[17]。
    • 民間的譯名還有「待兼唐瑟」，在四字的限制內盡量保留原名的所有元素。
    • 遊戲中譯為「待兼唐懷瑟」，比上面的民間譯名多一字。
  • えい、えい、むん！
    • 源於遊戲中舉辦與馬娘動畫版的聯動活動時，任務頁面中詩歌劇的台詞。
    • 這個還是監督指定的
    • 活動結束後的台詞則是換成了「ばばんば～」，不過毒性比えいえいむん差一截反而比較影薄
    • 後來NHK也來了一次えいえいむん，但那又是另一回事了
    • 現在可以在官網的角色頁面中聽到該台詞。
