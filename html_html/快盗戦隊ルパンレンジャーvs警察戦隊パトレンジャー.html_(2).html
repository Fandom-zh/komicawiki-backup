作品介紹

  • 超級戰隊系列第42+43作，2018年2月11日起首播，全51話
  • 監製是宇都宮孝明，主編劇是香村純子
    • 參考宇都宮的過去的經驗，本作兩位紅戰士一起開無雙的機率會高達八成
  • 旁白是安元洋貴，道具音效是知名旁白三村ロンド
  • 雖然不是紀念作，每話的ルパンコレクション都是前輩的道具
  • 本作於2019年獲得銀河賞月間賞，為戰隊系列首次，也是繼特搜戰隊的星雲賞、烈車戰隊的日本鐵道賞後第三部獲獎的戰隊作品[1]

故事概要

  • 由稀代大怪盜「亞森・羅蘋」留下的神秘寶物群「ルパンコレクション」，被來自異世界的犯罪集團「ギャングラー」奪去，並利用コレクション做出各種壞事。
  • 不能讓ギャングラー逍遙法外…！為了奪回重要之人而戰的「快盗戦隊ルパンレンジャー」、為了守護世界和平而戰的「警察戦隊パトレンジャー」，絕對不會有交集的「快盜」和警察，宿命的宿敵（們）如今開戰！
    • 時而對立，時而並肩作戰的兩隊的戰隊，您會為哪一隊打氣？

角色介紹

魯邦連者（ルパンレンジャー）

  • 為了奪回在過去被捲入ギャングラー的惡行而失去的「重要之人」而戰、被民眾們視為俠盜的三名快盜
    • 根據三人的回憶，他們的「重要之人」被一名戴著帽子的怪人以結冰後粉碎的方式殘忍殺害
    • 平日在GSPO日本分部基地附近的法國餐廳「Bistrot Jurer（ビストロ・ジュレ）」打工，但收到コグレ傳來的指示就會馬上打烊出動
      • 但真正有在工作的只有透真和初美花，魁利幾乎天天都在外面還會翹班去照顧流浪狗，還被透真吐槽「不是一直都這樣嗎」
      • 每次出動都會變裝，並佩戴具有擾亂臉容辨識機能的眼罩
        • 有時候不會按照自己的代表色穿[2]
        • 第48話在ドグラニオ的要求下在對全世界直播的「處決表演」上脫下眼罩，正式向世人展示自己的真面目
    • 身份暴露後被GSPO全國通緝，他們也放棄Bistrot Jurer於別處另立據點活動
  • 第50話三人為與ザミーゴ決一死戰而進入ドグラニオ的保險庫空間內，擊敗ザミーゴ後用コグレ給的書本將裡面的寶物轉移出去，雖協助巡邏連者擊敗並拘捕ドグラニオ但他們終究無法從裡面出來
    • 一年後，由勝利、彩、詩穗所組成的新的快盜三人組找到了在劇場版結束後被遺忘許久的ジャックポットストライカー，利用其能力打開ドグラニオ的保險庫解放三人
    • 為搜集餘下的ルパンコレクション再度現身在與ギャングラー殘黨戰鬥的巡邏連者面前重現第1話單刀直入的場面，兩隊戰隊圍繞ルパンコレクション的爭鬥將會繼續持續下去
  • 擁有在外出買菜的時候必逢ギャングラー怪人的特殊體質[3]，後續部分話數也保持著這個慣例
  • 初期戰士三人名字的頭文字組合起來是「快盜（かいとう）」


  • 夜野魁利/ルパンレッド【演：伊藤あさひ】
    • 為了奪回兄長・勝利【演：柴浩二】而成為快盜的青年，說話時會在句尾加上法文
      • 幼時曾經憧憬於兄長的優秀而立志成為像他的人，長大後卻比不上兄長就一時意氣離家出走，目睹兄長在眼前遇害後為此相當愧疚
    • 性格冷酷不好戰、看似大膽的行事也以回（奪）收（取）ルパンコレクション為先，不會貿然衝動行事也有意外地通情達理的一面[4]，但一旦被燃起鬥心就絕不放過眼前的敵人
      • 表面上說為了達成目標不會去拯救同伴、只要留下一個人完成使命即可，但在第7話卻為了救初美花而被メルグ吃掉[5]
      • 在ルパンマグナム給予的試煉裡甚至為達成目標而對兄長的幻影開槍，認為自己「果然最適合成為快盜」以表達早已不顧自身的想法
      • 當受到ヤドガー的「強制歸家」能力時並未受到影響可知其實他不信任身邊的夥伴也不認為自己還有「歸宿」，直到ジュゴーン事件後才注意到自己藏頭藏尾的做法為身邊人帶來無謂的擔心，例如擔心他在餐廳吃太撐後被不知從哪飛來的啄木鳥刺破肚皮爆發四散之類的
    • 有在開玩笑的時候會捏初美花鼻子的習慣，這舉動曾經被つかさ發現並懷疑他們三人
      • 但讓つかさ崩潰的卻是之後查出他們身份時才驚覺之前托管的三個小孩原來是被變小的快盜們
    • 因戰鬥理念的沖突以及私下生活的關係，與圭一郎有相當多的接觸
      • 第5話與パトレン1号對決中遭對方質疑行徑錯誤時，表示自己「除了成為快盜已經沒有其他選擇」的想法
      • 第30話為回收トリガーマシンスプラッシュ再次與パトレン1号對決，卻被對方委托「去拯救市民」時的言辭想起兄長生前對自己說過相同的話，對自己的做法產生動搖
      • 在共同經過一些事件（「結伴」旅行、ヤドガー事件）後開始變得會無意地將對自己友好的圭一郎與兄長的身影重疊，之後因各種機緣巧合被圭一郎試圖探求內心想法反而因此鬧翻產生排斥，使他討厭與對方扯上關係
      • 在身份公開以後不得不離開Jurer，但出於調查目的依然主動在圭一郎面前現身重現第1話初次見面的一幕，與對方交談後發現對方已經無意逮捕自己後產生動搖，憤然離去時被告知被ザミーゴ帶走的人會被造成化身皮囊之事
    • 在劇場版《VS騎士龍》中，依然是通輯犯，自稱「魅」，開了偵探所尋找餘下的ルパンコレクション
      • 不過大戰後因為已經沒有了ルパンコレクション，似乎跟巡邏連者也不再爭奪


  • 宵町透真/ルパンブルー【演：濱正悟】
    • 為了奪回未婚妻・大平彩【演：柚木渚】而成為快盜的腹黑紳士（公式）
    • 冷靜毒舌，經常板著臉吐槽和作出各種危險行動，對ギャングラー的態度是既要打倒他們也要收回寶物，對他們的憎恨甚至會失去理智
      • 第3話在巡邏連者初訪餐廳時想偷走他們的VSチェンジャー，甚至想單身攻入GSPO強搶，在魁利的開導和想起未婚妻「迂迴可開拓出新的可能性」的話語後才冷靜下來
    • 吐槽控，經常對當時場景進行一針見血的吐槽
    • 被ピョードル以寶物的力量洗腦成為門生的時期是他永遠不想再提起的黑歷史
    • 在劇場版《VS騎士龍》中，在其他地方重開餐廳，因為依然是通輯犯的緣故，與未婚妻保持距離
      • 但不幸被報答カナロ而邀去喝茶的つかさ撞破，似乎又要換地方了
      • 大戰結束後，原本應承アスナ做法國菜給她吃，但對方點的是麻婆豆腐，雖然最後還是板著臉答應
  • 早見初美花/ルパンイエロー【演：工藤遥】
    • 為了奪回女性摯友・一ノ瀨詩穗【演：小松もか】而成為快盜、喜歡打扮的妹系女子
      • 童年遭同學們嘲笑時詩穗出面庇護，兩人因而成為摯友
      • 詩穗遇害前一刻推開了她使其未遭毒手，反讓初美花深陷被他人保護而失去重要之人的愧疚，為了成為快盜而高中退學
    • 性格偏坦直率容易露出真實情感，但其實心思纖細，同時也是一個努力家
      • 在與咲也單相思的相處中發現對方一直相信著自己，在被其盤問過去的事用於判斷真身是否為快盜時對自己的做法產生了猶豫
      • 同時與巡邏連者多日的相處也產生了希望解決ザミーゴ後也能友好相處的願望，但在身份公開後依舊選擇了離開Jurer
      • 冒失鬼屬性[6]，但率真的性格也經常獲得他人的認可和理解
    • 在劇場版《VS騎士龍》中比本篇時更為成熟。除了維持被通輯外，現況未明，但按つかさ的說法，應該因為自己的身份遠離了詩穗。
      • 大戰結束後跟另外兩人同樣沒有明顯與魯邦連者繼續戰鬥的理由，對於咲也的追求似乎還是沒有答案
    • 曾經是偶像（絕對無誤）[7]
    • 重度LLer（也是無誤）[8]

巡邏連者（パトレンジャー）

  • 隸屬國際特別警察機構（GSPO）的戰力部隊，初期三人當中圭一郎和つかさ是警察學校時已相識的同期生，咲也則是他們的後輩
    • 兩屆警校人員裡只有三人通過審核，第四個人則是臥底
  • 初期戰士三人名字的頭文字組合起來是「警察（けいさつ）」


  • 朝加圭一郎/パトレン1号【演：結木滉星】
    • 為了剷除犯罪努力實幹的熱血警官
    • 貫切自己的正義熱血得做事不懂變通，熱血到不容任何罪犯的存在，但依然會堅守職責和做事懂分緩急
      • 在快盜們取得ルパンコレクション後逃走時，つかさ曾經問他是否要追擊，結果他回答要優先討伐危險性更高的ギャングラー
      • 第5話初戰ルパンレッド時表明自己「不論你們有什麼理由，成為快盜的那一刻開始便是錯誤」的想法
        • 輸掉對決後為再戰甚至像快盜們般刻意放跑ギャングラー怪人，遭つかさ打了一巴掌並被她要求想起自己在剛加入GSPO時說過的話，之後在戰鬥中想起自己當時說的是「我們所有的是為了保護市民才存在的力量，絕對不能搞錯使用的方法」記起初衷
        • 事件過後為自己的失職向ヒルトップ管理官道歉，同時思考對決時ルパンレッド所說的話的意義，想了解對方藏在面具下的真正目的
      • 第30話從走私組織處回收トリガーマシンスプラッシュ時遇上要來搶寶物的ルパンレッド，雙紅再戰之際為應對カンクス引發的大火災甘願讓移動速度較快的對方拿走它去拯救市民
      • 第32話在與ノエル的一對一決鬥中雖然佔上風，但在要作決勝一擊時卻刻意留手而反敗
    • 在ナリズマ事件以後開始調查所有連環失蹤案件中的受害者和ギャングラー活動之間的關係，意外發現魁利三人親屬的名字，才終於開始懷疑魁利等人
      • 經過仔細調查後確認三人正是快盜們的真身，卻因此陷入公私矛盾
        • 在魁利身負重傷依然要與ザミーゴ決一死戰時本想阻止他、甚至表示自己願意放棄警察身份幫助他們但被對方回絕，之後為阻止ドグラニオ不得不與他離別
    • 在與ドグラニオ的決戰中雖得ヒルトップ管理官下格殺令但依然想救出在他保險庫內的快盜三人，內必掙扎最後將他拘捕送入監獄
      • 一年後則在剿滅ギャングラー殘黨的行動中再遇魯邦連者，得知他們平安無事同時發現他們繼續做著快盜，於是又一次展開ルパンコレクション的爭奪戰
    • 戀愛白癡（公式）[9]
    • 其實是現八郎的徒弟（大誤）
  • 陽川咲也/パトレン2号【演：横山涼】
    • 三人組的後輩（悟退出後的補員），擅長駕駛、射擊和與人交際，自稱是被讚會成長的類型
      • 警校時期成績不佳甚至曾想過放棄，但因為受到憧憬的兩位前輩鼓勵發奮上進才有今天的成就
    • 三人裡唯一對快盜們有好感的，在初訪Bistrot Jurer時更對初美花一見鍾情
      • 在與（被）初美花約（利）會（用）過後不經意地成為了被快盜用作刺探GSPO情報的人物
      • 在ノエル的幫忙下再約會一次，但被對方以「你是個好人」而拒絕
      • 在GSPO鎖定初美花為快盜三人真身之一時急於想替她證明清白，甚至想約她出來約會想問清楚
        • 後來看到初美花自曝身份時悲痛欲絕，又為她因奪回親友而成為快盜之路而憂傷
        • 但在在初美花和透真執意要去與ザミーゴ決戰時，與つかさ一同將收押的VS Changer交還給二人
  • 明神つかさ/パトレン3号【演：奥山かずさ】
    • 既冷酷又男子氣的三人組中的紅一點，但十分喜歡布偶之類的可愛東西
      • 當初為了繼承父業、[10]優良薪金和待遇而加入GSPO
      • 幼年時雙親去世而與祖父一起生活，為了過著和平的生活而立志成為一名警察
      • 一直覺得自己的興趣與警察身份不合所以刻意隱藏，但其實早在警校時期已經被圭一郎知道了
    • 對快盜們態度中立，卻是最早接近揭露快盜們真身的人（雖然被對方成功逃過耳目）
      • 在特殊情況下會通融協助快盜們
      • 後期也在初美花和透真執意要去與ザミーゴ決戰時，與咲也一同將收押的VS Changer交還給二人

追加戰士

  • 高尾ノエル/ルパンエックス、パトレンエックス【演：元木聖也】
    • 由GSPO巴黎總部派出潛入魯邦連者的臥底警官，製作VSチェンジャー及トリガーマシン1、2、3号的研究員
      • ルパンエックス模式下佩戴警徽和警用肩甲，パトレンエックス模式下擁有快盜的裙擺和頭飾
      • 還會在自己的警察手帳和GSPO制服上標明自己是Undercover Police
    • 真正身份是與コグレ一樣是侍奉羅蘋家的技師，改造寶物的技術也是從此而來，因為得知GSPO擁有寶物的情報於是以研究員身份潛入加以利用
      • 因此與魁利初次見面就已知道他的身份、還潛入快盜們藏身的Bistrot Jurer
      • 與魁利等人一樣有著「想取回的重要之人」
    • 職務關係不論是快盜還是警察的戰鬥都會參與，但每當戰況傾向一方時便會馬上切換變身協助另一方，必要時還會提供エックストレイン協助作戰
      • 自己也會去奪取ルパンコレクション，但對於如何處置它們似乎有自己的主意，甚至與コグレ抬槓交換別的寶物
    • 在取得快盜和警察雙方信任的同時也被魁利和圭一郎懷疑，猶如雙重間諜一般處於相互利用的狀態
      • 然而在行動表現上明顯地傾向偏幫快盜，讓巡邏連者們開始提防他起來甚至要隱暪他執行計劃
      • 第32話為了回收實驗體身上的5個ルパンコレクション甚至要求警察協助快盜們惹得圭一郎不爽，只能以雙方決鬥解決
    • 因裡面的人的關係而懂得翻滾，以比喻方式說話，但令人難懂
    • 有著某個不可告人的秘密，第42話發現被ゴーシュ「看到」後變得十分焦燥
