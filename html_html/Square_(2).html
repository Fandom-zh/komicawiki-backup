Square Enix

Square Enix/SQEX/史克威爾艾尼克斯/方塊社/SEX/手游天尊

目錄

簡介編輯

  • Square Enix為日本遊戲開發廠商，主力遊戲類型為手機遊戲與SNS角色扮演遊戲，以最終幻想與勇者鬥惡龍聞名。
    • 特技是把舊作的遊戲源代碼搞丟、項目管理不當、把經費花在奇怪的地方、讓人不禁懷疑他們怎麼撐到現在
  • 前身為Square社與Enix社，在2003年合併成立Square Enix，以Enix為存續公司不是史克威爾吃掉Enix嗎。2008年改組成立控股公司Square Enix Holdings，Square Enix為其子公司。
    • 2005年以交換股票方式收購Taito以此進軍大型電玩市場所以ZUN和小夜現在算是SE社的員工[1]，2008年意圖收購特庫摩被攔截，2009年收購古墓奇兵系列的製作公司Eidos。
      • 因此歐美原產遊戲角色中最知名的蘿拉‧卡芙特也被納入SE社名下，因殺人如麻而進入三大惡女候選
  • Square創社於1983年，在1986年改名為Square，創社人宮本雅史[2]。在1987年於FC上推出最終幻想成名。
    • 在2001年因最終幻想電影版導致虧損，尋求諸位金主救援。Sony於此時入股。
    • 2003年與同樣碰到財務危機的Enix合併。
  • Enix為1982年由福嶋康博成立[3]。於1986年發售勇者鬥惡龍聞名。
    • 1991年以《月刊少年ガンガン》加入漫畫市場。
    • 2001年遭逢エニックスお家騒動，出版事業分割[4]。
      • 兩方一度對簿公堂，後於2003年達成和解。
      • 不過後來新連載的《鋼之鍊金術師》大受好評，危機解除。
    • 另一方面因為遊戲經營不善。在2003年與Square合併。
  • 因為是發行廠商，導致很多遊戲讓人以為都是Square Enix製作的。
    • 像是Cavia的誓血龍騎士。
    • 不過也有像是tri-Ace的女神戰記和銀河遊俠這樣的案例。
  • 除了遊戲之外，還設有出版部門，其中以上述的《鋼之鍊金術師》最為知名。
    • 2014年發生高分少女事件。安定的出版部
    • 《月刊少年ガンガン》是一切罪惡的根源
  • 於2004設立了SQUARE-ENIX小說大賞，現在改名為SQUARE-ENIX輕小說大賞
    • 也有發行不少輕小說

Square Enix作品編輯

  • 超時空之鑰
    • 嚴格來說是合併之前的合作遊戲，但是卻意外比樓上幾位還有人氣
  • 誓血龍騎士系列
    • 由橫尾太郎領銜製作的ARPG。
    • 除了畫面和遊戲性其他都很好（尤其是音樂）的遊戲，不管哪一代都是虐主角及玩家不手軟的
      • 雖然有多週目劇情及分歧結局，不過下一個結局基本上只會更致鬱，甚至連武器的背景故事都很惡趣味。
      • 而其精神續作《尼爾》依舊繼承了這種虐主角不手軟的風格。
  • 最後的遺跡
    • 即使到了現在，還放在PS3未發售名單上...
    • 仁王，食人大鷲:你要代替你的學長嗎
  • 混沌之環
    • 以手機為出發點的RPG，特殊的風格廣受玩家好評。
  • 人生街道
    • 主打DQ與FF共演的雙六遊戲
  • 百萬亞瑟王
    • 包括擴散性百萬亞瑟王和乖離性百萬亞瑟王等
    • 然後被玩死了
  • 勇氣默示錄
    • 主打傳統FF系列的模式，詳見主條目
  • 歧路旅人
    • 採用HD-2D的點陣風格呈現懷舊遊戲的美術畫面，詳見主條目

相關工作人員編輯

製作人/導演/編劇/主要開發人員編輯

  • 坂口博信（前員工）
    • 最終幻想系列的生父。
    • 但因為電影事業的失敗，離開Square Enix。
    • 現在在Mistwalker擔任總裁做手遊，結果手遊新作就炎上了。
      • 雖然已經離開，但是常常跟SE社借人
  • 北瀨佳範
    • 主要從事導演和編劇工作。
      • 意外的很會回應粉絲。所以FF10後面才會這樣
      • 爛好人
  • 田畑端（前員工）
    • 手上所執導的作品除了沒做完的FF15基本上都有品質保證。
      • 專業級PSP遊戲製作人。雖然第三次生日因為某人的關係獲得差評，不過大家不會介意
    • 據傳接下工作後正在想辦法幫FF15增加女性角色的出場機會
      • 結果FF15還是爆炸了，為了逃離憤怒粉絲的追殺選擇跑路，還讓DLC斷尾
  • 野島一成
    • 工作是遊戲編劇
    • FF7開始的正傳有不少劇情都是出自他的手筆，尤以FF10為最高峰。
      • 結果後來由他主筆的FF10-2.5小說和FF10/10-2HD版附送廣播劇，因劇情把10-2的完美結局給破壞殆盡+所有角色全部崩壞而差評如潮，完全淪為黑歷史。
  • 伊藤裕之
    • ATB系統的開發者。專業除錯專家
    • 通常經手的作品戰鬥系統都很不錯，但也造成沒有他戰鬥系統會崩壞的情形。
    • 比起劇情方面，更注重在遊戲性。
  • 松野泰己（前員工）
    • 手上持有兩個被Fami通評為滿分40分的遊戲。[5]
    • 因為生病的關係，離開了Square Enix。結果搞得太十二變成了莫名其妙的遊戲
    • 劇本風格比較嚴肅晦澀，即使是莫名其妙的FF12也能明顯感到與其他系列作的區別。
    • 在SE社經手的幾作[6]均使用大致相同的世界觀，之後被統合為“Ivalice Alliance”。
    • 非常喜歡皇后樂隊。
      • 甚至將兩首皇后樂隊作品的曲名當作副標題加進自己開發的皇家騎士團遊戲上。
    • 後來被吉田直樹請回來為FF14的支線寫劇情
  • 鳥山球求めない
    • 工作是遊戲監督和遊戲編劇。但他作為主編劇的作品劇情都非常不忍說。
      • 出道作為《神龍奇兵》，其中女主角的劇情給玩家留下了嚴重的心理創傷。
      • 導致FF13系列做到爆炸和《第三次生日》劇情差評的頭號元兇。
      • 雖然經常被婊，但一直都在SE社活躍。看起來只是需要有人管住他
      • 因為他的關係，導致部分作品的開發時間一延再延
      • 近年其主導的FF7 REMAKE第9章部分的劇情改編大獲好評或許稍微有讓評價回升了那麼一點點。
    • 自我感覺良好的中二病，在FF13系列的製作期間發表了不少問題發言，例如：
      • ただの石
      • 科幻電影《阿凡達》是受到FF13系列的影響而生人家詹導醞釀這個電影都花了14年你居敢說這種大話
      • 在曬老婆現場LRFF的公佈影片內說系列女主角雷光是整個SE以至日本遊戲界的牌面某個花牌製造商表示：
    • 無可救藥的雷光廚，LRFF時不僅逼她玩換裝play+大放殺必死，還用一眠大一吋的理由給她加乳量
  • 吉田直樹
    • 原屬ENIX側，參與過DQ10的製作。
    • 為了收拾FF14 1.0的爛攤子而被拉來，結果卻創造了MMORPG史上最大的奇蹟：不僅在短短兩年內把整個遊戲打掉重做，還利用一連串內外鋪陳為無藥可救的舊版本寫下了最轟轟烈烈的結局。
    • 後來更把FF14一步一步經營成史上最成功的MMORPG之一，可說是進年來SE最令玩家信賴的製作人。
    • 本人是網遊廢人，對玩家的反響很熱烈。
    • 現在是FF16的製作人。

美術設計編輯

  • 天野喜孝
    • 著名插畫家，最早在龍之子製作公司參與人物設計。
    • 後來轉為插畫家，在FF系列早期擔任人物設計，從FF4開始亦擔當每部FF正傳的LOGO設計。
    • 現在已經是個在國際上享有盛譽的藝術家。
      • 風格以獵奇強烈的幻想氣氛為特徵。
    • 此外也經常被邀請畫插畫，但實際成品狀況有好有壞。
      • 2019年和FGO連動推出畫冊，結果除了黑貞和嚴窟王以外的角色都……
  • 野村哲也
    • 工作是角色設計和遊戲總監。
      • 因為天野喜孝的風格在當時無法用3D技術忠實呈現，所以在FF7之後開始接手人物設計。
      • 實際上FF7最初的故事梗概就是由當時還尚未擔當要職的他創作出來的。
    • 遊戲的戰鬥設計很合男性玩家的胃口，但角色設計方面卻是女性玩家。所以FF15有一堆男生是很正常的事
    • SE社的最後良心，但卻有挖坑不填繼續節外生枝的傳統藝能
    • 中二病患者，後來凡是有他的作品都有各種造語+難解的劇情設定+拉練皮帶+刺蝟頭
      • 導致王國之心系列劇情理解困難的元兇，還因此衍生出了一個專有名詞：Nomlish（ノムリッシュ）
      • 迪士尼粉絲：我就只是想玩迪士尼世界旅行，為什麼會有一堆黑漆漆的中二來亂啊？
    • 因為以上因素外加KH3劇情後期的收尾問題，經常被不滿SE近年作風的玩家帶頭開幹
    • 目前未婚，還被宇多田光調侃過。
  • 上国料勇（前員工）
    • 雖然名字看起來像是上国 料勇，但應該是要念成「上国料 勇」才是正確的。[7]
    • 最終幻想系列的美術設計。
      • 一張圖畫需要畫好幾個禮拜是正常的事情。
    • 現在已經從SE獨立。
  • 吉田明彥
    • 角色設計。
    • 基本上參與的FF作品都與松野泰己或“Ivalice Alliance”有關。

作曲家編輯

  • 植松伸夫（前員工）
    • 最終幻想系列的音樂製作人。在其手中誕生了無數的經典名曲，可謂遊戲音樂界的一代巨匠。
      • 到了十代之後開始採取跟其他作曲家合作的方式。
      • 有自己的樂隊「The Black Mages」，不過已經解散。
      • 2020年NHK的節目【全ファイナルファンタジー大投票】的音樂部門中從多達3,000首曲子中票選出的前20名，除了第5名的【閃光】（由浜渦所作）以外其他19首曲子全部都是他的作品，可見其作品人氣之旺。
    • 很常被人邀請作曲
  • 下村陽子（前員工）
    • 王國之心系列的音樂製作人，也是《寄生前夜》、《LIVE A LIVE》、《超級瑪利歐RPG》的編曲者。
      • 負責FF15的作曲。為此遊戲中還放了一個以她為原型的NPC[8]。
    • 以前曾在卡普空工作過，並參與了快打旋風2的音樂製作。現在為自由作曲家。
    • 小提琴愛好者，也很常被人邀請作曲
  • 崎元仁
    • 音樂作曲家。
    • 基本上參與的FF作品都與松野泰己（ry
    • 以管弦交響樂為主。
      • 在Final Fantasy XII之後參與的作品配樂風格都差不多
  • 浜渦正志（前員工）
    • 最終幻想系列後期的音樂作曲家。
      • 因為太愛最終幻想，所以從德國跑到日本
    • 原為聲樂科出身，擅長合唱的調性拿捏。
    • 現在在自己成立的工作室工作。
  • 石元丈晴（前員工）
    • 音樂作曲及編曲家。
      • 雖然DFF系列的原曲改編評價有點兩極，但由其全權作曲的FF零式音樂就十分好評。
    • 擅長重低音強烈的搖滾風。
    • 於2017年離職。

逸聞編輯
