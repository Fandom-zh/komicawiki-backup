故事經歷編輯

  • 因為生活都被父母給安排好，每天都活得戰戰兢兢，並努力著不讓父母失望。
    • 當然，和網路遊戲本來是沒有任何關係的。
  • 就在感到生活壓力越來越大，有些喘不過氣時知道了SAO的存在。
    • 原本遊戲是哥哥買來玩的，但SAO開始營運當天哥哥出差，戴上NerVGear的她從此不幸被捲入SAO事件中。
      • 結果來說哥哥反而逃過一劫
  • SAO（艾恩葛朗特篇）
    • 故事開始兩年前（2022年），完全虛擬實境遊戲SAO正式營運的第一天，因為遊戲設計者茅場晶彥的陰謀，與其他一萬名玩家被困在SAO的遊戲中。
    • 因為害怕死亡而躲在初始城鎮的客店，害怕自己因為被困在SAO中而無法按照父母的期望去做而對她感到失望，在覺悟到要早日破關回到原本的生活只能靠自己，便開始積極熟悉遊戲的各項操作和鍛鍊劍術。
      • SAOP則是決定自己不要輸給這個世界、至少能選擇死法而在地底迷宮無止盡地戰鬥，後被桐人救出並在解開心結後與他一同行動。於是二人不過第三層關係就演進到近乎老夫老妻
      • 為了回到現實，在初期幾乎可以用被稱為「攻略之鬼」的態度成為攻略組的要員。
    • 後來接受攻略組中第一的公會KoB的團長希茲克利夫的邀請加入了KoB，並擔任副團長。
      • 因為希茲克利夫對攻略以外的事項沒有興趣，所以KoB幾乎都是靠亞絲娜在管理。
      • 有著KoB副團長身分後，攻略各樓層BOSS的會議皆由她來主持，且態度嚴厲。
      • 不過在某次攻略BOSS時，提出了犧牲城鎮中的NPC來攻略BOSS的方案，卻被桐人以不犧牲任何一名NPC的方式打倒BOSS後而有了改觀。
    • 之後多次和桐人有所接觸而被他吸引，在主動追求下成為了情侶。
      • 克拉帝爾事件後退出KoB，並在當晚和桐人…
    • 然後過了大約兩週左右的新婚生活，並在這期間遇見了兩人的「女兒」結衣；從希茲克利夫那得知75層BOSS的攻略出問題後和桐人再度回到前線。
    • 75層BOSS被打倒，並於希茲克利夫被桐人揭穿其就是茅場晶彥的事實後被茅場以系統限制住，但在桐人敗給對方後突破系統限制的替他擋下致命一擊而死亡。
    • 恢復意識後只看見艾恩葛朗特逐漸崩毀，並發現自己和桐人並未死亡，在聽完茅場究竟為何要這麼做之後與桐人交換了現實的情報。
      • 不過並沒有醒來，而是被看準機會的須鄉伸之將她和其他SAO生還者共三百名轉移到ALO之中囚禁著。
  • ALO（妖精之舞篇）
    • 以妖精王王妃「蒂塔妮亞」的角色被囚禁在世界樹頂的籠子，日夜被身為遊戲GM的須鄉調戲。
    • 曾想過要逃跑並付諸實行，但因為無法打開被程式封鎖的大門而失敗。
      • 但是弄到了GM程式的卡片，並丟下世界樹被桐人撿到。
    • 桐人突破重圍來到世界樹頂端後被須鄉以系統限制、玩弄，自己也險些被須鄉侵犯。
      • 不過桐人得到茅場暗助，將須鄉打倒後獲救，也成功令明日奈登出。
    • 回到現實世界後，在醫院第一次和桐人的本尊桐谷和人見面。[6]
    • 之後進入政府為SAO生還者成立的特殊學校就讀，並與和人過著甜蜜的學生生活，亦告知他RECT的後續。
    • ALO被新公司買下重新開始營運後，將自己的種族設定為擅長治療魔法的水精靈，但因為時常拋下治療師的工作並持劍衝上前線，而獲得了不優雅的「狂暴補師」稱號。
  • GGO（幽靈子彈篇）
    • 在皇居與和人約會，並得知他要把桐人轉移到GGO的事
    • GGO決賽當日和莉法、莉茲貝特、西莉卡、克萊因等人於ALO中觀看轉播，在看見死槍的微笑棺木印記後找來委託和人的菊岡誠二郎。
    • 知道委託的實際內容後，趕到和人所在的醫院支持著他。
    • 死槍事件結束後，在艾基爾開設的咖啡廳中和詩乃見面。終於知道和人在GGO中的偷腥貓的真正身份。
  • 新生ALO（聖母聖詠篇）
    • 在ALO自己與桐人購買的小屋中，從莉茲貝特那知道了四處挑戰他人、並以有著十一連擊的OSS[7]當賭注的玩家「絕劍」的消息。
    • 下線後與母親共進晚餐，並因為轉校、結婚對象的事發生了爭執。
      • 母親認為明日奈應該走家族的精英路線──及早上頂尖大學、和名門成婚來拓展自己的事業。
      • 但明日奈不以為然，直指母親吸收不了ALO事件的教訓，甚至出言諷刺母親是不滿自己並非出身名門、看不起在貧困農作中拉拔她長大的外祖父母，才如此執著要明日奈嫁入豪門。
    • 與「絕劍」有紀在ALO的艾恩葛朗特浮游城24層相遇，被莉茲貝特推出來後展開挑戰，結果敗在對方的OSS之下。
    • 挑戰結束後，被有紀拉去和沉睡騎士的團員阿淳、提奇、達爾肯、小紀、朱涅見面，並被拜託幫忙協助沉睡騎士完成攻略BOSS好留名在劍士之碑的計畫。
      • 雖然有點疑惑，不過最後還是同意幫忙了。
    • 之後被母親強迫斷線後再度有了爭執，並逃出家在夜裡哭泣。
    • 第一次和沉睡騎士出動攻略BOSS失敗，發現其他玩家團隊的蹤跡後提出立即進行第二次攻略的計畫。
    • 在BOSS的房間前被其他玩家團隊阻止進入，正當危急時桐人與克萊因趕到阻止玩家團隊，亞絲娜則和沉睡騎士團員一同進入BOSS房間內展開攻略。
    • 擊敗BOSS後與有紀和其他沉睡騎士團員們一同慶祝，但在提出要加入沉睡騎士的要求沒有得到有紀的回應，並看到對方流淚後下線的模樣。
    • 在和人提供情報後，於橫濱都筑區的醫院裡知道了有紀的本尊紺野木棉季是醫療用完全潛行機器「Medicuboid」的試驗者，與她已是AIDS末期症狀的事實。
    • 和有紀於ALO中對話，並得知沉睡騎士團員全是絕症患者，和為何要執著於在劍士之碑留名的真相。
      • 在有紀提出想去學校的願望後，在和人的幫助下透過明日奈的手機連線完成了這願望。
    • 之後在有紀的要求下去了有紀的家，聽完有紀的話後在附近的公園向有紀訴說了和母親無法互相理解的事。
      • 在這之間，有紀提到了和人和自己其實都有「在不同的意義下，活在非現實世界」的感覺。
    • 明日奈在有紀的開導下決定和母親好好對話一次，之後透過AmuSphere重現了類似母親老家的場景後談了外祖父母的事，之後母女總算互相理解。
      • 比起為了自己而向前跑的人生，明日奈更希望像外祖父母那樣，成為他人的支柱、帶給身邊的人幸福而活著。
      • 母親理解她的心意與覺悟後，接受她留在現在學校、與桐人交往的決定。但也說想支持他人的話，自己就要先變得強大，因此要求明日奈至少要讀大學。
    • 在桐人等夥伴的幫助下，和有紀與沉睡騎士的團員們在ALO與現實世界中進行了各式各樣的冒險與旅行。
    • 約於四月底時得到有紀病情急轉直下的消息，之後兩人在ALO中再會，有紀將自己的OSS「聖母聖詠」交給亞絲娜後，在眾人的守護下於亞絲娜的懷中過世。
    • 在有紀的葬禮上領悟了許多道理，並決定要將有紀的事傳給自己的孩子。
    • 葬禮後，和朱涅的本尊安施恩碰面，在安慰了對方後從和人與有紀的主治醫師處知道了Medicuboid的開發者，並從和人的談話中知道了Medicuboid的基礎設計者其實是茅場晶彥的事而百感交集。
  • Underworld（Alicization篇）
    • 在艾基爾的咖啡廳裡與和人、詩乃會面，並聊起了和人在RATH公司打工的各種已知訊息。
    • 結束後與和人一同回家，結果遭到死槍事件中在逃的第三名死槍金本敦的襲擊，和人被對方注射了琥珀膽鹼而倒下，明日奈則在救護車上目睹了和人心跳停止的一幕。
    • 其後在與直葉、詩乃討論此事時，從女兒結衣那裡取得的資料得知失蹤後的桐人所在處，以及神代凜子的幫助下，順利的安全潛入Ocean Turtle，並找尋到目前在Soul Translator中進行神經復原的和人，也得知菊岡誠二郎等人暗中進行的「A.L.I.C.E」計劃的真相。
    • 為了鼓勵失意且靈魂受損的桐人而使用GM帳號「生命之神史提西亞（ステイシア）」登入Underworld。
    • 帶領著好友們與日本玩家並肩作戰抵抗PoH所煽惑的數萬中韓玩家聯合軍。在絕望之際感受到有紀在天之靈的鼓勵而使出UW中本應不存在的OSS──聖母聖詠反擊PoH。而歷經這些惡戰後終於等到桐人甦醒，被他救下，最後贏得UW戰爭的勝利。
    • 在桐人爭取時間下，保護愛麗絲離開UW到達現實世界。此時UW的極限加速被開啟，遊戲內的時間速度變成現實世界的五百萬倍，亞絲娜自願留下來陪伴因戰鬥錯過登出機會的桐人。
    • 兩人一同留在Underworld內近二百年的時間，在這期間與桐人結婚被民眾推舉為Underworld的星王后，與身為星王的桐人一起治理整個Underworld。
    • 經過了現實世界的兩個月後，於現實世界的愛麗絲的呼喚下奇蹟似與桐人一同醒來，爾後與桐人協議在這百年間的記憶全數刪除。
  • 劇場版：序列爭戰
    • 時間上設定於聖母聖詠篇與Alicization篇之間。
      • 此時離有紀過世約一個月，明日奈與母親的關係也已經好轉，母親甚至想見見和人。
    • 有寫日記的習慣，很重視在SAO時期許下的約定──讓桐人帶著亞絲娜去看流星。
    • 在AR──擴增實境遊戲「序列爭戰」中表現突出，有非常高的排名。和人則對AR不太抱持興趣。
    • 在數次BOSS戰中以MVP之姿率眾擊倒BOSS。但在一次戰鬥中，Eiji故意將希莉卡推倒在BOSS前，亞絲娜為了保護希莉卡不受傷而被BOSS擊敗。此時藏於AR裝置中的程式啟動，奪走了亞絲娜於SAO時期的記憶。
    • 雖然與桐人的感情還是存在，仍會因為無法想起與桐人相識相知的記憶而難過，因此恐懼戰鬥、甚至有些害怕出門；但同時，也在日記中鼓勵著自己可以和桐人創造新的回憶。
      • 察覺明日奈心意的和人將她推倒床上，承諾自己一定會取回她的記憶。之後兩人有沒有H就請自行想像，據導演說本來是要把床戲做出來的。
    • 與希莉卡等人參加YUNA的演唱會，但這其實是一個誘捕SAO生還者記憶的大型陰謀。SAO的BOSS們接連出現並攻擊觀眾，危急時剛結束與Eiji戰鬥的和人前來救助無法戰鬥的明日奈，並給了她禮物──在SAO約定好、要在現實中送她的訂婚戒指。
    • 和人等人利用AR裝置中的完全潛行功能進入第一百層打算解決事件，留下明日奈在現實中守候。此時人工智慧悠那告知，一旦現場玩家都被恐懼和BOSS壓垮，AR裝置就會啟動大功率掃描，以穿戴者死亡的代價來收集記憶。
    • 雖然一度想幫和人拿下裝置，但深知和人不會選擇這種只有自己逃避的道路。於是自行克服了恐懼、拜託優娜讓她也潛行，幫助苦戰中的桐人。
    • 於SAO頂層紅玉宮，SAO、ALO、GGO時期的所有朋友們皆來助陣，結衣並讓亞絲娜等人穿回SAO時期的裝備。亞絲娜與有紀的幻影一起使出劍技「聖母聖詠」、與桐人將BOSS打敗。
    • 正式將SAO全部通關的桐人以獎勵的超強武器回到現實，將所有BOSS輕鬆掃平、結束了會場的混亂。悠那前來將記憶歸還給明日奈後消失。
    • 結尾，在約好看流星的日子，與和人到山上一起實現約定。明日奈將戒指交到和人手中，讓他親手為她戴上，兩人最後在浪漫的夜空下擁吻。
      • 在ED中，明日奈最終帶著和人回自己家見母親。
